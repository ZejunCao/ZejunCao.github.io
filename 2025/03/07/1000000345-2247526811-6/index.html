310-B2.1｜可动画化3D头像与人体重建系列；3D说话头系列

AIGCResearch
主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）｜编辑支持｜李观丽（GuanliLi）发布日期｜2025年2月28日｜周五
2025-02-27｜TUMunich,MetaRealityLabs
http ://arxiv.org/abs/2502.20220v1https ://tobias-kirschstein.github.io/avat3r/
Avat3r是一个创新的模型，旨在从仅四张输入图片中重建高质量、可动画的3D人头头像。传统的3D头像创建通常需要昂贵的多视角捕捉设备和复杂的后期处理，限制了其在日常场景中的应用。Avat3r通过结合先进的3D重建技术和深度学习方法，显著降低了计算需求，使得用户只需使用手机拍摄的几张照片即可快速生成逼真的3D头像。该方法能够处理输入图像的不一致性，例如不同的面部表情，并且无需长时间录制面部动作，极大地简化了3D头像的捕捉过程。
Avat3r的核心方法分为几个关键步骤：
输入处理：模型接受V张输入图像I，并利用DUSt3R和Sapiens预训练模型生成每张图像的位置信息和特征图。这些信息为后续的3D重建提供了基础。
特征提取：通过VisionTransformer架构，模型对输入图像进行patch化处理，并将位置信息与特征图结合，以生成低分辨率的中间特征图。
动画生成：利用交叉注意力机制，模型能够根据表达代码动态调整生成的3D头像的面部表情，确保动画的自然流畅。
高质量输出：最终，模型将生成的高维高斯属性图进行上采样，并通过自定义的渲染过程，输出可从任意视角观察的3D头像。
在实验部分，Avat3r的性能通过与现有的3D头像生成方法进行比较来验证。研究者使用了Ava256数据集进行训练，评估了模型在创建3D头像的能力，包括从单一或多个输入图像生成头像的效果。通过多种指标（如PSNR、SSIM和LPIPS）来量化生成图像的质量，结果显示Avat3r在多个评估标准上均优于现有方法。此外，模型在处理不同光照和视角条件下的输入图像时表现出良好的鲁棒性，能够生成高质量的3D头像，证明其在实际应用中的潜力。
Avat3r的工作原理可以简单理解为：它从几张照片中提取出人脸的关键信息，然后利用这些信息创建一个3D模型。首先，模型会分析每张照片，找出人脸的具体位置和特征，比如脸的形状、颜色等。接着，它会把这些信息结合起来，形成一个完整的三维头像。为了让这个头像能表现出不同的表情，Avat3r还会根据用户的需求调整面部表情。最后，生成的3D头像可以从任何角度查看，就像在真实世界中一样。这一过程不仅快速，而且只需要普通的手机拍摄几张照片，就能生成高质量的3D头像，极大地方便了用户。
2025-02-27｜Meta,RutgersU
http ://arxiv.org/abs/2502.19739v1
LUCAS（LayeredUniversalCodecAvatars）是一个创新的三维头像重建模型，旨在提升虚拟环境中的真实感和互动体验。该模型通过将面部和头发分离建模，克服了传统头像重建方法在动态场景中面临的挑战，如面部表情和头发运动的复杂交互。LUCAS引入了一种分层的通用先验模型（UniversalPriorModel,UPM），使得在实时渲染和高保真度生成之间取得平衡。该方法不仅支持在移动设备上进行实时渲染，还能在未见过的身份上实现准确的表情和姿势转移。实验表明，LUCAS在动态性能和视觉质量上均优于现有的单网格和高斯头像模型，推动了三维头像重建的技术进步。
LUCAS的核心方法包括以下几个方面：
分层建模：将面部和头发的建模分开，使其独立变形，从而提高了动态表现的准确性。
通用先验模型：利用多用户数据进行训练，使模型具备跨身份的泛化能力，能够适应未见过的用户。
表达编码器：通过表达编码器生成统一的表达代码，确保面部和头发的同步变形。
像素解码器：采用像素级别的解码技术，直接从每个像素生成颜色，避免了固定纹理图的限制。
多网格渲染：通过将面部和头发的几何体图结合，进行联合处理，实现更精细的渲染效果。
在实验部分，LUCAS采用了三种广泛使用的评估指标：峰值信噪比（PSNR）、结构相似性指数（SSIM）和学习感知图像补丁相似性（LPIPS）进行性能评估。实验结果显示，LUCAS在多个身份的重建中均表现出色，特别是在处理细微的面部和头发细节方面。与现有的uPiCA和URAvatar等方法相比，LUCAS在动态场景中的表现更为优越，能够准确跟踪头发的运动，并在不同头部姿势下保持良好的视觉效果。此外，LUCAS在零-shot驾驶场景中也展现了强大的泛化能力，能够在未见过的身份上准确转移表情。
LUCAS的工作原理可以简单理解为将一个人的头像分成两个部分：脸和头发。通过这种分离，LUCAS可以让脸和头发根据不同的表情和动作独立地移动，避免了传统方法中常见的头发和脸部不协调的问题。它使用了一种叫做通用先验模型的技术，这意味着它在训练时学习了很多人的数据，这样即使面对新的人，LUCAS也能快速适应并生成真实的头像。此外，LUCAS的解码过程非常聪明，能直接从每个像素生成颜色，这样可以避免一些传统方法中的限制。总之，LUCAS让虚拟头像的表现更自然，互动体验更真实。
2025-02-24｜SHU,FDU,TavusInc
http ://arxiv.org/abs/2502.19441v1
在近年来，基于高斯模型的人体重建技术取得了显著进展，尤其是在创建可动画化的虚拟化身方面。然而，现有的模型在充分利用SMPL（SkinnedMulti-PersonLinear）模型的先验知识和提升视觉效果方面仍面临挑战。本文提出了AniGaussian，一个新型框架，旨在解决这些问题。首先，我们引入了一种创新的姿态引导变形策略，使得动态高斯化身能够更好地捕捉细微的表面特征，同时在多种运动中保持解剖学的正确性。其次，我们解决了高斯模型在表现动态人体方面的局限性，结合了基于刚体的先验信息，以增强高斯模型的动态变换能力。通过引入分裂与缩放策略，AniGaussian显著提高了几何质量。我们的实验结果表明，该模型在定性和定量指标上均优于现有方法。
AniGaussian的核心方法包括三个主要部分：首先是姿态引导变形框架，结合了非刚性和刚性变形，以扩展3D高斯模型的应用。具体而言，我们使用SMPL模型提供的姿态参数作为引导，采用多层感知器（MLP）来学习非刚性变形，其输入为高斯点云的位置和SMPL模型的对应顶点。其次，刚性变形通过将高斯模型从规范空间转变到观察空间来实现，确保在不同视角下的几何一致性。最后，为了解决视觉伪影问题，我们引入了一种新的高斯分裂与缩放策略，以提升模型在细节丰富区域的表现。通过这种方法，AniGaussian能够在保持高质量视觉效果的同时，显著提高训练效率和稳定性。
在实验部分，我们对AniGaussian在两个数据集上的表现进行了评估，分别是PeopleSnapshot和ZJU-MoCap数据集。我们通过比较不同方法的合成结果，验证了AniGaussian在新视角合成和新姿态合成任务中的优越性。在PeopleSnapshot数据集中，AniGaussian在多个指标上表现出色，包括峰值信噪比（PSNR）、结构相似性指数（SSIM）和感知图像补丁相似性（LPIPS），显示出其在细节恢复和几何重建方面的优势。此外，通过消融研究，我们分析了各个组件对整体性能的贡献，证实了姿态引导变形和分裂与缩放策略对模型效果的重要性。最终结果表明，AniGaussian在训练时间和渲染速度方面也具有显著的优势。
AniGaussian的工作原理可以简单理解为一个智能的动画化身生成器。首先，它利用一个叫做SMPL的模型来了解人的身体结构和姿势。然后，通过一个复杂的计算过程，它可以根据这些姿势来调整和变形一个三维的高斯模型，确保这个模型在不同的动作下看起来依然自然。为了让这个过程更高效，AniGaussian还引入了一种方法，将大的高斯模型拆分成多个小的模型，这样可以保持更清晰的细节和更好的图像质量。最终，AniGaussian不仅能快速生成高质量的人物模型，还能在不同的视角和姿势下保持一致性，使得这些虚拟角色看起来更加真实和生动。
2025-02-27｜BUAA,GriffithU,RIKENAIP,UTokyo｜CVPR2025
http ://arxiv.org/abs/2502.20387v1https ://fictionarry.github.io/InsTaG/
InsTaG是一种创新的3D说话人合成框架，旨在从仅5秒的视频数据中快速学习个性化的3D说话头。现有基于神经辐射场（NeRF）的方法在处理新身份时，通常需要大量的训练数据和时间，而InsTaG通过引入轻量级的3D高斯点云（3DGS）技术，克服了这些限制。该框架实现了高质量的个性化唇同步和真实感渲染，同时保持低内存开销和实时推理能力。InsTaG的核心在于两项关键策略：身份无关的预训练策略和运动对齐适应策略，使得模型能够在少量数据下快速适应新身份，生成高保真度的个性化3D说话头。
InsTaG的工作流程主要分为两个阶段：预训练和适应。预训练阶段采用身份无关的预训练策略，旨在从长视频数据集中提取通用运动先验，避免身份冲突。具体步骤包括：
身份无关预训练：构建一个通用运动场，提取和存储多个视频中的共同运动知识。
运动对齐适应：将预训练的运动场与新身份的头部特征进行对齐，以实现快速适应。此策略包括：
运动对齐器：学习条件独立的坐标偏移和运动缩放因子，以确保个性化和通用运动的最佳匹配。
面部-口腔钩技术：通过将口腔运动与面部运动对齐，增强合成的鲁棒性和真实感。这些方法结合，使得InsTaG能够在极少的数据下生成高质量的个性化3D说话头。
在实验部分，InsTaG通过与多种现有技术的比较，展示了其在图像质量、个性化动态和适应速度等方面的优势。实验设置包括自重建和跨领域测试，评估指标涵盖PSNR、LPIPS、SSIM等。结果表明，InsTaG在自重建设置中表现最佳，尤其是在处理5秒训练数据时。通过对比，InsTaG的唇同步精度和视觉质量均优于其他方法，且在不同身份、性别和语言的场景下展现出卓越的适应性。用户研究也显示，InsTaG在视频真实感和身份保留方面得到了参与者的高度评价，进一步验证了其有效性。
InsTaG的工作原理可以简单理解为两个主要步骤。首先，在预训练阶段，系统从很多长时间的视频中学习说话人的共同特征，而不关注具体的身份。这就像在一个大课堂上，老师教学生们一些普遍的知识，而不是只教某个学生的个别情况。接下来，当我们给系统一个新的短视频（比如5秒钟）时，它会使用之前学到的知识，快速调整并生成一个个性化的3D说话头。这个过程就像一个学生在考试时，能迅速运用之前学到的知识来解答新的问题。为了确保说话头的嘴巴和脸部动作协调，系统还设计了一种“钩子”技术，帮助嘴巴的动作与脸部动作保持一致。这种方法使得InsTaG在短时间内就能生成高质量的、个性化的说话人动画。
2025-02-27｜UTokyo,RIKENAIP
http ://arxiv.org/abs/2502.20323v1http ://xg-chu.site/project_artalk/
ARTalk是一种新颖的自回归模型，旨在实时生成基于语音的3D头部动画，能够生成高度同步的唇动和逼真的头部姿态。这一研究背景源于现有的扩散模型虽然能够生成自然的动画，但其生成速度较慢，限制了其应用潜力。ARTalk通过学习语音与多尺度运动代码本之间的映射，实现了实时生成。这种方法不仅能够适应未见过的说话风格，还能生成具有独特个人风格的3D虚拟角色。研究表明，ARTalk在唇同步精度和生成质量上均优于现有方法，展现出广泛的应用潜力。
ARTalk的核心方法包括以下几个关键步骤：
多尺度运动VQ自编码器：该模型首先将运动序列编码为多尺度的离散运动表示。通过共享的代码本，模型能够高效处理复杂的运动动态。
条件自回归生成器：该生成器在生成每个时间窗口的运动代码时，基于当前窗口的语音输入和前一个窗口生成的运动，确保了运动的时间连续性。
风格编码器：集成的变换器风格编码器从样本运动序列中提取风格代码，使得生成的动画能够反映个体特征，简化了语音与运动之间的复杂映射关系。
训练与推理：模型通过使用混合损失函数进行训练，确保生成的运动具有高保真度和时间一致性。在推理阶段，模型能够快速生成与输入语音相适应的动画序列。
ARTalk的实验部分主要包括数据集的选择、定量和定性评估。研究使用了TFHP数据集，包含来自588个受试者的1,052个视频片段，经过精确跟踪，生成了约238.5万个动作帧。定量评估中，ARTalk在唇同步精度、风格对齐和生成质量等指标上均显著优于现有的基线方法。定性评估则通过用户研究，比较了不同方法在唇同步、面部表情和风格一致性上的表现，结果显示ARTalk在这些方面均获得了较高的用户满意度。此外，ARTalk在生成效率上也表现出色，能够在短时间内生成高质量的动画。
ARTalk的工作原理可以简单理解为一个“翻译”过程。首先，它将语音信号“听懂”，然后把这些声音转换成动画的动作。这个过程分为几个步骤：首先，ARTalk会把说话的声音分成小段，类似于把一段音乐切成小节；接着，它会把每一小段声音转化为动作代码，这些代码就像是动画的指令。然后，ARTalk会根据这些指令生成动画，这个过程是实时的，意味着用户几乎可以立刻看到动画效果。最后，ARTalk还可以学习不同人的说话风格，生成个性化的动画，使得每个虚拟角色都能表现出独特的风格和情感。

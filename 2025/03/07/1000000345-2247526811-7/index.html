310-B2.2｜VR环境360度视频质量评估；单目动态场景渲染，无需相机内外参的GS；基于物理的HOI模拟

AIGCResearch
主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）｜编辑支持｜李观丽（GuanliLi）发布日期｜2025年2月28日｜周五
2025-02-27｜BUAA,THU
http ://arxiv.org/abs/2502.19644v1https ://github.com/ZhouKanglei/ASAL_CVQA
随着虚拟现实（VR）技术的发展，360度视频的质量评估变得愈发重要。用户在使用VR设备时，往往会遭遇视频失真，这会显著影响他们的沉浸体验。传统的虚拟现实视频质量评估（VR-VQA）方法通常依赖于静态数据集进行训练，难以适应动态变化的内容分布。为了解决这些问题，本文提出了一种新颖的方法——自适应评分对齐学习（ASAL）。ASAL通过结合相关性损失和误差损失，增强了预测结果与人类主观评分之间的对齐程度，并提高了对感知质量的预测精度。此外，ASAL还引入了自适应记忆重放机制，旨在解决VR设备的计算和存储限制，从而实现对不断变化的VR环境的持续适应。
ASAL方法的核心在于两个主要组成部分：评分对齐学习和自适应记忆重放。首先，评分对齐学习通过以下步骤实现：
特征平滑：利用重新参数化策略平滑特征空间，以提升模型对未见内容的泛化能力。
双目标优化：结合相关性损失和精度损失，确保模型不仅与人类评分一致（相关性），还具备高预测精度。
损失函数设计：通过最小化预测误差和最大化与人类评分的一致性，优化模型性能。自适应记忆重放则通过以下方式增强模型的学习能力：
关键帧提取：从视频中选取具有代表性的关键帧，减少存储需求。
特征适配：将关键帧的特征重构为完整视频特征，以提高计算效率并保持时间一致性。通过这两个部分，ASAL能够有效应对VR内容的非静态变化，同时满足存储和计算效率的要求。
为了验证ASAL方法的有效性，研究者们进行了全面的实验评估。首先，建立了一个综合基准测试，以比较ASAL与其他基线模型的性能。实验结果显示，ASAL在静态联合训练设置中，相关性提升达4.78%，而在动态持续学习设置中，相关性提升更是达到12.19%。这些结果表明，ASAL在处理VR视频质量评估中的固有挑战方面具有显著优势。此外，研究团队还引入了新的数据拆分策略和评估指标，以确保评估的全面性和公正性。通过对不同数据集的测试，ASAL在多个类别中均表现出优异的性能，证明了其在真实世界应用中的适用性和有效性。
ASAL方法的工作原理可以简单理解为一个双管齐下的策略，旨在提高VR视频的质量评估。首先，它通过一个叫做“评分对齐学习”的过程，使得计算机的评分结果更接近人类的主观评价。这就像是让计算机学习如何理解和评估视频质量，以便它能给出更准确的评分。其次，ASAL还引入了“自适应记忆重放”，这意味着它能记住之前学习的内容，并在需要时提取出重要的信息。通过选择关键帧，ASAL减少了存储空间的需求，同时确保了视频的完整性。整体来看，这种方法不仅提高了视频质量评估的准确性，还使得计算机能在面对不断变化的内容时，依然保持高效的学习能力。
2025-02-27｜NUS｜AAAI2025
http ://arxiv.org/abs/2502.20378v1
在动态场景渲染领域，如何从单目视频中高效地重建三维场景是一项重要而复杂的任务。本文提出了一种名为高效动态高斯点云（EDGS）的新方法，旨在解决当前动态场景渲染中存在的效率低下和冗余高斯点的问题。以往的方法往往需要处理大量冗余的高斯点，导致渲染速度缓慢且画面抖动。EDGS通过引入稀疏时间变化属性建模，利用稀疏锚点网格表示动态场景，并通过经典的核表示来计算运动流，从而显著提高渲染速度和图像质量。实验结果表明，EDGS在多个真实世界数据集上都取得了优于现有最先进技术的表现。
EDGS的核心在于高效建模动态场景的时间变化属性。其方法步骤如下：
锚点网格初始化：从稀疏点云中初始化锚点网格，捕捉场景的几何和外观信息。
属性推导：使用轻量级多层感知机（MLP）解码高斯点的时间不变属性（如颜色和不透明度），并筛选出与静态区域相关的锚点。
运动建模：通过经典的径向基函数（RBF）核计算锚点的运动流，以稀疏的方式建模可变形高斯点的运动。
时间掩码：引入时间掩码MLP，有效区分静态和动态区域，仅对动态区域的锚点进行时间变化属性查询，进一步减少冗余和提升渲染效率。
高斯点密集化与修剪：在渲染过程中，通过计算梯度对锚点进行密集化，并剔除不必要的冗余锚点，以优化渲染性能。
在实验部分，EDGS在两个真实世界数据集上进行了广泛的评估：NeRF-DS和HyperNeRF。通过多种评估指标（如峰值信噪比PSNR、结构相似性SSIM和学习感知图像块相似性LPIPS），结果显示EDGS在渲染速度和质量上均优于现有的最先进方法。例如，在NeRF-DS数据集上，EDGS实现了最高的PSNR和SSIM分数，且渲染速度达到了117帧每秒，显著高于其他方法。同时，EDGS在静态区域的渲染中表现出更少的抖动现象，确保了更高的视觉稳定性。通过对比实验和消融研究，验证了锚点-体素策略和时间掩码的有效性。
EDGS方法的核心是如何高效地处理动态场景中的信息。首先，它从一个稀疏的三维点云中创建了一组“锚点”，这些锚点就像是场景中的关键位置。接着，它利用小型神经网络来提取这些锚点的颜色和透明度等信息，同时识别出哪些锚点是静态的，哪些是动态的。对于动态部分，EDGS使用了一种数学工具（径向基函数）来计算这些锚点在不同时间点的运动，从而更准确地模拟场景变化。为了提高效率，EDGS只关注那些动态的锚点，避免了不必要的计算。这种方法不仅加快了渲染速度，还减少了画面抖动，使得最终生成的图像更为平滑和清晰。
2025-02-27｜USTC,IndividualResearcher
http ://arxiv.org/abs/2502.19800v1
本研究提出了一种新颖的3D高斯点云渲染方法，旨在解决在没有已知相机内外参数的情况下进行3D高斯点云绘制的问题。传统的3D高斯点云方法依赖于预先计算的相机参数，如焦距和相机姿态，这不仅耗时而且在复杂场景中容易产生误差。为了克服这一限制，本文提出了一种联合优化策略，能够在图像集合中同时优化相机的内外参数和3D高斯模型。通过理论推导相机内参的梯度，研究中实现了在训练过程中动态更新这些参数。此外，结合全局跟踪信息和多视图几何一致性，显著提高了渲染质量和相机姿态的估计精度。经过广泛的实验验证，提出的方法在多个公共和合成数据集上均表现出色，达到了当前的最先进水平。
本研究的核心方法是联合优化3D高斯点云模型和相机参数的训练过程。具体步骤如下：
相机参数初始化：使用图像对应关系和深度信息来初步估计相机的内外参数。
梯度推导：通过理论推导相机内参的梯度，使得在训练过程中可以通过反向传播更新这些参数。
全局跟踪信息整合：利用跟踪的3D点来选择与每个跟踪点相关的高斯核，确保训练过程中多视图几何的一致性。
损失函数设计：结合重投影损失、跟踪损失和尺度损失，构建综合损失函数，以优化相机参数和3D高斯模型。
训练过程：通过不断迭代优化，调整高斯核的大小和位置，确保其分布在场景表面附近，最终实现高质量的图像合成。
在实验部分，研究团队在多个真实世界和合成数据集上进行了广泛的评估。使用的主要数据集包括TanksandTemples和CO3D-V2，涵盖了多种场景和相机运动模式。实验结果显示，提出的方法在新视图合成和相机姿态估计上均优于现有的基线方法。具体而言，在TanksandTemples数据集中，所提方法在图像质量指标上（如PSNR和SSIM）表现出色，且在CO3D-V2数据集中，尤其在复杂相机运动下，准确性显著提升。通过与COLMAP辅助的3D高斯点云模型的比较，结果表明，本文方法在合成质量和相机参数估计上均取得了优越的性能，为未来的3D重建和视图合成提供了新的解决方案。
在这项研究中，科学家们开发了一种新方法来生成3D图像，而不需要知道相机的具体设置。想象一下，你在拍摄一组照片，但你并不知道相机的焦距或角度。这个方法通过分析这些照片来推测这些参数。它的工作原理是：首先，使用这些照片来估算相机的基本设置，然后通过一种数学方法来不断调整这些设置，确保生成的3D图像尽可能真实。研究者还利用了一些额外的信息，比如图像之间的相似性，来帮助提高图像质量。最终，他们在多个不同的数据集上测试了这个方法，结果显示它的表现比之前的技术要好。这意味着，即使在复杂的拍摄条件下，这种新方法也能提供清晰、准确的3D图像。
2025-02-27｜UIUC,ElectronicArts｜CVPR2025
http ://arxiv.org/abs/2502.20390v1https ://sirui-xu.github.io/InterMimic/
InterMimic是一种新颖的框架，旨在实现物理基础的人类与动态物体的交互模拟。该研究的核心目标是通过利用不完美的运动捕捉（MoCap）数据，训练出一种能够泛化的控制策略，使得模拟人类能够与各种形状和动态的物体进行复杂的全身交互。框架的创新之处在于采用了教师-学生的训练策略，其中教师政策通过模仿和精炼数据来指导学生政策的学习。InterMimic不仅能够处理多物体的动态交互，还能通过强化学习（RL）微调，提升策略的质量，超越简单的演示复制。实验结果表明，InterMimic在多个人类-物体交互数据集上表现出色，展示了其在生成复杂交互方面的潜力。
InterMimic的方法分为几个关键步骤：首先，定义任务形式，即学习一个控制策略π，以生成与给定参考数据高度匹配的模拟人类-物体运动。其次，采用Markov决策过程（MDP）来框定学习问题，包括状态、动作、转移动态和奖励函数。接着，采用两阶段的训练框架：第一阶段是训练多个教师政策，这些政策在小规模数据子集上并行训练，专注于模仿和精炼运动数据；第二阶段是将这些教师政策的知识蒸馏到一个可扩展的学生政策中。为了应对MoCap数据中的不完美，设计了接触引导奖励机制，并优化轨迹收集，以提高学习效率。此外，采用物理状态初始化（PSI）来改善初始状态的质量，从而减少训练中的错误。
在实验中，研究者评估了教师政策在模仿不完美的人类-物体交互参考方面的能力，并测试整个教师-学生框架在大规模数据集上的可扩展性和零-shot泛化能力。实验结果表明，InterMimic在成功率、模仿持续时间和跟踪误差等指标上均优于现有基准方法。具体而言，教师政策能够有效修正MoCap中的错误，如手部位置不准确和接触浮动等问题，从而实现更长时间的交互跟踪。此外，研究还通过消融实验验证了物理状态初始化和交互早期终止机制的有效性，进一步证明了参考蒸馏在提升性能方面的重要性。这些实验结果展示了InterMimic在处理复杂交互任务时的优势。
InterMimic的工作原理可以简单理解为一个老师和学生的学习过程。首先，系统会创建多个“老师”政策，这些老师会在小规模的数据上学习如何模仿人类与物体的互动。老师们会从各种不完美的运动捕捉数据中提取动作，并进行修正，确保这些动作看起来更自然和真实。然后，学生政策会从这些老师那里学习，吸收他们的知识，最终形成一个能够处理多种复杂动作的智能体。为了让学习过程更加有效，系统还设计了一种奖励机制，帮助学生在模仿时能够更好地处理动作中的接触问题。此外，系统会使用一种叫做物理状态初始化的技术，确保学生在开始学习时的状态更接近真实情况，从而提高学习的质量和效率。这样，InterMimic就能让模拟的人类在与不同物体互动时表现得更自然和灵活。

2025-02-27｜CQUPT,Meituan,GreatBayU,UMacau｜⭐️⭐️
http ://arxiv.org/abs/2502.20307v1https ://mobius-diffusion.github.io/
本研究提出了一种名为Mobius的新方法，旨在从文本描述直接生成无缝循环视频。与传统方法不同，Mobius不需要用户注释或图像作为外观参考，而是利用预训练的文本到视频潜在扩散模型，创造出更具动态性和视觉质量的循环视频。该方法通过构建潜在循环，连接视频的起始和结束噪声，确保视频在生成过程中保持时间一致性。此外，Mobius的潜在循环可以是任意长度，从而扩展了循环视频的生成能力，适用于多种应用场景。
Mobius的方法主要包括以下几个步骤：
文本到视频潜在扩散模型：利用预训练的模型，将文本描述转换为潜在空间中的视频表示。
潜在循环构建：通过连接视频的首尾潜在噪声，构建一个潜在循环列表，为后续的去噪过程做好准备。
潜在位移策略：在每次去噪步骤中，将首帧的潜在表示逐步移动到最后一帧，通过这种方式实现视频的循环生成。
帧不变解码：为了解决由于3D变分自编码器（3DVAE）解码导致的伪影问题，设计了一种帧不变的解码方法，确保每帧在时间维度上的一致性。
旋转位置编码插值：通过插值方法修正潜在空间中的位置编码，以支持生成更长的循环视频，提升生成效果。
在实验中，研究团队基于开源的CogVideoX模型进行验证，采用480x720的分辨率和50步的推理设置。通过选取140个文本提示，评估生成视频的质量和一致性。实验结果显示，Mobius在视觉质量、时间一致性和动态表现等方面优于现有的基准方法。具体评估指标包括均方误差（MSE）、Frechet视频距离（FVD）和CLIP评分。此外，研究还进行了用户研究，参与者在视觉质量、视频一致性和动态性方面对Mobius进行了排名，结果表明该方法在各项指标上均表现优异。
Mobius的工作原理可以简单理解为将文本描述转化为视频。首先，系统会读取你给出的文字描述，然后使用一个已经训练好的模型来生成视频。这个过程不需要任何额外的图片，只需文字就能开始。接下来，系统会创建一个循环的视频效果，这就像把视频的开头和结尾连接在一起，让它看起来可以无限重复。为了确保每一帧都流畅且一致，Mobius会在生成过程中不断调整每一帧的位置，使得视频的每一部分都能自然地过渡。最后，这种方法还可以生成更长的视频，适用于多种场景，创造出更加生动的视觉效果。
2025-02-27｜NWPU,MBZUAI,9009.ai,LinkopingU,ANU｜⭐️🟡
http ://arxiv.org/abs/2502.19868v1https ://github.com/WesLee88524/C-Drag-Official-Repo
C-Drag是一种基于链式思维的运动控制器，旨在实现可控视频生成，尤其是在多物体交互场景中。传统的基于轨迹的方法通常仅关注生成物体的运动轨迹，忽略了物体与周围环境之间的动态交互。C-Drag通过引入物体感知模块和链式思维（CoT）推理模块，克服了这一局限。该方法首先通过视觉语言模型（VLM）进行物体识别与定位，然后利用推理模块进行多阶段的运动轨迹推理，最终生成具有高运动真实感的视频。研究中还构建了一个新的视频对象交互（VOI）数据集，以评估该生成方法在运动控制视频生成中的表现。
C-Drag的核心方法分为三个主要模块：物体感知模块、链式思维推理模块和基于轨迹的视频生成模块。具体步骤如下：
物体感知模块：使用无类别分割模型生成输入图像中受控物体的掩码，并利用开放集检测方法提取所有相关物体的信息，包括位置和类别。
链式思维推理模块：该模块通过逐步推理来捕捉物体之间的动态交互，分为多个阶段：场景理解、关系推理、交互轨迹推理、迭代推理和最终验证。每个阶段都会利用从物体感知模块获得的信息来进行推理。
轨迹生成：生成的物体轨迹随后被输入到预训练的轨迹生成模型中，生成具有多个物体交互的视频。
实验部分主要通过构建的VOI数据集进行评估，数据集包含72个视频，涵盖三种典型的物体交互类型。使用多种评估指标，包括FrechetVideoDistance（FVD）、FrechetInceptionDistance（FID）和运动一致性（MOC），对C-Drag的性能进行了全面比较。实验结果显示，C-Drag在多个指标上均优于现有方法，尤其在运动一致性方面表现突出，表明其在复杂场景下的有效性。此外，实验还展示了不同模块和推理阶段对最终生成效果的影响，验证了C-Drag的结构设计和推理流程的合理性。
C-Drag的工作流程可以简单理解为三个步骤。首先，它会看一张图片，识别出图片中的所有物体，比如小鸟、球等，并确定它们的位置和种类。接下来，C-Drag会通过一系列的思考步骤，理解这些物体之间的关系，比如小鸟是否会碰到球，或者球在重力下会怎么移动。最后，C-Drag根据这些关系生成一个视频，展示这些物体如何相互作用和移动。这个过程就像是人类观察和推理一样，分阶段进行，确保生成的视频既真实又连贯。通过这种方法，C-Drag能够在复杂的场景中生成高质量的视频，展现出物体之间的动态交互。
2025-02-27｜SIchuanU｜⭐️🟡
http ://arxiv.org/abs/2502.19894v1
本研究提出了一种新颖的照明可控视频扩散模型（LightingControllableVideoDiffusionmodel,LCVD），旨在实现高保真、可重光照的肖像动画。该模型的核心在于有效区分肖像的内在特征（如身份和外观）与外在特征（如姿态和光照），并通过专门的子空间进行操作。现有的肖像动画方法往往无法有效操控光照条件，因此本研究通过引入3D网格、姿态以及光照渲染的阴影提示，来增强对动画生成过程中的光照、姿态和表情的精细控制。经过广泛的评估，LCVD在光照真实感、图像质量和视频一致性方面均优于现有的最先进方法，树立了可重光照肖像动画的新标杆。
LCVD的实现分为两个主要阶段：肖像属性子空间建模阶段和重光照与动画阶段。在建模阶段，通过使用DECA模型提取视频帧中的关键参数（如光照、姿态和形状），并将其渲染为阴影提示。随后，使用阴影适配器和参考适配器分别将阴影提示和参考图像转换为外在和内在特征子空间。在重光照与动画阶段，利用视频中的姿态和参考图像的形状，渲染阴影提示，并将两种特征结合以生成最终的肖像动画。同时，采用多条件无分类器引导方法来调整外在特征的影响程度，从而实现对光照的精确控制。通过这种方法，模型能够生成具有指定光照、姿态和身份的肖像动画，显著提升了生成结果的质量和一致性。
为验证LCVD的有效性，研究团队在CelebV-HQ和VFHQ数据集上进行了模型训练，确保视频质量的同时，评估了生成效果。实验中，首先对HDTF和FFHQ数据集中的视频进行了重光照处理，生成了不同光照条件下的动画结果。通过计算光照误差（LE）、身份保留（ID）和图像质量（IQ）等指标，比较了LCVD与多种现有方法的性能。结果表明，LCVD在所有评估指标上均表现优异，尤其在视频保真度和图像质量方面，分别提高了32%和14.6%。此外，用户研究显示，LCVD在用户评分上也优于其他方法，进一步证明了其在肖像动画生成中的有效性和实用性。
在我们的研究中，我们开发了一种新技术，可以让静态的肖像图像在视频中动起来，同时还能根据用户的需求改变光照效果。这个过程分为两个主要步骤。第一步是提取肖像图像的关键信息，比如它的光照和姿势。我们使用一种特殊的工具（叫做DECA）来帮助我们从视频中获取这些信息。然后，我们将这些信息转化为可以用于动画的格式。第二步是把这些信息结合起来，生成动画。我们会根据视频中人物的动作和表情，调整光照效果，使得生成的动画看起来真实自然。通过这种方法，我们可以制作出高质量的肖像动画，不仅能准确反映人物的外观，还能根据不同的光照条件进行调整。

310-B1｜MLLM指令微调相关，探讨视觉基础模型能否学习人类视觉系统的低级特征，基于MLLM的视频帧选择用于视频理解

AIGCResearch
主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）｜编辑支持｜李观丽（GuanliLi）发布日期｜2025年2月28日｜周五
2025-02-27｜UMacau,UESTC

本研究提出了一种新颖的参数高效调优框架——AsymLoRA，旨在解决多模态大语言模型（MLLM）在指令微调过程中面临的数据冲突与共性问题。现有方法通常分别处理任务间的冲突与共性，导致模型在多任务环境中的适应性下降。AsymLoRA通过引入不对称的低秩适配策略，结合共享知识和任务特定特征的低秩矩阵，能够同时捕捉不同任务之间的共性与冲突。研究表明，AsymLoRA在多个基准测试中表现优于传统的LoRA及其扩展版本LoRA-MoE，显示出其在多模态任务中的有效性和高效性。
AsymLoRA的架构设计采用了不对称的低秩适配方法，具体包括以下几个关键点：
共享矩阵A：用于捕捉跨任务的共性知识，从而减少可训练参数的数量并增强模型的泛化能力。
任务特定矩阵B：为每个任务提供独立的适配路径，以便有效处理任务间的冲突。这种设计确保了不同任务的优化目标可以被独立处理，避免相互干扰。
专家混合机制：在多任务学习中，通过动态选择最合适的任务特定矩阵B，AsymLoRA能够根据输入特征自适应地调整模型的输出，进一步提升性能。
实验部分对AsymLoRA进行了广泛的评估，涵盖了单任务和多任务场景。首先，在单域任务中，AsymLoRA在多个基准测试（如TextVQA和MME）中取得了显著的性能提升，证明了其在整合文本与视觉信息方面的优势。其次，在多域设置下，AsymLoRA持续优于其他方法，尤其在处理复杂的视觉问答任务时表现出色。实验结果表明，AsymLoRA不仅在准确性上超越了传统LoRA和MoE-LoRA，还在参数效率上显著降低了计算开销，展示了其在多模态指令微调中的强大能力。
AsymLoRA的工作原理可以简单理解为一个“聪明的助手”，它在处理不同任务时，既能记住大家共同的知识（通过共享的A矩阵），又能根据每个人的需求进行个性化调整（通过任务特定的B矩阵）。想象一下，如果你在一个团队中，每个人都有自己的专长，但大家也有一些共同的知识。AsymLoRA就像是一个团队领导，它确保每个人都能发挥自己的特长，同时又不会忘记团队的目标。通过这种方式，AsymLoRA能够更好地处理各种任务，从而提高整体工作效率和效果。
2025-02-27｜UCambridge｜CVPR2025
http ://arxiv.org/abs/2502.20256v1
本研究旨在探讨计算机视觉基础模型（如DINO和OpenCLIP）是否能够学习人类视觉系统的低级特征，特别是对比度检测、对比度掩蔽和对比度恒常性等方面。通过自监督学习，这些模型在大规模图像数据集上进行训练，而人类视觉系统的能力也受到自然界中颜色和模式的统计分布的影响。因此，研究者们设计了一套九种测试类型的协议，评估45个基础和生成模型的图像编码能力。结果表明，某些基础模型（如DINO和OpenCLIP）在某些方面与人类视觉系统相似，但其他模型则显示出较少的相似性。研究发现，基础模型对低对比度的敏感性较低，并在不同频率的对比度响应上表现不规则。
本研究采用了一种系统化的测试协议，旨在评估计算机视觉模型与人类视觉系统之间的相似性。测试主要集中在以下几个方面：
对比度检测：通过使用Gabor图案和均匀背景，测量模型对低对比度模式的检测能力。
对比度掩蔽：评估在不同掩蔽条件下模型的对比度检测能力，使用相同频率和相位的正弦波作为掩蔽模式。
对比度恒常性：通过对比不同频率和亮度的正弦波，测试模型在匹配对比度时的表现。此外，研究者们还开发了一种模型对齐评分机制，以量化模型响应与人类数据之间的相似性，采用Spearman相关系数和均方根误差等统计方法进行分析。
在实验中，研究团队测试了45个计算机视觉模型，包括DINO、DINOv2和OpenCLIP等，采用了一系列的对比度检测和掩蔽测试。实验结果显示，DINO和OpenCLIP在对比度掩蔽测试中表现出较高的相似性，与人类视觉系统的数据相符。然而，在对比度检测方面，模型的表现则相对较差，尤其是在低对比度条件下。通过对比不同的模型，研究发现，DINOv2在多个测试中展现出最接近人类视觉系统的特征，而其他模型则显示出不稳定的响应模式。整体而言，实验结果表明基础模型在某些低级视觉特征上与人类视觉系统有一定的相似性，但仍存在显著的差异。
在这项研究中，科学家们想要了解计算机视觉模型是否能像人类一样识别图像中的细微差别。他们设计了一系列测试，主要关注三个方面：首先，测试这些模型能否发现低对比度的图案；其次，看看在复杂背景下，模型是否能够准确检测图案；最后，检查模型在不同光线条件下是否能保持对比度的稳定性。为了评估这些模型的表现，研究人员计算了它们与人类视觉的相似度。通过这些测试，他们发现一些模型在某些方面表现良好，尤其是在处理复杂背景时，但在低对比度检测方面则表现不佳。这项研究帮助我们理解计算机视觉如何与人类视觉相似或不同。
2025-02-27｜CMU,UCF,Amazon
http ://arxiv.org/abs/2502.19680v1
本研究提出了一种基于多模态大语言模型（M-LLM）的轻量级视频帧选择方法，旨在提高视频理解的效率。传统的均匀采样方法在处理长视频时，可能会导致重要上下文信息的丢失，从而影响模型对特定问题的回答能力。为了解决这一问题，本文提出了一种自适应帧选择方法，通过分析与用户查询相关的帧来优化输入，确保下游模型在进行视觉推理和问答时能够获得足够的信息。该方法通过两种监督信号进行训练：空间信号和时间信号，分别用于评估单帧和多帧的重要性。实验结果表明，所提出的帧选择器在多种视频问答基准测试中显著提高了模型的性能。
本研究的方法主要包括以下几个步骤：
帧选择器设计：利用经过微调的LLM作为帧选择器，输入为视频帧和相关问题，输出为每帧的重要性分数。该设计充分利用了LLM的语言理解能力，以识别最相关的帧。
重要性评分生成：采用空间伪标签和时间伪标签两种方式来生成帧的重要性评分。空间伪标签通过提问LLM来评估单帧的相关性，而时间伪标签则通过分析帧的字幕信息来捕捉时间上的关系。
贪婪非极大值抑制（NMS）：在选择帧时，采用贪婪算法结合NMS，以避免选择相似信息的相邻帧，从而提高所选帧的信息量。
训练过程：通过两个阶段的指令微调来训练帧选择器，第一阶段冻结预训练模型的参数，优化对齐投影器，第二阶段专注于重要性评分预测任务。
在实验部分，研究者首先建立了一个包含多种视频问答数据集的训练集，涵盖了短、中、长视频的问答任务。通过均匀采样和基于所提出的帧选择器的采样进行比较，评估其在不同模型上的性能提升。实验结果显示，使用帧选择器的模型在多个基准测试中均表现出色，尤其是在长视频的理解上，能够以更少的帧数获得更高的准确率。此外，研究还进行了消融实验，验证了帧选择框架中各个组件的有效性，并展示了所选帧在实际问答任务中的优越性。
在这个研究中，科学家们想要解决一个问题：当我们看视频时，很多时候并不是每一帧都对理解视频内容有帮助。为了让视频理解变得更高效，他们设计了一种聪明的选择方法，只选出那些与我们的问题最相关的帧。具体来说，他们使用了一种叫做多模态大语言模型的技术，这种模型不仅能理解文字，还能处理图像信息。研究者们通过两种方法来判断每一帧的重要性：一种是看这帧是否能回答我们的问题，另一种是考虑到时间顺序，看看这帧是否与前后的内容有关。最后，他们通过实验发现，这种选择方法让机器在理解视频时表现得更好，同时也减少了处理的帧数，使得整个过程更快、更有效。

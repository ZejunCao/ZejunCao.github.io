310-A3｜3D生成先验点云补全，基于神经表面变形的形状插值，开放词汇3D人类语义部件分割；双角色交互动作生成

主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）｜编辑支持｜李观丽（GuanliLi）发布日期｜2025年2月28日｜周五
2025-02-27｜NUAA｜CVPR2025
http ://arxiv.org/abs/2502.19896v1
在本研究中，作者提出了一种名为GenPC的零-shot点云补全框架，旨在通过利用3D生成先验来重建高质量的真实世界扫描。传统的点云补全方法通常依赖于预定义的合成训练数据集，在面对真实世界的扫描数据时表现不佳。GenPC的创新之处在于，它不需要特定的训练数据集，而是使用经过大量互联网数据训练的3D生成模型，从而在零-shot设置下进行3D生成。该方法的核心是DepthPrompting模块和GeometricPreservingFusion模块，前者通过深度图像将部分点云与图像到3D生成模型连接，后者则通过自适应调整生成形状的姿态和尺度，确保生成的结果保持原始结构的完整性。实验结果表明，GenPC在真实世界数据集上的性能优于现有方法，并显著减少了推理时间。
GenPC的输入包括一个部分点云P和一个对应的文本提示T。该方法的主要步骤如下：
DepthPrompting模块：首先，利用M个摄像头均匀地围绕部分点云P布置，然后选择可见点最多的摄像头作为扫描视角V。通过对点云进行投影，生成初步的深度图D。接着，使用深度图进行缺失区域的填充，得到高质量的深度图D_c，并与文本提示T一起输入到深度条件生成模型中，生成RGB图像I_g。
GeometricPreservingFusion模块：生成的3D形状P_gen需要与输入的部分点云P进行对齐。此模块通过动态调整生成形状的尺度和姿态，确保最终结果在语义和结构上都与输入部分点云一致。
优化与细化：在初步完成的点云基础上，进一步利用SDS损失进行优化，减少多阶段误差累积，从而提高补全质量。
为了验证GenPC的有效性，作者在三个真实世界数据集上进行了广泛的实验，包括Redwood、ScanNet和KITTI。实验中，使用ChamferDistance(CD)和EarthMover’sDistance(EMD)作为评估指标，比较GenPC与现有最先进方法的性能。结果显示，GenPC在所有数据集上均取得了优越的性能，尤其是在处理稀疏和噪声点云时，表现出更强的鲁棒性。通过与其他学习方法的比较，GenPC在补全质量上显著优于传统方法，且在推理速度上也有明显提升。实验还包括对不同模块的消融研究，以评估各个组件对最终性能的贡献。
在GenPC的方法中，首先，我们需要输入一个不完整的3D点云和一些文字描述。接着，系统通过设置多个摄像头来观察这个点云，选择最佳的视角来获取深度信息。然后，利用这些深度数据，我们生成一张完整的深度图，这就像给3D物体画了一层“底色”。接下来，我们将这张深度图和文字描述一起用来生成一幅完整的RGB图像，这个图像展示了我们想要的物体形状。最后，系统会对生成的3D形状进行调整，使其与原始的部分点云完美对齐，确保补全的部分看起来既真实又自然。整个过程就像是在为一个缺失的拼图块找到合适的补充，让整个图案更加完整。
2025-02-27｜TUMunich,MCML,UBonn｜CVPR2025
http ://arxiv.org/abs/2502.20208v1
4Deform是一种新颖的神经网络方法，旨在通过稀疏的时间序列点云生成逼真的中间形状。传统的形状插值方法多依赖于结构化数据（如网格），而4Deform则利用神经隐式表示（NIR）来处理不规则的点云数据。该方法不仅能够处理形状的非刚性变形，还能适应拓扑变化和部分缺失的情况。通过结合物理和几何约束，4Deform在生成中间形状时保持了物理合理性，且无需对中间形状进行监督训练。实验结果显示，该方法在处理复杂形状插值时表现优于现有技术，具有广泛的应用潜力，包括人类运动序列的插值和真实世界数据的处理。
4Deform的方法主要包括以下几个步骤：
对应块：通过无监督的非刚性3D形状匹配方法，获得稀疏的形状对应关系，而不依赖于真实的对应数据。
隐式场与速度场：将源点云到目标点云的变形建模为速度场，通过时间变化的隐式神经场来表示运动。
损失函数设计：引入两个新损失函数，防止变形过程中的扭曲和拉伸，确保生成的中间形状符合物理规律。
训练过程：在训练阶段，使用优化算法同时训练隐式网络和速度网络，更新潜在向量以生成中间形状。训练过程中不需要全量对应关系，仅需部分样本即可。
推理阶段：在推理时，利用训练好的模型生成任意时间步的中间形状，实现高效的插值。
实验部分对4Deform的性能进行了全面评估，使用了多种数据集，包括人类形状、非刚性动物形状和部分形状数据。研究者通过定量指标（如表面面积的标准差、Chamfer距离和Hausdorff距离）来比较4Deform与其他基线方法的表现。实验结果表明，4Deform在各类形状插值任务中均优于现有方法，尤其在处理大变形、非刚性和部分观察形状时表现突出。此外，4Deform还展现了在真实世界数据集上的良好适应性，能够处理噪声和不完整的输入数据，显示出其实际应用的潜力。
4Deform的工作原理可以简单理解为一个智能的“变形工匠”。它首先通过一种叫做“对应块”的技术，找到不同形状之间的相似点，而不需要完全匹配每个点。接着，它使用一个“速度场”，就像给形状施加了一种“力量”，使其在时间上逐渐变化。为了确保这些变形看起来自然，4Deform设计了一些规则，防止形状在变形过程中变得奇怪或拉伸得过度。训练时，它只需要少量的示例来学习如何变形，而不是每一个细节都知道。在实际使用中，4Deform可以快速生成新的形状，帮助我们更好地理解和模拟复杂的三维物体，比如人类的动作或动物的姿态。
2025-02-27｜UCSD,Qualcomm｜3DV2025
http ://arxiv.org/abs/2502.19782v1
本研究提出了一种首个开放词汇的3D人类语义部分分割方法，旨在解决当前3D视觉和增强现实/虚拟现实领域中3D部件分割的挑战。传统的监督学习方法因缺乏足够的3D标注数据而无法有效推广到新形状和类别。借助预训练的视觉-语言模型的零-shot能力，现有的开放世界3D分割方法虽在3D场景和物体上取得了一定进展，但在3D人类数据的泛化能力上仍显不足。本研究的框架通过渲染多视角图像并利用预训练的视觉-语言模型，能够根据文本提示对3D人类进行细致的语义分割，生成非重叠的3D掩码，且支持任意查询。通过设计简单有效的MaskFusion模块，该方法在多个3D人类数据集上表现出色，超越了现有的开放词汇3D分割方法。
本方法的核心框架包括多个关键步骤。首先，我们通过预设的摄像机视角将3D人类模型渲染为多视角的2D图像。接着，利用SegmentAnythingModel(SAM)生成类无关的2D掩码提案。这些掩码随后被输入到新提出的HumanCLIP模型中，该模型负责将掩码编码为统一的CLIP特征空间中的嵌入。接下来，文本提示被转换为文本嵌入。最后，MaskFusion模块对来自不同视角的掩码进行分类和融合，生成最终的3D语义分割结果。此方法的设计不仅提高了推理效率，还有效地解耦了掩码提案生成和文本分类的过程。通过这种方式，我们能够实现对3D人类模型的灵活分割，满足用户的多样化需求。
在实验部分，我们对提出的方法进行了全面评估。我们在五个标注的3D人类数据集上进行了定量比较，分别是MGN、SIZER、CTD、THuman2.0和PosedPro。实验结果表明，我们的方法在整体准确率、平均类准确率和平均交并比等多个指标上均优于现有的开放集3D分割方法。此外，我们也评估了不同视角数量对分割质量的影响，结果显示随着视角数量的增加，分割精度得到了显著提升。我们还进行了与其他开放词汇3D分割方法的比较，结果表明我们的框架在处理复杂场景时表现出更好的适应性和准确性。最后，我们的模型在推理效率方面也显示出明显的优势，特别是在处理多次分割请求时。
在我们的方法中，首先，我们将3D人类模型从不同的角度拍摄成2D图片。然后，我们使用一个叫做SAM的工具来识别并生成这些图片中的不同部分，比如“左臂”、“右腿”等。接下来，我们用一个新开发的HumanCLIP模型将这些识别出的部分转换为计算机能够理解的形式。与此同时，我们也将用户输入的文字信息转换为计算机语言，以便进行匹配。最后，我们的MaskFusion模块将来自不同视角的识别结果进行整合，生成一个完整的3D分割图。这样，用户可以根据自己的需求，轻松地分割出他们想要的3D人类部件。整体来说，这个方法使得3D人类的分割变得更加灵活和高效。
2025-02-27｜ZJUCAD&CG,HKU,AntGroup,SZU｜ICLR2025
http ://arxiv.org/abs/2502.20370v1https ://zju3dv.github.io/ready_to_react/
本研究提出了一种名为“Ready-to-React”的在线反应策略，旨在生成两个角色之间的交互动作。传统的生成方法主要依赖于对方的完整动作序列或在特定条件下共同生成动作，这些方法未能真实地模拟人类之间的实时互动。相较之下，“Ready-to-React”允许每个角色独立反应，通过观察对方的历史动作，动态预测下一个动作。这种方法不仅能够生成自然流畅的动作序列，还能有效减少生成过程中的错误累积。研究中使用了拳击这一动态场景作为实验任务，验证了所提方法在生成长时间交互序列方面的优越性，并展示了其在虚拟现实等在线交互环境中的应用潜力。
本研究的方法主要分为几个关键步骤。首先，问题的表述是生成角色A的未来动作，基于对方角色O的动作和自身的历史动作。接下来，反应策略的设计通过将扩散模型与自回归模型相结合，允许角色根据观察到的历史信息生成下一个动作。在具体实现上，首先使用运动编码器将观察到的动作编码为潜在向量，然后通过历史编码器获取当前状态的信息。接下来，利用条件扩散模型预测未来的运动潜在编码，最后通过在线运动解码器将潜在编码转化为实际的角色动作。通过这种方式，角色能够在流畅的交互中实时生成动作，确保动作的自然性和多样性。
实验部分通过自收集的拳击数据集DuoBox，对所提方法的有效性进行了全面评估。实验设置包括三种场景：反应性动作生成、两个角色的交互生成以及长时间的交互生成。在反应性动作生成中，使用对方的真实动作作为输入，结果显示本方法在多个指标上均优于基线方法。在两个角色的交互生成中，从初始的四帧开始，角色们通过各自的反应策略生成后续动作，结果表明本方法能够有效捕捉角色间的动态互动。此外，在长时间交互生成中，所提方法能够生成1800帧的动作序列，展现了其在长时间生成中的稳定性和高质量。
在这个研究中，科学家们开发了一种新方法，叫做“Ready-to-React”，用来让两个角色在互动时表现得像真实人一样。想象一下两个拳击手在比赛中，他们的每一个动作都依赖于对方的动作。这个方法通过观察对方的历史动作，预测下一个动作。具体来说，首先系统会记住角色之前的动作，然后利用这些信息来决定接下来应该做什么。这个过程分为几个步骤：首先把角色的动作转化为一种简化的表示，然后分析对方的动作，最后生成新的动作。通过这种方式，角色能够实时、自然地进行互动，而不是依赖于对方的完整动作序列。这种方法在拳击场景中表现得尤为出色，能够生成长时间的动作序列，非常适合虚拟现实等需要实时反应的应用。

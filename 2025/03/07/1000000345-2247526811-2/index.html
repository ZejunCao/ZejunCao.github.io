2025-02-27｜MetaGenAI,ETHZurich,KAIST｜⭐️⭐️
http ://arxiv.org/abs/2502.20126v1
FlexiDiT是一种新型的扩散变换器（DiffusionTransformer），旨在降低高质量图像生成时的计算需求。传统的扩散模型在推理阶段需要固定的计算预算，这导致了高昂的资源消耗。FlexiDiT通过引入动态计算策略，允许模型在不同的去噪步骤中灵活调整计算量，从而在保持生成质量的同时，显著降低计算需求。研究表明，FlexiDiT在图像生成时的计算量可减少超过40%，而在视频生成中则可减少高达75%。这种灵活性使得模型能够更有效地适应不同的输入和条件模式，具有更广泛的应用潜力。
FlexiDiT的核心在于其灵活的计算分配机制，具体方法如下：
动态计算预算：模型在每个去噪步骤中根据需要动态调整计算预算，而不是固定分配。
调整补丁大小：通过改变图像的补丁大小来影响输入的序列长度，进而控制计算量。早期去噪步骤使用较大的补丁，以捕捉低频特征，而后期步骤则使用较小的补丁，处理高频细节。
共享参数：在多个去噪步骤中共享模型参数，减少内存需求，同时保持模型的表达能力。
推理调度器：通过推理调度器，模型可以在前期步骤使用较弱的模型进行快速推理，而在后期步骤切换到强模型，以提高生成质量。
在实验部分，FlexiDiT在多个任务上展现了其高效性和灵活性。首先，在图像生成任务中，FlexiDiT与传统模型相比，计算需求减少了超过40%，同时保持了相似的FID和CLIP评分。此外，在文本条件图像生成和视频生成任务中，FlexiDiT同样表现出色，能够在高达75%的计算节省下实现高质量的输出。实验结果表明，使用动态推理调度器时，生成的图像质量与全计算预算的模型相当，且在不同的计算预算下，模型的表现保持一致，显示出FlexiDiT的广泛适用性和高效性。
FlexiDiT是一种智能的图像生成工具，它能根据需要灵活调整计算量，帮助生成高质量的图像。想象一下，我们在制作一幅画时，早期阶段只需要粗略的轮廓，这时可以用大笔刷快速完成。随着细节的增加，我们换成小笔刷来处理复杂的部分。FlexiDiT正是这样做的，它在生成图像的不同阶段，调整使用的“笔刷”大小。通过这种方式，它不仅节省了大量的计算资源，还能确保最终的图像质量不受影响。这样的设计让FlexiDiT能够快速生成图像，甚至在处理视频时也能大幅降低计算需求，展现出它的灵活性和高效性。
2025-02-27｜TelAvivU,SnapResearch｜⭐️🟡
http ://arxiv.org/abs/2502.20376v1https ://tight-inversion.github.io
本研究提出了一种名为“TightInversion”的图像反演方法，旨在提升基于文本到图像的扩散模型在编辑真实图像时的效果。传统的图像反演技术通常依赖于将图像逐步添加噪声以进行重建，这种方法在处理细节丰富的图像时，往往面临重建精度与可编辑性之间的权衡。为了解决这一问题，研究者们探讨了条件选择对于反演质量的影响，发现与输入图像高度一致的条件能够显著提升反演效果。TightInversion方法通过直接使用输入图像作为条件，缩小了模型输出的分布，提高了重建精度和可编辑性。通过与现有的反演方法相结合，研究展示了该方法在多种图像编辑任务中的有效性。
TightInversion方法的核心在于通过选择最精确的条件——输入图像本身，来进行图像反演。具体方法包括以下几个步骤：
图像反演：从真实图像I出发，目标是预测一个噪声图像z，使得对其进行去噪后能够恢复出I。
条件选择：在反演过程中，使用输入图像I作为条件c，与传统的文本条件相比，这种选择显著提高了反演的质量。
集成现有方法：TightInversion不仅适用于DDIM反演，还可以与其他反演技术（如ReNoise和RF-Inversion）无缝结合，进一步提升性能。
评估标准：通过定量和定性指标（如PSNR、SSIM等）对反演结果进行评估，确保重建的图像在细节和结构上都能与原图相符。
为评估TightInversion的有效性，研究进行了大量实验，重点关注重建精度和可编辑性。实验中使用了多种扩散模型，包括标准扩散模型和少步扩散模型，测试了不同条件下的反演效果。结果表明，采用TightInversion后，重建的图像在细节保留和结构一致性上都有显著提升。在与现有的反演方法结合时，TightInversion能够在保持图像真实感的基础上，增强编辑的灵活性。实验还通过对比不同条件的影响，验证了使用输入图像作为条件的优势。最终，研究表明TightInversion在复杂图像的编辑任务中表现突出，能够有效提升编辑效果。
TightInversion是一种新方法，主要用于改善我们编辑真实图片的效果。传统方法通常需要逐步向图片中添加噪声来进行重建，但这常常导致我们无法很好地编辑细节复杂的图像。TightInversion的聪明之处在于，它直接使用我们想要编辑的图片作为参考条件，这样可以更准确地重建出原图，同时也让我们在后续编辑时更加灵活。具体来说，首先我们从真实图片出发，生成一个噪声版本，然后利用这个噪声版本去恢复原图。通过这样的方式，我们不仅能得到更好的重建效果，还能在编辑时保持原图的细节和结构。这种方法可以与其他现有技术结合使用，进一步提升编辑效果。总的来说，TightInversion让我们在编辑真实图片时更加得心应手。
2025-02-27｜KoreaU,SMWU,KAIST｜⭐️⭐️
http ://arxiv.org/abs/2502.19930v1
在这项研究中，作者提出了一种新的图像编辑方法，称为身份保留蒸馏采样（Identity-preservingDistillationSampling,IDS），旨在通过固定点迭代器来改善文本引导的图像生成。现有的得分蒸馏采样（ScoreDistillationSampling,SDS）在生成图像时常常导致模糊效果，尤其是在处理复杂的图像编辑任务时。为了解决这一问题，IDS利用了一种新的正则化技术，称为固定点迭代正则化（Fixed-pointIterativeRegularization,FPR），以修正与文本条件相关的得分，从而在生成过程中保持源图像的身份信息，包括结构和姿态。通过这种方法，研究人员期望提高图像编辑的清晰度和一致性，确保生成的图像在符合文本提示的同时，能够保留源图像的特征。
IDS方法的核心在于通过固定点迭代正则化（FPR）来修正得分。具体步骤包括：
分析文本条件得分：首先，研究人员分析与源图像相关的文本条件得分，以确定其在图像编辑过程中的有效性。
误差修正：通过FPR，修正与文本提示不一致的得分，确保得分能够更准确地反映源图像的特征。
更新源潜在：在每次迭代中，通过最小化FPR损失，更新源潜在图像，以便更好地保留源图像的身份信息。
生成目标图像：最终，通过引导噪声生成目标图像，确保在编辑过程中源图像的特征得以保留。
这种方法不仅提高了生成图像的质量，还显著改善了图像与文本提示之间的一致性。
为了验证IDS的有效性，研究人员进行了两项主要实验：图像编辑和可编辑的神经辐射场（NeRF）。在图像编辑实验中，作者将IDS与四种最先进的基线方法进行了比较，包括Prompt-to-Prompt、Plug-and-Play、DeltaDenoisingScore和ContrastiveDenoisingScore。实验结果显示，IDS在保持源图像结构完整性的同时，成功实现了与目标文本提示一致的图像编辑。定量评估指标如IoU、LPIPS和PSNR均表明，IDS在各项任务中均表现出色。此外，在NeRF编辑实验中，IDS也展示了在三维渲染中的优势，保持了图像的结构一致性，进一步证明了该方法的广泛适用性。
在这项研究中，研究者们提出了一种新方法，叫做身份保留蒸馏采样（IDS）。这个方法的目标是改善图像编辑的质量，特别是在根据文本提示生成图像时，避免模糊和失真。为了实现这一目标，研究者们使用了一种叫做固定点迭代正则化（FPR）的技术。简单来说，FPR就像是一个调整工具，它会不断修正图像中的错误，使得生成的图像更好地保留原始图像的特征。在操作过程中，研究者们首先分析与文本提示相关的得分，然后通过多次迭代调整图像，确保最终生成的图像不仅符合文本要求，还保留了原图的细节和结构。通过这样的方式，IDS能够生成更清晰、更真实的图像，提升了图像编辑的整体效果。
2025-02-27｜JHU,ByteDance｜⭐️🟡
http ://arxiv.org/abs/2502.20388v1https ://oliverrensu.github.io/project/xAR
本文提出了一种名为xAR的自回归视觉生成框架，旨在解决传统自回归模型在图像生成中的一些局限性。传统的自回归模型通常依赖于“下一个标记预测”的方式，然而在视觉生成中，图像的最优标记定义仍未确定。xAR通过引入“下一X预测”的概念，扩展了传统的标记定义，使得预测实体X可以灵活地表示为图像补丁、单元、子样本、整个图像或多个尺度。这种方法不仅提高了模型对不同上下文的捕捉能力，还通过避免依赖于教师强制（teacherforcing），有效减少了暴露偏差的问题。实验结果表明，xAR在ImageNet-256基准测试中表现优异，尤其是在生成速度和FID（FrechetInceptionDistance）上均达到了新的最佳水平。
xAR框架的核心由两个主要组件构成：下一X预测和噪声上下文学习（NoisyContextLearning）。下一X预测通过将图像转换为连续潜在表示，生成一系列灵活的预测实体X，具体步骤如下：
个体补丁：将图像分解为单个图像补丁，采用标准自回归建模。
单元：将图像划分为一个m×m的网格，每个单元由邻近的k×k标记组成，以捕捉局部结构。
子样本：通过均匀抽样构建非局部的标记分组。
整个图像或尺度：将所有标记合并为一个实体，或通过多尺度逐步细化图像。噪声上下文学习则通过训练模型在不完美的上下文中进行预测，降低对干净输入的依赖，从而提高模型的鲁棒性并减少错误累积。
在实验部分，xAR在ImageNet-256和ImageNet-512数据集上进行了广泛测试，评估指标包括FID、InceptionScore、精确度和召回率。实验结果显示，xAR的基础模型xAR-B（172M参数）在FID上达到了1.72，超越了其他大型模型如DiT-XL和SiT-XL，且推理速度提高了20倍。此外，xAR-H（1.1B参数）在ImageNet-256上创下了1.24的FID新纪录，展现出在高分辨率图像生成中的卓越能力。通过一系列消融实验，研究者还发现使用8×8的单元作为预测实体在性能上优于其他设计，进一步验证了xAR的有效性。
在xAR的方法部分，研究者们设计了一种新的图像生成方式，叫做“下一X预测”。这个方法的核心是将图像分成不同的部分进行处理。首先，研究者将图像划分为小块，像拼图一样，然后可以将这些小块组合成更大的单元，或者甚至是整个图像。这样做的好处在于，模型可以更好地理解图像中的结构和细节，而不只是孤立地处理每个小块。此外，xAR还引入了一种叫做“噪声上下文学习”的技术，允许模型在处理不完美的数据时学习，这样它就能更好地适应真实世界中的各种情况，减少错误的发生。这种方法不仅提高了生成图像的质量，还加快了生成速度。
2025-02-27｜BJTU,UTS,Meituan,GeorgiaTech｜⭐️🟡
http ://arxiv.org/abs/2502.20313v1https ://github.com/jiaosiyu1999/FlexVAR
FlexVAR是一种新颖的视觉自回归图像生成模型，旨在突破传统的残差预测范式。该模型通过灵活的自回归学习来生成图像，允许每一步独立地产生合理的输出，而无需依赖于对前一步的残差进行预测。FlexVAR的设计使其能够在训练时仅依赖低分辨率图像（≤256像素），但在推理时却能生成各种分辨率和纵横比的图像，甚至超出训练图像的分辨率。此外，FlexVAR还支持多种图像到图像的任务，如图像修复和扩展，展现出其强大的适应性和灵活性。在ImageNet256×256基准测试中，FlexVAR的表现优于其VAR对手，且在零-shot转移到ImageNet512×512时也取得了令人满意的结果。
FlexVAR的核心在于其独特的自回归建模方法，主要包括以下几个方面：
可扩展的VQVAE标记器：该组件将输入图像编码为多尺度的潜在特征，并通过量化过程生成离散的图像标记，支持在不同分辨率下的图像重建。
无残差的自回归学习：与传统的自回归模型不同，FlexVAR在每一步中直接预测当前尺度的真实值，而不是依赖于前一步的残差，这样可以确保相邻尺度之间的语义一致性。
灵活的推理步骤：在推理过程中，FlexVAR允许根据需要选择不同数量的推理步骤，从而在加快推理速度的同时提高生成图像的质量。通过随机采样不同的尺度，FlexVAR能够在每一步生成合理的图像，增强了模型的灵活性。
在实验部分，FlexVAR的性能通过与多种现有生成模型的比较得以验证。首先，FlexVAR在ImageNet-1K数据集上进行训练，使用的训练策略包括优化的AdamW优化器和适当的学习率。实验结果表明，FlexVAR在FID（FréchetInceptionDistance）和图像质量方面均超越了传统的自回归模型，如VAR和其他基于扩散的模型。此外，FlexVAR还展示了在零-shot条件下生成512×512分辨率图像的能力，尽管其训练数据仅限于256×256的图像。这种能力表明FlexVAR在图像生成任务中的强大泛化能力和灵活性。
在FlexVAR的设计中，最重要的创新是它如何生成图像。传统的图像生成模型通常会先生成一部分图像，然后再通过计算“残差”来调整这一部分，使得最终结果更接近真实图像。而FlexVAR则不依赖于这种方式，它在每一步生成图像时，直接预测出当前阶段应有的图像内容。这就像是在画画时，不需要依赖之前的画作，而是每次都从头开始画出新的部分。这样一来，FlexVAR可以灵活地生成不同尺寸和比例的图像，甚至可以在训练时只使用小图像，推理时却生成大图像。这种灵活性让FlexVAR在处理多种图像生成任务时表现得非常出色。
2025-02-27｜HIT,HIT(SZ),CWRU｜CVPR2025｜⭐️
http ://arxiv.org/abs/2502.19754v1https ://github.com/Qiu-XY/LDSB
本研究提出了一种新方法，旨在优化图像生成中的扩散路径，通过局部扩散薛定谔桥（LDSB）技术实现更高的生成质量和效率。扩散模型通常通过寻找两个分布之间的最优传输路径来生成图像，但传统的薛定谔桥方法在高维空间中计算复杂且耗时。为了克服这一限制，LDSB在扩散路径子空间内优化局部桥接，采用科尔莫哥洛夫-阿诺德网络（KAN），从而增强了扩散模型的训练效果。通过这种方法，研究者们能够在保持预训练去噪网络不变的情况下，显著提升图像生成的质量，尤其是在较少的采样步骤下，减少了生成图像的FID（FréchetInceptionDistance）指标，表明生成结果更接近真实图像。
本研究的方法主要包括三个步骤：首先，重新参数化扩散路径以准确捕捉时间步之间的关系。通过公式化扩散路径，LDSB定义了一个由图像和噪声权重生成的扩散路径子空间。其次，采用迭代比例拟合（IPF）的方法优化扩散路径中的权重。通过这种方式，LDSB能够在每个时间步上最小化生成图像与目标图像之间的距离，从而提高生成效果。最后，LDSB的训练目标是确保前向和反向扩散路径的相似性，通过优化损失函数来实现这一目标。这种方法的创新之处在于，仅优化扩散路径的权重，而不改变去噪网络的结构，从而降低了训练的复杂性和计算成本。
实验部分主要评估LDSB在图像生成中的有效性。研究者们在多个数据集上进行了测试，包括CIFAR10和CelebA，使用不同的采样步骤（NFE）进行比较。实验结果显示，LDSB能够在较少的采样步骤下生成更高质量的图像，FID指标显著降低，表明生成图像的质量提高。此外，LDSB还在与传统扩散模型（如DDIM和流匹配）结合时，展现出了更快的生成速度和更好的图像清晰度。通过对比实验，研究者们验证了LDSB在优化扩散路径方面的优势，尤其是在保持去噪网络不变的情况下，能够有效提高生成效率。
在这项研究中，研究者们使用了一种新的技术，叫做局部扩散薛定谔桥（LDSB），来帮助计算机生成更好看的图像。想象一下，生成图像就像在地图上寻找从一个地方到另一个地方的最佳路径。传统的方法在寻找这条路径时非常复杂且耗时，而LDSB则通过优化路径的某些部分来简化这个过程。具体来说，研究者们首先将路径进行重新定义，使得每一步都能更好地连接。接着，他们只调整路径中的一些关键点，而不改变生成图像的基础模型，这样可以更快地得到更好的结果。通过这种方式，研究者们发现，即使在步骤较少的情况下，生成的图像质量仍然可以大幅提升，像是让图像变得更清晰、颜色更真实。

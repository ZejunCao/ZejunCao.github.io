主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）｜编辑支持｜李观丽（GuanliLi）发布日期｜2025年2月26日｜周三
2025-02-25｜SYSU,HuaweiNoah’sArkLab,UiT,PengChengLab｜ICLR2025｜⭐️⭐️
http ://arxiv.org/abs/2502.17860v2https ://github.com/Li-Hao-yuan/UniGS
在近年来的多模态3D预训练研究中，联合学习文本、图像与点云的表示取得了显著进展。然而，传统的点云表示无法全面捕捉3D世界的复杂性，导致在学习联合多模态表示时存在明显的缺口。为了解决这一问题，本文提出了一种新的框架——UniGS（统一语言-图像-3D预训练），该框架集成了3D高斯溅射（3DGS）技术，以增强3D表示的能力。UniGS通过将3D场景建模为一组具有颜色和不透明度的3D高斯体，充分利用了3D场景的信息，并与2D图像建立了更强的联系。通过在大规模现实世界图像-文本对上进行预训练，UniGS实现了语言、图像和3D表示的统一学习，显著提高了多模态表示的质量。
UniGS的核心方法包括几个关键组件。
首先，3DGaussianSplatting（3DGS）作为3D表示的基础，通过将场景建模为3D高斯体，提供了高效的渲染和信息捕捉能力。
其次，UniGS采用了一个双分支的编码器结构，其中基础编码器从预训练的点云模型中提取先验信息，而高级编码器则专注于高层次特征的建模。为了实现多模态的对齐，UniGS使用了对比损失函数，分别对文本-3DGS和图像-3DGS进行对齐。
此外，UniGS引入了一种新的高斯感知引导模块，通过交叉注意力机制来引导3DGS特征的学习，从而改善特征的提取和对齐效果。
最后，UniGS在多个数据集上进行了扩展和初始化，以确保其在不同任务中的有效性。
为验证UniGS的有效性，研究团队在多个3D理解任务上进行了广泛的实验，包括物体检索、零样本分类和场景理解。实验结果显示，UniGS在不同数据集上均取得了领先的性能，尤其是在Objaverse、ABO、MVImgNet和SUNRGBD等数据集上，相较于先前的最先进方法，UniGS在零样本分类、文本驱动检索和开放世界理解任务上分别提高了9.36%、4.3%和7.92%的准确率。此外，研究还进行了详细的消融实验，分析了UniGS各个组件对模型性能的影响，验证了3DGS表示及高斯感知引导模块在多模态学习中的重要性。这些实验结果表明，UniGS在学习更强的跨模态表示方面具有显著优势。
UniGS的工作原理可以简单理解为一个多模态学习的“桥梁”，它帮助不同类型的数据（如文本、图片和3D模型）更好地相互理解。首先，UniGS使用一种叫做3D高斯溅射的技术，将3D物体表示为一组有颜色和透明度的点，这样可以更真实地反映物体的外观。接着，UniGS采用了双分支编码器，一个分支从已有的点云模型中获取基础信息，另一个分支则深入挖掘更复杂的特征。在学习过程中，UniGS通过对比不同模态之间的特征，确保它们能够紧密结合。此外，UniGS还引入了一种新的引导机制，帮助模型更好地理解和处理这些3D数据。通过这些步骤，UniGS能够在各种任务中表现出色，帮助计算机更好地理解我们所看到的世界。
2025-02-25｜ETHZurich,ETHAICenter,NVIDIA,Microsoft,UCMerced｜3DV2025｜⭐️🟡
http ://arxiv.org/abs/2502.18219v1https ://github.com/botaoye/ConsisSyn
本研究提出了一种新方法，通过优化参考图像的信息利用，来提高合成多视图图像的一致性。传统的扩散模型在从单一图像生成新视图时，往往面临着视图间一致性不足的问题。为了解决这一问题，本文提出了一种基于三维极线注意力机制的技术，该技术能够在不需要重新训练模型的情况下，支持任意相对相机位姿的生成。通过利用参考视图中的重叠信息，确保生成图像在几何和外观上的一致性，从而提升下游任务（如三维重建）的性能。实验结果表明，所提方法在合成的多视图图像中显著提高了一致性，同时保持了生成图像的质量。
本方法的核心在于使用极线几何来定位和检索参考视图中的重叠信息，并将这些信息融入目标视图的生成过程中。具体步骤如下：
极线线定位：对于目标视图中的每个点，计算其在参考图像中的极线位置，利用相机的相对旋转和位移信息。
特征匹配：在极线位置上通过特征匹配找到对应的参考特征，确保获取到的特征与目标视图中的点相匹配。
特征聚合：将从参考视图中检索到的特征与目标视图的生成过程结合，使用加权平均的方式融合特征，增强生成图像的一致性。
多视图扩展：在生成特定的新视图时，不仅考虑输入视图，还将之前生成的目标视图作为上下文信息，进一步利用极线注意力机制聚合多个视图的信息。
无训练需求：该方法采用参数复制策略，避免了额外的训练或微调，直接使用预训练模型的参数。
在实验部分，我们使用Google扫描对象数据集来验证所提方法在零-shot新视图合成中的能力。通过对生成图像质量和视图一致性进行评估，我们与基线模型Zero123及其他最新方法（如SyncDreamer）进行了比较。实验结果表明，所提方法在多个评估指标上均表现出色，包括PSNR、SSIM和LPIPS等。此外，我们还将合成的多视图图像应用于三维重建任务，结果显示生成的一致性图像进一步提升了重建效果。整体上，所提方法在保持生成图像质量的同时，显著提高了多视图图像的一致性，并且不需额外的训练过程。
在我们的研究中，我们提出了一种新方法，使得从一张图片合成多张新视图时，生成的图像之间更加一致。这个过程就像在看同一个物体的不同角度时，确保每个视角看起来都差不多。我们的方法通过使用一种叫做“极线注意力”的技术，来找到输入图像和目标图像之间的对应点。具体来说，我们先确定目标图像中每个点在输入图像中应该在什么位置，然后找到这些位置上的特征。接着，我们把这些特征与目标图像的生成结合在一起，确保生成的图像在外观和形状上都能保持一致。最棒的是，这一切都不需要重新训练模型，只需利用已有的模型参数就能完成。这种方法不仅提升了生成图像的质量，还能帮助后续的三维重建任务。
2025-02-25｜AlibabaTongyiLab｜⭐️🟡
http ://arxiv.org/abs/2502.17796v1
本文介绍了一种创新的“大头像模型”（LAM），旨在从单张图像中生成可动画的高斯头部。与以往需要大量训练视频序列或依赖辅助神经网络进行动画和渲染的技术不同，LAM能够在一次前向传递中直接生成可立即动画化和渲染的高斯头部。这一模型的关键在于其采用了FLAME模型的规范点作为查询，利用多尺度图像特征进行高斯属性的准确预测。LAM的设计使得生成的高斯头像能够在多个平台上实现实时渲染，适用于在线会议、游戏和虚拟现实等多种应用场景。实验结果表明，LAM在现有基准测试中超越了最新技术，展示了其在高效性和效果上的优势。
LAM的核心方法包括三个主要步骤。
首先，构建一个规范高斯属性生成器，利用预训练的视觉变换器（ViT）提取输入图像的多层特征。这些特征通过堆叠的交叉注意力模块进行处理，以生成高斯头像的规范属性。
其次，使用FLAME模型的顶点作为初始高斯位置，并通过学习可调参数来增强特征提取过程。
最后，生成的高斯头像通过标准线性混合蒙皮（LBS）和修正混合形状进行动画化，确保能够快速渲染。该方法的优势在于无需额外的神经网络或后处理步骤，使其能够无缝集成到传统的渲染管道中，实现实时性能。
本研究在VFHQ和HDTF数据集上进行了广泛的实验评估。实验设置包括从视频中提取图像并进行高斯头像重建，随后评估生成图像的质量。通过PSNR、SSIM和LPIPS等指标对合成图像的质量进行量化评估，结果显示LAM在自我重演和交叉重演任务中均表现优异。此外，LAM的动画和渲染速度也显著快于现有方法，尤其是在移动设备上的应用。实验还探讨了不同高斯点数量对重建质量和速度的影响，结果表明，80K点数的设置在重建质量和渲染速度之间取得了良好的平衡。
在LAM的工作流程中，首先我们会从一张图片中提取出很多细节信息，比如头发和皱纹等。这些信息会被送入一个智能系统，这个系统会根据这些细节来创建一个虚拟的3D头像。这个头像就像一个数字化的面具，可以根据需要做出不同的表情和姿势。接下来，我们利用一种叫做“线性混合”的技术，让这个虚拟头像能够快速地进行动画效果，比如微笑或摇头。整个过程不需要额外的复杂计算，所以可以在手机等设备上快速实现，适合各种应用，比如在线会议或者游戏。这样，无论你是在电脑上还是手机上，都能轻松看到这个虚拟头像的实时表现。
2025-02-25｜CMU,NTU,BUPT,THU｜⭐️🟡
http ://arxiv.org/abs/2502.17852v1
本研究提出了一种名为Sketch-1-to-3的框架，旨在从单一草图中重建高保真的三维人脸模型。尽管已有研究集中于基于图像的三维重建，但草图到三维人脸重建的领域仍然面临许多挑战。主要问题包括：从2D草图中准确提取面部关键点、保持多样的面部表情及细致纹理、以及在数据有限的情况下训练高性能模型。为了解决这些问题，本文引入了几种创新模块，包括几何轮廓和纹理细节（GCTD）模块，旨在增强草图中几何轮廓和细节的提取。此外，设计了一个深度学习架构，结合领域适应模块和定制损失函数，使草图与三维面部空间对齐，从而实现高保真的表情和纹理重建。为促进评估和进一步研究，构建了一个手绘面部草图数据集SketchFaces和一个合成草图数据集Syn-SketchFaces。实验结果表明，Sketch-1-to-3在草图基础的三维面部重建中达到了最先进的性能。
本研究的核心方法是Sketch-1-to-3框架，旨在从单一草图中精确重建三维人脸。该方法主要包括以下几个部分：
特征增强模块（GCTD）：该模块利用数字图像处理技术，提升草图中面部轮廓和细节的提取效果，增强草图的可识别性。
两阶段训练策略：第一阶段使用独立的形状编码器提取整体面部特征，并通过粗略编码器回归参数，重建初步的三维面部形状。第二阶段则利用细节编码器生成与位移图相关的潜在代码，添加更精细的结构细节。
领域适应：由于真实手绘草图数据集的稀缺，采用合成数据集Syn-SketchFaces以增强训练样本的多样性，并通过MixStyle技术桥接合成与真实草图之间的差异。
损失函数设计：引入加权标记损失、互距损失和光度损失等多种损失函数，确保重建结果的准确性和细节丰富性。
为了验证所提出方法的有效性，研究者进行了多项实验。首先，构建了两个数据集：SketchFaces（真实手绘草图）和Syn-SketchFaces（合成草图），以支持模型训练和评估。实验中，采用SSIM和GMSD等指标对重建结果进行定量比较，结果显示Sketch-1-to-3在各项指标上均优于现有方法。定性分析方面，通过与其他三维重建方法进行对比，展示了Sketch-1-to-3在捕捉面部细节和轮廓上的优势。此外，针对不同用户群体进行了用户研究，结果显示该方法在建模支持、便捷性和准确性等方面获得了良好评价，证明了其在实际应用中的有效性。
在Sketch-1-to-3方法中，我们首先利用一种叫做GCTD的特征增强模块来处理草图，使得草图中的面部特征更加清晰。这就像给草图加了一层清晰的滤镜，让我们能更容易看到面部的轮廓和细节。接下来，我们分两个阶段来训练模型。第一阶段是提取草图的整体面部形状，第二阶段则是在这个基础上增加更细致的面部细节。为了让模型在不同类型的草图上都能表现良好，我们还使用了一些合成的草图数据来丰富我们的训练样本。最后，我们设计了一些损失函数，确保模型能够准确地重建出对应的三维人脸。通过这些步骤，我们的模型能够从单一草图中生成非常真实的三维人脸模型。
2025-02-25｜USurrey,JNU｜⭐️🟡
http ://arxiv.org/abs/2502.18309v1
GCDance是一种创新的3D全身舞蹈生成框架，旨在通过音乐和文本提示生成符合特定音乐风格的舞蹈序列。舞蹈生成是一项复杂的任务，要求生成的舞蹈不仅要符合音乐的旋律和节奏，还要遵循特定的舞蹈风格和编排规则。GCDance的核心是一个无分类器的扩散模型，能够根据音乐和文本提示生成多样化的舞蹈动作。通过结合高水平的音乐特征和手工制作的特征，GCDance在FineDance数据集上展示了显著优于现有方法的性能，且在AIST++数据集上也表现出竞争力。该框架的设计使其能够生成高质量的舞蹈序列，并在实验中展示了其有效性和灵活性。
GCDance的生成过程分为几个关键步骤。首先，系统将音乐片段分割成多个段落，并提取每个段落的特征。具体方法包括：
音乐特征提取：使用Wav2CLIP模型提取音乐特征，同时结合传统的短时傅里叶变换（STFT）特征，以捕捉音乐的频率信息。
文本特征提取：通过CLIP模型将文本提示（如舞蹈风格描述）转化为特征向量，以便在生成过程中进行条件控制。
舞蹈动作生成：采用扩散模型，通过逐步去噪的方式生成舞蹈序列。模型根据音乐和文本特征进行条件生成，以确保生成的舞蹈符合指定的舞蹈风格。
运动表示：使用52个关节的运动表示，结合根关节的平移和足部接触状态，形成完整的舞蹈动作表示。
在实验部分，研究者对GCDance进行了全面评估，使用了FineDance和AIST++数据集。实验指标包括：
运动质量：使用FrechetInceptionDistance(FID)度量生成舞蹈与真实舞蹈之间的相似度。
生成多样性：通过计算生成动作之间的平均欧几里得距离来评估生成的舞蹈动作的多样性。
运动与音乐的相关性：采用节拍对齐分数（BAS）评估生成动作与音乐节拍之间的时间一致性。
物理可行性：使用物理足接触指标（PFC）来评估生成舞蹈的物理合理性。
结果表明，GCDance在多个指标上均优于现有的舞蹈生成方法，展示了其在生成高质量和多样化舞蹈方面的潜力。
GCDance的工作原理可以简单理解为一个智能舞蹈生成器。首先，它会听一段音乐，并根据音乐的风格和节奏来决定舞蹈的类型。然后，它将音乐分成小段，提取出音乐的特点，就像给每段音乐打上标签。接下来，它还会读取一些文本提示，比如“这是爵士舞”来帮助它理解要生成什么样的舞蹈动作。最后，GCDance使用这些信息来生成舞蹈动作，确保这些动作不仅与音乐的节拍匹配，还能展现出特定的舞蹈风格。通过这种方式，GCDance能够创造出多样化且富有表现力的舞蹈，给观众带来视觉享受。

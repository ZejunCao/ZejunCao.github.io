2025-02-25｜LiAuto｜⭐️🟡
http ://arxiv.org/abs/2502.18302v1https ://zrealli.github.io/LDGen
本文提出了一种新颖的图像生成方法LDGen，旨在通过整合大型语言模型（LLMs）来提升文本到图像的合成效果。传统的文本编码器如CLIP和T5在处理多语言生成时存在局限性，限制了图像生成的多样性和质量。LDGen通过利用LLMs的先进语言理解能力，采用一种语言表示策略，结合分层标题优化和人类指令技术，以提高语义信息的准确性。实验结果表明，该方法在遵循提示和图像美学质量方面均优于现有基线模型，并且能够高效支持多语言生成。
LDGen的方法主要包括三个关键部分：语言表示策略、LLM对齐和跨模态精炼器。首先，语言表示策略通过分层标题优化和人类指令优化，帮助LLMs更好地提取语义信息，确保生成的图像与文本描述相符。其次，LLM对齐模块通过线性层对LLM输出与T5特征进行对齐，优化特征空间的匹配，显著减少训练时间。最后，跨模态精炼器利用自注意力机制和交叉注意力机制，增强文本和图像特征之间的交互，提升模型对跨模态内容的理解能力。通过这些模块的协同作用，LDGen能够有效整合LLM到现有的扩散模型中，实现高效的多语言图像生成。
在实验部分，LDGen的性能在多个基准测试中与现有方法进行了比较。通过使用DPG-Bench和Geneval等公开评测标准，评估了生成图像的质量和文本一致性。实验结果显示，LDGen在遵循提示和图像美学方面明显优于基线模型PixArt-α，尤其在多语言生成能力上表现出色。该方法的训练过程相对高效，仅需约26%的计算资源与时间，相比于传统方法显著降低了训练成本。此外，LDGen能够在没有多语言图像-文本对的情况下，实现零-shot多语言生成，展现出良好的应用潜力。
LDGen的工作原理可以简单理解为三个步骤。首先，研究团队使用了一种特殊的语言处理方法，帮助计算机更好地理解用户的描述，并将这些描述转化为生成图像所需的信息。接着，他们设计了一种对齐机制，确保计算机生成的图像与描述之间的关系更加紧密。这就像是把语言和图像的“语言”对齐，让它们能够更好地沟通。最后，他们还引入了一个精炼器，进一步优化图像和文本之间的互动，确保生成的图像不仅准确，而且美观。通过这些措施，LDGen能够快速、准确地生成符合用户需求的多语言图像。
2025-02-25｜NKU｜⭐️⭐️
http ://arxiv.org/abs/2502.18461v1https ://k-lora.github.io/K-LoRA.io/
K-LoRA是一种创新的训练-free方法，旨在有效融合不同主题和风格的LoRA（Low-RankAdaptation）模型。传统的LoRA方法在生成图像时，通常需要额外的训练以确保内容和风格信息的有效结合。然而，K-LoRA通过利用LoRA的内在特性，提出了一种简化的融合策略，能够在不增加训练负担的情况下，提升生成图像的质量。该方法通过比较每个LoRA的Top-K元素，在每个注意力层中选择最具代表性的特征，从而实现更优质的风格和内容融合。实验结果显示，K-LoRA在多种图像生成任务中表现出色，超越了现有的训练基础方法，且在用户友好性上也具备显著优势。
K-LoRA方法的核心在于其独特的Top-K选择机制和分层加权策略。具体而言，方法步骤如下：
Top-K选择：在每个注意力层中，K-LoRA会计算LoRA权重的绝对值，并选择具有最高值的Top-K元素，确保在融合过程中保留最重要的特征。
加权调整：引入比例因子（α和β），根据当前的扩散步骤动态调整内容和风格LoRA的权重，以便在生成过程中更好地平衡对象重建和风格细节的融合。
分层处理：在扩散的初期步骤中，优先使用内容LoRA以构建对象的基础，而在后期步骤中，则逐渐引入风格LoRA以增强细节和风格特征的表现。
无额外训练：K-LoRA的设计使其能够直接应用于现有的LoRA权重，无需额外的训练或参数调节，简化了用户的操作流程。
在实验部分，研究者对K-LoRA的性能进行了全面评估，比较了其与其他先进方法（如ZipLoRA和B-LoRA）的效果。实验使用了多种数据集，包括DreamBooth和StyleDrop，涵盖了不同主题和风格的图像生成任务。通过量化指标（如CLIP和DINO评分），K-LoRA在主题相似性和风格一致性方面均表现优越。此外，用户研究显示，K-LoRA生成的图像在保留对象特征和风格细节方面得到了用户的高度认可。实验结果表明，K-LoRA不仅在技术上实现了目标，同时在用户体验上也展现了其易用性和高效性。
K-LoRA的工作原理可以简单理解为一种智能的选择和组合方式。想象一下，你在做一道菜，需要将不同的食材（内容和风格）混合在一起。K-LoRA首先会评估每种食材的重要性，挑选出最好的几个（Top-K），确保每种食材的味道都能被充分展现。接着，它会根据做菜的进度，适时调整各食材的用量，比如在开始时多放一些主料（内容），而在最后阶段再加入调料（风格），这样才能让菜肴既丰富又美味。最棒的是，K-LoRA不需要重新学习如何做这道菜，只需利用已有的食材组合，就能迅速制作出令人满意的成果。
2025-02-25｜MSRA,THU,PKU,USTC｜⭐️🟡
http ://arxiv.org/abs/2502.18364v1https ://art-msra.github.io/
本文提出了匿名区域变换器（ART），一种用于生成可变多层透明图像的新方法。传统的生成模型通常只能创建单一层次的图像，限制了用户对特定元素的独立编辑能力。ART通过引入匿名区域布局，允许生成模型自动识别每个区域应生成的视觉元素，从而简化了图像生成过程。该方法不仅提高了生成图像的效率，还显著减少了计算成本，尤其是在处理超过50层的复杂图像时。此外，ART在层间一致性和用户控制方面表现出色，展现了其在图形设计和数字艺术领域的潜力。
ART的核心由三个主要组件构成：多层透明图像自编码器、匿名区域变换器和匿名区域布局规划器。具体方法如下：
多层透明图像自编码器：该组件负责对多层透明图像及其潜在表示进行编码和解码，支持在不同分辨率下生成多个透明层。
匿名区域变换器：该模块同时生成全局参考图像、背景图像和多个透明前景层。其通过对噪声的去噪处理，实现对透明层的高效生成。
匿名区域布局规划器：此规划器根据用户提供的文本提示，预测一组匿名边界框，简化了传统布局方法所需的复杂标注过程。通过这种布局，ART能够在生成图像时减少人力干预，提高整体效率。
在实验部分，我们通过对比ART与传统方法的性能，评估其在多层透明图像生成中的有效性。实验使用了包含多层图像的高质量数据集，并进行了用户研究，以评估生成图像的美观性、提示遵循性和层间和谐性。结果显示，ART在各个评估维度上均优于当前最先进的方法，尤其是在层间一致性和生成效率方面。此外，ART能够同时生成多个透明层，避免了以往方法中存在的层冲突问题，进一步提升了用户体验和生成质量。
在ART方法中，首先，我们使用一个叫做多层透明图像自编码器的工具来处理图像，这个工具可以同时处理多个图层，确保每个层的细节都能被很好地编码和解码。接着，匿名区域变换器会根据输入的提示，自动生成图像的背景和前景层，省去了手动标注的麻烦。最后，匿名区域布局规划器根据用户的文本提示，预测出图像中不同部分的布局，这样生成的图像不仅美观，还能更好地满足用户的需求。通过这种方式，ART能够快速生成高质量的多层透明图像，同时让用户在设计过程中有更多的控制权。
2025-02-25｜UIUC,UCLA｜⭐️🟡
http ://arxiv.org/abs/2502.17793v1https ://github.com/HyeonjeongHa/SYNTHIA
本研究提出了SYNTHIA，一个新颖的概念设计框架，旨在生成具有功能一致性和视觉新颖性的设计概念。尽管现有的文本到图像（T2I）模型在生成语义和风格变体方面取得了一定成就，但它们在整合多个功能性特征（即“可供性”）以形成统一概念方面仍存在不足。SYNTHIA通过引入分层概念本体，能够将设计概念分解为各个部分及其可供性，从而为功能一致的设计提供结构基础。实验结果表明，SYNTHIA在生成新颖性和功能一致性方面，显著优于当前最先进的T2I模型，证明了其在AI驱动设计中的应用潜力。
SYNTHIA的构建方法包括三个主要步骤：首先，构建一个分层概念本体，该本体将视觉概念分解为组成部分及其相关可供性。其次，采用基于课程学习的优化策略，通过逐步增加可供性距离来训练T2I模型，使其能够学习从基本的概念-可供性关联到复杂的可供性组合。具体而言，该方法包括以下几个要点：
分层概念本体构建：将概念分解为其组成部分和可供性，提供结构化表示。
可供性采样策略：通过选择具有足够不同概念的可供性对，促进新颖功能的整合，避免冗余组合。
对比学习与课程学习结合：通过对比目标，确保生成的图像不仅反映所需的可供性，还与现有概念保持区别，从而提高视觉新颖性。
为评估SYNTHIA的有效性，研究团队构建了一个包含500个新颖可供性对的数据集，并通过自动化与人工评估相结合的方法进行测试。实验设计包括四个评估指标：忠实度、新颖性、实用性和一致性。结果显示，SYNTHIA在所有评估指标上均显著优于基线模型，特别是在新颖性和功能一致性方面，分别提高了25.1%和14.7%。此外，研究还分析了课程学习的影响，结果表明，通过逐步增加训练难度，模型在早期阶段即可产生显著的性能提升。这些结果证明了SYNTHIA在生成功能一致且视觉新颖的设计概念方面的有效性。
SYNTHIA的工作原理可以理解为一种结构化的设计方法。首先，研究人员创建了一个“概念字典”，将各种设计元素和它们的功能进行分类。接着，通过一种逐步学习的方式，让计算机从简单的设计开始，逐渐学习如何将不同的功能组合在一起。比如，想象一个既能煮咖啡又能送到床边的咖啡机。SYNTHIA通过选择不同的功能进行组合，确保每个设计不仅看起来新颖，而且在实际使用中也能发挥作用。最后，通过对比学习，SYNTHIA确保生成的设计与已有的设计不同，保持独特性。这种方法使得设计过程更具创造性，同时也更实用。
2025-02-25｜UEdinburgh,UManchester,THU,FancyTech,UWashington,UCopenhagen｜⭐️🟡
http ://arxiv.org/abs/2502.18116v2
在图像生成领域，如何实现对生成内容的精确控制和保持语义一致性是一个重要的挑战。为了应对这一难题，研究团队提出了BayesGenie，这是一个将大型语言模型（LLMs）与贝叶斯优化相结合的创新框架。该方法能够通过自然语言描述来修改图像，而无需手动标记区域，同时保持原图的语义完整性。与现有需要大量预训练或微调的技术不同，BayesGenie展示了在不同LLMs上的适应性，提供了一种即插即用的解决方案。通过自动优化推理过程参数，该方法实现了高精度的图像编辑，且用户干预最小。通过对多种场景的广泛实验，BayesGenie在编辑准确性和语义保留方面显著优于现有方法。
BayesGenie的系统架构结合了LLMs和贝叶斯优化技术，以实现图像编辑。具体方法包括以下几个步骤：首先，LLM处理原始图像和修改需求，生成描述所需更改的文本提示。然后，将该提示与原始图像输入到扩散模型中，生成初步修改的图像。接下来，LLM对生成的图像进行评估，基于满足需求的程度打分，并反馈需要改进的地方。贝叶斯优化随后迭代调整关键参数，特别是文本和图像的CFG权重，以最大化输出质量。该过程通过动态提示优化和参数调整，确保生成图像的高质量和准确性，避免了依赖任何形式的掩模指导。
在实验中，研究团队采用了一系列评估标准，以验证BayesGenie的有效性和性能。实验包括多个图像编辑任务，如添加、移除和修改图像中的对象。研究人员构建了一个包含500多幅图像的评估数据集，并使用传统的图像处理指标（如SSIM和PSNR）与基于LLM的评分相结合，进行全面评估。结果显示，BayesGenie在满足用户编辑要求、整体视觉质量和保留原始图像上下文方面表现优异。与其他无掩模指导方法相比，BayesGenie在多种场景中显示出更高的稳定性和准确性，证明了其在精细图像编辑中的强大能力。
BayesGenie的工作原理可以简单理解为一个智能助手，它能够根据用户的描述来修改图像。首先，用户只需用自然语言描述想要的修改，比如“在厨房桌旁加几把椅子”。这个助手会分析描述，生成一个初步的修改图像。接下来，助手会对这个图像进行评分，看看它是否满足用户的要求。如果不够好，助手会根据反馈进行调整，反复优化，直到生成的图像符合用户的期望。整个过程不需要用户手动选择或标记特定区域，极大提高了图像编辑的便利性和准确性。通过这种方式，BayesGenie让图像编辑变得更加直观和易用。
2025-02-25｜SberAI,AIRI｜⭐️🟡
http ://arxiv.org/abs/2502.18417v3
GHOST2.0是一种新颖的高保真头部交换框架，旨在解决头部重演和背景融合中的多种挑战。该模型由两个主要模块组成：Aligner和Blender。Aligner模块负责头部重演，能够在多种姿态下保持身份信息的完整性，并对极端姿态变化具有鲁棒性。Blender模块则负责将重演的头部无缝地融入目标背景，通过皮肤颜色转移和不匹配区域的修补，确保最终合成的自然与真实。GHOST2.0在复杂场景中表现出色，尤其是在源头和目标之间存在显著发型差异的情况下，展现了其优越的性能。
GHOST2.0的核心方法包括两个模块：Aligner和Blender。Aligner模块通过以下步骤实现头部重演：
身份编码：使用两个编码器分别提取源头的身份特征和目标的运动特征，确保在多尺度下捕捉到细致的身份信息。
运动编码：通过一个运动编码器将目标图像的姿态和表情信息编码为向量，确保重演过程中保持自然的动态变化。
生成器条件化：将身份和运动信息拼接后，输入到基于StyleGAN的生成器中，生成重演的头部图像。
Blender模块则通过以下步骤实现背景融合：
数据预处理：对重演的头部和目标背景进行灰度化处理，并生成相应的掩膜，定义需要进行颜色转移和修补的区域。
颜色参考创建：通过学习源头和目标图像的语义区域之间的关系，生成颜色参考，以确保头部颜色的自然过渡。
最终融合：将重演的头部与目标背景通过UNet进行融合，输出最终的合成图像。
在实验部分，研究团队使用VoxCeleb2数据集对GHOST2.0进行训练和评估，分别在512×512和256×256的分辨率下进行。通过对比多种基线方法，评估了Aligner和Blender模块的性能。实验结果显示，GHOST2.0在身份保留、运动转移和整体生成质量方面均优于现有的方法。具体评估指标包括LPIPS、SSIM、PSNR等，确保了生成图像的视觉质量。此外，针对交叉重演场景进行了用户研究，结果表明GHOST2.0在用户体验方面也表现优异，能够更好地保留源头身份和自然运动。
GHOST2.0的工作原理可以简单理解为两个主要步骤：重演和融合。首先，Aligner模块负责“重演”头部，这就像把一个人的头部动画化，使其能够在不同的姿势和表情下移动。为此，它会提取源头的特征，比如面部形状和头发样式，然后将这些特征与目标图像的运动信息结合起来，生成一个新的头部图像。接下来，Blender模块则负责将这个重演的头部与背景结合在一起。它会分析目标背景的颜色和形状，然后将重演的头部颜色调整得与背景相匹配，确保看起来自然。最终，GHOST2.0能够生成一个无缝融合的图像，达到高保真的效果。

309-B1｜理解复杂的数学和科学定理，探讨VLM内部的神经元功能，CLIP视觉与文本可解释

AIGCResearch
主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）｜编辑支持｜李观丽（GuanliLi）发布日期｜2025年2月27日｜周四
2025-02-26｜UWaterloo,VoteeAI,VectorInstitute
http ://arxiv.org/abs/2502.19400v1https ://tiger-ai-lab.github.io/TheoremExplainAgent/
本研究提出了TheoremExplainAgent，这是一个旨在生成多模态定理解释的智能系统。该系统通过结合文本和视觉元素，帮助用户更好地理解复杂的数学和科学定理。尽管大型语言模型（LLMs）在文本推理方面表现出色，但在生成连贯且具有教育意义的视觉解释方面仍存在挑战。为了解决这一问题，研究团队开发了TheoremExplainBench，一个涵盖240个定理的基准测试，涉及多个STEM学科，并提出了五种自动评估指标来系统性地评估多模态定理解释的效果。实验结果显示，TheoremExplainAgent在生成长达10分钟的解释视频方面表现良好，成功率高达93.8%，但在视觉元素布局方面仍需改进。
TheoremExplainAgent的工作流程分为几个关键部分。首先，模型接收一个定理及其简短描述作为输入，这些信息帮助模型识别定理的背景。接下来，系统的输出是一个包含动画、结构化推导和语音解说的视频，旨在提供全面的定理解释。整个过程由两个主要代理组成：规划代理和编码代理。规划代理负责创建视频的高层次计划，包括多个场景的详细描述。然后，编码代理将这些场景转换为Python代码，使用Manim库生成相应的动画和视觉效果。此外，系统还采用了增强型生成技术，通过动态检索与定理相关的文档和代码示例，以提高生成的质量和准确性。
在实验阶段，研究团队使用TheoremExplainAgent对240个定理进行评估，涵盖了数学、物理、化学和计算机科学等多个领域。通过比较不同代理的表现，发现o3-mini模型在生成长视频方面表现最佳，成功率在各个难度水平上均高于其他模型。实验还显示，视频的成功率与定理的难度和学科密切相关，数学领域的成功率普遍较高，而化学领域则相对较低，主要由于化学概念的复杂性。此外，通过与人类制作的视频进行比较，研究团队评估了AI生成的解释在准确性、逻辑流畅性和视觉一致性等方面的表现。
TheoremExplainAgent的工作原理可以简单理解为“制作视频的机器人”。首先，它接收一个定理和一些背景信息，帮助它理解要讲解的内容。接下来，它的“规划师”部分会设计一个视频的基本结构，决定视频中会有哪些场景和动画。然后，“编码员”部分就会将这些场景转化为代码，使用一个叫做Manim的工具来制作动画和视觉效果。最后，所有的动画和解说会组合成一个完整的视频，帮助观众更好地理解这个定理。这种方法使得复杂的数学和科学概念变得更加直观和易于理解。
2025-02-10｜USTC,MSRA
http ://arxiv.org/abs/2502.18485v1
随着视觉-语言模型（VLMs）的迅猛发展，它们在多个领域的应用日益广泛。然而，由于这些模型的复杂性和不透明性，确保其透明度和可解释性变得至关重要。本研究旨在深入探讨VLM内部的神经元功能，通过分析神经元对输入图像和文本的激活反应，揭示其独特的功能特性。我们发现，神经元可以分为三类：专门处理视觉信息的视觉神经元、专门处理文本信息的文本神经元，以及同时处理二者的多模态神经元。此外，我们还构建了一个自动化框架，利用GPT-4o对神经元进行解释，以提升对其功能的理解。
本研究采用系统性的观察方法，分析VLM中神经元对视觉和文本令牌的反应。具体方法如下：
神经元分类：我们定义了四种类型的神经元，包括视觉神经元、文本神经元、多模态神经元和未知神经元，通过分析神经元在不同输入样本中的激活模式来进行分类。
自动化解释：引入基于GPT-4o的自动化解释框架，利用神经元的高激活样本生成对应的功能解释。对于视觉神经元，输入激活调制的图像；对于文本神经元，输入文本及其激活值。
激活模拟：构建视觉激活模拟器和文本激活模拟器，分别预测视觉和文本令牌的激活值，从而评估解释的可靠性。使用统计分析来揭示不同类别神经元的特征和行为。
在实验部分，我们基于LLaVA1.5模型进行了广泛的实验，旨在验证不同类型神经元的功能。首先，我们收集了23,000对图像和文本样本，以分析神经元的激活模式。通过观察，我们发现一些神经元对视觉令牌的响应显著，而另一些则对文本令牌表现出高响应。我们进一步利用自动化框架生成解释，并通过模拟实验评估这些解释的有效性。结果显示，视觉神经元和文本神经元在早期层中表现出高频率，而多模态神经元则在高层中更为活跃。此外，我们还评估了去除未知神经元对模型性能的影响，结果表明去除这些神经元对视觉问答任务的影响较小，进一步验证了我们的分类和分析方法的有效性。
在本研究中，我们通过观察神经元对图像和文本的反应来理解它们的功能。我们把神经元分为三类：只对图像有反应的视觉神经元、只对文本有反应的文本神经元，以及同时对两者都有反应的多模态神经元。为了更好地解释这些神经元的功能，我们使用了一种自动化工具，GPT-4o，来生成关于每个神经元的解释。我们还设计了模拟器，帮助我们预测这些神经元在不同输入下的激活情况。这种方法使我们能够更清晰地了解神经元是如何工作的，以及它们在处理视觉和文本信息时的不同反应。这些发现不仅帮助我们理解VLM的内部机制，也为未来的AI系统提供了更高的透明度和信任度。
2025-02-26｜CityU,HKUST,SenseTime
http ://arxiv.org/abs/2502.18816v1https ://github.com/Cyang-Zhao/Grad-Eclip
本文提出了一种名为Grad-ECLIP的梯度基础视觉和文本解释方法，旨在解释CLIP模型在特定图像-文本对上的匹配结果。尽管CLIP在多模态学习中取得了显著进展，但对其内部机制的理解仍较为不足。Grad-ECLIP通过分解编码器架构，探讨匹配相似性与中间空间特征之间的关系，生成有效的热图，展示图像区域或文本对CLIP结果的影响。与以往方法不同，Grad-ECLIP不仅关注自注意力图，还通过通道和空间权重对特征进行加权，产生高质量的视觉解释。实验结果表明，Grad-ECLIP在定性和定量评估中均优于现有最先进的解释方法，进一步揭示了CLIP的工作机制及其在图像-文本匹配中的优势与局限性。
Grad-ECLIP的核心在于通过梯度信息生成图像-文本匹配的解释。具体方法如下：
特征提取：从CLIP的图像和文本编码器中提取特征，分别表示为F_I和F_T。
匹配分数计算：利用余弦相似度计算图像特征F_I与文本特征F_T之间的匹配分数S(F_I,F_T)。
梯度计算：对匹配分数S相对于图像特征F_I的梯度进行计算，以获取特征的重要性权重。
通道与空间权重：通过定义通道重要性w和空间重要性λ，生成加权特征图。通道权重基于梯度计算，空间权重则通过松散的注意力机制获得。
热图生成：最终，通过将通道权重与空间权重结合，生成图像区域的热图，从而可视化对匹配结果的贡献。
在实验部分，作者对Grad-ECLIP进行了全面评估，包括定性和定量分析。定性评估通过对比不同解释方法的热图，展示Grad-ECLIP在不同文本提示下的解释能力。定量评估则通过与众多基线方法（如Grad-CAM、GAME等）进行比较，验证Grad-ECLIP的有效性。此外，进行了消融研究，分析了空间权重和参与层数对结果的影响。实验结果表明，Grad-ECLIP不仅在视觉解释上表现优异，也能有效提升CLIP在下游任务中的性能，特别是在细粒度理解和区域匹配方面。
Grad-ECLIP的工作原理可以简单理解为“找出关键部分”。首先，它从CLIP模型中提取图像和文本的信息，然后计算这两者之间的匹配程度。接着，通过观察这个匹配程度的变化，Grad-ECLIP能够识别出哪些图像部分或文本单词对最终的匹配结果影响最大。为了实现这一点，Grad-ECLIP使用了一种叫做“梯度”的数学工具，来衡量每个特征的重要性。此外，它还对图像的不同区域和文本的不同单词赋予不同的权重，从而生成一个热图，清晰地显示出哪些区域或单词是最重要的。这样，研究人员可以更好地理解CLIP是如何进行图像与文本匹配的。

主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）｜编辑支持｜李观丽（GuanliLi）发布日期｜2025年2月26日｜周三
2025-02-25｜SJTU,ShanghaiAILab,NJU,FudanU,ZJU
http ://arxiv.org/abs/2502.18411v1https ://github.com/PhoenixZ810/OmniAlign-V
本研究提出了OmniAlign-V，一个旨在提高多模态大语言模型（MLLMs）与人类偏好对齐的综合数据集。随着开源多模态大语言模型的发展，现有的研究主要集中在增强模型的基础能力，而忽视了人类偏好的对齐问题。OmniAlign-V包含约20万个高质量的训练样本，涵盖多样的图像、复杂的问题以及多种响应格式，以此来提升MLLMs的对齐性能。此外，研究还引入了MM-AlignBench，这是一个专门设计的人类标注基准，旨在评估MLLMs与人类价值观的对齐能力。实验结果表明，通过使用OmniAlign-V进行模型微调，可以显著提高人类偏好的对齐，同时保持或提升在标准视觉问答基准上的性能。
本研究的方法主要分为以下几个部分：首先，构建OmniAlign-V数据集，采用开放式、多样化和创造性的问题，确保涵盖广泛的主题和响应格式。其次，采用图像选择策略，利用图像复杂性模型（IC9600）来筛选出富含语义的图像，确保训练数据的质量。然后，设计数据生成管道，通过使用先进的语言模型（如GPT-4o）生成多样化的问题和答案对，确保每个问题都与图像内容密切相关。最后，进行数据后处理，通过人类专家的评审和合并不同模型的响应，提升数据的整体质量和一致性。在这些步骤中，特别强调了指令跟随能力和知识丰富的问答对的生成，以提高模型的对齐效果。
在实验部分，研究者使用OmniAlign-V进行多项实验，评估其对MLLMs人类偏好对齐的影响。首先，通过与现有的基准数据集进行对比，验证了使用OmniAlign-V后，模型在多模态对齐方面的显著提升。其次，采用监督微调（SFT）和直接偏好优化（DPO）两种策略，对比不同数据集的微调效果。实验表明，使用OmniAlign-V进行微调的模型在多个评估基准上表现出色，尤其是在MM-AlignBench和WildVision等人类偏好对齐基准上。此外，研究还探讨了不同任务类型对模型性能的影响，发现指令跟随数据对整体性能的提升至关重要。最终，实验结果证明了OmniAlign-V在增强MLLMs与人类偏好对齐方面的有效性。
在这项研究中，科学家们创造了一种新的训练数据集，叫做OmniAlign-V，目的是让计算机更好地理解和回应人类的需求。首先，他们收集了大量的图片和问题，确保这些问题既复杂又有趣，能够引导计算机给出更丰富的答案。接着，他们利用一种智能算法来筛选出最有意思的图片，确保这些图片能够激发出更深层次的思考。然后，他们使用一种强大的语言模型，像GPT-4o，来生成与这些图片相关的各种问题和答案。最后，他们请人类专家来检查和优化这些数据，确保其质量。这一系列步骤的最终目标是让计算机在与人类互动时，能更准确地理解我们的意图和情感，从而提升交流的质量。
2025-02-24｜UEdinburgh｜🟡
http ://arxiv.org/abs/2502.17540v1https ://github.com/saxenarohit/postersum
在学术交流中，科学海报扮演着重要角色，通常结合文本、图像和图表来展示研究成果。然而，将这些视觉复杂的海报准确地总结成简洁的文本摘要，尤其是面对复杂布局和技术术语时，仍然是一个巨大的挑战。为此，研究团队提出了POSTERSUM，一个全新的基准数据集，旨在推动视觉语言模型在科学海报摘要生成方面的发展。该数据集包含16,305个科学海报及其对应的研究摘要，涵盖了多个科学领域，主要来源于机器学习领域的顶级会议，如ICLR、ICML和NeurIPS。通过对现有多模态大型语言模型（MLLMs）的评估，研究发现这些模型在处理科学海报时的表现较差。因此，研究者们提出了一种新的方法——SEGMENT&SUMMARIZE，旨在通过分段和总结的方式提高摘要生成的质量。
SEGMENT&SUMMARIZE方法采用分层策略，将科学海报的摘要生成过程分为三个关键步骤：
首先是“分段与聚类”，该步骤将海报图像分割成多个连贯的区域，以便于后续处理。
接下来是“局部总结”，在这一阶段，使用多模态大型语言模型提取和解释每个区域的文本内容，生成局部摘要。
最后是“全局总结”，将所有局部摘要结合起来，使用文本基础的大型语言模型生成一个连贯的整体摘要。该方法的优势在于，不需要额外的训练或微调，可以有效捕捉每个区域的细节，同时保持海报的整体结构。通过这种方式，SEGMENT&SUMMARIZE能够处理复杂的视觉元素，并在摘要生成中取得显著的性能提升。
在实验中，研究团队对多种开源和闭源的多模态语言模型进行了评估，以测试它们在科学海报摘要生成任务中的表现。实验结果显示，尽管现有的多模态大型语言模型在其他任务上表现优异，但在处理科学海报时仍面临显著挑战。特别是，最佳的闭源模型GPT-4o在ROUGE-L评分上仅为22.30，表明了该任务的复杂性。通过引入SEGMENT&SUMMARIZE方法，研究团队在ROUGE-L评分上实现了24.18的成绩，较现有模型有了3.14%的提升。此外，实验还表明，通过聚类步骤，摘要生成的准确性得到了显著提高，进一步证明了该方法在科学海报摘要生成中的有效性和实用性。
在这项研究中，研究者们开发了一种新的方法，叫做SEGMENT&SUMMARIZE，来帮助计算机更好地理解和总结科学海报。这个方法分成三个简单的步骤。首先，他们会把海报分成几个小块，每个小块都包含相关的信息。接着，计算机会查看每个小块，提取出重要的内容，生成小块的摘要。最后，他们会把所有的小摘要合并成一个完整的、流畅的摘要，这样就能够清晰地传达整个海报的核心信息。这种方法的好处是，它可以专注于每个部分的细节，同时保持整体的逻辑性和连贯性。通过这种方式，研究者们发现这种方法在生成摘要时比现有的技术要更有效。
2025-02-25｜USTC,AlibabaTongyiLab,SJTU
http ://arxiv.org/abs/2502.18017v1https ://github.com/Alibaba-NLP/ViDoRAG
ViDoRAG（VisualDocumentRetrieval-AugmentedGeneration）是一种新颖的多代理框架，专门针对视觉丰富的文档进行复杂推理和信息检索。现有的检索增强生成（RAG）方法往往侧重于图像问答（VQA）任务，而忽视了在密集视觉文档中进行有效检索和推理的挑战。为了填补这一空白，ViDoRAG引入了一个多代理的粗到细检索框架，结合了文本和视觉特征的高效集成。该框架通过采用基于高斯混合模型（GMM）的混合检索策略，增强了多模态信息的处理能力。此外，ViDoRAG还通过引入动态迭代推理代理，提升了模型的推理能力，使其在处理复杂查询时更加高效。通过构建ViDoSeek这一新数据集，ViDoRAG在大规模文档检索和推理方面取得了显著进展，并在相关基准测试中展现出超过10%的性能提升。
ViDoRAG的核心方法包括两个主要模块：多模态混合检索和多尺度视图生成。具体而言：
多模态混合检索：该模块通过文本和视觉两条管道进行信息检索，利用高斯混合模型动态确定最优的检索结果数量K，从而实现更精准的信息获取。
多代理生成：该模块包含三个主要代理：探寻代理、检查代理和回答代理。探寻代理负责快速筛选相关图像，检查代理进行详细审核并提供反馈，而回答代理则整合信息生成最终答案。此过程通过迭代反馈机制提高了生成答案的准确性和一致性。
通过这种多代理的协作方式，ViDoRAG能够在复杂的视觉文档环境中有效地进行信息检索和推理，确保生成的答案既准确又具有高效性。
在实验部分，ViDoRAG的有效性通过在ViDoSeek数据集上的一系列测试得到了验证。实验设置包括使用多种基线模型进行比较，评估其在检索和生成任务中的表现。具体而言，使用了GPT-4o和Qwen2.5等闭源和开源模型进行性能测试，结果显示ViDoRAG在多种查询类型上均优于传统的RAG方法。此外，实验还涉及动态检索的时间效率分析，表明尽管多代理系统引入了一定的延迟，但总体生成质量得到了显著提升。最终，ViDoRAG在多个评估指标上表现出色，证明了其在复杂视觉文档处理中的潜力和优势。
在ViDoRAG的方法中，有两个主要部分帮助我们更好地从丰富的视觉文档中获取信息。首先是“多模态混合检索”，这就像是一个聪明的搜索引擎，它不仅会查找文本信息，还会同时查看图片，确保找到最相关的内容。为了做到这一点，它使用了一种叫做高斯混合模型的数学工具，帮助它决定要查找多少条信息，以便既能找到足够的参考资料，又不会浪费时间在不相关的内容上。其次，ViDoRAG还引入了三个不同的“代理”来处理信息：探寻代理像一个侦探，负责快速找到相关的图片；检查代理则像是一个审查员，仔细检查找到的信息；最后，回答代理会整合所有的信息，给出一个准确的答案。通过这样的合作，ViDoRAG能够高效地处理复杂的问题，确保我们得到的信息既准确又有用。
2025-02-24｜UCLA,UCMerced,AdobeResearch
http ://arxiv.org/abs/2502.17651v1https ://metal-chart-generation.github.io
在现代数据分析和可视化领域，图表生成变得尤为重要。METAL（多智能体框架）是一个基于视觉语言模型的创新框架，旨在通过分解复杂的图表生成任务来提高自动化图表生成的效率。该框架通过四个专门的智能体进行协作，分别负责生成、视觉评估、代码评估和修正。这种多智能体的设计使得每个智能体能够专注于特定任务，从而提高整体性能。实验结果表明，METAL在图表生成任务上相较于现有最佳方法提升了5.2%的F1分数，并且在计算预算增加时表现出显著的性能提升，验证了其有效性与潜力。
METAL框架的设计包括四个主要智能体，各自承担不同的职责。首先，生成智能体（G）负责从参考图像生成初始代码；其次，视觉评估智能体（V）对生成的图表进行视觉差异分析，识别与参考图表的不同之处；接着，代码评估智能体（C）审查生成的代码，提供改进建议；最后，修正智能体（R）根据视觉和代码评估的反馈进行代码修改。这一过程是迭代进行的，直到生成的图表达到预定的质量标准或达到最大尝试次数。此外，METAL引入了多标准验证机制，通过对颜色、文本和整体结构进行评估，确保生成图表的准确性和一致性。
为评估METAL的性能，我们使用了ChartMIMIC数据集，并与三种基线方法进行了比较：直接提示、提示增强和最佳选择。实验结果显示，METAL在所有评估指标上均表现优异，特别是在文本、类型、颜色和布局等方面，F1分数明显高于基线方法。以LLAMA3.2-11B为基础模型，METAL在图表生成任务中实现了11.33%的性能提升。对于GPT-4O，METAL的平均F1分数达到86.46%，显示出其在复杂图表生成任务中的强大能力。此外，消融研究进一步验证了各智能体在整体性能中的重要性，强调了视觉和代码评估智能体的分离设计对提升自我修正能力的贡献。
METAL的工作原理可以想象成一个团队合作的过程。首先，团队中的“生成者”负责从一张参考图表中创建一个初步的代码，这就像是画出图表的草图。然后，另一个成员“视觉评估者”会仔细检查这个草图，找出与参考图表的不同之处，比如颜色、布局等。接下来，“代码评估者”会审查生成的代码，指出哪里可以改进。最后，“修正者”会根据前面两个成员的反馈，对代码进行修改，直到生成的图表达到预期的效果。这一系列的步骤是反复进行的，确保每次修改都能使结果更接近理想状态。通过这种分工合作，METAL能够更高效地生成高质量的图表。
2025-02-24｜UCLA,UIUC,TAMU｜🟡
http ://arxiv.org/abs/2502.17709v1
在当今的多模态模型（LMMs）中，识别新概念和低资源概念仍然是一个挑战。由于这些模型主要依赖于预训练知识，它们在捕捉细微的视觉细节方面能力有限，导致在处理视觉相似或常被误解的概念时表现不佳。为了解决这一问题，本文提出了一种新的对比视觉数据增强方法（CoDA），旨在通过提取目标概念与混淆概念之间的对比特征，帮助LMMs更好地对齐视觉特征与语言信息。CoDA利用多模态生成模型生成合成数据，经过自动过滤以确保生成数据的质量。通过在多个低资源概念和多样场景识别数据集上的测试，CoDA显著提高了模型的准确性，展示了其在视觉数据增强方面的有效性和效率。
CoDA方法主要包括四个步骤：特征提取、特征过滤、特征控制的图像生成和增强图像过滤。首先，对于每个目标概念，CoDA会识别出一个最相似的混淆概念，并提取它们之间的对比文本和视觉特征。特征提取的过程中，使用大型语言模型（LLM）和视觉语言模型（VLM）来获取目标概念的关键特征。接着，通过自动过滤，确保提取的特征具有较高的区分性和生成性。然后，经过特征过滤的特征会传递给文本到图像生成模型，以生成增强的视觉数据。最后，通过对生成的图像进行验证，确保其包含了目标概念的特征，从而用于进一步更新LMMs。
为了评估CoDA在新概念和混淆概念识别中的有效性，研究者在三个数据集上进行了实验：iNaturalist、SUN和NovelSpecies。实验设置包括固定真实数据和固定计算资源两种情境，分别模拟现实中的数据稀缺和丰富场景。结果显示，CoDA在使用少量真实图像的情况下，能够显著提高模型的识别准确率，尤其是在NovelSpecies数据集上，表现出相对于现有方法的绝对增益。此外，CoDA还在传统分类器和其他LMMs上取得了良好的效果，验证了其在视觉数据增强中的广泛适用性。
CoDA方法可以看作是一个四步流程，帮助机器学习模型更好地识别新事物。第一步，模型会找到一个与目标概念相似的混淆概念，并提取它们的视觉和文本特征。第二步，模型会对提取的特征进行筛选，确保这些特征能够有效区分目标概念和混淆概念。第三步，经过筛选的特征会被用来生成新的合成图像，帮助模型学习。最后，模型会检查这些生成的图像，确保它们确实包含了目标概念的特征。通过这样的方法，CoDA能够有效地提升模型对新概念的识别能力，特别是在数据稀缺的情况下。

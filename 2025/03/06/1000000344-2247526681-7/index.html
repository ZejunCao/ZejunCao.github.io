2025-02-26｜PT,IIT｜⭐️🟡
http ://arxiv.org/abs/2502.18745v1https ://gabrieletiboni.github.io/MaskPlanner/
本研究提出了一种新颖的深度学习方法——MaskPlanner，旨在解决以3D点云为基础的对象中心运动生成（OCMG）问题。OCMG在工业应用中至关重要，如机器人喷涂和焊接等，要求算法具备高效性、可扩展性和通用性。当前的解决方案通常依赖于复杂的优化过程或特定几何假设，限制了其在现实场景中的适应性。MaskPlanner通过从3D点云中学习专家路径模式，能够在不依赖于手工设计的优化目标的情况下，直接生成多个长时间轨迹。该方法通过预测局部路径段和路径掩码的方式，捕捉局部几何特征和全局任务需求，显示出在未见对象上的几乎完整覆盖率（超过99%），并在现实场景中验证了其可行性。
MaskPlanner的核心是将OCMG问题转化为从3D对象点云到一组不规则输出路径的映射。该方法主要分为以下几个步骤：首先，使用深度神经网络对输入点云进行特征提取，并预测路径段和路径掩码。路径段是表示机器人末端执行器位置的6D姿态序列，而路径掩码则用于标识哪些段属于同一路径。其次，采用多组件损失函数，通过Chamfer距离优化路径段预测的准确性。最后，设计了一种后处理步骤，将属于同一路径的段连接起来，形成完整的可执行路径。MaskPlanner的设计允许在一次前向传递中同时完成局部和全局规划决策，增强了模型的灵活性和效率。
本研究在扩展的PaintNet数据集上进行了广泛的实验，以评估MaskPlanner的性能。数据集包含3088个样本，涵盖不同复杂度的3D对象。实验结果表明，MaskPlanner在未见对象上能生成高质量的喷涂路径，且与人类专家的路径质量相当。通过与其他基准方法的比较，MaskPlanner在路径数量的准确性、路径的平均绝对误差和喷涂覆盖率等多个指标上表现优异。此外，MaskPlanner在执行时间上表现出色，能够在约100毫秒内生成多个路径，显示出其在实际工业应用中的潜力。实验还验证了该方法在处理复杂形状和路径配置时的灵活性和有效性。
MaskPlanner的工作原理可以简单理解为一个智能助手，它通过观察专家如何在复杂的3D物体上进行喷涂，学习并模仿这些动作。首先，它会将物体的形状转换成一组点，这些点就像是物体的“蓝图”。然后，MaskPlanner会分析这些点，找出喷涂所需的路径。具体来说，它不仅预测喷涂的每个小段，还会同时标记哪些小段是属于同一条喷涂路径。这样一来，MaskPlanner就能在一次计算中快速得出所有喷涂路径，避免了逐段处理时可能出现的错误。最终，它会将这些路径组合成完整的、可以直接在机器人上执行的喷涂轨迹，确保喷涂的质量和效率。
2025-02-26｜HKUST(GZ)｜ICRA2025｜⭐️🟡
http ://arxiv.org/abs/2502.19128v1https ://github.com/3DAgentWorld/SCA3D
在跨模态3D检索任务中，目标是实现文本描述与3D形状之间的有效匹配，这在机器人技术中尤为重要。现有的跨模态3D检索方法主要依赖于有限的3D数据，导致模型在复杂真实场景中的泛化能力不足。为了解决这一问题，本文提出了一种名为SCA3D的新颖在线数据增强方法，通过生成丰富的3D形状和文本配对数据来提高检索性能。该方法利用LLaVA模型为每个3D形状的分段部分生成描述，进而创建大量新的3D-文本对，增强了语义特征的多样性。通过对3D形状的组件进行精细对齐，SCA3D显著提升了文本与3D形状之间的相似性匹配，进而提高了检索的准确性和效率。
SCA3D的核心由三个主要组件构成：数据增强模块、单模态编码器和匹配模块。首先，数据增强模块从现有的3D形状中提取多个部分，生成新的3D形状和相应的文本描述。具体步骤包括：
分段描述生成：使用LLaVA模型为每个3D形状的部分生成文本描述。
组件库构建：将生成的部分及其描述存储在组件库中，以便于后续的随机采样。
形状生成：随机选择多个部分并使用文本模板生成新的3D形状及其描述，确保这些部分在空间上合理排列，避免重叠。
相似性计算：通过地球移动者距离（EMD）计算3D形状与文本描述之间的相似性，优化匹配过程。该方法结合对比学习，提升了跨模态对齐的效果。
在实验部分，我们使用Text2Shape数据集进行评估。实验结果显示，SCA3D在形状到文本（S2T）和文本到形状（T2S）检索任务中均取得了显著的性能提升。具体而言，SCA3D的Shape-to-TextRR@1得分从20.03提升至27.22，Text-to-ShapeRR@1得分从13.12提升至16.67。此外，我们还进行了消融实验，以验证数据增强、相似性度量和对比学习损失等各个组成部分的重要性。结果表明，数据增强方法显著提高了检索性能，而EMD作为相似性度量能够更好地捕捉3D形状与文本之间的细微关系。
SCA3D的方法可以简单理解为一个智能的“拼图游戏”。首先，它从已有的3D物体中提取出各个部分，比如椅子的座位、靠背和腿。接着，利用一个强大的语言模型（LLaVA）为每个部分生成描述，比如“这是一个木质椅子的座位”。然后，系统随机组合这些部分，形成新的椅子，并为新组合生成合适的描述。为了确保这些新椅子看起来合理，系统会调整各个部分的位置，避免它们重叠。最后，通过计算新椅子和描述之间的相似度，系统可以准确地匹配文本和3D形状。这种方法不仅提高了检索的准确性，还让系统能够理解和生成更多样化的3D物体。
2025-02-26｜HKBU,UCampinas,PKU,A*STAR,CUHK
http ://arxiv.org/abs/2502.19125v1https ://github.com/luo-ziyuan/NeRF_Signature
随着神经辐射场（NeRF）在三维内容表现中的广泛应用，版权保护问题变得日益重要。现有的数字水印技术在NeRF中的应用，往往忽视了模型级别的考虑，导致了隐蔽性和鲁棒性的降低。为了解决这些问题，本文提出了一种新颖的水印方法——NeRFSignature，采用了代码本辅助签名嵌入（CSE）技术。这种方法在不改变模型结构的情况下，能够有效地嵌入水印，从而保持隐蔽性和增强鲁棒性。此外，本文还引入了一种联合姿态-补丁加密水印策略，将签名隐藏在特定视角渲染的补丁中，以提高鲁棒性。实验结果表明，所提出的方法在隐蔽性和鲁棒性方面均优于现有基线方法。
本文的方法主要包括以下几个步骤：首先，构建一个签名代码本，用于生成签名表示。然后，通过代码本辅助签名嵌入（CSE），将生成的签名表示直接添加到NeRF模型的参数中，确保模型结构不变。接着，为了增强鲁棒性，采用联合姿态-补丁加密水印策略，将签名嵌入特定视角下渲染的补丁中。最后，实施复杂度感知密钥选择（CAKS）方案，选择视觉复杂度高的补丁进行水印嵌入，以进一步提升隐蔽性。在整个过程中，水印的嵌入和提取均依赖于一个秘密密钥，以确保只有NeRF所有者能够验证模型的版权。
为评估所提出的方法，实验在四个标准数据集上进行，包括Blender、LLFF、Tanks&Temples和Mip-NeRF360数据集。通过与四种基线方法进行比较，评估了准确性、隐蔽性和鲁棒性三个关键方面。实验结果表明，NeRFSignature在比特准确率和图像质量（PSNR、SSIM、LPIPS）方面均表现优异。尤其是在高比特数情况下，所提出的方法在隐蔽性和鲁棒性方面显著优于其他方法。此外，通过对不同视角渲染的补丁进行验证，进一步确认了水印的有效性和安全性，确保了模型的版权保护。
本文的方法可以简单理解为一个三步走的过程。首先，我们创建了一个“签名代码本”，就像一本记载了各种签名的手册。接着，使用这个手册，我们将签名信息嵌入到NeRF模型中，而不改变模型的基本结构，这样做的好处是水印不会被轻易发现。最后，为了确保水印更加安全，我们采取了一种特殊的方式，把签名藏在特定视角下的图像补丁中，只有知道特定秘密的用户才能提取这些签名。这种方法不仅能有效保护版权，而且操作起来也相对简单，方便用户随时更新他们的签名。
2025-02-26｜TUWien,Google,Inria,UCA

3DGaussianSplatting(3DGS)是一种有效的三维场景表示方法，能够实现高质量的实时新视图合成。与基于体积渲染的神经辐射场（NeRF）相比，3DGS采用了一种混合渲染方案，结合了体积渲染和原始光栅化的优势。尽管3DGS在性能上表现优异，使用了一系列简化假设和近似，但这也引发了一个重要问题：如果采用更为准确的体积渲染方法，是否能够提升3DGS的视觉质量。为此，本文深入分析了3DGS中所采用的各种近似和假设，探讨了在不同高斯原始体素数量下，体积渲染与3DGS的性能差异。研究结果表明，尽管更准确的体积渲染在低数量原始体素时有所帮助，但在高高斯数量的情况下，3DGS的效率和优化能力使其在视觉质量上优于传统的体积渲染方法。
本文提出了一种系统的数学框架，旨在分析3DGS的不同近似和假设对视觉质量的影响。具体方法包括：
数学框架构建：从体积渲染的基本原理出发，明确3DGS和准确体积渲染之间的区别，特别是透明度与消光基于渲染的关系。
引入消光基于的光栅化算法：为分析3DGS与体积渲染的区别，开发了消光基于的光栅化算法，并提供了自我衰减高斯的闭合形式解决方案。
实验评估：通过逐步替换3DGS中的各项近似，评估其对视觉质量的影响，特别是在高斯数量变化时，分析不同渲染方法的效果。
为验证所提出方法的有效性，研究团队进行了多项实验，比较3DGS与传统体积渲染方法在不同高斯数量下的表现。实验设计包括：
场景选择：使用NeRF合成数据集中的LEGO场景，设置不同数量的高斯原始体素（如4k和100k高斯）。
性能评估：对比3DGS与消光基于体积渲染在视觉质量和渲染速度上的差异，尤其关注在高高斯数量条件下的表现。
结果分析：通过定量和定性的方式评估渲染结果，分析不同渲染方法的优劣，揭示在高高斯数量情况下3DGS的优势。
在这项研究中，作者们通过建立一个数学模型，来比较两种渲染技术：3DGaussianSplatting（3DGS）和传统的体积渲染。首先，他们明确了这两种方法在处理透明度和光的消散方面的不同。接着，他们设计了一种新的算法，专门用来更好地处理光的消散现象。通过实验，研究人员发现，当使用较少的高斯点时，传统的体积渲染效果更好，但随着高斯点数量的增加，3DGS的表现反而优于传统方法。这表明，虽然3DGS在某些方面做了简化，但在实际应用中，它的效率和效果可以超越更复杂的渲染技术。

主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）｜编辑支持｜李观丽（GuanliLi）发布日期｜2025年2月26日｜周三
2025-02-25｜THU,UCBerkeley｜⭐️🟡
http ://arxiv.org/abs/2502.18137v1https ://github.com/thu-ml/SpargeAttn
在大型模型的推理过程中，注意力机制的计算复杂度通常呈现平方级增长，导致推理速度缓慢。现有的稀疏注意力方法虽然在特定模型中取得了一定的加速效果，但普遍适用性仍然不足。为了解决这一问题，本文提出了一种名为SpargeAttn的通用稀疏注意力机制。该机制能够适用于多种模型类型，并在不损失模型性能的情况下显著提高推理速度。实验表明，SpargeAttn在多种生成任务中表现出色，能够在语言、图像和视频生成等领域实现2.5倍至5倍的速度提升。
SpargeAttn采用了一种两阶段的在线过滤器来优化注意力计算。具体步骤如下：
稀疏掩码预测：在第一阶段，通过快速准确地预测注意力图，构建稀疏掩码，该掩码指示需要计算的非零条目，从而跳过不必要的矩阵乘法。
在线Softmax计算：在第二阶段，设计了一种在线Softmax算法，利用全局最大值和局部最大值之间的差异，进一步减少一些乘法运算的需求。
集成量化：将稀疏注意力方法嵌入8位量化的SageAttention框架中，以实现更高效的计算。
该方法不仅提高了模型的推理速度，还保持了高准确性，适用于各种输入和任务，解决了传统稀疏注意力方法在效率和准确性之间的平衡问题。
在多种生成任务上评估了SpargeAttn的性能，包括语言建模和图像/视频生成。实验结果显示，SpargeAttn能够在保持模型整体性能的同时，显著加快推理速度。与现有的稠密和稀疏注意力模型相比，SpargeAttn的速度提升在2.5倍到5倍之间，且在各类任务中均表现出色。此外，SpargeAttn在高序列长度下（如45K-128K）也能有效运行，表明其在处理大规模数据时的优越性。所有实验均在L40GPU上进行，确保了结果的可靠性和可比性。
SpargeAttn的方法可以简单理解为一种聪明的计算方式，它通过两步走来加速注意力计算。首先，它快速判断哪些数据是重要的，哪些可以忽略，这样就能省去一些不必要的计算。然后，它在计算的时候，利用一些小技巧，进一步减少计算量。这样一来，虽然计算的内容减少了，但最终得到的结果依然准确。这种方法不仅适用于各种类型的模型，还能在保持性能的同时大幅提升速度，适合处理各种复杂的任务。
2025-02-25｜SonyAI,DondersInstitute,SonyGroupCorporation,imec｜⭐️
http ://arxiv.org/abs/2502.18197v1https ://github.com/sony/vct
本研究提出了一种基于变分噪声耦合的训练一致性模型的新方法。近年来，一致性训练（CT）作为扩散模型的替代方案，在图像生成任务中表现出色。然而，传统的非蒸馏一致性训练常常面临高方差和不稳定性的问题。为了解决这些问题，研究者们探讨了训练动态的分析与改进。本文的主要贡献在于提出了一种新的噪声耦合方案，灵感来源于变分自编码器（VAE）的架构。通过训练一个数据依赖的噪声发射模型，本文的方法可以间接学习噪声到数据映射的几何结构，从而在多个图像数据集上显著提升生成性能，超越了现有基线并实现了在CIFAR-10和ImageNet数据集上的最先进成果。
本文的方法主要包括以下几个步骤：
噪声耦合学习：通过引入一个编码器网络，学习数据与噪声之间的条件耦合函数。这一过程使得模型能够在生成过程中自适应地调整噪声输出，从而提高生成样本的质量。
流匹配框架：将一致性训练框架与流匹配方法相结合，利用流函数来构建从数据分布到噪声分布的映射。通过这种方式，模型能够更有效地捕捉数据的内在结构。
变分正则化：在训练过程中，采用Kullback-Leibler散度作为正则化项，以确保生成的噪声分布与理想的高斯分布相近。这种方法不仅提高了模型的稳定性，还增强了生成样本的多样性。
离散化一致性训练：在训练过程中，优化离散化损失函数，结合总导数的计算，以提高训练效率和生成质量。
在实验部分，研究者们对所提出的模型进行了广泛评估，使用了多个标准数据集，包括Fashion-MNIST、CIFAR-10、FFHQ和ImageNet。实验结果表明，采用变分噪声耦合的模型在一和两步生成设置下均显著优于传统一致性训练模型，并在多个基准上实现了最先进的性能。特别是在CIFAR-10和ImageNet上，模型的FID（FrechetInceptionDistance）得分显示出明显的改善。研究还通过对比不同的超参数设置，分析了KL散度权重对生成性能的影响，验证了模型在高维数据生成方面的有效性。此外，研究者们还探讨了模型训练过程中的梯度方差，发现引入的耦合机制有助于降低训练中的方差，进一步提升生成效果。
在这项研究中，研究者们提出了一种新方法，旨在改善图像生成模型的训练过程。简单来说，他们通过建立一个“桥梁”，让噪声和数据之间的关系更加紧密，从而使生成的图像质量更高。具体来说，他们使用了一个特殊的网络，来学习如何将输入的噪声转化为更符合真实数据的样本。在训练时，研究者们还加入了一种叫做变分正则化的技术，这就像是在训练过程中给模型提供额外的指导，帮助它更好地理解噪声的形状和分布。最后，他们在多个数据集上进行了实验，结果表明，这种方法不仅提高了生成图像的清晰度，还使得生成的图像更加多样化，表现优于传统的方法。
2025-02-19｜AlibabaGroup,BUPT｜⭐️🟡
http ://arxiv.org/abs/2502.13995v1https ://fantasy-amap.github.io/fantasy-id/
本研究提出了一种名为FantasyID的创新框架，用于生成身份保持的动态视频。该框架旨在通过增强面部知识，克服现有文本到视频生成（IPT2V）方法在面部动态和身份一致性方面的挑战。FantasyID结合了3D面部几何先验，以确保生成视频中的面部结构稳定，同时采用多视角面部增强策略，提升面部表情和头部姿态的动态表现。通过无调优的方式，FantasyID能够在保持个体身份的同时，生成更自然、更具表现力的视频。实验结果表明，该方法在视频质量、身份保持和时间一致性等方面均优于现有的调优方法。
FantasyID的核心方法包括以下几个步骤：
多视角面部集合：从每个训练视频中提取面部图像，构建多视角面部集合，以避免模型学习到简单的复制粘贴策略。通过选取不同视角的面部图像，增强模型对个体特征的理解。
面部抽象器：使用C-Abstractor模块提取面部特征，确保空间结构的完整性，避免特征间的干扰。
3D约束：利用DECA框架提取3D面部特征，以增强生成视频的面部稳定性。这一过程有助于区分身份相关特征和与身份无关的因素（如表情和姿态）。
层次感知控制信号注入：根据每层对控制信号的敏感性，动态调整特征的整合，以优化面部视频生成的质量和身份保持。这些方法的结合使得FantasyID在生成动态视频时，能够更好地捕捉面部表情的细微变化，确保高保真度和一致性。
在实验部分，研究者使用了包含全身和肖像数据的多样化数据集进行训练。为了评估模型的性能，采用了包括ArcFace嵌入相似度、参考相似度、帧间相似度和面部运动度等多个指标。实验结果显示，FantasyID在身份保持和动态表现上均优于对比模型，如ID-Animator和ConsisID。具体而言，FantasyID在面部运动的自然流畅性和整体视频质量方面表现突出。用户研究也表明，参与者对FantasyID生成的视频在整体质量、面部相似性、结构稳定性和动态表现等方面给予了较高评价。此外，通过消融实验，验证了每个模块在提升模型性能中的重要性，为进一步的优化提供了依据。
FantasyID的工作原理可以简单理解为一个多步骤的过程。首先，它会从视频中提取不同角度的面部图像，这样模型就能更全面地了解一个人的面部特征，而不是仅仅依赖于一张静态图像。接下来，模型使用一种叫做面部抽象器的工具来提取这些面部特征，确保在生成视频时不会出现失真。然后，模型还会利用3D技术来保持面部结构的稳定性，这样即使在动态场景中，面部特征也不会失真。最后，FantasyID会根据不同层的需要，灵活调整生成的面部特征，以确保视频的质量和个体身份的保持。这一系列的步骤使得生成的视频不仅真实感强，而且动态表现也非常自然。
2025-02-25｜HIT-SZ,SoochowU,CSU,PengChengLab｜⭐️🟡
http ://arxiv.org/abs/2502.17863v1
本研究回顾了视频生成领域中的时空一致性问题，强调了在生成高质量视频时，空间和时间一致性的重要性。视频被视为一系列连续的图像帧，生成过程需要确保这些帧在空间上保持一致，避免出现色彩、形状和物体位置的扭曲，同时在时间上也要确保帧与帧之间的变化平滑自然。该文献综述了近年来在视频生成技术方面的进展，特别关注五个关键领域：基础模型、信息表示、生成方案、后处理技术和评估指标。通过系统性地分析这些方面，旨在填补现有文献中关于时空一致性研究的空白，以推动视频生成技术的进一步发展。
在视频生成中，研究者们采用了多种基础模型以维护时空一致性，主要包括以下几种：
生成对抗网络（GAN）：通过生成器和判别器之间的博弈，生成高质量的视频帧，同时优化生成过程中的时空特征。
自回归模型：当前帧的生成依赖于前面帧的信息，确保视频帧之间的连贯性。
扩散模型：最初用于图像生成，后转向视频生成，通过逐步去噪的过程生成视频数据，保持时空一致性。
掩码模型：通过随机掩盖时间序列中的部分元素，训练模型准确预测被掩盖的部分，从而提高生成效果。
视觉编码器：将输入图像或视频转化为特征表示，结合上述模型，以提取和表示视频的深层时空信息。通过这些方法，研究者们能够在生成过程中有效维护视频的时空一致性。
本研究通过对比分析不同视频生成模型的性能，评估其在保持时空一致性方面的有效性。实验分为多个阶段，包括基础模型的选择、信息表示的优化、生成方案的实施以及后处理技术的应用。具体来说，通过对比GAN、自回归模型和扩散模型在生成视频时的表现，评估其在空间一致性和时间连贯性方面的能力。此外，采用了一系列评估指标，如峰值信噪比（PSNR）、结构相似性指数（SSIM）和弗雷歇视频距离（FVD）等，来量化生成视频的质量和一致性。实验结果表明，综合运用上述模型和技术能够显著提升视频生成的质量，并解决时空一致性问题。
在视频生成的过程中，研究者们采用了几种不同的方法来确保生成的视频在视觉上是连贯和自然的。首先，他们使用生成对抗网络（GAN），这种方法通过两个“竞争”的网络来生成高质量的图像。其次，自回归模型让每一帧的生成依赖于前面的帧，这样可以保持视频的连贯性。扩散模型则是通过逐步去除噪声来生成视频，就像把一幅模糊的画慢慢变清晰。掩码模型则通过隐藏视频中的某些部分，训练模型去预测这些部分，从而提高生成的准确性。最后，视觉编码器帮助把视频转化为可以更容易处理的特征信息。通过综合运用这些方法，研究者们能够生成更高质量、更一致的视频。

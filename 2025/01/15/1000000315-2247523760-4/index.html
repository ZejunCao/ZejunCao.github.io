2024-12-07｜XJTLU,SJTU
http ://arxiv.org/abs/2501.01959v1
本研究提出了一种名为STEAM-EEG的新框架，旨在提高脑电图（EEG）信号分析的效果。EEG信号在生物医学研究和临床应用中具有重要作用，但由于其复杂性和非平稳性，分析和解释这些信号面临诸多挑战。STEAM-EEG结合了现代计算机图形技术与生物信号模式识别，特别是采用马尔可夫转移场（MTFs）对EEG时间序列进行成像。该框架能够捕捉EEG信号的时空动态，并将其转化为视觉信息丰富的图像，从而促进数据探索、模式识别和决策制定。通过与多种现有方法的比较，STEAM-EEG在多个EEG数据集上显示出显著的准确性和鲁棒性，证明了其在EEG信号分析中的潜力。
STEAM-EEG的核心方法包括以下几个关键步骤：
趋势-季节性分解（SSA）：首先，使用奇异谱分析（SSA）将原始EEG时间序列分解为趋势、季节性和噪声成分。这一过程有助于提取信号中的重要特征并减少噪声干扰。
并行一维卷积神经网络（1D-CNNs）：将分解后的EEG成分输入到并行的1D-CNN中，以学习区分性特征。该网络采用了改进的注意力机制，能够捕捉跨通道依赖性，从而提高特征学习的能力。
马尔可夫转移场成像（MTF）：通过MTF成像对EEG信号的时空动态进行建模，生成信息丰富的视觉表示。MTF模型通过概率图模型捕捉不同区域之间的空间和时间依赖关系。
二维残差卷积神经网络（2D-ResNet）：最后，将生成的MTF图像输入到2D-ResNet中，以提取更具辨别性的特征并进行分类。
为评估STEAM-EEG的性能，研究者使用了来自UCR时间序列分类档案的多种EEG数据集，并与多个基线方法进行比较，如FCNN、EEGNet、DeepConvNet和STFT-CNN。实验结果表明，STEAM-EEG在分类准确性和F1分数上均优于现有方法。此外，进行的消融研究显示，各个方法组件的贡献，特别是MTF成像和SSA分解，对模型性能的提升至关重要。通过可视化分析，研究者进一步验证了SSA在提取EEG信号中关键特征的有效性。
在STEAM-EEG方法中，我们首先将EEG信号进行“拆解”，就像把一首复杂的乐曲分解成旋律、和声和节奏。这个步骤叫做趋势-季节性分解（SSA），它帮助我们找出信号中真正重要的部分。接下来，我们使用一种特殊的神经网络（1D-CNN）来学习这些拆解后的信号特征，这就像训练一个音乐家去识别不同的乐器声音。为了更好地理解信号的变化，我们还使用了马尔可夫转移场（MTF）技术，这种方法可以帮助我们看到EEG信号在时间和空间上的变化，生成易于理解的图像。最后，我们将这些图像输入到另一个神经网络（2D-ResNet）中，进一步提取特征并进行分类。这样的方法组合使得我们能够更准确地分析和理解复杂的EEG信号。
2025-01-06｜Stanford,THU,MIT
http ://arxiv.org/abs/2501.03225v1https ://yuhui-zh15.github.io/AutoConverter-Website/
随着视觉语言模型（VLM）的快速发展，对其进行严格和可靠的评估显得尤为重要。当前的视觉问答（VQA）基准往往使用开放式问题，这使得评估变得困难，因为模型生成的自然语言答案具有很大的变异性。为了解决这一问题，本文提出了AutoConverter，这是一种自动将开放式问题转换为多项选择题的框架，从而提高评估的客观性和可重复性。通过使用AutoConverter，研究者们构建了VMCBench，这是一个统一的多项选择VQA基准，包含9,018个问题，旨在为未来的VLM研究提供标准化的评估工具。实验证明，AutoConverter生成的问题在挑战性和正确性上均超过了人类创建的问题，为VLM的评估设定了新的标准。
在AutoConverter的设计中，核心目标是将开放式问题转换为多项选择格式，以便于评估。该过程分为几个关键步骤：
确保正确性：使用GPT-4o作为评估代理，检查每个选项与正确答案的相似性，并为每个问题打分，确保每个问题仅有一个正确答案。
增加难度：通过引入多个代理，生成具有挑战性的干扰项，确保问题不会因明显错误选项而变得简单。干扰项的生成基于常见错误类型，如概念误解和视觉误读。
迭代精炼：在生成初步干扰项后，使用评审代理对其进行评估，并根据反馈进行调整。这一过程不断迭代，直到达到所需的正确性和难度标准。
最终选择：经过多轮评估和调整后，选择最具挑战性的干扰项，确保生成的问题既正确又具有良好的区分能力。
在实验中，AutoConverter被应用于多个现有的多项选择VQA数据集，包括MMMU、MathVista和AI2D。通过与人类创建的问题进行比较，结果显示，VLM在AutoConverter生成的问题上表现出相似或更低的准确率，表明这些问题在难度上具有挑战性。为确保生成问题的正确性，研究团队设计了一套内部评估机制，对每个转换后的问题进行评分和验证。最终，经过人工审核，确认绝大部分生成的问题都是准确的，表明AutoConverter在生成高质量多项选择问题方面的有效性。
AutoConverter的工作原理可以简单理解为一个智能助手，它帮助将开放式问题转化为多项选择题。首先，它会检查每个问题，确保只有一个正确答案。接下来，AutoConverter会创造一些看似合理但实际上是错误的选项，这些选项被称为“干扰项”。为了让问题更具挑战性，系统会不断评估和调整这些干扰项，确保它们不会太简单。最后，经过几轮的调整和评估，AutoConverter会选择出最具挑战性的干扰项，形成最终的问题。这种方法不仅提高了问题的质量，也让评估变得更加客观和一致。
2025-01-06｜Aibee,THU
http ://arxiv.org/abs/2501.02964v2https ://github.com/aibee00/SocraticQuestioning
复杂的视觉推理仍然是当前研究中的一项重要挑战。传统方法如思维链（ChainofThought,CoT）和视觉指令调优在处理视觉推理任务时表现良好，但如何将这两种方法有机结合以实现更好的效果尚未得到充分探讨。此外，模型产生幻觉（hallucinations）和高训练成本的问题依然存在。为此，本文提出了一种创新的多轮训练和推理框架，称为苏格拉底式提问（SocraticQuestioning,SQ），该框架适用于轻量级多模态大语言模型（MLLMs）。SQ方法通过自我提问引导模型聚焦于与目标问题相关的视觉线索，从而减少幻觉现象，并增强模型对图像细节的描述能力。这一方法在复杂视觉推理和问答任务中表现出色，并且我们还创建了一个名为CapQA的多模态小型数据集，以便于视觉指令调优和评估。
苏格拉底式提问（SQ）方法是通过以下四个步骤来实现复杂视觉推理的：
自我提问：模型首先识别出进行推理所需的细粒度信息，并生成相关问题。
自我回答：模型根据自我提出的问题，从图像中获取所需的细节信息。
整合与组织：模型将生成的问题与答案对进行整合，形成对图像的详细描述。
总结与浓缩：最后，模型提取出最相关的信息，生成简洁的总结性标题。
通过这种多轮对话格式，SQ方法有效地利用了自我提问的策略，增强了模型的推理能力，同时减少了幻觉现象的发生。此外，SQ方法的架构设计集成了问题生成、回答和视觉总结的功能，使得训练过程在内存和计算上都更加高效。
在实验部分，研究团队对SQ方法进行了广泛的评估，使用了新创建的CapQA数据集。实验结果表明，SQ方法在多个基准测试中显著提高了模型的表现，尤其是在幻觉评分上改善了31.2%。通过与多种先进方法的对比，SQ方法在视觉推理和问答任务中展现出卓越的能力，尤其是在处理复杂问题时，采用三轮推理模式进一步减少了幻觉现象。实验还表明，SQ方法不仅提高了问题质量评分，还在减少训练成本的同时，提升了模型的整体性能。
在苏格拉底式提问方法中，模型通过自我提问来帮助自己更好地理解图像中的内容。首先，模型会想出一些相关的问题，比如“这个人正在做什么？”接着，它会根据这些问题从图像中寻找答案。然后，模型会把这些问题和答案整理在一起，形成一段详细的描述，最后再将这些信息浓缩成一句简短的总结。这样，模型就能更准确地理解图像，并减少错误信息的产生。通过这种方式，模型不仅能更好地处理复杂的视觉任务，还能在不增加额外计算负担的情况下，提高自己的推理能力。
2024-12-30｜THU,Infinigence-AI,PKU,SJTU
http ://arxiv.org/abs/2501.01986v1https ://github.com/thu-nics/FrameFusion
随着视频理解任务的复杂性不断增加，处理长时间和高分辨率视频对大型视觉语言模型（LVLMs）造成了巨大的计算压力。这些模型在视频理解中表现出色，但现有的令牌减少方法主要集中于基于重要性的剪枝，未能充分考虑由帧相似性和重复视觉元素引起的冗余。在此背景下，本文提出了一种新方法FrameFusion，通过分析LVLM中的视觉令牌相似性，结合相似性和重要性来优化令牌减少。研究表明，随着模型层数的加深，令牌相似性分布逐渐集中，但排名一致性保持稳定。FrameFusion通过在剪枝之前识别和合并相似令牌，为令牌减少提供了新的视角。实验结果显示，FrameFusion能够将视觉令牌减少70%，实现3.4至4.4倍的LLM加速和1.6至1.9倍的端到端加速，且性能损失小于3%。
FrameFusion的设计分为两个主要阶段：合并和剪枝。具体步骤如下：
相似性计算：首先，FrameFusion在初始层计算相邻帧之间视觉令牌的余弦相似性。通过仅计算相邻令牌的相似性，避免了N×N的复杂计算，从而降低了计算开销。
相似性合并：在相似性超过预设阈值的情况下，将相似的视觉令牌合并。合并过程在较浅的层进行，以充分利用高相似性，确保后续层的计算效率。
重要性剪枝：在合并完成后，FrameFusion利用累积注意力分数进行重要性剪枝，保留重要和独特的视觉令牌。根据用户设定的计算预算，确定保留的令牌数量。
通过这种两阶段的令牌压缩策略，FrameFusion能够在保持模型性能的同时，显著降低计算负担。
在实验中，FrameFusion与多种主流的令牌剪枝方法进行了比较，包括StreamingLLM和FastV。实验使用了多个视频理解基准，包括VideoMME、NExT-QA和VideoNIAH。结果表明，FrameFusion在30%的令牌预算下，能够在各个基准上获得与原始模型相当的性能，性能下降小于3%。此外，FrameFusion在效率上也表现出色，达到3.4至4.4倍的速度提升，同时显著降低了内存消耗。通过对不同模型和输入帧数量的测试，FrameFusion展示了良好的可扩展性和适应性。
FrameFusion的工作原理可以简单理解为一个高效的“信息筛选器”。首先，它会查看视频的每一帧，找出那些相似的部分。这就像是看电影时，发现某些镜头几乎一模一样，这些镜头的信息其实是重复的。FrameFusion会把这些相似的部分合并在一起，这样就减少了需要处理的信息量。接下来，它会根据每个部分的重要性来决定哪些信息可以保留，哪些可以去掉。这样，FrameFusion不仅能减少处理的数据量，还能保持视频理解的准确性。通过这个方法，FrameFusion能够快速处理视频，同时节省计算资源，确保在需要时仍然能够做出准确的判断。
2025-01-06｜THU,ZhipuAI
http ://arxiv.org/abs/2501.02955v1https ://motion-bench.github.io
随着视觉语言模型（VLMs）在视频理解领域的迅速发展，细粒度的运动理解能力却仍然未得到充分探索。为填补这一空白，研究团队提出了MotionBench，一个综合评估基准，旨在评估VLM在细粒度运动理解方面的表现。MotionBench通过六种主要的运动导向问题类型对模型的运动感知能力进行评估，并收集了来自多种来源的数据，确保涵盖广泛的现实世界视频内容。实验结果显示，现有的VLM在理解细粒度运动方面表现不佳，且仍有显著的提升空间。该基准的创建旨在引导和激励更强大的视频理解模型的发展，强调细粒度运动理解的重要性。
MotionBench的构建包括以下几个关键步骤：
数据收集：从多个公共数据集（如Panda-70M、MedVid、SportsSloMo等）和自生成的视频中收集视频片段。确保视频内容的多样性和复杂性，以便全面评估运动理解能力。
问题设计：根据运动理解的需求，设计六种主要类型的问题，包括运动识别、位置相关运动、动作顺序、重复计数、运动相关物体和摄像机运动。这些问题旨在考察模型在不同运动场景下的理解能力。
注释过程：对收集到的视频进行手动注释，生成问题和答案对。注释过程包括对动态变化的描述，以确保问题的准确性和多样性。每个视频的注释密度达到每秒12.63字，为模型的训练提供了丰富的资源。
在实验中，研究团队对多种现有VLM进行了评估，包括不同规模和架构的模型。结果表明，当前的主流模型在细粒度运动理解方面的准确率普遍低于60%。通过引入TEFusion（通过编码器融合）方法，研究团队实现了在高帧率输入下的显著性能提升，尤其是在压缩比高的情况下。实验还显示，TEFusion在MotionBench的表现优于其他压缩方法，进一步验证了该方法在细粒度运动理解中的有效性。通过对比不同模型在MotionBench上的表现，研究团队揭示了当前模型在处理快速和复杂运动时的局限性。
MotionBench的核心方法可以简单理解为一个评估系统，旨在测试视频理解模型对细微动作的理解能力。首先，研究团队从各种来源收集了大量视频，这些视频展示了人们在日常生活中可能遇到的各种动作。接着，他们设计了一系列问题，像是“视频中发生了什么动作？”这样的问题，帮助评估模型是否能准确识别和描述这些动作。最后，研究团队对这些视频进行了详细的注释，确保每个问题都是清晰和准确的。这一过程不仅帮助模型学习如何理解动作，还为未来的研究提供了丰富的资源。通过这样的方式，MotionBench不仅揭示了现有模型的不足之处，还推动了视频理解技术的进一步发展。
2025-01-06｜MSU,CornellU,ASU,UCBerkeley,UTAustin,Yale
http ://arxiv.org/abs/2501.02765v1https ://github.com/JackYFL/awesome-VLLMs
视觉大语言模型（VLLMs）在多模态学习和计算机视觉领域中正逐渐成为研究的热点。这些模型通过集成视觉和语言信息，展现出强大的推理能力和多任务处理能力。尽管VLLMs在一般用途和特定应用上取得了显著进展，但现有文献对其应用场景的全面探讨仍显不足。本研究旨在通过对VLLMs的应用场景进行系统性梳理，识别相关的伦理问题和挑战，并探讨未来的发展方向。通过汇总这些内容，本文希望为VLLMs的未来创新和更广泛的应用提供指导。
本研究采用了综合文献综述的方法，主要分为以下几个步骤：
文献收集：从多个数据库中收集与VLLMs相关的研究文献，涵盖不同的应用领域。
分类分析：根据应用场景将VLLMs分为三个主要类别：视觉到文本、视觉到行动和文本到视觉。每个类别下又细分为不同的子任务。
案例研究：对每个类别中的代表性模型进行深入分析，探讨其架构、训练方法和应用效果。
伦理与挑战讨论：识别VLLMs在应用中的伦理问题，并提出相应的解决方案和未来研究方向。
总结与展望：综合分析结果，提出对VLLMs未来发展的建议。
在实验部分，我们通过对比分析不同VLLMs在特定任务上的表现，评估它们的有效性。实验包括：
数据集选择：使用多个公开数据集，如COCO、VisualGenome等，进行模型训练和评估。
模型评估指标：采用准确率、召回率和F1分数等指标对模型进行性能评估。
对比实验：将VLLMs与传统视觉语言模型进行对比，分析其在多任务处理和推理能力上的优势。
用户反馈收集：通过用户调查获取对模型输出的反馈，评估模型在实际应用中的表现。
在本文的方法部分，我们主要通过几步来研究视觉大语言模型（VLLMs）。首先，我们从不同的学术数据库中找出与VLLMs相关的研究文章，确保涵盖多个领域。接着，我们把这些模型按照它们的应用方式分成三类：从视觉到文本（比如图像描述）、从视觉到行动（例如机器人控制）和从文本到视觉（如图像生成）。然后，我们选择一些代表性的模型，深入分析它们的工作原理和实际效果。我们还会讨论这些模型在应用中可能面临的伦理问题，比如隐私和安全，并提出解决方案。最后，我们会总结这些研究的发现，展望VLLMs未来的发展方向。通过这种方式，我们希望能让大家更好地理解VLLMs的应用和潜力。

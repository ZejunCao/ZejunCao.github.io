2025-01-06｜NJU,ByteDance,SWU｜⭐️
http ://arxiv.org/abs/2501.02976v1https ://nju-pcalab.github.io/projects/STAR
本研究提出了一种新颖的空间-时间增强框架（STAR），旨在利用文本到视频（T2V）模型来提升实际视频超分辨率（VSR）的质量。传统的VSR方法通常面临着在复杂退化情况下保持视频的空间细节和时间一致性的问题。为了解决这些问题，STAR引入了局部信息增强模块（LIEM）和动态频率损失（DFLoss），以改善视频恢复过程中出现的伪影和失真。通过对比实验，STAR在合成和真实数据集上均表现出优越的性能，证明了其在空间细节和时间一致性方面的有效性。
STAR框架的设计主要包括以下几个关键模块：
局部信息增强模块（LIEM）：
该模块在全局自注意力块之前引入，旨在增强局部细节，并减轻复杂退化所带来的伪影。
LIEM通过结合全局和局部特征，使模型能够更有效地处理细节恢复。
动态频率损失（DFLoss）：
该损失函数通过引导模型在不同扩散步骤中关注低频和高频信息，优化视频恢复的细节。
在扩散过程的早期，模型优先恢复低频结构，而在后期则集中精力于高频细节，从而提高整体恢复质量。
文本编码器和T2V模型：
STAR利用强大的T2V模型（如I2VGen-XL），通过文本嵌入生成高质量的超分辨率视频。
控制网络（ControlNet）用于引导T2V模型，使其在生成过程中更好地结合文本信息和视频内容。
在实验部分，研究团队使用多个合成和真实数据集对STAR进行了评估。通过比较不同的超分辨率方法，STAR在多个标准评估指标（如PSNR,SSIM,LPIPS等）中均表现优异。具体实验包括：
数据集选择：使用OpenVid-1M数据集进行训练，并在UDM10、REDS30和VideoLQ等数据集上进行测试。
定量评估：通过计算不同方法的PSNR、SSIM等指标，STAR在大多数情况下都取得了最佳性能。
定性评估：通过视觉对比展示STAR在恢复视频细节和时间一致性方面的优势，结果显示STAR生成的图像在视觉上更为自然和清晰。
在这个研究中，科学家们开发了一种新方法，叫做STAR，来帮助我们将低质量的视频变得更加清晰和流畅。这个方法的核心在于两个创新的部分：一个是局部信息增强模块（LIEM），另一个是动态频率损失（DFLoss）。LIEM就像是一个放大镜，帮助模型更好地看到视频中的细节，从而减少模糊和失真。而DFLoss则像是一个聪明的助手，指导模型在不同的处理阶段该关注哪些细节，比如在视频的早期阶段关注大的结构，而在后期则关注细小的纹理和边缘。通过这些聪明的设计，STAR在各种测试中都表现得非常出色，能够生成更加真实和自然的视频效果。
2025-01-04｜InstituteofScienceTokyo｜⭐️
http ://arxiv.org/abs/2501.02269v1
本文提出了一种新颖的时序一致性扩散模型（TDM），旨在实现多种真实世界视频恢复任务的统一解决方案。传统的视频恢复方法往往依赖于特定的模型来处理不同类型的降质问题，导致在实际应用中效率低下且不够灵活。TDM利用预训练的StableDiffusion模型和微调的ControlNet，通过单一模型处理多种视频降质类型，如去雾、去雨、去噪声和超分辨率等。该方法的主要创新在于其高效的训练策略和推理策略，确保了恢复视频的内容保持和时序一致性。通过在五种视频恢复任务上的广泛实验，TDM展现出优越的泛化能力和时序一致性，超越了现有的最先进方法，并为视频恢复任务设定了新的标准。
TDM的核心方法包括两个主要部分：训练和推理。
训练阶段：
任务提示指导（TPG）：通过使用与任务相关的文本提示（如“去噪声”或“去雨”），微调ControlNet，以便在多种恢复任务中提供指导。这种方法利用了StableDiffusion模型的零样本分类能力，允许模型在没有额外计算的情况下处理多任务。
单图像输入：在训练过程中，TDM仅需单图像输入，这样可以避免在训练时对多个视频帧的高内存需求，使得模型能够在单个GPU上高效训练。
推理阶段：
滑动窗口跨帧注意力（SW-CFA）：为了解决视频帧之间的时序一致性问题，TDM引入了SW-CFA机制，该机制通过在当前帧周围的滑动窗口中平均键和值，增强了对快速变化的平滑处理，从而提高了时序一致性。
DDIM反演：结合DDIM反演的方法，确保输入噪声在相邻帧之间保持一致，从而提升了内容的保留和时序的一致性。
在实验部分，研究团队对TDM进行了广泛的评估，涉及五种视频恢复任务，包括去雾、去雨、去噪声、MP4压缩伪影去除和超分辨率。使用了多个代表性数据集进行训练和测试，以确保模型的泛化能力。实验结果显示，TDM在图像质量和时序一致性方面均优于现有的最先进方法。具体而言，TDM在FID和KID等指标上表现出色，且在恢复质量方面，能够在不同的降质条件下保持一致性。此外，TDM还通过消融实验验证了TPG、SW-CFA和DDIM反演对模型性能的贡献。
在我们的研究中，我们提出了一种新方法，叫做TDM，旨在提升视频的质量，特别是在视频受到各种干扰（如雨、雾、噪声等）时。我们的模型就像一个多功能的工具箱，只需一个模型就能解决多种问题。我们通过给模型提供简单的指令（比如“去掉噪声”），让它知道我们想要做什么，这样模型就能在训练时学习如何处理这些任务。训练时，我们只用单张图片，这样可以节省计算资源。
在使用模型时，我们引入了一种新方法，叫做滑动窗口跨帧注意力（SW-CFA）。这就像在看视频时，我们不仅关注当前的画面，还会参考前后的画面，这样可以更好地保持视频的连贯性。我们还使用了一种叫做DDIM反演的方法，确保在处理视频时，保持画面的细节不被破坏。通过这些方法，我们的模型在各种视频恢复任务中表现得非常出色，能够有效地提升视频的质量。
2025-01-06｜ZJU,UCAS,HUST｜🟡
http ://arxiv.org/abs/2501.02706v1
随着扩散模型的快速发展，AI生成视频的内容和质量呈现出显著的变化，这给传统的视频质量评估（VQA）框架带来了新的挑战。尽管AI生成视频的流行程度不断上升，但现有的评估方法大多集中在用户生成内容（UGC）上，针对AI生成视频的质量评估方法仍显不足。为了解决这一问题，本文提出了一种多层次语义感知模型（MSA-VQA），旨在通过引入CLIP基础的语义监督和交叉注意机制，对AI生成视频进行质量评估。该模型通过对视频内容进行逐帧、分段和整体的视频分析，确保了评估的准确性和全面性。实验结果表明，MSA-VQA在多个指标上均达到了当前最先进的性能。
MSA-VQA框架专为AI生成视频的质量评估而设计，采用多维度分析策略，分为以下几个关键部分：
数据增强：
帧级增强：对每个视频帧进行独立的质量评估，增强模型对细节的敏感性。
段级增强：将视频分为多个段进行分析，提升模型的鲁棒性和准确性。
提示语义监督模块（PSS）：
利用CLIP的文本编码器处理条件提示，确保生成视频与提示之间的语义一致性。
语义突变感知模块（SMA）：
通过CLIP的图像编码器捕捉视频帧之间的语义变化，利用交叉注意机制评估帧间的细微变化。
多层模型集成策略：
在帧、段和视频三个层面独立训练模型，采用特定的损失函数优化各自的性能，最终通过加权求和的方式整合各层的输出。
本研究使用T2VQA-DB数据集进行实验，数据集包含10,000个AI生成的视频，并将其分为训练集和验证集。通过对比现有的多种基线模型，MSA-VQA在多个评估指标上表现出显著的优势，特别是在平均分数（AvgScore）和相关性系数（SRCC、PLCC）上，均达到了当前最先进的水平。此外，通过消融实验验证了PSS和SMA模块的有效性，结果显示这两个模块的引入显著提升了模型的整体性能。
在这项研究中，MSA-VQA模型采用了一种分层的方法来评估AI生成视频的质量。首先，模型通过对视频的每一帧进行独立分析，确保能够捕捉到每一个细节。接着，模型将视频分成多个段进行处理，这样可以更好地理解视频的整体结构和流畅性。为了确保生成的视频内容与我们给出的提示信息一致，模型还引入了一个叫做提示语义监督模块的部分，它会检查视频内容是否符合提示的要求。此外，模型还设计了一个语义突变感知模块，能够检测视频帧之间的细微变化，这样就能更准确地评估视频的质量。最后，模型将这些不同层次的分析结果结合起来，形成一个全面的视频质量评估。这样的方法使得MSA-VQA能够在评估AI生成视频时达到更高的准确性和可靠性。
2025-01-05｜ZJU,SJTU｜AAAI2025
http ://arxiv.org/abs/2501.02427v1https ://github.com/jialong2023/MetaNeRV
MetaNeRV（MetaNeuralRepresentationsforVideos）是一种新颖的隐式神经表示（INR）框架，旨在提高视频分析的效率。传统的NeRV方法在处理大量视频时效率不高，因为每个视频都需要单独训练一个模型。MetaNeRV通过元学习框架优化参数初始化，使得在适应新视频时可以快速收敛。此外，MetaNeRV还引入了空间-时间引导机制，增强了视频表示能力。通过多分辨率损失和渐进学习策略，该框架能够有效捕捉不同分辨率下的信息，并逐步精细化拟合过程。大量实验结果表明，MetaNeRV在视频表示和压缩任务中表现优越，显著提高了训练效率和效果。
MetaNeRV的核心方法包括以下几个方面：
元学习框架：采用MAML（Model-AgnosticMeta-Learning）算法，从多个视频中学习到一个优化的初始参数配置。这一配置为后续新视频的快速适应提供了良好的起点。
空间引导：引入多分辨率损失，允许模型在不同分辨率阶段获得监督信号。这种方法有效地提高了模型的拟合精度和收敛速度。
时间引导：采用渐进训练策略，初始阶段只学习单帧，逐步增加任务难度。这种方式帮助模型在处理复杂视频时更有效地学习参数，提升训练效率。
在实验部分，MetaNeRV在多个视频数据集上进行了广泛的定量和定性评估，包括UCF101、HMDB-51等。实验结果显示，MetaNeRV在视频表示任务上显著优于其他NeRV基础方法，尤其是在少量迭代下，模型的PSNR和MS-SSIM指标均表现出色。此外，MetaNeRV还在视频压缩和去噪任务中展现了强大的能力，能够在保持高质量重建的同时，显著减少训练和推理时间。通过与传统视频压缩算法的比较，MetaNeRV在多个BPP（每像素比特数）下均表现出更优的压缩性能。
MetaNeRV的工作原理可以简单理解为一个快速学习的系统。首先，它通过观察许多视频，学习到了一种“初始设置”，就像教会学生一个基础知识。这个设置可以帮助模型在看到新视频时迅速适应，而不需要从头开始学习。接下来，MetaNeRV还引入了两种帮助学习的技巧：空间引导和时间引导。空间引导就像在不同的课堂上同时学习不同难度的内容，而时间引导则是让学生从简单的任务开始，逐步增加难度。这样的方法使得MetaNeRV能够更快、更好地处理视频，为我们提供高质量的视频分析和压缩效果。
2025-01-06｜UAdelaide,SonyAI,SonyGroupCorporation｜⭐️
http ://arxiv.org/abs/2501.02786v1
Binauralaudiogeneration(BAG)是一种将单声道音频转换为立体声的技术，通常依赖于视觉提示以捕捉空间和语义信息。然而，现有模型在训练环境中容易过拟合，且无法保留细致的空间细节。为了解决这些问题，本文提出了一种新的音频-视觉双向生成模型，称为CCStereo。该模型引入了一种音频-视觉条件归一化层，能够动态调整目标差异音频特征的均值和方差，并通过新的对比学习方法来增强空间敏感性。CCStereo通过挖掘来自随机打乱视觉特征的负样本，提升了模型对空间信息的理解。此外，模型还采用了一种高效的测试时增强策略，以优化性能。实验结果表明，CCStereo在FAIR-Play和MUSIC-Stereo基准测试中达到了最先进的生成准确性。
CCStereo模型的核心方法包括以下几个方面：
音频-视觉条件归一化层：该层利用视觉上下文动态调整目标差异音频特征的均值和方差，从而提高生成过程的精确度。
对比学习方法：通过从随机打乱的视觉特征中挖掘负样本，增强模型的空间敏感性。这种方法帮助模型更好地学习音频和视觉之间的关系。
测试时增强（V-TTA）：在推理阶段，模型利用视频数据的冗余信息进行增强。具体来说，将视频分为多个连续帧集，并对每个帧集应用五种裁剪增强策略，从而提高模型的鲁棒性和预测准确性。
U-Net架构：模型采用U-Net结构，结合多头注意力机制，以便有效地融合音频和视觉特征。
在实验部分，CCStereo模型在两个主要数据集上进行了评估：FAIR-Play和MUSIC-Stereo。FAIR-Play数据集包含1871个10秒的视频片段，使用专业的双耳麦克风录制，确保了高质量的双耳音频。实验采用了10分割和5分割的评估协议，以测试模型的泛化能力。结果表明，CCStereo在多个指标上超越了现有的最优模型，尤其是在STFT、ENV、Mag和SNR等方面。MUSIC-Stereo数据集的评估进一步验证了CCStereo的可扩展性和泛化能力。通过对模型各个组成部分的消融研究，分析了不同损失函数对模型性能的影响，结果显示对比学习和条件归一化层的结合显著提升了生成音频的质量。
在CCStereo模型中，我们使用了一种聪明的方法来让音频和视频一起工作，以生成更好的立体声。首先，我们有一个特别的“归一化层”，它像一个调音师，根据视频中的视觉信息来调整音频的特性，让生成的声音更真实。接下来，我们还引入了“对比学习”，这就像是在比赛中找出哪些声音是相似的，哪些是不一样的，从而帮助模型更好地理解空间感。最后，在推理时，我们使用了一种称为“测试时增强”的技巧，利用视频中的冗余信息来提高模型的表现。通过这些方法，CCStereo能够更准确地生成立体声，让听者感受到更丰富的音效体验。

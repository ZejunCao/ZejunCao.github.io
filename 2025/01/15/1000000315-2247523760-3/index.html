2025-01-06｜HuaweiNoah’sArk-Paris,GustaveEiffelU｜⭐️🟡
http ://arxiv.org/abs/2501.02913v1
本文提出了一种名为PointmapDiffusion的新框架，用于单图像的新视角合成（NVS）。该方法首次利用点图（即栅格化的3D场景坐标）作为条件信号，从参考图像中捕捉几何先验，以引导扩散过程。通过嵌入参考注意力模块和控制网络（ControlNet），该模型在生成能力和几何一致性之间取得了平衡，使得在不同视点下的视图合成更加准确。大量实验表明，PointmapDiffusion在多种真实世界数据集上表现出高质量的多视图合成，并且相比于其他基线方法，所需的可训练参数显著减少。
PointmapDiffusion的核心方法包括以下几个关键步骤：
点图生成：从参考图像及其深度图生成点图，表示场景中每个像素的3D坐标。
双流架构：采用双流U-Net架构，其中一个U-Net处理输入视图，另一个U-Net生成目标视图。两者通过点图特征进行连接。
控制网络：引入PointmapControlNet以增强扩散模型的3D意识，通过提取几何特征来建立参考视图和目标视图之间的空间对应关系。
参考指导的跨视图注意力：在目标U-Net中引入参考视图注意力机制，确保生成过程中的几何一致性和语义对应。
训练策略：使用预训练的扩散模型进行微调，确保模型在新的视图合成任务中保持高效和一致性。
在实验部分，PointmapDiffusion在多个数据集上进行了评估，包括室内的RealEstate10K、ScanNet++和户外的KITTI-360。通过与现有的最先进方法进行对比，实验结果显示，PointmapDiffusion在生成质量和一致性上均优于其他方法。具体来说，模型在短期和长期视图合成任务中都表现出色，量化指标如PSNR、SSIM、FID和KID均显示出显著的提升。此外，模型在处理不同场景时表现出良好的适应性，证明了其在实际应用中的潜力。
PointmapDiffusion的工作原理可以简单理解为一个高科技的图像处理工具，它能从一张图片中生成新的视角图像。首先，它会从参考图像中提取3D信息（就像给图像加上一个深度地图），这帮助它理解场景的结构。然后，它使用一个双管道的神经网络：一个负责分析原始图像，另一个则负责生成新的视角图像。在这个过程中，模型会通过一个特别的控制网络来确保生成的图像在几何形状和视觉效果上与原图保持一致。最后，模型经过训练后，可以在不同的场景中表现出色，生成的图像质量高且一致性好，适合用于虚拟现实等领域。
2025-01-05｜SYSU(SZ),NUDT｜⭐️🟡
http ://arxiv.org/abs/2501.02519v1
Layout2Scene是一种基于3D语义布局的文本到场景生成模型，旨在创建高保真度的3D场景几何和外观，同时遵循用户提供的对象排列约束。该方法利用3D语义布局（即一组语义边界框）和文本提示（如“一个客厅”）作为输入，通过将对象位置的精确控制引入生成过程，显著提高了生成场景的合理性和真实感。Layout2Scene的训练时间为1.5小时，且在NVIDIAV100GPU上以30帧每秒的速度进行渲染，显示出其高效性和实用性。
Layout2Scene的方法分为几个关键部分：
场景混合表示：该方法将3D场景表示为混合形式，利用高斯分布表示对象，而背景则用多边形表示。这种方式能够有效地区分对象和背景，便于后续处理。
几何优化：首先通过语义引导的几何扩散模型对初始化的场景几何进行细化。该模型使用2D扩散模型的先验知识，以确保生成的几何符合提供的3D布局约束。
外观生成：在几何优化完成后，使用语义-几何引导的扩散模型生成场景的外观。这一过程结合了多个条件（如语义、法线和深度图），以提高生成结果的多样性和真实感。
在实验中，Layout2Scene在20个不同场景布局上进行评估，涵盖了多种文本提示类型。实验结果表明，该方法在生成的场景结构上显著优于现有的基于提示和布局引导的方法。通过与DreamFusion、ProlificDreamer和Text2Room等方法的比较，Layout2Scene在生成的RGB图像、法线图和深度图的质量上均表现出更高的真实感和一致性。此外，Layout2Scene在训练和渲染效率上也展现了其优势，训练时间仅需1.5小时，渲染速度达到30FPS。
在Layout2Scene的方法中，首先我们将场景中的物体和背景分开处理。想象一下我们在画一幅画，画中的每个物体（比如沙发、桌子）都用一个小的“气球”表示，而背景（像墙壁和地板）则用更简单的形状（如矩形）来表示。接下来，我们通过一种叫做“几何扩散”的技术来细化这些气球的形状，确保它们看起来更真实并且符合我们想要的布局。最后，我们使用另一种技术来为这些物体上色，确保它们的颜色和纹理看起来也很自然。这一系列步骤确保了我们最终得到的3D场景既美观又符合我们最初的设计意图。
2025-01-06｜SJTU,Imperial(ICL)
http ://arxiv.org/abs/2501.02973v1https ://hawor-project.github.io/
本研究提出了一种名为HaWoR的高保真3D手部运动重建方法，旨在从自我中心的视频中恢复手部运动。传统的3D手势估计方法通常集中在相机视角下的单图像重建，忽视了手部在世界坐标系中的运动。这种局限性在自我中心的视频场景中尤为明显，因为手部和相机都在不断移动，导致手部运动的准确估计变得复杂。HaWoR通过将手部运动估计与相机轨迹估计解耦，结合自适应的自我中心SLAM框架，克服了传统方法的不足。实验结果表明，HaWoR在手部运动重建和相机姿态估计方面均达到了最先进的性能，尤其是在手部处于视野之外的困难情况下。
HaWoR的核心方法分为三个主要模块：
手部运动估计网络（M）：该网络从输入的视频中提取每一帧的手部特征，并利用基于视觉变换器的模型进行3D手部姿态重建。为了解决手部被截断的问题，网络引入了时间图像注意模块（IAM），通过时间自注意机制增强特征的时序一致性。
相机轨迹估计模块：采用自适应自我中心SLAM方法，该模块能够在动态场景中准确预测相机的运动。通过将手部区域从SLAM重建中排除，确保相机轨迹的估计不受手部运动的干扰，从而提高了重建的准确性。
运动填充网络（F）：该网络专门设计用于预测缺失帧的手部姿态，以完成完整的3D手部运动序列。通过对输入序列进行规范化和转换，运动填充网络能够有效地恢复被遮挡或超出视野的手部运动。
在实验部分，研究者使用了多个数据集来评估HaWoR的性能。首先，使用DexYCB数据集评估相机框架下的3D手部运动重建，结果显示HaWoR在不同遮挡率下均表现出色，相比于其他最先进的方法，HaWoR在严重遮挡情况下的性能提升显著。其次，在HOT3D数据集上进行世界框架下的手部运动重建评估，HaWoR在手部运动重建方面显著优于基线方法，验证了其在复杂动态场景中的有效性和鲁棒性。此外，通过消融实验，研究者分析了各个模块对最终性能的贡献，进一步证明了IAM和PAM等模块在提升重建质量方面的重要性。
HaWoR的方法可以简单理解为三个主要步骤。首先，手部运动估计网络负责从视频中识别手部并重建其运动轨迹。为了更好地处理手部被遮挡的情况，这个网络会考虑到手部在视频中前后帧的变化。其次，相机轨迹估计模块帮助确定录制视频时相机的位置和运动，这个步骤非常重要，因为相机的运动会影响手部运动的判断。最后，运动填充网络则专门用来填补那些因为手部被遮挡而无法看到的帧，确保我们可以得到完整的手部运动序列。这种方法组合在一起，使得HaWoR能够在复杂的动态环境中准确地重建手部的3D运动。
2025-01-06｜CAS,UCAS,CASIA,Google｜AAAI2025
http ://arxiv.org/abs/2501.02845v1https ://iscas3dv.github.io/HOGSA/
在机器人技术和虚拟现实领域，理解双手与物体的交互至关重要。然而，由于手与物体之间存在显著的遮挡以及高自由度的运动，收集和标注高质量的大规模数据集变得十分困难，这限制了双手-物体交互相关基线的进一步改进。为了解决这一问题，本文提出了一种基于3D高斯点云的增强数据框架（HOGSA），该框架能够将现有数据集扩展为大规模的、具有多种手-物体姿态和视角的逼真数据。通过建立网格基础的3DGaussianSplatting（3DGS）模型，结合姿态优化模块和超分辨率模块，HOGSA能够有效地生成高质量的合成数据，从而提升双手-物体交互理解的准确性。
HOGSA数据增强方法主要由以下几个模块组成：
3D高斯点云建模：基于输入的手-物体交互图像，建立网格基础的3D高斯点云模型，以有效合成手-物体交互图像。
姿态优化模块（POM）：通过优化原始姿态，生成多样化的双手-物体交互姿态，扩展数据集中的姿态分布。
超分辨率模块（SRM）：针对由3DGS生成的低质量图像，使用卷积神经网络（CNN）提升图像的渲染质量，确保生成图像的真实感。
数据集扩展：将生成的增强数据与原始数据结合，进行模型的微调，以提高双手-物体交互的基线性能。
该框架的设计旨在通过高效的图像渲染、姿态多样性和逼真的图像合成，提升双手-物体交互理解的效果。
我们在两个主要基准数据集H2O和Arctic上评估了HOGSA方法的性能。通过与现有的基线方法进行比较，HOGSA显著提高了双手-物体交互理解的准确性。实验结果表明，经过数据增强处理的数据集在交互姿态估计和接触关系预测方面表现出更好的性能。具体来说，HOGSA方法在减少接触偏差、运动偏差和加速度误差等指标上均取得了显著改善。此外，我们还进行了消融实验，以分析各个模块对模型性能的贡献，结果显示超分辨率模块和姿态优化模块在提升生成图像质量和姿态多样性方面发挥了关键作用。
HOGSA方法的核心在于通过几个步骤来增强双手与物体的交互数据。首先，我们使用一种叫做3D高斯点云的方法来创建手和物体的3D模型，这样我们就能在不同的姿势和角度下生成交互图像。接着，我们引入一个姿态优化模块，这个模块可以根据原始的数据来生成多种不同的手势，这样会让我们的数据集更加丰富。最后，我们使用超分辨率模块来提升生成图像的清晰度和真实感，确保合成的图像看起来更自然。通过这些步骤，我们不仅扩展了数据集，还提升了模型在理解双手与物体交互时的准确性。
2025-01-04｜UCLA
http ://arxiv.org/abs/2501.02158v1https ://genforce.github.io/JOSH/
在本研究中，我们提出了一种名为JOSH（JointOptimizationofSceneGeometryandHumanMotion）的新方法，旨在实现自然环境中的4D人类场景重建。该方法通过联合优化全局人类运动和场景几何，解决了在单目视频中重建人类与环境互动的挑战。传统方法往往在受限环境中表现良好，但无法有效处理来自网络视频的多样化和自然的人类运动及场景上下文。JOSH利用密集场景重建和人类网格恢复技术作为初始化，随后通过人类与场景的接触约束进行联合优化，显著提高了人类运动估计和场景重建的准确性。
JOSH的方法可以分为以下几个步骤：
初始化：利用现有的密集场景重建和人类网格恢复技术，初步获取场景几何和人类姿态的估计。这一阶段的目标是为后续的优化提供可靠的初始参数。
联合优化：在初始化的基础上，通过接触损失约束，进行全局人类运动和场景几何的联合优化。具体来说，接触损失确保了人类与环境的接触点在空间和像素坐标中保持一致，从而提高了运动和场景重建的精度。
人类-场景接触损失：在优化过程中，基于人类与场景的接触信息，制定相应的损失函数。这些损失函数帮助调整人类运动和场景几何，使得重建结果更加真实和一致。
高效模型设计：为了解决处理大规模视频数据的效率问题，我们设计了一个轻量级的模型JOSH3R，能够实现实时推断。
在实验部分，我们在多个数据集上评估了JOSH的性能，包括EMDB、SLOPER4D和RICH数据集。我们分别测量了全局人类运动估计和密集场景重建的准确性。结果显示，JOSH在全局人类运动估计上超越了现有的最先进方法，并在场景重建中也表现出色。此外，我们还进行了消融实验，以评估各个组件对整体性能的贡献。实验结果表明，联合优化策略显著提高了重建的准确性和一致性。
在这个研究中，我们开发了一种新方法，叫做JOSH，它能够从普通视频中重建人类的运动和周围环境。这个过程可以想象成一个拼图游戏，我们首先用一些现有的工具来拼出一个初步的图案，然后通过一些规则来优化这个图案，使得人类的动作和环境的形状更加协调。具体来说，我们会先找出人类和环境接触的地方，比如脚与地面的接触点，然后确保这些点在不同的画面中保持一致。这样做的目的是让重建出来的场景看起来更自然、更真实。为了让这个方法在处理大量视频时也能快速有效，我们还设计了一个轻便的版本，能够实时处理数据。

2025-01-06｜HKUST(GZ),HKUST,AdobeResearch｜⭐️⭐️
http ://arxiv.org/abs/2501.03006v1https ://wileewang.github.io/TransPixar/
本文介绍了TransPixar，一种先进的文本到视频生成方法，专注于生成带有透明度的RGBA视频。传统的文本到视频生成模型在生成RGB视频方面取得了显著进展，但在生成RGBA视频时面临许多挑战，主要是由于缺乏足够的数据集和现有模型的适应性问题。RGBA视频的生成至关重要，因为它允许透明元素（如烟雾和反射）无缝地融入场景中，从而提升视觉效果。TransPixar通过引入LoRA层和新颖的透明度通道自适应注意力机制，成功地在生成RGB和alpha通道的同时保持了视频的高质量和一致性。
TransPixar的核心方法分为以下几个步骤：
模型架构：基于扩散变换器（DiT）架构，TransPixar在预训练的视频生成模型上进行扩展，添加了alpha特定的标记，以便同时生成RGB和alpha通道。
注意力机制优化：通过改进注意力机制，TransPixar能够在生成过程中实现RGB和alpha通道之间的强对齐。这包括：
文本-到-RGB的注意力：确保文本信息能够有效影响RGB生成。
RGB-到-alpha的注意力：允许RGB信息为alpha通道的生成提供反馈，从而提高生成的质量。
移除不必要的文本-到-alpha的注意力：避免干扰，保留原始模型的生成能力。
训练与推理：使用FlowMatching或传统的扩散过程进行训练，以提高模型的生成能力和一致性。
在实验部分，研究团队使用了VideoMatte240K数据集进行训练，该数据集包含丰富的前景和alpha通道信息。实验表明，TransPixar在生成RGBA视频时表现出色，能够生成多种运动模式的动态对象，并在透明效果方面表现突出。通过与现有的生成-再预测管道进行比较，TransPixar在RGB和alpha通道的对齐度和运动质量上均显著优于其他方法。此外，用户研究结果显示，参与者更倾向于选择TransPixar生成的视频，表明其在视觉效果和运动一致性方面的优势。
TransPixar的工作原理可以简单理解为一个聪明的“视频制作助手”。首先，它使用一种叫做扩散变换器的技术，这种技术可以帮助它理解和生成视频。为了让视频看起来更真实，TransPixar不仅生成常规的颜色通道（RGB），还额外生成一个透明度通道（alpha），这就像给视频添加了“隐形墨水”，使得某些部分可以是透明的，比如水或烟雾。接下来，它通过特别的注意力机制，确保生成的颜色和透明度之间能够很好地配合，避免出现不协调的情况。最后，TransPixar通过大量的训练，学会了如何在不同的场景中生成动态效果，比如动物奔跑或烟花绽放，从而使视频更具吸引力。
2025-01-06｜MetaGenAI,MetaFAIR,HUJI｜⭐️🟡
http ://arxiv.org/abs/2501.03059v1https ://guyyariv.github.io/TTM/
在本研究中，提出了一种名为“Through-The-Mask”的新颖图像到视频生成（I2V）方法，旨在将静态图像转换为基于文本描述的动态视频。尽管现有技术在生成高质量图像方面取得了显著进展，但在处理多对象场景时，生成一致且真实的运动仍然面临挑战。为了解决这些问题，本文提出了一种两阶段的生成框架：第一阶段生成显式的中间表示，第二阶段则基于该表示生成最终视频。关键创新在于引入了基于掩模的运动轨迹作为中间表示，这种表示不仅捕捉了对象的语义信息，还有效地描绘了对象的运动，从而在多对象场景中实现更准确的动画效果。
本研究的方法分为两个主要阶段：
图像到运动生成：在第一阶段，模型根据输入图像、分割掩模和运动提示生成掩模运动轨迹。这些轨迹代表了各个对象的动态行为，确保在生成过程中能够准确反映对象的运动和互动。
运动到视频生成：在第二阶段，利用生成的运动轨迹、对象特定提示和输入图像，生成最终的视频。该阶段通过引入学习的注意力层，确保生成的视频与先前阶段的语义和运动保持一致。
这种两阶段的过程使得模型能够在不同层次上处理对象的运动和语义信息，从而提高了生成视频的质量和一致性。
为评估所提出方法的有效性，研究者在多个基准数据集上进行了实验，包括SA-V-128基准和Image-Animation-Bench。通过对比不同模型的表现，结果表明“Through-The-Mask”在时间一致性、运动真实感和文本提示的忠实度等多个指标上均表现出色。特别是在多对象设置下，该方法显著优于现有的图像到视频生成模型。此外，研究还引入了人类评分，以评估生成视频的质量和运动一致性，进一步验证了所提方法在真实应用场景中的优越性。
在这个研究中，研究者们开发了一种新方法，可以将静态图片转变成动态视频，且这个过程是基于输入的文本描述来进行的。这个方法分为两个步骤。第一步是根据给定的图片和一些说明，生成一个“运动轨迹”，这个轨迹就像是给每个物体画出的一条运动路线，帮助模型理解物体应该如何移动。第二步则是根据这个运动轨迹和图片，生成最终的视频。在这个过程中，模型会特别关注每个物体的运动和它们之间的互动，确保生成的视频看起来真实且连贯。通过这种方式，研究者们能够更好地处理多个物体一起运动的复杂场景，生成更自然的视频效果。
2025-01-05｜CUHKMMLab,AvolutionAI｜⭐️🟡
http ://arxiv.org/abs/2501.02690v1https ://wkbian.github.io/Projects/GS-DiT/
在视频生成领域，GS-DiT提出了一种新颖的框架，通过引入伪4D高斯场和高效的稠密3D点跟踪技术，显著提升了视频生成的能力。该方法的核心在于能够控制视频中的4D内容，包括摄像机姿态和物体运动编辑，克服了现有方法无法支持复杂镜头技术的局限性。GS-DiT通过构建伪4D高斯场并为所有视频帧渲染该场，从而生成多摄像机拍摄的视频。在训练阶段，GS-DiT对预训练的DiffusionTransformer（DiT）进行微调，以生成符合渲染视频的内容。此外，GS-DiT还展示了强大的泛化能力，能够在不同的摄像机参数下生成具有相同动态内容的视频，极大地丰富了创意视频制作的可能性。
GS-DiT的实施方法包括几个关键步骤：
稠密3D点跟踪（D3D-PT）：该方法首先从输入视频中提取参考帧，并估算其在后续帧中的稠密3D点轨迹。D3D-PT的设计旨在提高跟踪精度，并加速推理速度，表现优于现有的稀疏3D点跟踪方法。
伪4D高斯场构建：通过对稠密3D点轨迹的分析，GS-DiT构建伪4D高斯场。该场的高斯原语具有固定的颜色值，并通过输入视频的像素值生成，其他参数如不透明度保持不变。
视频生成微调：在生成阶段，GS-DiT利用渲染的伪4D高斯场作为指导，微调预训练的DiT模型，最终生成符合条件的视频。该方法通过调整摄像机内外参数，实现了对视频生成的4D控制，包括多摄像机拍摄和镜头语言编辑。
在实验部分，GS-DiT的表现通过多个基准数据集进行评估，主要集中在稠密3D点跟踪和4D视频控制的有效性上。实验结果显示，D3D-PT在稠密3D点跟踪任务中表现优异，显著提升了跟踪精度并加快了处理速度。此外，GS-DiT在多摄像机视频生成和4D控制方面也展示了强大的能力。通过与其他方法如GCD和MonST3R的比较，GS-DiT在视频质量和摄像机控制的表现上均有显著优势。实验还展示了GS-DiT在不同场景下的泛化能力，能够有效处理来自真实世界的动态视频。
在GS-DiT的方法中，首先要进行的是一种叫做“稠密3D点跟踪”的过程。想象一下，我们要从一段视频中追踪每个物体的运动轨迹。我们选择一个视频帧作为参考，然后用一种智能算法来追踪这个帧中每个点在后续帧中的位置。这就像在玩一场追逐游戏，追踪每个点的移动。接下来，我们将这些点的运动信息组合起来，形成一个“伪4D高斯场”，这个场就像是一个动态的画布，能够根据不同的摄像机角度和运动生成视频。最后，我们利用这个伪4D高斯场来指导我们的生成模型，微调它，使得生成的视频不仅看起来真实，还能根据不同的摄像机设置进行调整。这样，我们就能创造出多视角的视频，甚至可以在视频中编辑物体的运动，给创作者带来了更多的创作自由。
2025-01-06｜FDU,HuaweiNoah’sArkLab｜ICASSP2025｜⭐️🟡
http ://arxiv.org/abs/2501.02741v1
随着视频生成技术的不断发展，长视频生成已成为一个重要的研究领域，尤其是在内容创作和媒体应用中。然而，现有的基于扩散模型的方法在生成长视频时面临诸多挑战，如计算资源的消耗和数据需求的增加，导致大多数模型只能生成较短的视频。本文提出了一种名为Brick-Diffusion的新方法，旨在通过一种无训练的砖墙去噪策略来生成任意长度的长视频。该方法通过将潜在空间分段去噪，模拟砖墙的构建过程，从而在不同视频帧之间实现信息传递，最终提高视频的质量和一致性。通过定量和定性实验，Brick-Diffusion在高保真视频生成方面优于现有的基线方法。
Brick-Diffusion的核心在于砖墙去噪技术，该技术包括以下两个主要步骤：
潜在切片：将长视频的潜在表示切分成多个短段落，每段的长度与预训练扩散模型的输入大小相匹配。在每个去噪步骤中，利用一个非零的步幅对潜在表示进行偏移和重新切片，以确保不同段落之间的信息交流。
逐段去噪：对每个切片进行独立的去噪处理，采用扩散模型进行去噪。通过这种方式，随着去噪的进行，视频帧之间的交流逐渐增强，使得最终生成的视频在内容和风格上更加一致。
这种方法的设计允许在不需要额外训练的情况下，充分利用预训练模型的能力，同时通过并行化处理提高效率。
在实验中，Brick-Diffusion与多种基线方法进行比较，包括直接拼接、Gen-L-Video、FreeNoise和FIFO-Diffusion等。我们使用VBench作为评估基准，采用了多个指标进行量化评估，包括主题一致性、动态程度、美学质量和视频文本一致性。实验结果显示，Brick-Diffusion在动态程度、美学质量和整体视频文本一致性方面均取得了最高分数，表明其在生成高保真长视频方面的有效性。同时，我们还进行了消融实验，评估不同步幅对结果的影响，发现步幅为1时，效果最佳。
Brick-Diffusion的工作原理可以想象成建造一堵砖墙。首先，我们把长视频的内容切分成小段，就像将砖块一个个放在墙上。每一小段都独立处理，确保每一块砖（视频片段）都经过精心打磨（去噪）。在处理每一段时，我们会稍微移动一下位置，这样在下一个处理阶段时，前一段的信息可以传递给后一段，就像砖块之间有缝隙，可以相互连接。通过这种方式，最终生成的视频就会更加流畅、一致，避免了不同片段之间的突兀感。这个方法不仅高效，而且不需要重新训练模型，充分利用了已有的技术。

2025-01-06｜CMU,Meta｜⭐️🟡
http ://arxiv.org/abs/2501.03120v1
在本研究中，我们提出了一种名为内容自适应图像标记器（Content-AdaptiveTokenizer,CAT）的新方法。CAT旨在根据图像的内容复杂性动态调整图像的表示能力，从而提高图像压缩的质量和效率。现有的大多数图像标记器采用固定的压缩比，无法有效处理不同图像的复杂性，可能导致重要视觉细节的丢失或计算资源的浪费。CAT通过引入基于文本的复杂性评估系统，利用大型语言模型（LLMs）预测图像的内容复杂性，并确定最优的压缩比例。经过多种压缩比的训练，CAT在图像重建和生成任务中表现出色，显著提高了FID评分并提升了推理吞吐量。
CAT的实现分为几个关键步骤：
复杂性评估：利用图像的文本描述，CAT首先获取图像的复杂性评分。这一评分基于图像的语义复杂性、视觉复杂性和感知复杂性等因素。
压缩比分类：根据复杂性评分，将图像分类为不同的压缩比例（如8x、16x、32x）。复杂性越高，所需的压缩比越低。
嵌套变分自编码器（NestedVAE）：CAT采用嵌套的VAE架构，能够在单一模型中处理多种压缩比例。通过动态路由输入，生成具有不同空间维度的潜在特征。
训练过程：在训练阶段，CAT对不同复杂度的图像进行压缩，使用多种损失函数（如重建损失、KL散度和感知损失）来优化模型性能。
在实验部分，我们对CAT在图像重建和生成任务中的表现进行了评估。实验使用了多个数据集，包括自然场景（COCO）、人脸（CelebA）和文本重图像（ChartQA）。我们比较了CAT与固定压缩比的基线模型，结果显示CAT在重建质量（如rFID、LPIPS和PSNR）上普遍优于这些基线模型。特别是在复杂图像上，CAT的重建质量提升显著，且在文本重图像任务中表现尤为突出。通过分析不同压缩比的分布，CAT能够有效学习压缩特征，从而在生成任务中实现更高的图像质量和更快的处理速度。
CAT的工作原理可以简单理解为：它根据一幅图像的描述来判断这幅图像的复杂程度。比如，如果图像比较简单，CAT就会选择更高的压缩比，这样可以省下存储空间；而如果图像复杂，CAT则会使用较低的压缩比，以保留更多的细节。整个过程就像是给不同的图像量身定制不同的“包裹”，确保每幅图像都能以最合适的方式被保存和使用。通过这种方法，CAT不仅提高了图像的压缩效率，还确保了图像质量，使得后续的图像生成和处理更加高效。
2025-01-06｜MetaFAIR,UCBerkeley,TTI-Chicago｜⭐️
http ://arxiv.org/abs/2501.03229v1https ://brjathu.github.io/gmae
本研究提出了一种新的自监督图像表示学习方法，称为高斯掩蔽自编码器（GMAE），旨在同时学习语义抽象和空间理解。传统的掩蔽自编码器（MAE）虽然能够有效地学习语义特征，但缺乏对空间结构的明确建模。GMAE通过引入3D高斯表示作为中间层，重建图像的同时捕捉其空间特征。该方法不仅保留了MAE在高层语义表示方面的优势，还增强了图像分层、边缘检测等零样本学习能力。研究表明，GMAE在多个视觉任务中表现良好，为未来高保真视觉数据建模技术的发展提供了新的思路。
GMAE的核心在于结合像素级学习和潜在空间学习。具体方法如下：
自监督掩蔽自编码器：使用掩蔽技术随机遮挡输入图像的一部分，通过重建这些被遮挡的部分来学习图像的整体结构。
3D高斯原语：引入3D高斯作为中间表示，定义每个高斯的颜色、位置、尺度和方向。这些高斯通过体积渲染技术转化为图像，允许模型在训练过程中动态调整它们的分布。
ViT编码器与解码器：使用视觉变换器（ViT）作为编码器，将可见图像块编码为潜在嵌入。解码器则预测高斯参数，并通过可微分的体积渲染生成重建图像。
损失函数：采用均方误差损失来评估重建图像与原始图像之间的差异，从而优化高斯参数的学习。
在实验部分，研究者对GMAE的性能进行了多方面的评估。首先，利用ImageNet数据集进行分类测试，结果表明GMAE在不同高斯数量的情况下表现出色，且随着高斯数量的增加，分类准确率逐渐提高。其次，GMAE在COCO数据集上的目标检测和分割任务中也取得了与MAE相当的结果，显示出其在下游任务中的有效性。此外，研究还探讨了高斯的规模、掩蔽比例和损失函数对模型性能的影响，发现适当的高斯规模和掩蔽比例能够显著提升重建质量和分类性能。
在GMAE方法中，我们首先将图像分成多个小块，并随机遮住一些块。接着，模型会学习如何根据剩下的可见部分来重建被遮住的部分。这就像在拼图游戏中，只能看到部分图案，依靠已有的线索来猜测缺失的部分。为了更好地理解图像的空间结构，我们使用了3D高斯模型，这种模型可以动态调整其大小和位置，帮助我们更准确地表示图像中的不同对象。最后，模型通过一种叫做“体积渲染”的技术，将这些3D高斯转换回2D图像，确保重建的图像尽可能接近原始图像。通过这种方式，GMAE不仅能识别图像中的物体，还能理解它们在空间中的关系。
2025-01-04｜NYCU,NationalTaiwanU,NVIDIA
http ://arxiv.org/abs/2501.02355v1https ://corrfill.github.io/
在参考图像基础上的图像修复任务中，研究者们致力于通过提供额外的参考图像来恢复受损目标图像的原始状态。近年来，扩散模型（如StableDiffusion）的发展为这一任务提供了新的思路。然而，现有的基于扩散的方法往往缺乏对参考图像与受损图像之间相关性的明确约束，这导致修复结果在忠实度上有所欠缺。为了解决这个问题，本文提出了一种名为CorrFill的模块，它旨在增强参考图像与目标图像之间的几何关联意识。通过在修复过程中引入基于对应关系的约束，CorrFill显著提高了图像修复的效果，使得生成的图像更能忠实于参考图像的内容。
CorrFill的核心在于利用图像之间的对应关系来指导修复过程。具体步骤包括：
图像拼接：将参考图像和目标图像拼接为一个输入，以便在扩散模型的自注意力层中实现跨图像的关注。
对应关系估计：在每个去噪步骤中，利用自注意力分数计算目标图像和参考图像之间的对应关系，生成匹配图。
引导修复过程：通过对应关系作为约束，更新输入张量，从而引导后续的去噪步骤，确保生成的图像忠实于参考图像。
循环增强：通过迭代更新对应关系和修复过程，逐步优化修复结果，达到更高的忠实度。
这种方法不需要额外的训练模块，适用于多种基于扩散的图像修复模型，提供了一种通用的改进策略。
在实验部分，研究者使用了RealEstate10K和MegaDepth两个数据集进行评估。通过与四种基线方法的比较，CorrFill在多个评估指标（如PSNR、SSIM和LPIPS）上表现出显著的改进。实验结果表明，CorrFill有效地提升了图像修复的质量，尤其在处理复杂的场景时，其对参考图像的忠实度得到了显著增强。此外，研究者还进行了消融实验，验证了CorrFill中各个关键组件的有效性，进一步确认了引入对应关系的必要性和有效性。
在CorrFill的方法中，我们首先将参考图像和需要修复的目标图像拼接在一起，这样计算机就可以同时看到这两幅图像。接下来，计算机通过一种叫做自注意力的技术来找出这两幅图像之间的相似部分，生成一个“对应关系”的图表，告诉我们在目标图像的哪个位置可以找到与参考图像相关的内容。然后，计算机会利用这些对应关系来指导修复过程，确保生成的图像尽量与参考图像相似。最后，这个过程会重复进行，逐步改善修复效果，直到得出一个更完美的结果。这样的设计使得CorrFill不仅提高了修复的质量，还避免了需要重新训练模型的复杂步骤。
2025-01-04｜UTS(Sydney),BaiduInc,ZJU
http ://arxiv.org/abs/2501.02376v1
本研究提出了一种新的任务，称为“文本引导图像生成模型的来源识别”（ID2），旨在识别由文本引导的图像生成模型生成的图像的原始来源。随着这些模型在创意和商业领域的广泛应用，误用的风险也在增加，例如传播虚假信息、侵犯版权和规避内容追踪。因此，开发一个有效的来源识别方法显得尤为重要。为此，研究者们不仅构建了一个首个专注于此任务的数据集OriPID，还提出了一种理论上保证的线性变换方法，以最小化生成图像的变分自编码器（VAE）嵌入与其原始图像之间的距离。这一方法在多种扩散模型上均表现出良好的泛化能力。
该方法的核心在于实现来源识别的线性变换。具体步骤如下：
存在性证明：首先，研究者们证明了存在一种线性变换矩阵W，可以使生成图像g的VAE嵌入E(g)与原始图像o的VAE嵌入E(o)之间的距离足够接近。
泛化能力：接着，研究者们展示了这种线性变换在不同扩散模型之间的泛化能力，意味着训练时使用的变换可以有效地应用于新的扩散模型，即使它们的VAE参数或架构不同。
训练过程：通过梯度下降优化一个度量学习损失函数，研究者们能够学习到理论上预期的矩阵W。
数据集构建：为支持上述方法，研究者们构建了OriPID数据集，包含大量的原始图像和引导提示，以便进行模型训练和测试。
实验部分评估了所提出方法的有效性和泛化能力。研究者们在OriPID数据集上进行了一系列实验，比较了不同模型的性能。实验结果表明：
所有现有的相似性基础方法在OriPID测试集上均未能达到理想的性能。
提出的线性变换方法在未见数据上显著优于其他方法，提升了31.6%的平均精度（mAP）。
该方法不仅在准确性上表现出色，同时在效率上也优于其他方法，训练和匹配速度大幅提升。
最后，研究者们还进行了消融研究，验证了不同VAE编码器和损失函数对模型性能的影响。
在这项研究中，科学家们开发了一种新的方法来识别那些通过文本描述生成的图像的原始版本。想象一下，你在网上看到了一张经过处理的图片，可能是因为某人想要改变它的内容，或者是为了逃避版权检查。研究者们的工作就是为了找出这张图片最初的样子。为此，他们首先证明了可以通过一种数学方法，将处理后的图片和原始图片的特征联系起来，然后他们训练了一个模型来识别这些特征。为了进行训练，他们创建了一个专门的数据集，里面有很多原始图片和对应的描述。通过这些步骤，他们的方法不仅能准确识别图像的来源，还能快速处理大量数据，这在实际应用中非常重要。
2025-01-04｜CityU,NTU,JDExploreAcademy
http ://arxiv.org/abs/2501.02207v1https ://github.com/MZMMSEC/AIGFD_EXIF.git
随着生成对抗网络（GAN）和扩散模型等技术的快速发展，AI生成的面孔已经变得几乎与真实面孔无法区分。这种情况引发了对其潜在滥用的担忧，因此开发有效的检测方法至关重要。传统的检测方法通常将这一任务视为二分类问题，但由于AI面孔生成器的迅速演变，这些方法往往难以适应新型生成器。为了解决这一问题，本文提出了一种基于自监督学习的异常检测方法，旨在通过学习相机内在特征和面部特征来检测AI生成的面孔。该方法通过设计一个预训练任务，训练特征提取器来对面孔图像进行排序和分类，进而识别出AI生成的面孔。
本研究的核心在于自监督表示学习和高斯混合模型（GMM）训练。具体方法如下：
自监督表示学习：
设计一个图像特征提取器(f(·;θ))，将输入图像转换为视觉嵌入。
利用五个预测头(g(·;ϕ))来计算与四个EXIF标签排序相关的任务和一个面部操控分类任务的“logits”。
对于EXIF标签排序，构建二元标签以比较图像对中每个标签的大小。
GMM训练：
将学习到的特征分布建模为高斯混合模型，通过最大化似然函数来估计模型参数。
在测试阶段，计算测试图像的对数似然，低于阈值的图像被标记为AI生成的面孔。
损失函数优化：
综合考虑四个EXIF标签排序任务和面部操控分类任务的损失，优化特征提取器的学习效果。
在实验中，研究团队使用了200,000张面孔照片，并从九个先进的生成模型中收集了AI生成的面孔图像。通过在CelebA-HQ数据集上进行训练和在预留的5,000张图像上进行测试，评估了所提方法的有效性。实验结果表明，该方法在检测准确率和平均精度方面均优于其他六种代表性检测器。此外，通过不同的高斯成分数的调优，发现八个成分的组合能获得最佳性能，进一步验证了模型的有效性。
我们的方法主要分为两个步骤：首先是自我学习特征，接着是使用高斯模型来判断面孔是否是AI生成的。具体来说，我们设计了一个特征提取器，它像一个智能的过滤器，可以从真实的面孔和AI生成的面孔中提取关键信息。为了让这个过滤器更聪明，我们给它设定了一些任务，比如比较面孔的不同属性（比如光圈、曝光时间等），并通过这些比较来学习。接下来，我们使用高斯混合模型来分析这些特征，判断哪些面孔的特征与真实面孔的特征相差很大，从而识别出AI生成的面孔。通过这样的方式，我们可以更准确地检测出网络上那些伪造的面孔。
2024-12-28｜ByteDance,TikTokInc
http ://arxiv.org/abs/2501.01973v2
随着大型语言模型（LLMs）和大型视觉模型（LVMs）的快速发展，多模态人工智能系统在工业应用中展现了巨大的潜力。然而，这些系统也引发了一系列伦理挑战，包括加剧有害内容的传播和社会偏见的强化。现有的评估框架往往忽视了内容生成的对齐和社会偏见敏感领域，导致其在公平性评估方面存在显著的局限性。为了解决这些问题，本文提出了INFELM，旨在对广泛使用的文本到图像模型进行深入的公平性评估。我们的研究贡献包括：开发了一种先进的肤色分类器，提出了一个偏见敏感的内容对齐测量方法，并进行了大规模的实验分析，以评估文本到图像模型在六个社会偏见敏感领域的表现。研究结果表明，现有模型普遍未能满足公平性标准，且在表现偏见方面的问题更为突出。
INFELM的方法包括多个关键步骤，以确保对文本到图像模型的公平性进行全面评估：
肤色分类器的改进：我们提出了一种新的肤色分类器，结合面部拓扑信息和精细的肤色像素表示，以提高分类精度，至少提升16.04%。
内容对齐测量：通过使用来自六个偏见敏感领域的提示，我们生成图像以测量生成内容的对齐程度，从而理解模型的社会影响。
代表性偏见评估：我们提出了一种通用的代表性偏见评估指标，适用于多种人口统计群体，特别关注性别和肤色群体。
大规模分析：对九个工业文本到图像模型生成的图像进行广泛分析，发现大多数模型在公平性标准下普遍存在偏见，并且肤色对齐误差显著高于性别误差。
在实验部分，我们首先描述了实验设置和使用的数据集。我们进行了大量的模型公平性分析，特别关注肤色分类和代表性偏见。通过对比现有基线模型，如HEIM和VIT，我们发现INFELM在肤色分类任务上表现优越，尤其是在均方误差（MSE）和精确度方面。实验结果显示，尽管大多数模型在性别公平性方面表现较好，但在肤色表现上存在明显的偏见。此外，我们还分析了模型生成内容的对齐误差，发现大多数模型在这一方面的表现不佳，特别是在肤色生成的准确性上。
在我们的研究中，我们开发了一种新的方法来评估文本到图像模型的公平性。首先，我们创建了一种新型肤色分类器，它能更准确地区分不同肤色，这样我们就能更好地分析模型的输出。接下来，我们设计了一种方式来检查模型生成的图像是否与输入的提示内容一致，比如如果我们要求生成一个“医生”的图像，模型是否能生成符合这个描述的图像。我们还通过分析大量生成的图像，发现大多数模型在表现肤色和性别时存在偏见，尤其是在肤色的表现上更为明显。通过这些方法，我们希望能够帮助开发出更加公平和符合社会价值观的人工智能模型。

在单目深度估计领域，传统的方法通常依赖于数据驱动或模型驱动的技术。尽管这些方法在准确性上取得了显著进展，但它们往往面临着推理速度慢的问题。本文提出了一种名为DepthMaster的新方法，旨在通过整合生成特征与判别任务，提升深度估计的效率与效果。DepthMaster基于单步的扩散模型，采用特征对齐模块和傅里叶增强模块，以克服传统方法中的过拟合问题，并在保持细节的同时提高模型的泛化能力。通过这两个模块的协同作用，DepthMaster在多个数据集上实现了最先进的性能，特别是在细节保留方面超越了其他扩散模型方法。
DepthMaster的设计包括几个关键模块，确保其在单步推理中有效捕捉深度信息。具体方法如下：
单步确定性范式：模型首先将RGB图像编码为潜在空间，然后通过去噪网络直接生成深度图。这种方法避免了传统扩散模型中的多步骤迭代，从而提高了推理速度。
特征对齐模块：引入高质量的外部视觉表示，通过对齐扩散模型的特征分布与外部编码器的特征分布，增强去噪网络的特征表示能力。这一过程有助于减少对纹理细节的过拟合。
傅里叶增强模块：该模块在频域内操作，动态平衡低频结构特征与高频细节特征，以改善深度图的视觉质量。通过对不同频带的信息进行自适应调整，模拟多步骤去噪过程的效果。
两阶段训练策略：第一阶段专注于学习全局场景结构，第二阶段则利用傅里叶增强模块优化细节保留。这种策略不仅提升了模型的泛化能力，还提高了细节的清晰度。
在多个数据集上进行的实验表明，DepthMaster在零-shot深度估计任务中表现优异。实验中，模型在KITTI、NYUv2、ETH3D等基准数据集上进行了评估，结果显示其在绝对相对误差（AbsRel）和准确率（δ1）等指标上均优于当前最先进的模型。此外，模型的推理速度大幅提升，验证了其在实际应用中的潜力。通过消融实验，研究还探讨了不同模块的有效性，证明了特征对齐和傅里叶增强对细节保留的积极影响。
DepthMaster的工作原理可以简单理解为一个高效的深度图生成器。首先，它通过一个快速的编码过程将普通的RGB图像转换成潜在的深度信息，而不是像传统方法那样逐步迭代生成。接着，DepthMaster使用一个特征对齐模块，这就像是把图像中的重要信息与外部的高质量信息进行匹配，从而避免只关注不重要的细节。最后，傅里叶增强模块则帮助它在细节和整体结构之间找到平衡，确保生成的深度图既准确又清晰。通过这种方式，DepthMaster不仅加快了推理速度，还在多个测试中表现出色，能够在实际应用中提供更好的深度估计效果。

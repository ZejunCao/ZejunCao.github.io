2025-01-03｜THU,PengChengLab,Lenovo｜⭐️⭐️
http ://arxiv.org/abs/2501.02064v1
近年来，文本引导的风格迁移技术取得了显著进展，主要得益于扩散模型的创新。这些模型通过利用文本或图像来指导采样过程，展现出强大的条件生成能力。然而，现有的直接条件引导方法在平衡文本语义的表现力与输出结果的多样性方面存在挑战。为了解决这些问题，本文提出了一种新颖的框架ArtCrafter，专注于文本到图像的风格迁移。ArtCrafter通过引入基于注意力的风格提取模块、文本-图像对齐增强组件及显式调制技术，极大地提升了生成图像的风格表现力和多样性。实验结果表明，该方法在视觉风格化方面表现优异，能够有效捕捉和生成参考图像的细腻风格特征。
ArtCrafter的整体架构基于扩散模型，包含三个关键组件：
基于注意力的风格提取：该模块通过多层架构和感知器注意力机制，捕捉图像中的细腻风格细节，确保生成的风格编码更为一致和准确。
文本-图像对齐增强：此组件通过精心设计的注意力交互，增强图像和文本嵌入之间的对齐。它将图像和文本嵌入映射到共享特征空间，提高生成图像对文本条件的响应能力。
显式调制：该模块通过线性插值和拼接方案，灵活地融合原始图像和文本嵌入，生成增强嵌入。这种方式使得ArtCrafter能够生成与文本提示相关且视觉表现多样的图像。
在实验设置中，ArtCrafter使用约500,000对真实图像-文本数据进行训练，数据来自LAION-Aesthetic和新构建的ArtMarket数据集。实验评估包括定量和定性分析，比较了ArtCrafter与其他现有方法的性能。结果显示，ArtCrafter在文本一致性、图像一致性和多样性等多个指标上均优于其他方法。此外，用户研究表明，ArtCrafter生成的图像在艺术风格的表现和文本一致性方面受到专业人士的高度评价，验证了其在实际应用中的有效性和优越性。
ArtCrafter的工作原理可以简单理解为一个“艺术家助手”，它帮助我们将特定的艺术风格融入到我们想要的图像中。首先，它通过一种特殊的“注意力机制”来分析参考图像，抓取其中的细节和风格，比如颜色、纹理和构图。接着，它会把这些风格与我们提供的文本描述结合起来，确保生成的图像不仅在样式上与参考图像相似，同时也能反映出文本的内容。最后，ArtCrafter会灵活地调整这些信息的组合方式，使得生成的图像既美观又多样，满足不同的创作需求。通过这种方式，ArtCrafter能够创造出既符合我们的艺术要求，又表现出丰富变化的图像。
2025-01-05｜CQUPT｜⭐️🟡
http ://arxiv.org/abs/2501.02523v1https ://github.com/ddw2AIGROUP2CQUPT/Face-MakeUp
在现代图像生成领域，面部图像的生成具有广泛的应用前景。然而，尽管现有的大规模文本-图像扩散模型表现出强大的生成能力，仅依靠文本提示来生成所需的面部图像仍然面临挑战。为了解决这一问题，本文提出了Face-MakeUp模型，通过多模态面部提示优化图像生成技术。我们构建了一个包含400万高质量面部图像-文本对的数据集（FaceCaptionHQ-4M），并在此基础上训练Face-MakeUp模型。该模型不仅能够生成符合输入文本的面部图像，还能保持与参考图像的一致性。通过在两个面部相关测试数据集上的验证，Face-MakeUp展示了最佳的综合性能，证明了其在面部图像生成任务中的有效性。
Face-MakeUp模型的核心方法包括多个步骤和组件。首先，构建了FaceCaptionHQ-4M数据集，以保证高质量的面部图像-文本对。其次，模型架构设计包括以下几个关键部分：
面部特征提取：使用通用和专门的视觉编码器提取面部的多尺度特征，并利用PoseNet模块学习面部姿态信息。
输入数据：模型接受参考面部图像、提取的姿态图和文本提示作为输入。
交叉注意力模块：通过交叉注意力机制将面部特征与文本特征融合，确保生成图像与输入文本和参考图像的一致性。
特征融合：将提取的面部特征与文本特征相结合，通过前馈层进一步处理，以增强生成图像的细节和个性化特征。
在实验部分，我们对Face-MakeUp模型进行了多个测试，以评估其性能。首先，使用了两个数据集进行评估，每个数据集包含50个参考图像，并设计了20种不同的文本提示。实验结果表明，Face-MakeUp在生成的面部图像的真实感和面部相似性方面显著优于其他模型。此外，通过定量指标（如VLM-score、CLIP-I等）对生成图像的质量进行评估，Face-MakeUp在多项指标中表现出色，证明了其在面部图像生成中的有效性和优越性。同时，手动评估也显示出Face-MakeUp在图像真实感和面部相似性方面的优势。
Face-MakeUp模型的工作原理可以简单理解为一个“智能画家”，它不仅能根据你给出的文字描述来画出面部图像，还能参考你提供的样本图像。首先，我们收集了大量高质量的面部图像和相应的文字描述，构建了一个丰富的数据集。接着，模型会分析你提供的面部图像，提取出各种特征，比如面部的形状、颜色和姿势等。然后，模型将这些特征与文字描述结合起来，生成一个新的面部图像，确保它既符合你的文字要求，又保持与参考图像的一致性。通过这种方式，Face-MakeUp可以生成非常真实且个性化的面部图像，满足不同的需求。
2025-01-04｜UOulu,SEU｜⭐️🟡
http ://arxiv.org/abs/2501.02260v1https ://github.com/weimengting/MagicFace
MagicFace是一个创新的面部表情编辑框架，旨在通过精确控制面部动作单元（AUs）来生成高保真度的表情编辑效果。该模型的核心是一个基于扩散模型的架构，能够在保留个体身份、姿态和背景的同时，实现细致、连续和可解释的表情变化。通过使用AUs作为面部肌肉活动的解剖标记，MagicFace能够灵活地对特定个体的表情进行调整。相较于传统的图像生成方法，MagicFace在表达变化的准确性和灵活性方面表现出色，尤其在处理不同背景和姿态时，能够保持高一致性的面部细节。
MagicFace的工作流程主要包括以下几个步骤：
AUs提取：首先，通过现成的工具（如LibreFace）从输入的肖像图像中提取AUs的强度。这些AUs用于表示目标表情的变化。
ID编码器：为了保持输入个体的面部特征，设计了一个ID编码器，该编码器通过自注意力机制合并外观特征，确保高一致性。
属性控制器：该模块负责处理目标图像的背景和姿态信息，以保持生成图像的背景和姿态一致性。
扩散模型：将AUs变体注入到去噪的UNet中，模型通过迭代去噪过程生成所需的表情图像。这一过程利用了预训练的StableDiffusion模型，以提高生成图像的质量。
在实验部分，研究团队使用了来自Aff-Wild数据集的30,000对图像进行训练，每对图像具有相同的身份但不同的表情、姿态和背景。通过评估生成图像在AUs强度、身份保留、背景保留和头部姿态保留等多个方面的表现，MagicFace展示了其优越的生成质量和灵活性。实验结果表明，MagicFace在保持面部特征的同时，能够实现对表情的精确控制，且在定量和定性评估中均优于现有的面部表情编辑方法。
MagicFace的工作原理可以简单理解为一个高级的照片编辑工具，专门用于修改人们的面部表情。首先，它会从一张照片中提取出关于面部表情的详细信息，这些信息称为面部动作单元（AUs）。接着，MagicFace会利用这些信息来精确调整面部表情，比如让一个人看起来更开心或更悲伤。同时，它还会确保这个人看起来依旧是同一个人，不会因为表情变化而改变身份。为了做到这一点，MagicFace会使用一种特殊的算法，将不同的面部特征和背景信息结合在一起，最终生成一张既真实又符合预期表情的照片。这使得即使是没有专业技能的人，也可以轻松地编辑出自然的面部表情。
2025-01-05｜AlibabaTongyiLab｜⭐️🟡
http ://arxiv.org/abs/2501.02487v2https ://ali-vilab.github.io/ACE_plus_page/
本文介绍了ACE++，一种基于指令的扩散框架，旨在处理各种图像生成和编辑任务。该框架受到FLUX.1-Fill-dev所提出的图像修复任务输入格式的启发，改进了ACE中引入的长上下文条件单元（LCU），并将此输入范式扩展到任何编辑和生成任务。为了充分利用图像生成的先验知识，ACE++采用了两阶段的训练方案，以减少对强大的文本到图像扩散模型（如FLUX.1-dev）的微调工作。第一阶段使用文本到图像模型的0-ref任务进行预训练，第二阶段则使用ACE中定义的所有任务对模型进行微调。通过这种方法，ACE++在图像质量和指令跟随能力方面展现了优越性，且提供了全面的模型集，以支持不同场景的广泛应用。
ACE++的核心在于改进的长上下文条件单元（LCU++），它兼容多种任务并促进不同任务之间知识的整合。其方法分为以下几个部分：
条件标记化：将文本、图像和掩码信息整合成条件单元（CU），以支持生成任务。
改进的LCU：通过将条件上下文和噪声上下文在通道维度上进行连接，减少了上下文感知的干扰，从而提高了模型的训练效率。
模型架构：整合LCU++到FLUX.1-dev的全注意力框架中，形成CU特征图，并通过嵌入层映射到序列token。
两阶段训练方案：
第一阶段：基于0-ref任务对模型进行预训练，利用文本到图像模型的生成能力。
第二阶段：使用所有收集到的0-ref和N-ref任务对模型进行微调，以支持一般指令的生成。
在实验部分，ACE++的模型基于ACEHan等人（2024a）收集的多种任务数据进行训练。首先，使用FLUX.1-dev作为基础模型，在第一阶段进行0-ref任务数据的预训练，随后进行全面微调。实验还包括对特定领域的任务模型进行轻量级微调，以适应不同的应用场景。通过使用AdamW优化器和适当的学习率和权重衰减，确保训练过程的稳定性。最终，ACE++展示了在图像生成、参考生成和局部编辑等多种任务中的出色性能，特别是在保持主体一致性和进行细致的图像编辑方面。
ACE++的方法可以简单理解为一种智能化的图像处理工具，它通过将指令与图像结合来生成或编辑图片。首先，它使用一种叫做条件单元的结构，把描述（比如“让这个角色穿上蓝色衣服”）和图片信息整合在一起。接着，通过一种新的方式把这些信息进行处理，确保模型在理解时不被干扰，从而更快速有效地学习。ACE++还采用了分阶段的训练方法，先让模型学会基础的图像生成，再通过更多的任务来提升它的能力。这样，它能够在处理各种图像任务时表现得更加灵活和高效。
2025-01-01｜AcademiaSinica(taiwan),NCCU｜⭐️
http ://arxiv.org/abs/2501.01998v1
在文本到图像生成领域，StableDiffusion模型已经取得了显著的进展，能够根据文本提示生成高质量的图像。然而，这些模型在准确表示复杂的3D空间关系时常常面临挑战，尤其是在涉及深度和位置的情况下。为了解决这一问题，本文提出了一种新方法——SmartSpatial，旨在增强StableDiffusion模型的空间排列能力。通过引入3D空间感知和注意力引导机制，SmartSpatial能够提高对象定位的精确度，并在生成图像时更好地遵循文本提示。此外，本文还提出了SmartSpatialEval，一个综合评估框架，用于评估生成图像的空间关系，以量化空间准确性并与传统的图像质量评估方法相辅相成。
SmartSpatial的核心方法包括两个主要创新：3D信息集成和注意力引导控制，以及基于视觉语言模型的空间准确性评估。具体而言：
3D信息集成与注意力引导控制：
使用深度估计器生成参考图像的深度图，以捕捉3D空间关系（如“在前面”、“在后面”）。
将深度信息注入到ControlNet中，通过对交叉注意力块的选择和调整，提升空间精度并保持图像质量。
设计特定的损失函数，以确保注意力权重集中在指定区域，从而优化生成的图像的空间关系。
空间准确性评估：
利用视觉语言模型生成对图像中对象关系的描述，并通过依赖解析技术将这些描述转化为图形结构。
通过设计的DistanceScore和SpatialRelationshipScore来量化图像中对象之间的空间关系，确保评估与真实情况的匹配。
本文在两个数据集上进行了实验：COCO数据集的子集和一个自定义的SpatialPrompts数据集。实验评估使用了CLIPScore、IoU和mAP等指标，以比较不同空间控制方法的效果。结果表明，SmartSpatial在空间控制方面显著优于现有方法，尤其是在SpatialPrompts数据集上，IoU提升约68%，mAP提升70%。在更复杂的COCO数据集上，尽管整体得分较低，但SmartSpatial仍然展示了强大的空间控制能力，证明了其在处理复杂3D关系时的有效性和可靠性。
在SmartSpatial的方法中，研究者们通过两种主要方式来提高图像中物体的空间排列准确性。首先，他们利用深度估计技术来获取图像中物体的3D位置，这就像给每个物体加上了一个“深度感知”的眼镜，帮助模型理解哪些物体在前面，哪些在后面。然后，他们在生成过程中使用了一种“注意力引导”的机制，确保模型专注于那些重要的区域，从而更准确地放置物体。其次，他们设计了一种新的评估方法，通过将图像中的对象关系转化为易于理解的描述，来量化这些空间关系。这样，SmartSpatial不仅能生成更准确的图像，还能通过明确的标准来衡量这些图像的空间准确性。

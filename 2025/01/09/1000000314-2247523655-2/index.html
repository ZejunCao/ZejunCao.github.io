2025-01-03｜USTC,MSRA｜⭐️⭐️
http ://arxiv.org/abs/2501.01722v1https ://hanxinzhu-lab.github.io/AR4D/
在近年来，生成模型的进步引发了对动态三维内容创作（即4D生成）的广泛关注。现有方法主要依赖于得分蒸馏采样（SDS）来推断新视角视频，但常常面临多样性不足、时空不一致和提示对齐差等问题。为了解决这些问题，本文提出了一种新的无SDS的4D生成范式——AR4D。该方法分为三个阶段：初始化阶段、生成阶段和优化阶段。首先，通过预训练的专家模型从单目视频的第一帧构建3D表示，并进一步微调以作为生成4D内容的规范空间。接着，利用自回归生成方式，基于前一帧的3D表示生成当前帧的表示，从而提高几何和运动估计的准确性。最后，通过全局变形场和每帧的3D表示的几何结构进行优化，以减少自回归生成引入的外观漂移。
AR4D的实现分为三个主要阶段：
初始化阶段：从单目视频的第一帧生成3D表示。利用预训练的多视图扩散模型生成多个新视图，并使用大规模3D重建模型恢复对应的3D高斯表示。通过微调这些表示，以确保其与参考帧的一致性。
生成阶段：采用自回归方式为每一帧生成3D表示，利用局部变形场建模相邻帧之间的变形。该过程结合逐步视图采样策略，以减少过拟合并提高运动和几何估计的精度。
优化阶段：为了解决外观漂移问题，利用全局变形场和每帧的深度图进行优化，确保每帧的几何结构保持稳定，同时调整外观以提高时空一致性。
在实验中，AR4D被评估在视频到4D生成任务上，与多种现有方法进行比较。通过量化指标如PSNR、SSIM、LPIPS等评估生成内容的质量，结果显示AR4D在各项指标上均优于对比方法，尤其是在时空一致性和与输入提示的对齐上。此外，通过消融实验验证了各个阶段的有效性，表明微调、逐步视图采样和优化阶段的结合显著提升了生成效果。实验结果表明，AR4D不仅能生成更高质量的4D内容，还能在多样性和一致性上获得显著改善。
AR4D的工作流程可以简单理解为三个步骤。首先，在初始化阶段，我们从视频的第一帧提取出三维信息，就像用一张照片来创建一个模型。然后，在生成阶段，我们根据前一帧的信息来逐帧生成新的三维数据，这就像在拍摄一部电影时，每一帧都是基于前一帧的内容。最后，在优化阶段，我们确保生成的图像在外观上不会出现不一致的情况，就像在后期制作中调整每个镜头的色彩和细节，以确保整部电影看起来都很和谐。通过这种方式，AR4D不仅能生成更加真实的动态三维内容，还能保持图像的流畅和一致性。
2025-01-03｜USTC｜⭐️🟡
http ://arxiv.org/abs/2501.01589v1https ://ustc3dv.github.io/D3Human/
D3-Human是一种新颖的方法，旨在从单目视频中重建动态分离的数字人类几何形状。以往的单目视频人类重建主要集中在重建耦合的全身或仅重建衣物，这使得在动画制作等应用中存在困难。该方法的核心挑战在于衣物遮挡身体所造成的可见区域与不可见区域之间的复杂关系。为了应对这一挑战，D3-Human结合了显式和隐式表示，构建了一种新的人体几何模型，使得衣物和身体可以被有效分离和重建。通过这一方法，研究者们不仅能够实现高质量的分离重建，还能够在动画制作和服装转移等实际应用中取得良好效果。
D3-Human的重建过程主要分为几个关键步骤：
hmSDF表示：定义了一种新的人体流形签名距离场（hmSDF），用于分离可见衣物和身体区域。hmSDF能够通过简单的2D人类解析信息对3D衣物和身体进行准确分割，而无需3D衣物先验知识。
区域聚合：通过图像和人类解析分割序列信息，重建可见区域的衣物和身体网格。然后，利用SMPL模型完成不可见身体区域的重建，生成分离的衣物和身体模板。
变形场：采用线性混合皮肤变形（LBS）方法建模大范围的变形，同时使用非刚性变形场独立处理衣物和身体的细微变形，以适应不同的运动规则。
训练策略：分为模板生成和细节变形优化两个阶段。通过RGB损失、掩模损失等多种损失函数，确保重建结果的准确性和细节。
为了验证D3-Human的有效性，研究团队进行了定量和定性实验。定量实验使用Synthetic数据集进行几何重建质量评估，结果显示，D3-Human在重建精度上优于现有方法，尤其在衣物和身体的分离重建方面表现突出。定性实验则采用PeopleSnapshot和SelfRecon数据集，展示了D3-Human在实际应用中的效果，包括动画制作和服装转移。实验结果表明，D3-Human不仅能够实现高质量的重建，还能在动态交互和个性化应用中展现出广泛的潜力。
D3-Human的方法可以简单理解为一个高科技的“3D人类重建器”。首先，它通过分析视频中的人，识别出哪些部分是衣物，哪些是身体。接着，它利用一种叫做hmSDF的技术，将这些部分分开，就像把衣服从人身上脱下来一样。然后，它会把不可见的身体部分（被衣物遮挡的地方）用已有的模型填补完整。最后，D3-Human通过两种不同的方式来模拟身体和衣物在运动时的变化，确保重建的效果既真实又细腻。这样，我们就能得到一个可以进行动画制作或换衣服的3D数字人了。
2024-12-30｜UMassAmherst｜⭐️
http ://arxiv.org/abs/2501.01449v1
人类动作合成是近年来备受关注的研究领域，尤其是在游戏、电影制作和虚拟现实等多个行业中具有广泛应用。传统的基于文本的动作合成方法通常依赖于计算密集型的扩散模型，这使得训练和推理过程耗时较长。为了解决这一问题，本文提出了一种新颖的框架，利用潜在空间中的生成对抗网络（GANs）来实现更快速的训练和推理，同时保持与最先进的扩散模型相当的生成质量。具体来说，该方法通过将文本输入映射到潜在空间中的动作序列，从而实现文本到动作的合成。实验结果表明，所提出的潜在空间GAN（LS-GAN）在多个基准数据集上表现出色，显著提高了动作合成的效率和质量。
本研究的方法主要包括以下几个关键步骤：
变分自编码器（VAE）：使用VAE将动作序列编码到潜在空间。VAE的编码器负责将输入的动作序列转换为潜在表示，而解码器则将潜在表示转换回动作序列。VAE在训练过程中采用均方误差（MSE）和KL散度损失进行优化。
条件输入：为了实现文本到动作的合成，使用预训练的CLIP模型将文本提示编码为潜在空间中的条件输入。对于动作条件，使用可学习的嵌入表示每个动作类别。
生成对抗网络（GAN）：选择GAN作为生成模型，主要有两个架构：简单的VanillaGAN和更复杂的DeepGAN。生成器通过潜在向量和条件输入生成伪造的动作潜在表示，而判别器则学习区分真实和伪造的动作潜在表示。在训练过程中，GAN通过最小化生成器的损失和最大化判别器的损失来进行优化。
在实验部分，研究者在多个数据集上进行了广泛的测试，包括HumanML3D和HumanAct12数据集。通过对比不同模型的性能，发现所提出的LS-GAN在FID（FréchetInceptionDistance）和计算效率方面表现优异。具体而言，DeepWGAN-GP模型在FID上达到了0.482，并且在计算量上相比于传统的扩散模型减少了91%。此外，LS-GAN在多样性和多模态性方面也展示了良好的性能，表明其能够生成高质量且多样的动作序列。实验结果验证了LS-GAN在文本和动作条件下的有效性，并展示了其在实际应用中的潜力。
在这项研究中，科学家们设计了一种新方法，旨在通过简单的文本描述来生成复杂的人类动作。例如，如果你输入“一个人正在跳舞”，这个系统会理解这个指令并生成相应的舞蹈动作。为了实现这一点，研究者首先使用一种叫做变分自编码器（VAE）的技术，将动作数据压缩成一种简化的形式，便于后续处理。接着，他们利用一种叫做生成对抗网络（GAN）的技术来生成动作。GAN的工作原理是有两个部分：一个生成器负责创建新的动作，另一个判别器则负责判断这些动作是真实的还是伪造的。通过这种方式，系统可以不断改进生成的动作，使其更加自然和真实。最终，实验表明这种新方法不仅快速，而且生成的动作质量也很高，适合在游戏和虚拟现实等领域应用。

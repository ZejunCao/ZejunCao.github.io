271-B2｜单目视频重建3D，基于跨视角GS的大型场景重建，基于并行GS的大型城市建筑细粒重建，通过RGB图像估计布料3D状态

2025-01-03｜UTAustin,UPenn,Stanford,JHU,Meta｜🟡
http ://arxiv.org/abs/2501.01949v1https ://videolifter.github.io
在计算机视觉领域，从单目视频高效重建准确的3D模型一直是一项关键挑战，这对虚拟现实、机器人技术和场景理解等应用至关重要。现有的方法通常依赖于预先计算的相机参数和逐帧重建流程，这导致了误差累积和显著的计算开销。为了解决这些局限性，研究者们提出了VideoLifter，一个新颖的框架，利用可学习模型的几何先验，从视频序列中直接进行稀疏到密集的3D表示优化。VideoLifter通过将视频序列分割为局部窗口，匹配和注册帧，构建一致的片段，并以层次方式对齐这些片段，从而生成统一的3D模型。该方法通过跟踪和传播稀疏点对应关系，逐步优化相机姿态和3D结构，从而最小化重投影误差，显著提高了重建的准确性和鲁棒性。
VideoLifter的方法可以概括为两个主要创新：稀疏点基片段注册和层次3D高斯对齐。
稀疏点基片段注册：该方法采用分段处理策略，将输入视频切分为不重叠的片段。每个片段由关键帧和其邻近帧构成，利用三维先验信息进行相机外部参数的优化，从而消除了对传统结构自运动（SfM）的依赖。
层次3D高斯对齐：在进行密集3D重建时，该方法通过引入层次结构来减少长视频序列中的累积误差。通过在已注册的片段中初始化局部3D高斯，并基于光度一致性进行优化，最终将局部表示合并为全局一致的场景表示。
为了验证VideoLifter的有效性，研究者们在多个真实世界数据集上进行了广泛的实验，包括TanksandTemples和CO3D-V2。通过与现有的最先进方法进行比较，VideoLifter在图像质量和计算效率上均表现出色。实验结果显示，该方法在重建时间上减少了82%以上，同时在视觉保真度上超越了其他方法。此外，VideoLifter不需要预先计算的相机参数，使其在处理复杂场景时更加灵活和鲁棒。定量评估显示，VideoLifter在SSIM、PSNR等指标上均取得了显著提升，进一步证明了其在3D重建领域的潜力。
VideoLifter的工作原理可以简单理解为将视频分成小块，然后逐块处理每个小块。首先，它从每段视频中提取出一些关键帧，像是从每段视频中选出几张最清晰的照片。接着，VideoLifter会利用这些照片之间的相似性来估计相机的位置和场景的三维结构，而不需要依赖于传统的复杂步骤。然后，它会把这些小块的信息结合起来，形成一个完整的3D模型。这样做的好处是，VideoLifter不仅能更快地完成任务，还能在处理过程中减少错误，使得最终的3D模型更加精确和真实。
2025-01-03｜BIT
http ://arxiv.org/abs/2501.01695v1
在大规模场景重建中，3D高斯点云技术（3DGaussianSplatting,3DGS）正逐渐成为一种重要的场景表示和重建方法。它通过密集分布的高斯原语实现高分辨率图像的实时渲染。然而，现有的3DGS方法在处理视角变化较大的场景时会遇到优化挑战，导致重建效果不佳。为了解决这一问题，本文提出了一种新颖的跨视角高斯点云方法（CrossView-GS），该方法利用双分支融合策略独立重建来自空中和地面的视图。通过引入梯度感知正则化策略，该方法有效缓解了由于视角差异较大而导致的平滑问题，从而提升了重建质量。实验结果表明，与现有最先进的方法相比，CrossView-GS在新视图合成方面表现出色。
CrossView-GS方法的核心在于双分支模型的构建和优化。具体步骤如下：
双分支初始化：分别从空中视图和地面视图重建模型，生成高斯分布的初始点云。这一过程为后续的跨视角重建提供了可靠的先验信息。
梯度感知正则化：在重建过程中，采用梯度感知的正则化策略，以减少由于视角变化造成的梯度平滑问题。通过记录不同视图下高斯原语的梯度和可见性计数，动态调整密集化策略。
独特的高斯补充：通过评估双分支模型的重建质量，将独特的高斯原语整合到跨视角模型中，进一步优化重建效果，确保最终重建质量优于单视图重建。
该方法在处理大规模场景时，能够有效利用来自不同视角的信息，从而实现高质量的场景重建。
为验证CrossView-GS的有效性，本文在多个基准数据集上进行了广泛的实验，包括MatrixCity、UC-GS和ISPRS等。实验结果显示，CrossView-GS在新视图合成的质量上明显优于当前最先进的方法，尤其在空中视图的重建中，平均PSNR提高了1.15dB。此外，定量评估使用PSNR、SSIM和LPIPS等指标对重建质量进行了评估，结果表明CrossView-GS在所有场景中均表现出色。通过对比实验，CrossView-GS在处理大视角变化时有效克服了传统方法的不足，展现了其在大规模场景重建中的潜力。
CrossView-GS方法可以理解为一种利用不同视角（比如从空中和地面拍摄的图像）来重建三维场景的技术。首先，我们从这两种视角分别创建模型，就像用两种不同的图纸来设计一个建筑。接着，我们会对这些模型进行调整，确保它们在不同视角下的表现都很好。为了避免因为视角差异而导致的细节丢失，我们引入了一种新的技术，这种技术会关注图像中的重要变化，确保在重建过程中不会因为某个视角的信息过于平滑而丢失重要细节。最后，我们将从两个视角得到的信息结合起来，形成一个更完整的三维场景。通过这种方法，我们能够更准确、更快速地重建出复杂的场景，特别是在面对大规模数据时。
2025-01-03｜WHU
http ://arxiv.org/abs/2501.01677v1https ://github.com/TFWang-9527/PG-SAG
本文提出了一种名为PG-SAG的并行高斯散射方法，旨在通过语义感知分组实现大规模城市建筑的细粒度重建。随着3D高斯散射（3DGS）技术的进步，研究者们逐渐能够处理大型场景，然而传统方法在处理高分辨率图像时常常需要进行下采样，导致重建精度下降。PG-SAG充分利用语义信息进行场景分区和高斯核优化，从而在不降低图像分辨率的情况下，实现城市建筑表面的精细重建。该方法通过引入边界感知法线损失和梯度约束平衡负载损失，解决了建筑边界模糊和计算负载的问题。实验结果表明，PG-SAG在建筑表面重建方面表现优于其他现有的3DGS方法，显示出其在大规模城市应用中的潜力。
PG-SAG方法主要包括以下几个步骤：
语义感知数据分组：
利用预训练的语言分割模型（LSA）对建筑区域进行初步分割，生成粗略的建筑掩膜。
通过可靠性评分的多视图投票过滤器，精炼掩膜，确保多视图一致性。
采用DBSCAN算法基于地理位置对建筑区域进行分组，每个子组包含相关的稀疏点云、原始图像和精炼掩膜。
边界感知法线损失：
在优化高斯核时，通过引入边界感知法线损失，针对建筑边界的法线矢量进行加权处理，以减少法线模糊对重建的影响。
梯度约束平衡负载损失：
该损失函数旨在根据场景复杂度调整每个像素的高斯球体数量，促进线程间均衡负载分配，从而减少训练时间和提高重建效率。
网格提取和合并：
对每个建筑的优化高斯进行网格生成，采用多视角深度图合成技术，确保重建结果的几何完整性和边界清晰度。
实验中，PG-SAG在GauU-Scene数据集上进行了性能验证，评估指标包括精度、召回率和F1分数。PG-SAG与多种最先进的方法（如Metashape、2DGS和SuGaR）进行了对比，结果表明PG-SAG在建筑重建的各项指标上均表现出色，尤其在处理复杂建筑布局时，能够有效保留边界细节。具体而言，PG-SAG在俄罗斯建筑和现代建筑场景中均展现了更高的重建精度和更清晰的边界定义。此外，通过消融实验，验证了边界感知法线损失和梯度约束平衡负载损失对提高重建质量和减少训练时间的贡献。
PG-SAG的工作原理可以简单理解为将大规模城市建筑的重建过程分成几个小步骤。首先，它使用一种智能工具（LSA）来识别建筑物的位置，生成一个初步的“建筑掩膜”，就像给建筑贴上一个标签。然后，PG-SAG会通过一个投票机制来确保这些标签在不同角度的照片中都是一致的，确保没有遗漏或错误的地方。接着，PG-SAG会对建筑物的边缘进行特别处理，以确保在重建时不会模糊掉重要的细节。此外，它还会根据场景的复杂性来调整每个部分的处理速度，确保计算资源得到合理利用，从而加快整个重建过程。最后，通过这些步骤，PG-SAG能够生成高质量的建筑网格，保持建筑的真实外观和细节。
2025-01-03｜KTH,CMU｜CoRL2024
http ://arxiv.org/abs/2501.01715v1http ://kth-rpl.github.io/cloth-splatting
Cloth-Splatting是一种新颖的方法，旨在通过RGB图像估计布料的3D状态。该方法基于预测-更新框架，结合了动作条件的动态模型与3D高斯点云技术。研究的关键在于将3D网格表示与高斯点云结合，使得布料的状态空间与图像空间之间形成可微分的映射。这种映射使得可以使用基于梯度的优化技术，仅通过RGB图像来精确修正不准确的状态估计。实验结果表明，Cloth-Splatting在状态估计精度上优于现有基线，同时将收敛时间减少约85%。此研究为机器人在复杂环境中操控变形物体提供了新的思路。
Cloth-Splatting方法的核心在于通过预测-更新框架解决布料的3D状态估计问题，主要包括以下几个步骤：
状态预测：使用图神经网络（GNN）来近似状态转移概率函数，这样可以根据当前状态和机器人动作预测下一时刻的布料状态。
测量更新：利用高斯点云作为测量模型，将预测的状态映射到图像空间中。通过计算预测观测与实际RGB图像之间的误差来更新状态估计。
优化过程：通过最小化测量误差，使用梯度下降法迭代更新状态估计。此过程确保了在更新时能够更好地捕捉布料的动态特性。
通过这种方法，Cloth-Splatting能够在仅依赖RGB图像的情况下，有效地估计布料的3D状态，并实现更快速的收敛。
在实验部分，研究者们对Cloth-Splatting进行了全面的评估，主要包括以下几个方面：
数据集生成：由于缺乏真实场景的地面真值，研究团队生成了包含75个不同场景的合成数据集，涵盖多种布料类型，并记录了RGB-D图像和机器人动作。
性能比较：通过与多个基线方法（如动态3D高斯和MD-Splatting）进行比较，Cloth-Splatting在状态估计精度、位置准确性和生存率等指标上均表现出色。
操作案例：展示了Cloth-Splatting在机器人布料操控中的应用，验证了其在闭环优化中的有效性。
实验结果表明，Cloth-Splatting不仅在准确性上超越了现有方法，而且在收敛速度上也表现出显著优势。
Cloth-Splatting的工作原理可以简单理解为一个“预测-更新”的过程。首先，系统会根据布料当前的状态和机器人即将执行的动作，使用一种特殊的神经网络来预测布料下一时刻的状态。接着，系统会通过摄像头拍摄到的RGB图像来验证这个预测的准确性。如果预测和实际观察之间存在差距，系统会进行调整，优化布料的状态估计。这个过程就像是在不断地修正和优化布料的“模型”，使得机器人能够更准确地理解和操作布料。通过这种方式，Cloth-Splatting能够在仅依赖视觉信息的情况下，有效地追踪和操控布料，大大提高了机器人在处理变形物体时的能力。

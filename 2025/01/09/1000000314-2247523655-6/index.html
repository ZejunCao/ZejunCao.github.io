2025-01-02｜BUPT,BNU｜ICLR2025｜🟡
http ://arxiv.org/abs/2501.01243v1https ://face-human-bench.github.io/
在社交互动中，面孔和人类是至关重要的元素，因此对它们的深刻理解能够显著提升多模态助手的响应质量和应用范围。现有的多模态助手社区缺乏对面孔和人类理解能力的全面、科学的评估。为此，本文提出了一个层次化的能力分类法，包含三个能力层级，并基于此分类法，收集了来自公共数据集的图像和注释，建立了一个半自动化的数据处理管道，生成新的基准测试问题。最终，Face-Human-Bench基准包括900个开发问题和1800个测试问题，支持英语和中文评估。通过对25个主流的多模态大语言模型（MLLMs）进行评估，关注能力之间的相关性、目标相对位置对性能的影响，以及链式思维（CoT）提示对性能的影响。此外，还探讨了在多模态代理中，哪些能力需要由专业模型进行补充。
本研究的方法分为几个关键步骤：
层次化能力分类法：首先构建了一个三层次的能力分类法。第一层（L1）从目标和认知过程两个视角出发，分别包括面孔理解和人类理解，以及感知和推理。第二层（L2）细分为10种能力，第三层（L3）进一步细化为18种能力维度。
数据集收集与处理：基于上述分类法，从面孔和人类社区收集了16个公共数据集，涵盖每个L3能力。随后，利用半自动化的数据管道将原始图像和注释转化为多模态问答问题。
问题生成：每个问题采用多项选择格式，包含图像、问题、选项和答案。通过图像处理管道对原始图像进行处理，生成多样化的问答问题，并确保选项的清晰性和唯一性。
实验部分主要包括以下几个方面：
评估协议：使用多项选择问题的加权准确率作为评估分数，确保不同能力在总分中的权重均等，避免模型偏向某一选项。
模型评估：对25个来自13个模型系列的MLLMs进行评估，涵盖不同规模的开源和闭源模型，分析其在不同能力层次上的表现。
主要结果：结果表明，开源模型在面孔和人类理解能力的评估中表现出色，尤其是InternVL-Chat-v1.2-Plus和LLaVA-NeXT-34B等模型在各项能力上取得了较高的分数。此外，研究还探讨了能力之间的相关性和目标相对位置对模型性能的影响。
在这项研究中，研究者们首先制定了一个清晰的框架，用来评估计算机在理解面孔和人类行为方面的能力。他们将这些能力分成几个层次，从最基本的面孔识别到更复杂的人际关系理解。接下来，研究团队从现有的数据集中收集了大量图像和信息，并通过一种半自动化的方法将这些数据转化为可以测试的问题。每个问题都以选择题的形式呈现，方便计算机模型进行回答。最后，研究者对多种计算机模型进行了测试，看看它们在理解面孔和人类行为方面的表现如何。这种方法不仅帮助他们找出哪些模型表现最佳，还能为未来的多模态助手提供改进方向。
2025-01-02｜KAUST
http ://arxiv.org/abs/2501.01121v1https ://zhyever.github.io/patchrefinerv2/
在高分辨率深度估计领域，尽管现有方法取得了显著成果，但由于依赖于重量级模型和多次推理步骤，计算效率常常不高。为了解决这一问题，本文提出了PatchRefinerV2（PRV2），该方法用轻量级编码器替代了重型精炼模型，从而降低了模型的规模和推理时间。PRV2还引入了一个粗到细（C2F）模块和一个引导去噪单元，以改善精炼特征的质量，并通过噪声预训练策略充分利用轻量级精炼分支的潜力。此外，PRV2采用了尺度和位移不变梯度匹配（SSIGM）损失，以增强合成到真实域的迁移能力。实验结果表明，PRV2在准确性和速度上均优于现有的高分辨率深度估计方法，展示了其在多个真实数据集上的广泛适用性。
PRV2的核心方法包括以下几个部分：
轻量级框架：将重型深度模型替换为轻量级编码器（如MobileNet），显著减少参数数量和内存使用，从而加快推理速度并支持端到端训练。
粗到细（C2F）模块：该模块通过引导去噪单元（GDU）来逐层处理特征，利用粗特征引导去噪和增强精炼特征，确保高分辨率信息的有效提取。
噪声预训练（NP）：在高分辨率训练之前，采用随机噪声替换输入特征，迫使精炼分支学习与深度相关的特征，增强特征表示能力。
尺度和位移不变梯度匹配（SSIGM）损失：在合成到真实域的迁移过程中，利用该损失函数提高模型在边界处的准确性，优化深度估计的细节。
PRV2在多个数据集上进行了评估，包括UnrealStereo4K、CityScapes、ScanNet++和KITTI。实验结果显示，PRV2不仅在UnrealStereo4K数据集中取得了新的最优结果（RMSE值显著降低），而且在推理速度上比现有模型快了多达10.7倍。在真实数据集上，PRV2的边界识别能力也得到了显著提升，尤其是在CityScapes数据集上，边界F1得分提高了17.2%。此外，通过一系列消融实验，验证了C2F模块和噪声预训练策略对模型性能的贡献，并展示了SSIGM损失在合成到真实迁移中的有效性。
在我们的研究中，我们提出了一种新方法，叫做PatchRefinerV2（PRV2），旨在提高从单幅图像中获取深度信息的准确性和速度。我们的主要思路是使用更简单、更轻便的模型，这样不仅能加快处理速度，还能减少所需的计算资源。具体来说，我们设计了一个“粗到细”的处理流程，首先获取整体的粗略深度信息，然后再通过细致的处理来提升深度的精确度。为了让模型更聪明，我们还采用了一种方法，在训练时给模型添加一些随机噪声，迫使它学习更有用的特征。最后，我们引入了一种新的损失函数，帮助模型在真实场景中更好地识别物体的边界。通过这些创新，我们的模型在多个测试中表现出色，证明了它在各种实际应用中的潜力。
2025-01-01｜BIT,BUAA,Kuaishou,Inceptio｜NeurIPS2024
http ://arxiv.org/abs/2501.00873v1https ://kiwixr.github.io/projects/dusa
本研究探讨了生成模型与判别模型结合的潜力，尤其是利用扩散模型中隐藏的语义结构来增强图像分类器在测试时的适应能力。我们提出了一种新方法，称为DUSA（Diffusion-basedStructuredPriorforTest-timeAdaptation），旨在从单一去噪扩散步骤中提取结构化的语义先验知识，从而避免传统方法中基于大量时间步的蒙特卡洛采样所带来的计算负担。通过理论分析，我们揭示了扩散模型中的判别先验，并展示了DUSA在多种测试场景下的有效性。我们还进行了详尽的消融研究，探讨了DUSA的关键组成部分，最终结果表明该方法在多种任务中均表现优越。
DUSA方法的核心在于利用扩散模型中的结构化语义先验，具体步骤如下：
理论基础：首先，我们提供了一个理论框架，揭示了扩散模型的得分函数与条件概率之间的关系，表明可以从中提取隐含的判别先验。
目标构建：接着，我们构建了一个测试时的优化目标，该目标通过将预训练的任务模型与扩散模型结合，利用得分函数来引导模型的适应过程。
候选选择模块：为提高效率，我们设计了候选选择模块（CSM），从任务模型的输出中选择最相关的类，以减少计算量并提高适应性。
噪声聚合：通过聚合条件噪声的估计，构建一个强化的目标函数，使任务模型能够更好地学习扩散模型中的结构化信息。
我们在三个基准数据集上进行了详尽的实验，评估DUSA在不同任务上的表现。首先，在ImageNet-C数据集上进行的全面测试时间适应实验表明，DUSA在多个噪声类型下均显著提高了分类准确率。其次，在ADE20K数据集上进行的语义分割实验中，DUSA同样展现了出色的适应能力，尤其在处理图像腐蚀时的表现优于现有方法。最后，我们进行了消融研究，验证了DUSA中各个组成部分对整体性能的贡献，进一步证明了从生成模型中提取结构化先验的有效性。
在DUSA的方法中，我们主要是利用一种名为扩散模型的技术来增强图像识别模型的表现。首先，我们发现扩散模型不仅仅用于生成图像，它的内部结构也能帮助我们更好地理解数据。接着，我们设计了一个智能的选择机制，只关注那些最有可能的分类，这样可以减少计算量。最后，我们通过整合来自扩散模型的噪声信息，帮助我们的图像识别模型在面对新数据时更快适应，提升了准确性。简单来说，DUSA就像是给我们的图像识别模型装上了一双“慧眼”，让它能够在复杂的环境中更好地辨别事物。
2024-12-31｜SYSU,HiDream.ai｜ECCV2024
http ://arxiv.org/abs/2501.00437v1https ://jianjieluo.github.io/SynthImgCap
近年来，零-shot图像描述（Zero-shotImageCaptioning,ZIC）逐渐受到关注，该方法仅依赖文本数据进行训练。尽管文本到图像的扩散模型在生成合成图像-文本对方面取得了显著进展，但合成图像中突出的细节缺陷导致语义不匹配，从而影响了描述的质量。为了解决这个问题，本文提出了一种新颖的“基于补丁的跨模态特征混合”（Patch-wiseCross-modalFeatureMix-up,PCM）机制，旨在以细粒度的方式在训练过程中自适应地减轻合成内容中的不真实部分。该机制可以集成到大多数编码-解码框架中，形成我们的PCM-Net。通过在MSCOCO和Flickr30k数据集上的广泛实验，验证了PCM-Net相较于现有最先进的视觉语言模型（VLMs）方法的优越性。
本文的方法主要包括以下几个步骤：
合成图像生成：利用文本到图像的扩散模型生成合成图像，构建合成数据集。每个文本句子通过模型生成对应的高保真合成图像。
补丁级别的跨模态特征混合：首先提取合成图像的全局和局部补丁特征。通过在CLIP空间中进行零-shot实体分类，检测图像中的显著视觉概念。然后，根据补丁与显著概念特征之间的相似度，进行特征混合，替换掉合成图像中不真实的细节。
视觉-语义编码器：将混合后的特征输入到视觉-语义编码器中，进一步提炼特征。最后，通过解码器生成描述句子。
CLIP加权交叉熵损失：设计一种新的损失函数，动态调整训练损失，根据合成图像与真实文本之间的相似度来加权，从而优先考虑高质量的图像-文本对。
在实验部分，研究者在MSCOCO和Flickr30k数据集上评估了PCM-Net的性能。在与其他最先进的方法进行比较时，PCM-Net在多项评估指标上均表现出色，尤其是在CIDEr得分上，达到了113.6%，相较于其他方法有显著提升。此外，研究者还进行了消融实验，验证了每个组件对整体性能的影响，结果表明，补丁级特征混合和CLIP加权损失的结合显著提升了模型的效果。实验结果表明，PCM-Net在零-shot图像描述任务中具有良好的鲁棒性和迁移能力。
在我们的方法中，首先，我们使用一种叫做扩散模型的技术，根据文本生成合成图像。接下来，我们从这些图像中提取出重要的视觉特征，并通过一种叫做“补丁级特征混合”的方法，将这些特征与文本特征结合起来。这样做的目的是为了减少合成图像中可能存在的缺陷，比如模糊或不真实的细节。然后，这些混合后的特征被送入一个模型中，模型会根据这些信息生成描述图像的句子。最后，为了确保模型在训练时更加关注高质量的图像和文本对，我们设计了一种新的损失函数，使得训练过程能够自动调整，优先考虑那些更好的数据。通过这些步骤，我们的模型在生成图像描述方面表现得更加准确和自然。
2025-01-01｜CQUT,CUHK
http ://arxiv.org/abs/2501.00882v1https ://github.com/Chianglu/FullTranNet
随着视频数据量的急剧增加，视频摘要技术应运而生，旨在从原始视频中提取出简洁且具代表性的内容，以便于用户快速浏览和理解视频信息。当前主流的视频摘要方法多基于递归神经网络（RNN）或卷积神经网络（CNN），甚至是仅使用编码器结构的变换器（Transformer）。本研究提出了一种新颖的全变换器架构——FullTransNet，采用了编码-解码结构，旨在处理视频摘要任务，并通过局部-全局稀疏注意力机制来提高效率。该方法不仅能有效捕捉视频的长距离依赖关系，还能减少计算成本，最终在SumMe和TVSum两个公共数据集上取得了优异的效果。
本研究的方法主要包括以下几个关键部分：
全变换器架构：采用全变换器结构的编码-解码模型，将视频摘要视为序列到序列的学习问题。编码器处理原始视频帧序列，而解码器生成对应的摘要序列。
稀疏注意力机制：引入局部-全局稀疏注意力（LGA）机制，替代传统的全注意力机制。局部注意力关注相邻帧的信息，增强了对关键帧的选择能力，而全局注意力则允许模型捕捉整个序列的全局信息。这种组合有效地平衡了性能与计算效率。
训练过程：采用监督学习方式进行训练，使用教师强制方法（teacherforcing），即在解码过程中以真实的目标输出作为输入，以提高模型的学习效果。
在实验中，使用了两个公共数据集SumMe和TVSum进行评估。SumMe数据集包含25个用户视频，提供了多种用户标注的关键帧摘要；TVSum数据集则包含50个视频，标注为每帧的重要性分数。通过在这两个数据集上的广泛实验，验证了FullTransNet在视频摘要任务中的有效性。实验结果显示，该模型在F-measure指标上分别达到了54.4%（SumMe）和63.9%（TVSum），并且在计算和内存需求上相对较低，证明了其高效性。
在我们的研究中，我们使用了一种名为FullTransNet的全新架构来处理视频摘要。这个方法可以理解为一个聪明的工具，它像一个视频剪辑师一样，能快速找到视频中最重要的部分。我们特别设计了一个叫做局部-全局稀疏注意力的机制，简单来说，它就像是一个过滤器，能帮助我们的工具在观看视频时，既能关注到每一帧的细节，又能把握住整个视频的主题。这样，我们的工具在生成摘要时，不仅能快速处理大量视频内容，还能确保所选的关键帧真实反映视频的核心信息。通过这种方式，我们的模型能够在多个视频数据集上表现出色，帮助用户更高效地获取他们想要的信息。

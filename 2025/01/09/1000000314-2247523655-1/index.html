2025-01-03｜ECNU｜⭐️⭐️
http ://arxiv.org/abs/2501.01700v1
本文提出了一种名为EmoMV的情感驱动音乐到视觉图像的操控方法，旨在通过音乐情感增强图像的情感表达。尽管近年来深度学习在图像识别领域取得了显著进展，但在情感表达的理解和控制上仍然面临挑战。现有的研究往往侧重于理论任务，如乐器识别和节拍分析，缺乏对音乐情感多维度的深入探讨。EmoMV通过结合音乐的底层元素（如音高和节奏）与图像的视觉特征（如色彩和光照），实现了音乐与视觉艺术的跨模态融合，从而提升了图像的情感深度。本文的研究不仅填补了音乐与视觉之间的情感连接空白，也为创意产业和人机交互技术开辟了新的研究方向。
EmoMV采用了一个两阶段的方法，主要包括Mus-Vis文本对齐和情感感知美学图像精炼两个模块。具体步骤如下：
Mus-Vis文本对齐:
提取音乐的基本元素（如音高、节奏）并将其转化为情感表达。
采用编码器-解码器框架生成音乐元素描述，使用HTSAT进行音乐编码，BART进行解码。
将生成的音乐描述与图像输入结合，利用多模态模型生成情感相关的描述。
情感感知美学图像精炼:
将音乐情感映射到图像的美学属性上，运用顶层处理方法将音乐情感转化为图像的核心元素。
通过情感描述、叙事描述和图像特征的结合，确保在图像编辑前后语义的一致性。
最后，利用IC-Light模型对图像进行编辑，以增强其情感表达。
为验证EmoMV的有效性，研究团队构建了一个包含3.8万对音乐与图像的数据集，并设计了多维度的评估框架。评估内容包括图像美学、图像质量和人类反馈等多个方面。通过对比实验，结果显示EmoMV生成的图像在情感表达和美学质量上均优于传统方法。具体分析包括对图像清晰度、对比度、色彩丰富度等指标的量化评估，以及基于EEG信号分析用户的情感反应。实验结果表明，EmoMV不仅能够有效提升图像的情感共鸣，还在艺术治疗等应用场景中展现出积极效果。
在我们的研究中，我们创造了一种新方法，叫做EmoMV，来帮助我们通过音乐来改变和提升图像的情感表达。这个方法分为两个主要步骤。首先，我们会分析音乐，找出它的基本元素，比如音调和节奏，并将这些元素转化为情感描述。接着，我们会把这些描述与图像结合起来，确保图像在编辑前后保持一致的主题和情感。第二个步骤是将音乐的情感映射到图像的视觉特征上，比如光线和颜色。最终，我们会使用一种特殊的编辑工具来调整图像，使其更好地表达与音乐相符的情感。这种方法不仅能让图像看起来更美，更能让人们在观看时感受到与音乐相匹配的情感。
2025-01-03｜HIT,HuaweiNoah’sArkLab,PazhouLab｜⭐️🟡
http ://arxiv.org/abs/2501.01633v1https ://github.com/120L020904/ACE
在文本到图像（T2I）模型的快速发展中，尽管其生成高质量图像的能力显著提升，但也引发了对潜在非法内容创作的担忧，如侵犯版权的图像。现有的概念抹除方法在防止通过文本提示生成被抹除的概念方面表现良好，但在图像编辑过程中却常常失效。为了解决这一问题，本文提出了一种新的反编辑概念抹除（ACE）方法，旨在同时在生成和编辑过程中抑制目标概念的产生。ACE通过将抹除指导注入到条件和无条件噪声预测中，使模型能够有效地防止在编辑和生成过程中创造被抹除的概念。此外，引入了一种随机修正指导，以解决无关概念的侵蚀问题。通过一系列实验，ACE在抹除知识产权（IP）角色、明示内容和艺术风格等方面表现出色，显示出其在生成和编辑过滤中的优势。
ACE方法的核心在于其双重抹除机制，具体步骤如下：
抹除指导注入：在训练过程中，将抹除指导融入条件和无条件噪声预测，以确保模型在生成和编辑时都能有效抑制目标概念的生成。
条件噪声预测：通过调整条件噪声预测，使其与目标概念的抹除指导相一致，从而引导模型远离被抹除的概念。
无条件抹除指导：引入无条件抹除指导，进一步增强模型在生成过程中对目标概念的抑制能力，确保生成的图像不包含被抹除的概念。
先验概念保护机制：为防止抹除过程中的过拟合，ACE还引入了先验概念保护机制，通过随机选择相关概念并引导模型保持其生成能力，以确保非目标概念的生成不受影响。
在实验部分，ACE方法被应用于多个任务，包括知识产权角色抹除、艺术风格抹除和明示内容抹除。实验使用了多个对比方法，如ESD、SPM、AdvUnlearn、MACE和RECE，评估其在抹除目标概念及保持非目标概念生成能力方面的效果。结果显示，ACE在抹除目标概念的同时，能够有效保留其他相关概念的生成能力，且在编辑过滤中表现出色。定量评估使用了CLIP和LPIPS等指标，结果表明ACE在抹除和编辑过滤任务中均优于现有方法，进一步证明了其有效性和鲁棒性。
ACE方法可以理解为一种智能的图像处理技术，旨在确保在生成或编辑图像时，不会出现某些特定的内容。想象一下，如果你想让图像中不再出现“皮卡丘”，ACE会在生成图像时和编辑图像时都采取措施，确保“皮卡丘”不会被再次生成或编辑出现。具体来说，ACE会在训练过程中教模型如何识别和去除这些不想要的内容，同时还会保护其他相关的内容不受影响。通过这种方式，ACE不仅能有效地抹去特定的概念，还能确保在编辑时不破坏其他图像的元素。这种方法不仅提高了图像生成的安全性，还为用户提供了更灵活的编辑选择。
2025-01-03｜KunlunInc.｜⭐️🟡
http ://arxiv.org/abs/2501.01790v1https ://github.com/feizc/Ingredients
本研究提出了一种名为“Ingredients”的新框架，旨在通过视频扩散变换器将多个特定身份照片整合进视频创作中，以实现个性化视频生成。该方法的核心目标是确保在多个参考图像中保持一致的面部身份，同时灵活地融入用户定义的文本提示。现有的个性化方法面临诸多挑战，包括训练过程中的高资源消耗和难以扩展性。与之相比，Ingredients通过引入三个关键模块：面部特征提取器、多尺度投影器和身份路由器，大大提升了个性化视频生成的效率和质量。该框架在经过精心策划的文本-视频数据集上进行了训练，实验结果表明，Ingredients在生成高质量、个性化视频内容方面表现优异，标志着向更有效的生成视频控制工具迈出了重要一步。
Ingredients框架主要由三个模块组成：
面部特征提取器：该模块从全局和局部视角提取每个身份的面部特征。全局特征通过人脸检测和合成图像来获取，以减少因裁剪或缩放导致的失真；而局部特征则通过面部识别网络和CLIP图像编码器提取，确保每个身份的特征独立保留。
多尺度投影器：该模块负责将提取的面部嵌入映射到视频扩散变换器的上下文空间。全局面部嵌入通过与潜在噪声输入的直接拼接来处理，而局部嵌入则通过多步骤融合，利用Q-former结构实现与视频视觉标记的交互。
身份路由器：此模块动态地将每个面部嵌入分配到对应的时空区域，避免身份混合。通过位置敏感的路由网络，确保每个视频帧的特定区域对应唯一身份特征。
在实验部分，研究团队使用自建的多身份文本-视频数据集进行训练，评估了Ingredients在多身份视频生成中的表现。实验结果显示，Ingredients在身份保留、视觉质量和文本相关性方面均优于现有基线方法。具体来说，使用面部相似度、CLIP得分和FID等指标进行定量评估，结果表明，Ingredients不仅能够有效保持各个身份的独特性，还能在生成过程中保持与文本提示的一致性。此外，研究还进行了消融实验，探讨了不同模块和损失函数对生成质量的影响，进一步验证了该框架的有效性和灵活性。
在我们的研究中，我们开发了一种名为“Ingredients”的方法，能够将多张人脸照片融入到视频中，生成个性化的内容。这个过程主要分为三个步骤。首先，我们有一个面部特征提取器，它会仔细分析每张照片，提取出每个人的独特面部特征。这个步骤确保我们能够准确地识别每个人的脸，即使在不同的照片中。接下来，我们使用一个多尺度投影器，这个工具会将提取到的面部特征与视频中的其他信息结合起来，确保它们在生成的视频中能够自然地呈现。最后，我们有一个身份路由器，它会智能地将每个人的特征放到视频中的正确位置，避免不同身份的特征混在一起。通过这样的方式，我们不仅能够生成高质量的视频，还能确保每个人在视频中的表现都是独特且一致的。
2025-01-03｜ZhejiangLab,LiAuto,HIT,ZJU,UCAS｜⭐️
http ://arxiv.org/abs/2501.01808v1
随着音频驱动的头像生成技术的迅速发展，生成生动的谈话头像视频已取得显著进展。然而，要制作出逼真的谈话头像视频，不仅需要精准的音频同步，还需要捕捉广泛的情感和细腻的面部表情。现有方法面临着两大挑战：一是缺乏针对单一基本情感表达的建模框架，限制了复杂情感（如复合情感）的生成；二是缺乏丰富的人类情感表达数据集，限制了模型的潜力。为了解决这些问题，本文提出了“情感专家混合模型”（MoEE），该模型通过解耦六种基本情感，实现了单一和复合情感状态的精确合成。同时，我们还构建了“DH-FaceEmoVid-150”数据集，专门收录了六种常见人类情感表达及四种复合情感，从而扩展了情感驱动模型的训练潜力。通过大量的定量和定性评估，我们证明了MoEE框架在生成复杂情感表达和细腻面部细节方面的优越性，设立了该领域的新基准。
本研究的核心是情感专家混合模型（MoEE），其结构包括多个关键模块：
模型概述：MoEE接收一个肖像图像、音频片段序列和情感条件，目标是生成与音频同步的动画视频。模型的基础是去噪U-Net架构，能够在特定条件下去噪输入的多帧噪声潜在变量。
情感专家模块：我们训练了六个情感专家网络，每个专家专注于一种基本情感。这些专家通过交叉注意机制进行协调，以生成丰富的情感表达。
情感到潜在模块：该模块接受多种模态输入（如文本、音频和标签），并将这些条件映射到共享的情感潜在空间，以确保对情感的良好控制。
训练与推理：模型采用两阶段训练过程，首先对参考网络和去噪U-Net进行微调，然后优化情感专家模块，以提升情感控制的准确性和多样性。
在实验部分，我们使用了多种数据集，包括DH-FaceEmoVid-150和HDTF数据集，进行模型性能评估。我们对模型进行了定量和定性分析，评估指标包括感知相似度、生成图像的真实性、面部关键点对齐及音视频同步效果。实验结果显示，MoEE在所有评估指标上均优于现有方法，尤其是在情感控制和面部细节生成方面。此外，我们还进行了用户研究，参与者对MoEE生成的结果给予了最高评分，进一步验证了其在情感表达和自然性方面的优越性。
在我们的研究中，我们设计了一种新模型，叫做情感专家混合模型（MoEE），它帮助计算机生成更自然的谈话头像。这个模型的工作方式可以简单理解为：它首先从一张图片和一些音频中获取信息，然后通过不同的“情感专家”来判断应该表达什么情感。每个情感专家就像一个小助手，专门处理一种情感，比如快乐或悲伤。这样一来，当你给它一段音频时，它可以根据音频内容和情感指令，生成一个生动的头像，嘴巴和表情都能同步，听起来很自然。此外，我们还创建了一个新的数据集，包含了很多不同的情感视频，帮助模型学习如何更好地表达这些情感。通过这些方法，我们的模型能够生成既真实又富有表现力的头像动画。
2025-01-03｜JD.Com,HKU｜⭐️
http ://arxiv.org/abs/2501.01798v1https ://joy-mm.github.io/JoyGen
本研究提出了一种名为JoyGen的创新性双阶段框架，旨在改善音频驱动的3D说话人物视频编辑中的唇音同步和视觉质量问题。尽管近年来在说话面孔视频生成方面取得了显著进展，但在编辑现有视频时，准确的唇音同步和高视觉质量仍然面临挑战。JoyGen通过利用3D重建模型和音频到运动模型，分别预测身份和表情系数，从而实现精确的唇动生成。此外，研究团队还构建了一个包含130小时高质量视频的中文说话面孔数据集，以支持模型训练。实验证明，JoyGen在唇音同步和视觉质量方面优于现有方法。
JoyGen的工作流程分为两个主要阶段。第一阶段涉及3D面部重建和音频到运动的转换。具体步骤如下：
3D重建模型：从单张图像中提取身份系数，这些系数用于构建3D面部网格。
音频到运动模型（Audio2Motion）：利用流增强变分自编码器，将音频信号转换为表情系数，确保生成的唇动与输入音频高度一致。
深度图生成：通过深度信息增强唇动生成的准确性，结合面部深度图和音频特征，提供全面的监督信号。
图像编码和解码：使用预训练的图像编码器将输入图像映射到潜在空间，并通过解码器生成最终的面部图像。
训练细节：采用L1损失函数进行优化，确保模型在潜在空间和图像空间之间的精确对齐。
在实验部分，研究团队使用了开放源代码的HDTF数据集和自建的数据集进行模型训练。数据预处理包括视频分割、面部关键点提取和3DMM系数预测。通过比较JoyGen与Wav2Lip和MuseTalk等现有方法的性能，结果表明，JoyGen在唇音同步和视觉质量上均表现出色。具体评估指标包括FID、LSE-D和LSE-C，JoyGen在这些指标上均优于其他方法。此外，研究还通过定性分析展示了不同方法在唇动同步和视觉质量上的差异，进一步验证了JoyGen的有效性。
JoyGen方法的核心在于将音频信号和面部图像结合起来，以生成更加自然的说话面孔视频。首先，它从一张面部照片中提取出人的特征，比如身份和表情。接着，系统会分析输入的音频，找出与说话时嘴唇运动相关的部分。然后，它会利用这些信息生成一个三维的面部模型，确保嘴唇的运动与音频完全同步。最后，经过一系列处理，系统将生成的面部图像转换成最终的视频输出。这个过程不仅提高了视频的视觉效果，还确保了唇动与音频的精准匹配，使得生成的说话视频更加真实自然。

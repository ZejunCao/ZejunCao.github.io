2025-01-02｜NTU,ByteDance｜⭐️⭐️
http ://arxiv.org/abs/2501.01320v1https ://iceclear.github.io/projects/seedvr/
SeedVR是一种新型的扩散变换器模型，旨在解决视频恢复任务中的多种挑战，如保持图像质量和时间一致性。该模型能够处理任意长度和分辨率的视频，克服了传统方法在处理复杂和未知退化时的局限性。SeedVR采用了移位窗口注意力机制，显著提高了长视频序列的恢复效果，支持在空间和时间维度上可变大小的窗口，从而有效地应对传统窗口注意力的分辨率限制。尽管SeedVR的参数数量达到2.48亿，但其速度比现有的扩散视频恢复方法快了超过两倍，同时在合成和真实世界基准测试中表现出色。
SeedVR的核心方法包括以下几个方面：
移位窗口注意力机制：该机制允许模型在处理视频时使用更大的注意力窗口（64×64），从而捕捉长距离依赖关系，并减少计算复杂性。
因果视频变分自编码器（CVVAE）：通过时空压缩，CVVAE显著降低了视频恢复的计算成本，同时保持了高质量的重建效果。
多阶段渐进训练策略：该策略在训练过程中逐步增加输入视频的长度和分辨率，帮助模型在大规模数据集上快速收敛。
大规模混合数据集训练：SeedVR在图像和视频的混合数据集上进行训练，增强了模型的泛化能力，并提高了对真实世界视频退化的适应性。
在实验部分，SeedVR在多个视频恢复基准测试中表现出色，包括合成数据集和真实世界数据集。通过与现有的多种视频恢复方法进行比较，SeedVR在多个指标上（如PSNR、SSIM和LPIPS）均取得了优越的性能。实验结果表明，SeedVR在处理不同类型的视频退化时，能够有效恢复细节和纹理。此外，模型的训练效率也得到了显著提升，尤其是在处理高分辨率视频时，SeedVR显示出更快的生成速度和更低的计算成本。
SeedVR的工作原理可以简单理解为它如何高效地处理视频恢复。首先，它使用一种叫做“移位窗口注意力”的技术，这种技术允许模型在分析视频时关注更大范围的内容，这就像在阅读时能够看到更多的文字，而不是只看一小块。其次，SeedVR还设计了一种特殊的自编码器，能够在压缩视频信息的同时，保留重要的细节，这样可以在处理视频时减少计算量。最后，SeedVR在训练时会逐步增加视频的复杂度，帮助模型更好地适应各种视频情况。通过这些创新，SeedVR不仅提高了视频恢复的速度，还确保了恢复效果的质量。
2025-01-02｜HKU,AlibabaDAMOAcademy,HupanLab,HUST｜⭐️🟡
http ://arxiv.org/abs/2501.01427v1https ://videoanydoor.github.io/
在视频生成领域，插入对象到视频中一直是一个挑战性任务，尤其是在保持对象外观细节和准确建模运动一致性方面。为了解决这一问题，本文提出了VideoAnydoor，一个高保真度的视频对象插入框架，具备精确的运动控制能力。该框架基于文本到视频模型，利用身份提取器注入全局身份信息，并通过框选序列控制对象的整体运动。VideoAnydoor不仅能够实现单个对象的插入，还支持多对象的迭代添加和交换，且无需针对特定任务进行微调，展现出显著的优越性。
VideoAnydoor的工作流程主要包括以下几个关键部分：
任务定义：该框架旨在实现高保真的视频对象插入，保持未编辑区域与源视频一致，同时准确控制目标对象的运动。
整体流程：通过2D图像修复扩展到视频生成，首先将源视频和目标对象的掩码输入到3DU-Net模型中，利用潜在扩散模型确保时间一致性。
像素变换模块：利用关键点和对应的运动轨迹作为输入，进行像素细节的变换，以实现外观细节与运动的联合建模。
重加权重构损失：通过增强关键点贡献的方式，提高运动和身份建模的精度，同时采用图像和视频混合训练策略，解决高质量视频数据稀缺的问题。
在实验部分，研究者们进行了全面的定量和定性评估。通过构建包含多种视频类别的基准数据集，评估了VideoAnydoor在内容和运动控制方面的性能。实验结果表明，VideoAnydoor在多个自动化指标上均优于现有方法，特别是在ID保留和运动一致性方面。此外，用户研究也显示，VideoAnydoor在视频编辑的质量、流畅度和多样性等方面均获得了较高的评分，验证了其在实际应用中的有效性。
VideoAnydoor的工作原理可以理解为一个智能的视频编辑工具。首先，它会接收一段视频和你想要插入的对象的图片。然后，系统会分析这段视频，找出合适的位置和运动轨迹。接下来，一个“像素变换器”会根据这些轨迹调整插入对象的外观，确保它在视频中看起来自然且运动一致。最后，为了提高效果，系统还会根据关键点的位置来优化插入的质量。这样，用户只需提供目标对象的图片和运动指令，VideoAnydoor就能生成一个完美融合的新视频。
2025-01-02｜FDU,NTU｜⭐️⭐️
http ://arxiv.org/abs/2501.01425v1https ://henghuiding.github.io/SynFMC/
本研究提出了一个名为SynFMC的合成视频生成数据集，旨在实现可控的相机和对象运动。随着视频生成技术的发展，控制动态对象和相机的运动成为一个重要的研究方向。然而，现有的数据集往往缺乏全面的运动注释，导致现有算法无法同时控制相机和对象的运动，从而限制了生成内容的灵活性。为了解决这个问题，SynFMC数据集提供了多样化的对象资产和环境，并包含完整的6D姿态信息，支持多种运动模式的生成。研究还提出了一种新的方法——自由形式运动控制（FMC），使得用户可以独立或同时控制对象和相机的运动，从而生成高保真度的视频。
FMC方法的核心在于两个主要组件：相机运动控制器（CMC）和对象运动控制器（OMC）。具体步骤如下：
资产选择：从多种类别中选择3D对象和HDRI环境贴图，以丰富合成内容。
运动类型选择：
对象运动：包括水平线、曲线、静态等多种运动模式。
相机运动：根据视角类型、高度类型和距离类型进行控制，确保相机的运动符合对象的动态。
动画序列生成：根据选择的资产和运动类型，生成3D动画序列。用户可以通过绘制曲线或指定运动类型来输入相机和对象的轨迹。
数据生成管道：使用UnrealEngine进行合成视频渲染，生成的视频包括详细的注释，如6D姿态、实例分割图和深度图等。
为了验证SynFMC数据集和FMC方法的有效性，进行了多项实验。实验结果显示，FMC方法在多个场景中优于现有技术，能够实现更高质量的对象和相机运动控制。具体评估指标包括FID（用于视觉质量）、FVD（用于时间一致性）和CLIPSIM（用于语义相似性）。通过与MotionCtrl等方法的对比，FMC在同时控制相机和对象运动方面表现出色，且用户研究结果显示FMC在质量和准确性上均优于其他方法。
在这项研究中，科学家们开发了一种新方法，帮助我们在合成视频中更好地控制相机和对象的运动。想象一下，当你在拍摄一段视频时，你希望相机能够跟随一个跑步的人，同时也想让他身边的动物做出不同的动作。为此，他们首先收集了各种3D模型和背景图像，然后设计了一套规则，用来生成这些对象和相机的运动轨迹。用户只需简单地通过画线或选择运动类型，就可以轻松地控制这些动作。最终，通过先进的技术，他们成功地生成了高质量的视频，并且在与其他方法的比较中，表现得更加出色。这项研究为视频制作提供了更多的灵活性，使得即使是没有专业技能的人也能创造出精彩的视觉效果。
2025-01-02｜AdobeResearch,KAIST,UCL｜⭐️🟡
http ://arxiv.org/abs/2501.01409v1https ://paulchhuang.github.io/jog3rwebsite
本研究提出了一种名为JOG3R的统一模型，旨在同时实现视频生成和相机姿态估计。随着图像生成模型在3D能力上的突破，视频扩散模型也迅速发展，能够从文本或图像提示中生成真实且多样化的视频。研究者们关注于视频生成模型中是否存在原生的3D意识，并探讨其在相机姿态估计中的应用。通过将OpenSora视频生成模型与结构光束法（SfM）结合，研究表明，尽管视频生成模型的特征在3D意识方面有限，但通过特定任务的监督，可以显著提高相机姿态估计的准确性。最终，JOG3R模型在生成视频质量的同时，也实现了高质量的相机姿态估计。
JOG3R模型的核心在于将视频生成和相机姿态估计任务进行统一。其具体方法包括以下几个步骤：
视频生成模型：采用OpenSora作为基础视频生成模型，利用其在低维潜在空间中进行扩散处理。每个输入视频帧被映射到潜在空间，随后通过逐步添加高斯噪声来生成视频。
相机姿态估计模块：使用DUSt3R框架作为相机跟踪模块，处理输入图像对以估计相机的相对姿态。该模块利用视觉变换器（ViT）编码器独立编码每张图像，并通过解码器进行交叉视图信息共享。
特征路由：将视频生成模型的中间特征路由到相机姿态估计模块。通过对这些特征进行微调，研究者能够提高相机姿态估计的准确性，尤其是在优化相机姿态和深度估计时。
训练目标：在训练过程中，设置生成损失和重建损失，分别用于优化视频生成和相机估计的性能。通过联合训练，模型能够在生成视频的同时，准确估计相机姿态。
在实验部分，研究者对JOG3R模型的设计选择进行了广泛的评估。主要实验包括：
视频到相机姿态估计（V2C）：通过向输入视频添加噪声，进行去噪处理，并将特征映射路由到DUSt3R解码器，最终注册点图以获得相机姿态。
文本到视频生成（T2V）：评估在文本引导下生成视频的效果，并分析额外的相机姿态监督对生成质量的影响。
联合生成和相机估计（T2V+C）：在不需要额外噪声添加的情况下，直接生成视频并同时估计相机姿态，展示了模型在两项任务中的一致性和有效性。
在这个研究中，科学家们设计了一种新模型，叫做JOG3R，目的是让计算机能够同时生成视频和估计相机的位置。首先，模型会将每一帧视频转换成一种特殊的数字格式，然后逐步添加噪声，最终生成出平滑的视频。接着，模型会利用一种叫DUSt3R的工具来分析视频中的每一帧，计算出相机的角度和位置。为了让模型更聪明，研究者将生成视频时得到的特征信息传递给相机分析工具，这样可以帮助模型更准确地判断相机的姿态。在训练时，模型不仅要学会生成视频，还要学会如何准确估计相机的位置。通过这样的方式，JOG3R能够在生成高质量视频的同时，提供准确的相机位置信息。
2024-12-30｜Lightricks｜⭐️🟡
http ://arxiv.org/abs/2501.00103v1https ://github.com/Lightricks/LTX-Video
LTX-Video是一种基于变换器的潜在扩散模型，旨在通过优化视频生成过程中的各个组件的相互作用，提升效率和质量。与现有方法将视频变分自编码器（Video-VAE）和去噪变换器视为独立模块不同，LTX-Video采用了一种整体性的方法，旨在实现更高的压缩比和更好的视频质量。其核心是一个精心设计的Video-VAE，压缩比达1:192，能够在32×32×8的时空分辨率下高效操作。该模型支持文本到视频和图像到视频的生成，能够在NvidiaH100GPU上以超过实时速度生成768×512分辨率的24fps视频，表现超越同类模型，成为可访问且可扩展的视频生成基准。
LTX-Video采用了一种整体性的方法来优化Video-VAE与去噪变换器之间的相互作用，以实现快速且高质量的视频生成。其主要方法步骤包括：
高维潜在空间：模型在高压缩比（1:192）和时空下采样（32×32×8）的潜在空间中操作，减少信息冗余。
VAE解码器的任务重构：VAE解码器不仅负责将潜在表示转换为像素，还执行最后的去噪步骤，以确保生成高频细节。
共享去噪目标：通过在VAE解码器与变换器之间共享去噪目标，模型能够有效融合去噪过程，提升生成质量。
改进的损失函数：引入了重建GAN（ReconstructionGAN）和多层噪声注入等新颖损失函数，以提高生成图像的清晰度和细节表现。
变换器架构优化：采用旋转位置嵌入（RoPE）和查询键归一化（QKNormalization）等技术，提升视频生成的时空一致性和鲁棒性。
在实验部分，研究团队使用ADAM-W优化器对模型进行了训练，并在多个视频生成任务上进行了评估。通过与现有模型（如MovieGen和CogVideoX）进行对比，LTX-Video在生成质量和速度上均表现优异。具体实验步骤包括：
训练数据集的构建：使用公开数据和许可材料，确保多样性和丰富性。
人类评估：通过人类调查评估生成视频的质量，参与者根据视觉质量、运动一致性和提示符合度进行评分。
消融实验：对不同模型架构和损失函数的影响进行了分析，验证了重建GAN和整体性去噪策略的有效性。
结果分析：LTX-Video在文本到视频和图像到视频任务中均显示出显著的优越性，尤其是在高运动场景下，生成的细节更加清晰。
LTX-Video的工作原理可以简单理解为将视频生成过程中的多个步骤整合在一起，以提高效率和质量。首先，模型使用一种特殊的压缩方法将视频数据转化为更小的表示，这样可以更快地处理。接着，模型的解码器不仅负责将这些压缩的数据转换回可视的图像，还会在这个过程中进行去噪，确保生成的画面清晰、细节丰富。此外，模型通过共享去噪的目标，让不同部分协同工作，从而生成更自然、更连贯的视频。最后，LTX-Video还引入了一些新技术来提升生成效果，比如通过特定的数学方法优化视频的时间和空间表现，使得生成的内容更加生动、真实。这一切都使得LTX-Video成为一个快速且高质量的视频生成工具。

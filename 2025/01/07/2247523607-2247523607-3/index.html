2025-01-01｜SJTU,TencentYoutuLab,ZJU｜⭐️🟡
http ://arxiv.org/abs/2501.00880v1https ://github.com/sjtuplayer/IAR
在视觉生成领域，利用大型语言模型（LLMs）进行图像生成的研究逐渐成为热点。然而，现有方法往往仅将LLM架构直接迁移到视觉生成上，未能深入探讨语言与视觉之间的根本差异。这种忽视可能导致视觉生成能力的次优利用。本文提出了一种改进的自回归视觉生成方法IAR（ImprovedAutoRegressiveVisualGeneration），旨在通过探索视觉嵌入空间的特性，提升生成质量和训练效率。我们的研究发现，视觉嵌入之间的相关性能够显著提高生成结果的稳定性和鲁棒性。通过引入平衡的K均值聚类算法重新安排视觉代码本，并设计了一种聚类导向的交叉熵损失函数，我们的IAR方法在多个实验中展现了卓越的性能，尤其是在训练效率和生成质量上。
本研究的核心方法包括两个主要部分：代码本重新排列和聚类导向交叉熵损失。具体步骤如下：
代码本重新排列：使用平衡的K均值聚类算法，将视觉代码本中的嵌入重新组织为相似特征的聚类。每个聚类内的嵌入具有高相似性，确保生成的图像在内容上相似。
聚类导向交叉熵损失：在训练过程中，模型首先量化图像补丁并使用重新排列的代码本。然后，计算每个聚类的概率分布，通过聚类导向的交叉熵损失引导模型预测正确的聚类索引。这种方法使得即使模型预测了错误的具体标记，该标记仍有较高的概率落在正确的聚类中，从而保证生成图像的高质量。
我们在多种参数规模的LlamaGen模型上进行了广泛的实验，以验证所提出方法的有效性。实验结果表明，IAR在训练效率和生成质量上均显著优于传统的LlamaGen模型。例如，在相同的FID值下，IAR减少了50%的训练时间。我们还对比了不同模型大小和图像标记的生成效果，结果显示IAR在不同情况下均能保持较高的生成质量。此外，通过消融实验，我们探讨了代码本重新排列和聚类导向交叉熵损失的独立贡献，进一步验证了各个模块的有效性。
在我们的研究中，我们提出了一种新方法来改进图像生成。想象一下，我们有很多图像，每张图像都被转化为一组数字，这些数字就像图像的“指纹”。我们的目标是让计算机能够更好地理解这些“指纹”，从而生成新的图像。为此，我们首先使用一种聪明的方式将这些指纹分成几组，每组里的指纹都很相似。然后，在计算机学习的时候，我们不仅告诉它应该生成哪个指纹，还告诉它这个指纹属于哪一组。这样，即使计算机偶尔猜错了具体的指纹，只要它能猜对组别，它生成的图像仍然会与目标图像相似。通过这种方法，我们的计算机生成的图像质量更高，训练速度也更快。
2024-12-31｜CMU,Yale,BytedanceSeed｜⭐️🟡
http ://arxiv.org/abs/2501.00289v1https ://zijieli-jlee.github.io/dualdiff.github.io/
本文提出了一种名为双重扩散变换器（D-DiT）的全新多模态扩散模型，旨在同时处理图像生成和文本理解任务。当前的多模态生成模型大多依赖于自回归机制，无法有效地进行图像到文本（I2T）和文本到图像（T2I）的双向生成。D-DiT通过引入一个统一的框架，结合了图像和文本的条件分布，并通过一个单一的损失函数进行联合训练，从而实现了更高效的多模态学习。该模型的灵活性使其能够在图像生成、图像描述和视觉问答等多种任务中表现出色，并在性能上与现有的多模态模型相媲美。
D-DiT模型采用双分支结构，分别处理图像和文本的扩散过程。其主要方法步骤如下：
连续扩散与离散扩散：图像分支使用连续扩散模型，而文本分支则采用离散掩蔽扩散模型。这种设计允许模型在处理图像和文本时具有不同的扩散策略。
联合训练目标：模型通过一个简单的联合损失函数来同时训练图像和文本的条件分布。这一损失函数结合了两种扩散模型的优点，允许在不同模态间进行有效的信息传递。
流匹配方法：在训练过程中，采用流匹配技术来优化模型的速度预测，从而提高生成质量。通过这种方式，模型能够更好地捕捉到图像和文本之间的复杂关系。
在实验部分，D-DiT模型在多个多模态任务上进行了评估，包括图像生成、视觉描述和视觉问答。实验结果显示，D-DiT在这些任务上的表现优于现有的多模态模型，尤其是在图像生成和文本理解的能力上。具体来说，模型在VQAv2、VizWiz等视觉问答基准测试中取得了显著的准确率，表明其在理解和生成图像与文本方面的强大能力。此外，通过与其他最新模型的比较，D-DiT展现出更高的灵活性和更好的生成质量，证明了其在多模态生成领域的潜力。
在D-DiT模型的设计中，我们采用了两个不同的部分来处理图像和文本。想象一下，一个部分专注于处理图像，而另一个部分则专注于处理文本。当我们给模型一个图像时，图像部分会开始工作，生成与之相关的文本；而当我们给它一段文本时，文本部分则会生成相关的图像。这种双重工作机制使得模型能够更好地理解图像和文本之间的联系。为了让这两个部分能够高效地合作，我们使用了一种叫做“联合训练”的方法，这样模型在学习时可以同时提升处理图像和文本的能力。通过这样的设计，D-DiT模型在生成图片和理解文字方面表现得更加出色，能够更准确地回答问题或生成描述。
2025-01-02｜HUST｜⭐️🟡
http ://arxiv.org/abs/2501.01423v1https ://github.com/hustvl/LightningDiT
在这项研究中，我们探讨了潜在扩散模型（LatentDiffusionModels,LDMs）中的优化困境，特别是在重建和生成性能之间的权衡。尽管采用Transformer架构的潜在扩散模型能够生成高质量图像，但研究表明，增加视觉标记器中的每个标记特征维度可以提高重建质量，但这会导致生成性能显著降低。现有的解决方案往往是妥协，导致生成图像出现视觉伪影或训练过程不收敛。为了解决这一问题，我们提出了一种新的方法，通过在训练视觉标记器时，将潜在空间与预训练的视觉基础模型对齐，从而显著改善潜在扩散模型的重建和生成性能。

对齐损失（VFLoss）设计：该损失函数由两个部分组成：边际余弦相似性损失和边际距离矩阵相似性损失。前者确保特征之间的相似性，后者则关注特征的相对分布。
自适应加权机制：为了确保不同损失函数在优化过程中的平衡，我们根据梯度的比例动态调整各部分的权重，确保训练过程的稳定性。
高维潜在空间训练：通过使用视觉基础模型的特征，优化高维潜在空间的学习，增强生成能力的同时保持重建能力。
我们在ImageNet数据集上进行了系列实验，评估了不同标记器配置下的重建和生成性能。实验结果显示，使用VA-VAE的LightningDiT系统在64个训练周期内达到了2.11的FID分数，相较于原始DiT实现了超过21倍的收敛速度提升。此外，系统在生成性能上也达到了最先进水平（FID为1.35），表明我们的方法有效地解决了重建与生成之间的优化困境。
在这项研究中，我们提出了一种新方法来解决图像生成中的一个重要问题。简单来说，生成高质量图像的过程就像是画画，而我们使用的工具（标记器）越复杂，画出的细节就越丰富，但同时也会让整个过程变得更慢，甚至可能画不好。为了让我们的画既好看又快，我们引入了一种新的训练方式，叫做“对齐损失”。这就像是在学习画画时，参考了一些大师的作品，帮助我们更好地理解如何使用工具。通过这种方式，我们不仅能提高画的质量，还能在更短的时间内完成画作。最终，我们的实验表明，这种方法的效果非常好，能够生成更清晰、更细致的图像。
2024-12-31｜PKU,HuaweiNoah’sArkLab｜⭐️🟡
http ://arxiv.org/abs/2501.00603v1https ://github.com/YuchuanTian/DiC
在本文中，研究者们提出了一种新的扩展卷积扩散模型DiC（DiffusionCNN），该模型以简单但高效的3x3卷积为基础，旨在提升生成任务的速度和性能。传统的扩散模型多依赖于复杂的自注意力机制，导致推理速度缓慢。DiC通过重新设计网络架构，采用编码-解码的Hourglass结构，并引入稀疏跳跃连接，显著提高了模型的可扩展性和生成质量。此外，研究者们还在条件嵌入和中间块注入等方面进行了改进，使得DiC在多种设置下表现出色，超越了现有的扩散变换器模型，并保持了良好的速度优势。
DiC的设计方法主要包括以下几个方面：
基础组件选择：选择标准的3x3卷积作为模型的核心组件，利用其在现代深度学习框架中的高效硬件优化。
架构设计：采用编码-解码Hourglass结构，以更好地扩展3x3卷积的感受野，并通过稀疏跳跃连接减少冗余，提高信息流动效率。
条件嵌入改进：引入阶段特定的条件嵌入，使每个阶段使用独立的嵌入表，增强模型对不同条件的适应能力。
中间块条件注入：在每个卷积块的中间层注入条件信息，以提高生成质量。
激活函数优化：替换传统的激活函数为GELU，以提升模型的表现。
研究者们在多个规模和设置下对DiC进行了实验，以验证其性能。实验结果显示，DiC在生成图像的FID（FréchetInceptionDistance）和IS（InceptionScore）指标上均优于现有的扩散变换器模型。同时，DiC在推理速度上也展现出显著的优势，适合实时和大规模应用。通过对比不同模型架构的性能，研究者们发现稀疏跳跃连接和阶段特定嵌入的结合显著提升了生成效果，表明这些方法在扩展卷积模型中的有效性。
在DiC模型中，研究者们通过使用简单的3x3卷积来代替复杂的自注意力机制，来加快生成图像的速度。首先，他们设计了一种新的网络结构，像一个沙漏，能够更好地处理信息。然后，他们引入了稀疏跳跃连接，这就像在不同层之间建立了“快捷通道”，让信息流动更加顺畅。此外，为了让模型更好地理解不同的输入条件，他们为每个阶段设计了专门的条件嵌入，就像为每个阶段准备了不同的“说明书”。最后，他们还优化了模型内部的激活函数，使得生成的图像质量更高。通过这些方法，DiC不仅生成的图像质量更好，而且速度也更快，适合用于需要快速反应的应用场景。
2024-12-30｜Oxford,ZhejiangLab｜⭐️
http ://arxiv.org/abs/2501.00124v1
扩散模型（DiffusionModels,DMs）在生成高质量和多样化图像方面取得了显著成功。然而，它们在计算资源和生成速度方面的高需求限制了其在普通消费者设备和资源受限环境中的应用。本文提出了一种新颖的后训练量化方法（PQD），旨在通过时间感知的优化框架提高扩散模型的推理效率。PQD通过直接将全精度模型量化为8位或4位模型，保持了在无训练情况下的性能，尤其是在无条件图像生成任务中表现出色。此外，PQD还首次成功应用于512x512的文本引导图像生成，展示了其广泛的适用性和兼容性。
本文方法主要包括以下几个步骤：
扩散模型概述：扩散模型通过马尔可夫链生成图像，包含前向过程（逐步添加噪声）和反向过程（逐步去噪）。反向过程是关键，因为它通过条件分布从噪声中恢复图像。
后训练量化（PTQ）：PTQ技术允许在不重新训练的情况下进行模型量化，适合数据有限的场景。该过程包括确定量化操作、收集校准数据和确定量化参数。
PQD方法：PQD通过选择适当的校准数据集，确保其分布与真实数据相似，从而优化量化过程。特别地，PQD在多时间步的动态激活过程中进行校准，以提高量化精度。
高分辨率图像生成的扩展：为了解决高分辨率生成的计算挑战，PQD在潜在特征上应用量化，而不是直接在输入特征上进行，从而实现高效的低位推理。
实验部分主要分为无条件生成和文本到图像生成两个方面。首先，在无条件生成实验中，使用预训练的DDIM采样器对ImageNet进行图像合成，结果显示PQD在生成质量上显著优于传统方法。其次，在文本引导图像生成实验中，使用MS-COCO数据集进行评估，PQD展示了在高分辨率生成中的优势，能够在8位量化下保持较高的图像质量和较低的FID评分。实验结果表明，PQD不仅在性能上接近全精度模型，而且在生成质量上具有竞争力。
在这项研究中，作者们提出了一种新方法来提高扩散模型的效率，称为PQD。简单来说，PQD是一种量化技术，它可以在不重新训练模型的情况下，将复杂的、高精度的模型转换为更小、更快的版本。这个过程就像把一张高分辨率的照片压缩成一个小文件，同时尽量保留照片的细节和质量。方法的关键在于选择合适的数据来帮助模型理解如何压缩，而不是让模型自己去学习，这样可以节省大量时间和资源。此外，PQD特别适合处理大图像，比如512x512的图像，确保即使在较低的位数下，生成的图像依然清晰、真实。通过这种方式，PQD不仅提升了图像生成的速度，也保持了较高的质量，使得更多人可以在普通设备上使用这些强大的生成模型。
2024-12-30｜PKU｜⭐️
http ://arxiv.org/abs/2501.00116v1
近年来，文本到图像生成（Text-to-ImageSynthesis）吸引了广泛的关注，尤其是在人工智能生成内容（AIGC）日益流行的背景下。尽管扩散模型和自回归模型在这一领域展现出了优异的表现，并逐渐取代了生成对抗网络（GAN），但它们在推理速度和训练成本上仍面临一些挑战。为了解决这些问题，本文提出了TIGER，一种基于预训练表示的文本到图像生成GAN。TIGER结合了视觉增强的判别器和高容量生成器，旨在提高模型的性能和生成速度。该模型通过吸收预训练视觉模型的复杂场景理解能力和领域泛化能力，增强了对复杂场景的处理能力，并通过高容量生成器实现高效的文本图像融合。
TIGER的架构主要包括两个创新组件：视觉增强的判别器和高容量生成器。具体而言：
视觉增强的判别器：该判别器由多个子判别器组成，每个子判别器从不同的预训练视觉模型中获取表示。通过集成多种表示，增强了模型对复杂场景的理解能力和泛化能力。
高容量生成器：该生成器采用多个新颖的高容量融合块（HFBlock），每个HFBlock包含多个深度融合模块和一个全局融合模块。深度融合模块主要负责跨模态的文本图像融合，而全局融合模块则增强了模型在空间和通道维度上的表达能力。
此外，TIGER还引入了语义对比损失，以促进生成图像与文本描述之间的语义一致性，从而提高生成图像的质量。
在实验过程中，TIGER在多个标准和零-shot文本到图像合成任务上进行了评估。实验结果表明，TIGER在COCO和CUB这两个挑战性数据集上都达到了最先进的性能，FID（FréchetInceptionDistance）分别为5.48和9.38。此外，在零-shot任务中，TIGER以更少的模型参数和更小的训练数据集，取得了与其他大型模型相当的表现，并且推理速度比现有的扩散模型快120倍。通过一系列定量和定性实验，TIGER的优越性得到了充分验证，尤其是在生成图像的真实感和文本匹配度方面。
在TIGER的设计中，我们主要使用了两个重要的技术来提高文本到图像生成的效果。首先是视觉增强的判别器，可以理解为一个“评审团”，它由多个小评审组成，每个评审都来自不同的“学校”（预训练视觉模型）。这些小评审共同努力，帮助TIGER更好地理解复杂的图像场景，从而提高生成的图像质量。
其次是高容量生成器，它就像一个“艺术家”，能够将文本描述和随机噪声结合起来，创造出新的图像。这个艺术家使用了多种工具（融合块），其中一些工具专注于细节（深度融合模块），而另一些则帮助它在整体上把握图像的风格（全局融合模块）。通过这些技术，TIGER不仅能生成更真实的图像，还能确保这些图像与输入的文本描述相符。

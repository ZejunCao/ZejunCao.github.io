2024-12-29｜ANU
http ://arxiv.org/abs/2412.20470v1
在计算机视觉领域，3D人类身体的生成建模是一个备受关注的研究方向。现有方法在设计紧凑的潜在表示方面面临挑战，难以同时满足表达能力与语义可解释性。本文提出了一种新的生成框架JADE（Joint-awareLatentDiffusion），通过细致控制人类形状的变化，推动了3D人类生成建模的发展。JADE的核心是一个联合感知的潜在表示，该表示将人体分解为骨骼结构（通过关节位置建模）和局部表面几何特征（通过附加在每个关节的特征描述）。这种解耦设计使得几何和语义信息的解释成为可能，从而为用户提供更灵活的控制能力。此外，JADE采用了级联的扩散管道，通过两次扩散分别建模骨骼结构和局部表面几何的分布，展现了在多项任务中相较于现有方法的有效性。
JADE的实现主要包括以下几个关键部分：
联合感知潜在表示：将人体编码分散到各个关节，形成一个由关节标记序列组成的整体模型。每个关节标记进一步分为外部参数（编码骨骼结构）和内部特征（建模局部表面几何）。
自编码器架构：采用Transformer架构的自编码器进行潜在表示学习，首先将3D人类点云转化为稀疏的关节标记集合。接着，通过混合网络增强标记之间的上下文信息，生成联合感知的潜在表示。
级联扩散管道：通过两个扩散模型，分别学习外部参数和内部特征的分布。第一个扩散模型生成骨骼结构的信息，第二个则在此基础上生成局部几何细节。该设计允许生成的每个表面点同时获取全局骨骼信息和局部结构样式，从而实现更高质量的形状生成。
本文在多个公开数据集上进行了广泛的实验，以评估JADE在不同任务中的表现，包括人类形状重建、编辑和生成。实验结果表明，JADE在重建精度、编辑可控性和生成质量方面均优于现有方法。具体而言，在DFAUST和SPRING数据集上，JADE展示了更低的每个顶点位置误差（MPVPE），并且在生成多样性和真实感的评估中表现出色。此外，JADE能够通过插值操作实现对人体形状的灵活编辑，证明了其在实际应用中的潜力。
在JADE的工作中，研究者们设计了一种新方法来生成和编辑3D人类模型。首先，他们将一个人类的形状分解成多个部分，每个部分对应一个关节。这样做的好处是，我们可以更容易地控制每个关节的运动和形状。接着，他们使用一种叫做自编码器的技术来学习如何将这些关节的信息转化为一个整体的人类形状。最后，他们使用一种叫做扩散模型的方法，逐步生成这些形状，确保每个生成的部分都与整体协调一致。通过这种方式，JADE不仅能够生成高质量的人类形状，还能让用户方便地进行各种编辑，比如改变姿势或外观。
2024-12-30｜CAS,UCAS,Google｜AAAI2025
http ://arxiv.org/abs/2412.20657v1https ://iscas3dv.github.io/DiffGrasp/
本研究提出了DiffGrasp，一个创新的框架，旨在生成高质量的全身人类抓取动作序列，特别是针对物体运动的指导。随着动画、虚拟现实（VR/AR）和机器人等领域的发展，生成自然且逼真的人类与物体交互的动态序列变得尤为重要。然而，如何在复杂的物体形状、大小和运动轨迹下，确保双手的协调动作和抓取的真实感，依然是一个难题。DiffGrasp通过将身体、手部和给定物体的运动序列共同建模，使用单一的扩散模型来解决这一问题。该方法引入了新颖的接触感知损失和数据驱动的指导策略，实验结果显示其在生成逼真抓取序列方面优于现有的最先进方法。
DiffGrasp的核心在于其条件扩散模型，主要包括以下几个部分：
数据表示：
人类动作表示：使用SMPL-X模型表示人类全身姿势，包含全局位移和关节信息。
物体运动表示：通过全球3D运动序列和几何信息来表示物体的运动和形状。
条件输入：将物体形状、运动和人类身份映射为特征向量，供模型使用。
条件扩散模型：
模型架构：采用Transformer架构进行序列到序列的生成任务，分为条件编码器和去噪声网络。
去噪声输出：模型学习输入条件与生成的全身运动序列之间的潜在关系。
接触感知损失：
引入接触标签和两种接触感知损失，以增强模型对手部与物体接触的感知能力，确保生成的抓取姿势稳定且自然。
指导策略：
采用数据驱动的指导策略来优化生成结果，确保手部与物体之间的稳定接触，减少穿透现象并增强视觉真实感。
在实验部分，DiffGrasp使用GRAB和ARCTIC两个数据集进行评估。每个数据集包含多个受试者与不同物体的交互序列。通过与基线方法OMOMO的比较，DiffGrasp在多个评估指标上表现出色，包括手部关节位置误差、接触度和碰撞深度等。实验结果表明，DiffGrasp能够生成更自然的抓取动作，并且在手部与物体接触的连续性和精确性方面优于其他方法。此外，消融实验验证了接触感知损失和指导策略对模型性能的提升作用。
DiffGrasp的工作原理可以想象成一个智能机器人在学习如何用手抓东西。首先，它需要知道人类的身体是如何运动的，包括手和身体的每个部分是如何协调的。然后，它还需要了解物体的形状和运动方式。通过使用一种叫做“扩散模型”的技术，DiffGrasp能够逐步学习如何将这些信息结合起来，生成自然的抓取动作。在这个过程中，DiffGrasp引入了一些“指导”，就像教练一样，确保机器人的手不会滑动或穿透物体，保持稳定的抓取。这种方法使得生成的抓取动作不仅看起来真实，而且感觉也很自然。
2024-12-28｜THU,ShanghaiQiZhiInstitute,ShanghaiAILab
http ://arxiv.org/abs/2412.20104v1https ://syncdiff.github.io/
在日常生活中，人类经常与物体进行复杂的交互，这种多体人-物交互（HOI）在虚拟现实（VR）、增强现实（AR）和人类动画中具有重要意义。本文提出了一种新方法SyncDiff，旨在合成多体人-物交互的运动。SyncDiff通过同步运动扩散策略，能够处理任意数量的人、手和刚性物体，克服了传统方法在多体运动同步中的挑战。其核心创新包括两个新的多体运动同步机制：对齐分数和相应的对齐损失函数L，以及推理中的显式同步策略。这些机制帮助SyncDiff在多种交互场景中实现真实、协调和物理上合理的运动合成。
SyncDiff的设计基于两个关键见解。首先，采用单一扩散模型来捕捉多体运动的联合分布，但仅依赖数据样本分数的模型通常不足以保证个体运动之间的对齐。为此，SyncDiff引入了一组新的对齐分数，以指导模型合成同步的运动。其次，在推理阶段，通过联合优化样本和对齐的似然性，SyncDiff将多体HOI合成视为一个运动同步问题，构建在一个图模型上，其中节点表示个体运动，边表示两体之间的相对运动。具体方法包括：
因子化运动表示：将个体运动和相对运动分解为低频和高频成分。
隐式联合建模：通过单一扩散模型同时建模所有个体和相对运动。
显式同步策略：在推理过程中，结合样本和对齐分数进行全局同步，以实现最大似然推断。
为了验证SyncDiff的有效性，研究者在四个不同的多体交互数据集上进行了广泛的实验，分别为TACO、CORE4D、GRAB和OAKINK2。这些数据集涵盖了不同数量的人、手和物体的交互场景。实验结果显示，SyncDiff在物理准确性和动作语义方面均优于现有的多体运动合成方法。具体评估指标包括接触表面比率、插入体积和动作识别准确率等，结果表明SyncDiff在各类设置中都能实现更高的运动协调性和真实感。此外，通过消融实验，研究者验证了各个关键设计对模型性能的影响，进一步强调了频率域运动分解和显式同步的必要性。
SyncDiff的工作原理可以想象成一个精密的乐队指挥。首先，乐队中的每个乐器（每个人或物体）都有自己的乐谱（运动轨迹），而SyncDiff就像是一个指挥，确保所有乐器在合适的时机演奏，以产生和谐的音乐（协调的运动）。为了实现这一点，SyncDiff会将每个乐器的音符（运动）分解为高音和低音部分（低频和高频运动），这样可以确保即使是细微的音符（小幅度运动）也不会被大的音符（大幅度运动）淹没。指挥在每个乐章的开始（推理时）都会检查每个乐器的音符是否对齐（同步），确保它们在一起演奏时不会产生冲突。通过这种方式，SyncDiff能够合成出自然流畅的人与物之间的互动运动。
2024-12-30｜NTHU｜AAAI2025
http ://arxiv.org/abs/2412.20767v1
在计算机视觉领域，利用稀疏的二维图像重建高质量的三维模型引起了广泛关注。本文提出了一种新颖的框架——KeyGS，旨在提高单目图像序列的相机姿态估计和新视图合成的效率。与传统的神经辐射场（NeRF）和三维高斯喷溅（3DGS）方法相比，KeyGS通过结合结构光束法（SfM）与3DGS，显著降低了训练时间并提高了重建精度。该方法通过快速获取粗略相机姿态并利用频率感知的密集化技术进行联合优化，解决了现有方法在复杂环境下的局限性，如噪声、低分辨率和光照变化等问题。
KeyGS方法的核心在于其高效的相机姿态估计与重建流程，主要包括以下几个步骤：
粗略相机姿态获取：采用COLMAP的顺序模式快速获取初步相机姿态，显著降低计算复杂度。
频率感知的密集化：通过引入一种粗到细的频率感知密集化策略，优化相机姿态的估计，防止在高频信号下出现过度分裂的问题。
联合优化：在获得初步姿态后，利用3DGS进行场景重建，同时对相机姿态进行细化，确保重建的准确性和真实感。
高斯分布建模：使用显式的三维高斯分布表示场景，通过体积渲染技术将颜色信息从高斯体投影到图像平面，提升渲染效率。
为了验证KeyGS的有效性，本文与多种现有的联合优化模型（如BARF、CF3DGS和Nope-NeRF）进行了比较。实验使用TanksandTemples和CO3DV2数据集，评估了重建质量和相机姿态的准确性。结果表明，KeyGS在PSNR、SSIM和LPIPS等指标上均优于其他方法，且训练时间显著缩短，从数小时降至约10分钟。此外，通过消融实验，进一步确认了频率感知密集化和联合优化的关键作用，展现了该方法在实际应用中的潜力。
KeyGS的方法可以理解为一个高效的相机和场景重建工具。首先，它通过快速的相机姿态估计来获取拍摄场景的初步位置，就像用一个简单的地图来找到方向。接着，KeyGS会利用一种聪明的技术来逐步提高这个地图的精确度，确保在不同光线和环境下都能得到准确的结果。最后，它通过将三维场景用高斯模型表示，快速生成高质量的图像，像是在制作一幅精美的画作。这个过程不仅快速，而且能在复杂的环境中保持高质量的效果，适合在实时应用中使用。
2024-12-29｜ShanghaiAILab,BUAA,UTokyo
http ://arxiv.org/abs/2412.20522v1https ://github.com/kaikai23/MaskGaussian
本研究提出了一种名为MaskGaussian的新方法，旨在通过概率掩膜实现自适应的三维高斯表示。这种方法主要解决了传统的三维高斯点云在实时渲染中存在的高内存消耗问题。尽管3D高斯点云（3DGS）在新视图合成和实时渲染方面表现出色，但其对内存的需求限制了其在虚拟和增强现实等实时场景中的应用。MaskGaussian通过将高斯视为概率实体，而非固定的点，动态调整其存在概率，从而优化了渲染效果和内存使用。通过引入掩膜光栅化技术，未被采样的高斯也能参与梯度更新，使得模型能够更好地适应场景的动态变化。
MaskGaussian的核心方法包括以下几个步骤：
高斯建模：将每个高斯视为具有中心、透明度、尺度和旋转属性的概率实体，而不是静态的点。
掩膜生成：为每个高斯分配可学习的掩膜分数，并使用Gumbel-Softmax技术从掩膜分布中采样，以指示高斯的存在与否。
光栅化处理：在渲染过程中，所有高斯都被标准化处理，掩膜在颜色累积和透射衰减过程中应用。这确保了即使高斯未被采样，它们仍能接收梯度更新。
动态调整：通过掩膜光栅化技术，动态评估高斯的贡献，调整其存在概率，从而使高斯在场景演变中保持更新和优化。
在实验部分，MaskGaussian在三个真实世界数据集上进行了评估，包括Mip-NeRF360、Tanks&Temples和DeepBlending。通过与传统方法（如3DGS和Compact3DGS）的比较，MaskGaussian在保持图像质量的同时，实现了62.4%、67.7%和75.3%的高斯点数削减，且渲染速度分别提高了2.05倍、2.19倍和3.16倍。实验结果表明，MaskGaussian在渲染质量和计算效率方面均优于现有的高斯处理方法，尤其是在处理小而透明对象时，能够有效保留细节。
在MaskGaussian方法中，我们将三维高斯看作是概率性存在的点，而不是固定不变的点。首先，我们为每个高斯点生成一个掩膜，这个掩膜就像是一个开关，指示这个点在渲染时是否参与。接着，在渲染图像时，我们会把所有高斯点都计算一遍，但只有那些被掩膜允许的点才能真正影响最终图像。这样，即使某些高斯点没有被直接使用，它们仍然可以通过掩膜接收反馈，从而在后续的渲染中调整自己是否继续存在。这种方法使得我们的系统在动态变化的场景中更加灵活，并且能有效减少内存使用，同时保持图像的高质量。
2024-12-30｜FDU,USurrey,Oxford｜ICLR2024
http ://arxiv.org/abs/2412.20720v1
动态场景的建模和从捕获视频中生成新视角图像是增强现实（AR）、虚拟现实（VR）和元宇宙应用所需的沉浸式体验的关键。然而，由于复杂的非约束现实场景及其时间动态，这一任务面临诸多挑战。本文提出了一种新的方法，将动态场景视为一个时空四维体积学习问题，采用了最小的运动假设，提出了一种称为四维高斯溅射（4DGS）的框架。该框架通过一系列四维高斯原语来明确表示目标动态场景，能够捕捉空间和时间中的相关信息，并支持高分辨率、光真实感的新视角实时渲染。通过大量实验验证了4DGS在视觉质量和效率方面的优越性，适用于多种动态场景相关任务。
本研究提出的四维高斯溅射（4DGS）方法包含以下几个关键步骤：
三维高斯溅射基础：首先回顾了三维高斯溅射（3DGS）的方法，为动态场景建模奠定基础。
四维高斯表示：动态场景中的每个像素不仅由空间坐标（u，v）标识，还需引入时间戳（t）。因此，4D高斯的均值和协方差矩阵被扩展为四维格式，以便更好地捕捉动态信息。
渲染过程：采用可微分的高斯溅射技术，通过对每个像素的颜色和透明度进行加权计算，生成最终图像。该过程通过将条件三维高斯与一维边际高斯结合，实现了高效的渲染。
优化框架：在训练过程中，采用渲染损失作为监督信号，进行端到端的优化，确保模型能够有效处理动态场景的变化。
为验证4DGS的有效性，作者进行了大量实验，涵盖不同的动态场景和任务，包括新视角合成和场景理解。实验使用了多个数据集，如PlenopticVideo数据集和D-NeRF数据集，评估了模型在视觉质量（如PSNR、SSIM）和处理速度方面的表现。结果显示，4DGS在动态场景合成中优于现有方法，尤其在复杂的动态区域表现出色。此外，作者还进行了消融实验，以分析不同组件对整体性能的影响，进一步验证了模型的鲁棒性和灵活性。
在这项研究中，科学家们提出了一种新的方法来处理动态视频场景。想象一下，我们想从一个视频中提取出一个场景的三维信息，并且能够在不同的时间点和视角中查看它。这个方法的核心是使用一种叫做“四维高斯”的数学工具，它不仅考虑空间（上下左右前后），还加入了时间的因素。通过这种方式，科学家们可以更准确地捕捉到场景的动态变化。
具体来说，研究人员首先使用已有的三维高斯模型来表示静态场景，然后扩展到四维，以便同时处理时间信息。在渲染图像时，他们通过计算每个像素的颜色和透明度，结合空间和时间的信息，生成最终的图像。整个过程通过优化算法进行训练，确保模型能够快速而准确地处理各种动态场景。这种方法在实际应用中表现优异，尤其是在需要实时渲染的增强现实和虚拟现实场景中。

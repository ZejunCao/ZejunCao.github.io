2025-01-02｜Unknown｜⭐️
http ://arxiv.org/abs/2501.01116v1
在图像合成中，通常需要将一个图像中的前景对象提取出来，并将其粘贴到另一个图像的背景中。这一过程可能会导致前景和背景之间在颜色和光照上出现不一致，进而使合成图像显得不自然。为了解决这个问题，图像和谐算法（IHA）被提出，以调整前景对象的外观，使其更好地与背景匹配。然而，现有的图像质量评估（IQA）方法往往无法准确反映人类的视觉偏好，因此急需一种有效的图像质量评估方法。本文提出了第一个针对图像和谐的质量评估数据库（HarmonyIQAD），包含1350张由9种不同IHA生成的和谐图像及其对应的人类视觉偏好评分，并基于此数据库提出了HarmonyIQA模型，旨在更好地预测人类对和谐图像的偏好。
HarmonyIQA的设计基于大规模多模态模型，采用了指令调优和低秩适配（LoRA）技术。主要方法步骤如下：
数据收集：构建HarmonyIQAD数据库，包含1350张和谐图像和28350个主观质量评分，图像内容多样，涵盖不同的前景对象和颜色风格。
模型架构：模型输入包括和谐图像和用户提示，首先提取图像和文本特征，然后通过一个可训练的投影器将视觉特征映射到语言空间，生成视觉标记。
特征融合：将用户提示的文本标记与视觉标记结合，输入到预训练的大规模语言模型（LLM），生成最后的隐藏状态。
双阶段训练：第一阶段使用交叉熵损失进行训练，以生成格式正确的文本和初步的质量评分；第二阶段采用均方误差（MSE）损失，进一步优化质量评分的准确性。
在实验部分，首先将HarmonyIQAD数据集按4:1的比例划分为训练集和测试集，评估HarmonyIQA在和谐图像质量评估中的表现。通过与20种传统和学习基础的IQA方法进行比较，HarmonyIQA在多个指标上均表现出色。此外，还进行了消融实验，评估LoRA适配和投影器训练的有效性，结果表明这些技术显著提升了模型性能。最后，HarmonyIQA在多个其他IQA基准数据集上进行评估，均取得了优异成绩，验证了其作为一种通用图像质量评估方法的潜力。
在这项研究中，HarmonyIQA的工作原理可以简单理解为一个“评分系统”，它帮助我们判断合成图像的质量。首先，研究人员收集了大量的图像样本，并请专业人士对这些图像的和谐程度进行评分。接着，他们设计了一个模型，这个模型能够从图像中提取出重要的特征，并结合用户的描述来生成一个质量评分。这个过程分为两个阶段：第一阶段是让模型学会生成正确的文本描述，第二阶段则是让模型通过这些描述来预测图像的质量分数。通过这样的方式，HarmonyIQA不仅能够判断图像的和谐程度，还能在更广泛的图像质量评估中发挥作用。
2025-01-01｜PolyU,OPPO,ByteDance,HKUST｜⭐️
http ://arxiv.org/abs/2501.00740v1https ://github.com/leeruibin/RORem
在图像处理领域，物体移除是一项重要的任务，旨在将用户指定的物体从图像中去除并用真实的背景填充。然而，现有的物体移除方法面临着许多挑战，如不完全移除、合成内容错误和合成区域模糊等问题。这些问题主要源于缺乏高质量的配对训练数据以及自监督训练范式的局限性。为了解决这些问题，本文提出了一种半监督学习策略，结合人类反馈来生成高质量的配对训练数据，最终训练出一个名为“鲁棒物体移除模型”（RORem）。通过收集60K个训练样本并利用人类反馈选择高质量样本，本文的方法显著提高了物体移除的成功率，达到了超过18%的提升。
本研究的核心方法包括以下几个步骤：
初始模型训练：首先，从两个开源数据集中收集60K个训练三元组，包括原始图像、编辑后的图像以及对应的掩码。利用这些数据训练一个基于稳定扩散的图像修复模型。
人类注释与反馈：将初步训练的模型应用于测试集，邀请人类标注者选择高质量的物体移除样本。这些样本随后被添加到训练集中，以增强数据的质量和多样性。
自动化标注：通过训练一个判别器，利用人类反馈数据来自动标注高质量的训练样本。通过多轮迭代人类和自动注释阶段，最终构建了一个包含超过200K个物体移除样本的数据集。
模型微调与蒸馏：利用收集到的数据集对预训练的稳定扩散模型进行微调，并引入可训练的LoRA层以提高推理效率。最终，模型能够在四个扩散步骤内完成物体移除任务。
实验部分主要评估了RORem模型的性能。通过与其他先进的物体移除方法进行定量和定性比较，结果表明RORem在多个指标上表现优异。具体而言，RORem的成功率在512×512和1024×1024分辨率下均超过了其他方法，显示出其在物体移除任务中的强大能力。此外，模型的推理速度也得到了显著提升，经过蒸馏后的模型在处理时间上大幅减少，达到了每张图像平均0.50秒的处理速度。实验结果表明，RORem在图像质量和处理效率上均优于现有的物体移除技术。
在这项研究中，我们开发了一种新的方法来帮助人们从照片中去掉不需要的物体。首先，我们收集了大量的照片和它们的编辑版本，也就是去掉物体后的样子。然后，我们请一些人来检查我们生成的去除效果，挑选出好的样本。通过这种方式，我们不断改进我们的数据集，使其更丰富、更高质量。接下来，我们训练了一个智能模型，让它学习如何更好地去掉这些物体。为了让这个模型运行得更快，我们还采取了一些技术手段，确保它能在短时间内完成任务。最终，我们的模型不仅能更准确地去掉物体，还能在更短的时间内完成这个工作。这意味着无论是在拍照、广告还是电影制作中，使用我们的技术都能得到更好的效果。
2025-01-02｜FDU｜AAAI2025｜⭐️
http ://arxiv.org/abs/2501.01125v1https ://github.com/Maplebb/DuMo
在当今的文本到图像（T2I）生成模型中，生成的不当内容（如色情或侵犯版权的图像）引发了广泛的安全担忧。为了应对这些问题，研究者们提出了多种方法来消除不适当的概念，但这些方法往往会对模型生成其他概念的能力产生负面影响。为了解决这一挑战，本文提出了双编码器调制网络（DuMo），旨在在尽量减少对非目标概念生成能力的损害的同时，精准地消除不当目标概念。DuMo通过引入EraserwithPriorKnowledge（EPR）模块，专注于修改U-Net的跳跃连接特征，从而实现对图像细节（高频成分）的概念消除。通过保持主干网络参数不变，DuMo在多种任务上表现出色，显著提高了模型的生成能力与安全性。
DuMo的核心在于两个主要组件：EraserwithPriorKnowledge（EPR）和时间-层调制过程（TLMO）。具体方法如下：
EPR模块：
该模块专注于跳跃连接特征的调整，而不改变主干网络的参数，从而保持非目标概念的结构完整性。
在EPR中，利用原始跳跃连接特征提供的隐式语义信息来指导目标概念的消除，确保生成能力的保留。
TLMO过程：
该过程通过引入时间特定和层特定的调制因子，动态调整EPR模块输出的消除效果。
TLMO的设计使得在不同的时间步和层级上能够自动平衡消除效果与生成能力，从而优化模型的性能。
通过这两项技术的结合，DuMo能够有效地消除不当概念，同时最大限度地保留其他非目标概念的生成能力。
为验证DuMo的有效性，研究者在三项任务上进行了广泛的实验：显式内容消除、卡通概念移除和艺术风格消除。实验使用了多个基准方法进行比较，结果显示DuMo在各项任务中均表现出色。具体结果如下：
显式内容消除：DuMo成功识别并消除了34个裸体部位，显示出最低的裸体部位数量。
卡通概念移除：在移除卡通角色（如“Snoopy”）的实验中，DuMo显著优于其他方法，保持了非目标角色的生成能力。
艺术风格消除：在去除特定艺术家风格（如“VanGogh”）的任务中，DuMo不仅成功消除了目标风格，还保留了图像的整体结构和细节。
整体来看，DuMo在消除不当内容的同时，显著提升了模型的生成能力，展示了其在安全AI领域的潜力。
DuMo的方法可以简单理解为一个“清理工具”，专门用来去掉图片中的不当内容，比如裸体或其他不合适的元素。这个工具有两个主要部分：
EPR模块：想象一下，EPR就像一个精细的画笔，它只针对那些需要去掉的细节，而不碰触其他部分。这意味着，即使我们在清理图片，其他的好看部分也能保持完好。
TLMO过程：这个过程就像是调音师，根据不同的情况调整EPR的效果。比如，在不同的时间点和不同的层次上，它会决定应该去掉多少内容，确保清理工作既有效又不会影响到其他的美观部分。
通过这两个部分的合作，DuMo能够在确保图片安全的同时，尽量保留原本的美感和细节。
2025-01-01｜MSU｜⭐️
http ://arxiv.org/abs/2501.00962v1
在现代的文本到图像生成模型（T2I）中，生成的图像往往反映出关于文化和职业等概念的视觉偏见和刻板印象。现有的刻板印象量化方法多基于统计平衡，未能与社会学对刻板印象的定义相匹配，因此常常错误地将偏见分类为刻板印象。为了解决这一问题，本文提出了一种新的定量测量刻板印象的方法，称为OASIS（开放集刻板印象评估工具），旨在量化T2I模型中生成数据集的刻板印象，并探讨其来源。OASIS包括两个主要的刻板印象评分指标，以及两种方法来理解刻板印象在T2I模型中的起源。尽管图像生成的质量有了显著提升，研究显示，诸如FLUX.1和SDv3等新型T2I模型依然存在强烈的刻板印象倾向，生成的图像中普遍包含刻板印象属性。
OASIS的核心是通过两个主要指标来量化刻板印象：
刻板印象评分（M1）：该评分用于衡量生成图像数据集中刻板属性的分布性违规，具体来说，它对比了生成数据集中刻板属性的分布与真实世界中该属性的分布。
加权对齐评分（M2）：该评分用于测量生成图像在特定刻板属性上的光谱变化，评估图像在该属性上的多样性。
此外，OASIS还包含两种方法来探讨刻板印象的来源：
通过优化提示发现刻板属性（StOP，U1）：该方法旨在识别T2I模型内部与特定概念相关联的刻板属性。
刻板印象传播指数（SPI，U2）：该方法量化在图像生成过程中，刻板属性在潜在空间中的聚合程度。
本研究通过OASIS对三种开放权重的T2I模型（SDv2、SDv3和FLUX.1）进行了比较分析。首先，为每种模型生成了2000张关于不同国籍的图像。接着，依据生成图像的数据集，使用大型语言模型（LLM）生成刻板印象候选集和描述。实验结果表明，尽管新模型在图像生成的保真度上有所提高，但在某些情况下，仍然存在较强的刻板印象倾向，尤其是对于网络足迹较低的国籍，生成的图像中刻板属性的数量更为显著。此外，OASIS的结果显示，刻板印象的形成往往在图像生成的早期步骤中就开始显现，这表明T2I模型在处理特定概念时容易内化刻板印象。
OASIS是一种新工具，帮助我们理解和测量文本生成图像模型（T2I）中存在的刻板印象。简单来说，它通过两个主要指标来评估生成的图像是否存在刻板印象。第一个指标看生成的图像中某些属性（如“男人”或“穿着传统服装”）出现的频率是否过高，是否与现实中的比例相符。第二个指标则关注这些图像的多样性，看看是否只是重复生成相似的图像。除了评估这些刻板印象外，OASIS还帮助我们找出这些刻板印象是如何在模型内部形成的，比如它是如何将某些特定的属性与某些概念（比如国籍）联系在一起的。通过这些方法，研究者们可以更好地理解和改进这些生成模型，减少它们在生成图像时可能带来的偏见。
2024-12-31｜HUST,SkyworkAI｜⭐️
http ://arxiv.org/abs/2501.00463v1
随着人工智能生成图像的普及，如何有效地进行水印嵌入以保护知识产权和识别虚假内容变得尤为重要。现有的基于训练的方法在不同提示下的泛化能力较差，并且常常会产生明显的伪影。为了解决这个问题，本文提出了一种可证明具有良好泛化能力的图像水印方法，称为自增强训练的潜在扩散模型（SAT-LDM）。该方法通过引入自由生成分布来对齐训练和测试阶段，从而增强水印模块的泛化能力。理论上，我们证明了自由生成分布在不需要收集新数据的情况下，有助于提高泛化界限。大量实验结果表明，SAT-LDM在保持水印的稳健性同时，显著提升了水印图像的质量。我们的目标是为保护高保真度的AI生成内容提供一种实用且方便的解决方案。
SAT-LDM方法的核心在于自增强训练机制，它通过以下几个步骤来提升水印的有效性和泛化能力：
自由生成分布的引入：与传统方法依赖外部数据集不同，SAT-LDM利用内部生成的自由生成分布来训练水印模块。这种分布更能反映扩散模型的自然工作条件。
训练和测试阶段的对齐：通过在训练阶段使用自由生成分布，SAT-LDM确保水印模块能够在多种提示下有效泛化，避免了因使用偏见数据集而导致的性能下降。
理论分析：我们提供了自增强训练方法的泛化界限分析，证明了自由生成分布可以显著减少训练和测试阶段之间的分布差异，从而提高模型的泛化能力。
损失函数设计：引入了结合均方误差（MSE）和感知损失（LPIPS）的复合损失函数，以优化水印图像的质量和水印的稳健性。
在实验部分，我们使用了多个数据集来评估SAT-LDM的性能，特别是在水印图像质量和水印稳健性方面。实验步骤包括：
数据集选择与生成：选择StableDiffusion模型生成512×512的图像，并使用GPT-4生成多样化的提示，确保测试的全面性。

稳健性评估：通过对水印图像施加不同类型的攻击（如高斯模糊、亮度变化、JPEG压缩等），评估水印的稳健性。
实验结果分析：结果显示，SAT-LDM在保持高质量水印图像的同时，具有显著的稳健性和较强的泛化能力，适合于实际应用场景。
SAT-LDM的方法可以简单理解为一种智能的“水印”技术，它使用了一种特殊的训练方式，让水印能够在不同的图像生成情况下保持不变。首先，它不需要依赖大量的外部数据，而是利用自己生成的数据来进行训练，这样可以更好地适应各种不同的图像生成情况。其次，在训练过程中，SAT-LDM确保每次生成的图像都能保持高质量，同时又能有效嵌入水印。最后，通过精心设计的损失函数，SAT-LDM能够在保证图像质量的同时，使嵌入的水印更加稳健，不容易被攻击或修改。这种方法不仅提高了水印的隐蔽性和耐用性，还能适应更多的使用场景，确保生成的图像在不同情况下都能保持其版权信息的有效性。

267-A1.4｜第一人称空间视频的感知质量评估，立体图像内容质量与偏好评估；图像的符号解耦表示；安全（概念擦除，AI生图检测）

AIGCResearch
主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）发布日期｜2024年12月31日｜周二
2024-12-29｜SJTU,UNantes｜⭐️🟡
http ://arxiv.org/abs/2412.20423v1
随着扩展现实（XR）技术的快速发展，第一人称视角的空间视频（egocentricspatialvideos）在提供沉浸式和互动体验方面的潜力逐渐显现。然而，针对这类视频的质量评估研究仍然相对缺乏。本文提出了首个专注于第一人称空间视频的感知质量评估数据库（ESVQAD），包含600个视频及其对应的主观评分（MOS）。我们还提出了一种新颖的多维双目特征融合模型（ESVQAnet），该模型结合了空间、运动和语义特征，以准确预测视频的整体感知质量。实验结果表明，ESVQAnet在评估第一人称空间视频的感知质量任务中显著优于16种最先进的无参考视频质量评估（VQA）模型。
本研究的核心是提出的ESVQAnet模型，具体方法包括以下几个步骤：
特征提取器：
空间特征提取器：采用多阶段的视觉状态空间双重性（VSSD）模块和多头自注意力（MSA）模块，逐层提取双目视图中的空间特征。
运动特征提取器：利用3D卷积神经网络（3D-CNN）捕捉视频中的运动动态和失真。
语义特征提取器：通过级联视觉变换器层（visiontransformerlayers）提取视频的高层语义信息。
特征融合：
将提取的空间、运动和语义特征通过多层感知器（MLP）进行融合，以生成最终的质量评分。
模型训练与评估：
使用ESVQAD数据库进行模型训练，采用主观评分标准评估模型的性能，确保模型在多种视频质量评估任务中的泛化能力。
实验部分包括对所提出的模型进行全面的性能评估。我们与16种最先进的VQA模型进行比较，使用多种评估指标（如Spearman等级相关系数和均方根误差）来量化模型的效果。实验结果显示，ESVQAnet在左视、右视和融合视图的评估中均表现出色，尤其在融合视图评估中，模型的表现优于所有对比模型。此外，进行的消融实验验证了各特征提取器和模型组件对最终性能的贡献，进一步证明了多维特征融合的有效性。
在这项研究中，我们开发了一种新方法来评估第一人称空间视频的质量。首先，我们使用不同的技术从视频中提取三种主要特征：空间特征（视频中的静态画面）、运动特征（视频中的动态变化）和语义特征（视频传达的意义）。接着，我们将这些特征结合在一起，形成一个完整的质量评分。这个过程就像把不同的乐器声合成一首和谐的音乐。最后，我们通过与其他已有的方法进行对比，证明了我们的方法在评估视频质量方面的优势。这种方法不仅能帮助我们更好地理解视频的质量，还能为未来的研究提供基础。
2024-12-30｜Apple,WIS,MIT｜⭐️🟡
http ://arxiv.org/abs/2412.21127v1
随着虚拟现实（VR）技术的快速发展，评估立体图像的质量变得至关重要，以确保用户获得沉浸式和舒适的三维体验。然而，现有的立体图像质量评估方法往往只关注单一方面，例如视觉不适或图像质量，且缺乏足够的数据支持。为了解决这些问题，研究团队提出了SCOPE（立体内容偏好评估）数据集，涵盖了多种常见的感知失真和伪影，并通过VR头显收集了用户的偏好注释。此外，研究团队开发了iSQoE（沉浸式立体质量评估模型），旨在更全面地捕捉用户的视觉体验，并与现有方法进行比较，展示其在评估立体图像质量方面的优势。
本研究的核心方法包括以下几个步骤：
数据集构建：通过对立体图像进行多种失真处理，创建了2400个样本的SCOPE数据集。失真类型包括噪声注入、光度和空间变换等。
用户偏好收集：采用双选强迫选择（2AFC）测试，参与者在VR环境中比较不同失真的立体图像，选择他们更喜欢的版本。每个图像配有5个注释，以确保数据的可靠性。
模型训练：使用收集到的偏好数据训练iSQoE模型。该模型通过对立体图像对的特征进行处理，计算出一个综合质量评分，涵盖图像质量、舒适度和深度感知等多方面。
模型架构：iSQoE模型使用修改过的DINOv2网络，结合左、右图像的交叉注意力机制，确保信息在两个视图之间的有效流动。
在实验部分，研究团队对iSQoE模型进行了多项评估：
性能比较：将iSQoE模型与其他现有的立体图像质量评估方法进行了比较，结果显示iSQoE在准确性和与用户偏好的匹配度上均优于其他模型。
失真响应评估：测试模型在面对未见过的失真时的表现，发现iSQoE能够有效地外推并保持相对一致的评分。
不同观看媒介的影响：通过在不同设备（如VR头显和2D屏幕）上进行用户偏好测试，验证了VR环境下对立体图像偏好的独特性。
实际应用评估：将模型应用于评估市面上现有的单图转立体生成技术，结果显示iSQoE能够准确反映用户的偏好。
在这项研究中，研究人员想要找到一种更好的方法来评估立体图像的质量。他们首先创建了一个数据集，这个数据集中包含了许多经过不同处理的立体图像，像是加入噪声或改变颜色等。接着，他们让参与者在虚拟现实的环境中比较这些图像，看看哪个更好，收集了大量的偏好数据。然后，研究人员使用这些数据训练了一个新的模型，叫做iSQoE，这个模型可以综合考虑图像的质量和用户的舒适度，给出一个评分。最后，他们通过实验发现，这个新模型在评估立体图像时，比以前的方法更准确，能够更好地反映人们的真实感受。
2024-12-25｜MIPT,AIRI,FRCCS&CRAS｜⭐️🟡
http ://arxiv.org/abs/2412.19847v1
在本研究中，作者提出了一种新颖的架构，称为ArSyD（ArchitectureforSymbolicDisentanglement），用于实现图像的符号解耦表示。解耦表示的核心思想是将数据简化为生成因素的集合，每个因素对应一个向量。通过这种表示，研究者可以通过调整特定因素的值来修改对象的属性。然而，传统的高维向量表示使得确定每个坐标对应的生成因素变得困难。ArSyD通过使用超维计算的原理，构建了一种新的符号解耦表示，使得每个生成因素也以相同维度的向量表示，从而简化了这一过程。该方法在dSprites和CLEVR数据集上进行了广泛的分析，并引入了新的解耦度量标准，以便于比较不同方法的表现。
ArSyD的设计基于以下几个关键步骤：
对象表示：通过超维计算的原则，将每个生成因素表示为一个高维向量，这些向量的维度与最终对象表示的维度相同。
生成因素投影：使用编码器网络提取对象的中间表示，并通过生成因素投影模块获取生成因素的值。
特征交换模块：在目标对象和捐赠对象之间交换特征向量，以实现对象属性的修改。
弱监督学习：模型通过重建目标和捐赠对象的图像来学习解耦表示，而不需要额外的假设或标签信息。
评估解耦效果：采用新的解耦度量标准（如DMM和DCM），这些标准能够评估模型在不同类型的潜在表示中的解耦效果。
在实验部分，研究者使用了dSprites和CLEVR数据集来验证ArSyD的有效性。实验设计包括以下几个方面：
数据集生成：从dSprites和CLEVR生成成对图像，确保每对图像在一个生成因素上存在差异。
模型训练：使用AdamW优化器对ArSyD进行训练，通过最小化重建损失来优化模型性能。
性能评估：通过FID和IoU等指标评估图像重建的质量，并与其他解耦表示方法（如BetaVAE和FactorVAE）进行比较。
结果分析：展示了特征交换的效果，验证了模型在单一对象和多对象场景中的表现。
在ArSyD的工作中，研究者希望通过一种新的方法来更好地理解和修改图像中的对象。想象一下，你有一个玩具模型，它的颜色、形状和大小都是可以单独调整的。ArSyD就像是为这个模型设计的一个控制面板，每个旋钮对应一个特定的属性。研究者使用一种特殊的数学方法，确保这些旋钮之间不会互相干扰，这样你只需调整一个旋钮，就能改变模型的一个属性，而不会影响其他属性。在训练过程中，模型学习如何从图像中提取这些属性，并能够在不同的图像之间交换这些属性，从而实现更灵活的图像编辑。这种方法不仅可以帮助我们更好地理解图像数据，还能在需要时快速修改图像的特定部分。
2024-12-29｜USTC,ElizaLabs,NTU,BUPT,A*STAR,AlibabaTongyiLab,TeleAI,BUAA｜⭐️
http ://arxiv.org/abs/2412.20413v2
在大型文本到图像（T2I）扩散模型中，去除不需要的概念，同时保持生成质量，是一个尚未解决的挑战。随着StableDiffusion（SD）v3和Flux等新兴范式的出现，现有的概念去除技术的可转移性受到限制。为此，本文提出了EraseAnything，这是首个专门针对流式T2I框架内的概念去除方法。该方法将概念去除视为双层优化问题，采用LoRA（低秩适应）参数调优和注意力图正则化来选择性地抑制不必要的激活。此外，本文还引入了自对比学习策略，确保去除不需要的概念不会对无关概念的性能产生负面影响。实验结果表明，EraseAnything在多种概念去除任务中取得了最先进的性能，成功填补了新T2I范式下的研究空白。
EraseAnything的方法主要包括以下几个关键步骤：
双层优化（Bi-levelOptimization）：将概念去除和无关概念的保留视为相互依存的任务。下层优化专注于去除特定概念，而上层优化则确保无关概念的生成质量不受影响。
LoRA参数调优：通过对流式模型的LoRA进行微调，调整与目标概念相关的激活，确保去除过程的有效性。
注意力图正则化：利用注意力图的索引信息，选择性地抑制与待去除概念相关的激活，从而实现精准的内容去除。
自对比学习策略：通过生成与无关概念相关的样本，构建自对比损失，确保去除目标概念不会对生成其他概念产生负面影响。
在实验部分，作者对EraseAnything进行了全面评估，涵盖了从具体到抽象的多种任务，例如：体育、建筑和艺术风格等。实验使用Flux.1模型，并通过FID和CLIP等指标对生成图像进行评估。特别是在去除裸体概念的任务中，EraseAnything显示出显著的效果，生成的图像在保持生成质量的同时，有效减少了不当内容的出现。此外，用户研究表明，EraseAnything在多个维度上均表现出色，用户对其整体性能给予了高评价。通过与现有方法的比较，EraseAnything在去除效率和无关概念保留方面均展现出优越性。
EraseAnything的方法可以简单理解为一个双层的“去除与保留”策略。首先，我们设定一个目标概念，比如“裸体”，并希望在生成图像时将其去除。为了实现这一点，我们使用一种叫做LoRA的技术来微调模型，使其在生成图像时不再关注这个概念。同时，我们还要确保其他无关的概念，比如“女孩”，能够正常生成。为此，我们采用了一种注意力图的正则化方法，这就像在图像生成过程中给模型提供一个“指南”，告诉它哪些部分可以忽略。最后，我们还引入了自对比学习的方法，通过生成与目标概念不相关的样本，帮助模型更好地理解如何去除目标概念而不影响其他内容。这样一来，我们就能够有效地去掉不想要的概念，同时保持生成的图像质量。
2024-12-30｜LGAIResearch｜⭐️
http ://arxiv.org/abs/2412.20704v1
随着生成性人工智能技术的迅速发展，潜在扩散模型（LDM）在生成高质量图像方面取得了显著成就。然而，这也引发了对AI生成图像的检测和潜在负面影响的担忧。现有的检测方法通常依赖于训练样本，而在实际应用中，获取相关的真实和AI生成图像数据往往非常困难。为了解决这一问题，本文提出了一种名为高频影响（HFI）的新方法，旨在无需训练数据的情况下检测LDM生成的图像。HFI通过分析重构图像中的高频信息失真程度来实现检测，能够有效识别不同生成模型的图像，并且在各种具有挑战性的基准测试中表现出色。
HFI方法的核心在于利用LDM的自编码器作为下采样和上采样的内核，测量重构图像中高频信息的失真程度。具体步骤包括：
高频成分提取：通过自编码器重构输入图像，并计算输入图像与重构图像之间的重构距离。
失真度量：通过高频成分的影响来评估重构图像的失真程度，利用低通滤波器对图像进行处理，以减少背景信息的影响。
无训练检测：HFI设计为在测试阶段直接应用，无需先前的训练数据，从而提高了检测的灵活性和效率。
集成策略：当多个自编码器可用时，HFI可以通过集成不同自编码器的结果来提高检测精度。
在实验部分，研究者们通过多个基准数据集评估了HFI的性能，主要包括自然图像和人脸图像数据集。实验分为两大场景：首先在交叉自编码器设置下比较HFI与现有基线方法的表现；其次，进行广泛的消融研究以优化HFI的参数设置。结果表明，HFI在检测AI生成图像方面的表现优于其他训练无关的方法，并且在许多情况下与训练基础方法相媲美。此外，HFI在处理不同生成模型的图像时，展现了良好的鲁棒性和高效性。
HFI方法的工作原理可以简单理解为一种“高频侦探”。想象一下，当我们拍摄一张照片时，照片中的细节（如人的眼睛、衣服的纹理等）就是高频信息。HFI通过比较原始图像和通过自编码器重建的图像，来检查这些细节是否被扭曲或失真。具体来说，它会查看这些细节在重建过程中是否被损坏，从而判断这张图像是否由AI生成。此外，HFI不需要任何先前的数据训练，就可以直接对新图像进行检测，这使得它在实际应用中非常灵活和高效。通过这种方式，HFI能够有效地区分真实图像和AI生成的图像，帮助我们更好地理解和管理生成性AI的影响。

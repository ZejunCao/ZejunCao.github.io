267-A1.2｜平衡内容真实感与艺术风格，增强视觉美学，野外现实世界图像编辑；直接优化扩散过程的端到端训练框架

主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）发布日期｜2024年12月31日｜周二
2024-12-25｜ReichmanU,Microsoft｜⭐️⭐️
http ://arxiv.org/abs/2412.19853v1https ://nadavc220.github.io/conditional-balance.github.io/
在图像生成领域，平衡内容真实感与艺术风格是一个重要挑战。传统的风格迁移方法和现代的去噪扩散概率模型（DDPMs）在实现这一平衡方面往往面临困难，可能导致风格、内容或两者的牺牲。为了解决这一问题，研究者们提出了一种新方法，通过分析DDPM的注意力层的敏感性，识别与不同风格特征相关的特定层。该方法允许在推理过程中仅将条件输入应用于这些敏感层，从而更好地平衡风格与内容，减少生成图像中的伪影，并提升艺术创作的自由度。研究结果表明，通过选择高敏感度的风格和结构层，可以实现更好的颜色和纹理控制，进而改善生成图像的质量。
本研究采用了一种新颖的分析方法，旨在识别DDPM注意力层在图像生成过程中的敏感性。具体方法如下：
图像集合创建：选择多个不同艺术风格，并为每种风格生成一组图像，确保每组图像在其他方面（如主题）保持一致。
层敏感性分析：提取每个层在不同时间步的自注意力特征，并计算每层的聚类得分，以评估其对特定艺术风格的敏感度。
条件输入选择：根据聚类得分，选择最敏感的K层来施加条件输入，而不是对所有层施加条件。这种选择可以更有效地平衡多重条件，并减少生成图像中的干扰。
控制参数设定：通过设定两个插值参数（λS和λT），用户可以灵活调整风格与内容的平衡比例，从而实现更稳定和创造性的图像生成。
为了验证所提出方法的有效性，研究者进行了多轮实验。首先，比较了不同条件组合下生成图像的质量，特别关注风格与内容之间的平衡。实验结果显示，使用所选敏感层进行条件注入的图像在风格一致性和内容保留方面表现显著优于随机选择层的图像。此外，研究还扩展了评估集，涵盖了更多艺术风格和文本提示，以测试方法在复杂条件下的稳健性。最终，用户研究表明，使用平衡方法生成的图像在视觉质量上更受欢迎，显示出显著的偏好。
在这项研究中，科学家们想要解决一个问题：如何在生成图像时同时保持内容的真实感和艺术风格的美感。为了做到这一点，他们首先创建了一组不同风格的图像，比如印象派或立体主义。接着，他们分析了图像生成模型的不同部分，找出哪些部分对风格和内容的影响最大。然后，他们决定只在这些关键部分应用风格和内容的条件，而不是在所有部分都施加条件，这样可以更好地平衡两者。最后，他们通过实验和用户反馈发现，这种方法确实能够生成更高质量的图像，让人们更满意。
2024-12-30｜ByteDanceInc,USTC｜⭐️🟡
http ://arxiv.org/abs/2412.20800v1https ://vmix-diffusion.github.io/VMix/
在文本到图像生成领域，尽管扩散模型展现出卓越的能力，但在生成高美学图像方面仍存在不足。尤其是在颜色、光照和构图等细微维度上，生成的图像与真实世界的美学图像之间仍存在差距。为了解决这个问题，本文提出了一种新的适配器——交叉注意力值混合控制（VMix），旨在提升生成图像的质量，同时保持对视觉概念的普适性。VMix通过将输入文本提示分解为内容描述和美学描述，从而增强文本的保真度和视觉美感。此外，该方法通过值混合交叉注意力将美学条件整合到去噪过程中，以实现更好的图像生成效果。通过一系列实验，VMix在文本保真度和视觉美学方面超越了其他最先进的方法，并且能够与社区模型（如LoRA、ControlNet等）兼容，显示出其广泛的应用潜力。
VMix的核心在于其创新的条件控制方法，主要包括以下几个步骤：
输入文本提示的解耦：将输入文本分为内容描述和美学描述，以便更好地指导模型生成符合人类偏好的图像。
美学嵌入初始化：构建一个美学嵌入模块，将美学标签通过CLIP模型处理成嵌入。这一过程仅在训练开始时进行，确保模型在后续训练中能够有效利用这些美学信息。
交叉注意力混合控制：在U-Net架构中引入交叉注意力模块，分别处理内容特征和美学特征。通过共享相同的注意力图，确保模型在增强美学表现的同时，不会损害文本与图像的对齐能力。
训练与推理阶段：在训练阶段，冻结基础模型的参数，仅训练新添加的美学嵌入和交叉注意力模块。在推理阶段，结合用户输入的美学条件，生成高质量的图像。
为验证VMix的有效性，研究团队进行了广泛的实验，包括定量和定性评估。实验设置包括使用不同的基准数据集（如MJHQ-30K和LAION-HQ10K），并采用多种评估指标（如FID、CLIP分数和美学分数）进行性能评估。结果显示，VMix在美学质量、文本保真度和生成图像的整体表现上均优于现有的最先进方法。此外，研究还进行了消融实验，分析了不同组件对模型性能的影响，进一步验证了VMix的有效性和灵活性。
在VMix的方法中，我们首先把输入的文本信息分成两部分：一部分是描述图像内容的，比如“一个穿白色裙子的金发女孩”，另一部分是描述图像美感的，比如“自然光、和谐的色彩”。这样做的目的是让模型更清楚地理解我们想要什么样的图像。接下来，我们使用一个叫做美学嵌入的模块，将这些美感描述转化为模型可以理解的格式。然后，我们在模型中加入一个交叉注意力的机制，这样模型就能在生成图像时同时考虑内容和美感，而不会互相干扰。通过这种方式，VMix不仅可以生成更美的图像，还能确保生成的图像与我们输入的文本信息相符。在训练和推理的过程中，VMix展现出了极大的灵活性，可以与许多现有的图像生成模型兼容使用。
2024-12-30｜HKUST,AntGroup,Stanford,CUHK｜⭐️🟡
http ://arxiv.org/abs/2412.21079v2https ://ezioby.github.io/edicho/
在计算机视觉和图像处理领域，实现跨不同图像的一致性编辑是一项重要且具有挑战性的任务。Edicho是一种新提出的方法，旨在解决这一问题，特别是在多样化的真实世界图像中。该方法通过利用显式的图像对应关系，实现了对图像的统一编辑，确保了编辑效果的一致性。Edicho的核心设计思想是采用无训练的扩散模型，并引入了显式对应关系的概念，以指导图像的编辑过程。该方法在多个应用场景中表现出色，包括产品照片的美化、活动主题的统一风格以及内容创作者的多图像风格一致性。
Edicho方法的实现分为几个主要步骤：
显式对应关系提取：首先，通过预训练的对应关系提取器，从输入图像中提取显式的语义对应关系。这一过程确保了后续编辑的准确性。
自注意力机制增强：在图像编辑中，Edicho通过增强自注意力机制，借用源图像的特征来指导目标图像的编辑。这一过程称为“对应注意力”（Corr-Attention），通过显式的对应关系来提高编辑的一致性。
分类器无关引导（CFG）：在去噪过程中，Edicho还对分类器无关引导（CFG）进行了改进，以确保在处理噪声潜在空间时，编辑效果的连贯性和高质量。通过将显式对应信息引入CFG，Edicho能够在多个图像间实现更高的一致性。
为了验证Edicho方法的有效性，研究者们进行了广泛的实验。实验设置包括使用StableDiffusion作为基础模型，并与多种现有图像编辑方法进行比较。在定量评估中，Edicho在文本对齐和编辑一致性方面均表现优异，尤其在本地和全局编辑任务中都取得了最佳成绩。此外，用户研究也显示，Edicho在实际应用中得到了用户的高度评价。实验结果表明，Edicho能够有效地保持输入图像的完整性，同时在多种编辑任务中实现一致的视觉效果。
Edicho方法的核心在于如何确保在多个图像中进行一致的编辑。首先，我们从输入图像中提取出相互之间的关系，这就像找到了不同图像之间的“连接点”。接着，Edicho利用这些连接点来指导图像的编辑，确保每个图像的编辑效果是相互协调的，而不是单独处理后可能出现的不一致。最后，Edicho还优化了去噪过程，使得在处理图像时，不仅保持了高质量的输出，还能确保不同图像间的风格和元素都能和谐统一。这样一来，无论是用于产品展示还是活动主题的照片，Edicho都能帮助用户轻松实现想要的效果。
2024-12-30｜FDU,ShanghaiAcademyofArtificialIntelligenceforScience｜⭐️
http ://arxiv.org/abs/2412.21044v1
在生成模型领域，扩散模型作为一种强大的生成框架，已经取得了显著的成功。然而，它们面临着一些固有的挑战，包括训练与采样过程之间的差距、逐步加噪过程中的信息泄露，以及在训练中无法有效整合高级损失函数（如感知损失和对抗损失）。为了解决这些问题，本文提出了一种创新的端到端训练框架，旨在通过直接优化最终重构输出，来对齐训练和采样过程。该方法通过将训练过程视为从纯噪声到目标数据分布的直接映射，消除了训练与采样之间的差距，并实现了感知损失和对抗损失的有效整合。通过在COCO30K和HW30K等基准数据集上的广泛实验，结果表明，该方法在生成质量和一致性方面优于传统的扩散模型，展现了端到端训练在提升扩散生成模型方面的潜力。
本文提出的端到端训练框架的核心思想是直接优化生成过程，以实现从纯高斯噪声到重构目标数据的转化。具体方法如下：
背景：扩散模型通过逐步添加高斯噪声到数据样本中，定义了前向过程，并通过反向过程重构干净数据。
训练目标：与传统方法不同，我们的训练目标是优化最终重构结果，而不是在每个时间步预测噪声。我们通过单一损失函数（重构损失）来直接优化重构的潜在表示。
损失函数的整合：该方法支持感知损失和对抗损失的整合，以增强生成输出的质量和一致性。此外，采用指数移动平均（EMA）来稳定模型参数的更新，从而提高训练的稳定性和模型性能。
在实验部分，我们首先设置了多个基准数据集，包括COCO30K和HW30K，以评估所提方法的性能。实验评估指标包括FrechetInceptionDistance（FID）和CLIP得分。我们与多种先进的文本到图像生成模型（如SDXL、PixArt-δ等）进行了比较，结果显示，所提方法在图像质量和语义一致性上均表现出色。此外，进行了一系列消融研究，以分析不同损失函数组合对模型性能的影响。结果表明，结合感知损失和对抗损失的组合能够显著提升生成图像的细节和真实感。最终，实验结果验证了我们提出的端到端训练框架在生成质量和效率上的优势。
在这项研究中，我们提出了一种新的训练方法，旨在提高生成图像的质量。传统的生成模型通常在训练和生成阶段使用不同的方式，这导致了生成的图像可能不够理想。我们的创新之处在于，我们通过一种叫做“端到端”的方法，让模型在训练时就直接关注最终生成的图像，而不是在每个步骤中逐步调整。这就像在学习绘画时，不是每次都只画一部分，而是一次性完成整幅画。我们还引入了一些新的损失函数来帮助模型更好地理解图像的细节和与描述的关系。这种方法经过多次实验验证，结果显示，我们生成的图像质量更高，细节更丰富，能够更准确地反映输入的描述。

2024-12-29｜HPC-AITech｜⭐️⭐️
http ://arxiv.org/abs/2412.20404v1https ://github.com/hpcaitech/Open-Sora
Open-Sora是一个开源视频生成模型，旨在为所有人提供高效的视频制作工具。该模型利用先进的深度学习架构，支持多种视觉生成任务，包括文本到图像、文本到视频以及图像到视频的生成。Open-Sora能够生成高保真度的视频内容，最长可达15秒，分辨率可达720p，并支持任意宽高比。通过引入空间-时间扩散变换器（STDiT），Open-Sora有效地解耦了空间和时间注意力机制。此外，模型还采用了一种高压缩的3D自编码器，以加快训练过程并提高表示的紧凑性。通过这一开创性工作，Open-Sora旨在推动AI内容创作的创新、创造力和包容性，所有资源均可在其GitHub页面上公开获取。
Open-Sora的构建方法主要包括以下几个方面：
数据来源与处理：
利用多个开放数据集（如Webvid-10M、Panda-70M等）生成30M个视频剪辑，确保模型训练的可重复性。
数据预处理采用完整的管道，包括场景切割、视频过滤和文本生成，以确保高质量的视频文本对。
模型架构：
使用3D自编码器对视频进行压缩，减少计算负担，并通过文本编码器处理文本输入。
采用空间-时间注意力机制，使得模型在生成视频时能够同时考虑空间和时间信息。
训练策略：
采用多阶段训练策略，分为三个阶段以优化训练效率。
通过动态分辨率和宽高比的支持，模型能够适应不同的视频生成任务，提升生成质量。
在实验部分，Open-Sora通过多个阶段的训练验证其有效性。第一阶段使用Webvid-10M数据集进行初步训练，重点在低分辨率视频的生成。第二阶段转向更高质量的Panda-70M数据集，过滤出高美学分数的视频进行训练。最后阶段则使用来自多个来源的高质量视频，进一步提升模型在720p和1080p分辨率下的生成能力。通过VBench评分进行的评估表明，Open-Sora在视频生成任务中表现优异，达到了开源模型中的领先水平。
Open-Sora的构建方法可以简单理解为一个三步走的过程。首先，团队从多个开放的数据源收集了大量的视频和图像，这些数据经过处理后形成高质量的训练样本。接着，他们设计了一个复杂的模型，能够同时处理视频和文本信息，确保生成的视频既美观又符合文本描述。最后，训练过程分为几个阶段，逐步提高模型的性能，使其能够生成各种分辨率和长度的视频。总的来说，Open-Sora就是通过这种系统化的方法，让每个人都能轻松制作出高质量的视频内容。
2024-12-28｜ZJU,WHUT,CJLU｜⭐️🟡
http ://arxiv.org/abs/2412.19978v1
本文提出了MAKIMA，一个无调优的多属性开放域视频编辑框架。尽管现有的基于扩散的文本到图像（T2I）模型在视频编辑方面取得了显著进展，但它们主要关注全局视频修改，且在实现特定属性的细致编辑方面依然面临挑战。MAKIMA旨在通过引入掩膜引导的注意力调制（Mask-guidedAttentionModulation），提升不同属性之间的相关性，同时抑制跨属性干扰，从而实现多属性编辑。该框架通过一致的特征传播机制，优化了视频帧生成质量与效率的平衡，显著提升了编辑精度和时间一致性，同时保持了计算效率。
MAKIMA的核心方法分为三个主要阶段：
掩膜生成：为每个待编辑属性生成相应的二进制掩膜，通过语义分割和跟踪技术来识别视频中的目标区域。
DDIM反演：对源视频进行DDIM反演，获取初始噪声和自注意力图，并缓存卷积特征。这些特征在后续的去噪过程中用于保持原视频的结构信息。
注意力调制：在去噪过程中，通过增强同一属性的注意力分数，同时抑制不同属性之间的干扰，确保编辑的精确性。具体而言，利用掩膜引导的调制策略来调整自注意力和交叉注意力层，优化属性与其对应文本标记之间的关联。
在实验部分，研究团队对MAKIMA在33个视频序列上的表现进行了评估，使用手动注释的文本描述生成99对视频-文本进行验证。通过与多种基线方法进行比较，MAKIMA在编辑质量和时间一致性方面表现出色，尤其在CLIP-T和Frame-Acc评估指标上超越了其他方法。此外，用户研究显示，MAKIMA在文本一致性、时间连贯性和结构保留等方面均得到了参与者的高度评价，进一步验证了其有效性。
在MAKIMA的工作中，我们首先为想要编辑的视频中的每个特定部分生成一个“掩膜”，就像在画画时用遮罩保护不想涂色的区域。接下来，我们使用一种叫做DDIM的技术来提取视频的基本特征和信息，这些特征将帮助我们在编辑过程中保持视频的原始结构。最后，在编辑过程中，我们会调整不同部分的注意力，使得相似属性的部分能够相互加强，而不同属性的部分则会被抑制。这就像在调音时，让某些乐器的声音更响亮，而让其他乐器的声音变得柔和，从而确保最终的编辑效果既自然又和谐。
2024-12-29｜VIVO,CUHK(SZ),NWPU,OpenNLPLab｜AAAI2025｜⭐️🟡
http ://arxiv.org/abs/2412.20378v1https ://tri-ergon.github.io/Tri-Ergon/
Tri-Ergon是一种新型的视频到音频生成（V2A）模型，旨在解决现有技术在生成高质量音频方面的局限性。当前的V2A方法通常仅依赖视觉信息，难以生成与视频内容精确同步的音频。为了解决这一问题，Tri-Ergon引入了多模态条件和细粒度的音频控制，特别是通过使用LUFS（相对于全尺度的响度单位）嵌入，赋予用户对音频响度变化的精确控制。该模型能够生成高达44.1kHz的高保真立体声，支持最长60秒的音频输出，显著优于现有的V2A技术。Tri-Ergon的创新之处在于其能够利用文本、音频和像素级的视觉提示，实现语义丰富且时序同步的音频合成，极大地拓展了其在影视、游戏和自动化视频处理等领域的应用前景。
Tri-Ergon的设计基于潜在扩散模型（latentdiffusionmodel），其核心包括以下几个方面：
多模态条件控制：模型通过结合文本、音频和视觉提示，增强音频生成的语义相关性。这种多模态条件的整合使得生成的音频不仅与视觉内容一致，还能反映出更丰富的上下文信息。
响度控制：引入LUFS嵌入，允许用户对音频的响度进行精确调整。LUFS提供了一种标准化的响度测量方法，能够在音频生成过程中动态控制响度变化，确保生成音频的质量和一致性。
可变音频长度：Tri-Ergon支持生成长度可变的音频，采用时间嵌入技术，使得生成的音频长度可以与输入视频的时长精确匹配。这一特性解决了传统模型生成固定长度音频的局限性，提供了更大的灵活性。
在实验中，Tri-Ergon模型的性能通过与现有V2A技术进行对比评估。模型在两个主要数据集上进行训练：VGGSound和新构建的MM-V2A数据集。结果显示，Tri-Ergon在音频质量和时间同步性方面均显著优于基线模型，如SpecVQGAN和Diff-Foley。具体而言，Tri-Ergon-S和Tri-Ergon-L模型在生成音频的响度控制和语义一致性上表现出色，能够生成高保真立体声，并在60秒的音频生成任务中也保持了高质量的输出。通过不同条件的组合实验，模型的灵活性和适应性得到了进一步验证，表明Tri-Ergon在多模态音频生成领域的潜在应用价值。
Tri-Ergon的方法可以简单理解为一个智能音频生成系统，它能根据视频内容自动生成相应的声音。首先，它结合了多种信息来源，包括文字描述、视频画面和已有的音频样本，这样可以更好地理解需要生成什么样的声音。其次，Tri-Ergon还引入了一种特别的响度控制技术，让用户可以精确调整声音的大小，确保生成的音频听起来更加自然。最后，Tri-Ergon可以生成不同长度的音频，确保声音和视频的时长完全匹配。这种灵活性使得Tri-Ergon在电影、游戏和其他需要音频与视频完美配合的场合中非常有用。
2024-12-26｜LiAuto,HIT,ZJU｜⭐️
http ://arxiv.org/abs/2412.19860v1
在当今数字化时代，利用音频输入生成栩栩如生的对话头像视频已成为热门研究领域。为了实现真实的对话头像生成，不仅需要准确的唇部同步，还需要灵活自然的面部和头部动态、摄像机运动以及逼真的光影效果。现有方法在这些方面的综合控制能力有限，难以满足高质量生成的需求。为此，本文提出了一种新方法UniAvatar，旨在提供对多种运动和光照条件的广泛控制。通过采用FLAME模型，我们将所有运动信息渲染至单一图像上，既保持了三维运动细节的完整性，又实现了精细的像素级控制。此外，我们的研究还设计了独立模块来管理三维运动和光照，允许单独或组合控制，经过大量实验验证，UniAvatar在运动和光照控制的广泛性上超越了其他方法。
UniAvatar的设计包括多个关键步骤，确保对运动和光照的有效控制。首先，模型接收一幅肖像图像和一段音频序列，目标是生成与音频同步的对话视频。具体方法包括：
运动控制：使用FLAME面部模型，将多个运动信号整合为单一图像，确保三维运动信息的完整性，同时通过3D运动模块实现像素级控制。
光照控制：采用光照感知渲染技术，通过一个轻量级的光照编码器独立提取光照信息，使得生成的视频在不同光照条件下保持一致性。
数据集构建：为克服现有数据集在运动和光照多样性方面的不足，特意收集了两个新数据集：DH-FaceDrasMvVid-100和DH-FaceReliVid-200，提供丰富的头部运动和多样化的光照场景。
训练策略：采用两阶段训练策略，首先生成运动视频帧，然后在第二阶段结合音频和光照信息进行细化，以确保生成视频的连贯性和一致性。
在实验部分，我们对UniAvatar的性能进行了全面评估。实验使用了多个数据集，包括HDTF、CelebV-HQ和新构建的DH-FaceDrasMvVid-100及DH-FaceReliVid-200。通过与其他现有方法的比较，我们重点评估了生成视频的质量、稳定性和多样性。结果显示，UniAvatar在多个控制条件下均表现出色，尤其在光照控制和运动稳定性方面，超越了其他方法。此外，定量评估采用了多种指标，包括峰值信噪比（PSNR）、学习感知图像补丁相似度（LPIPS）和Fréchet视频距离（FVD），均表明UniAvatar在生成真实、富有表现力的对话头像动画中的优势。
UniAvatar的方法可以简单理解为一个智能的动画生成器，它能够根据你说的话（音频）来创建一个“会说话”的头像。这个过程分为几个步骤：首先，我们给这个生成器提供一张你想要的头像照片和一段音频。然后，生成器会分析这张照片，提取出头像的各种细节，比如面部表情和头部位置。同时，它会根据音频中的语音内容来决定头像的嘴巴如何移动，确保嘴巴的动作与声音同步。此外，UniAvatar还可以调整光线效果，使得头像在不同的光照条件下都看起来真实。为了训练这个生成器，研究人员还特意收集了大量的头像视频，确保它能够在多种情况下表现良好。通过这些技术，UniAvatar能够生成看起来非常自然和生动的对话头像视频。
2024-12-28｜SZU,UExeter｜ICASSP2025｜⭐️
http ://arxiv.org/abs/2412.20148v1https ://github.com/CVI-SZU/DEGSTalk
在本研究中，我们提出了一种新颖的3D高斯散射（3DGS）方法，称为DEGSTalk，旨在合成具有长发的人物说话视频。该方法解决了现有技术在处理动态面部特征和长发保留方面的挑战。DEGSTalk利用可变形的预嵌入高斯场，通过动态调整预嵌入的高斯原语，能够以高帧率（超过100FPS）生成高质量的合成视频，同时保持面部动态和细微表情的准确捕捉。此外，我们还开发了一种动态发型保留的肖像渲染技术，以增强合成视频中长发运动的真实感。实验结果表明，DEGSTalk在合成质量和真实感方面显著优于当前的主流方法，特别是在复杂面部动态和发型保留的处理上。
DEGSTalk的核心方法包括以下几个关键步骤：
预嵌入高斯场：我们使用可变形的预嵌入高斯场来构建面部和嘴部区域的静态场。这些高斯场通过动态调整其原语参数（如位置、缩放和旋转）来适应输入的音频特征和面部表情。
动态发型保留渲染：为了准确重建长发，DEGSTalk采用了一种动态发型保留的肖像渲染技术。该技术通过空间整合头发和面部区域，确保两者的特征得到保留，避免了传统方法中常见的噪音和伪影。
训练过程：训练分为三个阶段：静态初始化、运动学习和微调。在静态初始化阶段，我们优化高斯参数以构建粗略的头部结构；在运动学习阶段，重点优化面部动态和嘴部同步；最后，在微调阶段，通过精细调整确保生成图像的现实感。
我们在多个公共视频数据集上对DEGSTalk进行了实验，使用了六个高分辨率的说话面孔视频。实验设置包括对比DEGSTalk与多种现有方法（如AD-NeRF、ER-NeRF、SyncTalk和TalkingGaussian）在图像质量、感知相似度和面部特征准确性等方面的表现。评估指标包括峰值信噪比（PSNR）、学习感知图像补丁相似度（LPIPS）、地标距离（LMD）和结构相似性指数（SSIM）。结果显示，DEGSTalk在所有评估指标上都优于其他方法，且推理速度达到114FPS，展示了其高效性和优越的合成质量。
在DEGSTalk的方法中，我们主要使用了两种技术来合成说话视频。首先，我们创建了一种叫做“可变形预嵌入高斯场”的模型，这个模型能够根据输入的音频和面部表情动态调整面部的形状和表情。就像给面部“穿衣服”一样，这种模型可以让面部的每个细节都更真实。其次，我们设计了一种新的渲染技术，专门用来处理长发。传统方法常常在头发和面部交接处出现噪音，而我们的技术则能在合成时更好地结合头发和面部，使得最终的视频看起来更自然、更流畅。这种方法让我们能够快速生成高质量的说话视频，尤其适合那些长发的角色。

2024-12-31｜POSTECH,InhaU,YonseiU｜AAAI2025｜⭐️⭐️
http ://arxiv.org/abs/2501.00645v1https ://soundbrush.github.io/
在本研究中，作者提出了一种名为SoundBrush的模型，该模型利用声音作为编辑工具来修改视觉场景。SoundBrush的创新之处在于它扩展了潜在扩散模型（LatentDiffusionModel,LDM）的生成能力，结合音频信息进行视觉场景的编辑。与传统的文本引导方法不同，SoundBrush能够根据输入的声音，精确地调整场景，甚至插入发声物体，从而实现对2D和3D视觉场景的有效操控。研究表明，声音作为一种控制信号，能够提供丰富的视觉信息，超越文本描述的局限性，特别是在捕捉场景的细微变化方面。
SoundBrush的设计分为几个关键步骤：
数据集构建：创建一个声画配对的数据集，包括合成和真实数据。合成数据集利用GPT-4生成与声音事件相关的图像对，真实数据集则从VGGSound数据集中提取音频和图像对。
声音特征提取：使用预训练的音频编码器（如CLAP）提取音频特征，并通过映射网络将这些特征转换为LDM的文本空间中的音频令牌。
条件编辑：将生成的音频令牌输入到LDM中，以实现基于音频的视觉场景编辑。该过程通过对比损失函数优化音频特征与视觉特征的对应关系，从而提高编辑的精确性和一致性。
训练与优化：模型采用联合训练的方式，通过低秩适应（LoRA）技术，优化映射网络和LDM的协同工作，以实现更有效的编辑结果。
为了验证SoundBrush的编辑能力，研究者进行了多项实验，包括：
数据集与评估指标：使用VGGSound数据集构建评估数据集，并通过音频-视觉相似性（AVS）、图像-图像相似性（IIS）和文本-视觉相似性（TVS）等指标评估模型性能。
与现有方法的比较：将SoundBrush与其他声引导视觉编辑模型（如AudioToken、GlueGen和InstructAny2Pix）进行比较，分析其在2D和3D场景编辑中的表现。
用户研究：通过人类评估，分析编辑结果的质量，特别关注音频反映的准确性和原始结构的保持程度。实验结果显示，SoundBrush在各项指标上均优于其他方法，特别是在音频-视觉相似性和图像-图像相似性方面，表现出色。
SoundBrush是一种新颖的技术，它能让我们通过声音来编辑和修改图像。想象一下，如果你听到雨声，SoundBrush可以让你在图像中看到下雨的场景，甚至可以在图像中添加雨伞或水洼。这个过程是通过一个特殊的计算机模型完成的，首先，它会从声音中提取出关键信息，然后将这些信息转化为可以编辑图像的指令。整个过程就像是给图像上色，声音就像是一支画笔，帮助我们创造出更符合我们想象的视觉效果。通过这种方式，SoundBrush不仅能处理静态的图像，还能扩展到三维场景的编辑，让我们在数字世界中更自由地表达创意。
2024-12-31｜SJTU,CUMT,UESTC,THU,HKUST(GZ)｜⭐️🟡
http ://arxiv.org/abs/2501.00375v1https ://github.com/EvelynZhang-epiclab/DaTo
本研究提出了一种名为动态感知令牌修剪（DaTo）的方法，旨在提高稳定扩散模型在文本到图像生成中的效率和图像质量。稳定扩散模型在生成能力上取得了显著的成功，但其迭代去噪机制导致了高计算成本和生成速度缓慢，限制了其广泛应用。虽然社区已经提出了多种方法来降低计算负担，如特征缓存，但简单重复使用先前计算的特征会导致特征动态性降低，从而影响生成图像的质量。为了解决这一问题，DaTo方法通过选择性地修剪低动态的令牌，使得高动态的令牌参与自注意力层的计算，从而有效延长特征在时间步之间的动态性。实验结果表明，该方法在ImageNet和COCO-30k数据集上分别实现了高达9倍和7倍的加速，同时在图像质量上也得到了提升。
DaTo方法的核心在于动态感知令牌修剪，主要包括以下几个步骤：
基础令牌选择：根据不同时间步之间的特征差异，选择具有最大动态变化的令牌作为基础令牌。这一过程通过计算相邻时间步之间的特征差异得以实现。
自注意力层优化：在自注意力层中，仅允许高动态的令牌与其他令牌进行交互，从而避免低动态令牌对生成质量的负面影响。
令牌修剪与恢复：从剩余的令牌中选择与基础令牌具有最高相似度的令牌进行修剪，并在后续计算中通过从基础令牌中复制相似的令牌来恢复被修剪的令牌。
优化策略搜索：通过进化算法（如NSGA-II）搜索最优的特征缓存和令牌修剪策略，以平衡计算延迟和生成质量，确保在不损失图像质量的情况下提升生成速度。
在实验中，我们对DaTo在多个数据集上的性能进行了评估。具体而言，使用了StableDiffusionv1.5模型，在ImageNet和COCO-30k数据集上生成512×512和1024×1024的图像。实验结果显示，使用DaTo方法可以在生成速度上实现高达9倍的加速，同时在FID（FréchetInceptionDistance）指标上也有显著降低，表明图像质量得到了提升。此外，通过与其他加速方法（如DeepCache和ToMeSD）进行对比，DaTo在生成质量和速度方面均表现出更优的平衡。实验还表明，DaTo在处理复杂文本提示时能够更好地对齐生成内容，提高了生成的准确性。
在这项研究中，我们提出了一种新方法，叫做“动态感知令牌修剪”，它可以帮助计算机更快地生成图像，同时保持图像的质量。想象一下，你在制作一个拼图，拼图的每一块都代表图像生成过程中的一个“令牌”。我们的目标是找到最重要的拼图块，以便快速完成拼图。我们首先观察相邻拼图块之间的差异，选择那些变化较大的拼图块作为基础块。接着，在拼图的过程中，我们只让那些重要的拼图块互相影响，这样就不会让不重要的拼图块干扰最终的结果。最后，我们通过一种智能的搜索方法来确定最优的拼图策略，确保在拼图的速度和质量之间取得最佳平衡。这种方法不仅让生成速度加快了很多，还保证了生成图像的清晰度和细节，让最终的作品更加完美。
2025-01-01｜AmirkabirUniversityofTechnology｜⭐️🟡
http ://arxiv.org/abs/2501.00946v1https ://github.com/omidiu/ca_tome
近年来，扩散模型在生成高质量、高维度图像方面取得了显著进展，但其高计算成本和慢推理速度仍然是主要的挑战。这主要源于自注意力机制在输入大小方面的平方计算复杂度。为了应对这一问题，研究者们提出了多种方法，其中之一是减少输入到自注意力中的token数量，即token合并（ToMe）。本文提出了一种新的方法——缓存自适应token合并（CA-ToMe），该方法通过计算token之间的相似性并合并相似性高于阈值的token，以减少计算量。为了进一步提升效率，CA-ToMe引入了自适应阈值和缓存机制，存储相似token对以便在多个相邻步骤中重复使用。实验证明，该方法在去噪过程中实现了1.24倍的加速，同时保持了与现有方法相同的FID分数。
CA-ToMe方法的核心在于改进token合并过程，其主要包括以下两个方面：
自适应token合并：该方法首先将输入token分为源和目标两组，并计算它们之间的余弦相似性。通过设定一个相似性阈值t，只有相似度高于该阈值的token对才会被合并。这种动态合并策略能够根据不同的时间步变化，优化合并效果，从而提高生成图像的质量。
计算缓存机制：在每次合并操作中，都会计算token之间的相似性矩阵，然而这一过程计算量巨大。CA-ToMe通过缓存相似性矩阵，避免在每个时间步都重复计算，从而显著降低了计算成本。具体来说，在特定的检查点上计算相似性，并在后续的推理步骤中重用这些结果，以减少冗余计算。
为了评估CA-ToMe的性能，研究团队在ImageNet-1k数据集上生成了2000张512×512像素的图像，使用StableDiffusionv1.5模型。通过设置多个不同的阈值t，团队评估了生成图像的FID、PSNR和SSIM等质量指标。实验结果表明，在阈值为0.7时，CA-ToMe在加速图像生成的同时，保持了较高的图像质量，FID分数为34.05，平均生成时间为6.09秒。此外，通过对比不同的配置，发现特定的检查点分布能够显著提升性能，进一步验证了该方法的有效性。
在CA-ToMe方法中，我们主要通过两种方式来加快图像生成的速度。首先，我们会检查每个token（图像中的小部分）之间的相似性，只有那些相似度高的token才会被合并在一起，这样可以减少不必要的计算。想象一下，如果我们在做拼图，只有那些颜色和形状相近的拼块才会被放在一起，这样能更快完成拼图。其次，我们还引入了一种缓存机制，类似于在厨房里准备好的食材，我们不需要每次都重新切菜，而是可以直接使用之前切好的食材。通过这样的方式，我们在生成图像时，不仅加快了速度，还保持了图像的质量。
2025-01-01｜ShaanxiNormalU,SMU｜⭐️⭐️
http ://arxiv.org/abs/2501.00816v1
在当前的草图提取技术中，多数方法依赖于大量的训练数据或者无法捕捉到丰富的艺术风格，限制了其实际应用的广泛性。针对这一问题，本文提出了一种名为Mixture-of-Self-Attention（MixSA）的无训练草图提取方法。此方法利用强大的扩散先验，以提高草图的感知效果。MixSA的核心在于其混合自注意力机制，通过将参考草图的键和值替换到自注意力层中，实现对初始轮廓图像的无缝整合。这种方法不仅可以精确控制纹理密度，还能在不同风格之间进行插值，从而生成新颖的、未见过的艺术风格草图。通过多种感知指标的评估，MixSA在草图质量、灵活性和适用性方面表现优异，突破了现有方法的局限性，使用户能够生成更高保真度的草图。
MixSA的工作流程主要包括以下几个步骤：
自注意力机制的混合：通过引入混合自注意力机制，将参考草图的自注意力特征（键和值）整合到彩色图像的自注意力计算中。这一过程确保了生成的草图能够融入参考图的笔触风格，同时保持原始图像的纹理和轮廓。
初始轮廓生成：利用边缘检测算法（如TEED）生成初始草图轮廓。这些轮廓作为草图生成过程中的起始点，帮助保持图像的结构和布局。
特征混合与调整：在草图生成过程中，通过调整两个参数（ζ和β）来控制参考样式与原始图像特征的融合程度。ζ控制草图与参考样式的相似度，而β则影响草图中纹理的保留程度。通过这些参数的灵活调整，MixSA能够生成从稀疏到高度详细的多种风格草图。
在实验部分，MixSA在多个数据集上进行了测试，包括4SKST、FS2K和Anime数据集。通过与多种基线方法进行比较，MixSA在多个指标上表现出色，如LPIPS、PSNR和FID等。实验结果表明，MixSA不仅在生成草图的质量上优于现有方法，还能有效处理未见过的艺术风格。具体来说，MixSA在Anime数据集上取得了最佳的LPIPS（0.4214）和FID（217.70）分数，显示出其在多样化艺术风格生成中的强大能力。此外，通过消融实验，验证了各个模块的贡献，证明了MixSA的设计合理性和有效性。
MixSA是一种新颖的草图提取技术，简单来说，它可以让我们从彩色图像中提取出草图，而不需要大量的训练数据。这个方法的核心在于一种叫做“混合自注意力”的技术，想象一下我们在画画时，先用铅笔画出轮廓，然后再用不同的颜色来填充。MixSA就是通过把参考草图的笔触风格融入到我们想要提取的彩色图像中，达到这种效果。具体来说，它会先生成一个初步的轮廓，然后根据我们想要的风格来调整草图的细节，比如让草图变得更细致或者更简约。通过这种方式，MixSA能够生成多种风格的草图，甚至是之前从未见过的风格，满足不同艺术创作的需求。
2025-01-01｜UCBerkeley,CMU｜⭐️🟡
http ://arxiv.org/abs/2501.00912v1https ://github.com/para-lost/AutoPresent
在现代社会中，设计结构化视觉内容（如演示文稿）变得越来越重要。本文提出了一种名为AUTOPRESENT的工具，旨在自动生成高质量的演示文稿。AUTOPRESENT基于一个经过训练的8BLLM（大语言模型），其训练数据来源于7,000对自然语言指令和相应的代码。这种方法不仅能生成文本内容，还能控制图像、布局和颜色等视觉元素。研究中还引入了SLIDESBENCH，这是第一个专门用于幻灯片生成的基准测试，包含7,000个训练示例和585个测试示例，涵盖了多个领域。通过对生成幻灯片的评估，研究发现AUTOPRESENT的性能与闭源模型GPT-4O相当，并且通过自我迭代设计优化，进一步提高了幻灯片的质量。
AUTOPRESENT的核心方法是通过自然语言指令生成代码，从而实现幻灯片的自动化设计。具体方法包括：
数据构建：创建自然语言指令与Python程序的配对，用于训练模型。每个指令都对应一个可以执行的程序，生成幻灯片。
SLIDESLIB工具库：为了简化代码生成过程，构建了一个功能库SLIDESLIB，包含基本的幻灯片操作，如添加标题、插入图像等。这使得生成的代码更简洁，便于模型理解和执行。
迭代优化：模型在生成初步幻灯片后，可以根据用户反馈和设计原则进行自我优化，提升幻灯片的设计质量。通过这种迭代过程，模型能够改善内容布局、颜色和元素间的间距等方面。
本研究通过SLIDESBENCH基准测试评估AUTOPRESENT的效果。实验分为多个场景，包括详细指令与图像、仅详细指令和高层次指令等。评估指标包括参考基础的相似度（如内容、布局、颜色匹配）和参考自由的设计质量评估。实验结果表明，AUTOPRESENT在多个任务中表现出色，特别是在详细指令场景中，其生成的幻灯片质量接近人类设计师。通过对比不同模型的表现，发现AUTOPRESENT在生成结构化视觉内容方面具有明显优势。
AUTOPRESENT的工作原理可以简单理解为：用户给出一些关于幻灯片的指示，比如想要的内容和样式。然后，AUTOPRESENT会将这些指示转换成一段Python代码，这段代码可以创建幻灯片。为了让这个过程更简单，研究人员还制作了一个工具库，里面有许多常用的操作，比如添加标题、插入图片等，这样生成的代码就不会太复杂。最后，AUTOPRESENT还会根据生成的幻灯片进行自我调整，以确保最终的幻灯片看起来更好、更符合设计要求。这种方法让制作幻灯片变得更高效，帮助用户快速创建出专业的演示文稿。
2025-01-01｜UCSD,FairmontPreparatoryAcademy｜⭐️
http ://arxiv.org/abs/2501.00811v1
本研究探讨了一种新的面部美学优化策略，利用回归引导的生成对抗网络（GAN）实现自动化的面部美化。传统的美化方法依赖于基于规则的预定义变换，往往受限于静态的美学标准，而本研究提出的策略则通过学习面部美的复杂模式，动态优化面部图像。研究中，作者们采用了预训练的GAN模型，将面部图像映射到潜在空间，并通过一个新开发的面部美学评估回归网络来指导潜在点的优化，从而实现美化效果的增强。该方法不仅展示了在面部美学提升方面的潜力，还为未来的自动化美学增强提供了新的方向。
本研究的方法分为几个关键步骤：
GAN反演：首先，将输入的面部图像投影到预训练的StyleGAN2的潜在空间中。这一过程称为GAN反演，旨在尽可能准确地重建原始图像。
潜在点优化：在获得初始潜在点后，利用梯度优化算法进行多次迭代，以优化该潜在点，确保生成的图像与输入图像在感知特征上尽可能相似。
美学优化：通过协方差矩阵适应进化策略（CMA-ES），对潜在点进行优化，使生成的图像在满足美学标准的同时，尽量保留原始面部特征。
损失函数设计：为指导优化过程，定义了多种损失函数，包括感知损失和美学损失，以平衡图像的美学增强与原始特征的保留。
在实验部分，研究者们使用SCUT-FBP5500数据集进行模型训练和评估。通过5折交叉验证，评估了模型在面部美学评估任务中的表现。实验结果表明，所提出的面部美学评估回归网络在美学评分的预测上优于许多现有模型，且生成的图像在视觉上明显优于原始图像，显示出有效的面部美化效果。通过对比原始图像与生成图像，研究者们观察到多个面部特征的显著改善，例如肤色和清晰度、眉毛的形状、下巴的轮廓等，证明了该方法在面部美学优化中的有效性。
在这项研究中，科学家们开发了一种新方法来自动改善人们的面部美观。首先，他们将一张照片转化为一种特殊的数字格式，这样计算机可以理解。接着，他们使用一种智能算法来调整这张照片，使其看起来更美。这个过程就像是给照片上了一层美颜滤镜，但更智能，因为它不是简单地改变几个特征，而是综合考虑了多种因素。研究者们还设计了几种计算方法来确保生成的照片既美观又保持了原始的面部特征。最后，他们通过大量实验验证了这种方法的有效性，结果显示，经过处理的照片在视觉效果上明显优于原始照片，提升了整体的面部美感。

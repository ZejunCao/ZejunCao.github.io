267-A1.3｜图像属性操控的轻量级插件，针对个体面部特征的多样化妆容；文本引导时尚图像编辑，细粒度属性增强时尚视觉语言预训练

2024-12-28｜JagiellonianU｜⭐️🟡
http ://arxiv.org/abs/2412.20164v1
本文介绍了一种名为StyleAutoEncoder（StyleAE）的新方法，旨在通过结合AutoEncoder和预训练的StyleGAN模型来高效地操控图像属性。现代生成模型（如GAN和VAE）在生成高质量图像方面表现出色，但从头开始训练这些模型通常需要大量的计算资源。StyleAE作为一个轻量级的插件，能够在有限的计算资源下有效地修改图像的特定属性，展现出良好的实用性。通过与StyleGAN的结合，StyleAE不仅在图像属性的操控上与最先进的算法（如可逆归一化流）相当，而且在速度和设计灵活性方面更具优势。
StyleAE的核心在于对StyleGAN的潜在空间进行简化和增强，以便更方便地操控图像属性。其方法步骤如下：
预备知识：首先，介绍StyleGAN的结构，强调其潜在向量z和样式向量w的作用。样式向量w更便于操控图像的不同特征。
AutoEncoder架构：StyleAE采用AutoEncoder的结构，包含编码器和解码器。编码器将输入数据映射到潜在空间，解码器则将潜在空间的表示恢复为原始输入。
目标空间结构：每个图像实例与一个K维标签向量相关联，标签可以是离散或连续的特征。通过调整潜在空间中的特定坐标，可以改变图像的相应属性。
损失函数设计：StyleAE的损失函数包括属性损失和图像损失，确保在修改属性的同时保持图像质量。
为了评估StyleAE的效果，研究者在多个数据集上进行了广泛的实验，包括人脸图像和动物图像。实验结果表明，StyleAE在操控图像属性的准确性和保持图像其他特征的完整性方面表现优异。与现有的流模型（如StyleFlow和PluGeN）相比，StyleAE在时间效率和计算复杂度上具有显著优势。实验还显示，StyleAE在处理细微属性（如表情变化）时的表现尤为突出，能够有效生成符合预期属性的图像。
StyleAE的方法可以简单理解为一种智能的图像编辑工具。想象一下，你有一张照片，想要改变某些特征，比如让一个人看起来更年轻或添加一些胡须。StyleAE通过一个“编辑器”来实现这个目标。这个编辑器有两个部分：一个是“识别器”，它能够理解照片中的特征；另一个是“生成器”，它能够根据你的要求重新生成照片。这个过程就像是把照片的特征分开，然后你可以单独调整每个特征，最后再合成一张新照片。这样，你就可以轻松地改变照片中的某些特征，而不会影响到其他部分，保持整体的自然感。
2024-12-29｜UMalaya｜⭐️🟡
http ://arxiv.org/abs/2412.20381v1
随着数字化时代的到来，化妆不再仅限于物理应用，越来越多的人通过移动应用程序在照片上进行数字化妆容的尝试。然而，尽管现有技术使化妆变得更加容易，设计针对个体面部特征的多样化妆容仍然是一个挑战。现有的化妆推荐系统和转移技术往往依赖于用户的手动操作，缺乏创新和直观性。为了解决这一问题，研究者们提出了Protegé，一个基于生成对抗网络（GAN）的化妆应用程序，旨在自动学习和生成多样化的妆容。Protegé的创新之处在于其能够理解和模拟人类化妆师的直觉，提供个性化和创新的化妆方案，标志着数字化妆技术的重大进步。
Protegé的核心方法包括以下几个步骤：
数据集构建：使用由专业化妆师创作的化妆图像数据集，确保模型能够学习到丰富的化妆风格和技巧。
区域识别：通过高级面部解析技术生成二进制区域兴趣（ROI）掩码，以识别面部化妆的具体区域，从而确保化妆应用的精确性。
化妆学习：采用化妆无妆图像的提取技术，去除原始图像中的化妆痕迹，为化妆应用提供干净的基础。
化妆生成：利用GAN生成器，将学习到的化妆风格应用于目标面部，确保每层化妆的细节都能准确呈现。
面部身份保持：在化妆应用过程中，确保保持用户的面部身份，避免因化妆而改变用户的自然面貌。
在实验中，Protegé的表现通过定量和定性评估进行验证。定量评估使用FrechetInceptionDistance(FID)和ArcFace等指标，比较生成妆容与人类化妆师妆容的相似性。实验结果显示，Protegé在化妆风格学习和面部身份保持方面均优于传统方法，生成的妆容风格与目标妆容高度一致，且保持了用户的面部特征。此外，通过用户研究，参与者普遍偏爱Protegé生成的妆容，认为其更自然，更接近人类化妆师的效果。这些结果表明，Protegé在数字化妆应用中具有显著的潜力和应用价值。
Protegé的工作原理可以简单理解为一个智能化妆助手。首先，它从大量的化妆图片中学习，了解不同风格的化妆技巧。接下来，当用户上传自己的照片时，Protegé会识别出哪些区域需要化妆，并去掉原本的妆容，留下一个干净的面孔。然后，它利用学习到的知识，自动为用户的面部添加合适的化妆风格，确保每一层妆容都非常细致，最终生成的效果既美观又保持了用户的原始面貌。通过这种方式，Protegé让每个人都能轻松获得个性化的妆容，而不需要任何专业的化妆知识。
2024-12-28｜SEU,NWPU,ZJUT,AlibabaGroup｜⭐️🟡
http ://arxiv.org/abs/2412.20062v1
在时尚图像编辑领域，基于文本的图像编辑模型取得了显著进展，但直接将这些模型应用于时尚领域时面临两个主要挑战：一是编辑区域的定位不准确，二是编辑效果的强度不足。为了解决这些问题，本文提出了MADiff模型。该模型主要由两个部分组成：MaskNet和增强注意力的扩散模型。MaskNet通过输入前景区域、密集姿态图和掩模提示，利用轻量级的UNet网络来准确预测编辑区域。增强注意力的扩散模型则通过将噪声图、注意力图和MaskNet提供的掩模输入到注意力处理器中，生成精细的噪声图，从而增强编辑效果。通过在Fashion-E数据集上的广泛实验，结果表明，MADiff模型在时尚图像编辑中能够显著提高编辑区域的预测精度和编辑效果的强度。
MADiff模型的核心方法分为两个主要阶段：掩模预测和图像编辑。在掩模预测阶段，首先对输入图像和目标提示进行预处理，提取出前景区域、密集姿态图和掩模提示。然后将这些信息输入到MaskNet中，以预测出需要编辑的区域掩模。在图像编辑阶段，首先对输入图像进行DDIM反演和随机噪声采样，以获得注意力图和噪声图。接着，将来自MaskNet的掩模、注意力图和噪声图输入到注意力处理器中，生成精细的噪声图。最后，利用生成的噪声图进行混合编辑，输出编辑后的图像。MaskNet和增强注意力的扩散模型共同作用，使得MADiff能够在时尚图像编辑中实现高精度的区域定位和强烈的编辑效果。
为了验证MADiff模型的有效性，研究团队构建了Fashion-E数据集，该数据集包含29380个训练样本和2639个用于评估的样本，涵盖颜色、细节、材质和综合编辑任务。通过对不同模型进行广泛的比较实验，结果显示，MADiff在多项指标上均优于现有的最先进模型。具体来说，MADiff在CLIP文本评分（CLIP-T）和CLIP图像评分（CLIP-I）上表现出色，表明其在文本对齐和原始信息保留方面的优势。此外，实验还进行了消融研究，以分析MaskNet和注意力处理器对模型性能的影响，结果表明这两个组件的引入显著提升了模型的编辑能力。
MADiff模型的工作过程可以简单理解为两个主要步骤：首先是“找出要编辑的地方”，然后是“进行编辑”。在第一步中，系统会分析输入的图像和你希望达到的效果，比如你想改变衣服的颜色或形状。通过使用一些先进的技术，MADiff能够准确找到需要修改的区域。接下来，在第二步中，系统会使用这些信息来进行编辑。它会生成一个新的图像，确保修改后的效果与目标相符。整个过程就像是给图像“打补丁”，确保每个细节都符合你的要求。通过这种方式，MADiff不仅能找到需要编辑的区域，还能确保编辑的效果非常显著。
2024-12-28｜SEU,NWPU,ZJUT,AlibabaGroup｜⭐️🟡
http ://arxiv.org/abs/2412.19997v1
随着视觉-语言预训练（VLP）技术在多个领域的成功应用，时尚领域的研究也逐渐受到重视。时尚产品的独特性往往体现在其细粒度属性，如材质和纹理等，这些属性在检索和分类任务中至关重要。然而，现有的模型往往未能充分利用来自文本和图像模态的细粒度信息。为了解决这一问题，本文提出了一种新颖的时尚领域预训练模型——Fine-grainedAttributesEnhancedVLP（FashionFAE），旨在深入挖掘时尚数据的细节特征。通过引入属性强调的文本预测任务（AETP）和属性促进的图像重建任务（APIR），FashionFAE能够更好地捕捉和利用时尚数据中的细粒度信息，从而在多项时尚相关任务中实现了显著的性能提升。
FashionFAE的架构包括一个基于视觉变换器的图像编码器和一个基于BERT的文本编码器。其核心方法包括以下几个方面：
属性强调的文本预测任务（AETP）：该任务通过对时尚数据中的属性进行结构化描述，帮助模型关注文本中的重要特征。具体而言，模型随机掩盖一些属性，并通过上下文信息进行预测，以增强对细粒度属性的理解。
属性促进的图像重建任务（APIR）：在该任务中，输入图像被分割成多个补丁，并通过图像编码器生成标签。模型需要重建这些补丁的标签，并利用文本信息来辅助预测，从而提升对图像细节的捕捉能力。
图像-文本对比学习任务（ITC）：该任务通过对比图像和文本的嵌入，促进它们在潜在空间中的相似性，进一步提升模型的跨模态表现。
其他辅助任务：包括传统的掩蔽语言模型（MLM）和图像-文本匹配（ITM）任务，帮助模型在多种任务中进行有效的知识迁移。
在实验部分，使用FashionGen数据集对FashionFAE进行预训练和下游任务评估。FashionGen包含292k对文本-图像数据，涵盖60k种独特的时尚商品。实验结果表明，FashionFAE在多个检索和分类任务中均超越了现有的最先进方法，具体表现为在图像到文本（I2T）和文本到图像（T2I）检索任务中分别实现了2.9%和5.2%的性能提升。此外，在类别和子类别识别任务中，FashionFAE也显著提高了准确率和宏观F1分数。这些结果表明，FashionFAE能够有效提取和利用时尚数据中的细粒度信息，显著提升模型的表现。
FashionFAE的工作原理可以简单理解为通过两个主要的任务来帮助模型更好地理解时尚产品。首先，在属性强调的文本预测任务中，模型会学习如何从描述中提取重要的细节，比如衣服的材质和颜色。想象一下，如果你在网上购物，看到一件衣服的描述中有“黑色的牛仔衬衫”，模型就会被训练来识别这些细节，以便更好地匹配图片和文字。其次，属性促进的图像重建任务则是让模型通过图像中的不同部分来猜测它们的标签，比如“这块区域是袖子，颜色是黑色”。通过这样的方式，FashionFAE能够更全面地理解时尚产品的特点，从而在检索和分类时表现得更好。

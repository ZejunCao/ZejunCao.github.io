2025-01-02｜Tel-AvivU,SnapResearch｜⭐️🟡
http ://arxiv.org/abs/2501.01407v1https ://snap-research.github.io/NestedAttention/
本研究提出了一种名为“NestedAttention”的新型机制，旨在改善文本到图像模型的个性化表现。个性化文本到图像生成的挑战在于如何在保持主体身份的同时，确保生成图像与输入文本提示的一致性。传统方法通常依赖于单一文本标记来表示主体，这限制了表达的丰富性。相对而言，使用更复杂的表示方法虽然能增强表达能力，但往往会干扰模型的先验知识，导致生成结果与文本提示的对齐度下降。本文的NestedAttention机制通过在现有的交叉注意力层中注入丰富的图像表示，生成依赖于查询的主体值，从而有效地选择与生成图像各个区域相关的特征。这种方法不仅能更好地保留主体身份，还能在多种场景中生成高保真度的个性化图像。
本研究的方法主要包括以下几个步骤：
基础构建：利用预训练的文本到图像扩散模型，输入特定主体的图像与文本提示，以生成符合提示的新图像。
交叉注意力机制：在模型的交叉注意力层中，传统方法使用固定的值来表示文本标记，而我们引入了“嵌套注意力”机制，允许每个区域的注意力值因查询的不同而变化。
查询依赖的值生成：在嵌套注意力层中，针对每个生成图像区域，模型可以选择与该区域最相关的主体特征，而不是使用单一的固定值。这一机制通过两个注意力层的组合实现，外部层负责将主体与文本标记关联，内部层则生成针对每个区域的特定值。
编码器的使用：我们使用CLIP作为编码器，将输入图像转换为多个标记，这些标记再用于计算嵌套注意力层中的每个区域的注意力值。整个过程确保了多标记的表达能力，同时与单一文本标记保持关联。
实验部分通过对比不同个性化生成方法的性能，验证了NestedAttention的有效性。我们在多种数据集上进行训练，包括人脸和宠物图像。实验结果表明，使用NestedAttention的模型在身份保留和文本提示对齐方面表现优于其他主流方法。具体而言，我们通过定性和定量的方式评估生成图像的质量，包括与输入图像的相似度、文本提示的遵循程度以及用户偏好调查。结果显示，NestedAttention不仅能够生成高质量的个性化图像，还能在多个主体的情况下，保持良好的身份一致性和表达丰富性。我们还展示了在多个输入图像的情况下，如何进一步提升生成结果的质量。
在我们的研究中，我们提出了一种新的方法，叫做“嵌套注意力”，用来帮助计算机更好地根据文本生成个性化图像。想象一下，当你给计算机一个文本描述，比如“一个在海滩上的人”，计算机需要生成一个符合这个描述的图像。传统的方法通常只使用一个简单的标签来表示“人”，这就像用一个模糊的词来描述一个复杂的形象，结果往往不能很好地保留细节。我们的嵌套注意力方法则更聪明，它允许计算机在生成图像的每个部分时，使用不同的细节信息。例如，当计算机生成“人的眼睛”时，它可以特别关注眼睛的细节，而不是简单地用一个标签来表示整个“人”。这样，生成的图像不仅能更好地符合文本描述，还能保持主体的独特性。通过这种方法，计算机能够在多种情况下创造出更加生动和真实的个性化图像。
2025-01-02｜CMU,SnapResearch,Tel-AvivU｜⭐️🟡
http ://arxiv.org/abs/2501.01424v1https ://snap-research.github.io/visual-composer/
本研究提出了一种新颖的方法，通过在文本到图像的扩散模型中整合对象级视觉提示来生成图像。这种方法旨在解决在生成过程中保持视觉提示的身份特征，同时实现多样化场景的构图问题。通过引入一种称为KV混合交叉注意力机制的架构，研究者能够有效地将来自不同视觉表示的关键信息进行混合，从而在保持每个对象独特特征的同时，生成具有多样性和灵活性的图像组合。该方法不仅提升了生成图像的语义一致性，还扩展了文本提示的创造潜力，允许用户在不依赖于预定义布局的情况下，创建多种风格和场景的图像。
本方法的核心在于KV混合交叉注意力机制，该机制通过以下几个步骤实现目标：
视觉提示编码：将输入的视觉提示通过两个独立的编码器进行处理，一个用于提取布局信息（粗编码器），另一个用于捕捉细节特征（细编码器）。
特征混合：将来自粗编码器的关键信息与来自细编码器的值进行混合，以便在生成过程中平衡对象的身份保留与多样性。
交叉注意力机制：通过交叉注意力层将混合后的特征注入到扩散模型中，使得生成的图像能够灵活地调整对象的布局和姿态。
组合指导：在推理阶段，通过调整注意力图来强化输入视觉提示与生成图像之间的对应关系，确保每个对象的特征在最终图像中得以保留。
实验部分通过多个对比试验验证了该方法的有效性。首先，研究者采用多个基准数据集，评估生成图像对输入视觉提示的遵循程度及其场景布局的多样性。结果表明，所提方法在图像多样性和身份保留方面均优于现有的图像提示方法和多模态生成方法。此外，研究还通过消融实验分析了各个组件对最终性能的贡献，结果显示，KV混合交叉注意力机制和组合指导在提升图像生成质量方面具有显著效果。
在这项研究中，研究者们开发了一种新方法，可以通过图像来帮助生成新的图像。想象一下，你有一些图片，比如一只熊和一只鸭子，你希望生成一幅包含这两只动物的图像。研究者们的做法是先把这些图片分开，提取出它们的特征，然后用一种特殊的技术把这些特征结合在一起。这个过程就像是把熊的外观和鸭子的外观分别放在一个调色板上，然后画出一幅新画。通过这种方法，生成的图像不仅能保留每个动物的特点，还能在它们的姿势和位置上有更多变化。最终，这种方法可以让我们更容易地创建复杂的图像，而不需要事先设定好它们的布局。
2025-01-02｜ZJU,Alibaba,ECNU｜⭐️⭐️
http ://arxiv.org/abs/2501.01097v1https ://github.com/modelscope/DiffSynth-Studio.git
EliGen是一种新颖的实体级控制图像生成框架，旨在解决当前文本到图像生成模型在处理个别实体时的局限性。尽管基于扩散模型的生成方法已经取得了显著进展，但仅依靠全局文本提示无法实现对图像中每个实体的精细控制。EliGen通过引入区域注意力机制，能够无缝整合实体提示和任意形状的空间掩模，从而实现更高质量的图像生成和控制。该模型不仅在生成能力上超越了现有方法，还通过构建高质量的细粒度数据集来提升实体级控制的准确性。此外，EliGen还扩展了图像修复的能力，能够在多个实体的情况下进行高效的图像修复。
EliGen的核心在于其区域注意力机制，旨在实现对多个实体的精细控制。该方法的关键步骤包括：
问题定义：实体级控制生成涉及生成图像时同时考虑全局和实体级条件。每个实体由局部文本提示和空间定位组成。
区域注意力机制：该机制扩展了传统的注意力机制，使其能够处理任意形状的空间掩模。通过对文本和潜在嵌入进行联合注意力处理，EliGen能够在生成过程中精准控制每个实体的布局。
数据集构建：为了训练模型，我们构建了一个包含500,000个高质量注释样本的数据集，确保每个样本都包含图像、全局提示和多个实体注释。
模型训练：使用LoRA（低秩适应）进行高效的微调，确保模型在保持高效性的同时获得对布局的控制能力。
在实验中，EliGen的性能通过与现有的图像生成模型（如InstanceDiffusion、GLIGEN等）进行比较来评估。实验结果显示，EliGen在图像质量、一致性和实体准确性方面明显优于其他方法。具体来说，在处理简单到复杂的任务时，EliGen能够保持高质量的生成效果，尤其在多个实体交互的情况下，其表现尤为出色。此外，EliGen还展示了其在图像修复任务中的强大能力，能够在单次前向传递中实现多个实体的修复，而不会影响非修复区域的质量。
EliGen的工作原理可以简单理解为一个智能画家，它能够根据你的指示在画布上绘制多个不同的对象。首先，你给它一个整体的描述，比如“在桌子上有一台笔记本电脑和一盆花”。接着，你还可以详细地告诉它每个对象的具体位置和样子，比如“花盆是圆形的，放在桌子的左边，而笔记本电脑是打开的，放在右边”。EliGen使用一种叫做区域注意力的技术，能够关注每个对象的具体区域，而不仅仅是整个画面，这样就能更准确地把你的想法变成图像。它还能够修复图像中的某些部分，比如如果你想要改变花的颜色，它可以在不影响其他部分的情况下做到这一点。总之，EliGen让图像生成变得更加灵活和精准，给用户带来了更大的创造空间。

本文提出了一种新颖的测试时可控图像生成方法，旨在改善文本到图像生成中的空间约束问题。传统的生成模型通常依赖于简化的提示和空间条件，这导致生成结果的语义和几何一致性不足。为了解决这一问题，研究者们将空间条件分解为语义条件和几何条件，并在生成过程中分别强制它们的一致性。该方法通过优化文本提示与生成图像之间的语义匹配，同时精确定位生成图像中的对象，从而提高了生成图像的自然性和准确性。实验结果表明，该方法在Coco-stuff数据集上相较于现有的最先进方法（SOTA）有显著提升，特别是在布局一致性评估指标上提高了30%。
本方法主要包括以下几个步骤：
语义一致性：首先，通过分析文本提示与生成图像之间的语义差异，识别缺失的语义信息，并通过编辑提示来补充这些信息。具体而言，系统会识别提示中的关键语义，并确保这些语义在生成的图像中得以体现。
几何一致性：接下来，系统通过几何变换模块来确保生成对象的准确定位。该模块包括三个步骤：识别区域（RoI）、重新定位和填充。在此过程中，系统会利用注意力图来确定生成对象的区域，并将其移动到目标位置，同时通过扩散模型对该区域进行自然值的填充。
扩散模型的应用：在生成过程中，利用扩散模型对潜在空间进行迭代更新，以消除生成图像中的伪影，确保最终输出的图像质量更高。该方法不仅适用于传统的边界框约束，还可以扩展到更复杂的空间条件。
在实验部分，研究者使用Coco-stuff数据集来验证所提方法的有效性。该数据集包含80个对象类别和91个背景类别，适合进行布局到图像的生成任务。实验结果表明，所提方法在多个评估指标上均优于现有的最先进方法，包括布局一致性和图像质量。同时，通过消融实验，研究者分析了各个组件对整体性能的贡献，结果显示语义一致性和几何一致性在提升生成图像质量方面均发挥了重要作用。此外，实验还展示了该方法在处理复杂场景时的强大能力，能够生成与用户输入的自然语言描述和空间约束高度一致的图像。
在这项研究中，科学家们开发了一种新的方法来生成图像，特别是当用户提供了复杂的文本描述和空间布局要求时。这个方法的核心在于确保生成的图像不仅在内容上与文本描述一致，而且在空间上也能够准确反映用户的要求。首先，他们会检查文本描述中是否遗漏了重要的信息，并通过添加这些信息来改善描述。接着，他们会使用一种特殊的技术来确保图像中的物体被放置在正确的位置。这个过程就像是在画布上重新安排颜色和形状，确保每个元素都在它应该在的位置。最后，他们还使用了一种叫做扩散模型的技术来清理图像，去除任何可能的瑕疵。通过这些步骤，研究者能够生成更自然、更真实的图像，满足用户的期望。

LayeringDiff是一种创新的分层图像合成方法，旨在通过生成合成图像并将其分解为前景和背景层来提高图像编辑的灵活性和效率。传统的图像合成方法通常需要大量的训练数据来微调生成模型，而LayeringDiff通过利用现有的图像生成模型，显著降低了对大规模数据集的需求。该方法的核心在于将图像合成视为从合成图像中提取层的过程，这样可以避免从零开始生成每一层的复杂性。此外，LayeringDiff还引入了高频对齐模块，以增强分解层的细节质量，从而在各种实际应用中展现出更好的表现。
LayeringDiff的实现分为三个主要阶段：
初始图像生成阶段：该阶段使用现成的图像生成模型（如StableDiffusionXL）来合成一个初始的合成图像。输入为描述图像内容的文本提示。
前景确定阶段：该阶段基于输入的文本提示识别出前景区域，并生成一个表示其边界和透明度的alpha掩码。此过程利用了自动alpha掩码估计管道，以确保前景的准确提取。
分层阶段：在此阶段，利用前面生成的合成图像和alpha掩码，将图像分解为前景层和背景层。为了解决分层问题，LayeringDiff采用了前景与背景的扩散分解模块（FBDD）和高频对齐模块（HFA），以确保生成的层具有自然的细节和纹理。
在实验中，LayeringDiff表现出色，超越了现有的图像合成方法。通过与LayerDiffuse、Text2Layer等基线模型的比较，LayeringDiff在图像质量、前景和背景层的自然度上均取得了更好的结果。此外，研究还进行了用户评估，结果显示参与者对LayeringDiff生成的图像在文本对齐和美观性方面的评分均高于其他方法。这些实验表明，LayeringDiff不仅在合成质量上表现卓越，还在实际应用中展现了广泛的适用性。
LayeringDiff的工作原理可以简单理解为一个三步走的过程。首先，它根据用户提供的描述生成一幅完整的图像，就像用画笔画出一幅画。接着，系统会识别出这幅画中前景的部分，比如一个飞鸟，并标记出它的边界和透明程度。最后，LayeringDiff将这幅图像拆分成两个部分：前景（飞鸟）和背景（天空和山脉），这样用户就可以单独调整每一部分，而不会影响到其他部分。这种方法不仅节省了大量的训练时间，还能生成更自然、更细致的图像，使得图像编辑变得更加灵活和高效。

在源无关领域泛化（Source-FreeDomainGeneralization,SFDG）研究中，BatStyler方法被提出以增强多类别场景中的风格生成能力。传统的SFDG方法依赖于多模态模型的知识转移和风格合成，然而在多类别配置中，现有方法的表现较差，尤其是在风格合成的效率上。BatStyler通过设计两个模块——粗语义生成模块（CoarseSemanticGeneration）和均匀风格生成模块（UniformStyleGeneration），有效地提高了风格合成的多样性。实验结果表明，BatStyler在多类别数据集上的表现超越了当前的最先进方法，同时在少类别数据集上也保持了相当的性能。
BatStyler的核心方法分为两个主要模块：
粗语义生成模块（CSG）：
该模块通过提取粗粒度语义来减轻语义一致性对风格多样性的压制。
利用KMeans++对类别文本特征进行聚类，从而识别并提取共享的粗语义，减少冗余信息。
通过查询大型语言模型（如GPT-4），从每个聚类中提取出三种粗语义，确保语义一致性。
均匀风格生成模块（USG）：
该模块通过初始化均匀分布的风格模板来增强风格多样性。
利用神经崩溃（NeuralCollapse）特性，生成的风格向量在联合空间中均匀分布，确保更好的风格多样性和训练效率。
通过将风格模板与固定分类器结合，支持并行训练，显著提高训练速度。
在实验中，BatStyler在六个领域泛化基准数据集上进行了评估，包括PACS、VLCS、OfficeHome、ImageNet-R、DomainNet和ImageNet-S。通过与当前最先进的方法进行比较，BatStyler在多类别数据集上展现出优越的性能，尤其是在风格多样性和训练效率方面。此外，通过消融实验验证了各模块的有效性，结果显示，粗语义生成和均匀风格生成的组合显著提高了模型在多类别场景下的表现。实验还分析了不同初始化策略对性能的影响，表明神经崩溃初始化优于随机初始化。
在BatStyler的设计中，我们主要想解决两个问题：如何让风格生成更丰富多样，以及如何提高训练的效率。首先，粗语义生成模块就像一个筛子，它帮助我们从大量的类别中找出相似的部分，避免重复的信息干扰风格的多样性。想象一下，很多猫的名字可能都包含“猫”这个词，这样就会让模型在学习时变得困惑。我们通过聚类来归纳这些相似的名字，提取出更粗的类别信息，比如“猫”这一类。接着，均匀风格生成模块则是通过创建一些均匀分布的风格模板，让模型在学习时能够更好地探索不同的风格。这样一来，模型在训练时就能更快地学习到丰富的风格，而不是像以往那样慢慢来。总之，BatStyler通过这两个模块的配合，使得风格生成既丰富又高效。

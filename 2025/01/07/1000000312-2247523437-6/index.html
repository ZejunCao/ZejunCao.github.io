2024-12-30｜ZJU,XMU,AntGroup,UTübingen｜⭐️⭐️
http ://arxiv.org/abs/2412.21117v2https ://freemty.github.io/project-prometheus/
Prometheus是一种新颖的3D感知潜在扩散模型，旨在实现从文本到3D场景的生成。该方法的核心在于利用2D先验知识，进行高效且可泛化的3D合成。通过将3D场景生成形式化为多视图、前馈、像素对齐的3D高斯生成，Prometheus能够在秒级别内完成对象和场景层面的3D生成。该模型在预训练的文本到图像生成模型的基础上，进行最小调整，结合大量单视图和多视图数据集进行训练，从而确保其生成能力的广泛适用性。
Prometheus的方法分为两个主要阶段：
GS-VAE训练阶段：
利用多视图或单视图的RGB-D图像作为输入，GS-VAE模型生成每个像素对齐的3D高斯分布。
输入的深度图通过市面上现成的单目深度估计器进行估算，模型直接从RGB图像和深度图中提取特征，生成潜在表示。
MV-LDM训练阶段：
在第一阶段生成的潜在表示基础上，训练一个多视图潜在扩散模型（MV-LDM），该模型能够生成多视图RGB-D潜在代码，并结合相机姿态和文本提示进行预测。
通过结合多视图信息和相机姿态，模型可以生成高质量的3D场景，并确保生成的一致性和视觉保真度。
在实验部分，Prometheus在多个数据集上进行了广泛的评估，包括单视图和多视图场景。通过与现有的3D生成方法进行比较，Prometheus在3D重建和文本到3D生成任务中表现出色。实验结果显示，该方法不仅在几秒钟内生成高质量的3D场景，而且在多视图一致性和几何精度方面优于其他基线方法。定量评估使用了多种指标，如PSNR、SSIM和CLIP得分，结果表明Prometheus在各项任务中均取得了显著的性能提升。
在Prometheus的生成过程中，首先会训练一个称为GS-VAE的模型，这个模型的任务是将输入的图片和深度信息转化为一种“潜在表示”，类似于压缩文件。接着，第二阶段的MV-LDM模型会利用这些潜在表示，结合文本描述和相机角度，生成我们想要的3D场景。可以想象，第一步就像是在制作食材的准备工作，而第二步则是将这些食材烹饪成美味的菜肴。通过这样的方式，Prometheus能够快速且高效地生成出丰富的3D场景，满足用户的需求。
2024-12-29｜NationalTaiwanU,NVIDIA｜⭐️⭐️
http ://arxiv.org/abs/2412.20473v1
在复杂3D场景生成领域，现有的文本到3D生成方法在处理物体间的复杂关系方面面临挑战。为了解决这一问题，研究者们提出了一种新框架——GraLa3D，旨在通过场景图和布局信息来生成复杂的3D场景。该框架利用大型语言模型（LLM）来构建场景图，并结合布局边界框，以准确表示物体之间的空间关系和交互。GraLa3D的独特之处在于它不仅能够生成单个物体，还能够处理物体之间的交互，从而避免了物体之间的不必要重叠或干扰。通过实验验证，GraLa3D在生成与文本提示高度一致的复杂3D场景方面展现了出色的性能。
GraLa3D的实现分为三个主要阶段：
场景图构成：使用LLM将输入文本提示转换为场景图和布局边界框。场景图由节点（表示物体）和边（表示物体之间的关系）组成。节点根据交互类型被分为单对象节点和超节点，以便更好地处理物体之间的关系。
节点到3D生成：在这一阶段，GraLa3D利用2D扩散模型作为指导，以生成与场景图中每个节点相对应的3D模型。每个模型在其指定的边界框内生成，确保物体的空间位置和尺寸符合布局要求。
3D场景协调：最后，GraLa3D通过全局样式协调策略，确保所有生成的3D对象在纹理和外观上保持一致，从而形成一个视觉上连贯的3D场景。
为了评估GraLa3D的性能，研究者们进行了定量和定性的实验。定量评估使用CLIP分数来衡量生成的3D模型与输入文本提示的匹配程度，结果显示GraLa3D在所有测试场景中均获得了最高的CLIP分数。此外，定性实验通过展示生成的3D场景与输入提示之间的比较，验证了GraLa3D在处理复杂场景时的准确性和一致性。与现有的GraphDreamer和GALA3D方法相比，GraLa3D能够更好地生成具有明确布局和物体间交互的场景，从而展示了其在复杂3D场景生成中的优势。
GraLa3D的工作原理可以简单理解为三个步骤。首先，系统会根据你提供的描述（比如“一个宇航员骑在马背上”）生成一个“场景图”，这就像给每个物体打标签，并标明它们之间的关系。接下来，在“节点到3D生成”阶段，系统会根据这些标签和关系，利用计算机技术生成每个物体的3D模型，确保它们的大小和位置都合适。最后，在“3D场景协调”阶段，系统会把所有生成的物体整合在一起，确保它们的外观和风格一致，形成一个完整的3D场景。这样一来，无论是玩游戏还是制作动画，用户都能得到一个既真实又美观的3D场景。
2024-12-29｜Bar-IlanU,NVIDIA｜⭐️🟡
http ://arxiv.org/abs/2412.20422v1https ://3-to-4d.github.io/3-to-4d/
随着生成建模技术的进步，4D内容的生成（动态3D对象）逐渐成为现实。本研究提出了一种新方法，将静态3D对象转化为动态4D场景，基于用户提供的文本提示来指导动画生成。现有的4D生成方法在控制生成内容的外观和几何形状上存在局限，无法满足用户对动态表现的个性化需求。为了解决这一问题，我们的3to4D方法通过将3D对象与文本描述相结合，能够有效地保留原始对象的身份特征，同时生成符合用户期望的动态动画。通过这种方式，我们的研究不仅提高了4D内容生成的灵活性，还为虚拟世界、媒体和游戏等应用领域开辟了新的可能性。
我们的3to4D方法包括以下几个关键步骤：
初始化静态4D神经辐射场（NeRF）：首先将输入的3D网格转换为一个静态的4D表示，确保在每个时间点上对象的形态保持一致。
动态添加：通过使用图像到视频的扩散模型，基于文本提示来引入动态特性。该过程包括条件化生成，使得生成的运动与输入对象的身份相符。
增量视角选择：在优化过程中，采用增量的视角选择策略，以逐步采样不同的视角。这种方法有助于增强生成的动态效果，确保运动的真实感。
注意力掩蔽的得分蒸馏采样（SDS）：使用掩蔽的SDS损失，专注于与对象相关的区域进行优化，从而提高生成内容的质量和一致性。
通过以上步骤，我们的模型能够在保持原始3D对象特征的同时，生成符合文本描述的动态4D场景。
在实验部分，我们评估了3to4D方法的性能，主要关注三个方面：生成视频的时间一致性、与文本提示的一致性以及与输入3D对象的视觉相似性。我们使用GoogleScannedObjects（GSO）数据集中的20个高质量3D对象，并为每个对象设置62个不同的动态提示。通过与三种基线方法进行比较，我们的3to4D方法在身份保留、视觉质量和动态内容生成方面表现优异，LPIPS评分提高了三倍，显示出显著的视觉保真度。此外，我们还进行了消融实验，以验证不同组件对最终生成效果的贡献，结果表明，增量视角选择和注意力掩蔽的SDS对于提升生成效果至关重要。
我们的方法可以想象成是给静态的3D玩具“注入生命”。首先，我们把这个玩具的3D模型转化为一种特殊的4D格式，这就像是把它的每一个细节都保存下来。接着，我们会根据你给的描述（比如“这个植物在开花”）来让它动起来。为了让这个过程更自然，我们会从不同的角度来观察这个玩具，就像你在现实中走动并从不同方向看它一样。最后，我们会确保在玩具运动的时候，它的样子不会变得太奇怪，而是保持它的原始特征。这样，你就能看到一个既真实又符合你想象的动态场景。
2024-12-30｜SNU｜⭐️🟡
http ://arxiv.org/abs/2412.21206v1https ://hyunsoocha.github.io/perse/
本文提出了一种名为PERSE的方法，用于从单一肖像图像生成可动画的个性化3D头像。该方法的核心在于能够在保持个体身份的同时，灵活地编辑各种面部属性，如发型、胡须等。与传统的头像创建方法不同，PERSE不仅仅是创建一个精确的数字双胞胎，而是通过生成合成的2D视频数据集，来实现对面部属性的连续和可控的编辑。通过引入一种新颖的潜在空间正则化技术，PERSE能够在生成高质量的头像时，确保面部属性的平滑过渡和多样性，从而更好地满足用户在虚拟环境中的个性化需求。
PERSE的实现过程主要包括以下几个步骤：
个性化生成头像模型：基于PEGASUS模型，采用3D高斯点云表示法，允许对面部属性进行解耦控制，从而使得用户可以选择性地改变发型或鼻子等特征，同时保持身份一致性。
合成数据集生成：通过对输入肖像图像进行属性编辑，生成一个大规模的合成2D视频数据集。此过程包括对九个属性类别（如胡须、发型等）进行图像编辑，并利用文本提示生成相应的图像。
潜在空间正则化：为了增强生成模型的能力，采用插值2D面孔图像的正则化技术，确保潜在空间的平滑过渡。这一方法通过引入新的损失函数，来控制潜在代码的分布，使其更符合真实数据的特征。
高效的微调技术：使用低秩适应（LoRA）技术，对模型进行微调，以便整合新的面部属性，同时保持已学得的特征。
在实验部分，研究团队生成了一个合成数据集，以评估所提出方法的有效性。该数据集包含957个不同属性的编辑视频，涵盖了多种面部特征的变化。通过与多个基线模型进行比较，PERSE在重建性能和生成质量上均表现优异。具体评估指标包括峰值信噪比（PSNR）、结构相似性（SSIM）和感知相似性（LPIPS）。实验结果表明，PERSE在保留身份特征的同时，能够生成高质量的3D头像，并且在用户研究中获得了较高的满意度评分。
PERSE的工作原理可以简单理解为：首先，从一张你喜欢的肖像照片开始，系统会生成一个可以动的3D头像。这个头像不仅会模仿你的外貌，还能让你随意改变发型、胡须等面部特征，就像在虚拟世界中随时换装一样。为了实现这一点，研究团队创建了一个包含许多不同面部特征的视频库，这些视频帮助系统学习如何在保持你身份的同时，灵活地调整各种面部属性。最后，通过一些聪明的技术，确保头像在变化时看起来自然且真实，让你在虚拟环境中展现个性化的自己。
2024-12-30｜BNU,BUAA,PKU,BIGAI,CityUofMacau,NUS｜⭐️⭐️
http ://arxiv.org/abs/2412.20977v1http ://unrealzoo.site/
本研究介绍了UnrealZoo，一个基于UnrealEngine构建的丰富的照片级真实感3D虚拟世界集合，旨在反映开放世界的复杂性和多样性。UnrealZoo不仅包含多种可操作的实体，支持具身人工智能（EmbodiedAI）代理的训练，还提供了一系列易于使用的PythonAPI和工具，适用于数据收集、环境增强、分布式训练和基准测试等多种应用。通过对UnrealCV的优化，我们提高了渲染和通信效率，以支持多代理交互等高级应用。实验结果表明，UnrealZoo能够为具身视觉智能的基本能力（如视觉导航和跟踪）提供有价值的见解，揭示了多样化训练环境对强化学习（RL）代理的优势及当前具身视觉代理在开放世界中面临的挑战。
UnrealZoo的构建方法包括以下几个关键步骤：
场景收集：我们从UnrealEngine市场选择了100个高质量的场景，涵盖室内、室外和复合环境，确保多样性和现实感。
可操作实体：构建了多种类型的可操作实体，包括人类、动物、车辆和无人机等，支持不同的交互和行为模式。
优化API：开发了UnrealCV+，这是对原有UnrealCV的改进，提供了高效的渲染和通信能力，支持实时交互和多代理操作。
OpenAIGym接口：为标准化代理与环境的交互，提供了Gym接口，允许用户自定义任务配置和环境增强。
实验基准：通过在复杂动态环境中进行视觉导航和跟踪任务的基准测试，评估代理的性能，分析多样化训练环境对代理的影响。
在实验部分，我们利用UnrealZoo的环境进行了一系列基准测试：
视觉导航：选择了具有复杂空间结构的场景，训练和验证基于强化学习和视觉语言模型的代理，评估其在开放世界环境中的导航能力。
主动视觉跟踪：在不同场景中测试代理的跟踪能力，分析训练数据的多样性对代理性能的影响。
社交跟踪：在动态人群中评估代理的跟踪能力，考察不同控制频率对代理表现的影响。通过这些实验，我们揭示了多样化训练环境对提高代理的跨域泛化能力和鲁棒性的重要性。
UnrealZoo是一个虚拟环境集合，旨在帮助人工智能代理更好地在复杂的3D世界中导航和互动。为了实现这一目标，我们采取了几个步骤：首先，我们从一个大型市场中选择了多种多样的场景，比如房屋、超市和自然景观。其次，我们创建了多种可以操作的角色，例如人、动物和车辆，这样AI代理可以在这些环境中进行各种活动。为了让这些代理能够快速且高效地与环境互动，我们开发了一套新的工具和接口，使得使用这些虚拟环境变得更加简单。最后，我们通过一系列实验来测试这些代理在不同环境中的表现，确保它们能够适应各种复杂的情况。这一切都旨在推动具身人工智能的发展，使其在真实世界中能更好地工作。

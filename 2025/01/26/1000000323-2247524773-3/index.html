285-A2A3｜人类反馈改善视频生成，身份保留人类视频生成；多概念定制化3D生成，单次前向过程实现千张无序无位姿图像到3D重建

AIGCResearch
主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）发布日期｜2025年1月24日｜周五
2025-01-23｜CUHK,THU,Kuaishou,SJTU,ShanghaiAILab｜⭐️🟡
http ://arxiv.org/abs/2501.13918v1https ://gongyeliu.github.io/videoalign
本研究旨在改善视频生成技术，通过引入人类反馈来解决当前系统面临的诸多挑战，如运动不平滑、视频与用户提示之间的对齐不佳以及视觉质量不足等问题。尽管现有的视频生成模型在生成具有连贯运动和逼真细节的视频方面取得了显著进展，但仍然存在与用户期望不一致的问题。为此，研究者们构建了一个大规模的人类偏好数据集，并提出了一种基于人类反馈的多维视频奖励模型，以优化视频生成的质量和一致性。通过系统性的方法，研究探索了如何将人类的偏好有效地融入到视频生成模型的训练和评估中，从而提升最终生成视频的质量。
本研究的方法主要分为以下几个步骤：
数据集构建：构建了一个包含182,000个视频生成偏好的数据集，涵盖了视觉质量、运动质量和文本对齐三个维度。通过对12种视频生成模型生成的视频进行配对评估，获得了丰富的偏好注释。
多维视频奖励模型：提出了一种新的多维视频奖励模型，旨在通过分析不同设计选择对模型效能的影响，从而提升视频生成的质量。该模型整合了来自人类评估的反馈，以优化生成视频的各个方面。
对齐算法：引入了三种对齐算法，包括直接偏好优化（Flow-DPO）、奖励加权回归（Flow-RWR）和用于推理的奖励指导（Flow-NRG）。这些算法分别在训练和推理阶段优化视频生成，确保生成结果更符合人类的偏好。
实验部分对提出的模型和算法进行了全面的评估。首先，研究者们在多个基准数据集上测试了视频生成模型的性能，包括GenAI-Bench和VideoGen-RewardBench。通过与其他现有模型进行比较，分析了不同奖励模型的准确性和效果。此外，研究还进行了消融实验，以探讨不同奖励模型类型和参数设置对生成质量的影响。结果表明，基于人类反馈的奖励模型显著提升了视频生成的质量，尤其是在运动质量和视觉质量的评估中表现优异。
在这项研究中，研究人员采用了一种新方法来改善视频生成的效果。首先，他们收集了大量人类对视频的偏好数据，这些数据帮助他们了解人们喜欢什么样的视频。接着，他们创建了一种新的评分系统，可以根据人们的反馈来评估视频的质量。这种评分系统不仅考虑了视频的画面质量，还关注视频的运动流畅度和与文字描述的一致性。最后，研究者们设计了几种算法来优化视频生成过程，使得生成的视频更符合人们的期待。这些方法的结合使得他们的研究在视频生成质量上取得了显著的提升。
2025-01-23｜ByteDance｜⭐️🟡
http ://arxiv.org/abs/2501.13452v1
在视频生成领域，身份保留的文本到视频生成（IPT2V）技术取得了显著进展，但现有方法仍面临“复制粘贴”伪影和低相似度的问题，这主要源于对低级面部图像信息的过度依赖。为了解决这些挑战，本文提出了EchoVideo模型，采用了两种关键策略：首先，设计了身份图像-文本融合模块（IITF），该模块整合了来自文本的高级语义特征，以捕捉清晰的面部身份表示，同时剔除遮挡、姿势和光照变化等因素，以避免引入伪影；其次，采用了两阶段训练策略，在第二阶段随机使用浅层面部信息，以平衡浅层特征带来的精细度提升与过度依赖之间的矛盾。实验结果表明，EchoVideo在生成高质量、可控性和保真度视频方面表现优异，成功保留了面部身份和全身完整性。
EchoVideo的整体架构基于条件扩散变换器（DiT）模型，通过创新性地融合目标人物的面部图像和文本提示，精确保留个体身份特征，生成高质量的个性化视频内容。具体方法包括：
面部特征提取：使用面部提取器定位面部区域，并采用双分支特征提取架构获取面部视觉特征和身份信息，确保全面捕捉面部特征。
视觉特征：通过高性能的SigLip编码器提取整体面部结构的高层语义信息。
身份特征：通过ArcFace提取面部细节的局部特征。
身份图像-文本融合模块（IITF）：
面部特征对齐：将视觉特征与身份特征通过可学习的映射模块进行对齐，确保信息融合的准确性。
条件特征对齐：将文本描述中的关键信息与面部特征相结合，确保生成内容的一致性。
数据和训练策略：使用变分自编码器（VAE）编码面部区域作为条件输入，并在训练过程中随机丢弃部分浅层面部信息，以减少模型对低层特征的依赖。
在实验设置中，研究者利用从互联网收集的文本到视频数据集，筛选出单一个体的片段进行训练，并构建了多种数据类型以增强模型的泛化能力。具体包括：
配对数据：为每个文本到视频对均匀采样五帧面部图像，构建了300万对样本。
交叉配对数据：通过不同视频中的同一人物图像进行交叉配对，增加了30万样本。
生成交叉配对数据：利用身份保留的文本到图像模型生成多样化的面部图像，进一步增强数据集的多样性。训练过程中，采用了两阶段的训练策略，第一阶段专注于IITF模块的预训练，第二阶段则训练整个EchoVideo模型。评估时，使用多种指标对生成视频的质量和身份保留效果进行量化分析，结果显示EchoVideo在视频质量和身份一致性方面均表现优异。
EchoVideo的工作原理可以简单理解为一个智能的“视频生成器”，它能根据用户提供的图片和文字描述，生成与之相似的动态视频。首先，系统会从用户的图片中提取出面部特征，比如脸型、五官等，然后结合文字描述中的信息，比如性别和年龄，确保生成的视频角色与用户的身份一致。为了让视频看起来更自然，EchoVideo采用了两种训练方式：一是在初期学习如何提取面部特征，二是让模型在后期能灵活运用这些特征，避免出现僵硬的表情或不自然的动作。通过这种方法，EchoVideo不仅能够生成高质量的视频，还能保持用户的独特个性，解决了许多传统视频生成技术中的问题。
2025-01-23｜UNIST,NAVERCloud｜⭐️🟡
http ://arxiv.org/abs/2501.13449v1
随着3D模型生成技术的进步，单一概念的自定义已经得到了广泛研究。然而，多概念的3D自定义仍然是一个相对未被充分探索的领域。为了解决这一问题，本文提出了MultiDreamer3D，一种能够生成包含多个用户定义概念的连贯3D内容的方法。该方法通过分而治之的策略，首先生成3D边界框，然后利用选择性点云生成器为每个概念创建粗略的点云，最后通过概念感知的扩散引导进行优化。实验结果表明，MultiDreamer3D不仅能确保对象的存在性，还能保持每个概念的独特身份，并成功处理属性变化和复杂交互等复杂场景。
MultiDreamer3D的工作流程分为两个主要阶段，涉及两个核心模块：3D布局生成器（LG）和概念感知扩散引导（CDG）。
3D布局生成器（LG）：
使用大语言模型（LLM）生成3D边界框，确保每个概念对象的存在和布局的连贯性。
通过选择性概念点云生成器，为每个概念生成粗略的点云，并将其放置在对应的3D边界框内。
概念感知扩散引导（CDG）：
初始化3D高斯体（3DGaussian）并对其进行概念标记，以便在优化过程中提供精确的反馈。
通过概念感知的间隔得分匹配（CISM）更新3D高斯体，确保每个概念在3D模型优化过程中保持其独特身份，避免混合。
这种方法有效地解决了多概念自定义中的对象缺失和概念混合问题，能够生成高质量的多概念3D模型。
在实验部分，研究者使用了来自CustomDiffusion和DreamBooth数据集的真实概念图像数据，专门设计了47个文本提示以覆盖三个主要场景：多个对象、属性变化和交互。通过定性和定量评估，MultiDreamer3D在文本-3D和图像-3D对齐方面表现出色，显著优于其他基线方法。用户研究表明，参与者对MultiDreamer3D生成的3D模型在文本和图像对齐方面的表现给予了最高的评价。此外，消融实验显示，各个组件的有效性，进一步验证了MultiDreamer3D在保持概念身份和处理复杂场景中的能力。
MultiDreamer3D是一种新方法，可以帮助我们根据文字描述生成复杂的3D模型。它的工作方式可以分为两个主要步骤。首先，它会使用一种智能算法来确定每个对象在3D空间中的位置，就像在房间里放置家具一样，确保每个物体都在合适的位置。接下来，它会为每个对象创建一个粗略的3D形状，然后通过一种特殊的技术来优化这些形状，确保每个对象的特征和身份都不会混淆。这样，即使我们想要在一个场景中放置多个不同的物体，MultiDreamer3D也能确保它们各自的特点和位置都能被准确地表现出来。
2025-01-23｜Meta,UMichigan｜⭐️⭐️
http ://arxiv.org/abs/2501.13928v1https ://fast3r-3d.github.io/
Fast3R是一种新颖的多视角三维重建方法，旨在通过一次前向传播处理1000张以上的无序和无位姿图像，从而实现高效且可扩展的三维重建。当前主流的方法，如DUSt3R，采用成对处理的方式，需进行耗时的全局对齐，限制了其在复杂场景中的应用。Fast3R通过引入基于Transformer的架构，能够在单次前向传递中同时处理多个图像，避免了迭代对齐的需求。经过大量实验，Fast3R在相机位姿估计和三维重建方面展现出卓越的性能，显著提高了推理速度，降低了误差累积，为多视角应用提供了一个强有力的替代方案。
Fast3R的核心在于其Transformer架构，该架构允许对多个图像进行并行处理，具体方法如下：
输入处理：将N张无序的RGB图像输入模型，Fast3R首先对每张图像进行特征提取，生成图像补丁特征。
融合Transformer：通过一个12层的Transformer模型，执行全局自注意力机制，允许模型同时关注所有输入图像，增强上下文信息的捕获。
点图解码：Fast3R生成局部和全局的3D点图及相应的置信度图，利用这些信息进行三维重建。
损失函数：模型的训练目标是最小化预测点图与真实点图之间的差异，结合置信度加权，提升模型对标签噪声的鲁棒性。
在实验中，Fast3R在多个数据集上进行了验证，包括CO3D和7-Scenes等。实验结果显示，Fast3R在相机位姿估计和三维重建精度上均优于DUSt3R，尤其在处理大规模视图时，其推理速度超过251帧每秒，并且能够在一次前向传递中处理多达1500张图像。此外，Fast3R在不同视角数量的测试中表现出更高的准确性，表明其具有很好的扩展性和适应性。通过与其他方法的比较，Fast3R在重建精度和处理速度上均展现出显著优势。
Fast3R的工作方式可以想象成一个快速的拼图游戏。通常，我们在拼图时需要一张一张地对比图像，找出每一块拼图的位置，这就像传统方法处理图像对一样。而Fast3R则像是一个聪明的助手，它可以同时查看所有的拼图块，快速找出它们应该放在哪里，这样就节省了大量时间。具体来说，Fast3R首先将所有图像的信息提取出来，然后通过一个强大的“大脑”（即Transformer）同时分析这些信息，从而快速生成三维模型。这样一来，无论我们有多少张图像，Fast3R都能迅速而准确地帮我们拼出完整的三维场景。

2025-01-22｜AlibabaDAMOAcademy
http ://arxiv.org/abs/2501.13106v2https ://github.com/DAMO-NLP-SG/VideoLLaMA3
本文提出了VideoLLaMA3，一个更先进的多模态基础模型，旨在提升图像和视频理解能力。VideoLLaMA3的设计理念以视觉为中心，这意味着在训练和框架设计上都强调视觉信息的重要性。研究指出，高质量的图像-文本数据对于图像和视频理解至关重要，因此，研究团队专注于构建大规模且高质量的图像-文本数据集，而不是依赖庞大的视频-文本数据集。VideoLLaMA3的训练过程分为四个阶段：视觉对齐阶段、视觉语言预训练阶段、多任务微调阶段和视频中心微调阶段。通过这些设计，VideoLLaMA3在图像和视频理解基准测试中取得了显著的性能提升。
VideoLLaMA3的训练方法包括以下几个关键技术点：
视觉编码器适应：该阶段旨在通过多样化的图像输入来增强视觉编码器的能力，使其能够处理不同分辨率的图像。使用场景图像、文档图像和带文本的场景图像来训练编码器，使其捕捉更细致的视觉细节。
视觉-语言预训练：在这一阶段，模型通过大量的图像-文本数据进行训练，以建立多模态理解的基础。数据集包含详细的图像注释和文本信息，以增强模型的空间推理能力。
多任务微调：这一阶段使用多种多模态问答数据进行微调，旨在提升模型对自然语言指令的理解和执行能力，同时为视频理解打下基础。
视频中心微调：最后一阶段专注于提高模型在视频理解和视频问答方面的能力，使用包含视频-文本数据、图像数据和文本数据的多样化数据集进行训练。
为了评估VideoLLaMA3在图像和视频理解方面的表现，研究团队进行了多项基准测试，包括文档理解、图表理解和数学推理等。实验结果显示，VideoLLaMA3在各种测试中均表现出色，尤其是在图表理解和视觉相关数学问题上，其性能显著超过了现有的最先进模型。此外，在视频理解方面，VideoLLaMA3在大多数基准测试中也取得了领先地位，显示出其强大的动态内容处理能力。
VideoLLaMA3的训练方法可以简单理解为几个步骤。首先，模型需要“适应”不同类型的图像，比如风景照和文档，这样它才能理解各种细节。接下来，模型会通过大量的图像和文字配对进行学习，让它能够理解图像与文字之间的关系。然后，模型会接受多种任务的训练，比如回答问题，这样它就能更好地理解人类的指令。最后，模型会专注于视频的理解，学习如何从视频中提取信息。通过这些步骤，VideoLLaMA3能够更准确地理解图像和视频中的内容。
2025-01-22｜QUT,CSIROData61,CSIROAgriculture&Food,ANU｜🟡
http ://arxiv.org/abs/2501.13104v1
《NeuralRadianceFieldsfortheRealWorld:ASurvey》一文对神经辐射场（NeRF）进行了全面的回顾和分析。NeRF自推出以来，已经在3D场景重建、计算机视觉和机器人等领域得到了广泛应用。尽管已有大量研究对NeRF进行了改进，针对复杂现实场景的挑战仍然存在。本文旨在整合当前的理论进展和替代表示，探讨NeRF在重建任务中的应用，并识别文献中的研究空白与未来的研究方向。通过对现有数据集和工具的回顾，本文为研究者提供了宝贵的资源，以促进NeRF模型的进一步发展和应用。
在方法部分，NeRF通过一个连续的体积函数来定义3D场景，具体过程包括以下几个步骤：
光线投射与采样：每个像素关联一条光线，通过层次化体积采样来选择光线上的点，从而避免对空白区域的无效查询。
样本编码：对每个采样点应用位置编码，将其转化为5D坐标（包括位置和视角），使用正弦和余弦函数进行高频特征的嵌入。
辐射场估计：使用多层感知机（MLP）来估计颜色和密度，通过两个MLP分别处理。
体积渲染：通过数值积分方法结合光线的颜色和密度来生成最终的图像，采用加权平均的方式整合光线沿途的颜色信息。
优化：通过最小化重构颜色与真实颜色之间的误差来训练模型，优化过程中同时更新粗略和精细模型的参数。
在实验部分，作者探讨了NeRF在多种应用场景下的表现，包括重建任务和机器人视觉。通过对比不同的NeRF变种，作者分析了它们在处理复杂场景时的优势与不足。此外，文中还强调了现实世界中的挑战，如光照变化、动态场景和相机姿态不准确等对NeRF性能的影响。通过这些实验，作者展示了NeRF在实际应用中的潜力和局限，并为未来的研究提供了参考。
在方法部分，NeRF的工作原理可以简单理解为：首先，它通过光线投射技术，将每个图像的像素与3D空间中的光线相联系。接着，NeRF对这些光线上的点进行采样，获取它们的颜色和密度信息。为了更好地捕捉细节，NeRF使用了一种叫做“位置编码”的技术，将每个点的信息转化为更高维度的数据，以便深度学习模型能够更好地处理。然后，NeRF通过计算这些点的颜色和密度来生成图像，最后通过不断比较生成的图像与真实图像之间的差异，来调整和优化模型的参数。这样，NeRF能够从2D图像中重建出高质量的3D场景。
2025-01-22｜UCSD
http ://arxiv.org/abs/2501.12637v1
DWTNeRF是一种新提出的框架，旨在提高少量训练样本下神经辐射场（NeRF）的表现。NeRF在新视角合成和三维场景表示方面表现出色，但其实际应用受到慢速收敛和对稠密训练视图的依赖的限制。DWTNeRF结合了Instant-NGP的快速训练哈希编码，并引入了一种新的离散小波损失（DiscreteWaveletLoss），以便在训练目标中明确优先考虑低频特征，从而减少少量样本NeRF在高频特征上的过拟合。通过这种方法，DWTNeRF在3-shotLLFF基准测试中显著提高了PSNR、SSIM和LPIPS指标，表明其在极具挑战性的少量样本条件下的竞争力。
DWTNeRF的核心方法包括两个主要部分：离散小波损失和跨分支交互。具体而言：
离散小波损失（DWL）：
将渲染图像和真实图像分解为不同频率的子带，以最小化它们的频率表示差异。
通过对低频差异设置更高的权重，优先考虑低频特征的学习，从而在训练早期减少高频过拟合。
跨分支交互：
使用多头注意力机制，在输入和输出层之间捕捉颜色和密度之间的交互。
这种方法允许模型在不同的3D点之间建立联系，从而降低对高频细节的过拟合风险，同时保持整体的几何准确性。
这些方法结合在一起，使DWTNeRF在处理少量训练样本时表现出色，尤其是在高频细节的处理上。
在实验中，DWTNeRF在LLFF和NeRF-Synthetic数据集上进行了评估，采用了2/3/6/9个输入视图的测试条件。使用PSNR、SSIM和LPIPS等指标来评估新视角合成的质量。实验结果表明，DWTNeRF在各个少量样本条件下均优于现有的几种最先进的方法，尤其在3-shot条件下表现突出。此外，进行了多次消融实验，验证了离散小波损失和跨分支交互的有效性，显示出这两者的结合能够显著提高模型性能。
在DWTNeRF的方法部分，我们可以把它想象成一个两步走的过程。首先，研究人员使用了一种叫做离散小波损失的技术，这就像是把一幅图像拆分成不同的层次，类似于分解乐曲为不同的音符和和声。他们特别关注低频部分，因为这些部分通常代表了图像的基本结构，而不是细节。通过优先学习这些低频特征，模型在早期训练中不容易被复杂的细节所干扰。
其次，DWTNeRF利用了一种叫做跨分支交互的机制，这就像是在不同的乐器之间建立联系，确保它们的演奏是协调的。通过这种方式，模型能够更好地理解不同3D点之间的关系，从而在合成新视角时保持图像的一致性和准确性。这两种方法结合在一起，使得DWTNeRF能够在只有少量训练样本的情况下，仍然生成高质量的图像。
2025-01-22｜NUS,IRIT-UToulouse
http ://arxiv.org/abs/2501.13045v1
在3D场景表现领域，3DGaussianSplatting（3DGS）技术因其出色的照片级渲染效果而备受关注。然而，3DGS的高存储需求限制了其在实际应用中的广泛使用。为了解决这一问题，本文提出了一种新颖的混合高斯表示方法，称为SketchandPatch。这种方法将高斯分为两类：SketchGaussians用于捕捉场景的边界特征，而PatchGaussians则用于表示较平滑的区域。通过这种分类，SketchGaussians采用参数模型进行高效编码，而PatchGaussians则通过优化修剪和重训练来保持体积一致性和存储效率。实验结果表明，该方法在多种室内外场景中显著提高了视觉质量，同时将存储需求降低至原始模型的2.3%。
本研究的核心方法包括以下几个步骤：
高斯分类：将高斯分为SketchGaussians和PatchGaussians。SketchGaussians捕捉高频边界特征，PatchGaussians则覆盖平滑区域。
SketchGaussians编码：通过提取3D线段来识别SketchGaussians，利用参数模型（如多项式回归）对其属性进行编码，从而显著减少存储需求。
PatchGaussians优化：对PatchGaussians进行修剪和重训练，确保在保持视觉质量的同时，优化存储。修剪过程通过随机移除部分高斯实现，重训练则用于提高视觉一致性。
向量量化：在PatchGaussians上应用向量量化技术，以进一步减少存储空间，确保在压缩过程中保持视觉质量。
在实验部分，研究者对提出的方法进行了全面评估，使用了四个不同的场景（Playroom、Drjohnson、Room、Truck）进行验证。通过比较不同方法在相同模型大小下的视觉质量，结果显示SketchandPatch方法在PSNR、SSIM和LPIPS等指标上均优于其他方法。具体而言，在Playroom场景中，该方法的PSNR提高了18.96%，在Truck场景中则提高了32.62%。此外，存储效率方面，SketchandPatch方法的模型大小仅为原始3DGS模型的2.35%至5.41%不等，证明了其在存储和视觉效果上的双重优势。
在我们的研究中，我们提出了一种新的方法来处理3D场景的表现，称为SketchandPatch。想象一下，艺术家在画画时，先用铅笔勾勒出轮廓（这就是SketchGaussians），然后再用颜色填充更大的区域（这就是PatchGaussians）。我们的方法通过把3D场景中的高斯分成这两类，来提高存储效率和视觉质量。首先，我们找出那些重要的边界线，并用简单的数学模型来表示它们，从而节省存储空间。接着，对于那些较平滑的区域，我们会进行一些优化处理，确保它们的视觉效果也很好。最后，我们还使用了一种叫做向量量化的技术，进一步减少存储需求。通过这些步骤，我们的方法在保持高视觉质量的同时，大大降低了存储空间的需求。

284-A1.2｜文生图精确相机控制；单图3D物体操控，文本到RGB+深度+表面法线图的联合生成

主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）发布日期｜2025年1月23日｜周四
2025-01-22｜UZaragoza,AdobeResearch｜⭐️⭐️
http ://arxiv.org/abs/2501.12910v1https ://graphics.unizar.es/projects/PreciseCam2024
本研究提出了一种名为PreciseCam的框架，旨在为文本到图像生成提供精确的相机控制。传统的文本生成模型在相机视角和镜头畸变控制方面存在显著的局限性，往往无法有效传达艺术意图或情感。与以往依赖预定义镜头的技术不同，PreciseCam仅依赖四个简单的外部和内部相机参数，消除了对现有几何形状或多视图数据的需求。该方法通过一个包含超过57,000幅图像及其文本提示和真实相机参数的全新数据集进行训练和评估，展示了在生成图像时对相机视角的精确控制能力。
PreciseCam的核心方法包括以下几个部分：
相机参数设定：用户通过简单的滑块输入四个相机参数：滚动（roll）、俯仰（pitch）、垂直视场（vFoV）和畸变（ξ）。这些参数用于定义所需的相机视角。
统一球面（US）相机模型：采用UnifiedSpherical相机模型，将相机参数转化为透视场（PF）表示。这种表示方式便于存储和解读，同时允许对不同相机模型的灵活应用。
ControlNet模块：通过ControlNet对生成过程进行引导，确保生成的图像遵循用户提供的相机参数和文本提示。该模块的输出与基础生成模型的残差结合，以实现精确的相机控制。
数据集构建：构建了一个包含57,380幅图像的多样化数据集，每幅图像都配有相应的相机参数和文本描述，确保模型能够学习到相机参数对图像外观的影响。
在实验部分，研究团队评估了PreciseCam在相机控制方面的性能，并与基线模型（如SDXL和AdobeFirefly）进行了比较。实验结果表明，PreciseCam能够在各种艺术风格和真实场景中实现精确的相机视角控制。通过对相机参数的系统变化，生成的图像在保持一致性的同时，能够有效反映用户指定的相机设置。此外，研究还探讨了模型在背景生成和视频生成中的应用，展示了其在多个领域的潜力。
PreciseCam的工作原理可以简单理解为用户通过几个滑块选择相机的视角和效果。想象你在拍照时，可以调节相机的角度和焦距，这样你就能拍出不同的照片。PreciseCam允许用户通过设置四个参数来实现这种效果：滚动（相机左右倾斜）、俯仰（相机上下倾斜）、视场（相机的视野宽窄）和畸变（镜头的变形程度）。这些设置会影响最终生成的图像，使得生成的内容不仅美观而且符合用户的创意需求。通过这种方式，艺术家和设计师可以更自由地表达自己的想法，而不再受限于传统图像生成的局限。
2025-01-22｜ZJU,Harvard｜⭐️⭐️
http ://arxiv.org/abs/2501.12935v1https ://whalesong-zrs.github.io/OMG3D-projectpage/
本研究提出了一种新的框架OMG3D，用于在单幅图像中实现3D物体的操控与动态生成。传统方法在静态编辑和动态生成之间往往难以兼顾，导致物体外观和场景照明的真实感不足。OMG3D通过将2D物体转换为3D模型，结合生成模型的强大能力，显著提升了视觉表现。该框架不仅支持用户对3D模型的精确几何控制，还能实现生动的动态效果。实验结果表明，OMG3D在静态和动态场景中均表现出色，能够在保持物体外观一致性的同时，生成自然的运动效果。
OMG3D框架的设计包括三个主要步骤：
2D到3D的转换：首先，通过图像分割技术将物体从背景中提取出来，使用图像到3D的重建方法生成粗略的3D模型。
模型变形与操控：在优化后的3D模型上，用户可以使用自动绑定工具或手动构建骨架，调整骨骼以实现所需的姿态或动画效果。此过程确保了物体外观在不同姿态下的一致性。
渲染与照明处理：最后，OMG3D利用改进的照明处理模块IllumiCombiner，估计背景的照明条件，并对物体的颜色进行校正，以实现更真实的渲染效果。通过这些步骤，OMG3D能够在图像中实现高质量的物体操控和动态生成。
在实验部分，研究者采用了多种基线方法进行比较，包括传统的图像编辑工具和最新的图像到视频生成技术。实验评估了OMG3D在物体操控任务中的表现，特别是在图像编辑和视频生成方面。通过用户研究和定量评估，OMG3D在物体外观一致性、动态效果的自然性以及整体视觉质量上都表现优于现有方法。实验结果表明，OMG3D在处理复杂场景和实现用户导向的动态效果方面具有显著优势，展示了其在实际应用中的潜力。
OMG3D的工作原理可以简单理解为三个步骤。首先，它将你手中的2D图像（比如一张照片）转变成一个3D模型，就像把平面画变成立体的雕塑。接着，用户可以对这个3D模型进行操作，比如改变它的姿势或添加动画，这个过程就像在玩一个虚拟的动作玩具。最后，OMG3D会将这个3D模型和背景的光线进行匹配，确保它在新环境中看起来自然，光影效果真实。通过这三个步骤，OMG3D能够让静态的图像变得生动起来，给人一种物体“活”了的感觉。
2025-01-22｜GoogleDeepMind,GeorgiaTech,Waymo｜⭐️⭐️
http ://arxiv.org/abs/2501.13087v1https ://orchid3d.github.io
本文提出了Orchid，一个统一的多模态潜在扩散模型，旨在同时生成颜色、深度和表面法线。这一模型的创新之处在于其能够生成相互一致的输出，使得生成的颜色、深度和法线可以无缝转换为3D重建。Orchid不仅可以从文本描述生成3D场景，还能从单个颜色图像中提取信息，处理各种3D逆问题，例如在稀疏视图下的重建。与传统的单任务模型不同，Orchid通过捕捉外观与几何之间的内在关联，能够在多种任务中展现出更高的准确性和一致性。
Orchid的核心方法包含两个主要组成部分：联合潜在空间变分自编码器（VAE）和潜在扩散模型（LDM）。具体步骤如下：
联合潜在空间VAE：通过对颜色、深度和法线进行编码和解码，构建一个统一的潜在空间。VAE的输入包括颜色图像、深度和法线，设计上允许同时处理这三种模态。
潜在扩散模型：在联合潜在空间中注入噪声，并训练模型以去噪声化，生成颜色、深度和法线。通过对联合潜在的反向过程进行建模，Orchid可以根据文本提示生成相应的输出。
多模态任务：Orchid能够执行多个任务，包括从文本生成图像及其深度和法线，或根据颜色图像进行深度和法线的估计。通过微调，Orchid还可以用于无条件的多模态修复。
在实验中，Orchid被评估在三项主要任务上：文本条件下的颜色、深度和法线生成；颜色条件下的深度和法线预测；以及无条件的联合颜色、深度和法线修复。实验结果表明，Orchid在这些任务中表现出色，生成的结果不仅准确而且一致。特别是在单目深度和法线估计方面，Orchid的性能与现有的最先进方法相媲美。此外，Orchid在生成的深度和法线之间保持了高度的一致性，显示出其在3D场景生成中的潜力。
Orchid的工作原理可以简单理解为一个智能的“画家”，它不仅能够根据描述（比如“画一朵日本百合”）画出图像，还能同时生成这幅画的深度信息和表面细节。这个过程分为几个步骤：首先，Orchid会把输入的图像信息转化为一种“内部语言”，这就像把所有颜色、深度和细节信息压缩成一个小包。然后，Orchid会在这个小包里添加一些“噪声”，相当于给画面增加一些随机的变化。接着，它会逐步“清理”这个包，最终生成出清晰的颜色图像、深度信息和表面法线。这样，Orchid就能确保生成的图像和它的深度、细节之间有很好的配合，避免了传统方法中可能出现的错误和不一致。

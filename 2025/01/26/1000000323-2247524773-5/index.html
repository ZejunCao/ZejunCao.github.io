2025-01-23｜FDU,UMD｜🟡
http ://arxiv.org/abs/2501.13893v1https ://github.com/geshang777/pix2cap
本文介绍了Pix2Cap-COCO，这是首个旨在提升细粒度视觉理解的全景像素级描述数据集。该数据集通过自动化注释管道，利用GPT-4V生成与图像中每个实例精确对齐的像素级描述，从而使模型能够学习对象之间的细微关系。Pix2Cap-COCO的数据集包含167,254个详细的描述，平均每个描述22.94个单词。研究还引入了一项新任务——全景分割-描述，要求模型同时识别图像中的实例并提供详细描述。通过与现有数据集的比较，Pix2Cap-COCO在细粒度视觉理解和语言生成方面的表现显著提升，为多模态模型的训练提供了更高质量的数据支持。
Pix2Cap-COCO的数据集构建分为几个步骤：
实例标记：使用COCO提供的分割掩码，对图像中的对象进行标记，确保每个实例都有唯一标识。
描述生成：通过设计的提示，利用GPT-4V生成与每个对象实例相关的详细描述。这些描述不仅包含对象的外观信息，还包括其与周围环境的交互。
质量验证：对生成的描述进行人工审核，确保其准确性和与视觉信息的一致性。该过程由一组训练有素的注释员完成，确保每个描述经过至少两次验证。
数据集划分：最终将数据集分为训练集和验证集，以支持后续的模型训练和评估。
在实验部分，研究者使用Pix2Cap-COCO对GPT4RoI进行监督微调，评估其在多个基准上的表现。实验结果表明，使用Pix2Cap-COCO进行训练后，GPT4RoI在视觉理解任务中的性能显著提升，特别是在识别准确性和语言生成质量方面，分别提高了11.2%和22.2%。此外，研究还设计了简单基线模型，通过与其他现有模型的比较，验证了Pix2Cap在全景分割-描述任务中的有效性，展示了其在生成高质量分割掩码和详细描述方面的能力。
在Pix2Cap-COCO的构建过程中，首先我们使用COCO数据集中的信息来标记每幅图像中的不同物体。接着，我们设计了一些问题，利用一种叫做GPT-4V的智能工具来生成每个物体的详细描述。这些描述不仅告诉我们物体的颜色、形状等基本信息，还描述了它们在图像中是如何与其他物体互动的。为了确保这些描述的准确性，我们组建了一支由学生组成的团队来检查这些描述，确保每个描述都与图像内容一致。最后，我们将这些数据分成训练用和验证用两部分，以便于后续的模型训练和效果评估。这样的方法使得我们能够得到非常详细和准确的描述，有助于更好地理解图像中的内容。
2025-01-23｜WHU
http ://arxiv.org/abs/2501.13528v1
在数字时代，视频内容成为互联网流量的重要组成部分，因此高效的视频压缩技术日益受到关注。尽管传统的视频编码标准如AVC和HEVC已经被广泛应用，基于学习的神经视频压缩方法（NVC）在压缩性能上表现优异。本文介绍了一种新的视频压缩框架DiffVC，它将基础扩散模型与视频条件编码范式相结合，旨在提升视频的重建质量和压缩效率。DiffVC利用前一帧的时间上下文和当前帧的重建潜在表示来指导扩散模型生成高质量的重建结果。此外，为了加快推理过程，提出了一种称为时间扩散信息重用（TDIR）的策略，从而显著提高了推理效率。通过引入基于量化参数的提示机制（QPP），DiffVC能够有效应对不同比特率下的失真差异，展现出卓越的视觉质量和感知指标。
DiffVC框架主要由三个核心模块组成：运动模块、上下文模块和扩散模块。具体方法如下：
运动模块：
运动估计：通过网络估计前后帧之间的光流，获取运动向量。
运动向量编码：将运动向量编码为比特流，进行压缩和解码。
熵模型：采用超先验模型和四叉树分区进行熵编码，以提高编码效率。
上下文模块：
上下文编码/解码：利用提取的时间上下文将当前帧的潜在表示编码入比特流，并通过解码器重建。
时间上下文挖掘：对先前解码帧的潜在表示进行对齐，并提取多尺度的时间上下文，以指导当前帧的重建。
扩散模块：
噪声估计：利用StableDiffusion中的U-Net进行噪声预测，结合当前帧的重建潜在表示和时间上下文。
条件引导：将当前帧的潜在表示与时间上下文进行连接，作为条件输入到U-Net中进行噪声估计。
量化参数提示：通过量化参数的比值调整U-Net的特征，以适应不同比特率的失真水平。
在实验部分，DiffVC在多个数据集上进行了评估，包括Vimeo-90k、HEVC、UVG和MCL-JCV等，涵盖了多种视频分辨率。采用多阶段训练策略，确保每个模块的全面训练，使用不同的损失函数来针对特定模块进行优化。实验结果表明，DiffVC在感知指标和视觉质量上均表现优异，特别是在DISTS指标上达到了最先进的性能。与传统编码器和其他神经视频压缩方法相比，DiffVC在压缩效率和重建质量上均有显著提升，尤其是在处理不同比特率时，展现了良好的鲁棒性。
在DiffVC的工作中，我们主要关注如何更好地压缩视频。想象一下，我们在看一部电影，电影的每一帧都是一张图片。DiffVC通过三个部分来处理这些帧。首先，运动模块帮助我们了解帧与帧之间的变化，比如人物移动的速度和方向。接下来，上下文模块会利用之前帧的信息来帮助我们更准确地重建当前帧，就像在拼图时参考旁边的拼图块一样。最后，扩散模块通过一种智能算法来减少重建过程中出现的噪音，使得最终的画面更加清晰。为了让这个过程更快，我们还使用了一种技术，能让系统在处理当前帧时借用之前帧的信息，从而节省时间。通过这些方法，DiffVC能够在保证视频质量的同时，减少文件的大小，让我们在观看时体验更流畅。
2025-01-23｜UTurin,NIITokyo,TélécomParis
http ://arxiv.org/abs/2501.13558v1
本研究提出了一种名为GoDe（按需高斯）的新方法，旨在实现逐级细节（LoD）和可扩展压缩的高效3D渲染。当前的3D高斯点云技术（3DGaussianSplatting，3DGS）在实时渲染和新视角合成中表现优异，但通常需要大量存储和显存，限制了其在消费级设备上的应用。GoDe通过将高斯体组织成多个层级，结合最新的压缩技术，能够在不同的压缩比下动态调整模型，避免了重新训练的需求。通过对标准数据集的验证，GoDe展现出低失真和显著的可扩展性，能够在保证高质量渲染的同时，大幅降低存储需求。
GoDe方法的核心在于其逐级细节和可扩展压缩的实现，具体步骤如下：
高斯层级构建：从预训练的3DGS模型开始，利用迭代梯度信息构建高斯的逐级层级。每个层级包含不同数量的高斯，以便在渲染时可以选择合适的细节层级。
细调与光谱调优：在细调阶段，通过分析高斯的梯度，剔除不必要的光谱谐波，从而减少内存占用并提升重建质量。
可扩展压缩：应用基于K均值聚类的编码技术，对每个参数进行独立压缩，确保在不同的压缩率下都能保持模型质量。此外，模型可以根据实时性能需求动态调整渲染细节。
本研究在多个标准数据集上进行了广泛的实验，以评估GoDe的性能。实验结果显示，GoDe在多个压缩等级下均能保持较高的PSNR（峰值信噪比）和SSIM（结构相似性），同时显著降低了模型的存储需求。与现有的非可扩展压缩方法相比，GoDe不仅在质量上表现出色，还在渲染速度上取得了显著提升，最高可实现99.76%的模型大小减小和98.36%的原始高斯数量减少。实验还表明，GoDe的逐级细节结构在不同层级之间实现了平滑过渡，避免了常见的视觉伪影问题。
GoDe的方法可以简单理解为一个智能的3D渲染系统，它能根据需要调整画面的细节和清晰度。首先，它把3D场景分成多个层级，每个层级代表不同的细节程度。比如，当你远离一个物体时，系统会选择显示较少的细节，这样能更快地渲染画面；而当你靠近时，系统会自动显示更多的细节。其次，在准备这些细节时，GoDe会用一种聪明的方法去掉那些不必要的部分，确保画面看起来依然清晰，但占用的存储空间却大幅减少。最后，这种方法允许用户在不同的设备上都能享受到流畅的3D体验，无论是高端游戏机还是普通的电脑。
2025-01-23｜NTU,HUST,SenseTimeResearch
http ://arxiv.org/abs/2501.13335v1
Deblur-Avatar是一种新颖的框架，旨在从运动模糊的单目视频中重建高保真、可动画的3D人类。运动模糊在动态视频捕捉中普遍存在，尤其是在快速移动的人类场景中。现有方法通常假设输入为清晰图像，未能有效处理由于人类运动引起的细节损失。Deblur-Avatar通过将基于人类运动的运动模糊模型与3D高斯点云（3DGS）结合，优化人类运动轨迹与3D高斯的重建过程，从而恢复清晰的高质量人类。该方法通过姿态依赖的融合机制，有效区分模糊和清晰区域，显著提升渲染质量和效率。
Deblur-Avatar的核心方法包括几个关键步骤：
人类运动轨迹建模：通过对每个帧的开始和结束姿势进行建模，预测人类在曝光期间的运动轨迹。
非刚性变形：对初始化的3D高斯进行非刚性变形，以适应人体在运动过程中的形状变化。
虚拟图像生成：基于预测的人类姿势，生成一系列虚拟清晰图像，这些图像将用于合成最终的模糊图像。
姿态依赖的融合：通过计算每个像素的融合掩码，有效地将模糊区域与清晰区域结合，确保在训练过程中对不同区域的优化。
优化过程：通过多种损失函数（如RGB损失、感知损失等）对模型进行训练，确保生成的头像在视觉效果和细节上都能达到高标准。
在实验部分，研究团队首先收集了合成和真实的运动模糊数据集，以评估Deblur-Avatar的性能。实验结果表明，该方法在渲染质量和定量指标上显著优于现有的其他技术。通过与人类头像建模的最新方法进行比较，Deblur-Avatar在处理运动模糊的能力上表现出色，能够生成更清晰的头像和更真实的动画效果。此外，研究还进行了消融实验，以验证每个组件的有效性，结果显示，所有模块的结合能显著提升最终效果。
Deblur-Avatar的工作原理可以简单理解为一个运动模糊去除的过程。首先，它会分析视频中人类的动作，预测他们在拍摄期间的姿势变化。接着，系统生成一些清晰的虚拟图像，模拟人类在不同时间点的样子。然后，它会将这些清晰图像与模糊图像结合，通过一种智能的方式来决定每个像素应该显示清晰还是模糊。最后，整个过程经过优化，使得生成的人类不仅清晰，而且看起来非常自然。这种方法让我们能够从模糊的视频中复原出栩栩如生的3D人类，适用于虚拟现实、游戏和电影制作等多个领域。

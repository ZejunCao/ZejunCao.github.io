2025-02-11｜NUS,ShanghaiAILab,UTAustin｜⭐️🟡
http ://arxiv.org/abs/2502.07508v2https ://github.com/NUS-HPC-AI-Lab/Enhance-A-Video
本文提出了一种名为“Enhance-A-Video”的新方法，旨在提升基于扩散变换器（DiT）的生成视频的时间一致性和视觉质量。尽管DiT模型在生成真实感视频方面取得了显著进展，但在保持帧间一致性和细节方面仍面临挑战。Enhance-A-Video通过增强跨帧信息的利用，改善了视频生成的连贯性和视觉效果，且该方法不需要重新训练或微调现有模型，具有很好的适应性和效率。通过引入跨帧强度和增强温度参数，作者展示了该方法在多种DiT模型上的有效性，期望能为未来的视频生成研究提供启示。
Enhance-A-Video的核心在于两个创新点：跨帧强度和增强温度参数。具体方法分为以下几个步骤：
跨帧强度计算：通过计算非对角线元素的平均值，得到跨帧强度（Cross-FrameIntensity,CFI），该值用于捕捉帧间信息。
增强温度参数：引入一个可调的温度参数，以调整CFI的值，从而增强跨帧相关性。较高的温度值有助于模型关注更广泛的时间上下文，而较低的温度则有助于维持局部一致性。
增强块设计：在原有的时间注意力机制中添加一个增强块，该块计算CFI并结合温度参数，优化时间注意力输出。最终，增强后的时间注意力与原始输出进行融合，以提升生成视频的质量。
在实验部分，作者对多个基于DiT的视频生成模型（如HunyuanVideo、CogVideoX、LTX-Video和Open-Sora）进行了评估。实验通过在推理阶段集成Enhance-A-Video，比较了原始模型和增强模型在生成视频时的时间一致性和视觉细节。结果表明，应用Enhance-A-Video后，生成的视频在平滑过渡、细节保留和整体质量上都有显著提升。此外，作者还进行了用户研究，收集了参与者对视频质量的主观评价，结果显示使用Enhance-A-Video的模型在各项评估标准上都获得了更高的偏好度。
Enhance-A-Video是一种用于改进视频生成质量的方法。它的工作原理可以简单理解为：想象你在制作一个动画，通常每一帧之间的变化可能会让人觉得不自然。Enhance-A-Video通过“记住”前面几帧的信息，帮助每一帧之间的过渡变得更流畅。具体来说，它做了两件事：首先，它计算出不同帧之间的联系强度，就像给每一帧打分，看看它和前后帧的关系有多紧密；其次，它引入了一个“温度”参数，来调整这些联系的强度。通过这两个步骤，生成的视频在视觉效果上更加连贯，细节也更清晰。总之，Enhance-A-Video让视频看起来更自然，更像真实的场景。
2025-02-11｜FDU,ZJU,HuaweiNoah’sArkLab,WestlakeU｜⭐️⭐️
http ://arxiv.org/abs/2502.07531v2
VidCRAFT3是一种新颖的图像到视频生成模型，旨在实现对相机运动、物体运动和光照方向的同时控制。该模型通过引入一种名为VideoLightingDirection(VLD)的合成数据集，克服了传统图像到视频生成方法在多视觉元素控制方面的局限性。VidCRAFT3的核心贡献在于其能够提供细粒度的控制，生成高质量且自然的视频内容。通过结合3D点云渲染、稀疏轨迹编码和光照感知的注意机制，VidCRAFT3在控制精度和视觉一致性方面超越了现有的最先进技术。
VidCRAFT3的架构由三个主要组件组成，以实现对相机、物体和光照的精确控制：
Image2Cloud模块：将输入图像转换为高质量的3D点云，以便进行精确的相机运动控制。该模块利用DUSt3R算法进行单目和双目重建，确保生成的视频在空间上保持一致。
ObjMotionNet模块：通过稀疏轨迹编码来控制物体运动。该模块提取光流图中的运动特征，并将这些特征与UNet的输入进行加和，以确保物体运动的自然性和准确性。
SpatialTriple-AttentionTransformer模块：整合光照方向、图像和文本信息。这一模块通过并行交叉注意机制，确保生成视频在光照、文本和图像条件下的一致性，提升视频的视觉质量。
在实验中，VidCRAFT3在多个基准数据集上进行了广泛测试，以验证其性能。实验结果表明，VidCRAFT3在相机运动、物体运动和光照控制方面均表现出色，超越了其他先进方法。通过对比分析，VidCRAFT3在视频质量评估指标（如FID、FVD和CLIPSIM）中均表现优异，显示出其在生成高质量视频方面的潜力。此外，模型的训练采用了三阶段策略，逐步优化相机运动、物体运动和光照方向的控制，使得模型在复杂场景下的表现更加稳定。
VidCRAFT3的工作原理可以想象成一个导演在拍摄电影时，对镜头、演员和灯光的精确控制。首先，模型会将一张静态图像转换成一个3D模型，这就像是为电影场景搭建了一个虚拟的舞台。接下来，模型会通过分析图像中物体的运动轨迹，来让这些物体在视频中自然地移动。最后，模型还会根据用户的需求，调整场景中的光线方向，确保无论是在明亮的阳光下还是在昏暗的房间里，生成的视频都能看起来真实且美观。这种方式让用户能够轻松创建出高质量的动画视频，仿佛他们自己在操控整个拍摄过程。
2025-02-11｜PKU,HedraInc,NVIDIA｜⭐️🟡
https ://arxiv.org/abs/2502.07701v1https ://magic-141.github.io/Magic-141/
腾讯研究团队提出Magic1-For-1（Magic141），一种高效视频生成模型，通过优化内存消耗和推理速度，实现“1分钟生成1分钟视频”的目标。传统扩散模型需数百步生成视频，耗时长且计算成本高。Magic141将文本到视频任务分解为文本生成图像（T2I）和图像生成视频（I2V）两个子任务，结合多模态条件注入、对抗性步骤蒸馏和参数稀疏化等技术，显著提升效率。实验表明，模型可在3秒内生成5秒视频，并通过滑动窗口扩展至1分钟长视频，视觉质量和动态效果优于主流开源模型。
Magic141的核心是任务分解与多级优化。首先，文本生成图像阶段利用预训练模型或检索增强生成高质量首帧；随后，图像生成视频阶段通过多模态条件注入（融合文本与图像特征）提升模型收敛速度。针对推理效率，采用对抗性步骤蒸馏（DMD2算法）将扩散步骤从50步压缩至4步，并结合分类器无引导蒸馏（CFG）减少计算量。此外，通过参数稀疏化和int8量化技术，模型内存占用从40G降至28G，支持消费级GPU运行。
团队在定制化VBench和通用VBench上验证模型性能。实验显示，Magic141在肖像视频合成任务中，4步生成的视频质量接近56步基线模型，动态流畅度、美学评分等指标领先SVD-XT-1.0、VideoCrafter等模型。量化后模型在RTX5090等显卡上仅需30G显存，生成1秒视频平均耗时小于1秒。消融实验证实，图像到视频任务比文本到视频任务收敛更快，任务分解策略有效降低了训练难度。
Magic141的工作逻辑类似于“分步完成复杂任务”。比如制作动画时，先根据剧本画一张关键图（T2I），再让这张图“动起来”（I2V）。为了让动画快速生成，研究者用了三个“加速秘籍”：1）给模型同时看文字和图片，让它学得更快；2）把原本需要反复修改50次的动画优化过程，压缩到只需修改4次，且效果不打折（类似“步骤压缩”）；3）将模型“瘦身”，通过参数精简和数值精度调整，让大模型能在普通显卡上运行。这些技术组合，最终实现了“1秒生成1秒视频”的高效表现。
2025-02-10｜UTAustin｜IUI2025｜⭐️🟡
http ://arxiv.org/abs/2502.07096v1https ://creativecommons.org/licenses/by/4.0
随着短视频在社交媒体平台上的流行，创作者们面临着将长视频转化为短视频的挑战。传统的视频编辑方法往往耗时且复杂，尤其是在选择、提取和排列视频片段时。为了解决这一问题，研究团队提出了Lotus，一个结合了抽取式和生成式摘要技术的系统，旨在帮助创作者更高效地生成短视频。Lotus首先生成一个抽象短视频的脚本及其对应的语音，然后将长视频中的片段与生成的叙述相匹配。这样，创作者不仅能够保持原始内容的连贯性，还能灵活地覆盖更多内容。Lotus的设计目标是简化短视频创作流程，提升创作者的效率和创作体验。
Lotus的工作流程分为几个步骤，帮助用户从长视频中创建短视频。具体方法如下：
输入视频：用户首先将长视频上传至Lotus，并可选择指定希望提取的片段。
生成短视频脚本：Lotus利用生成模型（如GPT-4o）创建一个简化的脚本，并生成与之匹配的语音。
匹配视频片段：系统会根据生成的脚本自动匹配长视频中的相关片段，确保语音与视觉内容的对应。
编辑界面：用户可以在Lotus的编辑界面中进一步调整视频，添加抽取式片段，修改叙述文本，或选择不同的视觉元素。
渲染与预览：用户完成编辑后，可以渲染短视频并进行预览，以便进行最后的调整。
为了评估Lotus的有效性，研究团队进行了用户研究，招募了8名具有视频编辑经验的参与者。参与者被要求使用Lotus创建短视频，并与他们通常使用的编辑工具进行比较。研究重点包括评估Lotus生成的初步短视频的帮助程度、对视频编辑工作的支持以及与传统编辑方法的比较。参与者在使用Lotus后填写了满意度问卷，并参与了半结构化访谈。结果显示，参与者普遍认为Lotus的初步生成视频提供了良好的起点，并且在编辑过程中比现有工具更高效、更灵活。
Lotus的工作流程可以简单理解为一个智能视频编辑助手。首先，用户将长视频上传，Lotus会自动帮你生成一个简短的脚本和配音，这就像是在为你的视频写一个浓缩的故事。接着，Lotus会找出长视频中与这个故事相符的片段，就像是从书中挑选出重要的句子。然后，用户可以在Lotus的编辑界面中自由调整，添加自己喜欢的片段或修改配音，确保短视频既有趣又吸引人。最后，用户可以预览并渲染出最终版本，准备分享给大家。这样一来，创作者就能轻松地把长视频变成短视频，节省了大量时间和精力。

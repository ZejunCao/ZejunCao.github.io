2025-02-07｜NationalTaiwanU,UTokyo｜⭐️⭐️
http ://arxiv.org/abs/2502.06860v2
本文介绍了一种名为AutoSketch的创新方法，旨在实现风格感知的矢量草图补全。该方法能够根据用户提供的部分草图和文本提示，自动填补缺失的内容，并保持草图的原始风格。传统的草图生成方法通常是从零开始创作，而AutoSketch则通过利用预训练的视觉语言模型（VLM）来提取草图风格的自然语言描述，从而生成与原始草图风格一致的补全效果。研究表明，AutoSketch在多种草图场景中表现出色，能够有效克服现有技术在风格一致性和内容完整性方面的不足。
AutoSketch的实现分为两个主要阶段：风格无关的草图补全和草图风格调整。具体步骤如下：
文本提示风格化：首先，通过VLM提取用户提供的部分草图的风格描述，并将其添加到文本提示中，以增强提示的风格信息。
草图优化：在得到风格化的提示后，使用条件文本到图像（T2I）模型生成指导图像，然后优化草图的参数，确保补全的草图能够与风格化提示一致，填补缺失区域。
风格调整：最后，利用VLM生成可执行的风格调整代码，对补全的草图进行风格调整，以确保最终的草图在风格上与用户提供的部分草图一致。此过程包括对新生成的笔画进行宽度、曲率等属性的调整。
在实验部分，研究者对AutoSketch进行了全面的定量和定性评估。通过与现有的草图生成方法进行比较，结果显示AutoSketch在风格一致性和内容完整性方面具有显著优势。实验使用了多种草图样本和文本提示，评估指标包括LPIPS、DINO和VQA分数等。用户评估也表明，参与者更倾向于选择AutoSketch生成的草图，认为其在风格保持和内容表现上都优于对照组。研究还探讨了不同迭代草图补全的可能性，使用户能够在草图生成过程中进行实时调整和修改。
AutoSketch的工作原理可以简单理解为一个两步走的过程。首先，用户给出一幅部分草图和一个描述这幅草图的文字提示。然后，AutoSketch会分析这幅草图，提取出它的风格特征，比如笔触的粗细和形状。接着，它会根据这些特征和文字描述来填补草图中缺失的部分，确保补全的内容与用户的要求相符。最后，AutoSketch会对补全后的草图进行风格调整，确保整体风格与原始草图一致。这样，用户就能得到一幅既完整又保持原有风格的草图。
2025-02-11｜HKUST(GZ),HKUST,CSU｜CHI2025｜⭐️🟡
http ://arxiv.org/abs/2502.07556v1
随着文本到图像生成（T2I）模型的迅速发展，生成高质量图像的能力得到了显著提升。然而，非专业用户在创建合适的提示和指定细致的空间条件方面仍面临挑战，尤其是在涉及多个对象时。为了解决这一问题，研究者们提出了SketchFlex，一个交互式系统，旨在通过粗略区域草图提高空间条件图像生成的灵活性。SketchFlex能够自动推断用户的提示，并基于众包的对象属性和关系进行合理描述，从而确保生成图像的语义连贯性。此外，该系统还将用户的粗略草图精炼为基于Canny边缘的形状锚点，以确保生成质量与用户意图的一致性。实验结果表明，SketchFlex在生成图像的连贯性上优于现有的端到端模型，并显著降低了用户的认知负担。
SketchFlex的核心方法包括三个主要模块：
草图感知的提示推荐：该模块通过多模态大语言模型（MLLM）解析用户的提示和草图，生成与粗略草图相符的内容，并确保在多区域空间条件下的完整性。推荐的提示基于从大规模真实世界数据集中提取的细粒度视觉属性、状态和对象关系。
空间条件草图精炼：此模块旨在将用户的粗略草图转化为高质量且符合意图的空间条件。通过“分解与重组”的方法，首先聚焦于每个前景对象，生成更为真实的参考图像。其次，提取每个对象的精细化掩膜，并将这些掩膜重新组合成一个统一的掩膜，确保生成的图像精确地匹配用户选择的形状。
用户界面设计：SketchFlex提供了一个用户友好的界面，使用户能够轻松进行草图控制和提示编辑。用户可以在草图上进行自由绘制，并调整系统推荐的草图，进行逐步修正以达到理想效果。
为了评估SketchFlex的有效性，研究团队进行了一项用户研究，邀请12名参与者与现有基线系统进行比较。研究重点在于用户满意度和系统功能评估。结果显示，SketchFlex在用户表达创意意图的灵活性和生成精确结果方面表现优于其他系统。参与者反馈表明，SketchFlex不仅简化了草图输入和提示编辑的过程，还在生成过程中提供了更高的连贯性和一致性。这些实验结果证实了SketchFlex在提升用户体验和生成质量方面的潜力。
SketchFlex的工作方式可以简单理解为一个智能的绘画助手。首先，当你在画布上画出一个粗略的草图时，SketchFlex会自动分析你的草图，并根据它生成一些合适的文字描述，这就像是给你的画加上标签。接着，SketchFlex会把你的草图中的每个部分分开，逐个细化，确保每个对象看起来都很真实，比如将一个模糊的“猫”变成一只具体的、形状清晰的小猫。最后，你可以在一个易于操作的界面上，进一步调整这些对象的位置和大小，确保它们在画面中看起来很自然。通过这种方式，即使是没有绘画基础的人也能轻松创造出美观且有意义的图像。

MatSwap是一种新颖的方法，旨在实现图像中材料的真实感转移。其核心目标是将输入图像中的指定区域替换为用户所选择的材料，同时保持图像的光照和几何特征。由于图像中材料的外观、几何形状和光照之间的复杂关系，材料的编辑和转移往往是一个极具挑战性的任务。传统的图像编辑方法通常依赖于繁琐的手动注释或复杂的3D场景知识，这在实际应用中显得不切实际。为了解决这一问题，MatSwap利用了一种光照和几何感知的扩散模型，能够在不需要显式UV映射的情况下直接学习输入材料的外观与场景之间的关系，从而实现材料的无缝转移。
MatSwap的方法可以分为几个关键步骤：
数据准备：构建一个名为PBR的合成数据集，包含250,000对图像，这些图像展示了相同场景的不同材料和光照条件。
模型训练：采用一个光照和几何感知的扩散模型，该模型经过微调以适应材料转移任务。模型的输入包括目标图像、掩膜、法线图和辐射图。
条件生成：在推理阶段，模型结合用户提供的材料示例，通过将CLIP特征注入到生成过程中，确保生成的材料与目标图像的光照条件一致。
逐步去噪：通过迭代的去噪过程，逐步生成与目标图像相匹配的材质效果，保持高质量的视觉效果。
为了评估MatSwap的性能，研究团队在合成和真实图像上进行了广泛的实验。实验结果显示，MatSwap在材料转移的质量上优于现有的最先进方法，尤其是在保持光照一致性和细节方面。通过与其他材料转移和图像修复方法的比较，MatSwap在PSNR和LPIPS等指标上均表现出色。此外，研究还进行了消融实验，验证了不同输入条件（如法线图和辐射图）对最终结果的影响，进一步确认了光照信息在材料转移中的重要性。
MatSwap的工作原理可以简单理解为一个智能的图像编辑工具。首先，它会准备许多不同的图像，以便学习如何在不同的光照和材料条件下进行编辑。接着，它使用一种特殊的算法，能够理解图像中的光线和形状，然后将用户想要的材料应用到图像中的特定区域。比如，如果你想把一面墙的颜色换成砖红色，MatSwap会确保新颜色在光照下看起来真实，并且与周围的环境协调一致。这种方法特别适合那些不想花太多时间在复杂的3D建模上的艺术家和设计师，帮助他们快速实现想要的效果。

在这篇论文中，研究者们提出了一种新颖的基于掩蔽的图像内容与风格解耦方法，旨在解决当前文本到图像扩散模型在风格转移过程中的内容泄漏问题。尽管现有的扩散模型在生成符合文本提示的高质量图像方面取得了显著进展，但仍面临从风格参考图像中有效解耦内容和风格的挑战。内容泄漏通常发生在风格转移过程中，导致不必要的非风格元素被引入生成图像中。为了解决这一问题，作者提出了一种不需要调整模型参数的方法，通过掩蔽特定的图像特征元素来避免内容泄漏。通过理论验证和广泛的实验，研究者展示了该方法在多种风格下的有效性和优越性。
该研究提出的方法主要包括以下几个步骤：
特征提取：首先，从风格参考图像中提取图像特征，并与内容文本特征进行逐元素相乘，生成一个特征向量。
聚类分析：对生成的特征向量进行聚类，以识别与内容相关的元素。通过聚类，可以有效地找出哪些特征元素与内容文本特征相似。
掩蔽策略：为与内容相关的特征元素生成掩蔽向量，将这些元素的值设置为零，从而消除它们对生成图像的影响。
应用掩蔽：将掩蔽后的图像特征与文本特征结合，输入到扩散模型的交叉注意力模块中进行图像生成。
该方法的理论基础通过多个定理得到了支持，表明通过适当地选择较少的条件，可以降低生成图像分布与真实图像分布之间的差异，从而优化风格转移性能。
在实验部分，研究者们构建了一个包含10个内容对象和21种图像风格的评估数据集，以全面评估模型在避免内容泄漏方面的能力。通过与现有的最先进方法进行比较，作者展示了所提出方法在文本保真度和风格相似性方面的优势。实验结果表明，使用掩蔽方法的模型在多个风格下均能有效减少内容泄漏，同时保持较高的图像质量。此外，研究者还进行了消融实验，验证了掩蔽策略的有效性，并表明较少的条件选择能够显著提高风格转移的效果。
在这项研究中，研究者们提出了一种新的方法来处理图像风格转移中的内容泄漏问题。想象一下，当我们试图将一幅图像的风格应用到另一幅图像时，有时会出现一些不应该出现的内容，比如原始图像中的某些元素意外地出现在生成的图像中。为了避免这种情况，研究者们采用了一种“掩蔽”的策略。他们首先提取出图像的特征，然后通过与文本描述进行比较，找出哪些特征与目标内容相关。接着，他们将这些相关的特征“掩蔽”掉，也就是说，把它们的值设置为零，从而在生成新图像时不让这些内容影响最终结果。这样，生成的图像就能够更好地保持目标风格，同时避免了不必要的内容泄漏。通过大量实验，研究者们证明了这种方法的有效性，显示出它在风格转移任务中的优越表现。

本文介绍了一种名为“直接上升合成”（DirectAscentSynthesis，DAS）的方法，旨在揭示传统判别模型中潜在的生成能力。研究表明，判别模型不仅能有效映射输入到语义表示中，还蕴含着丰富的生成知识。通过对CLIP模型表示的多分辨率优化，DAS能够在不需要额外训练的情况下实现高质量的图像合成。这一方法挑战了判别模型与生成模型之间的基本区分，并为图像生成、风格迁移等多种应用提供了新的视角。实验结果显示，DAS不仅能够生成自然图像，还能保持图像的统计特性，避免了常见的对抗性模式。
DAS的核心在于通过多分辨率优化来解锁判别模型的生成能力。具体方法包括：
多分辨率分解：将图像表示为不同分辨率的组件，从1×1到224×224，逐步优化这些组件以匹配目标表示。
优化目标：通过最小化图像与目标表示之间的相似度损失，来引导生成过程。该目标同时考虑了像素的统计结构和语义一致性。
增强技巧：在优化过程中，采用随机的x-y位移和像素噪声等增强方法，以提高生成的稳定性和质量。
模型集成：通过对多个CLIP模型的梯度进行平均，使得生成过程更加鲁棒，进一步提升生成效果。
这种方法不仅能够生成高质量的图像，还能进行图像重建和风格迁移，展示了判别模型在生成任务中的潜力。
为了验证DAS的有效性，进行了多项实验，主要集中在生成质量、一致性、控制修改、重建精度和应用多样性等方面。实验结果表明：
生成质量与一致性：在多个运行中，DAS能够稳定生成语义一致且视觉上连贯的图像，表现出对复杂场景的理解能力。
控制修改：DAS能够根据文本提示进行局部和全局的图像修改，保持场景的结构和物理合理性。
重建能力：通过CLIP嵌入进行图像重建时，DAS能够恢复出主要的语义元素和风格特征，显示出其在高维压缩表示中的有效性。
多样化应用：DAS在生成国旗、图像修补等特定任务中表现出色，展示了其在多种场景下的适用性。
直接上升合成（DAS）是一种新方法，它可以从传统的判别模型中生成高质量的图像。想象一下，我们通常用一个模型来判断图像的内容，比如识别图片中的物体。这种模型也可以用来生成图像，但我们需要一种聪明的方式来实现。DAS的方法是将图像分解成不同的层次，就像分层的蛋糕一样，从最简单的层到最复杂的层进行优化。
具体来说，DAS会先从一个简单的图像开始，然后逐步调整每一层，确保最后生成的图像不仅看起来美观，还能准确反映我们想要表达的内容。为了让这个过程更稳定，DAS还会在每一步中加入一些随机变化，比如轻微的移动和噪声，这样生成的图像就不会总是一样。通过这种方法，DAS可以在不需要额外训练的情况下，快速生成各种风格的图像，甚至可以用它来修改现有图像的某些部分。

本研究提出了一种名为对比对抗训练（ContrastiveAdversarialTraining,CAT）的方法，旨在评估潜在扩散模型（LatentDiffusionModels,LDMs）中保护性扰动的鲁棒性。尽管LDM在图像生成任务中表现出色，使用未经授权的数据进行定制化引发了隐私和知识产权的重大问题。为了解决这一问题，研究者们开发了对抗样本作为保护性扰动，以防止未经授权的数据使用。本文通过定性和定量实验揭示了对抗样本在LDM定制中的有效性主要源于其潜在表示的扭曲。研究进一步提出了CAT方法，该方法利用适配器作为适应性攻击，评估现有保护方法的鲁棒性，并通过大量实验表明CAT显著降低了现有保护性扰动的有效性，促使社区重新考虑和增强现有保护方法的鲁棒性。
CAT方法的核心在于对比对抗损失和适配器的使用。具体方法如下：
对比对抗损失：我们定义了一个损失函数，旨在最小化保护图像样本的重构损失。通过对抗训练，模型的潜在表示被调整，以提高对抗样本的鲁棒性。
适配器的引入：在潜在自编码器的特定层中引入LoRA适配器，仅修改适配器的权重，而不影响整个模型。这种方法扩展了参数空间，降低了计算成本，并提升了模型在不同场景下的适应性。
潜在表示的重对齐：通过使用CAT适配器，保护图像的潜在表示被重新对齐，从而确保生成的图像能够有效抵消保护性扰动。
在实验部分，研究者使用CelebA-HQ和VGGFace2数据集进行物体驱动的图像合成和风格模仿任务的评估。研究者选取了多种保护性扰动方法，并应用CAT方法进行比较。实验结果表明，CAT方法在降低保护性扰动有效性方面表现显著，尤其在使用DreamBooth和LoRA进行定制时，CAT-both和CAT-en设置均显著提高了任务性能。此外，研究还探讨了CAT适配器的不同等级对结果的影响，发现更高的适配器等级通常能提高性能。
在这项研究中，我们提出了一种新的方法，叫做对比对抗训练（CAT），用来检查图像生成技术中保护隐私的有效性。简单来说，CAT的工作原理是通过对图像的潜在表示进行微调来增强模型的安全性。我们使用了一种叫做适配器的工具，这种工具可以在不改变整个模型的情况下，灵活调整模型的某些部分。通过这种方式，CAT能帮助模型更好地处理那些经过特殊保护的图像，使其在生成新图像时不受到保护措施的影响。我们在多个实验中测试了这种方法，结果显示它能有效降低现有保护措施的效果，帮助我们更好地理解如何在图像生成中保护个人隐私。

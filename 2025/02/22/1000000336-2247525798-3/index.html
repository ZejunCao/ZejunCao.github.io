2025-02-11｜PKU,MicrosoftResearch｜⭐️🟡
http ://arxiv.org/abs/2502.07737v2https ://renshuhuai-andy.github.io/NBP-project/
本研究提出了一种新的视频生成方法，称为“下一个块预测”（Next-BlockPrediction,NBP），该方法基于半自回归（semi-autoregressive）建模框架。传统的自回归（AR）模型在视频生成中采用逐个预测下一个标记的方式，虽然这种方法在自然语言处理领域取得了成功，但在视频生成中却面临着生成速度慢和空间依赖性不足等问题。NBP通过将生成单元从单个标记转变为等大小的块（例如行或帧），有效地提高了生成效率和质量。实验结果表明，NBP在UCF-101和K600数据集上均显著优于传统的下一个标记预测模型（NTP），实现了更高的生成速度和更好的视频质量。
NBP框架的关键在于其独特的生成策略和注意力机制。具体方法包括：
视频标记化：采用MAGVITv2作为视频标记器，将视频内容转化为离散的标记映射，确保图像和视频使用共享的视觉词汇。
块级生成目标：NBP将生成目标从逐个标记的预测转变为块级预测，即当前块中的每个标记同时预测下一个块中的对应标记。这种方法允许更强的空间依赖性捕捉。
双向注意力机制：在每个块内，模型利用双向注意力机制，使得当前块内的所有标记能够相互关注，从而更好地捕捉块内的空间关系。
推理过程优化：通过并行生成多个标记，NBP显著减少了推理步骤，提升了生成效率。
在实验部分，我们对NBP模型进行了详细评估，使用UCF-101和K600数据集进行视频生成任务。实验设置包括：
模型规模：我们测试了从700M到3B参数不同规模的模型，观察生成质量与模型规模之间的关系。
生成质量评估：使用Fréchet视频距离（FVD）作为评估指标，比较NBP与传统NTP模型的生成性能，结果显示NBP在所有模型规模上均表现优异。
推理速度测试：测量NBP模型在生成视频时的帧率（FPS），结果表明NBP模型在推理速度上较NTP模型有显著提升，最高达11倍的速度提升。
消融实验：通过不同块大小和形状的实验，分析其对生成质量和推理效率的影响，发现16的块大小在生成质量和速度之间取得了良好的平衡。
在这项研究中，我们提出了一种新方法来生成视频，叫做“下一个块预测”。想象一下，传统的方法就像逐个拼图块一样，一个一个地放上去，而我们的办法是一次性放下一整行或一整块拼图，这样不仅更快，而且能更好地看到每个拼图块之间的关系。为了实现这一点，我们首先把视频分成小块，然后用一种特殊的注意力机制，让每个小块可以“看到”它内部的所有部分，而不是只能看前面的部分。这样，我们就能更快地生成视频，并且生成的内容质量更高。我们的实验显示，这种方法在生成速度和视频质量上都超越了传统的方法，展现了更大的潜力。
2025-02-11｜CUHK,StepFun,Unaffiliated｜⭐️🟡
http ://arxiv.org/abs/2502.07590v2
本研究提出了一种名为DSV的新框架，旨在加速和扩展视频扩散变换器（DiTs）的训练，特别是针对高分辨率和长视频输入。随着大规模视频生成需求的增加，传统的3D全注意力机制由于其二次计算复杂度，成为了训练过程中的主要瓶颈。即使使用了变分自动编码器（VAEs）压缩视频表示，输入序列的token数量仍然会迅速增长，导致计算和内存负担加重。DSV通过利用动态稀疏性，开发了一种两阶段训练算法，专注于关键元素，并采用高效的定制内核来优化稀疏计算和通信。实验结果表明，DSV在训练吞吐量上实现了最高3.02倍的提升，同时几乎没有质量损失。
DSV框架的核心在于其两阶段训练算法，具体包括以下几个重要组成部分：
稀疏性分析：在第一阶段，训练稀疏性预测器，以近似每个自注意力模块的注意力得分分布。这一过程不干扰DiT模型的训练，确保稀疏性预测器能够准确捕捉动态稀疏性水平。
关键KV对估计：在第二阶段，使用训练好的预测器动态评估每个查询的关键KV对，允许进行完全稀疏的注意力计算。这一过程通过识别每个查询的重要KV对，显著减少计算时间。
专门的内核和并行处理：DSV引入了优化的内核，用于关键KV的估计和稀疏注意力计算，并探索了在稀疏设置下的上下文并行策略，以最小化通信开销并提高计算效率。
在实验部分，研究团队使用了基于PyTorch的DSV原型，并在一个高达64个H100GPU的测试平台上进行了评估。实验涉及多种视频生成数据集和不同规模的DiT模型，结果显示DSV在处理长度达260K的输入时，训练吞吐量相比基线方法提升了3.02倍，同时在视觉质量上保持了与全注意力基线相当的水平。此外，DSV还在最终推理效率上表现出色，端到端延迟相较于基线提高了3.5倍，进一步证明了其在大规模视频生成任务中的有效性。
在DSV方法中，我们采用了一个简单的两阶段训练流程。首先，我们训练一个“预测器”，它能够预测每个注意力模块中最重要的部分，也就是关键的KV对。这个阶段不涉及实际的模型训练，只是让预测器学会如何识别哪些部分是关键的。接着，进入第二阶段，我们开始使用这个预测器来指导模型的训练，专注于这些关键的KV对，这样可以大大减少计算量。为了让这一切更高效，我们还设计了一些特别的计算内核，确保在多个设备上处理时不会浪费资源。这种方法不仅提高了训练速度，还保持了生成视频的质量，适合处理更长、更复杂的视频输入。
2025-02-11｜CASICT,NUS｜⭐️
http ://arxiv.org/abs/2502.07327v1https ://github.com/Siaaaaaa1/video-source-bias
随着人工智能生成内容（AIGC）的迅速发展，AI生成视频的制作变得更加高效和简单，导致互联网上充斥着各种视频内容。然而，这些视频对内容生态系统的影响尚未得到充分探索。视频信息检索作为用户访问视频内容的重要手段，面临着如何处理AI生成视频的挑战。本研究的目标是调查在视频检索中是否存在隐含的排名偏差，尤其是AI生成视频是否会被检索模型优先考虑。为此，研究团队构建了一个包含真实视频和AI生成视频的综合基准数据集，并应用了一系列公正的评估指标来评估偏差。这项研究的发现表明，AI生成视频在检索任务中被明显优先考虑，这一现象在视频检索中表现得尤为复杂。
本研究采用了以下方法来探讨视频检索中的偏差问题：
基准数据集构建：研究团队创建了一个包含13,000个视频的基准数据集，其中包括9,000个用于训练的真实视频和4个测试集（每个包含1,000个视频），这些视频由两种先进的开源视频生成模型生成。
偏差评估指标设计：为了准确测量视频检索中的偏差，研究团队设计了一系列严谨的评估指标，包括新的评估指标NormalizedΔ，以减少语义差异对评估结果的影响。
模型训练与微调：研究使用了三种现成的视频检索模型，并通过对比学习方法对模型进行微调，旨在提高模型对真实视频的优先级，确保在检索结果中真实视频排在AI生成视频之前。
在实验部分，研究团队对构建的基准数据集进行了多轮测试，评估不同模型在检索任务中的表现。通过对比分析，发现视频检索模型普遍倾向于优先排名AI生成视频，而非真实视频。实验结果显示，当AI生成视频的比例增加时，检索模型对这些视频的偏好也随之加剧。为了验证这一现象，研究团队对模型进行了多次微调，结果表明，通过引入真实视频数据进行训练，可以显著降低模型的偏差，提升对真实视频的检索效果。最终，研究提供了关于AI生成视频对检索系统影响的深刻见解，并为未来的研究提供了宝贵的参考。
在这项研究中，科学家们想知道在视频搜索时，AI生成的视频是否会被优先推荐。为此，他们首先创建了一个包含真实和AI生成视频的大型数据库。接着，他们设计了一些新的评估标准，用来准确判断这些视频在检索中的表现。最后，他们测试了几种视频搜索模型，发现这些模型往往更喜欢AI生成的视频。为了改善这一点，科学家们对模型进行了调整，确保真实视频在搜索结果中更靠前。通过这些步骤，他们希望能让视频搜索变得更加公平，避免偏向AI生成内容。
2025-02-11｜GuangzhouQuwanNetworkTechnology｜⭐️
http ://arxiv.org/abs/2502.07203v1https ://playmate111.github.io
本文提出了一种新颖的两阶段训练框架Playmate，旨在生成更为逼真的人脸动画视频。该框架利用3D隐式空间引导的扩散模型，能够将静态图像与语音音频相结合，生成与特定身份相符的动态人脸视频。尽管现有的音频驱动人脸动画技术在合成视频质量方面取得了显著进展，但仍面临诸如嘴型同步不准确、头部姿态不当及表情控制不足等挑战。Playmate通过引入解耦隐式3D表示和情感控制模块，解决了这些问题，从而实现了更高质量的视频生成和情感表达的灵活控制。大量实验结果表明，Playmate在视频质量、嘴型同步及情感控制方面均优于现有的最先进方法。
Playmate的方法主要分为两个阶段，旨在提高生成的动画视频的质量和可控性。具体步骤如下：
解耦隐式3D表示构建：在第一阶段，模型首先构建一个解耦的隐式3D表示，能够将面部特征如表情、嘴型和头部姿态等进行分离，以实现灵活的面部动画控制。
音频条件扩散变换器：接着，训练一个音频条件的扩散变换器，该变换器能够直接从音频提示中生成运动序列，增强了表情和运动的解耦能力。
情感控制模块：在第二阶段，引入情感控制模块，通过编码情感信息至潜在空间，实现对生成动画中情感的精细控制，增强了模型在不同情感状态下的表现能力。
自适应归一化：为改善表情和头部姿态的解耦，采用自适应归一化策略，使得在训练过程中能够独立处理不同的面部特征。
在实验部分，Playmate在多个数据集上进行了验证，包括AVSpeech、CelebV-Text等。实验设置包括：
数据集：使用约80,000个视频片段进行训练，并从中选取30,000个情感标记视频进行情感控制模块的训练。
训练过程：第一阶段使用四个NVIDIAA100GPU进行为期三天的训练，第二阶段则在两个GPU上继续训练两天，固定扩散变换器的参数。
评估指标：使用FID、FVD、Sync-C、Sync-D等多项指标对生成结果进行评估，确保生成视频的质量和嘴型同步的准确性。
结果分析：实验结果显示，Playmate在视频质量、情感表现及嘴型同步等多个指标上均超越了现有的最先进方法，表现出更强的灵活性和可控性。
为了让大家更容易理解Playmate的工作原理，可以把它想象成一个非常聪明的动画导演。首先，它会从一张静态的照片开始，比如你的一张头像。然后，它会听一段语音，像是你在说话或者唱歌。Playmate会把这些信息结合起来，生成一个动态的人脸动画，让这个头像看起来像是在说话或表达情感。它的特别之处在于，它不仅能让嘴巴动得很同步，还能根据不同的情感需求，调整脸上的表情，比如开心、伤心或惊讶。通过这种方式，Playmate能够制作出非常生动且自然的动画视频，适用于各种场合，比如影视配音、互动游戏等。
2025-02-11｜URochester,UCIrvine,Imperial(ICL),FlawlessAI｜⭐️
http ://arxiv.org/abs/2502.07239v1https ://andypinxinliu
在数字人机交互中，生成与语音同步的手势是提升虚拟角色真实感和互动性的关键。尽管已有一些研究在这一领域取得了进展，但现有方法在准确识别音频中的节奏和语义触发点方面仍存在困难。为了解决这些问题，本文提出了一种名为“ContextualGesture”的框架，旨在通过三个创新组件来改善共语手势视频的生成。这些组件包括：时间上对齐的语音与手势，结合语音上下文的手势标记化，以及利用边缘连接的结构感知修正模块。这一框架不仅能够生成长达一分钟的手势视频，还能对现有视频进行编辑和手势模式迁移，展示了其在多种应用中的潜力。
ContextualGesture框架的核心是以下三个主要方法：
时间对齐的语音-手势对齐：通过引入时间序列对齐，确保手势的生成与语音内容在时间上高度一致。这一过程利用了对比学习方法，以捕捉语音与手势之间的细微动态。
上下文感知的手势表示：通过知识蒸馏，将语音上下文信息融入手势运动的标记化过程中。这一方法使得手势生成不仅反映出语音的内容，还能捕捉到语音中的情感和语境信息。
结构感知的图像修正模块：采用边缘热图作为指导，提升生成视频的清晰度和真实感。该模块通过对运动关键点的连接进行建模，确保在转换过程中保持结构的准确性，从而减少由于快速或大幅度运动引起的模糊现象。
为评估所提方法的有效性，研究团队进行了多项实验，主要包括手势运动生成和视频质量评估。实验使用了PATS数据集，包含来自25位发言者的84000个视频片段。通过与多种现有方法进行比较，结果表明，ContextualGesture在手势运动的准确性和视频质量上均显著优于其他方法。此外，用户研究显示，参与者普遍认为ContextualGesture生成的视频在真实感、手势多样性和语音同步性方面表现优异。具体的评估指标包括FrechetGestureDistance（FGD）、BeatAlignmentScore（BAS）等，进一步验证了该方法在生成高质量共语手势视频方面的优势。
在这项研究中，研究者们开发了一种新方法，旨在让计算机生成的手势与人类的说话更加协调。首先，他们确保说话和手势的动作在时间上是同步的，就像一个人说话时，手势能够准确地跟上语音的节奏。其次，他们通过分析说话的内容和情感，将这些信息融入手势的生成中，使得手势不仅仅是随意的动作，而是与说话内容紧密相关的表达。最后，他们还设计了一种智能的图像处理方法，确保生成的视频看起来更加真实，不会因为说话者的快速动作而模糊。通过这些技术，研究者们能够生成更自然、更生动的手势视频，提升了人机交互的体验。

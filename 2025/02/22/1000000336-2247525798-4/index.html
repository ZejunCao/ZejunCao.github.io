2025-02-11｜SFU｜⭐️🟡
http ://arxiv.org/abs/2502.07278v1
本文提出了一种新颖的方法ATOP（ArticulateThatObjectPart），旨在通过文本提示和运动个性化生成3D对象的部件关节运动。当前的3D对象模型通常缺乏对部件运动的意识，导致生成的动画效果不够真实。ATOP利用现代视频扩散模型的优势，通过少量的示例视频进行微调，生成与目标3D对象相匹配的运动视频。该方法的核心在于将从视频中获得的运动信息转移到3D网格中，以实现更为准确的部件运动。在此过程中，ATOP不仅能生成逼真的运动视频，还能够在未见过的形状上进行个性化运动生成，展示了其在3D对象关节运动建模中的潜力。
ATOP的实现分为三个主要步骤：
类别特定运动视频生成的微调：首先，利用预训练的多视图图像生成模型（如ImageDream）进行少量样本的微调。这一步骤旨在弥补现有视频扩散模型在关节运动方面的不足，通过生成与目标类别相关的多视图视频，捕捉部件的运动动态。
运动视频个性化：在生成的多视图视频基础上，结合文本提示和分割的网格信息，进行运动个性化处理。通过渲染多视图图像和相应的掩码，模型能够生成与特定3D对象的部件运动相匹配的视频，确保生成的运动是个性化的。
视频到网格运动转移：最后，利用可微渲染技术将生成的视频运动转移到3D网格中。通过优化与文本提示相对应的运动参数，模型实现了对目标3D对象的精确关节运动控制。
为验证ATOP的有效性，作者进行了定性和定量的实验评估。实验使用了PartNet-Mobility数据集，评估了模型在3D运动参数推断中的表现。通过与现有基线方法（如Shape2Motion和OpenablePartDetection）进行比较，结果显示ATOP在生成运动参数的准确性上具有显著优势。具体而言，ATOP在多个类别中的平均角度误差和位置误差均低于其他方法，表明其在3D物体运动建模中的潜力。此外，ATOP还展示了在未见过的对象上进行零-shot泛化的能力，进一步证明了其方法的通用性和有效性。
ATOP方法的核心可以理解为三个简单的步骤。首先，研究人员使用一些现有的视频来教计算机如何让特定物体的部件运动。接着，他们会把这些运动信息与描述这些运动的文字结合起来，帮助计算机生成与这些物体相关的运动视频。最后，计算机会根据生成的视频，调整3D模型的运动，让它们看起来更真实。这就像是给一个玩具车编写一个动作剧本，然后让它按照剧本里的动作真实地移动一样。通过这种方式，ATOP能够生成非常自然的物体运动，而不需要人工标注每一个细节，节省了大量时间和精力。
2025-02-10｜CMU,ShanghaiAILab,Stanford｜⭐️🟡
http ://arxiv.org/abs/2502.06957v1https ://humansensinglab.github.io/GAS/
本研究提出了一种通用且统一的框架，旨在从单个图像合成视角一致且时间连贯的人物头像。传统的单图像头像生成方法面临着多视角和时间一致性的问题，尤其是在处理稀疏输入信号时，常常无法保留外观信息。为了解决这一问题，本文结合了基于回归的3D人类重建能力与扩散模型的生成能力，通过初步重建的密集驱动信号，实现了对参考外观和结构的高质量合成。此外，研究还提出了一种模式切换器，能够在静态视图合成和动态姿势动画之间有效区分，从而提升生成的头像在不同视角和时间上的一致性。
本方法的核心在于利用单个输入图像生成视角和时间一致的头像渲染，具体步骤如下：
单视图通用化人类合成：通过一个回归基础的3D人类重建模型，生成与目标相机参数匹配的中间视角或姿态。
视频扩散模型：将重建的3D模型作为条件输入，利用视频扩散模型生成最终的头像渲染。该模型通过多层感知器（MLP）网络获取颜色和密度信息，确保生成的头像在外观和结构上与参考图像一致。
几何和外观线索整合：通过SMPL模型提供几何信息，同时结合人类辐射场的外观线索，增强合成的外观一致性。
模式切换器：引入切换器机制，有效区分静态视图合成和动态姿势动画，确保每个任务的性能和一致性。
在实验部分，研究使用了多个数据集进行评估，包括THuman2.1和2K2K数据集，分别包含高质量的3D扫描和多视角视频。通过定量和定性指标评估生成的头像质量，结果表明，本文方法在PSNR、SSIM和LPIPS等指标上均优于现有的基线方法。此外，还进行了一系列消融实验，验证了几何线索和外观线索对生成质量的重要性，结果显示，整合这两种线索显著改善了合成效果。最终实验结果表明，所提出的框架在真实场景中的表现具有良好的泛化能力。
在这个研究中，我们提出了一种新方法来从一张图片生成3D人像，确保这些人像在不同角度和时间上看起来一致。首先，我们使用一种技术来从单张图片中提取出人像的3D结构。这就像我们用一张照片来建立一个人的3D模型。接着，我们把这个模型输入到一个视频生成程序中，这个程序可以根据模型的信息生成多个角度的图像，甚至模拟人物的动作。为了让生成的人物在不同的视角下看起来更自然，我们还结合了几何信息和外观信息。最后，我们设计了一种切换机制，可以在生成静态图像和动态动画之间进行切换，这样可以保证每种生成的效果都很好。总的来说，我们的方法能够从一张图片中创造出高质量的动画人像，表现得非常真实。
2025-02-10｜UNevada,ZJU｜⭐️🟡
http ://arxiv.org/abs/2502.07007v1
近年来，人工智能生成内容（AIGC）在三维（3D）和四维（4D）生成领域取得了显著进展，极大地提升了生成内容的真实感。然而，现有方法多侧重于视觉一致性，忽视了物理原则的应用，导致生成内容中出现不合理的变形、运动不稳定和物体交互不一致等问题。因此，将物理先验知识整合到生成模型中成为一种新兴且重要的研究方向。本文综述了物理感知生成方法，系统分析了物理约束如何在3D和4D生成中被整合，旨在提升生成内容的结构完整性和运动真实感。
本研究首先回顾了物理先验在静态和动态3D生成中的应用，分类方法包括基于视觉的、NeRF（神经辐射场）和高斯点云（GaussianSplatting）等不同表示方式。其次，探讨了4D生成中的新兴技术，尤其是如何通过物理仿真来建模时间动态。最后，进行了主要方法的比较分析，强调了它们在不同材料和运动动态适用性方面的优缺点。具体方法包括：
3D生成方法：
采用生成对抗网络（GANs）和扩散模型（DMs）来合成高质量的3D结构。
整合物理仿真方法（如材料点法和有限元法）以确保生成内容的物理一致性。
4D生成方法：
通过时间依赖的辐射场和变形场来捕捉动态场景。
使用轻量级的高斯核表示法来维持高质量的动态场景重建。
物理仿真技术：
采用MPM、FEM等技术来模拟动态材料行为，确保生成内容符合物理法则。
实验部分评估了不同物理感知生成模型的性能，采用了多个数据集和评估指标，以验证模型在真实世界物理一致性和视觉质量方面的表现。使用的评估指标包括平均绝对误差（MAE）、物理常识（PC）和峰值信噪比（PSNR）。通过对比分析，发现不同方法在流体和塑料材料的模拟上表现出明显的差异，强调了集成物理先验在生成模型中的重要性。此外，实验结果显示，尽管已有模型在流体动力学和材料行为建模上取得了一定的进展，但仍需进一步提高对复杂物理交互的捕捉能力。
在这项研究中，我们主要关注如何将物理知识应用于计算机生成的三维和四维内容。可以想象一下，如果我们在生成一个虚拟的水杯时，只关注它的外观，而不考虑水的流动和杯子的重力影响，那么生成的水杯可能就会出现不自然的情况，比如水在杯子外面流动。为了避免这种情况，我们采用了一些方法：
生成模型：我们使用一些先进的技术，如生成对抗网络（GAN）和扩散模型（DM），这些技术可以帮助我们生成更真实的3D物体。
物理仿真：我们引入了物理仿真方法，比如材料点法（MPM）和有限元法（FEM），这些方法可以帮助我们模拟物体的动态行为，比如水的流动和物体的变形。
综合评估：通过对生成的内容进行测试和评估，我们可以确保它们在视觉上看起来真实，同时也符合物理规律。这样，生成的内容就能在游戏、动画等应用中更加真实可信。
2025-02-11｜NJU,FDU,HuaweiNoah’sArkLab｜ICLR2025｜⭐️🟡
http ://arxiv.org/abs/2502.07615v1https ://nju-3dv.github.io/projects/fds
在本研究中，我们提出了一种新的技术——流蒸馏采样（FlowDistillationSampling,FDS），旨在通过引入预训练的匹配先验来改善3D高斯点云（3DGaussianSplatting,3DGS）的几何重建质量。尽管3DGS在渲染质量和速度上表现出色，但其优化过程缺乏明确的几何约束，尤其是在观察视角稀疏的区域，导致几何重建效果不佳。我们的FDS方法通过结合从匹配模型中获得的光流信息，针对未观察视角进行精确采样，从而提升了高斯辐射场的准确性。通过在多个数据集上进行全面实验，我们验证了FDS在深度渲染、网格重建和新视角合成等任务中的显著优势。
FDS方法的核心在于通过引入预训练匹配先验来调节高斯辐射场的优化过程。具体而言，方法主要包括以下几个步骤：
3D高斯点云与辐射流生成：使用3DGS表示3D场景，并通过高斯分布优化其位置、透明度和颜色特征。辐射流（RadianceFlow）用于在不同视角间传递信息。
相机采样方案：在训练过程中，FDS随机采样靠近输入视角的未观察视角，并确保足够的视角移动以捕捉场景几何。通过深度自适应半径控制采样的重叠程度，确保光流的一致性。
流蒸馏采样损失：定义FDS损失函数，通过相互精炼的方式优化辐射流和匹配先验流。该损失函数旨在减小渲染结果与真实图像之间的差距，并通过匹配先验提升几何重建的准确性。
我们在多个数据集上评估了FDS方法的有效性，包括Mushroom、ScanNet和Replica数据集。实验设置包括对比传统3DGS和2DGS方法，并分析FDS在几何重建和新视角合成中的表现。结果显示，FDS显著提高了几何重建的准确性，尤其是在观察视角稀疏的情况下，”绝对相对误差”（AbsRel）指标的改善超过50%。此外，FDS的引入也改善了RGB图像的渲染质量，显示出在不同数据集上的鲁棒性和有效性。
流蒸馏采样（FDS）是一种新方法，用来改善3D图像的质量，尤其是在我们无法从多个角度观察到的情况。它的工作原理是这样的：首先，我们使用一种叫做3D高斯点云的技术来表示图像中的物体。然后，FDS通过从已有的图像中提取信息，帮助我们更好地理解未观察到的区域。具体来说，FDS会在训练过程中随机选择靠近我们已有图像的新视角，并确保这些视角之间有适当的距离，这样可以更好地捕捉到场景的形状和细节。此外，FDS还会通过一种特殊的损失函数来优化图像，使得生成的图像更接近真实情况。这种方法在各种测试中都表现出色，帮助我们在复杂场景中获得更高质量的3D重建效果。
2025-02-11｜NJU,Apple,HKUST｜⭐️🟡
http ://arxiv.org/abs/2502.07685v1https ://nju-3dv.github.io/projects/matrix3d
本研究提出了Matrix3D，一个统一的模型，能够执行多个光摄影测量任务，包括姿态估计、深度预测和新视角合成。传统的光摄影测量流程通常需要大量的图像，且涉及多个独立的处理阶段，这使得流程复杂且易于出现误差。Matrix3D通过多模态扩散变换器（DiT）整合不同模态的数据，如图像、相机参数和深度图，显著简化了这一流程。模型还引入了一种掩码学习策略，使其能够在数据不完整的情况下进行训练，从而提高了可用训练数据的数量。通过多轮交互生成，Matrix3D展现出在各个下游任务中的先进性能，证明了其在光摄影测量应用中的多样性和有效性。
Matrix3D的核心在于其多模态扩散变换器架构，支持灵活的输入和输出配置。具体方法包括：
多模态扩散变换器：该模型由多视角编码器和解码器组成，能够处理来自不同模态（RGB图像、姿态和深度）的条件数据，并将其嵌入共享的潜在空间，以便更好地进行跨视角和跨模态的特征整合。
掩码学习：与传统的掩码自编码器不同，Matrix3D在多视角和多模态设置中扩展了掩码策略，通过随机掩码特定视角或模态，模型学习在训练和推理过程中预测缺失的内容。
训练策略：采用分阶段训练策略，首先在较低分辨率的多视角模型上进行初步训练，随后逐步提高分辨率，以增强模型的生成能力。
数据准备：使用六个不同的数据集进行训练，并对不同模态进行特定的编码，以应对数据分布的多样性。
在实验部分，Matrix3D在多项光摄影测量任务上进行了评估，包括姿态估计、深度预测和新视角合成。实验结果显示，Matrix3D在姿态估计任务中相较于传统方法表现出显著的准确性提升。在新视角合成任务中，Matrix3D也优于现有的多视角扩散模型，展示了其在生成高质量图像方面的能力。此外，在深度预测任务中，Matrix3D能够从单幅图像中生成高质量的深度图，证明了其在处理稀疏输入时的有效性。综合来看，Matrix3D在多个任务中均表现出色，验证了其作为一个统一模型的潜力。
Matrix3D是一种新型的技术，可以将多种不同的输入（例如图片、相机位置和深度信息）结合在一起，生成三维图像。它的工作原理就像一个聪明的拼图游戏，能够根据已有的信息推测出缺失的部分。首先，它会将不同类型的信息放在一起，利用一种叫做“掩码学习”的方法，学习如何填补那些缺失的部分。接着，它会通过多轮训练，逐步提高自己的能力，最终能够生成高质量的三维图像。这个过程就像是从简单的草图逐步完善成一幅详细的画作。通过这种方法，Matrix3D可以帮助我们在光摄影测量中更加高效地工作，尤其是在我们没有很多图片的情况下。

2025-02-10｜GoogleDeepMind｜⭐️
http ://arxiv.org/abs/2502.07001v1https ://anuragarnab.github.io/
在计算机视觉领域，扩散模型的出现为图像和视频生成带来了前所未有的真实感。这项研究旨在探讨视频扩散模型在视觉理解任务中的表现，尤其是与图像扩散模型的比较。尽管先前的研究主要集中在图像生成上，但视频扩散模型的潜力尚未得到充分挖掘。本研究通过系统性比较同一模型架构在图像和视频生成任务上的表现，揭示了视频模型在多种下游任务中的优越性。此外，研究还分析了不同层次提取的特征、噪声水平的变化，以及模型规模和训练预算对表示和生成质量的影响。这项工作不仅填补了视频和图像扩散模型在视觉理解方面的研究空白，还提供了有关时间信息在表示学习中作用的重要见解。
本研究采用了基于扩散模型的WALT（Windowed-AttentionLatentTransformer）架构，重点分析其在图像和视频生成中的表现。具体方法如下：
扩散模型概述：扩散模型通过逐步去噪的方式学习数据分布，分为前向扩散和反向扩散两个阶段。前者通过添加噪声逐步破坏输入数据，后者则学习如何逐步还原数据。
WALT架构：WALT结合了时空注意力机制，以适应图像和视频的处理需求。通过窗口限制的自注意力块，模型能有效处理图像和视频帧，分别捕捉静态和动态信息。
特征提取与评估：使用探测框架从WALT模型中提取特征，并通过任务特定的读出头进行性能评估。评估任务包括图像分类、动作识别、单目深度估计和相机位姿预测等。
实验部分首先对I-WALT和V-WALT模型的特征进行了定性分析，接着进行了定量评估。通过比较图像和视频模型在多种任务上的表现，结果显示视频扩散模型在需要动态理解的任务上显著优于图像模型。具体结果包括：
定性观察：通过主成分分析可视化模型的特征，V-WALT在动态场景中的表现更为突出，能够更好地捕捉运动区域。
任务性能比较：在多个任务上，V-WALT相较于I-WALT表现出显著的性能提升，尤其是在动作识别和点跟踪任务中，表现出68%的提升。
模型规模与训练预算影响：研究还探讨了模型规模与训练预算对任务性能的影响，发现更大的模型在图像和视频分类任务中表现更佳，但在某些跟踪任务中小模型反而更具优势。
在这项研究中，我们使用了一种叫做扩散模型的技术，通过逐步去噪来生成图像和视频。我们特别关注了一种叫做WALT的模型，它可以处理图像和视频。我们的研究方法包括：
学习过程：模型首先通过添加噪声来“破坏”输入数据，然后学习如何将其恢复到原始状态。这就像我们在画画时，先用铅笔画草图，然后再用橡皮擦和颜料来完善它。
模型架构：WALT模型使用了一种叫做“窗口注意力”的方法，这种方法可以让模型在处理视频时更好地关注到动态部分，而在处理图像时则关注静态部分。
性能评估：我们通过给模型不同的任务，来测试它的表现，比如识别图像中的物体、理解视频中的动作等。这种评估方式就像给学生考试，看看他们在不同科目上的表现如何。
2025-02-11｜JagiellonianU,UCambridge
http ://arxiv.org/abs/2502.07754v1https ://github.com/gwilczynski95/meshsplats
本文提出了一种名为MeshSplats的新方法，该方法将高斯点云（GaussianSplatting，GS）转换为网格状结构，从而实现更高效的渲染。GS是一种用于3D计算机图形的先进技术，通常用于生成高质量的实时渲染效果，但在处理阴影和反射等复杂光照效果时存在局限性。MeshSplats通过将GS中的高斯元素转化为网格面，使得可以利用传统的光线追踪技术进行渲染，从而克服了GS的不足。该方法不仅保持了高质量的重建效果，还能够在现有的渲染环境中直接使用，具有良好的实用性和灵活性。
MeshSplats的核心方法包括以下几个步骤：
高斯生成：首先使用经典的GS方法创建一组高斯分布，这些分布由位置、协方差矩阵、透明度和颜色等参数定义。
高斯转网格：将高斯元素转化为网格面，保留其颜色和透明度特征。这一步骤通过将每个高斯的形状和方向近似为多个三角形来实现，从而确保网格在视觉上与高斯的表现一致。
优化与细化：在生成网格后，应用优化算法以减少可见的伪影和细化几何细节，以提高重建质量。该优化过程针对网格面而非高斯组件进行，从而在不需要额外训练的情况下提升渲染质量。
在实验部分，研究团队对MeshSplats进行了全面评估，使用了多个标准数据集（如Mip-NeRF360、TanksandTemples和DeepBlending）进行验证。通过量化指标（如SSIM、PSNR和LPIPS）比较了MeshSplats与现有技术的性能，结果表明MeshSplats在保留几何精度的同时，能够达到或超过其他最先进的方法。实验还显示，在复杂的室内场景中，MeshSplats的重建质量显著优于传统的3D高斯绘制方法。通过细化优化，MeshSplats能够有效消除伪影，提升渲染的整体视觉质量。
MeshSplats的工作原理可以用一个简单的比喻来解释：想象你在画一幅画，首先你用点（高斯分布）来表示不同的颜色和形状，然后你决定把这些点连接起来，形成一个网格（网格面），这样就能更清晰地展示你想要表达的内容。这个转换过程就像把一幅模糊的画变得更加清晰和立体。接着，为了让画看起来更真实，MeshSplats还会对这些网格进行细致的调整和优化，确保每个细节都尽可能完美。这样，最终的画作不仅视觉效果好，而且可以在许多不同的展示平台上使用，像是3D游戏或动画中。
2025-02-11｜MetaRealityLabs,UToronto,UCBerkeley｜⭐️
http ://arxiv.org/abs/2502.07785v1http ://yashkant.github.io/pippo
本研究介绍了一种名为Pippo的生成模型，能够从单张随意拍摄的照片生成高达1K分辨率的人物多视角视频。Pippo是一种多视角扩散变换器，无需额外输入，如参数模型或相机参数。模型在约30亿张人类图像上进行了预训练，随后在专业拍摄的人体数据上进行了多视角中期和后期训练。中期训练阶段通过去噪多个低分辨率视角来快速吸收数据集，并使用浅层多层感知器（MLP）粗略编码目标相机。在后期训练阶段，Pippo以高分辨率去噪较少的视角，并利用像素对齐控制（如空间锚和Plücker射线）实现3D一致性的生成。最终，Pippo通过一种注意力偏置技术，在推理时能够生成超过训练时所见的5倍视角，展现出卓越的多视角人类生成能力。
Pippo的训练采用三阶段策略：
图像预训练：在大规模人类中心数据集上进行，模型通过图像嵌入进行去噪，无需任何标签或注释。
多视角中期训练：在128×128的低分辨率下，联合去噪多达48个目标视角，并使用粗略的相机控制。此阶段不使用像素对齐的空间控制。
多视角后期训练：在1024×1024的高分辨率下，去噪1-3个视角，并通过ControlMLP模块注入空间控制，进一步提高3D一致性。
在推理过程中，Pippo能够生成平滑的360度视频，尽管训练时只联合去噪固定数量的视角。这一过程利用了注意力偏置技术，解决了在推理中增加视角数量导致内容质量下降的问题。
本研究通过多个实验验证了Pippo的有效性，使用了大规模的训练和测试数据集，包括来自iPhone的随意拍摄图像和专业拍摄的多视角数据。实验结果表明，Pippo在生成的3D一致性、身份保留以及图像质量等方面优于现有模型。特别是，我们提出的3D一致性度量方法，能够在没有成对真实数据的情况下评估生成结果的几何正确性。此外，实验还展示了Pippo在不同分辨率下的表现，表明高分辨率生成仍能保持3D一致性。
在Pippo的训练过程中，首先我们让它在大量的图片上学习如何生成图像，这个过程叫做预训练。接下来，我们让它在较低分辨率下学习如何从一张图片生成多个视角的图像，这个步骤称为中期训练。最后，我们在高分辨率下进行最后的训练，让Pippo能够更精确地生成图像，并确保这些图像在3D空间中是一致的。通过这种方法，Pippo可以从一张普通的照片中生成多个角度的高清图像，甚至可以在生成时加入一些空间信息，让图像看起来更加真实和连贯。这样一来，Pippo不仅能生成看起来很不错的图像，还能在不同的视角下保持一致性，让人感觉就像是在看同一个人从不同角度拍摄的样子。
2025-02-10｜LGElectronics,UToronto
http ://arxiv.org/abs/2502.07030v1
本文介绍了一种名为PrismAvatar的3D头部虚拟化模型，旨在实现实时动画和渲染，特别是在资源受限的边缘设备上。随着对逼真3D人类头像需求的增加，PrismAvatar结合了可变形神经辐射场（NeRF）和3D可变形模型（3DMM），通过构建一个混合的网格-体积模型来实现高效的渲染。该模型在训练时利用神经体积渲染的优势，确保在设备上以60帧每秒的速度运行，同时保持较低的内存占用。与传统桌面设备上的3D头像模型相比，PrismAvatar在图像质量上具有可比性，适用于移动设备、智能电视和车载信息娱乐系统等环境。
PrismAvatar的构建方法包括以下几个步骤：
输入处理：该方法从一系列经过抠图的头部图像中获取数据，并结合相机参数和3DMM参数进行处理。
混合模型训练：使用分析-合成方法训练一个混合的网格-体积模型。该模型基于FLAME3D可变形模型，适合大多数面部和颈部的建模。
PrismLattice构建：在FLAME网格上手动标记区域，构建一个棱柱网格，以重建复杂的头发结构。该网格通过细分和挤压操作生成多层棱柱，形成一个可变形的结构。
神经网络训练：训练三个神经网络来预测透明度、特征向量和颜色，支持体积渲染和纹理生成。
混合渲染：采用混合渲染方法，结合透明区域的体积渲染和FLAME网格的纹理渲染，以提高渲染效率。
模型导出：最终将训练得到的模型导出为带有纹理的三角网格，便于在边缘设备上进行高效动画和渲染。
在实验部分，研究团队对PrismAvatar进行了多项测试，包括使用单目视频和多视角视频数据集进行重建。通过与现有的3D头像重建方法（如PointAvatar、FLARE和INSTA）进行比较，评估了模型的性能。实验结果表明，PrismAvatar在图像质量上表现优异，尤其在发型和面部细节的重建上，能够有效捕捉复杂的几何形状和纹理。通过在移动设备上进行实时动画测试，PrismAvatar成功实现了60帧每秒的流畅输出，表明其在边缘设备上的应用潜力。
在PrismAvatar的构建过程中，首先我们需要一些经过处理的头部图像，这些图像通过抠图技术去掉了背景。接着，我们利用这些图像和一些相机参数来训练一个混合模型，这个模型可以同时处理头部的网格和体积数据。为了让模型能够准确表现头发等复杂部分，我们在头部模型上构建了一个特殊的棱柱网格，这个网格就像一个灵活的框架，可以根据头部的运动进行调整。然后，我们训练了几个神经网络来预测不同的属性，比如透明度和颜色，这样在渲染时就能得到非常真实的效果。最后，我们将训练好的模型导出为一个三角网格，这样就可以在手机等小设备上流畅地展示和动画化这些3D头像了。
2025-02-11｜Inriaet.al.,URennes,IRISA,CNRS,MPI-IS
http ://arxiv.org/abs/2502.07140v1
本文提出了一种新颖的神经隐式重建方法，旨在从稀疏视角的图像中恢复多个体的形状和外观。多人体场景的复杂性在于遮挡和杂乱的背景，这使得现有技术在处理这类场景时面临挑战。尽管一些单人重建方法取得了显著成就，但在多人体重建中，传统方法往往依赖于密集的视角和额外的分割信息。为了解决这些问题，研究者们利用几何约束，结合预先计算的SMPL人体模型，通过优化签名距离函数（SDF）来提升重建质量。此外，本文还引入了光线一致性和饱和度正则化策略，以提高在不同光照条件下的渲染效果。实验结果表明，该方法在真实和合成数据集上均表现出色，达到了当前最先进的性能。
本方法的核心是通过以下几个步骤来实现多人体的几何和外观重建：
场景表示与渲染：定义多人体表面为SDF的零水平集，利用多层感知器（MLP）来编码几何信息，并通过预训练的SMPL模型初始化该网络。
几何先验：通过独立训练SDF网络，利用SMPL模型的初步估计来正则化几何信息，确保在处理复杂场景时能够更好地捕捉细节。
混合渲染：采用混合渲染方法，将前景和背景分开处理，使用SMPL模型的边界框来定义前景体积，并通过体积渲染来提升渲染质量。
优化策略：通过最小化重建损失、Eikonal损失、光线一致性损失和饱和度损失来优化网络，确保在稀疏视角下仍能生成高质量的图像。
在实验部分，作者首先在CMUPanoptic数据集和合成的MultiHuman数据集上进行了评估。在真实数据集上，使用了多种训练视角（5、10、15和20视角），通过与现有基线方法NeuS和VolSDF进行比较，展示了该方法在表面重建和新视角合成方面的优越性。实验结果表明，本文提出的方法在处理多人体场景时，能够在较少的训练视角下仍然保持较高的重建质量，且在PSNR、SSIM等指标上均优于基线方法。此外，作者还进行了消融实验，验证了几何正则化、光线一致性和饱和度损失等各个组件在提升整体性能中的重要性。
在这个研究中，科学家们想要解决一个复杂的问题：如何从少量的照片中，准确地重建出多个正在一起的人物的形状和外观。他们的方法有几个关键步骤。首先，他们使用了一种叫做SMPL的模型来帮助理解人类的基本形状。接着，他们通过一些聪明的数学技巧，确保在处理这些复杂场景时，能更好地捕捉到细节。这就像是用一个模具来确保每次制作的形状都很一致。然后，他们将前景和背景分开处理，确保每个人的形象都能清晰呈现。最后，他们通过不断优化，确保生成的图像在不同光照条件下依然看起来真实。这种方法在测试中表现非常好，能够在只有少量照片的情况下，还能重建出高质量的图像。

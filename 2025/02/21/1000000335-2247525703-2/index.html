297-A1.2｜AIGC图像水印，AI生成媒体检测综述；统一扩散桥用于图像恢复，可逆引导一致性训练，无指导扩散用于少数样本生成

2025-02-10｜ZJU｜⭐️
http ://arxiv.org/abs/2502.06418v1
在当今人工智能生成内容（AIGC）日益普及的背景下，水印技术成为确保内容来源和检测深度伪造的重要手段。现有的水印技术通常注重抵抗真实世界的干扰（如JPEG压缩和噪声添加），但研究表明，这种“强健性”实际上会导致信息泄露的风险增加。本文提出了一种新型攻击框架，利用多通道特征提取来挖掘水印模式的泄露信息，从而实现水印的去除和伪造。与以往需要大量数据或检测器访问的研究不同，我们的方法仅需一张水印图像即可完成攻击，且在视觉保真度方面表现优异。实验结果显示，该方法在检测规避和伪造准确性上分别提高了60%和51%的成功率，揭示了现有水印技术在安全性与强健性之间的矛盾，为未来水印设计提供了新的视角。
本研究提出的攻击框架名为DAPAO，主要分为以下几个步骤：
特征提取：利用预训练的视觉模型对水印图像进行多通道特征提取，识别出可能包含水印信息的特征通道。
通道定位：通过聚类算法自动定位并优化特征通道，以便提取出水印信息。
攻击目标：分为两个主要目标：
检测规避：通过对特征通道的扰动，降低水印信息的可识别性，进而达到规避检测的目的。
伪造攻击：将提取的水印信息嵌入到无水印的干净图像中，实现伪造。
优化算法：采用投影梯度下降法（PGD）解决优化问题，以确保在扰动图像的同时保持视觉质量。
本研究通过两组数据集（COCO和DIV2K）进行实验，随机选择100张图像进行水印嵌入，并与100张无水印图像进行对比。我们测试了七种公开的水印方法，并与传统的图像降质技术（如JPEG压缩、加性高斯噪声等）以及现有的攻击方法进行了比较。评估指标包括结构相似性指数（SSIM）和峰值信噪比（PSNR），同时采用比特准确率和成功率来衡量攻击的有效性。实验结果表明，DAPAO在水印去除和伪造攻击上均显著优于现有方法，成功率分别提高了60%和51%。
我们的方法主要分为几个简单的步骤。首先，我们会使用一个已经训练好的计算机视觉模型，从水印图像中提取出不同的特征通道。这些通道就像图像的不同层次，包含了可能的水印信息。接下来，我们会通过一种聚类技术来自动找出哪些通道最有可能包含水印信息，并对其进行优化。然后，我们的目标有两个：一是通过对这些特征的轻微扰动，让水印变得难以被检测到；二是将提取到的水印信息添加到另一张干净的图像中，使其看起来像是原始水印图像。为了确保我们的操作不会影响图像的视觉效果，我们使用了一种叫做投影梯度下降的方法来解决这个问题。这一系列步骤的设计旨在有效地实现水印的去除和伪造，同时保持图像的质量。
2025-02-07｜BUPT,UCSB,CAS,CASIA｜⭐️
http ://arxiv.org/abs/2502.05240v2
随着人工智能生成媒体技术的快速发展，信息的真实性和社会信任面临着重大挑战，因此可靠的检测方法变得尤为重要。本文对AI生成媒体的检测方法进行了系统性综述，特别关注了从特定领域检测方法（Non-MLLM）到通用大型多模态语言模型（MLLM）检测方法的过渡。当前的检测方法可分为两大类：基于非MLLM的方法和基于MLLM的方法。前者依赖于深度学习技术，使用高精度的特定领域检测器，而后者则利用MLLM的通用检测器，能够处理文本、图像、音频和视频等多种输入类型，并生成高质量的文本输出。尽管这一领域取得了显著进展，但在文献中仍缺乏对这两类方法的全面比较，本文旨在填补这一空白，提供对现有方法的详细分类和分析。
本文的方法部分主要集中在对AI生成媒体检测技术的分类和比较，具体包括以下几个方面：
检测方法分类：
非MLLM检测器：这些方法通常针对特定任务优化，具有较高的检测精度，适用于资源有限的环境。
MLLM检测器：利用MLLM的跨模态处理能力，能够同时处理多种输入，进行真实性验证、可解释性分析和定位。
检测任务：
真实性检测：判断媒体样本是原创还是AI生成的，通常采用二分类任务。
可解释性：提供人类可理解的检测决策解释，通常以自然语言或视觉表现形式呈现。
定位：识别输入中的具体篡改区域，通常使用像素级或分段级分类。
技术优势：MLLM方法通过人类般的推理能力和丰富的知识获取，能够执行复杂的分析任务，并为后续分析提供灵活的框架。
在实验部分，作者对超过100项研究进行了深入评估，比较了不同检测方法的性能。通过分析这些方法在真实性检测、可解释性和定位等任务中的表现，揭示了它们的优缺点和适用场景。此外，研究还探讨了当前检测方法面临的主要挑战，特别是在安全性和伦理问题上的考量。实验结果表明，MLLM方法在处理复杂的多模态数据时表现出色，能够有效提高检测准确性，并为更复杂的任务提供有力支持。通过对现有基准和数据集的总结，本文为未来的研究方向提供了重要的参考依据。
在这一部分，我们可以更简单地理解本文的方法。首先，研究者将AI生成的媒体检测分为两大类：一种是专门针对特定任务的检测器，另一种是基于大型多模态语言模型（MLLM）的检测器。前者在特定领域内表现优异，适合在资源有限的环境中使用；而后者则能够处理多种类型的信息，比如文字、图片和视频，提供更全面的分析。
接下来，检测的任务主要有三个：首先是判断一段媒体是否是AI生成的，其次是提供关于检测结果的解释，最后是找出媒体中被篡改的具体部分。特别是MLLM方法，凭借其强大的理解和推理能力，能够在复杂的情况下提供更准确的结果，并帮助我们理解为什么会得出这样的结论。这种方法不仅提高了检测的准确性，也为后续的深入分析打下了良好的基础。
2025-02-09｜ShanghaiTech,MoEKeyLab,FDU｜⭐️
http ://arxiv.org/abs/2502.05749v2https ://github.com/UniDB-SOC/UniDB/
在这篇论文中，作者提出了UniDB，一个基于随机最优控制（SOC）的统一扩散桥架构。扩散桥模型在图像生成和恢复等领域取得了显著的进展，但现有的模型在处理任意分布之间的转换时存在局限性，主要是由于其对高斯噪声先验的固有假设。UniDB通过引入可调的终端惩罚系数来克服这些限制，从而在控制成本和终端惩罚之间实现最佳平衡，显著提高了图像细节的保留和输出质量。通过对现有扩散桥模型的统一化，UniDB展示了其在图像恢复任务中的优势，尤其是在超分辨率、去雨和修复等任务中。
UniDB的核心方法基于随机最优控制（SOC），其主要步骤如下：
构建扩散桥的前向随机微分方程（SDE）：该方程强制从预定的初始状态过渡到终端状态，确保生成的图像符合目标分布。
定义目标函数：目标函数结合了控制成本和终端惩罚，允许通过调整终端惩罚系数来优化图像细节的保留。
导出最优控制器：通过SOC问题的闭式解，UniDB能够推导出最优控制策略，以实现从初始状态到终端状态的平滑过渡。
与现有模型的整合：UniDB能够与现有的扩散桥模型（如DDBMs和GOUB）无缝集成，且只需进行最小的代码修改。
在实验部分，作者对UniDB在多个图像恢复任务中的表现进行了评估，包括超分辨率、去雨和图像修复。实验结果表明，UniDB在多个图像质量指标上超过了现有基线模型，如PSNR、SSIM、LPIPS和FID。具体来说，UniDB在超分辨率任务中显著提高了图像细节的保留，尤其在处理高频特征时表现优异。此外，作者还探讨了不同终端惩罚系数对模型性能的影响，发现适当的惩罚系数能够有效提升图像质量。
在UniDB的构建过程中，研究者们使用了一种名为随机最优控制的方法。这种方法的基本思想是通过设定一个目标，来寻找最佳的路径或策略。在图像处理的背景下，这个目标是让生成的图像尽可能接近我们想要的效果。具体来说，UniDB会首先定义一个起点和一个终点，然后通过调整一些参数（比如惩罚系数）来控制生成图像的质量。这个惩罚系数就像一个调节器，帮助模型在生成图像时平衡细节保留和整体风格。最终，UniDB不仅能生成更清晰、更真实的图像，还能与现有的图像处理模型兼容，方便使用。
2025-02-08｜NationalTaiwanU,NCTU｜⭐️
http ://arxiv.org/abs/2502.05391v1
本研究提出了一种新颖的训练框架，称为可逆引导一致性训练（iGCT），旨在提升图像生成和编辑的效率与质量。传统的扩散模型（DMs）在生成高质量图像时，通常需要大量的计算资源和时间，而iGCT通过引入数据驱动的方法，显著降低了对预训练模型的依赖，减少了计算需求。该方法解决了在高引导尺度下出现的饱和伪影问题，并通过对CIFAR-10和ImageNet64数据集的广泛实验，展示了其在FID（FréchetInceptionDistance）和精度方面的显著提升。iGCT的创新之处在于，它能够在不依赖于扩散模型的情况下，进行高效的图像生成和编辑。
iGCT的核心在于其数据驱动的引导学习机制，主要包括以下几个方面：
引导一致性训练：iGCT通过解耦目标清晰样本与源图像，捕捉无条件和条件噪声的影响。这种方法使得模型在训练过程中能够独立学习引导信息，而不依赖于复杂的预训练扩散模型。
可逆性设计：引入一个独立的噪声映射组件，该组件能够将图像映射到噪声空间，简化了图像编辑过程。这种可逆性使得在生成和编辑过程中，能够快速地进行反向映射。
一致性目标优化：iGCT通过优化一致性目标，直接推断出概率流ODE（OrdinaryDifferentialEquations）的终点，提升了生成效率和图像质量。
在实验部分，研究团队对iGCT的性能进行了全面评估，主要集中在CIFAR-10和ImageNet64数据集上。实验结果表明，iGCT在不同引导尺度下的生成效果优于传统的扩散模型和基于一致性蒸馏的方法。具体而言，iGCT在高引导尺度下显著降低了FID值，同时提升了精度和召回率。此外，iGCT在图像编辑任务中表现出色，能够在单步操作中快速完成目标类条件的图像编辑，展示了其在实时应用中的潜力。
iGCT的工作原理可以简单理解为一种智能的图像生成和编辑方式。首先，它通过学习如何将图像转化为噪声，从而掌握了图像的基本特征。接着，当我们需要生成新图像或编辑现有图像时，它能够快速地把这些噪声转回到清晰的图像中。与传统方法需要大量计算不同，iGCT只需一步就能完成这个过程，因此更快、更高效。此外，它还避免了一些常见问题，比如生成图像时的颜色失真，让结果更加自然。总的来说，iGCT是一个聪明且高效的工具，能帮助我们更轻松地处理图像。
2025-02-10｜KAIST｜⭐️
http ://arxiv.org/abs/2502.06516v1
在现代生成AI中，生成少数样本（minoritysamples）是一个重要的研究领域，尤其是在数据增强和创意内容生成等应用中。然而，现有的基于扩散的少数生成器通常依赖于计算开销较大的指导方法。为了解决这一问题，本文提出了一种简单而有效的无指导生成方法——Boost-and-Skip。该方法通过对标准生成过程进行两项最小化修改，即（i）方差增强初始化和（ii）时间步跳过，来提升少数样本的生成能力。尽管这些修改看似微不足道，但其背后有坚实的理论和实证支持，能够有效促进少数特征的出现。实验结果显示，Boost-and-Skip显著提高了少数样本的生成能力，甚至与依赖指导的最先进方法相媲美，同时计算成本显著降低，推动了该领域的进一步发展。
Boost-and-Skip方法的核心在于两项关键的修改：
方差增强初始化：在生成过程中，我们用增强的方差初始化噪声，而不是标准的高斯噪声。这种修改鼓励生成过程从低密度区域开始，增加生成少数样本的概率。具体来说，我们通过设置一个比例因子（γ>1）来放大初始化的方差，从而提高少数样本生成的可能性。
时间步跳过：为了增强低密度初始化的影响，我们在生成过程中跳过几个最初的时间步。这一策略的实施是通过选择一个合适的跳过量（Δ），确保在生成过程中依然能够保持有效的低密度引导。通过这两项修改，我们能够显著提升少数样本的生成性能，而单独使用其中任何一项修改效果有限。
本文进行了广泛的实验，以验证Boost-and-Skip方法的有效性。我们在多个真实世界的数据集上进行测试，包括CelebA、LSUN-Bedrooms和ImageNet等。实验结果表明，Boost-and-Skip在生成少数样本方面表现出色，生成的样本在质量和多样性方面均与现有的最先进方法相当，甚至在某些情况下超越了它们。此外，Boost-and-Skip的计算效率显著提高，所需的计算资源远低于依赖指导的生成方法。我们还进行了消融实验，表明同时应用方差增强和时间步跳过的组合效果最佳，进一步验证了该方法的有效性和实用性。
Boost-and-Skip方法的核心可以理解为两种简化的操作，帮助我们更好地生成少数样本。首先，想象我们在画画时，通常使用一种标准的颜色（高斯噪声）来开始，但如果我们用一种更鲜艳的颜色（增强的方差）来开始，就能更好地突出一些细微的特征，这就相当于我们在生成过程中使用了方差增强的初始化。其次，我们并不需要每一步都细致入微地进行，而是可以选择跳过一些步骤（时间步跳过），这样可以让我们更快地达到想要的效果。通过这两种方法的结合，Boost-and-Skip能有效地生成那些在数据中比较少见的样本，同时减少计算的复杂性，帮助我们在生成AI中更好地处理少数样本的问题。

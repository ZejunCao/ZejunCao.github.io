2025-02-10｜VAST,CUHK,UTAustin,ShanghaiAILab｜⭐️⭐️
http ://arxiv.org/abs/2502.06608v1
TripoSG是一种新颖的高保真3D形状生成模型，旨在解决当前3D生成技术面临的挑战，如输出质量、泛化能力和与输入条件的对齐问题。通过引入大型的整流流变换器，TripoSG在生成精细的3D网格方面达到了前所未有的精度。该模型的核心创新包括：1)基于高质量数据训练的整流流变换器，2)结合SDF、法线和Eikonal损失的混合监督训练策略，3)生成200万高质量3D样本的数据处理管道。通过这些创新，TripoSG不仅提升了3D形状生成的质量，还展示了其在多样化图像风格和内容上的强大生成能力。
TripoSG的框架主要由两个部分构成：数据构建系统和TripoSG模型。具体方法如下：
数据构建系统：
数据评分：为每个3D模型分配分数，仅高分模型进入后续处理阶段。
数据过滤：排除包含大平面基础、动画渲染错误和多对象的模型。
数据修复与增强：确保角色模型朝前，并为未纹理模型生成多视角法线图。
现场数据生成：将原始非水密网格转换为水密网格，以确保几何监督的计算。
TripoSG模型：
整流流变换器：使用标准变换器块构建的流架构，结合全局和局部图像特征进行条件生成。
混合监督策略：在训练过程中结合SDF损失、法线损失和Eikonal正则化，提高3D重建性能。
大规模训练：通过Mixture-of-Experts（MoE）架构扩展模型参数，从而提升生成能力。
通过这些方法，TripoSG能够生成高保真、几何精确的3D结构，并在多种输入条件下展示出良好的泛化能力。
在实验部分，TripoSG的性能通过一系列的设置实验进行了验证。首先，模型在大规模数据集上进行训练，以确保生成的3D形状具有高保真度和细节。实验结果显示，TripoSG在生成质量和输入对齐方面均超越了现有的3D生成模型。此外，模型还展示了出色的多样性，能够从不同风格和内容的图像中生成3D模型。通过对比实验，研究者们评估了各个组件的有效性，进一步确认了数据质量和数量对3D生成模型训练的重要性。最终，实验结果表明，TripoSG在3D形状生成领域达到了新的状态。
在TripoSG的研究中，科学家们设计了一种新的方法来生成3D模型。首先，他们创建了一个系统来处理和筛选大量的3D数据，确保只有质量高的模型用于训练。接着，他们使用了一种叫做整流流变换器的技术来帮助模型更好地理解和生成3D形状。这个技术就像是一个聪明的助手，可以从不同的图片中提取重要的信息，帮助模型生成更准确的3D图像。最后，研究团队通过大量的实验，验证了这种方法的有效性，发现TripoSG能够生成非常细致和真实的3D模型，并且可以适应不同风格的图片。这使得TripoSG在3D生成技术中占据了一个重要的位置。
2025-02-09｜PKU,BUAA,BeijingDigitalNativeDigitalCityResearchCenter,TUMunich｜AAAI2025｜⭐️🟡
http ://arxiv.org/abs/2502.05874v1https ://yangzhifeio.github.io/project/MMGDreamer
MMGDreamer是一种新型的三维室内场景生成框架，旨在通过输入提示生成可控的真实感场景。该方法在虚拟现实和室内设计等领域具有广泛的应用潜力，能够根据用户的需求精确控制场景中对象的几何形状。当前的图基方法主要依赖文本输入，缺乏对灵活用户输入的适应性，限制了对对象几何形状的精确控制。为了解决这一问题，MMGDreamer引入了一种创新的混合模态图（Mixed-ModalityGraph），结合了视觉增强模块和关系预测器，以提高生成场景的几何控制能力。实验结果表明，MMGDreamer在对象几何控制方面表现优越，达到了当前场景生成的最先进水平。
MMGDreamer的核心是混合模态图（MMG），它允许节点同时包含文本和视觉信息，从而增强对用户输入的适应能力。该方法的主要步骤包括：
场景图表示：构建一个有向图，其中节点表示场景中的对象，边表示对象之间的关系。
图形编码器：使用三元组图卷积网络（Triplet-GCN）处理场景图，更新节点和边的表示。
视觉增强模块：通过将文本信息编码为潜在向量并结合图像信息，生成节点的视觉特征，提高生成对象的几何精度。
关系预测器：推断节点之间缺失的关系，以确保生成的场景具有更好的布局一致性。
双分支扩散模型：在生成过程中，分别处理形状和布局，利用去噪器逐步生成最终的三维场景。
在实验中，MMGDreamer使用SG-FRONT数据集进行评估，结果表明其在场景生成的真实感和几何控制方面均优于现有的最先进方法。通过对比不同的场景生成方法，MMGDreamer在FID（FrechetInceptionDistance）和KID（KernelInceptionDistance）等指标上均表现出色，特别是在卧室和客厅等复杂场景中，生成的对象几何形状更为准确。此外，MMGDreamer在对象级生成质量的评估中也显著优于其他方法，展现了其在生成多样性和几何一致性方面的优势。
MMGDreamer的工作原理可以简单理解为一个智能的室内设计助手。首先，它会根据用户的要求（比如“我想要一个双人床和两个床头柜”）构建一个包含这些对象的图表。这个图表不仅会记录每个对象的类型，还会标注它们之间的关系，比如“床头柜在床的两侧”。接着，MMGDreamer会利用图像和文字信息来生成每个对象的外观，使得生成的场景不仅看起来真实，而且每个对象的形状和位置都很准确。最后，它会通过一个双重处理的方式，分别生成房间的布局和每个对象的形状，确保最终的三维场景既美观又符合用户的需求。
2025-02-10｜ShanghaiTech,Deemos,HUST｜⭐️🟡
http ://arxiv.org/abs/2502.06392v1https ://sites.google.com/view/tangled1
在当今数字媒体中，发型作为文化身份和个人表达的重要象征，其复杂性和多样性使得在计算机图形学中生成真实的3D发型变得尤为重要。现有的生成方法往往难以捕捉到多样的发型特征和细节，尤其是对于复杂的发型如辫子和卷发的表现力不足。为了解决这一问题，研究者们提出了TANGLED，一个基于多视角线条艺术的扩散框架，能够从任意风格和视角的图像中生成高质量的3D发型。该方法不仅依托于一个包含457种多样化发型的MultiHair数据集，还通过三步管道实现了对不同风格和视角的灵活适应，推动了发型生成的多样性和文化包容性。
TANGLED的方法主要分为以下几个步骤：
数据集构建：构建MultiHair数据集，包含457种发型，配有74种属性标注，涵盖多种文化和风格，特别是复杂的辫子和卷发。
扩散框架：提出一种基于多视角线条艺术的扩散模型。该模型利用线条艺术作为条件输入，提取发型的拓扑特征（如发束密度和发缝）并消除噪声，从而实现灵活的3D发型生成。
后处理模块：设计一个参数化后处理模块，以确保在生成复杂结构时保持一致性，特别是在辫子等发型的生成中，通过编码特定的几何约束来减少几何失真。
在实验中，TANGLED展示了其在多样化输入条件下生成3D发型的能力。通过与现有的发型生成模型（如HAAR和HairStep）进行比较，TANGLED在生成质量、细节保留和结构一致性等方面均表现出色。用户研究显示，参与者在真实场景和风格化场景中更偏爱TANGLED生成的发型，分别达到了63.8%和84.3%的偏好率。此外，实验结果表明，使用多视角输入显著提高了生成发型的准确性和细节捕捉能力，尤其是在处理复杂的发型结构时，TANGLED的优势更加明显。
在TANGLED的工作中，研究者们首先收集了大量不同风格的发型数据，以便更好地理解和生成各种发型。接着，他们开发了一种新的生成方法，这种方法利用了发型的线条艺术图像来帮助计算机理解发型的形状和结构。通过这种方式，计算机可以从不同的角度和风格生成3D发型，而不需要手动建模。最后，为了确保生成的发型在形状上看起来真实且一致，研究者们设计了一种后处理技术，专门处理那些复杂的发型，比如辫子。这样，TANGLED不仅能够创造出多样化和文化包容的发型，还能在动画和增强现实等领域中广泛应用。
2025-02-10｜UCambridge｜⭐️
http ://arxiv.org/abs/2502.06367v1https ://github.com/OllieBoyne/FOCUS
在这项研究中，作者提出了一种名为FOCUS的多视角足部重建方法，旨在通过合成训练的密集对应关系来生成高质量的三维足部模型。足部重建是计算机视觉领域的一个挑战性任务，尤其是在使用普通用户的手机拍摄的图像时。当前的重建方法通常依赖于大量重叠的图像和昂贵的扫描设备，而FOCUS通过引入一种新的表示方法——模板物体坐标（TOCs），结合合成数据和不确定性预测，显著提高了重建的准确性和效率。该方法在少量视角下仍能保持高质量的重建效果，并在速度上也具备优势。
FOCUS方法主要包含三个核心贡献：
合成数据集SynFoot2：该数据集扩展了现有的足部合成数据，包含100,000张高质量的足部图像，并提供了详细的表面法线和密集对应关系标签。数据集的背景和光照条件多样化，增加了足部的关节运动。
密集对应关系预测器：该预测器利用合成数据训练，能够同时预测每个像素的TOC值和对应的不确定性。这一过程采用了基于神经网络的模型，能够在合成数据上有效学习并在真实图像中进行推断。
三维表面重建方法：FOCUS提供了两种重建方法，分别是FOCUS-SfM和FOCUS-O。FOCUS-SfM采用传统的运动结构方法，通过多视图三角测量生成定向点云，而FOCUS-O则直接优化参数化的FIND模型，以适应密集对应关系的图像，生成水密的参数化网格。
在实验部分，作者对FOCUS方法的性能进行了评估，并与现有的重建方法（如COLMAP和FOUND）进行了比较。实验结果表明，FOCUS-SfM在较少的视角下表现出色，能够在10个视角下实现高质量的重建，而COLMAP则需要超过15个视角。FOCUS-O在重建精度方面也表现优异，尤其在法线重建上超越了COLMAP。速度测试显示，FOCUS方法在推理时间上显著快于现有方法，且FOCUS-SfM能够在没有GPU的情况下运行，适用于资源受限的设备。
FOCUS方法的核心是通过一种叫做模板物体坐标（TOCs）的新方式来帮助计算机理解和重建足部的三维形状。首先，研究人员创建了一个大型的合成数据集，里面有很多不同角度的足部图像，这些图像不仅清晰，而且背景和光照条件各异，增加了训练的多样性。接着，他们设计了一个智能模型，可以从这些图像中学习到每个像素的具体位置和形状，并预测出它们的不确定性。最后，FOCUS提供了两种不同的重建方法，一种是通过传统的三维重建技术来合成足部的三维模型，另一种则是直接调整一个已知的足部模型，使其更好地匹配图像中的足部形状。这种方法不仅能在较少的图像下获得好的效果，还能在速度上大幅提升，适合在普通手机上使用。
2025-02-08｜BUAA,Durham｜⭐️
http ://arxiv.org/abs/2502.05534v1
本文提出了一种新颖的框架Fg-T2M++，旨在解决文本驱动的人类运动生成中的细粒度建模问题。现有方法常常无法准确捕捉文本中描述的运动细节，主要原因在于缺乏有效的文本解析技术和对语言结构的全面建模。Fg-T2M++通过引入大语言模型（LLMs）进行语义解析，利用超曲面文本表示模块以及多模态融合模块，显著提高了运动生成的精度和多样性。通过对HumanML3D和KIT-ML数据集的广泛实验，Fg-T2M++展现了优于现有最先进方法的性能，证明了其在生成符合文本语义的运动方面的有效性。
Fg-T2M++框架由三个主要组件构成，分别是：
LLMs语义解析模块：该模块利用大语言模型从文本提示中提取细粒度的语义描述，将文本解析为各个身体部位的运动注释，深入分析词汇之间的语义角色，从而更好地理解复杂的语言结构。
超曲面文本表示模块：该模块通过构建依赖解析树并将其嵌入到超曲面空间中，捕捉文本提示的句法结构。超曲面几何具有低失真保留层次结构的优点，使得该模块能够更有效地捕捉文本中的层次关系。
多模态融合模块：该模块在条件扩散框架中，通过多个去噪步骤对文本和运动特征进行层次融合，结合全局和局部特征，以实现更全面的文本-运动映射。
在实验部分，Fg-T2M++在HumanML3D和KIT-ML数据集上进行了广泛的评估。使用多种指标进行性能评估，包括R-Precision、FrechetInceptionDistance(FID)、多模态距离（MM-Dist）等。实验结果显示，Fg-T2M++在R-Precision和FID等关键性能指标上均优于现有的最先进方法，验证了其在生成高质量、符合文本描述的细粒度运动序列方面的能力。此外，用户研究也表明，Fg-T2M++在用户偏好上表现突出，进一步巩固了其在文本驱动运动生成领域的有效性。
在Fg-T2M++的方法部分，我们使用了三种主要的技术来帮助计算机更好地理解文本并生成相应的人类运动。首先，我们利用大语言模型来解析文本，这就像是让计算机读懂文本中的每一个动作和细节，帮助它知道每个身体部位应该如何移动。接着，我们使用超曲面文本表示模块，它会把文本的结构转化为一种特殊的数学形式，这种形式能够更好地保留文本中的层次关系。最后，我们有一个多模态融合模块，它会把文本信息和运动特征结合起来，确保生成的运动不仅符合文本的整体意思，还能细致地捕捉到每个动作的细节。这样一来，计算机就能生成更加自然和准确的运动序列，真正实现了从文本到运动的精准转换。
2025-02-08｜UNCCharlotte｜⭐️
http ://arxiv.org/abs/2502.05432v1
本研究提出了一种新的大型人类运动基础模型MoFM，旨在理解复杂的人类动作语义。MoFM的设计灵感来源于基础模型（FoundationModels）在自然语言处理和计算机视觉中的成功应用。通过利用“运动书”（MotionBook），一个全面的人类运动字典，MoFM能够将人类运动映射到离散的潜在空间，从而实现高效的运动表示和理解。该模型的灵活性使其适用于多种下游任务，包括动作分类、少样本动作识别、自监督异常检测和监督异常检测。MoFM不仅能够捕捉人类运动的复杂性，还能在多种应用场景中提供强大的基础支持。
MoFM的核心方法包括以下几个步骤：
热图表示生成：通过在给定的2D骨架关键点上应用高斯函数，构建时空热图（U），并将其划分为多个热量立方体（ThermalCubes，C），用于捕捉动作的时空特征。
离散变分编码器-解码器（dVED）：设计了一个自定义的dVED模型，用于将热量立方体编码为离散的潜在表示（运动令牌）。dVED模型的编码器负责将热量立方体映射到潜在空间，而解码器则重建热图。
自监督训练：采用BERT风格的掩蔽训练，通过掩蔽热量立方体中的部分输入，训练MoFM以捕捉运动语义。该过程使模型能够在没有大量标注数据的情况下，学习人类运动的丰富表示。
任务适应性：MoFM的灵活结构允许在不同的下游任务中进行微调，只需简单的任务特定头部即可实现高效的迁移学习。
在实验部分，研究团队评估了MoFM在四个主要人类运动任务上的表现：动作分类、少样本动作识别、自监督异常检测和监督异常检测。实验结果表明，MoFM在各个任务中均能达到与当前最先进技术相媲美的性能。特别是在动作分类任务中，MoFM展示了其强大的泛化能力，能够有效适应不同的数据集和任务设置。此外，MoFM在自监督异常检测中表现出色，能够准确识别出异常行为，证明其在实际应用中的潜力。通过与现有模型的比较，MoFM的有效性和灵活性得到了充分验证，为人类运动理解领域提供了新的基础。
在这项研究中，MoFM模型的工作方式可以简单理解为一个“智能运动识别器”。首先，它通过观察人们的动作，生成一种特殊的“热图”，就像给每个动作加上了温度标签，显示出哪些地方的动作更为明显。接着，模型使用一种叫做“dVED”的工具，把这些热图转化为简单的符号，方便后续处理。这个过程就像把复杂的动作简化为易于理解的语言。然后，MoFM通过自我学习，不断提高对各种动作的理解能力，甚至在没有太多指导的情况下也能学习。最终，MoFM不仅能识别常见的动作，还能发现异常行为，比如有人在做不寻常的事情。这种能力使得MoFM在监控、医疗等多个领域都有着广泛的应用前景。

2025-02-08｜SJTU,Huawei,Tongji｜⭐️🟡
http ://arxiv.org/abs/2502.05415v1https ://github.com/zhijie-group/Show-o-Turbo
Show-oTurbo是一种新型的统一多模态理解与生成模型，旨在提升文本到图像及图像到文本生成的效率。该模型的显著特点在于其小巧的模型规模（仅1.3B参数）和开源特性。Show-oTurbo通过结合图像标记的离散扩散建模和文本标记的自回归建模，提供了一种新的生成视角。尽管Show-o在多模态生成任务中表现出色，但其推理过程面临着效率低下的问题，尤其是在生成图像和文本时需要经历数十到数百个采样步骤。为了应对这一挑战，Show-oTurbo引入了一种统一的去噪视角，并通过一致性蒸馏（ConsistencyDistillation）等策略，显著提高了生成速度和质量。
Show-oTurbo的核心方法包括以下几个关键步骤：
统一去噪视角：通过将图像和文本的生成过程视为去噪过程，建立一个统一的框架，便于同时优化这两个任务。
一致性蒸馏：采用一致性蒸馏技术，将多个文本标记的并行解码与图像标记的去噪过程结合，旨在减少生成步骤并提高效率。
轨迹分段与课程学习：将整个训练过程分为多个阶段，逐步减少每个阶段的轨迹段数，以提高模型的收敛速度和性能。
采样策略：在推理阶段，采用贪婪策略与Jacobi解码相结合，允许模型在一次前向传播中生成多个文本标记，进而加速生成过程。
在实验部分，Show-oTurbo在多个文本到图像生成（T2I）和多模态理解（MMU）任务上进行评估。通过与原始Show-o模型的对比，Show-oTurbo在不同采样步骤（如2步、4步和8步）中均表现出显著的生成速度提升，同时保持了较高的生成质量。具体而言，Show-oTurbo在4步采样下的GenEval得分为0.625，超越了原始Show-o在8步采样下的表现。此外，在图像到文本生成任务中，Show-oTurbo实现了1.5倍的推理加速，且性能损失微乎其微。实验结果表明，Show-oTurbo在多模态生成任务中不仅提高了效率，还在多个基准测试中取得了优异的表现。
Show-oTurbo的工作原理可以简单理解为：它把生成文本和图像的过程看作是去掉噪声的过程。想象一下，你在画一幅画，起初画面上都是模糊的颜色（噪声），而你需要逐步清晰化这个画面，直到它变得完美。Show-oTurbo通过一种叫“一致性蒸馏”的方法，帮助模型在生成文本时同时考虑多个信息，从而加快生成速度。它还将生成过程分成几个小阶段，逐步训练模型，确保每一步都能更快、更准确地生成所需内容。最终，Show-oTurbo能够在更少的步骤中，快速而高效地生成高质量的图像和文本，解决了传统方法的效率问题。
2025-02-10｜HSEU,AIRI,SkolkovoInstituteofScienceandTechnology｜⭐️⭐️
http ://arxiv.org/abs/2502.06606v2https ://github.com/ControlGenAI/MaterialFusion
在计算机视觉和图形学领域，操控图像中物体的材质外观至关重要，尤其在增强现实、虚拟原型制作和数字内容创作等应用中。本文提出了一种名为MaterialFusion的新框架，旨在实现高质量的材质转移。该框架允许用户调节材质应用的强度，从而在新材质特性与物体原有特征之间达到最佳平衡。MaterialFusion通过维护背景一致性和减少边界伪影，能够无缝地将修改后的物体融入场景。为了全面评估该方法，研究团队建立了一个真实世界的材质转移数据集，并进行了复杂的比较分析。通过定量评估和用户研究，结果表明，MaterialFusion在质量、用户控制和背景保留方面显著优于现有方法。
MaterialFusion的核心方法包括三个主要组成部分：StableDiffusionv1.5模型、IP-Adapter和Guide-and-Rescale（GaR）方法。具体步骤如下：
StableDiffusionv1.5模型：作为潜在扩散模型，处理低维潜在空间中的图像生成任务。
IP-Adapter：用于从源材质示例图像中提取材质特征，并将其嵌入到扩散模型中。它采用解耦交叉注意力机制，允许独立处理文本和图像特征。
Guide-and-Rescale方法：通过自我引导机制，确保在图像生成过程中保留原始图像的布局和结构，特别是在材质转移时。该方法包括两次掩膜策略，第一次在材质特征整合阶段应用，以限制材质转移到目标区域；第二次在去噪过程中进行，以维护背景一致性。
为了评估MaterialFusion的效果，研究团队创建了一个包含15个材质图像和25个物体图像的自定义数据集。通过与其他方法（如ZeST、GaR和IP-Adapter）进行定性和定量比较，结果显示MaterialFusion在材质转移效果和细节保留方面表现出色。定量分析使用了学习感知图像补丁相似性（LPIPS）和CLIP相似性分数来评估背景和物体几何形状的保留程度。用户研究表明，MaterialFusion在整体偏好、材质保真度和细节保留方面均优于ZeST，显示出其在实际应用中的优势。
MaterialFusion的工作原理可以简单理解为一个“材料魔法师”。首先，它使用一种特殊的模型（StableDiffusion），就像一个聪明的画家，可以在图像中添加新材质。然后，它通过一个“材料适配器”来识别和提取我们想要的材质特征，比如木头或金属的纹理。接下来，使用“引导和重标定”方法，确保在添加新材质时，原有物体的形状和细节不会被破坏。这个过程就像在给一件衣服换颜色，但仍然保持它的形状和风格。为了确保效果最佳，MaterialFusion还会在处理过程中使用掩膜，确保新材质只应用于目标区域，而不会影响背景。这样，最终的图像看起来既自然又真实。
2025-02-08｜HIT,USTC,WestlakeU｜⭐️🟡
http ://arxiv.org/abs/2502.05606v2https ://petershen-csworld.github.io/FreeBlend.
FreeBlend是一种新颖的概念融合框架，旨在有效地将不同的概念结合以生成新的对象。该方法通过反馈驱动的插值扩散和辅助推理机制，克服了以往方法在类别偏见和形状不匹配方面的不足。与传统的文本嵌入和结构相似性依赖方法相比，FreeBlend提供了一种更加稳健的解决方案，能够生成视觉上连贯且和谐的融合结果。通过广泛的实验，FreeBlend在语义一致性和图像质量方面均显著提升，设定了概念融合技术的新基准。
FreeBlend的核心由三部分组成：使用unCLIP的图像条件、逐步增加的插值策略和反馈驱动的去噪机制。具体方法如下：
图像条件：与传统的文本条件不同，FreeBlend利用通过unCLIP生成的图像嵌入作为条件输入，这样可以更精确地捕捉视觉细节。
逐步插值策略：在去噪过程中，FreeBlend采用逐步增加的插值策略，在每个时间步调整辅助潜变量的贡献，以确保生成图像的连贯性。
反馈驱动机制：该机制在去噪过程中动态更新辅助潜变量的影响，逐步引导融合潜变量向最终图像的生成，确保在不同阶段的细节逐渐增强。
在实验部分，研究者们构建了一个名为CTIB的数据集，包括11,400对文本-图像组合，涵盖20个不同的类别。通过对比实验，FreeBlend在多个指标上超越了现有的概念融合方法，包括CLIP-BS、DINO-BS和CLIP-IQA等。实验结果表明，FreeBlend不仅在视觉上生成了更高质量的图像，而且在用户偏好调查中也获得了最高的选择比例，验证了其在概念融合中的有效性和优越性。
FreeBlend的工作原理可以简单理解为一个“调味师”，它能够将不同的“食材”（即概念）混合在一起，创造出美味的“菜肴”（即新图像）。首先，FreeBlend使用图像而不是文字来告诉计算机想要什么样的图像，这样可以更好地捕捉细节。然后，它逐步调整这些“食材”的比例，确保每种成分都能在最终的“菜肴”中找到合适的位置。最后，FreeBlend通过不断反馈和调整，确保最终的图像既美观又和谐，就像一位经验丰富的厨师，最终呈现出一道令人满意的美食。
2025-02-09｜HSEU,AIRI｜⭐️🟡
http ://arxiv.org/abs/2502.05895v1https ://github.com/ControlGenAI/PersonGenSampler
本文探讨了个性化图像生成中的抽样技术，特别是如何在保持概念的真实性与适应新情境之间取得平衡。个性化文本到图像生成旨在根据用户定义的概念和文本描述生成定制图像。然而，现有方法在处理有限图像数据集时面临挑战，尤其是在生成高保真图像的同时，还要确保模型能够适应不同的文本描述。尽管改进的抽样策略为解决这一问题提供了可能的途径，但对这些方法的系统分析仍然相对不足。本文通过分析超类轨迹与概念轨迹的整合，提出了一种决策框架，以更好地评估和选择适合特定生成任务的抽样策略。
本文提出的研究方法主要包括以下几个方面：
抽样策略分析：系统分析不同的抽样策略，包括基于概念的抽样和基于超类的抽样，评估它们对生成结果的影响。研究不同组合的抽样方法，如切换抽样、混合抽样和掩蔽抽样。
无微调评估：在固定的微调模型上比较多种抽样方法，分析它们在概念保真度和适应性方面的表现。
选择框架：建立一个框架来选择最合适的抽样方法，考虑文本对齐、计算约束和保真度目标，以指导策略选择。通过实验确定各方法的优缺点，并提供具体的应用建议。
实验部分使用了Dreambooth数据集，涵盖了多个概念，并生成了大量的图像进行评估。为每个概念使用25个文本提示，生成7500个图像，以确保结果的可靠性。评估指标包括CLIP图像相似性和文本相似性，以测量生成图像与原始图像和文本提示之间的对齐程度。通过用户研究进一步验证不同抽样方法的效果，比较了基于抽样的生成图像与基线模型（如Dreambooth、CustomDiffusion等）的表现，以识别最佳的抽样策略。
在这个研究中，作者主要关注如何通过不同的抽样方法来改善个性化图像生成。想象一下，你有一张图片和一个描述，比如“在城市背景下的紫色狗”。为了生成符合这个描述的图像，模型需要做一些调整。研究者们发现，直接从已有的图像中学习可能导致生成的图像不够灵活，无法适应不同的描述。因此，他们提出了几种不同的抽样方法，比如“混合抽样”，这是一种结合了不同信息的方式，既能保留狗的特征，又能让它在城市背景中看起来合适。通过实验，研究者们比较了这些方法的效果，找出最适合特定任务的抽样策略，最终希望能在生成图像时取得更好的效果。
2025-02-09｜ASU｜⭐️
http ://arxiv.org/abs/2502.06023v1https ://github.com/sahsaeedi/DCPO/
在近期的研究中，针对文本到图像扩散模型的优化，提出了一种名为双重标题偏好优化（DCPO）的新方法。该方法旨在解决现有偏好数据集中存在的冲突分布问题，这种问题导致相同提示下的偏好图像和不偏好图像之间的分布重叠，从而影响模型的优化效果。此外，研究还发现输入提示中包含的无关信息会影响模型对不偏好图像的噪声预测能力。为了解决这些问题，DCPO引入了两种不同的图像标题，分别对应偏好和不偏好图像，并构建了一个新的数据集——Pick-DoubleCaption，以支持这一方法。通过三种生成标题的策略，DCPO在图像质量和与提示的相关性方面显著优于现有的多种方法，包括StableDiffusion2.1和Diffusion-DPO。
DCPO方法主要包括以下几个部分：
冲突分布问题：分析了当前偏好数据集中偏好图像和不偏好图像的分布重叠情况，指出了无关提示对模型优化的负面影响。
双重标题生成：通过引入两种不同的标题，分别为偏好图像（zw）和不偏好图像（zl），来优化扩散模型。具体方法包括：
标题生成方法（DCPO-c）：利用现有的标题生成模型为每个图像生成独特的标题。
扰动方法（DCPO-p）：通过对原始提示进行语义扰动，生成不太相关的标题，以增加偏好和不偏好图像之间的分布差异。
混合方法（DCPO-h）：结合标题生成和扰动方法，进一步提升模型的优化效果。
优化目标函数：在优化过程中，DCPO通过最大化偏好图像的生成概率，同时最小化不偏好图像的生成概率，来实现模型的优化。
在实验中，研究团队对StableDiffusion2.1模型进行了微调，并使用Pick-DoubleCaption数据集与其他方法进行了比较。实验结果表明，DCPO在多个指标上均显著优于其他方法，包括Pickscore、HPSv2.1、ImageReward和CLIPscore。特别是DCPO-h方法在优化性能上表现最佳，显示出其在处理偏好图像和不偏好图像时的有效性。此外，为了验证提出的假设，研究还进行了消融实验，探讨了扰动程度对模型性能的影响，结果显示，适度的分布差异能够有效提高模型的对齐性能。
在这项研究中，研究者们发现，现有的图像生成模型在处理人们的偏好时，常常会遇到一些问题。比如，当我们用一个提示生成图像时，生成的图像可能会有很多相似之处，以至于很难区分哪些是人们喜欢的，哪些是不喜欢的。为了解决这个问题，研究者们提出了一种新方法，叫做双重标题偏好优化（DCPO）。这个方法的核心在于为每幅图像生成两个不同的标题，一个是描述人们喜欢的图像，另一个是描述不太受欢迎的图像。这样做的好处是，可以更清晰地告诉模型哪些特征是受欢迎的，哪些是不受欢迎的，从而帮助模型更好地生成符合人们期望的图像。研究结果表明，这种方法显著提高了图像的质量和相关性，能够更好地满足人们的偏好。

本研究探讨了儿童、成人和人工智能（AI）在创意绘画任务中的表现，旨在建立一个系统化的框架来评估不同智能体之间的视觉创造力。我们收集了1338幅绘画作品，涵盖了儿童（分为两个年龄组）、成人和AI生成的图像，通过分析绘画的风格和内容来比较这些群体的创造性。研究中，我们定义了绘画的两个核心方面：风格（如墨水密度、墨水分布和元素数量）和内容（通过专家注释的类别和图像文本嵌入计算的概念多样性）。通过这些分析，我们发现不同群体在风格和内容上存在显著差异，儿童的绘画更具组成元素，AI的绘画墨水密度更高，而成人的绘画则展现出最大的概念多样性。这项研究为理解人类与人工智能的创造力提供了新的视角，并探讨了评估创造力时专家评分与自动评分之间的差异。

线条数量：通过Hough变换提取绘画的结构轮廓并统计直线数量。

内容评估则通过生成的图像嵌入和文本嵌入来计算概念多样性。

采用专家评分和自动化工具（AuDrA和OSC-figural）对绘画进行创造力评分。专家根据创造力和灵活性给出0-4的评分，而自动化工具则给出0-1的原创性评分。
在实验中，我们首先比较了儿童、成人和AI绘画在风格和内容上的差异。结果显示，AI绘画的墨水密度最高，儿童绘画的组成元素数量较多，而成人绘画则具有最大的概念多样性。具体来说，儿童在绘画中往往使用更多的独立元素，而AI的绘画则表现出较高的线条数量和墨水集中度。此外，我们还评估了不同群体的灵活性，发现成人在绘画主题上表现出更高的灵活性，而儿童在图式阶段后灵活性显著提高。通过对绘画主题的层次聚类分析，我们发现成人绘画包含更多复杂的主题，而儿童的主题则更富于想象力。AI绘画的主题多样性较低，提示的不同对AI生成内容的影响也得到了验证。
在这项研究中，我们主要想了解儿童、成人和人工智能在绘画方面的创造力有何不同。我们收集了很多绘画作品，并通过一些具体的方法来分析这些作品。首先，我们观察了绘画的风格，比如说绘画中墨水的使用情况、线条的数量等。其次，我们还分析了绘画的内容，看看每幅画表达了多少不同的主题。为了评估这些绘画的创造力，我们请专家给出评分，同时也使用了一些自动化工具来进行评分。通过这些方法，我们发现儿童的绘画通常包含更多的元素，而AI的绘画则在墨水的使用上更为集中，成人的绘画则表现出更丰富的主题和概念。这些发现帮助我们更好地理解不同智能体的创造力表现。

2025-02-10｜USTC,ZJU,HKUST(GZ)｜⭐️🟡
http ://arxiv.org/abs/2502.06527v1https ://xiaobul.github.io/CustomVideoX/
在个性化视频生成领域，尽管定制化图像生成已经取得了显著进展，个性化视频生成仍面临着时间不一致性和质量下降的挑战。为此，本文提出了CustomVideoX，一个基于视频扩散变换器（VDiT）的零-shot个性化视频生成框架。该框架通过引入3D参考注意力机制，实现了参考图像特征与视频帧的直接和同时交互，从而提高了生成视频的质量和一致性。此外，CustomVideoX还采用了时间感知注意力偏差（TAB）策略和实体区域感知增强（ERAE）模块，以动态调整参考特征的影响，确保在生成过程中保持高质量的时间一致性和视觉细节。通过建立一个新的评估基准VideoBench，本文展示了CustomVideoX在视频一致性和质量方面显著优于现有方法。
CustomVideoX的核心方法包括以下几个方面：
3D参考注意力：该机制允许参考图像特征与每一帧视频之间的灵活交互。通过在VDiT架构中直接注入参考特征，避免了传统方法中使用辅助网络引入的对齐问题。
时间感知注意力偏差（TAB）：TAB通过动态调整参考特征的影响力，确保在视频生成的不同阶段适当地调整参考图像的权重。这一机制使得在初始阶段，参考特征的影响较小，而在中间阶段逐渐增加，在最后阶段则减少，从而优化了视频的时间一致性。
实体区域感知增强（ERAE）：ERAE模块通过对关键实体区域的注意力进行动态调整，增强了视频生成过程中对重要区域的关注。这一方法通过激活阈值计算，确保生成的内容与参考图像保持一致，同时提升了生成的多样性。
在实验部分，CustomVideoX的性能通过与多种现有方法的比较进行评估。实验使用了CogVideoX-5B作为基础模型，并通过LoRA方法进行微调，确保了模型的高效性和稳定性。通过在VideoBench和DreamBench基准上进行定量评估，CustomVideoX在视频生成质量、时间一致性和主题保真度等多个指标上均表现出色。具体而言，CustomVideoX在CLIP-I和DINO-I等指标上显著优于其他方法，证明了其在保持视频一致性和细节方面的优势。此外，通过对不同模块的消融实验，验证了3D参考注意力、TAB和ERAE模块在提升生成效果中的重要性。
在CustomVideoX的工作中，有几个重要的步骤帮助我们生成个性化的视频。首先，3D参考注意力机制让我们能够将一张参考图片的信息直接融入到每一帧视频中，这样视频中的内容就能更好地反映参考图片的细节。接下来，时间感知注意力偏差则是个聪明的调整工具，它会在视频生成的不同阶段，灵活地控制参考图片的影响力。比如，开始时我们不想太依赖参考图片的信息，而在生成的中间阶段则逐渐增加这种依赖，最后又减少，这样可以保持视频的自然流畅感。最后，实体区域感知增强模块则确保了视频中重要部分的细节不会被忽视，让生成的视频既有趣又生动。通过这些方法，CustomVideoX能够生成高质量、个性化的视频，满足用户的需求。
2025-02-09｜HKUST,TencentAILab｜⭐️⭐️
http ://arxiv.org/abs/2502.05979v2https ://vfx-creator0.github.io/
本研究提出了一种名为VFXCreator的框架，旨在通过可控的扩散变换器生成动画视觉效果（VFX）。该框架的核心在于结合了空间和时间控制，能够以用户友好的方式生成动态效果。我们首先构建了Open-VFX数据集，这是首个高质量的VFX视频数据集，涵盖15种不同的效果类别，并提供了文本描述和实例分割掩码。VFXCreator通过最小化训练数据的需求，实现了实例级空间操控和精细的时间控制，从而提升了生成效果的准确性和一致性。我们的实验表明，该系统在生成真实和动态效果方面表现优异，超越了现有的技术水平。
VFXCreator的设计包括以下几个关键组成部分：
空间控制模块：通过引入掩码控制模块，用户可以进行实例级的空间操控，以便精确调整生成效果。例如，用户可以选择特定的对象或区域进行效果应用。
时间控制机制：该框架通过将开始和结束的运动时间戳与文本信息结合，使得用户可以精确控制效果的节奏和时机。这一机制通过将时间戳嵌入到扩散过程中的文本编码器中实现。
低秩适应（LoRA）：该方法允许模型在训练过程中有效地更新权重，减少计算和内存开销。通过将时间戳和文本信息的嵌入结合，VFXCreator能够更好地捕捉视频的时序动态。
在实验部分，我们对VFXCreator在Open-VFX数据集上的表现进行了全面评估。我们采用了多种评估指标，包括FID-VID、FVD和动态度，以衡量生成视频的质量和动态一致性。实验结果显示，VFXCreator在生成15种不同视觉效果时，均表现出色，尤其在生成复杂运动效果时，明显优于其他现有方法。此外，我们还进行了用户研究，评估用户对生成视频的文本对齐、对象保真度、运动流畅性和整体质量的满意度，结果表明VFXCreator在各个维度上均得到了较高的用户偏好。
VFXCreator的工作原理可以简单理解为一个智能工具，帮助用户生成电影特效。首先，用户可以选择想要的特效，比如“爆炸”或“融化”，并指定效果应用的对象。接着，系统会使用一些预先训练好的模型，自动生成这些特效的动画效果。为了让效果看起来更自然，用户还可以设置特效开始和结束的时间，使得特效在视频中的表现更加流畅和准确。通过这种方式，VFXCreator不仅节省了制作特效的时间和成本，还使得每个人都能轻松创建出高质量的视觉效果，真正实现了特效制作的“人人可用”。
2025-02-10｜CUHK,PolyU,THU,IntelliFusion,HKU,UESTC｜⭐️🟡
http ://arxiv.org/abs/2502.06734v1https ://senorita.github.io/
在近年来，视频生成技术取得了显著进展，但现有的视频编辑方法仍面临许多挑战。为了解决这些问题，本文提出了Señorita-2M，这是一个高质量的视频编辑数据集，包含约200万对视频编辑样本。该数据集通过精心设计的四个专业视频编辑模型构建，旨在提高视频编辑的效果和效率。这些模型在特定编辑任务上表现出色，并且数据集经过了一系列过滤步骤，以确保最终结果的质量。通过与当前主流的视频编辑架构结合，Señorita-2M为进一步的研究提供了坚实的基础，展示了在视频编辑领域的巨大潜力。
本文的方法主要包括以下几个方面：
视频专家的构建：
训练数据：利用Webvid-10M数据集进行训练，生成详细的字幕和对象识别。
设计与训练：开发四个视频编辑专家，包括全局风格化器、局部风格化器、文本引导的视频修复器和对象移除器。每个模型专注于特定的编辑任务，以实现最佳性能。
Señorita-2M数据集的构建：
数据来源：从Pexels网站收集高质量视频，确保数据的合法性和多样性。
编辑任务：设计了包括对象替换、局部风格转移等多种编辑任务，以满足不同的编辑需求。
过滤管道：
质量过滤：使用分类器识别成功与失败的编辑样本，确保数据集的高质量。
文本对齐：通过对比编辑样本与其对应的文本提示，排除文本对齐不佳的样本。
为了验证Señorita-2M数据集的有效性，研究团队训练了多个视频编辑模型，并与现有的编辑方法进行了比较。实验结果表明，使用Señorita-2M训练的模型在视觉质量、帧一致性和文本对齐方面均显著优于其他方法。具体来说，模型在Ewarp、CLIPScore和TemporalConsistency等多个指标上均取得了最佳成绩。此外，进行的消融研究显示，使用Señorita-2M样本的模型在相同训练条件下表现更佳，进一步证明了该数据集在提升视频编辑能力方面的有效性。
在我们的方法部分，我们主要介绍了如何创建一个强大的视频编辑系统。首先，我们训练了四个专业的视频编辑模型，每个模型都有自己擅长的任务，比如改变视频的风格、添加或去除对象等。接着，我们从一个叫Pexels的网站上收集了很多高质量的视频，这些视频是合法的，确保了数据的多样性和真实性。为了保证我们收集到的视频编辑样本的质量，我们设计了一套过滤系统，能够识别出哪些编辑效果好，哪些效果差，这样我们最终得到的数据集就会非常高质量。总的来说，我们的目标是通过这些方法，提升视频编辑的效果，让用户能够更轻松地实现他们想要的视觉效果。
2025-02-08｜XMU,PengChengLab｜⭐️🟡
http ://arxiv.org/abs/2502.05433v1https ://github.com/jidantang55/AdaFlow
随着图像扩散模型的快速发展，文本驱动的长视频编辑已经成为一个备受关注的研究领域。然而，现有方法在处理长视频时面临着巨大的内存和计算资源挑战，限制了其应用范围。为了解决这些问题，本文提出了一种新颖且无训练的长视频编辑方法，称为AdaFlow。该方法通过引入自适应注意力压缩和关键帧选择机制，显著提高了长视频的编辑效率与质量。AdaFlow能够在一次推理中处理超过1000帧的视频，表现出卓越的性能，尤其是在保持视频一致性和质量方面。
AdaFlow的核心方法包括两个创新设计：自适应注意力压缩（AAS）和自适应关键帧选择（AKS）。具体步骤如下：
关键帧选择（AKS）：根据视频内容将视频分割成多个片段，并在每个片段中选择具有代表性的关键帧。这种选择考虑了视觉内容的变化，确保在快速变化的场景中能够更频繁地采样关键帧，从而提高编辑质量。
自适应注意力压缩（AAS）：在进行关键帧翻译时，AAS会压缩不重要的键值（KV）序列，减少计算负担。通过只保留与查询相关的关键帧信息，AAS显著降低了计算复杂度，并提高了可编辑的帧数。
特征匹配的潜在传播：通过将关键帧的生成信息传播到非关键帧，确保整个视频的连续性和一致性。这一过程利用了从源视频中提取的特征，以实现流畅的编辑效果。
为了验证AdaFlow的有效性，研究团队构建了一个新的长视频编辑基准LongV-EVAL。该基准包含75个约1分钟长的视频，涵盖多种场景。实验结果表明，AdaFlow在视频质量、对象一致性和语义一致性等多个指标上均优于现有的先进方法。同时，AdaFlow的平均编辑时间仅为24分钟，远低于其他方法的40分钟以上。这些结果通过定量和定性的评估方法得到了充分支持，表明AdaFlow在长视频编辑任务中的高效性和有效性。
在AdaFlow的工作流程中，首先，我们需要从视频中提取出一些重要的画面，这些画面被称为“关键帧”。这些关键帧帮助我们理解视频的主要内容。接着，我们会使用一种叫做“自适应注意力压缩”的方法，只关注那些与我们想要编辑的内容最相关的部分。这就像我们在阅读一本书时，不会每一个字都仔细看，而是会选择重要的段落来理解大意。最后，AdaFlow会把关键帧的信息传播到视频的其他部分，确保整个视频在编辑后依然流畅自然。通过这些步骤，AdaFlow能够在一次处理中编辑出超过1000帧的视频，且视频质量和一致性都得到了很好的保持。

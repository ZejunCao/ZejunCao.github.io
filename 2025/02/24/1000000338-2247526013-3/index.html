301-A2&A3｜阶跃星辰文生视频大模型，复杂相机控制图生视频；场景级3D外观迁移，LLM驱动CAD设计

主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）发布日期｜2025年2月17日｜周一
2025-02-14｜StepFun｜⭐️⭐️
http ://arxiv.org/abs/2502.10248v2https ://github.com/stepfun-ai/Step-Video-T2V
在本技术报告中，我们介绍了Step-Video-T2V，这是一个先进的文本到视频预训练模型，具有300亿个参数，能够生成最长达204帧的视频。该模型采用了一种深度压缩变分自编码器（Video-VAE），实现了16x16的空间和8x的时间压缩比，同时保持了优异的视频重建质量。为了处理用户的中英文提示，我们设计了双语文本编码器。此外，使用了具有3D全注意力机制的扩散变换器（DiT），并通过流匹配方法进行训练，以去除输入噪声并生成潜在帧。为提高生成视频的视觉质量，我们还采用了视频基础的直接偏好优化（Video-DPO）方法。通过在新创建的基准数据集Step-Video-T2V-Eval上的评估，Step-Video-T2V展示了其在文本到视频生成中的领先性能，并探讨了当前扩散模型的局限性及未来的发展方向。
Step-Video-T2V的设计和实现包括多个关键组件：
深度压缩变分自编码器（Video-VAE）：
采用双路径架构，提供统一的空间和时间压缩，达到16x16和8x的压缩比。
通过因果3D卷积模块和中间块结合注意力机制来提高压缩效果。
双语文本编码器：
使用Hunyuan-CLIP和Step-LLM两个编码器，处理用户的中英文提示。
Hunyuan-CLIP用于生成与视觉空间对齐的文本表示，而Step-LLM则能处理长度不受限制的复杂文本序列。
扩散变换器（DiT）：
采用3D全注意力机制，优化了空间和时间信息的处理。
通过流匹配方法进行训练，预测噪声的瞬时变化，进而实现视频生成。
直接偏好优化（Video-DPO）：
通过引入人类反馈来提高生成视频的视觉质量，调整模型以更符合用户偏好。
我们在新创建的Step-Video-T2V-Eval基准数据集上评估了Step-Video-T2V的性能。该数据集包含128个多样化的提示，覆盖11个类别，并与多个开源和商业文本到视频生成引擎的结果进行了比较。实验结果表明，Step-Video-T2V在生成高质量视频方面表现优越，特别是在处理复杂的运动动态和文本内容时。此外，报告中详细讨论了在模型训练和推理过程中遇到的挑战，包括视频生成中的长时序依赖问题和复杂动作序列的生成难度。通过对这些问题的深入分析，我们为未来视频基础模型的研发提供了重要的见解和方向。
在这个研究中，我们开发了一种新的技术，叫做Step-Video-T2V，它可以根据用户输入的文本生成视频。我们首先使用一种叫做变分自编码器（VAE）的技术来压缩和处理视频数据，这样可以在不损失质量的情况下减少计算量。为了让模型理解中英文输入，我们使用了两种不同的文本编码器。接下来，我们利用一种先进的扩散变换器，这种技术能够更好地捕捉视频中的时间和空间信息。最后，我们还引入了人类反馈来优化生成的视频质量，使其更符合用户的期望。这种方法不仅提高了生成视频的质量，还帮助我们理解了在创建视频模型时可能遇到的挑战。
2025-02-14｜ZJU｜⭐️🟡
http ://arxiv.org/abs/2502.10059v1https ://zgctroy.github.io/RealCam-I2V
在图像到视频生成领域，现有的基于相机轨迹的方法在控制精度和视频质量上取得了显著进展，但在实际应用中仍面临诸多挑战。为了改善这些问题，本文提出了一种新颖的框架——RealCam-I2V。这一框架通过集成单目深度估计和三维场景重建，成功实现了复杂的相机控制，并在多个应用场景中展现出优越的性能，包括循环视频生成、帧插值和流畅场景过渡。RealCam-I2V通过对相机参数进行绝对尺度的训练，确保了不同真实场景下的一致性和可控性，显著提升了生成视频的质量和可操作性。
RealCam-I2V的核心方法包括以下几个步骤：
单目深度估计：使用深度预测模型从输入RGB图像中生成深度图，进而重建三维场景。这一过程为后续的相机控制提供了坚实的基础。
绝对尺度训练：在训练阶段，通过对相机参数进行绝对尺度的对齐，确保了生成模型对真实场景的准确理解。这一过程涉及将相机的相对参数转换为绝对值，增强了模型在实际应用中的适应性。
场景约束噪声塑形：引入了一种新机制，通过在高噪声阶段使用预览视频中的像素作为参考，来塑形生成过程。这种方法能够有效提升视频的动态表现和场景一致性。
交互式相机轨迹设计：在推理阶段，用户可以通过直观的界面在重建的三维场景中绘制相机轨迹，实时预览生成效果，从而实现更精确的相机控制。
在实验中，RealCam-I2V在RealEstate10K数据集上进行了全面评估，结果显示其在视频质量和可控性方面均显著优于现有方法。通过与多个基准模型的比较，RealCam-I2V在绝对和相对相机控制指标上均取得了显著提升，尤其在绝对尺度训练和场景约束噪声塑形的结合应用下，模型的动态表现和生成一致性得到了极大的改善。此外，实验还展示了该方法在多种复杂场景下的适用性，包括大幅度运动和快速旋转等动态场景的处理能力。
RealCam-I2V的方法可以简单理解为一个三步走的过程。首先，我们通过一款智能工具（单目深度估计）来“测量”一张图片中的深度信息，构建出一个三维模型。接着，在训练过程中，我们确保这个模型能够真实反映现实世界的尺度，这样生成的视频不会失真。最后，我们设计了一个用户友好的界面，让用户可以直接在这个三维模型上画出想要的相机轨迹，并实时看到生成的视频效果。这样，即使没有专业知识的用户，也能轻松控制相机，生成高质量的视频。
2025-02-14｜Stanford,ETHZurich,ZillowGroup｜⭐️🟡
http ://arxiv.org/abs/2502.10377v1https ://restyle3d.github.io/
本文介绍了ReStyle3D，一个新颖的框架，旨在将单一风格图像的外观转移到由多视图图像表示的真实场景中。传统的图像风格迁移方法虽然在将艺术风格应用于照片时表现出色，但在实际应用（如虚拟布置或室内装饰）中却存在不足。ReStyle3D通过结合显式语义对应关系和多视图一致性，解决了这一问题。该方法确保每个对象根据其语义匹配的纹理进行风格迁移，从而实现精确且连贯的风格化效果。通过在单视图上完成风格转移，ReStyle3D随后利用3D感知的风格提升技术，将风格扩展到多个视角，确保在多视图之间的一致性和细节的保留。
ReStyle3D的核心方法包括两个主要阶段：
单视图语义外观转移：该阶段使用一种训练无关的语义注意机制，将风格图像的外观信息转移到源图像的单一视图上。通过开放词汇的全景分割模型，ReStyle3D能够建立风格图像与源图像之间的密集实例级对应关系，确保每个对象根据其语义类别获得匹配的纹理。
多视图样式提升：在完成单视图的风格转移后，框架采用一个学习的变形与精炼网络，将样式提升到其他视角。该网络通过单目深度和逐像素光流引导，确保在多视图之间保持几何一致性，避免了传统方法中由于缺乏3D建模而导致的伪影和不一致性。
实验部分展示了ReStyle3D在结构保留、风格相似性和多视图一致性方面的优越性能。通过构建SceneTransfer基准，研究团队使用25个室内设计图像和31个室内场景进行评估，涵盖243个风格-场景组合。实验结果表明，ReStyle3D在结构保留和感知风格相似性方面均优于现有方法。此外，用户研究也表明，参与者更倾向于选择ReStyle3D生成的结果，认为其在保留结构的同时，能够更忠实地传达风格。
ReStyle3D的工作原理可以简单理解为一个两步的过程。首先，它会把你想要的风格（比如某种家具的样式）应用到一张照片上，确保每个家具或物品的外观都与风格相匹配。为了做到这一点，ReStyle3D会分析风格图像和源图像中的每个对象，确保它们在风格上是一致的。接下来，ReStyle3D会把这种风格扩展到多个视角，就像你在不同的角度看同一个房间一样。它使用一些聪明的技术来确保从不同角度看时，房间的样式和结构都保持一致，不会出现模糊或不协调的现象。这种方法不仅让房间看起来更美观，也保持了每个家具的真实感。
2025-02-13｜MIT,UWashington,Adobe｜⭐️🟡
http ://arxiv.org/abs/2502.09819v1
本研究提出了一种名为AIDL（AIDesignLanguage）的新型分层特定领域语言（DSL），旨在解决大型语言模型（LLMs）在计算机辅助设计（CAD）中的几何生成难题。尽管LLMs在各种生成任务中表现出色，但在CAD中生成程序性几何体时却面临着空间推理能力不足和复杂几何体生成所需的长程规划能力缺乏的问题。AIDL通过将空间推理的要求转移给几何约束求解器，使LLMs能够专注于高层次推理，同时在生成几何体的过程中实现更高的精确度和可编辑性。实验结果表明，AIDL在少量样本学习的情况下，生成的视觉结果更接近于输入提示，并且生成的对象更易于后处理和推理，相比于现有的OpenSCAD语言，AIDL显示出更优越的性能。
AIDL的设计围绕四个关键目标展开，旨在提升LLMs在CAD生成中的表现：
求解器辅助：AIDL允许LLMs通过指定几何关系而非直接进行计算，利用外部求解器处理复杂的几何约束，降低了模型的计算负担。
依赖关系和约束：AIDL支持隐式引用先前构建的几何体（依赖关系）和显式定义几何体之间的关系（约束），使得生成的设计更加灵活和精确。
语义化抽象：通过提供直观命名的操作，AIDL利用了LLMs在自然语言理解和处理上的优势，增强了设计操作的语义清晰度。
分层设计：AIDL采用分层结构，允许将设计过程分解为多个模块，提升了设计的可编辑性和复杂性管理能力。
通过这些设计决策，AIDL不仅提高了几何体生成的准确性，还增强了模型的可操作性和灵活性。
在实验中，研究者使用AIDL进行了一系列的2DCAD生成任务，评估了该语言在不同条件下的表现。实验流程包括：首先，向LLM提供AIDL的语法和示例程序，然后生成完整的AIDL程序。通过反复执行生成过程，直到找到一个语法正确的程序为止。实验结果显示，使用AIDL生成的几何体在视觉效果上与OpenSCAD生成的几何体相当，甚至更优。研究还通过消融实验比较了AIDL与其去掉层次和约束的变种，结果表明引入层次结构和约束能够显著提升生成模型的可编辑性。最终，研究者发现AIDL在生成准确性和可编辑性方面均优于现有的CAD语言。
AIDL的工作原理可以简单理解为一种特别为计算机设计而创建的语言。它允许设计师通过描述几何形状之间的关系，而不是直接计算每个形状的具体位置。这就像你在搭积木时，先决定哪些块要连接在一起，而不是一开始就去测量每个块的确切位置。AIDL通过使用外部求解器来处理这些关系，确保每个形状都能正确连接，同时保持设计的灵活性和可编辑性。通过这种方式，设计师可以更容易地创建复杂的形状，而不必担心每个细节的计算，从而使得设计过程变得更加直观和高效。

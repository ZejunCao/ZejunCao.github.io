2025-02-14｜Kuaishou｜ICLR2025
http ://arxiv.org/abs/2502.09925v1https ://github.com/Kwai-YuanQi/TaskGalaxy
TaskGalaxy是一个新提出的大规模多模态指令微调数据集，旨在解决现有数据集中任务多样性不足的问题。该数据集包含19,227种层次化任务类型和413,648个样本，依托GPT-4o进行任务类型的自动生成和丰富。通过从少量手动定义的任务出发，TaskGalaxy利用CLIP和GPT-4o对图像进行筛选，并生成相关的问答对。这一自动化过程显著提升了任务多样性和数据质量，减少了人工干预。将TaskGalaxy融入LLaVA-v1.5和InternVL-Chat-v1.0模型后，研究显示在16个基准测试中均有显著的性能提升，证明了任务多样性的重要性。TaskGalaxy的数据集已公开发布，供研究者和开发者使用。
TaskGalaxy的数据生成流程分为五个主要步骤，以确保高效、自动化的任务类型构建和问答对生成：
任务类型生成：首先，设计了一些基础的多模态任务类型作为种子，包括光学字符识别（OCR）、图像描述、逻辑推理等。接着，使用GPT-4o进行迭代扩展，生成更详细的任务类型。
图像收集：从多个开放数据集中收集多样化的图像，以供所有任务类型使用。
任务类型与图像匹配：利用CLIP模型，初步筛选出与每张图像强相关的任务类型，确保任务与图像之间的相关性。
问答对生成：使用GPT-4o为每种任务类型生成相关的问答对，确保内容的多样性和质量。
质量筛选：通过三个开源多模态模型对生成的任务类型和问答进行评分，确保最终数据集的高质量和准确性。
在实验部分，研究者将TaskGalaxy数据集应用于LLaVA和InternVL-Chat模型的微调，并评估其在16个基准测试中的表现。实验结果显示，使用TaskGalaxy的数据集后，LLaVA-v1.5-7B和13B模型的平均性能提升分别为4.5和3.83分，而InternVL-Chat-v1.0也有相应的提升。在MME基准测试中，LLaVA-v1.5-13B的得分增加了68分，显示出TaskGalaxy在增强模型泛化能力方面的显著效果。此外，研究还进行了消融实验，探讨了任务类型数量和样本量对模型性能的影响，结果表明任务类型的多样性和样本量的增加均能有效提升模型性能。
TaskGalaxy的生成方法可以简单理解为一个五步流程：
创建任务：首先，我们定义了一些基本任务，比如识别图像中的文字或描述图像内容。然后，使用智能程序（GPT-4o）来扩展这些任务，生成更多种类的任务。
收集图片：接着，我们从许多公开的图像库中收集了大量的图片，以便为每个任务提供合适的图像。
匹配任务和图片：然后，我们用一种叫做CLIP的工具来确保每张图片都与合适的任务匹配，确保它们之间有很强的关联性。
生成问答：接下来，GPT-4o会根据每个任务和对应的图片生成问题和答案，这样可以让模型学习如何回答不同类型的问题。
筛选质量：最后，我们使用三个不同的工具来检查这些问题和答案的质量，确保最终的数据集是高质量的。
通过这样的流程，TaskGalaxy能够生成一个包含许多不同任务和高质量问答的数据集，使得多模态模型在理解和回答问题时更加准确和全面。
2025-02-14｜KuaiShou,CASIA,NJU,USTC,PKU,Alibaba,MetaAI
http ://arxiv.org/abs/2502.10391v1https ://mm-rlhf.github.io/
在多模态大型语言模型（MLLMs）领域，尽管已有显著进展，但大多数先进模型在与人类偏好的对齐方面仍显不足。本研究提出了MM-RLHF（多模态强化学习与人类反馈），一个包含12万对精细标注的人类偏好比较的数据集，旨在填补这一空白。该数据集在规模、多样性和标注质量上均优于现有资源。通过利用MM-RLHF数据集，本文提出了多项创新，包括基于批评的奖励模型和动态奖励缩放，以提升奖励模型的质量和对齐算法的效率。实验证明，使用MM-RLHF进行微调的LLaVA-ov-7B模型在对话能力和安全性方面分别提升了19.5%和60%，展示了我们方法的有效性。
本研究的核心方法围绕MM-RLHF数据集展开，主要包括以下几个步骤：
数据收集：从多种来源收集多模态任务，确保涵盖10百万个数据实例，广泛代表各类任务。
数据选择：通过严格的重采样，提取30,000个代表性查询，确保在真实场景、数学推理和图表理解等多个领域的多样性。
模型响应生成：利用先进的模型（如Claude3.5-Sonnet和Qwen2-VL-72B）生成各类任务的响应。
精细人类标注：采用超过50名标注员进行细致的标注，评分、排名并提供文本解释，最终形成超过120,000对高质量比较对。
奖励模型：提出基于批评的奖励模型，通过生成对模型输出的批评，提供更具解释性的反馈。结合动态奖励缩放，优化高质量比较对的使用，提升训练效率。
为评估所提出的方法，研究团队在10个关键维度和27个基准上进行了广泛的实验。结果显示，MM-RLHF数据集结合MM-DPO算法显著提高了模型的整体性能。具体而言，经过微调的模型在对话能力上平均提高了11%，在安全性方面则降低了57%的不安全行为。此外，MM-RLHF-Reward-7B模型在多个奖励模型基准上达到了最先进的性能，显示出该方法的有效性和可扩展性。实验还包括对不同超参数设置的敏感性分析，进一步验证了所提方法的鲁棒性。
在我们的研究中，我们设计了一种新方法来使多模态大型语言模型（MLLMs）更好地理解和响应人类的需求。我们首先从多个来源收集了大量的数据，确保这些数据涵盖了各种任务，比如图像理解和视频分析。接着，我们从中挑选出一些代表性的例子，并利用先进的模型生成相应的回答。最重要的是，我们请了很多专业的标注员对这些回答进行评分和排名，确保每个答案的质量。最后，我们开发了一种新的奖励机制，让模型在学习过程中不仅能得到分数，还能理解为什么某些回答更好。这种方法帮助我们的模型在与人类交流时表现得更自然、更安全。
2025-02-14｜UArkansas,SUNY
http ://arxiv.org/abs/2502.09906v1https ://uark-cviu.github.io/projects/insect-foundation
本研究提出了一种新颖的多模态对话生成模型——Insect-LLaVA，旨在提升对昆虫领域知识的视觉理解。当前的多模态生成模型虽然在视觉和语言理解方面表现出色，但在昆虫识别与理解上却存在显著的知识缺口。为了解决这一问题，研究团队首先构建了一个大规模的多模态昆虫数据集，包含丰富的视觉指令数据，以支持模型学习昆虫的视觉与语义特征。此外，研究还引入了一种新型的昆虫基础模型，通过微特征自监督学习和基于补丁的相关注意机制，捕捉昆虫图像中的细微差异。实验结果表明，该模型在可视化昆虫理解任务上表现优异，超越了现有的标准基准。
本研究的方法主要包括以下几个步骤：
数据集构建：创建了一个名为“MultimodalInsectDataset”的大规模昆虫数据集，包含超过一百万张昆虫图像，涵盖34,212种不同的昆虫。数据集的标签体系采用了六级层次结构，包括亚门、纲、目、科、属和种。
视觉昆虫特征对齐预训练：通过随机生成描述昆虫图像的问题，并将其与相应的昆虫描述配对，构建预训练数据集。模型在这一阶段仅更新多层感知器连接器的权重，而固定视觉编码器和大语言模型的权重。
微特征自监督学习：提出了一种基于补丁的相关注意机制，旨在捕捉昆虫图像中的微小特征差异。该机制帮助模型在处理昆虫图像时，能够有效识别和区分细微的特征。
在实验部分，研究团队设计了多个基准测试，包括昆虫分类、检测和视觉问答（Insect-VQA）等任务，以评估Insect-LLaVA模型的性能。通过与现有方法的对比，结果显示该模型在各项任务上均取得了显著的性能提升，特别是在视觉昆虫理解的准确性和效率方面。此外，实验还验证了所提出的微特征自监督学习策略对模型学习细微特征的重要性，进一步证明了数据集的多样性和模型设计的有效性。
在这项研究中，科学家们想要让计算机更好地理解昆虫。为了做到这一点，他们首先创建了一个包含大量昆虫图片和详细描述的数据集。接下来，他们设计了一种新模型，能够通过学习这些图片和描述之间的关系来识别昆虫的不同特征。这个模型特别关注昆虫身上的小细节，比如它们的颜色、形状和纹理。通过这种方式，模型不仅能识别出昆虫的种类，还能回答与昆虫相关的问题，比如它们的生活习性和栖息环境。最终，实验结果表明，这个新模型在理解昆虫方面表现得非常出色，远超以前的方法。
2025-02-13｜UCambridge,UAlberta,IndependentResearcher,HKU,UTübingen,Oxford,AuburnU
http ://arxiv.org/abs/2502.09696v1https ://zerobench.github.io/
在当今的多模态模型（LMMs）研究中，尽管这些模型在语言任务上展现了显著的能力，但在视觉理解和推理方面却存在显著不足。为了解决这个问题，研究团队提出了ZeroBench，这是一个全新的视觉推理基准，旨在挑战现有的前沿模型。ZeroBench包含100个精心设计的问题和334个相关的子问题，所有参与评估的20个LMMs在这一基准上均得分为0.0%。通过对模型错误的深入分析，研究者希望推动视觉理解的进步，并为未来的模型评估提供一个长期有效的标准。
ZeroBench的构建方法主要分为以下几个步骤：
问题设计：研究团队由20多位创作者组成，手动设计了100个问题，确保每个问题都包含复杂的视觉元素和多步骤推理。问题的设计旨在超出当前模型的能力范围。
问题筛选：通过四步筛选流程，包括反馈、初步评估、审查和对候选问题的逐步改进，确保最终选出的100个问题具有足够的挑战性。这一过程还包括对问题的多样性和质量的严格把控。
子问题生成：为每个主要问题创建了334个子问题，以便更细致地评估模型的表现。这些子问题通过分解主要问题的步骤而来，确保模型在回答时能够展示出其推理能力。
评估标准：使用pass@1和pass@5等多种评估指标来量化模型在ZeroBench上的表现，并进行严格的错误分析，以识别模型在视觉理解和推理方面的共同失误。
在实验中，研究团队对20个前沿的LMMs进行了评估，发现所有模型在ZeroBench的主要问题上均得分为0.0%。虽然在子问题上部分模型表现出微弱的正确率，但整体结果显示这些模型在面对ZeroBench中的挑战时，仍然无法有效回答问题。通过对模型输出的细致分析，研究者识别出了多种错误类型，主要集中在视觉解释错误而非逻辑推理错误上。这一发现进一步证实了ZeroBench作为一个视觉推理基准的有效性，并强调了当前模型在视觉理解方面的不足。
在ZeroBench的构建过程中，研究团队采取了几个简单但有效的步骤来确保问题的难度和多样性。首先，他们设计了一系列复杂的问题，确保这些问题不仅需要视觉观察，还需要多步骤的推理。接着，他们通过严格的筛选流程，从大量候选问题中挑选出最具挑战性的100个问题。为了更好地评估模型的能力，每个主要问题还被分解成多个子问题，这样可以更清晰地看到模型在每一步的表现。最后，通过一系列标准来评估模型的答案，研究者发现这些现代模型在面对这些问题时几乎没有成功。这些步骤的设计旨在推动模型在视觉理解方面的进步，同时为未来的研究提供了一个可靠的评估工具。
2025-02-12｜AlibabaGroup｜🟡
http ://arxiv.org/abs/2502.09660v1
在图像和视频的细粒度交互式分割任务中，现有的基础分割模型如SegmentAnythingModels（SAMs）虽然展现了良好的泛化能力，但在处理需要高精度掩码的场景时，仍然存在性能下降的问题。为了解决这一问题，本文提出了一种新的框架——SAM2Refiner。该框架在SAM2的基础上，设计了一个局部增强模块，以利用局部上下文信息来增强全局特征，同时引入了掩码细化模块，以生成更高分辨率的分割掩码。通过广泛的实验，结果表明，SAM2Refiner在图像和视频分割任务中均能生成高精度的分割结果，超越了现有的最先进方法。

局部增强（LA）：将输入图像均匀切分为四个子图像，并通过交叉注意力机制将局部特征与全局特征进行融合，以提取细粒度信息。
提示重定向（PR）：通过精确对齐提示特征与增强的对象特征，提升模型对复杂结构的响应能力。该模块包含自注意力、交叉注意力和多层感知机（MLP）更新等步骤，以增强对象嵌入的敏感性。
掩码细化（MR）：结合对象特征与多尺度特征生成高分辨率的掩码，采用U-Net结构进行特征融合，确保细节的保留。
在实验部分，研究者在多个细粒度分割基准上评估了SAM2Refiner的性能，包括DIS、ThinObject-5K、COIFT和HR-SOD等数据集。在图像分割任务中，SAM2Refiner表现出色，mIoU和mBIoU指标显著高于其他方法。此外，针对视频对象分割，SAM2Refiner在LVOS-v1和LVOS-v2数据集上也取得了优异的成绩，充分验证了其在动态场景中的有效性和鲁棒性。
在我们的研究中，我们提出了一种新的方法，叫做SAM2Refiner，旨在提高图像和视频中的物体分割效果。这个方法就像一个聪明的助手，能帮助我们更好地识别和分割图像中的细节。我们的方法有三个主要部分：第一部分是局部增强，它会把一幅图像分成几个小块，帮助我们更好地理解每个部分的细节；第二部分是提示重定向，它会根据我们给出的提示（比如点击某个点）来调整模型的注意力，让它更关注复杂的结构；第三部分是掩码细化，它会把不同层次的图像信息结合起来，生成更清晰的分割结果。通过这些方法，我们的模型在处理图像和视频时，能够比其他方法更准确地识别出物体的边界和细节。
2025-02-14｜UCBerkeley,TranscEngram,MicrosoftResearch,HKU
http ://arxiv.org/abs/2502.10385v1https ://robinwu218.github.io/SimDINO
本文提出了一种名为SimDINO和SimDINOv2的简化模型，旨在改进现有的DINO和DINOv2模型在自监督学习（SSL）中的表现。这些模型在处理大规模无标签图像数据时，通常面临复杂的训练流程和众多超参数的调节。研究者们指出，现有模型在训练过程中容易出现表示崩溃的问题，即模型可能学习到将所有输入映射到相同输出的简单解决方案。为了解决这一问题，SimDINO通过引入显式的编码率正则化项，简化了训练管道。这种方法不仅减少了模型的复杂性，还提高了在下游任务上的表现，展现出更强的鲁棒性和更高的表示质量。
本研究的方法分为几个关键部分：
原始DINO管道回顾：DINO的目标是通过多视图学习来获取输入图像的聚合表示。模型通过随机采样不同的视图（如局部和全局裁剪），并确保这些视图生成的特征尽可能一致。训练过程中采用了教师-学生网络架构，其中学生网络的表示与教师网络的表示进行对齐。
从DINO到SimDINO：在SimDINO中，研究者们提出使用简单的欧几里得距离来替代复杂的后处理步骤，如高维softmax和中心化操作。通过直接比较局部和全局视图的特征，SimDINO实现了有效的非崩溃约束。
从DINOv2到SimDINOv2：SimDINOv2在SimDINO的基础上进一步简化，通过引入“iBOT头”和对特征的中心化操作来增强模型的性能。SimDINOv2同时关注于聚合特征和局部特征的学习，确保模型在处理不同视图时能够有效地提取信息。
在实验部分，研究者们对SimDINO和SimDINOv2进行了全面的评估，比较了其与原始DINO和DINOv2模型在多个下游任务上的表现。通过在ImageNet-1K数据集上进行训练，SimDINO和SimDINOv2在分类准确率上均超越了原始模型。此外，研究者们还在COCO、ADE20K和DAVIS-2017等数据集上进行无监督目标检测和分割任务的评估，结果显示，简化后的模型在这些任务中表现出更强的能力和更高的稳定性，证明了其在实际应用中的有效性。
在这项研究中，研究人员发现，现有的图像学习模型在训练时需要调节很多复杂的参数，容易出现问题。为了简化这一过程，他们设计了两种新模型SimDINO和SimDINOv2。简单来说，SimDINO和SimDINOv2通过减少模型的复杂性，使用简单的数学方法来比较不同视图的特征，从而避免了模型学习到简单而无效的解决方案。这种方法不仅让模型训练变得更容易，还提高了模型在实际应用中的表现，比如图像分类和目标检测等任务。通过这些改进，研究人员展示了如何在保持性能的同时，让学习过程变得更简单和稳定。
2025-02-14｜CMU,SJTU,HuaweiNoah’sArkLab,SCUT
http ://arxiv.org/abs/2502.09873v2
在图像恢复领域，扩散模型已经展现出了显著的成功，但其多步骤去噪过程带来了高昂的计算开销，限制了其实际应用。现有的方法在处理重度JPEG压缩伪影时表现不佳，尤其是在高度压缩的图像中。为了解决这些问题，本文提出了一种名为CODiff的压缩感知一体化扩散模型，专门用于JPEG伪影去除。CODiff的核心是压缩感知视觉嵌入器（CaVE），它提取并利用JPEG压缩先验信息来指导扩散模型的去噪过程。此外，作者还提出了一种双重学习策略，使CaVE能够在显式和隐式学习之间进行有效结合，从而增强模型对JPEG压缩的理解。实验结果表明，CODiff在定量和视觉质量指标上均优于现有的领先方法。
CODiff的设计包括两个主要阶段：第一阶段是训练压缩感知视觉嵌入器（CaVE），第二阶段是将CaVE集成到扩散模型中进行JPEG伪影去除。具体方法如下：
压缩感知视觉嵌入器（CaVE）：
显式学习：CaVE通过预测低质量图像的质量因子（QF）来明确学习JPEG压缩先验。
隐式学习：CaVE还学习从压缩输入重建高质量图像，增强对JPEG压缩过程的理解。
一体化扩散模型：
将低质量图像编码为潜在表示，然后通过单步去噪操作预测噪声，最终恢复高质量图像。
结合多种损失函数（如MSE损失、DISTS损失和GAN损失）来优化模型性能。
通过这种双重学习和一体化的设计，CODiff能够有效捕捉JPEG压缩特征，从而实现高质量的图像恢复。
在实验中，CODiff使用DF2K数据集进行训练和测试，涵盖多种图像质量因子（QF）。评估指标包括LPIPS、DISTS、MUSIQ等，涵盖了全参考和无参考的图像质量评估。结果显示，CODiff在LIVE-1、Urban100和DIV2K-Val等数据集上均优于其他对比方法，尤其是在高压缩图像（如QF=5）的恢复效果上。通过与现有的最先进方法进行比较，CODiff在图像重建的视觉质量和计算效率上均显示出明显的优势。此外，CODiff的复杂性分析表明，其所需的计算资源显著低于传统的多步骤扩散模型。
CODiff的工作原理可以简单理解为一个两步过程。首先，它通过一个叫做CaVE的工具来学习JPEG压缩的特征。CaVE会学习如何从低质量的图像中提取出有用的信息，比如图像的质量因子（QF），这就像是给图像打分，告诉模型这幅图有多清晰。然后，CODiff使用这些信息来帮助模型在一次性处理图像时，快速去除压缩带来的伪影，恢复出更清晰的图像。通过这种方法，CODiff不仅提高了处理速度，还能更好地保留图像的细节，使得恢复后的图像看起来更自然、更真实。

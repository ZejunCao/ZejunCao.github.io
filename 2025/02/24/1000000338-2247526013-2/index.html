2025-02-14｜PolyU,CAS,UCAS,CASIA｜⭐️🟡
http ://arxiv.org/abs/2502.09963v1
在本研究中，我们提出了一种名为RSIDiff的自我进化扩散模型，旨在通过递归自我训练提高图像生成的质量。传统的扩散模型在使用合成数据进行训练时常常遭遇训练崩溃的问题，这主要是由于缺乏感知对齐和生成幻觉的累积。为了解决这些问题，我们提出了三种策略：构建和筛选高质量的提示，采用偏好采样方法以识别和过滤出人类偏好的样本，以及通过分布加权机制来惩罚可能引入幻觉的样本。通过这些方法，RSIDiff能够在自我改进的过程中生成具有更高美学质量的图像，从而克服传统方法的局限性。
我们的研究方法主要包括以下几个步骤：
提示构建和筛选：我们从用户活跃的图像合成网站收集了大量提示，并通过清晰度、特异性和多样性进行筛选，以确保生成的图像更符合人类的认知偏好。
偏好采样：通过引入偏好采样策略，我们利用自动化指标评估生成样本的质量，优先选择与人类偏好高度一致的样本，从而减少生成幻觉的影响。
分布加权机制：我们设计了一种分布加权策略，对生成样本进行权重评估，确保在训练过程中优先使用符合主要分布的样本，减少离群样本带来的负面影响。
这些策略相辅相成，共同促进了模型的自我改进，使得RSIDiff在图像生成任务中表现出色。
为了验证所提出方法的有效性，我们在两个代表性的数据集上进行了实验：PartiPrompts和HPS测试集。实验设置包括使用StableDiffusion1.4作为基础模型，在每轮训练中生成5000个提示，并从中选择300个样本作为训练集。我们进行了多轮递归训练，评估生成图像的文本对齐度和美学质量。实验结果表明，RSIDiff在视觉吸引力和文本忠实度方面均显著优于基础模型和监督微调方法，尤其在第六轮训练时表现最佳。此外，用户研究也显示RSIDiff生成的图像更符合人类的偏好，进一步验证了我们方法的有效性。
在我们的研究中，我们开发了一种新方法来帮助计算机生成更好看的图像。我们的方法分为三个简单的步骤。首先，我们从互联网上收集了很多描述图像的文字（我们称之为“提示”），并挑选出那些清晰、具体且多样化的提示，确保它们能帮助计算机更好地理解我们想要的图像。其次，我们通过一种叫做“偏好采样”的技术，优先选择那些人们更喜欢的图像，这样可以避免计算机生成奇怪或不合适的图像。最后，我们还设计了一种方法来给不同的图像分配权重，这样计算机在学习时可以更关注那些更符合我们期望的图像。通过这三个步骤，我们的模型能够不断自我改进，最终生成的图像更加美观和符合人类的审美。
2025-02-14｜NUS,MicrosoftResearch｜⭐️🟡
http ://arxiv.org/abs/2502.10389v1https ://github.com/microsoft/RAS
本文提出了一种新的采样策略——区域自适应采样（RAS），旨在提升扩散模型（DiffusionModels,DMs）在生成任务中的实时性能。尽管扩散模型在图像合成、超分辨率等领域表现出色，但其依赖于多个顺序前向传递的特性使得实时应用受到限制。传统的加速方法主要集中在减少采样步骤或重用中间结果，未能充分利用图像中不同区域的变化。通过利用扩散变换器（DiffusionTransformers,DiTs）处理可变数量的令牌，RAS能够根据模型的关注点动态分配不同的采样比例，从而有效地更新图像的语义重要区域，显著提升生成速度和质量。
RAS的核心思想是根据当前图像的关注区域动态调整采样比例。具体方法包括：
区域识别：通过观察模型输出的噪声标准差，识别出需要快速更新的区域（快更新区域）和可以重用前一步噪声的区域（慢更新区域）。
动态采样比例：在每个采样步骤中，针对快更新区域进行高频次更新，而慢更新区域则保持之前的噪声输出，从而减少计算负担。
缓存机制：在每个步骤中缓存关键和数值信息，以便在后续步骤中重用，进一步提升效率。
调度优化：在扩散过程的早期阶段保持较高的采样比例，以确保生成图像的基础结构稳定，随后逐渐降低采样比例以提高效率。
在实验中，RAS被应用于StableDiffusion3和Lumina-Next-T2I模型，评估其在文本到图像生成任务中的表现。通过对比不同的采样配置，结果显示RAS在保持图像质量的同时，能够实现高达2.51倍的速度提升。此外，用户研究表明，RAS生成的图像在质量上与传统方法相当，同时在速度上具有显著优势。实验还对不同采样步骤和比例进行了详细分析，结果表明，RAS在各项评估指标上均表现优异，证明了其在实际应用中的有效性。
区域自适应采样（RAS）是一种聪明的策略，它帮助生成图像的计算机程序更快地工作。想象一下，当你画画时，有些地方需要多次细致描绘，而其他地方只需轻轻一笔。RAS就像是这样的画家，它可以识别出图像中重要的部分，并给这些部分更多的注意力。具体来说，RAS会在每一步生成图像时，先找出哪些区域是当前最重要的，然后集中资源去处理这些区域，而不那么重要的地方则可以用之前的结果来代替，这样就省去了很多不必要的计算。这样一来，生成图像的速度就快了很多，同时质量也没有受到影响。通过这种方式，RAS让计算机生成图像的过程变得更高效，也更符合人类的思维方式。
2025-02-14｜SamsungResearchIndia｜ICASSP2025｜⭐️🟡
http ://arxiv.org/abs/2502.10258v1
PromptArtisan是一种创新的多指令图像编辑方法，能够在单次操作中实现复杂的图像编辑，而无需耗时的迭代优化。该方法允许用户为图像中的每个特定区域提供多个编辑指令，并通过相应的掩码进行关联。这种灵活性使得用户可以进行涉及掩码交集或重叠的复杂编辑，极大地丰富了图像转化的可能性。PromptArtisan利用预训练的InstructPix2Pix模型和新颖的完全注意力控制机制（CACM），确保用户指令的精确执行，从而实现对编辑过程的细致控制。此外，该方法具备零-shot特性，无需额外训练，且在处理复杂性上优于传统的迭代方法，适合各类用户的创意和高效图像编辑需求。
PromptArtisan的核心在于其方法论，主要包括以下几个方面：
多提示嵌入生成：该方法使用掩码-提示对作为输入。每个提示的嵌入需单独计算，以确保其在特定掩码区域内的含义不受其他提示的影响。最终形成一个固定长度的嵌入序列，便于后续处理。
完全注意力控制机制（CACM）：CACM的设计旨在调节交叉注意力和自注意力层，以实现更细致的编辑控制。它确保每个提示仅在对应的掩码区域内产生影响，从而避免不必要的干扰，同时增强与用户指令的一致性。
复合掩码管理：PromptArtisan通过复合掩码的方式整合多个掩码-提示对，允许在同一图像中进行多区域的编辑。该机制确保在编辑过程中，掩码之间的互动不会影响最终生成的内容，从而提升编辑的精确性和质量。
为了评估PromptArtisan的有效性，研究团队与当前主流的文本与掩码基础图像编辑方法进行了比较，包括GLIDE、BlendedLatentDiffusion、DiffEdit和InstructPix2Pix。实验使用了新创建的基准数据集MiE-Bench，包含30个测试样本。通过定性、定量和主观分析，PromptArtisan在视觉质量和指令遵循方面均表现优异。用户研究显示，参与者普遍认为PromptArtisan在处理复杂指令和重叠掩码方面能力突出，最终生成的图像在美学特征上得到了最高评价。此外，定量指标如CLIP分数和PickScore也表明，PromptArtisan在忠实执行用户指令方面优于其他方法。
PromptArtisan的工作原理可以简单理解为一个智能的图像编辑助手。首先，它允许用户通过简单的指令来告诉计算机想要对图像做什么，比如改变颜色或添加某个物体。每个指令都有一个对应的区域（掩码），这样计算机就能清楚地知道在图像的哪个部分进行修改。其次，PromptArtisan使用了一种特殊的机制，确保每个指令只影响它应该影响的区域，而不会干扰到其他部分。这就像在绘画时，使用遮罩来避免颜色溢出到不该染色的地方。最后，PromptArtisan能够在一次操作中完成所有修改，而不需要反复调整，极大地提高了编辑效率。这种方法让任何人都能轻松进行复杂的图像编辑，无论他们的技术水平如何。
2025-02-14｜ULisbon,FaculdadedeCiências｜COLING2025｜⭐️🟡
http ://arxiv.org/abs/2502.10064v1
本研究提出了一种新颖的无监督图像编辑方法，旨在通过自然语言指令来实现图像的修改，而不需要特定的标注、掩膜或训练。这一方法通过整合预训练的大型语言模型（LLM）和图像生成模型，利用用户提供的编辑指令生成描述，并据此进行图像修改。传统的指令引导图像编辑方法依赖于监督学习和任务特定的数据集，存在标注成本高、领域适应性差等问题。相比之下，本文的方法通过消除这些障碍，展现出良好的编辑效果，并在多个基准测试中与现有的最先进方法相媲美，具有较大的改进潜力。
本方法的核心在于利用预训练的LLM生成图像的描述，并通过计算输入和输出图像描述的差异向量来指导图像编辑。具体步骤如下：
图像描述生成：使用LLM生成输入图像的描述和输出图像的描述，以便形成编辑方向。
差异向量计算：通过对比输入和输出图像的描述，计算出一个差异向量，指示需要进行的修改。
图像重构：将差异向量与输入图像结合，利用扩散模型生成最终的编辑图像。
去噪处理：采用DDIM（去噪扩散隐式模型）技术，将输入图像转化为噪声，然后再重构为目标图像，确保编辑效果符合指令要求。
实验部分使用了MAGICBRUSH数据集，对提出的方法进行了评估。通过与其他现有方法进行比较，验证了本方法在图像编辑任务中的有效性。实验结果显示，本文的方法在多项指标上均表现出色，尤其是在生成质量和编辑指令的遵循度上。此外，采用人类评估方法，比较了本方法与InstructPix2Pix和ZONE等先进方法的效果，结果表明本方法在可接受性上具有一定优势。
在这项研究中，研究人员开发了一种新方式，让计算机根据人们的自然语言指令来修改图像。想象一下，你有一张照片，想让照片中的某个物体变成不同的颜色或形状。传统的方法通常需要大量的手动标记和训练，而这个新方法则不需要。它使用一种大型语言模型，首先理解你给出的指令，然后生成描述这张图的文字。接着，它计算出原图和目标图之间的差别，并利用这些信息来创建修改后的图像。这样，计算机就能更智能地理解你想要的效果，而不需要事先学习很多特定的案例。
2025-02-14｜WarsawUniversityofTechnology,CISPA｜ICLR2025｜⭐️🟡
http ://arxiv.org/abs/2502.09935v1https ://t2i-text-loc.github.io/
在本研究中，作者探讨了扩散模型在生成图像时如何有效地处理文本内容。尽管现有的扩散模型在图像合成方面表现出色，但研究发现，文本生成的实际影响仅源于模型中不到1%的参数，主要集中在注意力层。基于这一发现，作者提出了一种新的方法，通过定位负责文本生成的交叉和联合注意力层，显著提高了文本生成的效率和性能。此外，研究展示了如何利用这些定位的层进行文本编辑，以及在生成过程中防止有害文本的生成，强调了该方法在多种扩散模型架构中的广泛适用性。
本研究的方法主要包括以下几个步骤：
激活补丁技术：利用激活补丁技术来识别扩散模型中负责生成文本内容的交叉和联合注意力层。研究表明，这些层在不同模型架构中表现出一致性。
选择性微调：针对定位到的层进行微调，采用低秩适配（LoRA）技术，以提高文本生成质量，同时保持模型的整体生成能力。通过仅微调这小部分参数，研究者能够在不损失多样性的情况下，显著提升文本生成的效果。
文本编辑与去除有害文本：研究展示了如何通过补丁技术进行图像中的文本编辑，并能够在生成过程中实时替换有害文本。这一过程不需要额外的数据或训练，且不会增加计算负担。
实验部分通过在多个基准数据集上评估提出的方法，验证其有效性。研究采用了两个基准：SimpleBench和CreativeBench，分别包含400个简单和复杂的提示。通过比较OCRF1分数、图像对齐度（SSIM和PSNR）等指标，评估生成文本的质量和图像的视觉一致性。实验结果表明，使用定位层的微调方法能够显著提高文本生成的准确性，同时保持图像质量。此外，研究还表明，利用该技术可以有效地去除生成图像中的有害文本，且在多种扩散模型上均取得了良好的效果。
在这项研究中，科学家们发现，生成图像时，模型中只有一小部分参数负责生成图像中的文本。为了提升文本生成效果，他们使用了一种叫做“激活补丁”的技术，来找出这些特定的参数。接着，他们只对这些参数进行微调，而不是对整个模型进行改动，这样既能提高文本的质量，又能保持生成图像的多样性。此外，研究者们还展示了如何利用这些找到的参数来编辑图像中的文本，甚至可以在生成图像时自动去掉不当的文字内容。这种方法不仅高效，而且适用于多种不同的图像生成模型，使得图像生成的过程更加安全和可靠。
2025-02-12｜DeakinU｜⭐️
http ://arxiv.org/abs/2502.09655v1https ://github.com/kvmduc/BDBM
本研究提出了一种新颖的双向扩散桥模型（BidirectionalDiffusionBridgeModel,BDBM），旨在解决现有图像对图像（I2I）转换任务中单向模型的局限性。传统方法通常需要分别训练前向和反向模型，这不仅增加了计算成本，还限制了模型的实际应用。BDBM通过一个单一网络实现了双向翻译，利用查普曼-科尔莫戈罗夫方程（Chapman-KolmogorovEquation）来处理数据分布在时间步长上的变化。特别地，当给定端点的边际分布为高斯分布时，BDBM的转移核在两个方向上都可以得到解析形式，从而实现高效学习。通过与现有桥接方法的比较，BDBM在高分辨率I2I转换任务中显示出优越的性能。
BDBM的核心方法基于查普曼-科尔莫戈罗夫方程，构建一个双条件马尔可夫过程（DCMP）以实现双向转换。具体步骤包括：
建模桥接过程：通过将初始分布设为Dirac分布，定义从一个域到另一个域的转移过程。
双条件马尔可夫过程：利用CKE将条件分布转化为边际分布的建模问题，确保在不同时间点间的分布一致性。
高斯分布的应用：当边际分布为高斯时，转移核可以简化为高斯形式，便于进行有效的学习和推断。
统一模型结构：通过掩码机制，使用单一网络来同时处理前向和反向的转移，避免了资源的重复消耗。
为了验证BDBM的有效性，研究团队在多个流行的I2I转换数据集上进行了广泛的实验，包括Edges↔Shoes、Edges↔Handbags等。实验结果表明，BDBM在视觉质量（通过FID评估）和感知相似性（通过LPIPS评估）上均优于现有的单向扩散桥模型。具体而言，BDBM在相同或更少的训练迭代下，显著提高了生成样本的质量。此外，BDBM在双向翻译任务中表现出更高的稳定性和一致性，进一步证明了其在实际应用中的潜力。
BDBM的工作原理可以想象成一条双向的桥，允许我们在两个不同的图像领域之间自由行走。传统的方法就像是分别建造两座桥，一座连接从图像A到图像B，另一座则连接从图像B到图像A，这样的做法不仅浪费时间，还需要更多的资源。BDBM则通过一个单一的桥来解决这个问题。它利用数学公式（查普曼-科尔莫戈罗夫方程）来确保无论是从A到B，还是从B到A，桥的两端都能保持一致。通过这种方式，BDBM不仅提高了生成图像的质量，还节省了计算资源，让图像转换变得更高效。

302-A1.2｜β自适应缩放CFG，无需CFG的模型引导；图像反演综述，解耦CLIP中类别与风格特征用于特定域生成，LDM水印

2025-02-14｜JagiellonianU,USTWrocław｜⭐️🟡
http ://arxiv.org/abs/2502.10574v1https ://github.com/gmum/beta-CFG
在现代文本驱动的扩散模型中，分类器无关引导（CFG）是一个重要机制。本文提出了β-CFG（β自适应缩放的分类器无关引导），旨在解决生成图像质量与与提示的对应性之间的权衡。在强引导下，生成的图像与条件文本完全匹配，但质量下降；而弱引导则能生成高质量图像，但与提示不符。β-CFG通过动态调整引导影响，利用基于梯度的自适应归一化和单模态β分布的时间依赖曲线，来稳定引导效果，从而改善生成图像的质量与提示的匹配度。实验结果表明，β-CFG在FID评分上优于传统的CFG，同时保持了与参考CFG相似的文本到图像的CLIP相似性评分。
β-CFG方法包括以下几个关键步骤：
动态引导控制：通过引入β分布，动态调整引导强度，确保在不同的扩散阶段能够平衡图像质量与提示的一致性。
梯度归一化：采用梯度基的自适应归一化技术来稳定引导过程，减少生成样本的多样性损失。
时间依赖性调整：利用单模态β分布的时间依赖曲线，调节引导强度在扩散过程中的变化，确保在初期阶段引导影响较小，而在中间阶段逐渐增加，最后再减少引导影响。
实现简便性：β-CFG模型的实现相对简单，易于集成到现有的扩散模型中，提供更好的生成效果。
实验部分包括多个方面的评估，首先是对β-CFG在2D示例中的视觉表现进行展示，然后通过定量和定性比较，评估其相对于传统CFG和CFG++模型的性能。使用50NFEDDIM采样的SDv1.5和SDXL模型，评估生成图像的FID评分和CLIP相似度。结果显示，β-CFG在FID评分上表现优越，且CLIP相似度保持稳定。此外，通过消融研究，分析了β分布参数和引导强度参数对生成质量的影响，验证了模型的有效性和灵活性。
在β-CFG方法中，研究人员通过一种叫做“动态引导控制”的技术来改善计算机生成图像的质量。这就像在画画时，画家可以根据不同的阶段调整颜色的浓淡。在图像生成的初期，β-CFG会让计算机生成的图像与提示的关联性较小，确保图像的多样性；在中间阶段，计算机会根据提示逐渐调整图像的细节，使其更符合要求；最后，在生成的尾声阶段，又会减少这种调整，以确保最终的图像质量。这样的调整方法使得生成的图像不仅与提示相符，而且质量更高，减少了生成不合适图像的可能性。
2025-02-17｜THU,MSRA｜⭐️🟡
http ://arxiv.org/abs/2502.12154v1https ://github.com/tzco/Diffusion-wo-CFG
本论文提出了一种新颖的目标函数——模型引导（Model-guidance,MG），旨在解决传统扩散模型中常用的无分类器引导（Classifier-freeGuidance,CFG）所带来的问题。MG方法不仅超越了单纯的数据分布建模，还引入了条件的后验概率，显著提升了训练效率和生成质量。通过简单的代码修改，MG可以作为现有模型的即插即用模块，训练速度提高了六倍以上，推理速度加倍，同时在多个数据集上表现出色，特别是在ImageNet256基准测试中取得了1.34的FID分数，达到了当前最先进的水平。
MG方法的核心在于重新思考无分类器引导的实现，主要包括以下几个步骤：
后验概率整合：MG通过将模型本身作为隐式分类器，直接学习条件的后验概率，从而优化生成过程。
模型引导损失函数：在训练过程中，MG使用修改后的损失函数，结合条件和无条件模型的概率，进行联合优化，提升生成质量和对条件的对齐。
实现细节：MG需要的仅是对现有模型进行简单的代码修改，支持多种超参数设置，如引导比例的自动调整，以适应不同的生成任务。
高效推理：MG方法在推理时只需一次前向传播，大幅减少了计算成本，相比于CFG需要两次前向传播，显著提高了推理速度。
为验证MG方法的有效性，研究者在多个标准数据集上进行了广泛实验，特别是在ImageNet256×256上进行了系统性比较。实验结果表明，应用MG方法后，模型在FID、InceptionScore等指标上均有显著提升，且在高分辨率图像生成任务中表现优异。此外，研究还通过消融实验探讨了超参数对模型性能的影响，结果显示MG在不同模型和数据集上均具有良好的可扩展性和适应性。
在这项研究中，研究者提出了一种新的方法，叫做模型引导（MG），目的是让图像生成模型更高效。传统的方法需要两次计算来生成图像，而MG只需要一次，这样可以大大节省时间和计算资源。MG的工作原理是通过模型自身来判断生成图像的质量，而不是依赖额外的分类器。这样做的好处是，生成的图像质量更高，同时训练速度也快得多。简单来说，MG就像给模型装上了一个“智能引导”，让它更聪明地生成图像，减少了不必要的计算，让整个过程变得更加流畅和高效。
2025-02-17｜ZJU,TencentYouTu,SWU,SJTU,NJU｜⭐️🟡
http ://arxiv.org/abs/2502.11974v1https ://github.com/RyanChenYN/ImageInversion
图像反演是生成模型中的一项基本任务，旨在将图像映射回其潜在表示，从而支持后续的图像编辑、恢复和风格转移等应用。随着StyleGAN系列的出现，图像反演技术的准确性和效率得到了显著提升。然而，现有的图像反演方法仍存在一定的局限性，包括基于编码器的单向方法通常效果不佳，而基于优化的方法则耗时较长，难以满足一般图像编辑和高精度应用的需求。近年来，扩散模型因其强大的生成能力和稳定的训练过程而受到青睐。本文综述了图像反演技术的最新进展，重点关注两大主要范式：生成对抗网络（GAN）反演和扩散模型反演。我们根据优化方法对这些技术进行了分类，分析了它们的理论基础、技术创新及实际权衡，并探讨了当前的挑战和未来的研究方向。
在方法部分，我们将图像反演技术分为两大类：GAN反演和扩散模型反演。对于GAN反演，主要有三种方法：
基于编码器的方法：这些方法通过构建直接的图像到潜在空间的映射来实现快速反演，注重速度和条件控制。
潜在优化方法：这类方法直接优化预训练GAN的潜在变量，以实现高重建精度，通常不依赖于额外的模块。
混合方法：结合了编码器和潜在优化的优点，通常先用编码器提供初步的潜在代码，然后通过优化进行细化，以实现更复杂的编辑任务。
对于扩散模型反演，方法则分为三类：
无训练方法：直接利用预训练的扩散模型进行图像反演，无需额外训练，强调高效性和灵活性。
微调方法：通过调整预训练模型的参数来提高其在特定任务上的适应性和性能。
额外可训练模块的方法：通过引入新模块或调整网络结构，增强扩散模型的功能，特别是在图像编辑任务中表现突出。
在实验部分，本文总结了近年来在图像反演领域的主要数据集和应用任务，包括对象编辑、属性编辑、风格迁移和图像恢复等。我们分析了不同方法在这些任务上的表现，利用LAION、FFHQ等数据集进行评估。这些数据集涵盖了多种类型的图像，确保了反演任务的多样性和有效性。此外，我们还探讨了在图像反演中面临的挑战，如重建精度与可编辑性之间的权衡，以及在多模态和跨领域反演中的困难。通过这些实验，本文为未来的研究方向提供了重要的参考。
在图像反演的过程中，我们主要是将现实中的图像转化为一种计算机可以理解的“潜在代码”，这个过程可以帮助我们对图像进行各种编辑和修改。我们的方法分为两大类：一种是基于生成对抗网络（GAN），另一种是基于扩散模型。对于GAN，我们可以通过三种方式来实现反演：第一种是直接通过编码器快速获得潜在代码，第二种是通过优化现有的潜在代码来提高图像的重建质量，第三种是结合前两者的优点，先用编码器得到一个大致的潜在代码，再进行细化。而对于扩散模型，我们可以选择不进行额外训练的方法，直接利用已有的模型，或者通过微调模型参数来提高性能，甚至可以添加新的模块来增强模型的功能。这样，我们就能更灵活地处理图像反演任务，让图像编辑变得更加高效和精准。
2025-02-17｜PKU,TencentWechatAI｜⭐️🟡
http ://arxiv.org/abs/2502.11532v1
近年来，文本到图像生成模型的发展取得了显著进展，尤其是在生成高质量图像方面。然而，这些模型在处理特定领域的风格时常常面临挑战，主要是因为现有的CLIP文本编码器在捕捉特定领域的语义时表现不佳。为了解决这一问题，本文提出了Control-CLIP框架，旨在解耦图像生成中的类别和风格特征。通过对CLIP模型进行特别设计的微调，Control-CLIP能够更好地理解文本描述中的分类信息和风格特征，从而提高生成模型在特定领域的表现。实验结果表明，该方法在多个领域的图像生成任务中表现出色，尤其是在生成与文本描述高度一致的图像方面。
Control-CLIP的核心在于对风格和类别特征的解耦。具体方法包括：
双编码器设计：引入风格编码器和类别编码器，分别独立进行微调，以增强对风格和类别特征的识别能力。
损失函数优化：采用交叉熵损失函数来分别最小化风格分类误差和最大化类别分类误差，从而有效分离风格特征和类别特征。
适配器模块：设计一个包含两个可学习线性层的适配器模块，以捕捉领域特定的风格和类别特征，优化学习过程。
交叉注意力机制：在生成过程中，通过交叉注意力机制将风格和类别特征结合到StableDiffusion模型中，实现对图像生成的精确控制。
在实验部分，Control-CLIP在多个图像分类数据集（如ImageNet-R、PACS等）上进行了评估。实验设置包括：
数据集选择：选择具有明确风格和类别标签的数据集进行训练和测试。
训练策略：采用16样本设置进行微调，并评估模型在风格和类别识别任务上的表现。
生成质量评估：通过用户评分和生成质量的客观指标（如FID和CLIP得分）来评估生成结果的质量。实验结果显示，Control-CLIP在风格和类别识别的准确性上均优于现有的微调方法，证明了其在特定领域图像生成的有效性。
Control-CLIP的工作原理可以理解为将图像生成过程中的“内容”和“风格”分开处理。想象一下，当你想要生成一幅特定风格的猫的图像时，Control-CLIP通过两个不同的“翻译器”来理解你的需求：一个专注于“猫”的类别，另一个关注“风格”，比如“动漫风格”或“油画风格”。通过这种方式，模型能够更准确地抓住你想要的风格和内容。此外，Control-CLIP使用了一种新的方法来结合这两个翻译器的结果，使得生成的图像不仅符合你的描述，还能保持多样性和高质量。这种方法不仅提高了生成图像的准确性，还让模型在处理不同风格时更加灵活和高效。
2025-02-14｜ECNU,HIT｜⭐️
http ://arxiv.org/abs/2502.10495v1
在图像生成领域，潜在扩散模型（LatentDiffusionModels,LDMs）因其高效和高质量的图像生成能力而受到广泛关注。然而，这一技术的迅速发展也带来了版权侵犯和生成内容滥用的严重担忧。现有的水印技术往往在生成图像中嵌入固定信号，导致这些水印易被恶意攻击者检测到。为了解决这一问题，本文提出了一种新颖的方法——SWA-LDM，它通过随机化嵌入过程来增强水印的隐蔽性，从而有效避免被检测。同时，该方法在保持图像质量和水印鲁棒性的前提下，增强了生成图像的版权保护和内容完整性。
SWA-LDM的核心思想是通过随机化水印嵌入过程来提高水印的隐蔽性。具体方法包括以下几个步骤：
水印嵌入：在潜在噪声中初始化并分割成密钥通道、噪声通道和水印通道。密钥通道用来生成随机种子，以确保每个图像的水印信息都是唯一的。
密钥构建：使用伪随机数生成器（PRNG）和洗牌算法对水印和噪声通道进行随机化，确保水印信息分散在潜在空间中。
水印验证：通过反向扩散方法恢复图像的潜在表示，并提取密钥以验证水印的存在。该过程能有效抵抗图像传输中的噪声和失真。
SWA-LDM的设计使得水印信息的嵌入和提取过程高效且不需要额外的管理开销，从而增强了水印的隐蔽性和鲁棒性。
在实验部分，研究者使用了三种广泛应用的LDM模型（StableDiffusionv1-5、v2-1和SDXL1.0）进行评估。通过生成大量图像，评估了SWA-LDM在水印隐蔽性和鲁棒性方面的表现。实验结果显示，SWA-LDM在防御水印存在攻击方面表现优异，相较于其他基线方法（如Tree-Ring和GaussianShading），其水印隐蔽性显著提高。此外，SWA-LDM在图像质量评估上也保持了与原始方法相当的水平，表明其在实际应用中的有效性和可行性。
SWA-LDM是一种新方法，旨在让生成的图像水印更隐蔽。想象一下，你在一张照片上添加了一个秘密标记，通常这个标记是固定的，容易被别人发现。SWA-LDM的做法是让这个标记变得随机化，每张照片的标记都是独特的。具体来说，它会把图像的潜在信息分成几个部分，其中一部分用来存放这个秘密标记。然后，它使用一种巧妙的方式来混合这些信息，确保即使有人试图查找这个标记，也很难找到。最后，当需要验证图像是否被水印标记时，SWA-LDM可以通过一种特殊的方法恢复出原始信息，从而确认是否有这个标记。这样一来，生成的图像既能保持高质量，又能有效保护版权。

2025-02-17｜NTU,ByteDanceSeed,A*STAR｜⭐️⭐️
http ://arxiv.org/abs/2502.12135v2https ://chaoyuesong.github.io/MagicArticulate
随着3D内容创作的迅速发展，市场对能够支持真实动画的关节化3D模型的需求日益增加。传统方法通常依赖于专业艺术家手动标注骨骼和权重，这不仅耗时且需要丰富的专业知识。为了解决这一问题，本文提出了MagicArticulate框架，旨在自动将静态3D模型转换为可进行动画处理的关节化版本。该方法的关键贡献包括：引入Articulation-XL数据集，包含超过33,000个高质量的关节化注释模型；提出了一种新的骨骼生成方法，通过自回归变换器处理不同3D模型中骨骼和关节的变动；并通过功能扩散过程预测皮肤权重，显著提高了在各种对象类别中的表现。
MagicArticulate的实现分为两个主要阶段：骨骼生成和皮肤权重预测。
骨骼生成：
使用自回归变换器将骨骼生成任务重新表述为序列建模问题，能够处理不同3D模型中骨骼和关节的变化。
通过对输入3D网格进行采样，提取点云并将其编码为固定长度的形状特征，作为生成骨骼的条件输入。
设计了一种分层序列排序策略，以确保生成的骨骼结构符合关节之间的层级关系。
皮肤权重预测：
采用功能扩散框架，利用体积测地距离先验信息来平滑地预测皮肤权重。
通过将皮肤权重视为在网格表面上的连续函数，能够捕捉权重之间的平滑过渡，适应复杂的网格拓扑结构。
模型在训练时利用真实的骨骼和皮肤权重进行监督学习，以提高预测的准确性。
在实验部分，MagicArticulate在Articulation-XL和ModelsResource两个数据集上进行了评估。实验结果表明，MagicArticulate在骨骼生成和皮肤权重预测方面均显著优于现有方法。具体而言，使用多种标准度量（如Chamfer距离）评估骨骼质量时，MagicArticulate的结果表现出更高的准确性和一致性。此外，皮肤权重的预测也通过精度、召回率和L1范数误差等指标显示出优越的性能。通过与传统模板方法和学习方法的比较，MagicArticulate在处理多样化对象类别时展现了更强的泛化能力，证明了其在实际动画制作中的应用潜力。
MagicArticulate的工作原理可以简单理解为一个两步过程。首先，它会分析输入的3D模型，并自动生成一个“骨架”，就像给一个玩具娃娃装上骨头一样。这个骨架的生成是通过一种智能算法来完成的，它能根据不同模型的形状灵活调整骨架的结构，确保每个关节都在合适的位置。接下来，MagicArticulate会计算每个关节对模型表面不同部分的影响力度，这个过程叫做“皮肤权重预测”。想象一下，如果你在玩具上按下某个关节，玩具的其他部分会如何随之移动。通过这种方法，MagicArticulate可以确保生成的模型在动画中看起来自然且流畅，而不需要人工干预。
2025-02-17｜ShanghaiTech,HKUST,HKU｜ICLR2025｜⭐️🟡
http ://arxiv.org/abs/2502.11697v1https ://soolab.github.io/MVTokenFlow
本文提出了一种名为MVTokenFlow的新型框架，旨在从单目视频中生成高质量的四维（4D）内容。尽管近年来生成模型（如视频扩散模型和多视图扩散模型）在二维和三维内容生成方面取得了显著进展，但在动态4D内容的创建中，保持空间和时间的一致性仍然是一项挑战。MVTokenFlow通过利用多视图扩散模型生成不同时间步的多视图图像，确保了不同视点间的空间一致性，并重建出合理的粗略4D场。随后，利用渲染的二维流作为指导，进一步再生所有多视图图像，以提高时间一致性。最终，生成的图像被用来精炼粗略的4D场，从而获得高质量的4D场。实验结果表明，该方法的有效性明显优于基线方法。
MVTokenFlow的核心方法分为三个主要步骤：
多视图视频生成：首先，利用预训练的多视图扩散模型Era3D，从输入的单目视频生成多个视角下的图像。该模型不仅生成RGB图像，还预测法线图。通过对每帧应用多视图扩散模型，确保在同一视点下的图像保持一致性，但不同时间步之间的图像可能存在时间不一致性。
动态3D高斯场重建：接下来，从生成的多视图视频中重建动态3D高斯场。该方法采用关键点控制的动态3D高斯表示，结合静态3D高斯和时间相关的变形场。通过训练过程，使用多种损失函数（如流损失和法线损失）来优化模型，以确保生成的场在几何上合理且具备高质量的渲染效果。
基于2D流的再生与精炼：最后，利用粗略的动态3D高斯场指导再生多视图图像。通过渲染的2D流来改善生成图像的时间一致性，再生的图像用于精炼动态3D高斯场，从而提高最终生成结果的质量和一致性。
在实验部分，研究者对MVTokenFlow进行了全面的评估，使用了Consistent4D数据集进行定量和定性比较。实验结果表明，MVTokenFlow在空间一致性、时间一致性和生成质量等方面均优于现有的基线方法。具体来说，作者通过PSNR、SSIM和LPIPS等指标评估了与参考视频的一致性，并使用CLIP分数来测量多视图一致性。定性比较中，MVTokenFlow在复杂场景下生成的结果显示出更好的细节和一致性，且在不同视点下的表现优于其他方法。此外，消融实验表明，流损失和关键帧特征传播对提高时间一致性和生成质量具有显著影响。
MVTokenFlow的工作原理可以简单理解为三个步骤：首先，模型从一段普通视频中提取多个视角的图像，就像我们在看一部电影时从不同的角度拍摄一样。接着，这些图像被用来构建一个动态的3D模型，这个模型可以随着时间的变化而变形，确保它在不同时间点和不同视角下都能保持一致。最后，模型通过分析这些图像之间的关系，重新生成更高质量的图像，确保在观看时不会出现模糊或不自然的效果。这样的过程使得生成的视频不仅看起来真实，而且在不同角度和时间上都能保持一致性，适用于虚拟现实、动画等多个领域。
2025-02-17｜KAIST,SKKU｜⭐️🟡
http ://arxiv.org/abs/2502.11642v1
在本研究中，我们提出了一种名为GaussianMotion的新型人类渲染模型，旨在从文本描述生成完全可动画的三维人类模型。与现有方法相比，GaussianMotion在实现高保真度和高效渲染方面表现出色，特别是在处理任意姿态时。该方法结合了可变形的三维高斯点和基于姿态的文本到三维分数蒸馏技术，使得模型能够在端到端的训练过程中捕捉到自然动作的细节。通过密集生成不同随机姿态，GaussianMotion有效地扩展了模型的运动范围，确保生成的三维人类模型不仅在静态状态下保持高质量，还能在动态场景中表现出色。
GaussianMotion的核心方法包括以下几个步骤：
高斯点优化：在规范空间中定义高斯点，这些点的位置信息通过输入姿态进行调整，从而实现对不同姿态的适应。
姿态可变形性：使用线性混合皮肤（LBS）函数，将三维高斯点从规范空间变换到观察空间，以适应用户指定的姿态。
动态姿态指导：在训练过程中，随机采样姿态，确保模型能够学习到多样化的姿态变化，并通过预训练的姿态条件扩散模型进行引导。
自适应分数蒸馏：提出了一种新的蒸馏策略，平衡细节保留与平滑性，以优化生成结果。该策略根据不同的时间步长选择性调整去噪分数，确保在生成过程中保持细节清晰与流畅。
为了验证GaussianMotion的有效性，我们进行了广泛的实验并与现有的最新技术进行了比较。在静态姿态生成和动画生成两个设置中，我们评估了模型的几何质量、纹理质量和文本一致性。实验结果表明，GaussianMotion在所有指标上均优于对比方法，生成的三维人类模型展现出更细致的几何结构和纹理细节。此外，通过用户研究，我们进一步确认了模型在视觉质量和文本一致性方面的优越性，用户普遍偏好我们的方法。
在GaussianMotion的工作中，我们的方法可以简单理解为一种“智能拼图”游戏。首先，我们在一个标准的三维空间中放置一些小点，这些点就像拼图的块。接下来，我们根据用户给出的姿态（比如“把手抬起来”）来移动这些点，使它们适应新的形状。为了让我们的拼图块能够灵活地变换姿势，我们在训练时会随机选择不同的姿态来帮助模型学习。最后，我们引入了一种聪明的“评分系统”，它会根据生成的图像质量来调整我们的拼图块，确保最后的图像既清晰又自然。通过这种方式，GaussianMotion可以创造出既生动又符合用户要求的三维人类模型。
2025-02-17｜HuzhouU,ZJU,Moxin,SUTD,WenzhouU｜⭐️⭐️
http ://arxiv.org/abs/2502.11586v1https ://syllables-to-scenes.github.io/
本研究提出了一种创新的方法，通过分析日本俳句，将其转化为可导航的三维场景，旨在保留诗歌的情感和语义深度。该方法名为HaikuVerse，利用先进的生成模型，特别是大型语言模型（LLM）和3D高斯点云技术，来解析和可视化抽象的文学概念。研究聚焦于俳句这一简洁而富有表现力的诗歌形式，通过对其内涵的深入理解，创建出高质量的三维场景，这不仅展现了传统文化的美，还为用户提供了沉浸式体验。此方法为文化遗产的数字化保存提供了新的视角，并展示了技术如何促进人类情感的表达。
本研究的方法分为两个主要阶段：层次化文学批评理论基础解析（H-LCTGP）和渐进维度合成（PDS）。
层次化文学批评理论基础解析（H-LCTGP）：
翻译与欣赏：首先，使用大型语言模型对俳句进行文化和历史背景的分析，理解其情感和象征意义。
推理与关键元素提取：通过LLM提取诗句中的关键意象和情感，分析其隐含的哲学意义。
LLM增强：生成更详细的文本提示，以便为后续的视觉化提供更丰富的内容基础。
渐进维度合成（PDS）：
从平面图像到全景图像：将俳句转换为高质量的2D图像，并生成360度全景图。
深度信息提取：从全景图中提取深度信息，为3D场景初始化提供支持。
3D高斯点云生成：利用3D高斯点云技术构建三维场景，确保场景的语义和几何一致性。
实时图像增强：在渲染过程中应用实时图像增强技术，提高视觉质量，确保用户能够从多个角度清晰地观察生成的3D场景。
在实验部分，研究团队使用了包含50个日本俳句的特定数据集，评估其方法的有效性。通过与两种现有的文本到3D场景生成方法（SceneScape和Dreamscene360）进行比较，结果显示本研究的方法在内容一致性和场景连贯性方面表现优越。用户研究通过5分制的平均意见评分（MOS）系统对生成的3D场景的准确性和整体质量进行评估，结果表明，使用LLM解析的场景获得了更高的评分，进一步验证了该方法在保持文学忠实度和空间一致性方面的有效性。
在我们的研究中，我们首先需要理解日本俳句的深层含义。我们使用一种叫做大型语言模型的技术，这种技术就像一个非常聪明的助手，能够分析诗句的情感和意象。接下来，我们将这些分析结果转化为生动的图像。我们先把诗句变成平面的图像，然后再将这些图像扩展成360度的全景图，最后利用这些全景图生成一个立体的三维场景。这个过程中，我们不仅要确保图像的质量，还要让用户能够在这个三维空间中自由探索，就像走进了一幅美丽的画作一样。这种方法让我们能够更好地体验和理解古老的诗歌，同时也为文化遗产的保护提供了新的方式。
2025-02-17｜SJTU,TencentXRVisionLabs,ANU｜⭐️⭐️
http ://arxiv.org/abs/2502.11390v1
在本研究中，提出了一种名为MARS（MeshAutoRegressiveModelfor3DShapeDetailization）的新型方法，旨在提升3D模型的细节化处理。现有的细节化方法多依赖生成对抗网络（GANs），其局限性在于无法在不同细节层次（LOD）之间保持几何一致性。MARS通过引入多层次、多类别的3D网格表示，学习在潜在空间中实现形状一致的网格表示。此外，该方法结合了自回归模型，能够通过预测下一个细节层次的网格标记来生成细致的3D形状。通过在挑战性3D形状细节化基准测试上的广泛实验，MARS在定性和定量性能上均优于现有方法，显示出其在细节生成和形状完整性保持方面的显著优势。
MARS的核心方法包括以下几个关键步骤：
多层次3DVQVAE：该模型首先采用多层次的3D向量量化变分自编码器（VQVAE），将输入的3D网格编码为多个细节层次的离散标记。这种编码方式使得模型能够有效捕捉不同细节层次的几何信息。
几何一致性监督：为确保模型在不同细节层次之间保持几何一致性，MARS引入了一种几何一致性监督机制。这一机制确保每个多层次标记都能对生成网格的整体质量和几何一致性做出贡献。
自回归模型生成：在完成VQVAE训练后，MARS利用自回归模型生成更高细节层次的网格标记。该模型通过逐步预测下一个细节层次的标记，从而实现3D形状的细节化处理。这一过程不仅提高了生成的细节质量，还保持了形状的一致性。
为验证MARS的有效性，研究团队进行了多项实验，包括定性和定量比较。定性比较中，MARS与当前最先进的细节化方法ShaDDR和DECOLLAGE进行了对比，结果显示MARS在生成复杂几何细节方面表现优越。定量比较则采用了严格的指标，如严格的交并比（Strict-IOU）和F-Score，结果表明MARS在各项指标上均超越了基线模型。此外，研究还进行了消融实验，探讨几何一致性监督和VQVAE码本大小对模型性能的影响，结果进一步证实了MARS在不同细节层次保持几何一致性的重要性。
MARS方法的核心可以看作是一个智能的3D模型细化工具。首先，它把粗糙的3D模型分成不同的细节层次，就像把一幅画分成几个不同的层次，逐层添加细节。接着，MARS利用一种叫做几何一致性监督的技巧，确保每一层的细节都能与其他层协调一致，避免出现不协调的地方。最后，MARS通过一个自回归模型，像拼图一样逐步预测和生成更细致的部分，最终形成一个完整的、细致的3D模型。这样的方法不仅能让生成的模型更美观，还能确保它们的形状保持一致，适合各种应用场景。
2025-02-15｜ShandongU,NUDT,SRIIIT｜⭐️🟡
http ://arxiv.org/abs/2502.10675v1
本文提出了一种基于预训练的大型语言模型（LLM）的分层结构开放词汇室内场景合成方法。室内场景合成旨在自动生成符合用户需求的多样化、真实的三维室内场景。尽管现有方法在生成场景描述方面表现良好，但直接将这些描述转换为物理可行的场景布局仍存在挑战。为了解决这一问题，作者引入了一种分层场景描述作为中介表示，采用三层结构：整个场景为根节点，功能区域为内部节点，物体为叶节点。该方法通过训练一个层次感知网络，推断物体之间的细粒度相对位置，并设计了一种分而治之的优化策略，以生成更合理的场景布局，从而更好地满足用户需求。
本研究的方法分为三个主要阶段：
层次结构生成：首先，使用预训练的LLM生成包含场景描述的层次结构。该结构包括场景的根节点、功能区域的内部节点及物体的叶节点，确保物体间的关系得到合理表示。
细粒度相对位置推断：接下来，训练一个层次感知图神经网络，以推断同一功能区域内物体的相对位置。该网络利用文本空间关系作为基础，能够在开放词汇设置中推断出合理的相对位置。
分而治之的布局优化：最后，采用分而治之的优化策略，分别优化每个功能区域的布局，然后将这些区域组织成完整的场景，以确保生成的场景布局在物理上是可行的。
为了验证所提方法的有效性，研究者在多个实验中进行了比较和消融研究，使用了3D-Front数据集。实验结果表明，所提方法在物理可行性（如避免物体重叠和超出边界）和合理的相对位置推断方面均优于现有的最先进方法。具体而言，所提方法在生成的场景布局的重叠率和超出边界率方面均为零，表明其生成的场景布局在物理上是合理的。此外，研究者还进行了感知研究，结果显示参与者对所生成场景的有效性、物理可行性和布局合理性给予了最高评分，进一步证明了该方法的实用性和有效性。
在这项研究中，研究人员开发了一种新方法，可以根据人们的描述自动生成室内场景。这个过程分为三个步骤。首先，他们会使用一个强大的语言模型来理解用户的需求，并生成一个三层结构的场景描述。这个结构就像一个树：最上面是整个房间，中间是不同的功能区域（比如卧室、客厅），最下面是具体的物体（如床、桌子）。接下来，他们使用一个智能网络来推断这些物体在房间里应该放在哪里，确保它们之间的关系合理。最后，他们会对每个区域进行单独的优化，然后将这些区域组合成一个完整的场景，这样生成的房间既美观又符合物理规律。通过这些步骤，他们能够创建出既符合用户需求又真实可信的室内场景。

302-A3.2｜从未校准稀疏视图前馈估计几何、外观与相机姿态，3D高斯修复；单视角人类扩散；逆向渲染材料建模；语音同步手势生成

2025-02-17｜ZJU,AntGroup,Oxford,Stanford｜⭐️🟡
http ://arxiv.org/abs/2502.12138v2https ://zhanghe3z.github.io/FLARE/
FLARE是一种前馈模型，旨在从未经校准的稀疏视图中高效地恢复高质量的相机姿态、几何形状和外观。该模型在处理相机围绕主题旋转的场景时表现出色，并且在实际应用中，特别是在室内环境捕捉时，展现了强大的泛化能力。FLARE的设计目标是同时优化几何重建与新视角合成，使其在处理仅有2到8张输入图像的情况下，仍能在不到0.5秒的时间内完成推理。这一方法利用了大规模公共数据集进行训练，取得了在相机姿态估计、几何重建和新视角合成等任务上的最新成果。
FLARE的方法分为几个关键步骤：
相机姿态估计：FLARE首先使用神经网络预测器估计粗略的相机姿态，为后续步骤提供几何和外观学习的基础。该过程通过将输入图像分割为非重叠的补丁并与学习到的相机潜在向量结合，来实现直接的姿态估计。
几何学习：在得到相机姿态后，FLARE采用两阶段几何学习方法。第一阶段是在相机坐标系中学习局部几何，然后通过一个神经场景投影器将其转换为全局几何表示。这种方法提高了重建的准确性和效率。
外观建模：最后，FLARE利用3D高斯分布来建模外观，并结合输入图像的特征进行渲染，以实现高质量的新视角合成。整个框架通过多任务学习进行训练，优化相机姿态、几何和外观的损失。
在实验中，FLARE在多个数据集上进行了评估，包括RealEstate10K和DL3DV。通过对比不同方法的相机姿态估计和稀疏视图重建结果，FLARE表现出色，尤其是在点云质量和新视角渲染方面。实验结果显示，FLARE不仅在准确性上优于传统的优化方法，还在推理速度上具有明显优势。此外，FLARE在处理极少视图时的泛化能力也得到了验证，展示了其在实际应用中的潜力。
FLARE的工作原理可以简单理解为一个三步走的过程。首先，它会观察输入的图片，像拼图一样把这些图片的不同部分组合起来，来估算出相机的角度和位置。接着，FLARE会在每张图片的视角下，仔细学习这些图像中可见的物体形状和特征，像是在画地图一样，把这些信息整理成一个整体的3D模型。最后，FLARE利用这些信息来生成新的图片，表现出这些物体在不同角度下的样子，就像是你从不同位置拍摄同一个场景一样。这种方法让FLARE在处理少量图片时，仍能快速而准确地重建出丰富的3D场景。
2025-02-17｜NationalTaiwanU,NVIDIA｜⭐️
http ://arxiv.org/abs/2502.11801v1
在3D场景重建和理解中，生成新视角图像扮演着重要角色。尽管近年来神经辐射场（NeRF）和3D高斯点云（3DGS）等方法取得了显著进展，它们在处理对象移除和填充时仍面临挑战。为了解决多视角一致性的问题，本文提出了一种新的框架——基于深度引导的跨视角一致性3D高斯填充（3DGIC）。该方法通过利用训练视图的深度信息和背景像素，优化了3DGS模型，实现了高保真度的3D填充效果。通过对多个基准数据集的广泛实验，3DGIC在定量和定性评估中均优于现有的最先进方法，展示了其在视觉一致性和背景内容保留方面的优势。
3DGIC的实现分为两个主要阶段：深度引导的填充掩码推断和填充引导的3DGS优化。具体步骤如下：
深度引导的填充掩码推断：
从多视角图像和对象掩码中提取可见背景像素。
通过投影和反投影技术，结合深度图，更新填充掩码，以确保在不同视角间保持一致性。
该过程确保填充区域仅包含在所有训练视角中不可见的区域。
填充引导的3DGS优化：
使用更新后的掩码对3D高斯模型进行优化，去除指定对象并引入新的高斯点。
通过引导填充过程，确保生成的3D场景在不同视角下保持一致性，进而实现高保真的新视角合成。
该方法有效地减少了对人工标注的依赖，提高了3D场景填充的自动化能力。
为验证3DGIC的有效性，研究团队在多个真实世界基准数据集上进行了实验。使用SPIn-NeRF数据集，该数据集包含多种场景的训练和测试图像。实验结果表明，3DGIC在FID、LPIPS等评估指标上均优于现有的3D填充方法。通过对比不同方法的填充结果，3DGIC不仅在视觉保真度上表现优越，而且在多视角一致性方面也显著改善。实验还包括对不同场景的定性评估，展示了3DGIC在处理复杂场景时的优势，如在移除物体时有效保留背景细节。
3DGIC的工作原理可以简单理解为两个步骤。首先，我们需要确定需要填充的区域。为此，系统会查看多个视角的图像，识别出在所有视角中都不可见的部分。这样可以确保填充时不会留下明显的空洞或不自然的边缘。接下来，系统会根据更新后的填充区域，调整3D模型，去掉不需要的物体，同时确保其他背景元素的一致性。通过这种方式，3DGIC能够生成高质量的3D图像，确保在不同角度下的视觉效果都很自然。这样，我们就能在虚拟现实或增强现实中获得更真实的体验。
2025-02-17｜SonyAI
http ://arxiv.org/abs/2502.12080v1https ://skhu101.github.io/HumanGif/
HumanGif是一个新提出的单视角人类扩散模型，旨在从单张输入图像中合成出具有视角和姿态一致性的3D人类图像。尽管以往的研究在新视角合成方面取得了显著进展，但在生成可动画的人类头像时仍面临挑战。HumanGif通过利用基础扩散模型的生成先验，采用单视角条件的人类扩散过程，来实现高质量的人类图像合成。该模型引入了HumanNeRF模块，以捕捉输入图像中的空间对齐特征，从而更好地捕捉相对相机和人类姿态的变化。此外，模型还采用图像级损失来缩小潜在空间与图像空间之间的差距，从而提高生成结果的一致性和细节表现。经过在RenderPeople和DNA-Rendering数据集上的广泛实验，HumanGif显示出在新视角和姿态合成方面的最佳感知性能。
HumanGif的核心方法包括几个重要模块：
单视角条件扩散模型：将3D人类新视角和姿态合成任务转化为单视角条件的扩散过程，利用生成先验来增强模型的表现。
HumanNeRF模块：该模块负责从输入图像中学习空间对齐特征，确保生成结果在姿态和视角上的一致性。通过线性混合皮肤（LBS）技术，模型能够有效地捕捉细节。
图像级损失：在训练过程中引入图像级损失，以确保潜在空间与图像空间之间的一致性，促进模型在生成图像时的精确性。
多视角和时间注意机制：通过整合多视角注意和时间注意模块，模型能够在合成过程中保持一致性，尤其是在处理目标图像序列时。
在实验部分，HumanGif在两个多视角人类建模数据集上进行了评估：RenderPeople和DNA-Rendering。研究团队从RenderPeople数据集中随机选择了450个对象进行训练，并从DNA-Rendering数据集中使用416个序列进行训练和评估。实验结果表明，HumanGif在多个感知质量指标上超越了基线方法，尤其是在新视角和姿态合成任务中。与其他方法相比，HumanGif能够更好地捕捉输入图像的细节，并保持生成结果的一致性。此外，模型在视频一致性方面的表现也得到了验证，显示出其在生成可动画人类视频方面的优越性。
HumanGif的工作原理可以简单理解为一个“智能画家”，它只需要一张人类的照片，就能画出该人类在不同姿势和角度下的样子。这个“画家”使用了一些聪明的工具。首先，它会分析这张照片，找出人物的特征，比如身形和姿势。接着，它会利用这些特征，结合一些数学模型，生成出不同视角下的人物图像。为了确保这些图像不仅看起来像真实的人，而且在不同角度和姿势下保持一致，它还会进行多次调整和优化。最后，通过对比生成的图像和真实图像，模型会不断学习和改进，确保生成的结果尽可能完美。这样，HumanGif就能在视频游戏、动画制作等领域，帮助创造出更生动、真实的人物形象。
2025-02-16｜CMU｜ICLR2025｜⭐️
http ://arxiv.org/abs/2502.10988v1
本研究提出了一种新颖的方法，名为OMG（OpacityMattersinGaussianSplatting），用于逆向渲染中的材料建模。逆向渲染旨在从一组图像中恢复物理属性，如几何形状、材料和光照条件。近年来，神经渲染的进展使得实现逼真的逆向渲染结果成为可能。然而，现有模型通常忽视了不透明度与材料属性之间的依赖关系，这导致了建模的不足。通过引入基于辐射传输理论的改进，本文将不透明度与材料属性相结合，提出了一种更为准确的建模方法。实验结果表明，该方法在新视角合成和材料建模方面均显著提升了性能。
本文的方法主要分为以下几个步骤：
不透明度建模：依据Bouguer-Beer-Lambert法则，提出不透明度应与材料属性相关联。通过引入一个神经网络，输入材料属性以输出交叉截面，从而实现物理上正确的激活函数。
高斯斑点作为吸收体：将每个高斯斑点视为一个吸收体，计算其光透过率。透过率的计算遵循Bouguer-Beer-Lambert法则，考虑材料的密度和交叉截面。
梯度流动：在优化过程中，材料属性不仅接收来自颜色的梯度，还接收来自不透明度的梯度。这种方法增强了材料学习的梯度流动，使模型在物理约束下进行优化。
在实验中，本文将所提出的方法应用于三种先进的基线模型：GaussianShader、GS-IR和R3DG。通过在合成数据集（如Synthetic4Relight和ShinyBlender）和真实数据集（如MIP-NeRF360）上进行验证，结果显示该方法在新视角合成、光照重建和材料属性估计等方面均有显著提升。量化指标如PSNR、SSIM和LPIPS均表明，本文方法在材料建模和渲染质量上优于现有技术，表明引入物理约束的有效性。
在我们的研究中，我们主要关注的是如何更准确地模拟材料在光线照射下的行为。简单来说，不同的材料对光的反应是不同的，比如玻璃允许光线通过，而木头则会阻挡光线。我们借鉴了一个叫做Bouguer-Beer-Lambert法则的物理原理，认为材料的不透明度（即光线被阻挡的程度）应该与材料的特性相关联。为此，我们设计了一个神经网络，它可以根据材料的特性来计算不透明度，从而使得我们的模型在处理光线时更加真实。此外，我们的方法还允许材料的特性在优化过程中相互影响，这样可以更好地反映现实世界中材料的行为。通过这些改进，我们的模型在生成图像时表现得更好，能够更准确地重现不同材料的特性。
2025-02-15｜CUC,SamsungResearchAmerica,BUAA
http ://arxiv.org/abs/2502.10729v2https ://github.com/mookerr/VarGES/
VarGes是一种创新的框架，旨在通过结合视觉风格线索来提升与语音同步的3D手势生成的多样性和自然性。随着虚拟角色和社交机器人在教育、娱乐和医疗等领域的广泛应用，生成自然且符合上下文的手势变得尤为重要。现有的手势生成方法虽然在性能上取得了显著进展，但往往受到数据集多样性和音频输入信息量有限的制约。VarGes通过引入Variation-EnhancedFeatureExtraction（VEFE）模块、Variation-CompensationStyleEncoder（VCSE）模块和Variation-DrivenGesturePredictor（VDGP）模块，旨在克服这些挑战，生成既流畅又自然的3D手势，捕捉人类表达的复杂动态。
VarGes的核心方法可以分为三个主要模块：

Variation-CompensationStyleEncoder(VCSE)模块：利用变压器编码器和自注意力池化层，VCSE将StyleCLIPS编码为深度学习表示，生成风格代码。这些风格代码在手势生成过程中起到指导作用，确保生成的手势在整体特征上保持一致。
Variation-DrivenGesturePredictor(VDGP)模块：该模块通过交叉注意力机制融合音频特征和风格代码，利用时间自回归模型生成手势序列。这一过程不仅捕捉了节奏和变化，还整合了来自风格参考视频的复杂手势风格，增强生成手势的多样性和自然性。
在实验中，VarGes的表现通过与现有方法的比较进行了评估。使用SHOW数据集，模型的生成手势在多样性、真实感和与音频的同步性上均表现出色。通过定量评价指标，如Variation、FrechetGestureDistance(FGD)和BeatConsistencyScore(BC)，VarGes在这些指标上均优于其他方法，显示出其生成的手势在多样性和自然性方面的优势。此外，通过用户研究和视觉比较，参与者对VarGes生成的手势在自然性、流畅性和与语音的协调性方面给予了更高的评价，进一步验证了该方法的有效性。
VarGes的工作原理可以简单理解为一个“音频与视觉结合”的手势生成系统。首先，它从语音中提取出一些重要的特征，就像我们听到某种语气或情绪时，能够感受到说话者的情感一样。然后，系统会参考一些风格视频，这些视频展示了不同的手势和动作风格。通过结合这些信息，VarGes能够生成多样化且自然的手势，确保它们与语音内容相匹配。最终，系统像一个指挥家一样，利用这些信息来演绎出与音频内容相符的生动手势，让虚拟角色的表达更加真实和富有表现力。

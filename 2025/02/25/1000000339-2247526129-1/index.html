2025-02-16｜FDU,AlibabaDAMOAcademy｜⭐️⭐️
http ://arxiv.org/abs/2502.11158v2https ://github.com/Stability-AI/stablediffusion
本研究提出了一种新颖的左提示引导（Left-Prompt-Guided,LPG）框架，旨在解决多种基于参考的视觉任务。该框架受到人类创作过程的启发，通过左侧和右侧图像的拼接构建上下文输入。我们进一步开发了AnyRefill模型，它是LeftRefill的扩展，能够有效地将文本到图像（Text-to-Image,T2I）模型适应于各种视觉任务。AnyRefill利用先进的扩散变换器（DiffusionTransformer,DiT）架构的修复先验，并结合灵活的任务特定组件，展现出卓越的数据效率，能够在保持高生成性能的同时，仅需最少的任务特定微调。通过大量的消融研究，AnyRefill的性能优于其他图像条件注入方法，并在许多任务中与最先进的开源方法相媲美。
AnyRefill方法的核心在于其LPG框架，具体包括以下几个关键步骤：
输入拼接：将参考图像放置在左侧，目标图像放在右侧，通过横向拼接形成一个统一的输入。这种设计消除了对额外图像特征编码器的需求，简化了输入结构。
任务特定LoRA：在AnyRefill中引入任务特定的低秩矩阵（LoRA），以有效微调模型，优化生成性能，同时保持较少的可训练参数。
扩散优化：AnyRefill结合了扩散过程的优化，利用拼接输入进行上下文信息的引导，确保生成图像与参考图像之间的空间关联。
多任务能力：该模型能够处理多种视觉任务，包括条件生成、感知和图像编辑，展现出强大的灵活性和适应性。
在实验部分，我们验证了AnyRefill在多个视觉任务中的表现，包括图像修复、条件生成和图像编辑。使用来自不同数据集的训练对进行评估，特别是在图像修复任务中，AnyRefill展示了优越的生成质量和数据效率。在条件生成任务中，AnyRefill能够在仅需少量训练对的情况下，生成高质量的图像，与其他方法相比，其生成的图像在视觉上更具一致性和准确性。此外，在图像编辑任务中，AnyRefill能够有效地进行年龄和性别修改，显示出其在处理复杂编辑任务时的能力。所有实验结果均表明，AnyRefill在多任务处理的灵活性和数据效率方面具有显著优势。
AnyRefill的工作原理可以想象成一个画家在创作时的直观过程。首先，画家将参考图像放在左边，作为灵感来源，而目标图像则放在右边。通过将这两幅图像拼接在一起，AnyRefill能够同时看到参考和目标，从而在创作时更好地捕捉两者之间的关系。为了使画作更加完美，AnyRefill使用了一种叫做“LoRA”的技术，这就像在画布上添加一些细微的调整，使得画作更符合我们想要的效果。而且，AnyRefill不需要大量的训练数据，只需少量的图像对就能表现出色，这就像画家只需看几幅作品，就能灵活地创作出新的艺术作品。通过这种方式，AnyRefill不仅能处理多种不同类型的视觉任务，还能保持高效和高质量的输出。
2025-02-16｜UPenn,CornellU,UCIrvine｜⭐️🟡
http ://arxiv.org/abs/2502.10999v1https ://github.com/bowen-upenn/ControlText
本研究提出了一种名为ControlText的创新方法，旨在实现可控的多语言文本渲染，且无需依赖字体注释。尽管近年来在视觉文本渲染领域取得了一些进展，但用户在开放世界中指定所需字体仍然面临挑战，尤其是在缺乏准确字体标注的情况下。为了解决这一问题，研究团队结合了条件扩散模型和文本分割模型，利用分割掩码捕捉和表示像素空间中的字体特征。这种自监督的方法使得用户能够在没有任何真实标签的情况下，自定义多语言文本的渲染。通过零样本的字体和文本编辑实验，ControlText展示了其在多种字体和语言上的有效性，为行业和学术界提供了有价值的见解。
ControlText的关键方法分为几个步骤：
数据驱动的洞察：通过对大量真实图像进行文本分割，提取出字体控制信息，而无需依赖任何字体标签。
训练管道：使用OCR模型进行文本区域的识别和过滤，以确保仅保留高质量的分割掩码作为字体控制。该模型在训练过程中从输入的图像中学习字体特征。
生成过程：在推理阶段，用户可以指定文本、字体和位置，模型会根据这些输入生成带有用户指定字体的文本，并将其无缝集成到原始图像中。
评估机制：为了评估生成文本的字体准确性，提出了新的度量标准，能够处理模糊字体的开放世界场景。
在实验部分，研究团队使用了一个包含139万张英文和160万张中文图像的数据集进行模型训练，并通过AnyText基准测试评估模型性能。实验结果显示，ControlText在文本准确性和图像质量方面均表现出色，尤其是在处理多样化字体信息时，表现优于现有方法。此外，研究还展示了模型在不同语言和字体上的适应能力，验证了其在实际应用中的有效性和灵活性。
ControlText的工作原理可以简单理解为一个智能的文本编辑器。首先，它从大量的图片中学习如何识别和渲染文字，而不需要任何特定的字体标签。想象一下，你有一张图片，想在上面加上文字，你只需要告诉这个程序你想要什么文字、使用什么字体，以及文字的位置。程序会自动处理这些信息，把你想要的文字完美地融入到图片中。为了确保生成的文字看起来真实且美观，ControlText会过滤掉那些不够好的文字区域，并且在生成过程中使用了一些聪明的数学工具来确保文字的样子和位置都很合适。这样，无论是英文、中文还是其他语言，用户都能轻松地实现个性化的文本渲染。
2025-02-17｜NYU,UIUC,UCSF｜⭐️🟡
http ://arxiv.org/abs/2502.11925v1
随着多模态大型语言模型（MLLMs）的迅速发展，文本与图像的结合成为了研究的热点。在现实世界中，文本和图像之间通常存在复杂的图结构关系，称为多模态属性图（MMAG）。然而，现有的MLLMs在处理这种图结构时面临着诸多挑战，如图的规模膨胀、非欧几里得特性以及层次化模态依赖等问题。为了解决这些问题，本文提出了GRAPHGPT-O，一个专门针对MMAG的多模态大型语言模型，旨在实现对图结构的深度理解和生成能力。通过对输入的语义和结构信息进行线性化处理、构建层次对齐器以及探索推理策略，GRAPHGPT-O能够有效地生成与节点相关的文本和图像对，展示出其在艺术和电子商务等领域的广泛应用潜力。
GRAPHGPT-O的核心方法包括以下几个方面：
个性化PageRank邻居采样：为了应对图规模膨胀的问题，采用个性化PageRank算法选择与目标节点最相关的邻居节点，从而提取关键信息。
图线性化策略：通过线性化处理，将图结构中的文本和图像特征转化为序列化输入，以适应MLLM的处理需求。这一过程包括模态选择、模态顺序和邻居数量的优化。
层次化对齐器：构建一个层次化对齐器，通过节点特征Q-Former和图结构Q-Former来捕捉节点级和子图级的模态依赖，增强对图结构的理解。
推理策略：探索了顺序和并行推理两种策略，以解决模态间的推理依赖问题，确保生成的文本和图像在语义上具有一致性。
在实验部分，研究团队在三个不同领域的多模态属性图数据集上进行了广泛的测试，分别是ART500K、Amazon-Baby和Amazon-Beauty。实验结果表明，GRAPHGPT-O在文本和图像生成的质量上显著优于基线模型。评估指标包括生成图像的质量、文本和图像之间的对应关系，以及生成结果与条件子图的对齐度。通过比较不同的邻居数量、模态顺序和推理策略，研究者们发现，合理的模态整合和顺序处理能够有效提升模型的表现，证明了GRAPHGPT-O在多模态内容生成任务中的有效性。
在GRAPHGPT-O的工作中，研究者们采用了一种新颖的方法来帮助计算机更好地理解和生成图像和文本。首先，他们使用了一种叫做个性化PageRank的算法，像是在图中找出最重要的节点，这样可以避免处理过多不必要的信息。接着，他们将图中的文本和图像“排成一队”，让计算机可以逐个处理，确保每个部分都能被理解。为了更好地抓住图的结构，他们设计了一个层次化的对齐器，帮助模型在生成文本和图像时，确保它们之间的关系是紧密的。最后，他们还测试了不同的生成顺序，发现有时先生成文本再生成图像，或者同时生成这两者，会让结果更好。这种方法的目标是让计算机在处理图像和文本时，能够更自然、更准确。
2025-02-17｜PKU,THU,Mila-QuébecAIInstitute｜⭐️🟡
http ://arxiv.org/abs/2502.12148v1https ://github.com/Gen-Verse/HermesFlow
在多模态大型语言模型（MLLMs）的研究中，理解与生成能力之间存在显著差距。尽管现有模型如Show-o、Transfusion和Emu3在图像理解和生成上取得了显著进展，但它们的理解能力往往优于生成能力。为了解决这一问题，本文提出了HermesFlow，一个旨在通过迭代优化和对齐理解与生成能力的框架。HermesFlow通过使用同源数据来策划理解和生成的偏好数据，并通过Pair-DPO（配对直接偏好优化）方法来缩小这两者之间的差距。实验结果表明，HermesFlow在多个基准测试中表现出色，显示了其作为下一代多模态基础模型的潜力。
HermesFlow的方法主要分为以下几个步骤：
同源偏好数据的策划：
理解偏好数据：利用预训练的MLLM生成与图像相关的多个标题，通过BERT相似性评分选择最优和最劣的标题，形成理解偏好数据集。
生成偏好数据：使用MLLM根据输入提示生成多张图像，并通过自我评估选择出最佳和最差的生成结果，构建生成偏好数据集。
Pair-DPO训练策略：
通过Pair-DPO方法同时优化理解和生成能力，确保模型能够相互促进。具体而言，优化目标是最大化优选输出的概率，最小化不优选输出的概率。
自我游戏迭代优化：
在每轮迭代中，模型生成新的理解和生成样本，并根据前一轮的结果更新偏好数据。通过这种方式，HermesFlow能够在多轮迭代中不断提升模型的理解和生成能力，逐步缩小两者的差距。
在实验部分，HermesFlow的性能通过与多种基准模型进行比较来评估。选取5,000对图像-标题作为同源输入数据，结合视觉问答（VQA）数据进行全面评估。结果表明，HermesFlow在理解和生成任务上均优于其他模型，尤其在对象计数和位置准确性方面。通过用户研究，HermesFlow在生成图像的质量上获得了广泛的认可，显示出其在多模态理解和生成领域的有效性和优势。
HermesFlow的核心方法可以简单理解为一个“学习和提升”的过程。首先，它从一组相似的图像和描述中提取信息，像是在问一组问题并得到不同的答案。然后，它选出最好的答案和最差的答案，确保模型能够更好地理解图像。接着，它利用这些信息来生成新的图像，并再次进行评估，选择出最好的生成结果。这个过程是循环进行的，每次都在改进模型的能力，帮助它在理解和生成方面变得更强。通过这样的方式，HermesFlow不仅提高了图像理解的准确性，还增强了生成图像的质量，最终实现了两者之间的平衡。
2025-02-17｜PKU,PrincetonU,THU｜⭐️🟡
http ://arxiv.org/abs/2502.12146v1https ://github.com/Gen-Verse/Diffusion-Sharpening
在现代生成模型中，扩散模型已经成为一个重要的研究方向，尤其在文本到图像合成和视频生成等任务中表现出色。然而，如何有效地微调这些模型以使其与用户的偏好和预定义标准对齐仍然是一个关键挑战。为了解决这一问题，本文提出了一种新方法——Diffusion-Sharpening。这一方法通过优化采样轨迹来增强模型的下游对齐能力，克服了现有强化学习（RL）方法在单时间步训练上的局限性。Diffusion-Sharpening不仅提高了训练效率，还在推理过程中减少了计算成本，并在多个评估指标上超越了传统的微调方法和采样轨迹优化技术。
Diffusion-Sharpening方法的核心在于通过路径积分框架优化采样轨迹。具体来说，方法包括以下几个关键步骤：
预处理和奖励评估：首先，通过对中间状态进行近似，评估生成样本的奖励。这一过程使用了扩散概率模型的逆过程，计算每个时间步的奖励。
轨迹选择：在训练过程中，从多个采样轨迹中选择最优轨迹。通过评估不同轨迹的累计奖励，选择出最优的轨迹进行反向传播。
两种实现方式：
SFT-Diffusion-Sharpening：利用现有的图像-文本数据集进行监督微调，优化反向轨迹。
RLHF-Diffusion-Sharpening：通过在线优化，无需预先准备的数据集，自动生成正负样本来进行微调。这种方法有效地利用了轨迹级别的反馈，确保模型在生成过程中能够获得更稳定和高质量的输出。
在实验部分，Diffusion-Sharpening方法在多个基准数据集上进行了评估。实验结果显示，与标准微调方法相比，Diffusion-Sharpening在训练和推理效率上均表现出色。具体而言，SFT-Diffusion-Sharpening和RLHF-Diffusion-Sharpening在多个性能指标上均超越了现有的RL微调方法和采样轨迹优化方法。实验还包括对不同奖励模型的评估，结果表明，Diffusion-Sharpening在文本-图像对齐、组合能力和人类偏好等方面均表现优异。通过对比不同的采样步骤和样本数量，进一步验证了该方法的有效性和鲁棒性。
Diffusion-Sharpening方法可以简单理解为一种“优化生成过程”的技术。想象一下你在画画，通常你会尝试不同的画法，然后选择最好的那一幅。在这个方法中，模型就像是一个画家，它会生成多个不同的图像（轨迹），然后根据一些标准（奖励）来评估每幅图像的好坏。通过这样的评估，模型可以学习到哪些画法更好，从而在下次创作时选择更优的方式。这个过程有两个主要的实现方式：一种是依赖于已有的画作（数据集）来进行微调，另一种是实时生成新的画作并进行优化。这样一来，模型不仅能更好地理解用户的喜好，还能在生成图像时更加高效和准确。
2025-02-17｜KAIST,MicrosoftResearch,HKUST｜⭐️🟡
http ://arxiv.org/abs/2502.11477v1
近年来，文本到图像的扩散模型在生成图像的能力上取得了显著进展。然而，控制生成过程以满足特定属性（如美学质量和用户意图）仍然是一个挑战。本文提出了一种新的方法，称为“基于GFlowNets的提示适应”（PAG），旨在通过将原始提示优化为模型偏好的提示，从而生成所需的图像。以往的研究主要采用强化学习（RL）来优化提示，但我们发现这种方法往往导致生成相似的后缀和确定性行为，因此我们转向一种基于概率推断的方法。我们的核心观点是，利用GFlowNets可以从未归一化的密度函数中进行采样，从而实现高质量和多样性的提示生成。
PAG的核心在于将提示适应重新框架化为一个概率推断问题，具体方法如下：
流动再激活机制：通过定期重新激活网络的最后一层，恢复潜在的神经通路，以增强模型的探索能力，防止模式崩溃。
奖励优先采样：在训练过程中，采用奖励优先的采样方法，以确保模型能够持续接触到高质量的提示，促进知识的有效保留。
奖励分解：通过细化的奖励结构，提供更细致的学习信号，从而改善信用分配效率，帮助模型在生成过程中更好地评估中间选择的贡献。
这些方法共同作用，确保模型在生成多样化提示的同时，保持训练的稳定性和有效性。
为了验证PAG的有效性，本文进行了广泛的实验。我们使用预训练的GPT-2语言模型作为策略模型，并在多个数据集上进行训练和评估。实验结果表明，PAG在奖励和多样性指标上均优于现有的基线方法。我们还测试了PAG在不同文本到图像扩散模型中的迁移能力，结果显示其在零样本条件下仍能生成高质量的图像。此外，消融实验表明，流动再激活机制、奖励优先采样和奖励分解等组件对整体性能贡献显著，验证了我们方法的有效性。
在我们的研究中，我们提出了一种新方法来帮助计算机更好地理解和生成图像。我们的方法叫做PAG，主要通过三种技巧来实现这一目标。首先，我们定期“唤醒”计算机模型中一些“沉睡”的部分，这样它就能更好地学习和适应新的提示。其次，我们采用一种聪明的方式来选择高质量的提示，这样模型就能记住哪些提示效果好，并在将来使用它们。最后，我们将奖励分解成更小的部分，这样模型在每一步都能获得更清晰的反馈，从而更好地学习如何生成多样化的图像。这种方法让我们的计算机不仅能生成漂亮的图像，还能在不同的情况下表现得很好。

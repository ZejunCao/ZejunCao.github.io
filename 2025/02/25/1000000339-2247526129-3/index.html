2025-02-16｜ByteDance｜⭐️⭐️
http ://arxiv.org/abs/2502.11079v1https ://phantom-video.github.io/Phantom/
随着基础模型在视频生成领域的不断发展，主题一致性视频生成仍处于探索阶段。该研究提出了一种名为“Subject-to-Video”的方法，旨在通过文本指令，从参考图像中提取主题元素，并生成主题一致的视频。研究的核心在于平衡文本和图像的双模态提示，从而实现文本内容与视觉内容的深度对齐。为此，研究团队提出了“Phantom”框架，适用于单一和多主题的参考图像。该框架在现有的文本到视频和图像到视频架构的基础上进行了重新设计，重点强调在生成过程中保持主题的一致性，尤其是在生成人类角色时，展现出相较于现有技术的优势。
Phantom框架的核心在于建立一个文本-图像-视频三元组的数据结构，以实现跨模态学习。具体方法包括：
数据结构构建：通过配对图像、文本和视频，形成三元组数据结构，确保模型可以同时处理这三种模态。
数据类型分类：将图像提示分为“配对数据”和“交叉配对数据”，前者确保图像和视频中的主题一致，而后者则通过不同视频中的元素匹配，避免单纯的“复制粘贴”现象。
模型架构设计：Phantom采用了改进的联合文本-图像注入模型，结合了VariationalAuto-Encoder（VAE）和CLIP技术，以确保有效的跨模态数据表示学习。
训练过程：通过对基础模型的微调，Phantom在生成过程中不仅保留了主题的一致性，还提高了对文本指令的响应能力。
在实验部分，Phantom的性能通过一系列定量和定性评估进行验证。为了评估主题一致性生成能力，研究团队构建了一个特定的测试集，包含50个参考图像，每个图像配有3个不同的文本提示，共生成450个视频。实验中，Phantom的表现与当前市场上最先进的商业工具（如Vidu、Pika和Keling）进行了比较，结果显示Phantom在主题一致性和文本响应能力上具有明显优势。此外，通过用户研究评估多主题生成的效果，结果表明Phantom在多主题一致性方面也表现良好，显示出其在实际应用中的潜力。
在Phantom方法中，研究团队首先创建了一种新颖的数据结构，结合了文本、图像和视频。这就像将一本书（文本）、一本画册（图像）和一部电影（视频）结合在一起，形成一个完整的故事。接下来，他们将数据分为两类：一类是直接对应的图像和视频，确保两者中的主题一致；另一类则是从不同视频中提取的元素，以避免简单的复制。然后，Phantom使用了一种特殊的模型架构，像拼图一样将这些信息组合在一起，确保生成的视频不仅视觉上连贯，还能准确反映文本的内容。这种方法允许Phantom在生成视频时保持主题的一致性，让观众看到的内容与他们期望的更为接近。
2025-02-15｜SkyworkAI｜⭐️🟡
http ://arxiv.org/abs/2502.10841v1https ://skyworkai.github.io/skyreels-a1.github.io/
SkyReels-A1是一种基于视频扩散变换器（DiT）框架的创新方法，旨在实现高保真度和表现力的肖像动画。尽管现有技术在肖像动画方面取得了一定进展，但仍面临身份扭曲、背景不稳定和不自然的面部动态等问题。SkyReels-A1通过利用DiT的强大生成能力，提升了面部运动转移的精确度、身份保留和时间一致性。该系统集成了一个表达感知的条件模块，使得通过表达引导的地标输入可以实现无缝的视频合成。此外，SkyReels-A1采用了多阶段训练范式，逐步优化表情与运动之间的关联，确保稳定的身份再现。大量实证评估表明，该模型在生成视觉连贯且构图多样的结果方面表现出色，适用于虚拟头像、远程通信和数字媒体生成等多个领域。
SkyReels-A1采用了一系列先进的技术来实现其肖像动画功能。其方法包括以下几个关键模块：
表达感知地标：通过从视频中提取面部地标，捕捉细腻的面部表情变化，确保动画的真实感和情感共鸣。
多阶段训练策略：该策略分为三个阶段，分别为运动驱动训练、身份保留训练和多模块联合微调。每个阶段都专注于不同的优化目标，逐步提高模型的动画生成能力。
面部图像-文本对齐模块：通过将面部特征映射到文本特征空间，增强身份一致性，确保在改变面部表情时保持角色的身份完整性。
时空一致性引导模块：利用3D因果编码器确保驱动信号与输入视频潜在表示之间的时空对齐，提升动画的自然流畅性和真实感。
在实验部分，SkyReels-A1的性能通过与多种基线模型的比较进行评估。实验设置包括使用多个数据集进行训练，并在不同阶段进行细致的训练调整。实验结果显示，SkyReels-A1在生成质量、运动精确度及身份保留等多个方面均优于现有的肖像动画方法。定量评估指标包括图像质量、表情差异和姿势差异等，结果表明SkyReels-A1在这些指标上均表现出色。此外，用户研究也证实了SkyReels-A1在运动准确性和人类相似性方面的优势，63%的参与者偏好于该模型生成的动画。
SkyReels-A1的工作原理可以简单理解为一个智能的动画生成器。首先，它从视频中提取面部的关键点，这些点帮助模型理解面部表情的变化。接着，它使用一种分阶段的训练方式，逐步教会模型如何生成自然的动画。每个阶段都有不同的重点，比如第一阶段关注运动，第二阶段则确保动画中的人物身份保持一致。最后，SkyReels-A1还会将面部特征与文本信息结合起来，这样可以更好地控制动画中的细节，确保角色在表情变化时依然保持原有的身份。这种方法使得生成的动画更加真实自然，适合用于虚拟角色、视频聊天等多种场景。
2025-02-17｜THU,InfinigenceAI,IllinoisTech,SJTU｜⭐️🟡
http ://arxiv.org/abs/2502.11897v1https ://github.com/thu-nics/DLFR-VAE
在本文中，研究者们提出了一种名为动态潜在帧率变分自编码器（DLFR-VAE）的新框架，旨在提高视频生成的效率。传统的视频生成模型通常采用固定的压缩率，而DLFR-VAE则根据视频内容的复杂性动态调整潜在帧率。这一创新的核心在于两个主要部分：一是动态潜在帧率调度器，它根据信息论中的内容复杂性对视频进行分段，并自适应确定最佳帧率；二是无训练适应机制，使得预训练的变分自编码器（VAE）架构能够处理具有可变帧率的特征。通过这些改进，DLFR-VAE能够作为一个即插即用的模块，与现有的视频生成模型无缝集成，从而加速视频生成过程。
DLFR-VAE的实现包括几个关键步骤。首先，信号的频率分析是基础，通过对视频信号的时域和频域特征进行分析，识别出不同视频段的复杂性。这一过程借助傅里叶变换来提取频谱信息。其次，动态帧率潜在空间的构建允许每个视频段根据其内容复杂性分配不同的帧率。具体来说，输入视频被划分为多个段，每个段的潜在表示通过动态下采样和上采样模块进行处理，以适应其复杂性。最后，将静态VAE转换为动态VAE，通过在编码器中引入动态下采样模块和在解码器中引入动态上采样模块，使得预训练的VAE能够处理可变帧率的特征，从而实现高效的视频生成。
实验部分主要评估了DLFR-VAE在视频重建质量和生成速度上的表现。研究者将DLFR-VAE应用于两个先进的预训练VAE模型，并对其在不同分辨率和帧率下的重建性能进行了比较。实验结果表明，DLFR-VAE在保持较高重建质量的同时，显著减少了潜在空间中的元素数量，进而提高了生成速度。此外，DLFR-VAE的性能在不同的压缩比下均优于静态VAE，展示了其在各种视频生成任务中的广泛适用性和强大鲁棒性。
DLFR-VAE的工作原理可以简单理解为根据视频的内容复杂程度来调整生成视频的帧率。想象一下，你在看一部电影，快速动作场景需要更多的细节和流畅度，而静态场景则不需要那么多帧。DLFR-VAE通过分析视频中的每一部分，判断哪些地方需要更多的帧（即更高的帧率），哪些地方可以减少帧数（即更低的帧率）。这样做的好处是，它可以在不牺牲画质的情况下，减少计算量和生成时间。总的来说，DLFR-VAE让视频生成变得更高效，就像在看电影时，智能地根据场景的需要调整画面的流畅度。
2025-02-16｜LMUMunich｜⭐️🟡
http ://arxiv.org/abs/2502.11234v1https ://compvis.github.io/maskflow/
在长视频生成领域，生成高质量且时长超过训练样本的视频一直是一个挑战。本文提出了一种名为MaskFlow的统一视频生成框架，通过结合离散表示和流匹配技术，能够高效地生成高质量的长视频。MaskFlow通过在训练过程中采用帧级别的遮罩策略，能够在生成视频时条件化于之前生成的未遮罩帧，从而实现生成长度是训练序列十倍的视频。该方法在多个数据集（如FaceForensics和DeepmindLab）上进行了验证，并通过FrechetVideoDistance（FVD）指标表现出与最先进的方法竞争的质量。此外，MaskFlow还展示了在采样效率方面的优势，能够在训练无关的方式下应用于时间依赖和时间独立的模型。
MaskFlow的核心方法包括以下几个方面：
长视频生成任务的定义：该方法将视频视为由多个帧组成的序列，并通过对帧进行分块生成来实现长视频的生成。
流匹配的初步概念：MaskFlow采用离散流匹配的理念，通过对每个帧施加不同的噪声级别来实现数据的腐蚀和重建。
帧级别遮罩训练：在训练过程中，模型通过对每个帧应用不同的遮罩比例，使得模型能够学习如何在不同的遮罩条件下恢复帧内容。这种方法允许模型在生成时灵活处理先前生成的帧。
块式自回归生成：通过将视频划分为重叠的块，MaskFlow能够在每个块中生成新帧，同时保持上下文信息的连贯性。
两种采样模式：MaskFlow支持流匹配（FM）和MaskedGenerativeModel（MGM）两种采样模式，前者适合平滑过渡，后者则能在少量采样步骤内生成高质量视频。
在实验中，MaskFlow在FaceForensics和DeepmindLab数据集上进行了评估。FaceForensics数据集包含704个深度伪造视频，而DeepmindLab则提供了300帧的随机游走视频。实验结果表明，MaskFlow在生成质量和采样效率方面均优于现有的其他方法。具体来说，MGM风格的采样方法显示出在生成长视频时的高效性，能够在较少的函数评估次数（NFE）下生成高质量视频。此外，在长视频生成任务中，MaskFlow能够无缝地在时间依赖和时间独立的生成模式之间切换，展示了其灵活性和适用性。
MaskFlow的方法可以简单理解为一种智能的“视频生成工厂”。首先，它把视频看作一系列的帧（就像电影中的每一帧画面），然后通过一种叫做“遮罩”的技术来训练模型。遮罩意味着在训练时，有些帧的部分内容会被隐藏，模型需要学习如何填补这些空白。这样，当我们让模型生成新视频时，它就能根据之前的帧来决定新帧应该是什么样子。为了提高效率，MaskFlow还将生成过程分成小块，每次生成一部分内容，这样可以减少计算时间。最后，它有两种不同的工作模式，可以根据需要选择，既能生成流畅的视频，也能快速生成高质量的内容。这种灵活的方法使得MaskFlow在生成长视频方面表现出色。
2025-02-17｜UCAS,RightBrain.AI,NTU,PKU,CityU｜⭐️🟡
http ://arxiv.org/abs/2502.11515v1
本文介绍了一种名为SayAnything的条件视频扩散框架，旨在通过音频输入直接合成口型运动，同时保留说话者的身份特征。近年来，音频驱动的口型同步技术取得了显著进展，但现有方法通常依赖于音频与视觉之间的固定对齐，导致训练过程复杂且口型运动不够自然。SayAnything通过设计多模态条件融合机制，克服了这些限制，实现了对未见角色的口型同步，同时在多种场景下表现出色。该框架包含三个专门模块：身份保留模块、音频引导模块和编辑控制模块，能够在没有额外监督信号的情况下生成高质量的视频，确保口型与音频的高度一致性。
SayAnything的核心在于其多模态条件融合机制，具体包括以下几个部分：
身份保留模块：使用高效的ID-Guider编码器提取身份特征，并控制其对口型风格的影响，确保生成视频中的说话者身份得到保留。
音频引导模块：利用Whisper音频特征提取器增强音频信号对口型运动生成的控制，通过跨注意力机制将音频特征与视频帧的潜在表示结合，从而改善音频与口型的对应关系。
编辑控制模块：采用自适应掩膜策略，确定口型编辑区域，消除周围像素运动对口型生成的影响，确保生成的口型与参考视频的视觉一致性。
这些模块共同作用，使SayAnything能够在没有额外监督信号的情况下，直接从音频输入生成自然流畅的口型运动。
为了验证SayAnything的有效性，研究团队进行了广泛的实验，使用了多个公共数据集，包括AVASpeech和HDTF。实验结果表明，SayAnything在视频生成质量、时间一致性和口型灵活性等方面均优于现有的最先进方法。定量评估使用了多项指标，如FID、SSIM和SyncNet信心分数，结果显示SayAnything在视觉保真度和音频-视觉同步方面表现出色。此外，通过用户研究，SayAnything在不同场景下的用户偏好率最高，进一步证明了其在多样化应用场景中的适用性和优越性。
SayAnything的工作原理可以简单理解为三个主要步骤。首先，身份保留模块确保生成的视频中说话者的特征保持一致，就像在电影中保持角色的外貌一样。其次，音频引导模块从音频中提取信息，帮助计算机理解说话者在说什么，这样生成的视频中的口型就能与音频同步。最后，编辑控制模块通过设置特定区域，确保口型编辑时不受周围动作的影响，保证整体画面的自然流畅。通过这三个部分的协作，SayAnything能够生成高质量的口型同步视频，让虚拟角色“说话”时看起来更真实。

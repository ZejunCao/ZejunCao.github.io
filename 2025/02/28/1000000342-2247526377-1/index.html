2025-02-21｜KoreaU｜⭐️🟡
http ://arxiv.org/abs/2502.15203v1
近年来，文本到图像（T2I）生成领域在将多个个性化概念整合到单张图像中的方法上取得了显著进展。然而，现有方法在处理包含多个对象的复杂场景时，往往会出现性能下降的问题，导致非个性化区域的失真或修改。为了解决这些问题，本文提出了一种名为FlipConcept的新方法，它能够在不需要额外微调的情况下，无缝地将多个个性化概念整合到单张图像中。FlipConcept通过引入引导外观注意力、掩码引导噪声混合和背景稀释等技术，有效地解决了多概念个性化中的挑战，生成了高质量的图像，同时最小化了概念泄漏。实验结果表明，FlipConcept在复杂场景中能够保持每个概念的连贯性，并保护背景不受失真或意外修改的影响，显著优于现有方法。
FlipConcept方法分为两个主要阶段：输入准备和自适应概念混合。在输入准备阶段，首先生成背景图像并从中提取掩码，然后对背景图像和个性化概念图像进行Edit-FriendlyDDPM反演，得到编辑友好的潜在表示和精确定义的掩码。在自适应概念混合阶段，利用上一步得到的潜在表示和掩码生成图像，通过三种核心技术实现多概念的融合。首先，引导外观注意力通过参考其他图像重建个性化概念图像的键和值，增强个性化概念对整体外观的影响。其次，掩码引导噪声混合基于提取的掩码选择性地整合噪声预测，确保仅修改包含个性化概念的区域，而非个性化区域保持不变。最后，背景稀释通过扩展对象掩码来减少指定区域外的影响，缓解自注意力操作中的概念泄漏，保持图像质量。
实验部分通过定量和定性评估验证了FlipConcept的性能。使用CustomConcept101数据集和StableDiffusion生成的图像进行测试，结果表明FlipConcept在CLIP文本相似度（T_CLIP）和图像相似度（I_CLIP）指标上均优于现有方法。定性比较显示，FlipConcept在处理包含多个个性化概念的复杂场景时，能够有效分离和整合各个概念，保持图像结构和细节的完整性。此外，时间效率比较显示，FlipConcept由于不需要微调，显著减少了预处理时间，适用于各种场景。消融研究进一步验证了每个组件对整体性能的贡献，表明引导外观注意力、背景稀释和掩码引导噪声混合在多概念个性化生成任务中起到了关键作用。
FlipConcept的核心思想是通过两个步骤来生成包含多个个性化概念的图像。首先，它准备背景图像和个性化概念的图像，并通过一种特殊的反演技术将它们转换为适合编辑的潜在表示。然后，它使用三种技术来混合这些概念：第一种技术是通过参考其他图像来增强个性化概念的外观；第二种技术是基于掩码选择性地修改图像，确保只有个性化概念的区域被改变；第三种技术是减少背景对图像的影响，防止概念泄漏。通过这些步骤，FlipConcept能够在不需要额外训练的情况下，生成高质量的图像，保持每个概念的独特性和背景的完整性。
2025-02-19｜UTS(Sydney),IITKharagpur,ISIKolkata｜⭐️🟡
http ://arxiv.org/abs/2502.14007v1
本文提出了一种新颖的草图到图像转换技术，利用预训练的潜在扩散模型（LDM）的特征泛化能力，无需重新训练即可实现高质量的图像生成。该方法通过引入一个可学习的轻量级映射网络（LCTN），将草图域的特征映射到图像域，从而在保持结构一致性的同时生成逼真的图像。与现有的基于生成对抗网络（GAN）的方法相比，该方法在计算成本和数据需求上更为高效，且在未见过的对象类别上表现出更好的泛化能力。实验结果表明，该方法在多个数据集上的定性和定量评估中均优于现有技术。
潜在扩散模型（LDM）：利用预训练的LDM，通过前向扩散和后向去噪过程生成图像。前向扩散逐步向输入数据添加噪声，后向去噪则通过学习网络参数来逆转这一过程，从而从噪声中恢复原始图像。
潜在代码翻译网络（LCTN）：设计一个轻量级的映射网络，将草图域的特征映射到图像域。LCTN通过训练边缘图而非手绘草图来减少结构模糊性，利用LDM的中间激活图计算输入特征空间，并通过最小化均方误差来优化映射。
采样策略：在生成图像时，首先对LCTN映射的潜在代码进行前向扩散，然后通过后向去噪过程生成最终图像。通过选择适当的扩散步数k，可以在结构一致性和视觉逼真度之间取得平衡。
实验在三个数据集（Scribble、QMUL和Flickr20）上进行了评估，比较了本文方法与现有草图到图像转换技术（如Pix2Pix、CycleGAN、AODA和LGP）的性能。通过视觉分析和定量评估（如FID、IS、PSNR、SSIM、LPIPS、ACC和MOS），结果表明本文方法在生成图像的视觉质量、结构一致性和类别准确性上均优于现有方法。特别是在处理具有相同形状的不同对象时，本文方法能够生成高质量且视觉上可区分的图像。
本文的方法就像是一个“翻译官”，它能够将简单的手绘草图“翻译”成逼真的照片。这个“翻译官”不需要从头学习，而是利用了一个已经训练好的“大师级画家”（即预训练的潜在扩散模型）。为了让翻译更准确，我们引入了一个“小助手”（即潜在代码翻译网络），它专门负责将草图的特征转换成照片的特征。这个小助手通过训练边缘图来减少手绘草图中的模糊性，从而确保翻译后的照片既符合草图的形状，又看起来非常真实。在生成照片时，我们还会对翻译后的特征进行一些“微调”，通过前向扩散和后向去噪的过程，确保最终的照片既逼真又符合草图的意图。实验证明，这种方法在各种复杂场景下都能生成高质量的照片，远远超过了现有的技术。
2025-02-21｜NVIDIA｜⭐️
http ://arxiv.org/abs/2502.15681v1https ://research.nvidia.com/labs/genair/f-distill/
本文提出了一种基于f-散度分布匹配的一步扩散模型，旨在加速生成速度，解决传统扩散模型采样过程缓慢的问题。现有的方法通过变分分数蒸馏将多步扩散模型蒸馏为单步生成器，但通常使用反向KL散度进行分布匹配，这可能导致模式丢失。本文提出了一种新的f-散度最小化框架，称为f-distill，涵盖了多种散度，能够在模式覆盖和训练方差之间进行权衡。通过推导教师和学生分布之间的f-散度梯度，本文展示了梯度可以表示为它们的分数差异与由密度比确定的加权函数的乘积。实验表明，使用Jensen-Shannon散度的f-distill在ImageNet64和MS-COCO的零样本文本到图像生成任务中达到了当前最先进的一步生成性能。
本文的方法部分主要包括以下几个关键点：
f-散度框架：提出了一种基于f-散度的分布匹配框架，称为f-distill，涵盖了多种散度，如反向KL、正向KL、Jensen-Shannon等。这些散度在模式覆盖和训练方差之间有不同的权衡。
梯度推导：推导了教师和学生分布之间的f-散度梯度，展示了梯度可以表示为它们的分数差异与由密度比确定的加权函数的乘积。这种加权函数在使用较少模式寻求的散度时，会自然强调教师分布中密度较高的样本。
加权函数的作用：加权函数根据选择的f-散度和密度比来确定，能够自适应地减少在教师分布低密度区域的分数匹配，从而提高模型的生成质量。
实验验证：在多个图像生成任务上验证了f-distill框架的有效性，定量结果表明，较少模式寻求的散度在f-distill中一致优于之前的变分分数蒸馏方法。
实验部分在多个图像生成任务上验证了f-distill框架的有效性，包括CIFAR-10、ImageNet64和MS-COCO的零样本文本到图像生成任务。实验结果表明，使用Jensen-Shannon散度的f-distill在ImageNet64和MS-COCO的零样本文本到图像生成任务中达到了当前最先进的一步生成性能。具体来说，f-distill在ImageNet64上实现了1.16的FID分数，在MS-COCO上实现了7.42的FID分数，显著优于之前的蒸馏方法和生成模型。此外，实验还验证了加权函数在减少低密度区域分数匹配误差方面的有效性。
本文提出了一种新的方法，叫做f-distill，用来加速生成图像的过程。传统的扩散模型需要很多步才能生成一张图片，这很慢，尤其是在需要快速生成图片的应用中。f-distill的核心思想是通过一种叫做f-散度的数学工具，把多步生成过程压缩成一步。f-散度有很多种，比如反向KL、正向KL、Jensen-Shannon等，每种散度在生成图片时都有不同的特点。比如，有些散度会让生成的图片更丰富多样，但训练时可能会不太稳定；有些散度则会让生成的图片更集中，但可能会丢失一些细节。f-distill通过一个加权函数，自动调整在不同区域的生成策略，特别是在图片细节丰富的地方，模型会更多地依赖教师的指导，而在细节较少的地方，模型则会减少依赖。实验证明，使用Jensen-Shannon散度的f-distill在多个图像生成任务中表现最好，生成的图片质量高，速度快。
2025-02-21｜Tongji,USTC｜⭐️
http ://arxiv.org/abs/2502.15167v1https ://github.com/strawhatboy/M3-AGIQA
随着AI生成图像（AGI）模型的快速发展，评估其质量变得愈发复杂，需要考虑感知质量、提示对应性和真实性等多个维度。为了解决这些挑战，本文提出了M3-AGIQA，一个多模态、多轮次、多方面的AGI质量评估框架。该框架利用多模态大语言模型（MLLMs）作为文本和图像的联合编码器，并通过低秩适应（LoRA）微调将在线MLLMs的高级图像描述能力蒸馏到本地模型中。M3-AGIQA还引入了结构化的多轮评估机制，生成中间图像描述以深入分析质量、对应性和真实性。为了与人类感知判断对齐，框架还包含一个由xLSTM和回归头构建的预测器，用于处理序列逻辑并预测平均意见分数（MOS）。在多个基准数据集上的实验表明，M3-AGIQA在捕捉AGI质量的细微差异方面表现出色，且跨数据集验证证实了其强大的泛化能力。
M3-AGIQA方法分为三个阶段：初始描述阶段、推理阶段和训练阶段。在初始描述阶段，通过在线MLLM生成图像描述，涵盖质量、对应性和真实性三个方面，并通过LoRA微调将这些能力蒸馏到本地MLLM中。推理阶段利用MLLM的零样本链式思维（CoT）能力生成中间推理过程，增强评估的准确性。训练阶段则通过xLSTM特征提取器和回归头将MLLM的输出与人类感知判断对齐，使用均方误差（MSE）损失进行优化。具体步骤包括：
初始描述阶段：使用在线MLLM生成图像描述，并通过两轮对话格式微调本地MLLM。
推理阶段：利用MLLM的CoT能力生成中间推理过程，增强评估的准确性。
训练阶段：通过xLSTM特征提取器和回归头将MLLM的输出与人类感知判断对齐，使用MSE损失进行优化。
实验在AGIQA-3k、AIGCIQA2023和AIGCIQA-20k三个数据集上进行，评估了M3-AGIQA在质量、对应性和真实性方面的表现。与现有方法相比，M3-AGIQA在大多数指标上表现优异，尤其是在捕捉AGI质量的细微差异方面。通过消融实验，验证了图像描述、蒸馏和微调以及MLLM作为编码器对整体性能的贡献。跨数据集验证进一步证实了M3-AGIQA的泛化能力。实验结果表明，M3-AGIQA在AGI质量评估任务中达到了新的最先进水平。
M3-AGIQA的核心思想是通过多模态大语言模型（MLLMs）来评估AI生成的图像质量。首先，它利用在线MLLM生成图像描述，涵盖质量、对应性和真实性三个方面，然后将这些能力通过微调技术转移到本地MLLM中。接着，M3-AGIQA通过多轮对话机制生成中间推理过程，帮助更准确地评估图像质量。最后，它使用一个特殊的预测器（xLSTM和回归头）将MLLM的输出与人类感知判断对齐，从而预测出图像的质量分数。简单来说，M3-AGIQA通过结合文本和图像信息，生成详细的图像描述，并通过多轮对话和预测器来确保评估结果与人类感知一致，最终达到更准确的图像质量评估效果。
2025-02-17｜AmazonBedrockScience｜⭐️🟡
http ://arxiv.org/abs/2502.14896v1
本文全面综述了文本到图像（T2I）扩散模型中的概念擦除技术。随着T2I模型在生成高质量、多样化图像方面的显著进展，其复制受版权保护的风格、敏感图像和有害内容的能力引发了严重的伦理和法律问题。概念擦除提供了一种主动的替代方案，通过修改T2I模型来防止生成不需要的内容。本文系统地分类了现有的概念擦除方法，基于其优化策略和修改的模型组件，将其分为参数更新的微调、高效编辑的闭式解以及无需权重修改的推理时干预。此外，本文还探讨了绕过擦除技术的对抗攻击，并讨论了新兴的防御策略。为了支持进一步研究，本文整合了关键数据集、评估指标和基准，用于评估擦除效果和模型鲁棒性。本文为概念擦除领域的演变、挑战和未来方向提供了全面的资源。
概念擦除方法主要分为三类：微调、闭式模型编辑和推理时干预。微调方法通过迭代优化模型参数来擦除不需要的概念，具体包括更新潜在扩散模型（LDM）的交叉注意力层或直接修改LDM的去噪过程。闭式模型编辑方法通过直接计算参数更新来实现高效的概念擦除，避免了迭代训练的计算开销。推理时干预方法则在不修改模型权重的情况下，通过调整分类器自由引导、编辑文本嵌入或使用大型语言模型（LLM）预处理提示来实现概念擦除。此外，本文还探讨了对抗攻击和防御机制，分析了如何通过对抗训练和鲁棒性增强技术来提高概念擦除的可靠性。
实验部分详细评估了概念擦除方法的有效性和模型保真度。擦除成功率（ESR）通过预训练分类器评估目标概念是否在生成图像中被成功移除。模型保真度则通过Fre´chetInceptionDistance（FID）和CLIPscore来评估生成图像的质量和文本-图像对齐度。常用的数据集包括用于评估NSFW内容移除的I2P数据集、用于对象概念移除的结构化提示数据集以及用于艺术风格擦除的ESD艺术家提示数据集。此外，本文还介绍了专门用于测试对抗性漏洞的数据集，如MMA-Diffusion基准和Ring-A-Bell数据集。这些评估指标和数据集为概念擦除方法的比较和未来研究提供了标准化框架。
概念擦除技术就像是给AI模型装了一个“过滤器”，让它不再生成某些特定的内容。比如，如果你不想让AI画出梵高风格的画，你可以通过微调模型，让它把“梵高”这个概念替换成“艺术家”，这样即使你输入“梵高风格的画”，AI也不会画出梵高风格的作品。还有一种方法是通过直接修改模型的某些参数，快速实现概念擦除，而不需要反复训练模型。此外，还有一种方法是在生成图像的过程中，通过调整一些设置或使用大型语言模型来过滤掉不需要的内容。这些方法各有优缺点，但目的都是为了确保AI生成的内容符合伦理和法律要求。
2025-02-21｜EPFL,SonyAI,WestlakeU｜⭐️🟡

本文提出了CopyJudge，一个自动化的版权侵权识别和缓解框架，专门针对文本到图像扩散模型。随着这些模型在生成高质量图像方面的能力不断增强，它们有时会记忆并复制训练数据中的内容，从而引发版权侵权的担忧。CopyJudge利用大规模视觉语言模型（LVLM）来模拟实际法庭流程，通过抽象-过滤-比较测试框架来评估生成图像与受版权保护图像之间的实质性相似性。此外，CopyJudge还引入了一种基于LVLM的缓解策略，通过优化输入提示词和探索潜在噪声向量来减少侵权风险，同时保留非侵权内容。实验结果表明，CopyJudge在识别侵权方面达到了与现有技术相当的性能，并且在泛化能力和可解释性方面表现优异。
CopyJudge的核心方法分为两个主要阶段：侵权识别和侵权缓解。在侵权识别阶段，框架首先通过抽象-过滤-比较测试来分解图像，提取受版权保护的元素，并评估其与生成图像的相似性。具体步骤如下：
抽象：将图像分解为不同的层次或元素，如构图、颜色和主题，以区分图像的基本概念和具体表达。
过滤：去除不受版权保护的部分，如公共和功能性表达，保留具有原创性的元素。
比较：通过多LVLM辩论机制，多个LVLM对过滤后的元素进行讨论和评分，最终由元法官LVLM根据共识给出侵权判定。
在侵权缓解阶段，CopyJudge通过以下两种策略来减少侵权风险：
提示词优化：利用LVLM迭代调整输入提示词，避免生成侵权表达，同时保留原始内容。
潜在噪声向量探索：通过强化学习在扩散模型的潜在空间中搜索非侵权噪声向量，即使不修改原始提示词，也能有效减少侵权风险。
实验部分验证了CopyJudge在侵权识别和缓解方面的有效性。首先，在侵权识别实验中，CopyJudge在多个数据集上表现出色，尤其是在卡通角色和艺术作品数据集上，显著优于传统的图像复制检测方法。其次，在侵权缓解实验中，CopyJudge通过提示词优化和潜在噪声向量探索，成功减少了生成图像的侵权风险，同时保持了图像与原始提示词的一致性。实验结果表明，CopyJudge不仅在识别侵权方面达到了与现有技术相当的性能，还在泛化能力和可解释性方面表现优异。
CopyJudge的工作方式可以类比为一个“智能法官”，它通过三个步骤来判断一张AI生成的图片是否侵犯了版权。首先，它会将图片分解成不同的部分，比如颜色、构图等，就像法官分析案件的不同细节。然后，它会过滤掉那些不受版权保护的部分，比如公共元素，只留下那些可能涉及侵权的部分。最后，它会通过多个“智能助手”进行讨论，最终决定这张图片是否侵权。如果判定为侵权，CopyJudge还会通过调整生成图片的提示词或修改图片的“内部参数”来减少侵权风险，就像法官给出解决方案一样。这种方法不仅高效，还能确保生成的图片既符合用户需求，又不会侵犯版权。

在生成式AI的繁荣时代，AI生成内容的可追溯性成为了促进负责任使用生成式AI的关键解决方案。图像水印技术被广泛认为是一种有效的手段，用于保护版权或防止虚假图像的滥用。然而，现有的水印系统在面对黑盒攻击时的鲁棒性仍然是一个挑战。本文提出了一种基于深度图像先验（DeepImagePrior,DIP）的黑盒水印去除方法，该方法无需任何水印图像数据集或水印系统的先验知识。通过DIP，我们能够从单张水印图像中生成高质量的去除水印的图像，并展示了DIP在去除不可见水印方面的有效性。此外，本文还探讨了DIP在对抗基于训练的可见水印时的局限性，并讨论了基于训练的可见水印在防止虚假信息滥用中的积极意义。
本文提出了一种基于深度图像先验（DIP）的黑盒水印去除方法，具体步骤如下：
DIP基础：DIP是一种利用未训练的深度神经网络（DNN）作为自然图像的隐式先验来解决图像恢复问题的技术。对于任何自然图像I，DIP将其参数化为I=Gθ(z)，其中Gθ是一个可训练的卷积神经网络（CNN），z是一个固定的输入（通常随机生成）。
水印去除公式：给定一个水印图像Iw，我们通过最小化Iw与Gθ(z)之间的均方误差（MSE）来去除水印。具体公式为：minθ||Iw-Gθ(z)||2。通过迭代优化，DIP能够逐步去除水印，同时保留图像的高质量。
频率分离原理：DIP在去除水印时，低频率成分（主导自然图像）比高频率成分（通常由噪声或水印引起）更快地被捕捉。这种频率分离使得DIP能够在去除水印的同时保持图像的主要结构。
算法实现：我们使用ADAM优化器对DIP进行优化，并在迭代过程中记录所有中间结果。通过查询水印检测API，我们能够找到成功去除水印的图像。
本文在MS-COCO和DiffusionDB两个大规模数据集上进行了实验，评估了六种代表性的后处理水印方法和一种最新的处理中水印方法（TreeRing）。实验结果表明，基于DIP的水印去除方法在去除不可见水印方面表现出色，能够生成高质量的去除水印的图像。具体评估指标包括峰值信噪比（PSNR）、结构相似性指数（SSIM）和像素差异的90%分位数。实验还对比了DIP与其他经典数字编辑方法和最新的净化方法（如DiffPure、Diffuser和VAE再生）的性能，结果显示DIP在大多数情况下表现最佳。
本文的方法可以理解为一种“智能橡皮擦”，它能够在不破坏图像质量的情况下，擦除图像中的水印。具体来说，我们使用了一种叫做深度图像先验（DIP）的技术，它就像是一个没有经过专门训练的“画家”，能够通过观察一张有水印的图片，逐步“修复”这张图片，去掉水印。DIP的工作原理是，它首先捕捉图片中的主要部分（比如风景或人物的轮廓），然后再处理细节部分（比如水印）。通过这种方式，DIP能够在去除水印的同时，保持图片的清晰度和真实感。实验证明，这种方法在去除不可见水印方面非常有效，而且生成的图片质量很高。

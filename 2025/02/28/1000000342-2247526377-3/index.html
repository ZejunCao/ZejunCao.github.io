主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）发布日期｜2025年2月24日｜周一
2025-02-21｜DKFZ,HelmholtzImaging,HeidelbergUniversity
http ://arxiv.org/abs/2502.15563v1
本文提出了一种资源高效的方法，用于生成特定领域的视觉语言模型（VLM）基准测试。现有的VLM基准测试虽然在多个领域和任务上提供了广泛的模型能力评估，但其设计异质性以及对少数图像领域的有限关注，使得跨领域性能比较和特定领域评估面临挑战。为了解决这一问题，本文提出了三个关键贡献：1）一个通过任务增强从单一任务生成多个多样化任务的框架，2）发布了七个新领域的VLM基准测试，包含162,946个经过人工验证的答案，3）对22个最先进的VLM在37,171个任务上进行了广泛基准测试，揭示了跨领域和任务的性能差异。该方法为资源高效的领域特定模型选择和未来研究方向提供了指导。
本文的方法主要包括以下几个步骤：
任务增强框架：通过从单一任务（如图像实例分割）生成多个多样化任务，利用来自多个来源（人类、预定义启发式、模型）的元数据注释，创建资源高效的领域特定VLM基准测试。
新基准测试的生成：应用该框架生成了七个新领域的VLM基准测试，每个基准测试都包含大量经过人工验证的答案，确保了数据的可靠性。
模型评估：对22个最先进的VLM在37,171个任务上进行了广泛评估，揭示了模型在不同领域和任务上的性能差异。
实验部分展示了所提出框架的多个优势：
任务多样性：通过任务增强生成的25个不同任务，涵盖了从像素级到图像级的多种视觉语言能力，展示了框架的多样性。
人类歧义基线：通过获取每个任务的六个人类评分，评估了任务的难度和歧义性，为模型评估提供了可靠的参考。
模型性能分析：对22个VLM在多个领域的性能进行了详细分析，揭示了模型在不同领域和任务上的显著性能差异，强调了领域特定评估的重要性。
本文的方法可以简单理解为：我们开发了一个“任务生成器”，它可以从一个简单的任务（比如给图片中的物体画框）生成多个复杂的任务（比如数一数图片中有多少只动物，或者判断哪些物体被遮挡了）。这个生成器不仅节省了时间和资源，还能生成多样化的任务，帮助我们更全面地评估AI模型的能力。我们用它生成了七个新领域的测试任务，并对22个最新的AI模型进行了测试，发现它们在处理不同领域的任务时表现差异很大。这说明，要想让AI在实际应用中表现更好，我们需要针对特定领域进行专门的测试和优化。
2025-02-19｜USTC,AlibabaGroup,SJTU
http ://arxiv.org/abs/2502.14914v1
随着多模态大语言模型（MLLMs）的快速发展，传统的视觉描述基准已经显得过时，因为它们主要评估简短的描述，并使用过时的指标。尽管最近的基准通过将描述分解为视觉元素并采用基于模型的评估方法，部分解决了这些局限性，但它们仍然不完整，忽略了关键方面，并且提供的评分模糊且缺乏解释性。为了弥补这一差距，我们提出了CV-CapBench，一个全面的视觉描述基准，系统地评估了描述质量，涵盖了6个视角和13个维度。CV-CapBench引入了每个维度的精确度、召回率和命中率指标，独特地评估了正确性和覆盖率。对领先的MLLMs进行的实验揭示了显著的能力差距，特别是在动态和知识密集型维度上。这些发现为未来的研究提供了可操作的见解。
CV-CapBench的设计包括多个维度的数据注释和评估方法。首先，我们设计了9个静态维度和4个动态维度，这些维度共同构成了一个完整的视觉描述。静态维度适用于图像或视频的特定帧，而动态维度则需要整个视频的信息。我们收集了近1000张图像/视频用于每个维度，并使用最先进的MLLMs进行预注释，随后手动校正每个样本。为了评估描述质量，我们使用GPT-4Turbo对每个维度进行评分，计算精确度、召回率和命中率。精确度表示描述中提到的内容的准确性，召回率表示所有样本中被正确描述的比例，命中率表示描述中尝试覆盖该维度的比例。我们还引入了“一代表所有”的预注释策略，通过随机选择图像中的对象来反映描述的正确性和覆盖率。
我们对多个领先的MLLMs进行了实验，评估了它们在CV-CapBench上的表现。实验结果显示，尽管所有模型在静态维度（如OCR和对象类别）上表现出色，但在动态维度（如相机运动和动作）上表现较差。例如，相机运动的召回率仅为0.40%至10.37%，F1得分也较低。此外，模型在字符识别等知识密集型维度上的表现也不尽如人意。我们还发现，不同模型在不同维度上的表现存在显著差异，Qwen2.5VL-7B在精确度上领先，而LLaVA-OneVision-7B在召回率和F1得分上表现最佳。这些结果表明，当前模型在动态和知识密集型维度上仍有很大的改进空间。
CV-CapBench是一个新的工具，用来评估AI模型生成图像和视频描述的能力。它不仅仅看描述是否准确，还看描述是否全面。我们设计了13个不同的方面来评估描述，比如描述中是否提到了所有重要的对象、颜色、场景等。对于视频，我们还评估了描述中是否包含了动作和相机运动等动态信息。为了测试这些模型，我们收集了大量的图像和视频，并让AI模型生成描述。然后，我们使用另一个强大的AI模型（GPT-4Turbo）来检查这些描述是否准确和全面。实验结果显示，虽然AI模型在描述静态信息（比如对象和文字）上做得不错，但在描述动态信息（比如动作和相机运动）上还有很大的提升空间。这个工具可以帮助研究人员更好地理解AI模型的优缺点，并指导未来的改进方向。
2025-02-21｜LMU,MCML
http ://arxiv.org/abs/2502.15457v1
本文探讨了多模态大语言模型（MLLMs）在处理视频流事件时的表现，特别是如何利用记忆来增强对当前事件的理解。研究表明，虽然记忆能够提供上下文和时间信息，帮助MLLMs更好地理解视频事件，但由于记忆依赖于对先前事件的预测，可能会导致信息失真，即“虚构”（confabulation）。为了解决这一问题，本文提出了一种名为CAMEO的虚构感知记忆修改方法，旨在减少虚构记忆对模型性能的负面影响。通过实验，本文展示了记忆在增强事件理解方面的潜力，同时也揭示了虚构记忆在流媒体场景中的误导性。
本文的方法主要分为三个部分：记忆作为上下文、虚构记忆的识别与修正。
记忆作为上下文：MLLMs通过将记忆作为上下文输入来增强对当前事件的理解。记忆分为长期记忆和短期记忆，长期记忆来自其他视频片段，短期记忆则来自当前视频片段中的先前事件。通过将记忆与当前事件结合，MLLMs能够更好地理解事件的上下文和时间关系。
虚构记忆的识别：在流媒体场景中，MLLMs无法访问真实的记忆，而是依赖于先前生成的叙述。这些叙述可能包含错误信息，导致虚构记忆的产生。本文通过语义熵来评估生成叙述的可信度，语义熵越高，叙述的可信度越低。
虚构记忆的修正（CAMEO）：CAMEO方法通过识别和修正虚构记忆来减少其对模型的影响。具体来说，CAMEO通过探测模型中的注意力头，找到那些容易产生虚构记忆的注意力头，并通过调整这些头的权重来减少虚构记忆的贡献。最终，CAMEO能够显著提高MLLMs在流媒体事件理解中的表现。
实验部分主要验证了记忆对MLLMs事件理解能力的增强作用，以及虚构记忆在流媒体场景中的负面影响。实验使用了Ego4D数据集，并对比了三种模型设置：无记忆、带有真实记忆和带有虚构记忆的MLLMs。结果表明，带有真实记忆的MLLMs在事件理解任务中表现最佳，而带有虚构记忆的模型则表现较差。通过引入CAMEO方法，模型在流媒体场景中的表现得到了显著提升，证明了CAMEO在减少虚构记忆影响方面的有效性。
想象一下，你正在看一部连续剧，每一集都依赖于前一集的情节来理解当前的故事。如果你能记住前一集的所有细节，那么你就能更好地理解当前的情节。这就是本文的核心思想：利用记忆来帮助MLLMs更好地理解视频中的事件。然而，问题在于，MLLMs的记忆并不是完美的，它们依赖于对先前事件的预测，而这些预测可能会有错误，就像你可能会记错前一集的某些细节一样。这种错误记忆被称为“虚构”，它会导致MLLMs对当前事件的理解出现偏差。为了解决这个问题，本文提出了一种名为CAMEO的方法，它就像是一个“记忆修正器”，能够识别并减少这些错误记忆的影响，从而让MLLMs在理解视频事件时更加准确。
2025-02-17｜WHU
http ://arxiv.org/abs/2502.14893v1
本研究提出了NOTA，一个大规模的多模态音乐符号理解数据集，旨在解决当前视觉大语言模型在音乐符号理解方面的不足。NOTA数据集包含来自全球三个地区的1,019,237条记录，涵盖了三个主要任务：音乐信息提取、跨模态对齐测试和音乐符号分析。基于该数据集，我们训练了NotaGPT，一个能够理解多模态音乐符号的视觉大语言模型。实验结果表明，NotaGPT在音乐理解方面取得了显著进步，展示了NOTA数据集和训练流程的有效性。
跨模态对齐预训练：首先，我们进行了一个预对齐训练阶段，旨在实现音乐乐谱图像中的音符与其在ABC符号中的文本表示之间的特征对齐。这一阶段通过冻结视觉编码器和语言模型，仅训练两层MLP视觉-语言连接器来实现。
音乐信息提取微调：接下来，我们利用NOTA数据集对模型进行微调，使其能够从乐谱图像中提取基本的音乐信息，如音符长度和调号等。这一阶段通过冻结视觉编码器并训练其余部分来提升模型的音乐信息提取能力。
音乐符号分析监督微调：最后，我们通过监督微调进一步训练模型，使其能够进行复杂的音乐符号分析，如音乐风格和节奏分析。这一阶段通过使用NOTA数据集中的音乐分析数据样本进行训练，增强了模型的音乐分析能力。
我们进行了三个子实验来验证NOTA数据集的有效性：音乐信息提取、跨模态对齐和音乐符号分析。在音乐信息提取实验中，NotaGPT在多个子任务中显著优于其他模型，特别是在音符长度和调号提取方面。在跨模态对齐实验中，NotaGPT在将乐谱图像转换为ABC符号文本的任务中表现出色，展示了其强大的跨模态对齐能力。在音乐符号分析实验中，NotaGPT在音乐风格和乐谱结构分析任务中表现优异，证明了其在复杂音乐理解任务中的有效性。
通俗易懂：想象一下，你有一本乐谱，里面有很多音符和符号，但电脑看不懂这些符号。NOTA就像是一个翻译官，它帮助电脑理解这些乐谱。首先，NOTA会教电脑如何把乐谱上的音符和电脑能理解的文字对应起来，这就像教电脑认字。然后，NOTA会教电脑从乐谱中提取出一些基本信息，比如音符的长度和调号，这就像教电脑读懂简单的句子。最后，NOTA会教电脑分析乐谱的风格和结构，这就像教电脑理解整篇文章的意思。通过这样的训练，电脑就能更好地理解和处理音乐乐谱了。
2025-02-16｜UTS(Sydney),SuZhouYierqi,USYD｜ICLR2025｜🟡
http ://arxiv.org/abs/2502.14889v1https ://github.com/LMBTough/NIB
本文提出了一种名为“NarrowingInformationBottleneckTheory(NIBT)”的新框架，旨在解决多模态图像-文本表示模型（如CLIP）的可解释性问题。尽管CLIP在图像和文本之间的复杂关联学习上表现出色，但其黑箱性质在高风险应用（如医疗诊断）中带来了挑战。现有的可解释性方法大多针对单模态任务，难以直接应用于多模态场景。传统的瓶颈方法虽然基于信息论，但受到强假设和内在随机性的限制。NIBT通过重新定义传统瓶颈方法，消除了随机性和超参数依赖，提供了更鲁棒和可靠的可解释性解决方案。实验表明，与现有方法相比，NIBT在图像可解释性上平均提升了9%，文本可解释性提升了58.83%，并加速了63.95%的处理速度。
NIBT的核心是通过理论推导，消除传统信息瓶颈方法中的随机性和超参数依赖。具体方法包括以下三个关键点：
瓶颈层的引入：在神经网络的第l层引入瓶颈层，通过参数λ控制信息流。λ是一个标量，用于独立控制每个特征维度的信息流，确保计算过程中没有内在随机性。
负属性的引入：提出了一种新的负属性概念，用于识别对模型预测产生负面影响的特征维度，从而进一步提升模型的可解释性。
理论推导：通过三个核心定理（定理1、定理2、定理3）证明了NIBT的有效性。定理1表明，通过减小λ的值，可以减少输入样本与其编码之间的互信息；定理2证明了当噪声方差趋近于零时，定理1的结论仍然成立，从而消除了随机性；定理3则通过积分形式计算特征维度的重要性，避免了超参数β的引入。
实验在三个数据集（ConceptualCaptions、ImageNet、Flickr8k）上进行，使用预训练的CLIP模型作为视觉编码器。评估指标包括图像置信度下降（ImageConfidenceDrop）、图像置信度提升（ImageConfidenceIncrease）、文本置信度下降（TextConfidenceDrop）和文本置信度提升（TextConfidenceIncrease）。实验结果表明，NIBT在所有数据集上均优于现有的可解释性方法（如M2IB、RISE、Grad-CAM等），在图像和文本的可解释性上均取得了显著提升。此外，NIBT在计算效率上也表现出色，处理速度显著高于其他方法。消融实验进一步验证了关键参数（如迭代次数和目标层数）对模型性能的影响，表明NIBT在不同设置下均能保持稳定的性能。
NIBT的核心思想是通过一种新的“瓶颈”方法来解释多模态模型（如CLIP）的决策过程。想象一下，CLIP就像一个黑箱，我们不知道它如何将图像和文本联系起来。NIBT就像给这个黑箱装了一个“过滤器”，通过控制信息的流动，让我们能够看到哪些特征对模型的决策最重要。这个“过滤器”通过一个参数λ来控制信息的多少，λ越小，过滤掉的信息越多，留下的就是最关键的特征。NIBT还引入了一个新概念，叫做“负属性”，它可以帮助我们识别那些对模型决策有负面影响的特征，从而进一步提升解释的准确性。通过这种方法，NIBT不仅提高了模型的可解释性，还大大加快了处理速度，使得它在实际应用中更加高效和可靠。

306-A2&3｜基于LLM智能体的逼真3D世界创建与定制；硬件友好静态量化用于视频DiT后训练量化，基于LVLM的生成视频检测

AIGCResearch
主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）发布日期｜2025年2月24日｜周一
2025-02-21｜HKUST,DartmouthCollege｜⭐️⭐️
http ://arxiv.org/abs/2502.15601v1
WorldCraft是一个创新的系统，利用大型语言模型（LLM）代理来创建复杂且逼真的3D虚拟世界。用户可以通过简单的文本指令生成包含精确几何和物理渲染（PBR）纹理的3D场景。该系统允许用户通过自然语言交互定制单个对象和场景布局，从而实现美学、功能和人体工程学设计。WorldCraft的目标是简化3D虚拟世界的创建过程，使其对非专业人士更加友好，从而让更多人能够实现他们的创意愿景。该系统在单对象定制到复杂的大规模室内外场景设计中展示了其多功能性。
WorldCraft的方法分为三个主要阶段：对象创建、布局生成和场景动画。首先，在对象创建阶段，系统通过ForgeIt代理生成3D对象。ForgeIt利用LLM代理来导航程序化生成器，生成具有精确几何和纹理的3D对象。它通过自动验证机制动态构建一个不断增长的手册，指导代理掌握复杂的程序化生成器。其次，在布局生成阶段，系统使用ArrangeIt代理来设计场景布局。ArrangeIt将场景布局建模为分层数值优化问题，并通过新颖的优化协议解决这些问题，确保布局符合功能性和美学要求。最后，在场景动画阶段，系统通过轨迹控制模块允许用户通过自然语言指令控制对象和摄像机的运动，从而生成动画场景。整个系统还兼容现成的深度3D生成器，以丰富场景资产。
WorldCraft在多个复杂场景生成任务中进行了评估，展示了其生成高质量3D场景的能力。实验结果表明，WorldCraft在一致性、美学和功能性方面均优于现有的最先进方法。通过用户研究和定量评估，系统展示了其在解释和执行复杂3D世界创建指令方面的能力。此外，ForgeIt和ArrangeIt模块的有效性通过对比实验得到了验证，证明了动态手册构建和分层优化在提升生成质量方面的作用。WorldCraft还展示了其在生成大规模室内外场景中的多功能性，用户可以通过自然语言交互进一步编辑和增强场景。
WorldCraft就像是一个虚拟世界的“魔法师”，它能够根据你的文字描述，快速生成一个逼真的3D场景。比如，你可以告诉它“我想要一个充满摇滚元素的卧室”，它就会自动生成一个包含吉他、鼓组和摇滚海报的房间。你还可以通过简单的对话来调整房间里的物品，比如把床移到窗边，或者在墙上挂一幅画。WorldCraft的核心技术是三个“小助手”：ForgeIt负责生成每个物品，ArrangeIt负责把这些物品合理地摆放在场景中，而轨迹控制模块则让你可以像导演一样，控制摄像机的移动，拍出电影般的场景动画。整个过程就像是在玩一个高级版的“模拟人生”，只不过你只需要动动嘴，就能创造出你想象中的世界。
2025-02-20｜Caltech,SamsungSemiconductorInc.｜⭐️
http ://arxiv.org/abs/2502.15077v1
本文提出了一种硬件友好的静态量化方法，用于视频扩散变换器（VideoDiffusionTransformers,DiT）的后训练量化。与动态量化不同，静态量化在推理过程中不需要实时计算量化参数，因此更适合资源受限的设备。该方法通过使用每步校准数据，为每个时间步提供后训练静态量化模型，并结合通道级权重量化和张量级激活量化。通过进一步应用平滑量化技术，该方法能够在静态量化模型上生成高质量的视频输出。实验结果表明，静态量化在视频扩散变换器上可以成为动态量化的可行替代方案，提供了一种更高效的方法，且不牺牲性能。
本文提出了两种静态量化方法：聚合静态量化和时间步静态量化。聚合静态量化通过聚合所有去噪步骤的激活来估计一组量化参数，适用于所有去噪步骤。时间步静态量化则为每个时间步单独估计量化参数。两种方法都采用了通道级权重量化和张量级激活量化。此外，本文还引入了平滑量化技术，通过平滑权重和激活的通道差异，进一步提高了量化质量。具体来说，平滑量化通过计算每个通道的最大绝对值来确定缩放因子，从而减少量化误差。时间步静态量化通过在校准阶段为每个时间步估计量化参数，避免了推理过程中的额外计算，从而实现了高效的硬件优化。
实验在STDiTv1.0模型上进行了广泛的评估，使用了不同的比特宽度和评估设置。量化参数通过校准集进行估计，校准集由10个提示生成。评估指标包括CLIPSIM、CLIP-temp、VQA-aesthetic和VQA-technical，分别用于衡量文本-视频对齐、时间语义一致性、视频美学和视频技术质量。实验结果表明，本文提出的静态量化方法在多个指标上优于动态量化方法。特别是，时间步静态量化在生成语义对齐的视频方面表现尤为出色。此外，实验还验证了平滑量化技术的重要性，并展示了不同比特宽度下的方法鲁棒性。
本文的方法主要是为了让视频生成模型在资源有限的设备上也能高效运行。通常，视频生成模型需要大量的计算资源，而动态量化虽然能减少计算量，但在推理过程中需要实时计算量化参数，这对资源受限的设备来说是不现实的。本文提出的静态量化方法则不同，它在模型训练后通过校准数据预先计算好量化参数，推理时直接使用这些参数，不需要额外的计算。具体来说，本文的方法对模型的权重和激活分别进行了量化，权重是按通道量化的，而激活是按张量量化的。此外，还引入了一种平滑量化技术，通过调整权重和激活的缩放因子，进一步减少了量化带来的误差。实验证明，这种方法不仅能保持视频生成的质量，还能显著提高模型的运行效率。
2025-02-20｜ColumbiaU｜⭐️
http ://arxiv.org/abs/2502.14994v1
随着生成模型在创建高质量视频方面的显著成就，数字完整性和隐私漏洞的担忧也随之增加。尽管在图像领域（如深度伪造）中，AI生成内容的检测已被广泛研究，但视频领域仍未被充分探索。大型视觉语言模型（LVLM）因其强大的推理和多模态能力，成为AI生成内容检测的新兴工具。本文提出了LAVID，一种基于LVLM的AI生成视频检测框架，通过显式知识增强来提高检测效果。LAVID的核心思想是利用LVLM调用外部工具提取有用信息，并通过结构化提示优化推理能力。与现有方法不同，LAVID无需额外训练检测器，仅依赖LVLM的推理能力进行检测。实验结果表明，LAVID在多个数据集上的F1分数比现有基线提高了6.2%到30.2%。
LAVID方法的核心包括以下几个步骤：
显式知识工具包选择：LAVID利用LVLM的推理能力，自动搜索并收集一组显式知识（EK）工具。这些工具包括光学流、深度图、饱和度等，能够从视频中提取有助于检测的信息。通过参考样本集，LAVID为每个LVLM定制一个工具包，确保工具的选择与模型的推理能力相匹配。
在线自适应与结构化提示：LAVID通过自适应的提示重写机制，实时调整结构化提示的格式。结构化提示为LVLM提供了一个“思考框架”，从而减少非结构化提示中的幻觉问题。在线自适应过程基于测试集的反馈，动态调整提示结构，避免模型过拟合。
模型特定的EK选择：不同的LVLM对显式知识的理解能力不同。LAVID通过预定义的评分指标，为每个LVLM选择最合适的工具集。评分指标综合考虑了模型的加权准确率和主观评估，确保工具的选择能够最大化检测效果。
显式知识增强的检测：LAVID通过提取显式知识特征，结合LVLM的推理能力，进行视频检测。每个工具的分析结果通过逻辑“或”操作进行集成，最终得出视频是否为AI生成的结论。
实验部分主要评估了LAVID在多个数据集上的表现。实验使用了四个领先的LVLM模型（Llava-OV-7B、Qwen-VL-Max、Gemini-1.5-pro和GPT-4o），并在新创建的VidForensic基准数据集上进行了测试。VidForensic包含了1400多个高质量视频，涵盖了多个视频生成工具（如Kling、RunwayGen3等）。实验结果表明，LAVID在所有数据集上均显著优于基线方法，F1分数提高了6.2%到30.2%。特别是在高质量视频数据集上，LAVID的表现尤为突出，展示了其在检测AI生成视频方面的强大能力。
LAVID的核心思想是利用大型视觉语言模型（LVLM）的推理能力来检测AI生成的视频。首先，LAVID会调用一些外部工具，比如分析视频的光学流、深度图等信息，这些信息可以帮助模型更好地理解视频内容。然后，LAVID通过一种叫做“结构化提示”的方式，让模型按照特定的格式进行推理，这样可以减少模型在推理过程中产生的错误。此外，LAVID还会根据不同的模型，选择最适合的工具集，确保每个模型都能发挥出最好的检测效果。最后，LAVID通过整合这些工具的分析结果，判断视频是否为AI生成。实验表明，LAVID在多个数据集上的表现都非常出色，比现有的方法提高了不少。

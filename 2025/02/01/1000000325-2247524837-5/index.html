2025-01-24｜UMichigan,CMU｜ICLR2025
http ://arxiv.org/abs/2501.14319v1https ://github.com/Xiaohao-Xu/SLAM-under-Perturbation.
本研究旨在重新定义自我运动估计和逼真的三维重建，重点解决现有模型对无噪声数据的依赖问题。虽然在理想条件下的评估简化了研究，但却无法反映实际环境中的复杂噪声因素，如动态运动、传感器缺陷和同步误差。这些因素在实际应用中导致性能显著下降，因此迫切需要一种能够在真实世界噪声下表现出色的框架。为此，研究提出了三项核心挑战的解决方案：可扩展的数据生成、全面的基准测试以及模型鲁棒性增强。具体而言，研究者们开发了一种可扩展的噪声数据合成管道，创建了一个名为Robust-Ego3D的基准测试，以评估现有模型在噪声条件下的性能，并提出了一种新颖的Correspondence-guidedGaussianSplatting（CorrGS）方法，旨在通过视觉对应关系增强几何对齐和外观恢复。
本研究提出的CorrGS方法结合了高斯点云表示和基于对应关系的可微优化，以实现鲁棒的自我运动估计和逼真的三维重建。具体方法包括以下步骤：
高斯表示：使用高斯点云作为地图表示，保持无噪声的状态，生成在估计姿态下的RGB-D图像。
可微优化：通过计算生成图像与观察到的噪声RGB-D帧之间的差异，进行可微优化，增强模型的鲁棒性。
对应关系引导：通过建立渲染和观察RGB帧之间的2D视觉匹配，将其提升为3D对应关系，增强几何对齐和外观恢复。
姿态初始化：利用对应关系初始化姿态，通过相对姿态计算来提升姿态参数学习的鲁棒性。
优化过程：通过最小化重建误差，得到最终的姿态估计，优化过程的计算复杂度与图像分辨率相关，确保低时间成本。
在实验部分，研究者们利用合成和真实世界的数据对CorrGS进行了广泛测试，评估其在快速运动和动态光照条件下的表现。实验结果表明，CorrGS在多种噪声环境中均优于现有的最先进方法，特别是在快速运动和动态光照的场景中，CorrGS能够实现更高的自我运动估计准确性和三维重建质量。此外，通过Robust-Ego3D基准测试，研究揭示了现有模型在动态运动下的姿态跟踪失败和三维重建质量下降的普遍问题，强调了需要在噪声条件下进行系统评估的必要性。
在这项研究中，我们提出了一种新方法，旨在帮助计算机更好地理解和重建三维环境，尤其是在现实生活中常见的噪声和不稳定条件下。我们的方法叫做CorrGS，主要分为几个简单的步骤。首先，我们使用一种特殊的技术来创建一个清晰的三维地图，并从中生成图像。接下来，我们将这些图像与真实的、可能有噪声的图像进行比较，以找出差异。然后，我们利用这些差异来调整计算机的“眼睛”，使其能够更准确地估计运动和重建场景。最后，我们通过这种方式，确保计算机在面对快速移动和光线变化等挑战时，依然能够保持准确性。这种方法的优势在于，它不仅可以处理理想条件下的情况，还能在真实世界的复杂环境中表现良好。
2025-01-24｜SNU
http ://arxiv.org/abs/2501.14277v1
本文提出了一种名为Dense-SfM的新型结构光重建框架，旨在从多视图图像中实现高密度和高精度的3D重建。传统的结构光重建方法通常依赖于稀疏的关键点匹配，这限制了在纹理稀少区域的准确性和点云密度。Dense-SfM通过结合密集匹配和基于高斯分布的轨迹扩展技术，克服了这一限制，提高了特征轨迹的连贯性和长度。此外，该框架还引入了一种多视图核化匹配模块，利用变换器和高斯过程架构，增强了多视图间的轨迹精细化。实验结果表明，Dense-SfM在ETH3D和Texture-PoorSfM数据集上显著提升了重建的精度和点云密度。
Dense-SfM框架的设计包含三个主要步骤：
初步SfM模型构建：通过密集匹配算法（如DKM或RoMa），从图像对中提取特征匹配，构建初步的3D点云模型。与传统方法不同，本方法采用非最大抑制技术来提高匹配的可靠性。
轨迹扩展：利用高斯分布对初步3D点进行轨迹扩展。通过将3D点投影到新图像上，验证其可见性，从而增加每个3D点的匹配长度。这一过程有助于解决传统方法中短轨迹的问题。
迭代精细化：通过多视图核化匹配模块和几何束调整，进一步优化相机姿态和3D点云。该模块结合特征信息和位置信息，利用深度学习技术对特征轨迹进行精细化处理，以提高重建的整体质量。
在ETH3D和Texture-PoorSfM数据集上进行了广泛的实验评估。实验结果显示，Dense-SfM在3D点云的准确性和完整性上均优于现有的最先进方法。具体而言，在不同的距离阈值下，Dense-SfM在多视图相机姿态估计中表现出色，尤其是在高阈值下的精度显著提高。此外，通过对比不同的匹配设计和精细化模块，结果表明，采用高斯分布的轨迹扩展和多视图核化匹配模块显著提升了重建效果。
Dense-SfM的工作原理可以简单理解为三个步骤。首先，它从多张图片中找到相似的部分，并用这些相似点来建立一个初步的3D模型。接下来，为了让这个模型更加完整，它会把这些3D点投影到其他图片上，检查哪些点在新图片中也能看到，从而延长这些点的“轨迹”。最后，通过不断地优化这些轨迹和相机的位置，Dense-SfM能够生成一个更加准确和丰富的3D模型。这个过程利用了现代深度学习技术，使得即使在缺乏纹理的环境中，也能得到高质量的3D重建。
2025-01-24｜BUAA,ShanghaiAILab｜AAAI2025
http ://arxiv.org/abs/2501.14231v1
在计算机视觉领域，3D重建是一项重要且具有挑战性的任务，尤其是在处理来自不同视角的非约束图像时。传统方法在保持一致的重建质量方面往往受到限制，尤其是在动态场景中。为了解决这些问题，本文提出了一种新颖的微观-宏观小波基高斯喷溅（MW-GS）方法。该方法通过将场景表示分解为全局、精细和内在三个组成部分，旨在有效处理动态外观变化和瞬态遮挡。MW-GS的核心创新在于微观-宏观投影和小波基采样，这两者结合能够从多尺度特征图中提取丰富的细节，显著提升3D重建的效果。通过广泛的实验，MW-GS在重建质量和渲染性能上超越了现有的技术，展示出其在实际应用中的潜力。
MW-GS方法的设计包括多个关键组成部分，以实现高保真度的3D重建。首先，从2D参考图像中提取全局外观特征，利用卷积神经网络（CNN）进行处理。其次，采用微观-宏观投影技术，通过在二维特征图上使用狭窄和宽广的圆锥截面进行自适应采样，从而增强精细外观特征的准确性和多样性。接着，构建层次残差融合网络（HRFN），有效整合全局、精细和内在特征，确保信息的全面融合。此外，采用小波变换对特征进行处理，以捕捉多分辨率信息，进一步提高重建精度。最后，使用可见性图来缓解瞬态对象的影响，优化重建过程。
在实验部分，MW-GS方法在多个数据集上进行了评估，包括布兰登堡门、圣心大教堂和特雷维喷泉。通过与现有方法（如NeRF-W、Ha-NeRF等）进行比较，MW-GS在峰值信噪比（PSNR）、结构相似性指数（SSIM）和LPIPS（感知图像相似性）等指标上均表现出色。定量结果表明，MW-GS在所有评估场景中均优于其他方法，尤其在细节捕捉和色彩重建方面。此外，通过消融实验，验证了微观-宏观投影、小波基采样和层次残差融合网络在提升重建质量方面的重要性。
在MW-GS方法中，我们通过几个步骤来实现高质量的3D重建。首先，我们从一张2D图片中提取出场景的整体特征，就像从一张照片中了解整个风景的样子。然后，我们使用一种叫做“微观-宏观投影”的技术，这就像用放大镜仔细观察某个细节，同时也不忘整体的样子。这种方法允许我们在不同的视角下捕捉到更多的细节。接下来，我们有一个层次融合的网络，它就像一个聪明的拼图师，能够把不同的特征拼接在一起，形成一个完整的画面。此外，我们还应用小波变换技术，这个技术可以帮助我们在不同的分辨率下提取信息，确保我们不会错过任何重要的细节。最后，我们使用可见性图来处理那些在照片中可能会被遮挡的物体，确保最终的3D重建尽可能准确和真实。
2025-01-24｜ETHZürich,Meta-CodecAvatarsLab,UTübingen
http ://arxiv.org/abs/2501.14726v1https ://neuralbodies.github.io/RFGCA
本研究提出了一种新颖的“可重光照全身高斯编码头像”（RelightableFull-BodyGaussianCodecAvatars）方法，旨在实现高保真、可重光照和生动动画的人体全身头像重建。该方法的独特之处在于其结合了学习的方向依赖散射传输和基于延迟着色的高光传输技术，能够有效模拟复杂的光传输现象，包括全局光照效果。通过这种方法，研究者们能够在不同的光照条件和未见姿势下，准确地重现全身头像的外观，并实现更自然的动画效果。
本方法的核心在于三个主要部分：几何建模、外观建模和阴影学习。首先，几何部分使用3D高斯点云表示全身头像，包括每个高斯体的位移、旋转、缩放和不透明度等属性。其次，在外观建模中，通过引入学习的区域谐波（ZonalHarmonics）来表示局部散射光传输，克服了传统球谐函数在全身关节运动中的局限性。此外，使用延迟着色技术来处理高光传输，确保在渲染过程中能够捕捉到高质量的高光效果。最后，阴影学习部分通过一个专门的网络来预测由于身体部位遮挡产生的非局部阴影，从而提高了头像在复杂光照环境下的真实感和一致性。
实验部分采用多相机光源捕捉技术进行数据采集，利用多种照明条件下的样本训练模型。研究者们通过对比不同方法的重光照和动画效果，验证了所提出方法在真实感和高效性上的优势。实验结果显示，所提出的模型在处理全身头像的局部和非局部光传输方面表现优异，能够在未见姿势和光照条件下保持较高的外观一致性和细节还原能力。
在我们的方法中，首先，我们用一种叫做3D高斯点云的技术来表示全身头像的形状。这就像用很多小点来构建一个复杂的形状，每个点都有自己的位置和特性。接下来，我们通过学习不同方向的光如何在这些点上散射来模拟头像的外观。这就像我们学习如何在不同的光照下拍摄照片，以便能在任何情况下都能看起来好看。最后，我们还设计了一个网络来预测由于身体各部分遮挡而产生的阴影，这样可以让头像在光线变化时看起来更真实。总的来说，我们的方法让全身头像在不同光照和姿势下都能保持自然和真实的效果。
2025-01-23｜TUMunich,MCML,UBonn｜ICLR2025
http ://arxiv.org/abs/2501.14038v1https ://github.com/Sangluisme/Implicit-surf-Deformation
本研究提出了一种新颖的无监督方法，旨在同时预测时间变化的神经隐式表面及其在一对点云之间的变形。我们的方法通过显式速度场来建模点的运动，并利用改进的水平集方程直接对时间变化的隐式场进行变形。这种方法确保了变形过程的物理合理性，采用了光滑性和无散度约束，从而恢复出自然且符合物理规律的中间形状。与现有的变形方法相比，我们的方法在质量和效率上显著优越，能够处理刚性和非刚性变形，且无需对中间形状进行监督。
我们的目标是从两个3D点云(P_0)和(P_1)中重建一个时间变化的隐式表示，并生成自然且物理合理的中间表面。方法主要包括以下几个步骤：
时间变化隐式场：定义一个时间变化的隐式场(f)，其输入为点的位置和时间，表示在时间(t)的表面形状为隐式函数的零水平集。
速度场建模：采用流体力学中的拉格朗日视角，通过显式速度场(V)来跟踪点的运动，确保物理合理性。速度场需要满足光滑性和无散度约束，以保证变形过程的自然性。
隐式场直接变形：通过改进的水平集方程直接对隐式场进行变形，避免显式网格渲染。这种方法使得每个组件都可微分，从而实现端到端的训练。
我们在多个数据集上验证了该方法的有效性，包括Faust、SMAL和SHREC16等。实验结果表明，与其他方法相比，我们的方法在处理不同类型的变形时，能够生成更高质量的中间形状。我们使用Chamfer距离和Hausdorff距离等指标进行定量评估，结果显示我们的算法在中间形状的重建上保持了较低的误差。此外，我们还测试了算法在处理稀疏和不完整输入点云时的鲁棒性，结果表明该方法能够有效恢复目标形状及其间接形状。
我们的方法可以理解为一种“时光机”，能够在两个不同形状之间生成平滑的过渡。这一过程就像在动画中让角色从一种姿势变换到另一种姿势。首先，我们从两个点云（可以想象成形状的点的集合）开始，定义一个“时间变化的形状”。接着，我们通过计算每个点的移动速度，来追踪这些点在时间变化中的轨迹。为了确保这些变形看起来合理，我们设定了一些规则，比如速度不能太快，形状的体积也不能随意改变。最后，我们通过一种特殊的数学方法，直接调整形状的表示，而不需要先绘制出形状的网格。这样，我们就可以在不需要额外数据的情况下，生成自然的中间形状，确保变形过程的流畅和真实。
2025-01-24｜SamsungR&DInstituteUK
http ://arxiv.org/abs/2501.14534v1
在3D重建领域，GaussianSplatting(GS)由于其快速的训练和推理速度以及高质量重建的特点而受到广泛关注。然而，GS方法通常需要处理数百万个高维高斯原始数据，这对计算资源有限的设备（如智能手机）造成了挑战。为了解决这一问题，本文提出了一种名为Trick-GS的方法，它通过系统性地结合多种策略，显著提高了GS的效率。Trick-GS在多个基准数据集上表现出色，实现了训练速度提高2倍、磁盘空间减少40倍以及渲染速度提升2倍，同时保持了与传统GS方法相当的准确性。这一方法的灵活性使得用户可以根据特定应用需求调整不同方面的优先级，如收敛速度和渲染质量。
Trick-GS的设计基于对现有高效GS方法的深入分析，结合了以下几种关键策略：
渐进式训练：通过逐步增加图像分辨率、噪声和高斯原件的规模，使得模型能够在初期学习到更粗略的场景表示，随后逐渐细化。
重要性修剪和掩蔽：通过学习每个高斯原件的显著性，动态地修剪对最终重建影响较小的高斯原件和球面谐波（SH）带，这样可以减少不必要的计算和存储开销。
加速GS训练框架：将SH的0阶和高阶带分开处理，优化计算过程，减少训练时间和资源消耗。此外，通过引入高斯模糊和渐进式分辨率策略，进一步提升训练的效率和效果。
这些策略的结合，使得Trick-GS在资源受限的环境中，能够以更快的速度和更小的存储需求完成高质量的3D重建。
在实验部分，Trick-GS在多个数据集上进行了严格的评估，包括Mip-NeRF360和Tanks&Temples。实验结果表明，Trick-GS在PSNR、SSIM和LPIPS等指标上均表现优异，且在存储和训练时间方面显著优于传统GS方法。具体而言，Trick-GS的模型大小降至仅10MB，并在大多数场景中保持约10分钟的训练时间。为了验证不同策略的有效性，进行了消融实验，结果显示，重要性修剪和渐进式训练策略对模型的存储和训练效率提升贡献最大。总体来看，Trick-GS在准确性与效率之间取得了良好的平衡，特别适合在设备资源有限的情况下进行3D重建。
在Trick-GS方法中，我们采用了一些聪明的技巧来提高3D重建的效率。首先，我们在训练过程中从简单的图像开始，逐渐增加图像的细节，这样可以让模型更好地学习场景的基本结构。其次，我们给每个高斯原件打分，看看哪些对最终效果影响不大，然后把这些不重要的部分去掉，这样可以节省存储空间和计算资源。最后，我们还优化了计算过程，使得每次更新的次数减少，这样可以更快地完成训练。通过这些方法，我们不仅让模型变得更小、更快，而且在保持准确度的同时，适应了不同设备的需求。

在计算机视觉领域，从单一图像合成新视角图像（NovelViewSynthesis,NVS）是一项具有挑战性的任务。传统的方法通常需要复杂的3D重建，导致计算成本高昂。本研究提出了CheapNVS，这是一种高效的端到端解决方案，专注于窄基线的单视图NVS。CheapNVS通过多个编码器和解码器的创新设计，能够在不进行全面3D重建的情况下，快速生成新视角图像。该方法利用摄像机姿态的嵌入信息，结合轻量级的学习模块，显著提升了合成效果和处理速度。与现有技术相比，CheapNVS在OpenImages数据集上的表现超越了最先进的方法，同时在速度和内存消耗上也表现出色。
CheapNVS的核心在于其高效的图像变换模块，能够快速近似3D图像扭曲。其方法步骤如下：
输入处理：接收输入图像及其深度图，并将其映射到共享的潜在空间。
编码器设计：采用共享的RGBD编码器和外部编码器，分别处理图像和摄像机姿态信息。
并行解码：通过流解码器和掩码解码器并行执行图像扭曲和遮挡区域填充，提升处理速度。
遮挡处理：在合成新视角图像时，使用生成的掩码来指导图像的填充过程，确保图像的完整性。
多阶段训练：采用分阶段训练策略，逐步激活不同解码器，以优化整体性能。
在实验部分，CheapNVS与当前的最先进方法AdaMPI进行了比较。通过在OpenImages和COCO数据集上进行训练与评估，CheapNVS在多个指标上表现出色，尤其在图像扭曲和填充质量上。定量结果表明，CheapNVS的运行速度是AdaMPI的十倍，同时内存消耗降低了6%。此外，CheapNVS的模型在移动设备上也能实时运行，达到每秒30帧以上的性能。视觉结果展示了CheapNVS在处理遮挡和边界伪影方面的优势，证明了其在实际应用中的有效性。
在CheapNVS的工作原理中，首先，我们需要输入一张图像和对应的深度信息。接着，系统通过两个不同的“编码器”来提取这些信息，一个处理图像，另一个处理摄像机的位置。然后，CheapNVS会同时进行两个主要操作：一是将图像进行扭曲，以模拟从不同角度观察的效果；二是填补图像中被遮挡的部分。这个过程是并行进行的，意味着它们可以同时完成，而不是一个接一个地进行，这样可以大大加快处理速度。最后，通过分阶段的训练，CheapNVS能够逐步提高其性能，使得生成的新图像更加真实和完整。这种方法不仅速度快，而且在手机等设备上也能顺利运行，非常适合实际应用。

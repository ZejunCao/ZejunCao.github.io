286-A1｜图像风格与内容迁移，产品广告图像设计生成，为特定任务找到合适的生成模型，多主体fMRI图像重建的隐私保护与个体差异

主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）发布日期｜2025年1月27日｜周一
2025-01-24｜MaxPlanckSociety,SapienzaUni.Rome,UZH,UCambridge｜⭐️🟡
http ://arxiv.org/abs/2501.14524v1
本研究提出了一种新方法，称为SkipInject，旨在利用U-Net中的跳跃连接来实现图像的风格与内容转移。尽管近年来扩散模型在图像生成领域取得了显著进展，但其内部潜在表示仍然不够清晰，限制了对生成结果的有效控制。SkipInject方法通过研究U-Net的跳跃连接，发现特定的连接在内容和风格的传递中起着关键作用。该方法不仅能够实现高质量的文本引导图像编辑，还能在风格转移任务中表现出色，展示了在多个基准测试上优于或持平于现有最先进技术的能力。
SkipInject方法的核心在于有效利用U-Net中的跳跃连接，具体步骤如下：
跳跃连接分析：研究U-Net的不同跳跃连接，识别出哪些连接对图像生成最为重要。特别是发现第4和第5组跳跃连接在传递空间信息方面表现突出。
图像生成过程：通过在生成的图像中注入这些跳跃连接的表示，控制内容和风格的转移，进而实现精准的图像编辑。
注入时机和强度调整：通过调整注入的时间步和强度，控制生成图像的背景和前景内容的保留或替换。采用分类器自由引导等技术来增强对编辑效果的控制，使得生成图像既符合文本提示，又保持结构完整性。
本研究在多个数据集上对SkipInject方法进行了评估，包括Wild-TI2I和ImageNet-R-TI2I。实验结果表明，该方法在文本引导的图像编辑和风格转移任务中均表现出色。通过定量指标（如CLIP相似度和DINO-ViT自相似性）和定性分析，SkipInject在文本对齐和结构保持方面均优于其他现有方法，显示出其在实际应用中的有效性和灵活性。
在我们的研究中，我们开发了一种叫做SkipInject的方法，旨在通过U-Net模型中的“跳跃连接”来改变图像的风格和内容。简单来说，U-Net就像一个复杂的图像处理机器，而跳跃连接则是它内部不同层之间的信息传递通道。我们发现，某些特定的连接在保持图像的整体结构和风格方面特别重要。通过将这些连接的特征注入到生成的图像中，我们可以灵活地调整图像的内容和风格。比如说，如果我们想把一只鸟的图像变成另一种鸟的样子，SkipInject可以帮助我们保留背景，同时只改变鸟的外观。此外，我们还可以控制这种变化的强度和时机，确保生成的图像既符合我们的要求，又看起来自然。这种方法在多个测试中表现出色，能够生成高质量的图像，使得艺术创作和图像编辑变得更加简单和高效。
2025-01-24｜AlibabaGroup,BUPT｜⭐️🟡
http ://arxiv.org/abs/2501.14316v1
在电子商务平台上，广告图像的设计至关重要，直接影响消费者的购买决策。为了提高广告图像的生成效率，研究团队提出了一种名为产品中心广告图像设计（PAID）的新框架。该框架旨在通过自动化生成广告图像，减少人工成本，并提升图像的视觉吸引力。PAID框架的输入包括产品前景图像、所需的标语和目标尺寸，通过四个阶段的处理，最终生成和谐且美观的广告图像。这四个阶段分别是：提示生成、布局生成、背景图像生成和图形渲染。通过与传统方法的比较，PAID展示了更高的视觉品质和更好的设计灵活性。
PAID框架的实现分为四个主要阶段：
提示生成：使用视觉语言模型（VLM）生成与产品和背景相匹配的提示。该模型通过分析产品的特征和市场需求，自动生成描述性文本。
布局生成：在此阶段，模型根据生成的提示和产品前景，合理安排广告图像中各个元素的位置，包括文本、图标和背景元素，以确保整体的美观性和可读性。
背景图像生成：利用控制生成的背景图像，确保其与产品前景图像之间的和谐。该过程通过深度学习模型进行训练，生成符合视觉美学的背景。
图形渲染：最后一步是将生成的元素整合到一起，形成最终的广告图像。此阶段使用特定的渲染技术，确保图像的各个部分协调一致。
为了验证PAID框架的有效性，研究团队进行了多项实验。首先，构建了两个高质量的数据集，分别用于广告图像设计（PITA）和背景生成（PIL）。通过对比PAID与其他现有方法，实验结果显示，PAID在视觉吸引力和设计灵活性方面均表现优越。具体来说，PAID的生成图像在美学评分和布局合理性上均优于对比方法，且在多个评价指标上均取得了更好的结果。此外，实验还表明，PAID在处理不同产品类型时，能够有效调整布局和元素位置，从而提高了广告图像的整体质量。
PAID框架的工作原理可以简单理解为一个四步走的过程。首先，它会根据你提供的产品图片和想要的广告文字，自动生成一段描述文本，这叫做“提示生成”。接下来，系统会决定这些文字和图片在广告中的具体位置，这就是“布局生成”。然后，它会生成一个背景图像，这个背景会与产品图片相匹配，确保整体看起来很和谐。这一步骤叫做“背景图像生成”。最后，所有的元素会被整合在一起，形成一个完整的广告图像，这叫做“图形渲染”。通过这个过程，PAID能够快速且高效地创建出吸引人的广告图像，减少了人工设计的时间和成本。
2025-01-23｜NJU｜⭐️🟡
http ://arxiv.org/abs/2501.13991v1
在近年来，生成模型取得了显著的进展，尤其是在图像生成领域。随着可用模型数量的激增，如何高效地为特定任务找到合适的生成模型变得尤为重要。传统的模型搜索方法，如基于文本匹配和下载量排名，往往无法满足用户的具体需求，特别是在用户目的与模型提供的示例图像不匹配的情况下。为了解决这一问题，本文提出了一种新的方法——条件生成模型识别（CGI），旨在通过用户提供的示例图像来有效识别最适合的模型，而不是依赖用户手动审查大量模型。
CGI方法的核心在于三个关键模块的结合：自动规格分配、需求生成和任务特定匹配。具体来说：
自动规格分配：该模块通过预定义的提示集或开发者提供的提示集，自动生成每个条件生成模型的功能规格。这一过程确保了模型的功能能够被清晰地描述。
需求生成：通过对用户上传的示例图像进行编码，生成用户任务的需求表示。这一过程将用户的需求与模型规格投影到统一的匹配空间中。
任务特定匹配：利用需求生成的结果，调整模型规格以更准确地匹配用户需求，最终识别出最适合的模型。该步骤通过计算相似性得分来实现，确保用户能够快速找到满足其具体需求的模型。
为验证所提CGI方法的有效性，研究团队构建了一个包含65个条件生成模型和9100个识别任务的基准测试。通过与基线方法和现有技术进行对比，实验结果显示，CGI方法在准确性和生成质量上均显著优于传统模型搜索技术。具体而言，CGI方法在识别任务中达到了高达92%的准确率，同时生成的图像质量也得到了提升。人类评估和GPT-4的评估结果进一步确认了CGI方法在识别和生成方面的优势。
在这项研究中，科学家们提出了一种新方法，帮助用户更容易找到适合他们需求的图像生成模型。想象一下，你需要生成一张特定风格的图像，但市场上有成千上万的模型可供选择，手动寻找非常耗时。这时，CGI方法就像一个智能助手，它会根据你提供的图像示例，快速分析并推荐出最合适的模型。这个过程分为三个步骤：首先，它会自动生成每个模型的功能说明；接着，它会理解你想要的图像特征；最后，它会把这些信息结合起来，找到最匹配的模型。这种方法不仅提高了寻找模型的效率，还能确保生成的图像更符合用户的期望。
2025-01-24｜LZU,NTU,ZJU｜AAAI2025｜⭐️🟡
http ://arxiv.org/abs/2501.14309v1https ://github.com/kunzhan/BrainGuard
BRAINGUARD是一个创新的框架，旨在解决多主体fMRI图像重建中的隐私保护和个体差异问题。传统的fMRI图像重建方法通常为每个个体训练独立模型，忽视了跨个体间的共同特征。尽管最近的研究开始探索多主体方法，但仍面临数据隐私和个体变异管理的重大挑战。BRAINGUARD通过引入一个隐私保护的协同训练框架，旨在利用多主体fMRI数据提升图像重建的准确性，同时确保个体隐私的安全。该框架采用全球-局部架构，个体模型在各自的本地数据上训练，并与一个共享的全球模型协同工作，从而有效捕捉和利用跨个体的模式，避免了数据聚合的隐私风险。
BRAINGUARD的核心方法包括以下几个步骤：
个体训练：每个主体使用其特定的fMRI数据独立训练模型，确保敏感数据保持本地化，避免集中存储。
参数聚合：在每轮训练后，个体模型将更新的参数发送至全球模型进行汇总。这一过程通过指数移动平均（EMA）来平滑参数变化，以增强模型的稳定性。
模型同步：全球模型更新后，个体模型通过混合同步策略整合来自全球模型的知识。这种策略确保在保留个体特征的同时，提升模型的泛化能力。
此外，BRAINGUARD通过动态融合学习模块（DFL）对高级层进行自适应调优，以便更好地反映个体特征和跨主体的共同模式，从而提升整体性能。
在实验部分，BRAINGUARD使用自然场景数据集（NSD）进行评估，数据集包括来自八名参与者的fMRI数据，涉及73,000张RGB图像。实验结果表明，BRAINGUARD在图像重建的低级和高级指标上均优于现有的最先进方法，特别是在像素相关性（PixCorr）和结构相似性（SSIM）等低级指标上，BRAINGUARD的表现显著提升。此外，BRAINGUARD在高层次特征的评估中也展现了卓越的性能，证明了其在图像重建领域的有效性和适用性。
BRAINGUARD的工作原理可以简单理解为一个团队合作的过程。想象一下，多个科学家（每个科学家代表一个参与者）在各自的实验室中进行研究，每个人都有自己的实验数据（fMRI数据）。每个科学家独立地分析自己的数据，得出一些结论，但他们也希望能够从其他科学家的研究中学习。BRAINGUARD允许这些科学家在不共享具体数据的情况下，汇总他们的发现。
具体来说，首先每个科学家使用自己的数据训练一个模型。然后，他们将这些模型的更新信息发送给一个中央模型，这个中央模型会整合所有人的信息，找出大家的共同点。最后，每个科学家再根据这个中央模型的知识来调整自己的研究。这种方式能够确保每个科学家的数据隐私，同时也能提高他们的研究成果，帮助他们更准确地重建他们所看到的图像。这样一来，BRAINGUARD不仅保护了个人隐私，还提升了整个团队的研究效率。

2025-01-24｜NTUS-Lab,SenseTimeResearch｜⭐️🟡
http ://arxiv.org/abs/2501.14677v1https ://pq-yang.github.io/projects/MatAnyone
MatAnyone是一个针对目标指定的人类视频抠图的强大框架，旨在解决复杂和模糊背景下的抠图问题。与传统的辅助视频抠图方法不同，MatAnyone不依赖额外的标注，而是通过输入帧的内在特征进行处理。该方法通过区域自适应记忆融合机制，有效整合前一帧的记忆信息，从而确保在视频中核心区域的语义稳定性，同时保持物体边界的细致细节。此外，MatAnyone还引入了一个新的训练策略，利用大规模的分割数据来增强模型的鲁棒性。通过这些创新，MatAnyone在各种真实场景下表现出色，超越了现有的抠图方法，提供了更高质量的抠图结果。
MatAnyone的核心方法包括以下几个方面：
一致性记忆传播机制：该机制通过对每一帧的像素进行记忆读取，结合前一帧的信息，确保在连续帧之间的语义一致性。对于每一帧，模型会根据与上一帧的差异来调整当前帧的alpha值，从而在大变化区域依赖当前帧的信息，而在小变化区域则保留先前的记忆。
区域自适应记忆融合：通过引入一个预测模块，动态估计每个查询像素的变化概率，使得在融合记忆时能够更加准确地处理核心区域和边界区域的不同需求。这一机制提升了边界区域的稳定性和细节保留。
新训练策略：MatAnyone通过直接利用真实的分割数据对核心区域进行监督，同时对边界区域采用改进的DDC损失函数，确保模型能够在缺乏真实alpha标签的情况下进行有效训练。该策略显著提升了模型在视频抠图任务中的表现。
在实验中，MatAnyone与多种最先进的方法进行了比较，包括辅助视频抠图和掩码引导的方法。使用标准的评估指标（如MAD、MSE、Grad和dtSSD），MatAnyone在多个数据集上均表现出色，尤其是在细节提取和语义准确性方面。在合成基准测试中，MatAnyone在所有评估指标上均取得了最佳结果，显示出其卓越的空间精度和时间稳定性。此外，在真实场景的基准测试中，MatAnyone的表现同样优于其他方法，尤其是在复杂背景下的抠图稳定性和细致性方面，展现了其在实际应用中的有效性。
在MatAnyone的工作流程中，首先，我们需要一个目标对象的分割图，这样系统就能知道我们想要抠出哪个人或物体。接下来，系统会逐帧地处理视频，每一帧都会借用前一帧的信息，确保抠图的连贯性。想象一下，如果你在看一部电影，屏幕上有一个人走动，MatAnyone会不断更新这个人的轮廓，确保即使背景复杂，也能准确地把他抠出来。为了做到这一点，系统会根据每个像素的变化情况，灵活选择使用当前帧的信息或保留之前的记忆，这样就能在抠图时保持细节的清晰度，尤其是在头发等边界模糊的地方。通过这种方式，MatAnyone不仅能在各种视频中表现出色，还能在没有额外标注的情况下，依然保持高质量的抠图效果。
2025-01-24｜KAIST,RutgersU｜⭐️🟡
http ://arxiv.org/abs/2501.14174v1
在这篇论文中，作者提出了一种名为Dreamweaver的新型神经网络架构，旨在从原始视频中学习层次化和组合性的世界表示。人类能够将对世界的感知分解为对象及其属性（如颜色、形状和运动模式），从而构建出对未来的想象。然而，将这种能力复制到人工智能系统中，尤其是在没有依赖文本、掩码或边界框等辅助数据的情况下，仍然是一个挑战。Dreamweaver通过一种新颖的递归块槽单元（RBSU）来分解视频内容，并通过多未来帧预测目标来捕捉动态和静态概念的解耦表示。实验结果显示，Dreamweaver在多个数据集的世界建模评估中超越了当前的最先进基准，并能够通过重新组合不同对象的属性生成新的视频。
Dreamweaver的核心是递归块槽单元（RBSU），它负责处理输入的视频序列并生成可组合的表示。具体方法包括：
输入编码：RBSU接收过去T帧的视频图像，利用CNN提取特征，并将其转化为特征映射。
状态更新：RBSU通过独立更新槽状态来维护一个解耦的状态表示，槽代表对象而块则表示对象内部的可重用概念（如颜色和形状）。
注意力机制：RBSU利用底层注意力机制从输入特征中提取信息，生成底层读出向量，并通过独立的递归模块对每个块进行更新。
未来帧预测：通过一个自回归图像变换器，Dreamweaver能够预测未来的帧，利用预测目标来促使模型捕捉动态概念。
在实验中，作者评估了Dreamweaver在无监督学习模块化概念、生成新视频以及支持超出分布（OOD）推理任务的能力。实验使用了多种数据集，包括不同复杂度的动态和视觉数据集。通过与现有的无监督表示学习基线（如RSSM、STEVE和SysBinder）进行比较，Dreamweaver在解耦性、完整性和信息量等指标上表现出色。此外，实验还展示了Dreamweaver在生成新视频和进行场景预测与推理任务中的强大能力，尤其是在处理动态概念时的出色表现。
Dreamweaver的工作原理可以简单理解为一个“视频理解和创造”的系统。首先，它会接收一段视频，并通过一种智能的方式分析每一帧，提取出重要的元素，比如颜色、形状和运动方向。接着，系统会将这些元素整理成一个“概念库”，就像把不同的乐器放在一起，随时可以组合成新的乐曲。然后，Dreamweaver不仅能理解这些元素，还能想象未来的场景，比如预测接下来会出现什么样的画面。通过这种方法，Dreamweaver能够生成新的视频，展示出它对世界的理解和创造能力，甚至在没有任何文字描述的情况下，也能进行组合和创新。
2025-01-24｜NTU,A*STAR,ZJU,THU｜ICLR2025｜⭐️
http ://arxiv.org/abs/2501.14195v1https ://github.com/hurunyi/VideoShield
随着人工智能生成内容（AIGC）的迅猛发展，视频生成模型（如文本到视频和图像到视频模型）在各个领域的应用日益广泛。然而，这些生成的视频内容面临着未经授权的使用和篡改的风险，因此需要有效的内容控制手段。传统的水印嵌入方法多集中于图像，而对视频的关注相对较少，且这些方法往往在后处理阶段进行水印嵌入，可能会影响视频质量。为此，本文提出了一种新的水印框架——VIDEOSHIELD，专为流行的扩散基础视频生成模型设计。该框架在视频生成过程中直接嵌入水印，避免了额外训练的需求，并引入了一种篡改定位特征，能够同时检测时间和空间上的变化。通过广泛的实验验证，VIDEOSHIELD能够有效提取水印并检测篡改，而不会损害视频质量。
VIDEOSHIELD的核心方法分为两个主要阶段：水印嵌入与提取，以及篡改定位。具体步骤如下：
水印嵌入：
将水印位映射到高斯噪声，并在生成视频的过程中嵌入水印。
通过加密算法生成模板位，这些模板位与每个像素一一对应，确保其在篡改时易于识别。
使用去噪扩散模型（DDIM）对生成的视频进行去噪，以便在后续步骤中提取水印。
水印提取：
在视频可能被篡改的情况下，通过将生成的视频压缩成潜在表示，再利用DDIM反演来恢复嵌入的水印位。
通过比较提取的位与模板位，判断哪些部分被篡改。
篡改定位：
采用时序模块和空间模块，分别处理时间和空间的篡改检测。
通过层次空间-时间细化（HSTR）模块，提升篡改定位的准确性和细粒度。
在实验部分，研究者选择了两个流行的开源模型进行测试：文本到视频模型ModelScope和图像到视频模型Stable-Video-Diffusion。通过生成不同类型的视频，评估VIDEOSHIELD的水印提取和篡改定位性能。实验结果表明，VIDEOSHIELD在水印提取的比特准确率和视频质量方面均优于多种基线方法。此外，在不同的篡改类型（如帧交换、插入和删除）中，VIDEOSHIELD表现出卓越的篡改定位能力。通过对比，VIDEOSHIELD在视频质量和提取准确率上均保持了高水平，验证了其在实际应用中的有效性和鲁棒性。
VIDEOSHIELD的工作原理可以简单理解为一个保护视频内容的“防盗门”。首先，它在视频生成的过程中就把一个隐秘的标记（水印）嵌入到视频中，而不是在生成后再加上去，这样可以保证视频的质量不会受到影响。接着，当视频被篡改时，VIDEOSHIELD能够通过比较视频中的标记和原始标记来发现这些改动。它不仅能检测到视频的某些部分是否被修改，还能判断这些修改发生在时间上还是空间上，确保我们能快速找到问题所在。这样的技术在保护视频版权和防止不当使用方面非常有用。
2025-01-24｜AiShiWeiLai,SCUT,BIT,BUPT｜⭐️
http ://arxiv.org/abs/2501.14646v1https ://syncanimation.github.io/
SyncAnimation是一种新颖的实时音频驱动人类姿态和说话头像动画生成框架，旨在解决现有方法在实时性能和视觉质量上的不足。该方法结合了音频驱动的生成技术，能够从单一图像和音频输入中生成高质量的头像动画，包括面部表情和上半身的动态变化。与传统方法相比，SyncAnimation采用了基于神经辐射场（NeRF）的生成模型，能够实现音频与动作的高精度同步，克服了现有技术在音频与面部表情、头部姿态及上半身动作之间的一致性问题。通过集成音频姿势同步器和音频情感同步器，该框架不仅提高了生成的稳定性和精确性，还确保了头像动画在无声状态下的自然表现。
SyncAnimation的方法主要由三个核心模块组成：
**音频姿势同步器(AudioPoseSyncer)**：该模块负责将音频信号映射到动态头部姿势，确保姿势与音频的稳定一致性。通过引入条件向量，模型能够更好地控制姿势的稳定性和多样性。
**音频情感同步器(AudioEmotionSyncer)**：该模块专注于根据音频生成逼真的面部表情。通过分析音频特征并结合面部动画数据，模型能够生成与音频情感相匹配的自然表情，确保面部动画的连贯性。
**高同步人类渲染器(High-SynchronizationHumanRenderer)**：负责将生成的头部姿势和上半身动作无缝整合，确保最终渲染的头像在视觉上保持一致性和自然性。该模块利用神经辐射场技术，实现高质量的图像输出，确保生成的动画在动态变化时仍然保持高保真度。
在实验部分，SyncAnimation的性能通过与多种现有技术进行比较来评估。实验结果表明，该框架在多个指标上均优于当前的最先进方法，包括图像质量、音频同步和面部表情的多样性。具体来说，SyncAnimation在峰值信噪比（PSNR）、结构相似性指数（SSIM）和音频与唇部同步的精度等方面表现突出。此外，SyncAnimation还展示了其在实时应用中的适用性，能够以每秒41帧的速度生成动画，这在音频驱动的头像生成领域中具有里程碑式的意义。通过对不同音频输入的测试，SyncAnimation显示出其强大的适应性和高效性，能够处理各种复杂的音频信号并生成自然流畅的动画。
SyncAnimation的方法可以简单理解为一个“智能动画工厂”。首先，它通过音频信号来“听”出说话者的情感和语气，这就像我们在与人交谈时，能根据对方的声音判断他们的情绪。接着，这个系统会根据声音生成相应的头部动作和面部表情，比如微笑或皱眉。这一过程就像给动画角色配音一样，不仅需要声音，还要让角色的表情和动作与声音相匹配。最后，SyncAnimation还会把头部的动作和上半身的动作结合在一起，确保整个动画看起来既自然又生动。通过这三步，SyncAnimation能够快速且高质量地生成一个栩栩如生的说话头像，无论是用于视频会议、直播还是其他实时应用。

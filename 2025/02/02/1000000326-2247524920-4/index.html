2025-01-26｜KAIST｜ICLR2025｜⭐️⭐️
http ://arxiv.org/abs/2501.15445v1https ://stochsync.github.io/
在这篇论文中，研究者们提出了一种名为StochSync的新方法，用于在任意空间中生成图像，尤其是360°全景图和3D网格纹理。该方法利用了预训练的图像扩散模型，以实现零-shot生成，消除了对新数据收集和单独训练生成模型的需求。研究表明，StochSync结合了DiffusionSynchronization（DS）和ScoreDistillationSampling（SDS）的优点，能够在条件较弱的情况下仍然生成高质量的图像。通过对这两种方法的分析，研究者们发现了它们之间的内在联系和差异，并提出StochSync能够在360°全景图生成方面超越传统的微调方法，同时在3D网格纹理生成中表现出与现有方法相当的效果。
StochSync方法的核心在于结合了DiffusionSynchronization和ScoreDistillationSampling的优势，具体包括以下几个方面：
最大随机性：在同步过程中引入最大随机性，优化了样本在不同实例空间之间的连贯性。这一策略通过调整噪声预测的方式，帮助消除样本间的接缝，提高了图像的整体流畅性。
多步去噪计算：将传统的单步去噪预测替换为多步去噪过程，以提高生成样本的真实感和一致性。这种方法允许在每个时间步骤上进行更精确的样本预测，进而提升生成图像的细节。
非重叠视图采样：在每个步骤中采用非重叠的视图采样策略，确保样本在时间上的同步，同时避免了由于视图重叠导致的真实感下降。这种方法通过交替采样不同的视图，提高了生成图像的质量。
在实验部分，研究者们对StochSync在360°全景图生成和3D网格纹理生成两个应用场景中的表现进行了评估。在360°全景图生成中，StochSync展示了优于现有零-shot方法和微调方法的性能，特别是在FID、IS和GIQA等指标上表现突出。对于3D网格纹理生成，尽管StochSync主要设计为无条件生成，仍然在有深度图作为条件输入时展现出与现有DS方法相当的效果。此外，用户研究结果显示，StochSync在避免几何失真方面表现更佳，用户偏好度明显高于其他方法。实验结果表明，StochSync在生成多种视觉内容方面具有广泛的应用潜力。
StochSync是一种新颖的图像生成方法，能够在不同的空间中，比如球面或3D模型表面，生成高质量的图像。它结合了两种已有的方法，分别是DiffusionSynchronization和ScoreDistillationSampling。简单来说，StochSync通过在生成图像时引入更多的随机性，使得不同视角的图像之间更加协调，减少了图像之间的接缝。此外，它还采用多步去噪的方式来提升图像的真实感，而不是一次性生成。最后，StochSync在每一步生成图像时，使用不同的视角，而不是重复使用同一个视角，这样可以避免图像失真。通过这些创新，StochSync在生成360°全景图和3D纹理方面的表现都非常出色，证明了其在视觉内容生成领域的广泛适用性。
2025-01-27｜KAUST,MiamiU｜⭐️⭐️
http ://arxiv.org/abs/2501.15981v1
在计算机图形学中，为3D模型分配逼真的材料仍然是一个重要的挑战。为了解决这一问题，研究者们提出了MatCLIP，一种新颖的方法，旨在通过提取对形状和光照不敏感的物理基础渲染（PBR）材料描述符，来为3D对象分配合适的纹理。这种方法能够基于图像（如潜在扩散模型（LDMs）的输出或照片）进行材料匹配，克服了传统方法中由于形状、光照和视角变化导致的材料分配不一致的问题。MatCLIP通过扩展Alpha-CLIP模型，生成能够在多种几何形状和光照条件下保持一致性的材料描述符，从而实现更高的分配准确性，达到了76.6%的Top-1分类精度，显著优于现有的最先进方法。
MatCLIP的方法主要包括以下几个步骤：
材料数据库构建：使用公开的高分辨率PBR材料数据库（如MatSynth），生成多种材料的样本。在多种环境光照条件下渲染这些材料，以确保材料描述符对形状和光照的鲁棒性。
材料表示：每种材料通过多个渲染生成的图像来表示，形成一个包含光照和几何变化的嵌入向量。这些向量能够捕捉材料在不同条件下的外观变化。
模型架构：MatCLIP由两个主要编码器组成：材料编码器和部分编码器。材料编码器处理PBR材料的渲染图像，而部分编码器处理输入图像的掩码区域。通过对比损失函数来优化两个编码器的输出，使得材料描述符与3D模型的特定部分进行匹配。
纹理采样：利用LDM生成合成目标图像，结合深度和边缘信息，确保生成的纹理与3D模型的几何形状相符。
在实验中，MatCLIP的性能通过多个维度进行评估，包括材料分类的准确性、对新输入的泛化能力以及数据集多样性的影响。实验结果显示，MatCLIP在材料分类上取得了76.69%的Top-1准确率，优于其他方法，如PhotoShape和MatAtlas。通过对比不同形状和环境地图的多样性，发现使用全数据集的模型表现最佳，而使用较少的形状和环境地图会显著降低准确性。此外，用户研究表明，参与者更倾向于选择使用MatCLIP生成的材料分配，显示出其在实际应用中的有效性。
MatCLIP的工作原理可以简单理解为一个“智能材料匹配器”。首先，它会从一个丰富的材料库中选取不同的材料，这个库包含了各种各样的纹理，比如布料、皮革等。在选择材料时，MatCLIP不仅考虑材料本身的特性，还会考虑到3D模型的形状和光照条件。然后，模型会通过两个“识别器”来分析材料和模型的细节，确保它们能够相互匹配。最后，MatCLIP会生成一个合成的图像，展示出选定材料在模型上的样子。通过这样的方式，MatCLIP能够为各种3D模型提供更真实和一致的材料分配，帮助设计师和开发者在创作过程中提升效率和效果。
2025-01-27｜CUHK(SZ),TencentXRVisionLabs,SJTU,ANU｜⭐️⭐️
http ://arxiv.org/abs/2501.16177v1https ://bag-3d.github.io/
本文提出了一种名为BAG（Body-Aligned3DWearableAssetGeneration）的新方法，旨在自动生成可穿戴的3D资产，如服装和配饰，以适应特定的人体模型。尽管近年来在3D多视图图像生成方面取得了显著进展，但如何利用这些技术自动生成与目标人体形状相匹配的可穿戴3D资产仍然是一个未被充分探索的领域。本研究通过控制3D生成过程，结合人体形状和姿势信息，成功实现了这一目标。BAG方法通过构建一个多视图图像扩散模型，生成与人体对齐的3D资产，并通过物理模拟解决资产与身体之间的穿透问题，最终实现了高质量的3D资产生成。
BAG方法的核心步骤如下：
一致性多视图图像生成：首先，构建一个单图像到一致性多视图图像的扩散模型，利用人体的SMPL模型生成四个视角的图像，以形成一个2x2的布局。这些图像作为输入，确保生成的多视图图像与目标人体形状一致。
相似性变换估计：通过多视图图像的监督，优化生成的3D资产的相似性变换（Sim(3)），以确保资产与人体的几何一致性。这一过程利用轮廓损失函数来最小化生成的轮廓与目标轮廓之间的差异。
处理资产与身体的穿透：在生成资产与人体对齐后，使用物理模拟解决可能存在的穿透问题。通过计算有符号距离场（SDF），识别并调整穿透的顶点，确保资产在与人体交互时不会发生不自然的穿透现象。
在实验部分，研究者评估了BAG方法的效果，采用多个指标来衡量生成形状的质量，包括Chamfer距离、法线一致性和平均点到表面欧几里得距离等。实验结果显示，BAG方法在形状多样性和生成质量方面显著优于现有的其他方法。通过与BCNet、ClothWild等最先进的方法进行比较，BAG在生成与输入图像一致的3D资产方面表现出色，展示了其在多种应用场景下的有效性和灵活性。
BAG方法的工作原理可以简单理解为：首先，我们从一张人穿着衣服的图片开始，利用一个智能系统把这张图片转换成四个不同角度的图像，就像我们用相机围绕着这个人拍照一样。接着，系统会分析这些图像，确保生成的3D衣物与人的身体形状完全匹配。为了避免衣物与身体之间出现穿透的情况，系统还会进行一些物理模拟，确保衣物自然地贴合在身体上。最终，BAG方法能够自动生成出高质量、符合人体形状的3D服装，适合在虚拟现实或视频游戏中使用。
2025-01-26｜SCUT,NTU,SenseTimeResearch,PengChengLab｜⭐️🟡
http ://arxiv.org/abs/2501.15616v1
在虚拟试衣技术中，IPVTON（基于图像的3D虚拟试衣框架）旨在通过输入一张人像图和一张服装图，重建出一个逼真的3D人类模型，使其能够自然地穿上所选的服装。传统的2D虚拟试衣技术虽然易于使用，但其局限于固定的视角，无法满足用户从多个角度观察服装的需求。而3D虚拟试衣技术虽然提供了多视角的优势，但过程复杂且劳动密集。IPVTON通过结合图像提示适配器与自定义扩散模型，优化了3D人类模型的几何和纹理表现，克服了这些传统方法的不足。该框架利用掩模引导的图像提示嵌入，确保模型在试衣区域的真实性，同时保留源身份的特征。
IPVTON的方法可以分为几个关键步骤：
3D混合人类表示初始化：使用SMPL-X模型作为基础，建立一个3D人类模型，确保其形状和姿态的合理性。
图像提示适配器（IP-Adapter）：通过IP-Adapter将目标服装图像的特征嵌入到扩散模型中，结合文本和图像提示，以便更好地捕捉服装的高级语义。
几何优化：在几何优化阶段，利用伪轮廓损失（PseudoSilhouetteLoss）来约束生成的3D模型的形状，确保其符合目标服装的轮廓。
纹理优化：在纹理优化过程中，使用SD损失（ScoreDistillationLoss）来确保生成的纹理与目标服装的图像特征一致，同时利用掩模引导的图像提示来聚焦在试衣区域，减少对非试衣区域的干扰。
在实验部分，研究者通过定量和定性分析评估了IPVTON的效果。定量实验使用CLIP评分来衡量生成结果在几何和纹理上的真实度，并与多个基线方法进行了比较。定性实验则展示了IPVTON生成的3D试衣结果的视觉质量，强调了其在捕捉服装细节和保持人像特征方面的优势。此外，研究者还进行了用户研究，邀请参与者对不同方法生成的结果进行偏好排序，结果表明IPVTON在用户体验上表现优越。
在IPVTON的工作流程中，首先我们使用一个叫做SMPL-X的模型来创建一个3D人类模型，这个模型可以根据输入的人像图像来调整身体的形状和姿势。接下来，我们引入一个图像提示适配器，它的作用是从服装的图像中提取特征，并将这些特征与文本描述结合在一起，以帮助模型更好地理解服装的样式和细节。当我们进行几何优化时，我们使用一种叫做伪轮廓损失的技术，确保生成的3D模型看起来像是穿上了目标服装。最后，在纹理优化阶段，我们会提取目标服装的颜色和纹理，并确保这些特征在生成的3D模型上得以准确体现。通过这些步骤，IPVTON能够生成出既真实又自然的3D试衣效果。

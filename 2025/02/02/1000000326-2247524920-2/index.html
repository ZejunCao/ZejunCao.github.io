287-A1.2｜肖像抠图，提示感知可控影子去除，草图查询图像分割；文生图安全性（概念擦除，性别偏见）

2025-01-27｜BNU,HUST｜⭐️🟡
http ://arxiv.org/abs/2501.16147v1
本文研究了一种高效的肖像抠图（portraitmatting）方法，旨在解决传统抠图技术在复杂环境下的局限性。传统的深度学习肖像抠图模型在简单场景中表现良好，但在处理细边缘和半透明区域时常常出现模糊和细节丢失的问题。此外，获得高质量的抠图数据集通常需要昂贵的手动标注或自动化方法，这些方法往往依赖于均匀背景，导致数据多样性不足。为了解决这些问题，本文提出了一种基于层扩散（LayerDiffusion）模型和连通性先验的肖像抠图数据集生成方法，构建了一个包含20,051个高质量肖像图像及其对应的透明度图（alphamatte）的数据集LD-Portrait-20K。实验结果表明，使用该数据集训练的模型在处理复杂边缘和背景时显著优于其他数据集训练的模型。
本文提出的肖像抠图数据集生成方法主要分为三个步骤：
输入提示生成：通过描述人类特征的提示，使用层扩散模型生成相应的RGB图像和透明度图。提示生成有两种方式：一种是通过系统性描述性特征（如性别、年龄、情绪等）自动生成，另一种是利用ChatGPT生成更丰富的提示。
快速筛选：对生成的图像和对应透明度图进行初步筛选，去除显著偏差的样本。通过对背景区域的连通性分析，保留那些半透明区域集中在人体边缘的图像。
连通性意识的透明度优化：在筛选后的图像中，使用连通性意识算法优化透明度图。这一过程通过广度优先搜索（BFS）和深度优先搜索（DFS）有效识别并保留边缘的半透明区域，同时修正其他区域的错误透明度值，从而提升整体图像质量。
在实验部分，本文将LD-Portrait-20K数据集与其他三种现有数据集进行了比较。使用ViTMatte-B和GFM模型对不同数据集进行训练，并在PhotoMatte85和PPM-100等验证集上进行评估。实验结果表明，使用LD-Portrait-20K训练的模型在多个评估指标上均优于其他数据集。特别是在处理复杂边缘和细节保留方面，LD-Portrait-20K展现出显著优势。此外，本文还进行了消融研究，探讨数据集容量对模型性能的影响，结果显示更大的数据集显著提升了模型的表现。
在这项研究中，研究人员想出了一个聪明的方法来生成高质量的肖像抠图数据集。他们首先通过描述人的外貌和情感等特点，生成了一些图像和透明度数据。接着，他们仔细筛选出那些边缘清晰且没有太多错误的图像，以确保后续处理的效果。最后，他们使用一种聪明的算法来优化透明度，确保图像中人的边缘和细节都能被准确保留。通过这种方法，他们成功创建了一个包含大量高质量肖像图像的数据集，这个数据集在处理复杂的背景和细节时表现得非常出色。
2025-01-25｜ZJU｜⭐️🟡
http ://arxiv.org/abs/2501.15043v1
本文提出了一种新颖的影子去除方法，称为“基于提示的可控影子去除”（PACSRNet），旨在解决传统影子去除技术面临的两个主要问题：一是无法精准控制影子的去除，二是对影子区域的精确标注依赖过重。PACSRNet允许用户通过简单的提示（如点、线或主体掩模）来指定需要去除影子的特定对象，从而实现灵活的影子去除。该方法不仅消除了对影子标注的需求，还提供了基于用户输入的可控性。PACSRNet由两个关键模块组成：提示感知模块和影子去除模块，前者生成特定对象的影子掩模，而后者利用这些信息恢复影子区域的内容。此外，研究团队还创建了一个新的数据集，专门用于提示驱动的可控影子去除，以支持该领域的进一步研究。
PACSRNet的核心在于其两个模块的设计。首先，提示感知模块负责根据用户提示生成影子掩模和指导特征。该模块利用空间-频率交互（SFI）块，通过小波变换提取影像的频率特征，并与空间特征进行交互，以增强影子的边界感知能力。其次，影子去除模块则使用提示感知模块生成的影子掩模来恢复影子区域的内容。该模块通过稠密-稀疏局部注意力（DSLA）机制聚合有效的上下文信息，从而提高影子去除的精确度。最后，PACSRNet通过最小化影子去除损失和影子预测损失的组合来进行训练，确保模型在去除影子时保持空间一致性。
在实验部分，研究团队使用定制的PCSRD数据集和ISTD+数据集对PACSRNet进行了评估。PCSRD数据集包含11,900个复杂场景样本，涵盖多种用户提示。实验结果表明，PACSRNet在影子去除任务中表现优异，尤其是在影子区域的去除效果上，显著优于五种现有的基线方法，包括ShadowFormer和ShadowDiffusion。定量分析显示，PACSRNet在多个指标上（如PSNR和SSIM）均取得了较高的分数，验证了其在影子去除领域的有效性和灵活性。此外，定性结果也展示了PACSRNet在处理不同类型用户提示时的鲁棒性，能够准确去除指定对象的影子。
PACSRNet的工作原理可以简单理解为一个智能的影子去除助手。首先，用户可以通过简单的提示（比如在图片上画一个点或线）告诉系统要去掉哪个对象的影子。接着，系统会分析这个提示，找出影子所在的区域，并生成一个影子掩模。然后，系统会利用这个掩模来恢复影子区域的内容，确保最终的图片看起来自然且没有影子。PACSRNet的设计使得用户不再需要手动标记影子区域，减少了繁琐的工作，同时也提高了影子去除的灵活性和准确性。这个过程不仅高效，而且用户友好，适合各种场景的影子去除需求。
2025-01-27｜USurrey-SketchX,iFlyTek-Surrey｜⭐️🟡
http ://arxiv.org/abs/2501.16022v1
本研究提出了一种创新的“草图循环”图像分割框架，旨在利用自由手绘草图作为主观图像分割的查询方式。该方法的核心理念是通过草图的主观性来增强图像分割的灵活性，允许用户在多种层次上进行分割，包括类别级、细粒度和部分级。与传统需要大量标注数据的分割方法不同，本框架不依赖于特定的数据集或掩膜，而是通过与大规模预训练模型的结合，利用草图作为输入，实现对视觉概念的有效分割。研究结果表明，该方法在多个基准数据集上表现优于现有的图像分割技术，展示了其在开放词汇环境下的广泛应用潜力。
本研究的方法主要包括以下几个步骤：
草图查询和特征提取：输入的草图通过预训练的视觉编码器（如CLIP和DINOv2）提取全局特征，生成一个草图特征向量。
相似度计算：利用余弦相似度计算草图特征与候选图像的局部特征之间的相似度，生成草图引导的相关性图。
掩膜生成：通过对相关性图进行双线性上采样和可微分阈值处理，生成二进制分割掩膜，标记出草图所查询概念的像素位置。
多层次分割：该方法支持多层次的分割，包括类别级（标记所有同类物体）、细粒度（标记特定实例）和部分级（标记物体的具体部分），通过草图的分割能力来实现更细致的图像处理。
正则化与损失函数：引入掩膜正则化损失，防止生成全白掩膜，确保模型的泛化能力。
在实验部分，我们使用Sketchy和Sketchy-Extended数据集进行训练和评估，分别针对类别级和细粒度分割进行测试。通过手动标注的分割掩膜构建测试集，评估指标包括平均交并比（mIoU）和每像素分类准确率（pAcc）。实验结果显示，提出的方法在各个评估指标上均优于对比基线，特别是在未见类别的测试中，表现出较强的泛化能力。为了进一步验证模型的有效性，进行了消融实验，探讨了不同组件对整体性能的影响，结果表明草图分割和正则化策略对性能提升至关重要。
在我们的研究中，我们提出了一种新颖的图像分割方法，允许用户使用简单的手绘草图来选择和分割图像中的对象。这个过程可以分为几个简单的步骤：首先，用户画出他们想要分割的对象的草图。然后，系统会分析这个草图，并与数据库中的图片进行比较，找出最相似的部分。接着，系统会生成一个掩膜，标记出草图中对象在图片中的位置。这个方法不仅可以识别整个类别的对象，例如所有的“猫”，还可以识别特定的实例，比如“坐着的猫”，甚至可以只标记出“猫的头”。通过这种方式，用户可以灵活地进行图像分割，而无需依赖复杂的标注数据。
2025-01-26｜ZJU,MBZUAI｜⭐️
http ://arxiv.org/abs/2501.15562v1
在当今的文本到图像（T2I）扩散模型中，生成高质量图像的能力已经取得了显著的进展。然而，随着对隐私和安全的关注日益增加，如何有效地从生成图像中删除不当内容（如NSFW内容）成为一个重要的研究课题。本文提出了一种名为CE-SDWV的框架，旨在通过对文本语义空间中的条件进行调整，来高效地抑制目标概念。这一方法不仅不需要重新训练模型的参数，还能够在保持生成质量的同时，减少存储和计算开销。通过构建与目标概念相关的词汇表，该框架能够有效地识别并消除不当内容，从而使生成的图像符合社会规范。
CE-SDWV框架的核心在于三个主要步骤：
语义驱动的概念表示：首先，利用大型语言模型（LLM）生成与特定目标概念相关的词汇，并构建一个语义矩阵，以增强目标概念在文本语义空间中的表示。通过对词汇进行处理，提取出与目标概念高度相关的主成分，构建出准确捕捉目标概念特征的语义空间。
自适应组件抑制：在这一阶段，框架会对文本条件中的每个令牌进行自适应的概念抑制，确保有效去除目标概念的信息。通过对文本令牌的嵌入进行奇异值分解（SVD），识别出与目标概念相关的成分，并将其从文本条件中移除。
梯度正交令牌优化：最后，采用端到端的梯度正交优化策略，对抑制后的文本令牌进行调整，以更好地适应原始图像的语义空间。这一过程有助于提高生成图像的细节质量，同时确保目标概念的有效抹除。
为验证CE-SDWV框架的有效性和效率，研究团队在I2P和UnlearnCanvas两个基准上进行了广泛的实验。实验结果表明，该方法在抑制不当概念的同时，能够有效保留无关概念的生成能力。特别是在去除性别相关内容、艺术风格和特定对象类别的实验中，CE-SDWV的表现优于现有的多种方法。此外，通过对抗性提示攻击的评估，CE-SDWV展现出了良好的鲁棒性，能够有效防止模型生成不当内容。实验还表明，CE-SDWV在存储需求和计算时间上具有显著优势，进一步证明了其在实际应用中的可行性。
CE-SDWV框架的工作原理可以简单理解为一个三步走的流程。首先，研究团队使用一种智能的语言模型来生成与特定不当内容（如色情内容）相关的词汇。这些词汇帮助模型更好地理解哪些信息需要被抹去。接着，模型会对文本中的每个词进行处理，找出和不当内容相关的部分，并将其移除。这一步骤确保生成的图像不会包含这些不当内容。最后，模型会进行一些额外的调整，以确保生成的图像在细节上依然清晰、真实。这种方法不仅能有效去掉不想要的内容，还能保持其他正常内容的生成能力，确保整个过程高效且不需要重新训练模型。
2025-01-27｜SMU,ANU｜⭐️
http ://arxiv.org/abs/2501.15775v1
文本到图像（T2I）模型近年来因其生成高质量图像的能力而备受关注，但其性别偏见问题引发了广泛担忧。以往研究表明，T2I模型在使用中性文本提示时，往往会加剧性别刻板印象。例如，在生成“CEO”时，模型更倾向于生成男性形象，而在生成“护士”时则倾向于女性形象。尽管已有研究提出了自动化的性别偏见检测工具，但缺乏对这些工具的全面比较，以及它们所检测到的性别偏见与实际情况的偏差理解。本研究旨在填补这一空白，通过使用人工标注的数据集验证现有的性别偏见检测器，并比较不同检测器识别的偏见水平与实际偏见之间的差异。
本研究采用了以下步骤来评估T2I模型的性别偏见检测器：
数据集构建：创建了一个包含6000幅图像的数据集，这些图像由三种先进的T2I模型生成，分别是StableDiffusionXL、StableDiffusion3和DreamlikePhotoreal2.0。数据集中的图像通过手动标注，分为男性、女性和低质量图像。
性别偏见检测器选择：选择了七种代表性的性别偏见检测器，包括基于视觉语言模型的CLIP及其变种、专门的性别分类模型（如FairFace）和API服务（如Face++）。这些检测器的性能被评估，以确定其在识别T2I模型性别偏见方面的有效性。
偏见测量：通过模型偏见评分和提示偏见评分来量化T2I模型的性别偏见。模型偏见评分评估生成的男性与女性图像的比例，而提示偏见评分则分析特定提示下生成图像的性别分布。
实验过程中，研究人员首先对生成的6000幅图像进行性别标注，以便于后续的偏见检测。通过比较检测器的输出与人工标注结果，发现现有检测器在准确识别T2I模型的性别偏见方面存在显著差异，部分检测器的偏见估计偏差高达26.95%。针对这些问题，研究团队提出了一种增强型检测器CLIP-Enhance，能够更准确地测量性别偏见，其偏差仅为0.47%-1.23%，并且有效过滤掉82.91%的低质量图像。这一新检测器的性能显著优于现有工具，表明其在性别偏见检测中的潜力。
在这项研究中，研究人员希望找出现有工具在检测文本到图像模型中的性别偏见方面的有效性。首先，他们创建了一个包含6000幅图像的数据集，这些图像是通过三种不同的图像生成模型生成的。接着，他们使用七种不同的工具来检查这些图像的性别偏见。通过比较这些工具的检测结果与人工标注的实际性别，发现现有工具在识别性别偏见时存在很大差异。为了解决这个问题，研究人员开发了一种新的工具，能够更准确地识别性别偏见，并有效地过滤掉那些无法判断性别的低质量图像。最终，这项研究不仅揭示了现有工具的不足，还提供了一个更好的解决方案，帮助我们更好地理解和应对AI模型中的性别偏见问题。

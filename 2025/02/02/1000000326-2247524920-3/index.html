287-A2｜视频重光照，评估人体姿势迁移模型在人类动作视频生成中的表现

仅用于站内搜索，没有排版格式，具体信息请跳转上方微信公众号内链接

AIGCResearch
主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）发布日期｜2025年1月28日｜周二
2025-01-27｜ShanghaiAILab,FDU,SJTU,ZJU,Stanford,CUHK｜⭐️⭐️
http ://arxiv.org/abs/2501.16330v1
在视频重光照的研究领域，RelightVid提出了一种新颖的框架，旨在通过时间一致的扩散模型实现高质量的视频重光照。传统的视频重光照技术面临着缺乏配对数据集和保持时间一致性的挑战。RelightVid利用扩散模型的潜力，结合背景视频、文本提示和HDR环境视频作为输入条件，能够在不需要复杂的内在分解的情况下，灵活地处理动态前景物体的光照变化。通过在大量真实视频和3D渲染数据上进行训练，该模型能够在多种光照条件下实现任意视频的重光照，保证输出的高保真度和时间一致性。
RelightVid的核心方法包括几个关键步骤：
LightAtlas数据收集：构建了一个高质量的视频重光照数据集，包含来自真实视频和3D渲染的数据，确保模型学习到丰富的光照先验知识。
模型设计：采用预训练的2D图像重光照扩散模型作为基础，通过将其扩展为3DU-Net结构来处理视频数据。模型中引入了时间注意力层，以捕捉帧间的时间依赖性，确保视频重光照的一致性。
多模态条件联合训练：模型支持结合背景视频、文本提示和HDR环境图进行重光照。通过编码这些输入，模型能够在编辑过程中动态整合信息，确保输出视频与给定的视觉和文本上下文相符。
光照不变集成策略：通过对输入视频进行亮度增强，生成多个增强版本，并在每个扩散步骤中计算噪声的平均值，从而提升模型在不同光照条件下的鲁棒性。
在实验部分，RelightVid经过了全面的评估，以验证其性能。首先，通过与基线方法的比较，展示了在视频背景条件下的重光照效果，采用了PSNR、SSIM和LPIPS等指标来评估图像质量和时间一致性。其次，在文本条件重光照的实验中，模型在处理文本提示时表现出色，能够保持前景物体的身份和光照一致性。最后，HDR条件的引入进一步提升了重光照的准确性，尤其是在复杂光照场景下。用户研究表明，RelightVid在视频平滑度和光照合理性方面超越了现有的其他视频编辑方法，证明了其在多种应用场景中的有效性。
RelightVid的工作原理可以理解为一个智能视频编辑工具，能够根据用户提供的背景视频、文字描述和环境光照信息，自动调整视频中的光照效果。首先，研究人员创建了一个庞大的视频库，里面有很多不同光照条件下拍摄的视频，以便模型学习如何在不同情况下进行光照调整。接着，他们设计了一个复杂的计算模型，像是一个有记忆的机器人，能够记住前后帧之间的变化，从而保证整个视频看起来连贯自然。当用户给出新的背景或描述时，这个模型会结合这些信息，生成新的光照效果，确保视频中的人物或物体在不同光照下依然保持真实感。最后，通过大量实验和用户反馈，RelightVid被证明能够在各种条件下有效地重光照，提升视频的视觉效果。
2025-01-26｜SSPS,Stanford｜⭐️
http ://arxiv.org/abs/2501.15648v1https ://github.com/matyasbohacek/pose-transfer-human-motion
本研究旨在评估三种先进的人体姿势转移模型——AnimateAnyone、MagicAnimate和ExAvatar——在生成逼真的人类动作视频方面的表现。姿势转移技术的目标是利用参考视频中的动作，通过新的身份进行重现。尽管这些方法在学术上有着重要贡献，并在娱乐、医疗和时尚等多个领域展现出应用潜力，但在真实世界的应用中仍面临挑战。通过对20种不同人类动作的控制实验，研究发现参与者仅在42.92%的情况下正确识别出所生成视频中的动作，并且只有36.46%的生成视频被认为与参考视频一致。这些结果表明，尽管存在一定的技术进步，这些模型在处理超出训练分布的身份时仍显不足。
本研究采用以下步骤来生成姿势转移视频并进行评估：
数据收集：
参考动作视频：从UCF101数据集中选取60个视频，涵盖20个动作类别。
新的人类身份：为AnimateAnyone和MagicAnimate收集14个不同身份的静态图像，而ExAvatar则需要15个身份的视频。
视频生成：
AnimateAnyone和MagicAnimate：这两种模型基于静态图像生成视频，采用扩散模型架构以提高时间一致性。
ExAvatar：使用3D高斯模型生成动态视频，依赖于输入视频以实现更自然的动作表现。
参与者研究：
设计了一项包含66个问题的调查，参与者需观看生成的视频并评估其质量和一致性。调查分为三个任务，分别测试生成动作的识别、动作一致性和视频质量的主观评价。
在实验中，共招募16名参与者参与对生成视频的评估。每位参与者观看相同顺序的视频，并对每个视频进行评价。实验结果显示，参与者在识别生成的动作时的准确率为42.92%，而在判断视频与参考视频的一致性时，结果为36.46%。不同模型的表现存在显著差异，ExAvatar生成的视频在一致性和真实感方面表现最佳，参与者认为其视频一致性达到55.00%。相较之下，AnimateAnyone和MagicAnimate的表现则较差，尤其在生成的动作识别和视频一致性方面，MagicAnimate的结果最为不理想。
在这项研究中，我们主要关注三种技术如何将一个人的动作转移到另一个人身上。首先，我们从一个大型视频库中选择了一些动作视频作为参考，然后为每种技术准备了不同的新身份图片或视频。接下来，我们用了这些技术生成新视频，让参与者观看并回答一些问题，判断他们能否识别出视频中的动作，以及这些动作是否与参考视频一致。结果显示，参与者的识别准确率并不高，尤其是在使用MagicAnimate技术时，许多人都难以理解视频中展示的动作。这表明，尽管这些技术在实验室条件下表现良好，但在实际应用中仍需改进，特别是在确保生成视频的真实感和动作一致性方面。

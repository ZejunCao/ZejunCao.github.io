2025-01-25｜THU,BJTU,DLMU｜⭐️🟡
http ://arxiv.org/abs/2501.15096v1
在本文中，我们提出了一种名为Rob-GS的框架，旨在为任意长度的视频序列提供更为稳健的3D姿态估计与优化。传统的3D高斯点云表示（3DGS）在训练时通常依赖于已知的相机姿态，而这些姿态往往需要通过耗时的结构光束法（SfM）预处理获得。Rob-GS通过逐步估计相机姿态，克服了这一限制，并通过“分而治之”的策略将视频序列分割为多个局部段落进行独立优化。这种方法不仅确保了重建的稳定性，还避免了在处理长视频时的内存溢出问题。通过在Tanks和Temples数据集及自收集的真实场景上进行的广泛实验，Rob-GS在渲染质量和姿态估计准确性上均优于现有的最先进方法。
Rob-GS的核心方法包括三个主要组成部分：
3D高斯表示：我们利用多个各向异性的3D高斯来表示场景。每个高斯与几何和外观属性相关联，通过优化这些属性来最小化渲染图像与真实图像之间的色差和深度差。
相机姿态估计：我们采用相邻帧跟踪方法，利用视频的时间连续性来确保相邻帧之间的姿态稳定。通过单帧深度估计和光流信息，逐步提升相机姿态的准确性。
自适应分段策略：为了处理任意长的视频输入，我们将视频分割为多个局部段，并独立优化每个段落。这种方法不仅提高了优化效率，还有效防止了内存溢出。
我们在多个真实场景数据集上进行了广泛的实验，特别是在Tanks和Temples数据集上，Rob-GS表现出色。我们与现有的最先进方法（如Nope-NeRF和CF-3DGS）进行了比较，结果显示Rob-GS在渲染质量（PSNR、SSIM等）和姿态估计准确性（相对姿态误差）方面均有显著提升。实验还表明，Rob-GS的训练速度远快于其他方法，处理复杂场景时的内存使用也更为高效。此外，我们还通过自收集的数据集验证了Rob-GS在现实世界应用中的可行性，结果同样优于对比方法。
在Rob-GS的工作流程中，我们首先将视频分成几个小段，然后逐段进行处理。每一段的处理方法是这样的：首先，我们从第一帧中提取出场景的深度信息，并用这些信息生成一个3D点云。接着，我们为每个点生成一个高斯分布，这样可以更好地表示场景的细节。然后，我们通过比较每一帧的图像，来不断调整相机的位置和角度，确保每一帧的姿态都尽可能准确。最后，当我们处理完一段后，会根据可见性和一致性来判断是否需要开始处理下一段。这种方法使得我们能够在不需要预先计算相机姿态的情况下，处理长视频，并且确保了渲染效果的高质量。
2025-01-27｜TUMunich｜⭐️
http ://arxiv.org/abs/2501.16312v2https ://nicolasvonluetzow.github.io/LinPrim
LinPrim是一种新颖的方法，利用线性原语（如八面体和四面体）进行可微分的体积渲染，以促进3D场景重建和新视图合成（NVS）。该方法提出了一种可微分的渲染管道，结合了实时光栅化技术，专为这些原始体设计，从而实现了交互式的帧速率。这种方法不仅提高了新视图渲染的效率，还展示了显式多面体原始体在现代NVS工作流程中的潜力。当前，体积渲染在新视图合成中扮演着重要角色，尤其是随着NeRF和3D高斯点等方法的兴起，LinPrim为3D场景的表示提供了新的视角和选择。
LinPrim的核心在于引入两种基于多面体的场景表示：八面体和四面体。每种原始体通过一组特征进行描述，包括几何参数、位置、透明度和颜色。具体的渲染过程分为以下几个步骤：
预处理：在渲染前，首先构建每个原始体，并根据相机位置进行变换，计算视图依赖的颜色。
全局排序与分块：利用全局排序将原始体映射到相应的像素，提升渲染效率。
光栅化：通过光线与三角形的交叉计算，确定每个像素的透明度和颜色。
优化：采用可微分渲染技术，通过已知视图优化原始体的特征，使用基于梯度的优化方法来调整原始体的几何和外观特征。
在实验中，LinPrim在Mip-NeRF360和ScanNet++v2数据集上进行了评估。结果表明，LinPrim在重建质量方面与现有的3DGS和Mip-Splatting方法相当，但使用的原始体数量显著减少。具体而言，LinPrim在ScanNet++上使用了255K个原始体，且在渲染速度上也表现出色，八面体方法达到了68帧每秒，而四面体方法则更快，达到175帧每秒。实验还表明，LinPrim的抗锯齿处理能够有效提升渲染质量，且其灵活性使得该方法在多种场景中都能实现高保真度的重建。
LinPrim的方法可以简单理解为使用特定形状的几何体（八面体和四面体）来构建3D场景。在这个过程中，首先要为每个几何体设置一些基本参数，比如它的位置、透明度和颜色。接下来，通过一个两步的渲染过程，先处理这些几何体，然后将它们映射到屏幕上的像素点。这样，当我们从不同的角度查看场景时，系统能够快速而准确地计算出每个像素的颜色和透明度。最后，LinPrim还会通过不断调整这些几何体的特征，确保场景的真实感和细节表现。这种方法不仅提高了渲染速度，还能保持高质量的视觉效果。
2025-01-25｜CityU,TencentGames｜🟡
http ://arxiv.org/abs/2501.15008v1
HuGDiffusion是一种新颖的3D高斯扩散学习管道，旨在通过单视图输入图像生成逼真的人类角色的多视角合成。当前的许多方法依赖于视频帧或经过校准的多视图图像作为输入，这限制了其在真实世界中应用的灵活性，尤其是在相机姿态未知或稀疏视图的情况下。为了解决这个问题，HuGDiffusion通过从单张图像中提取人类先验信息，生成一组3D高斯属性，并采用扩散模型来实现这一目标。通过多阶段生成策略，HuGDiffusion有效地学习和优化不同类型的3D属性，从而显著提高了视觉渲染的质量和效率。
HuGDiffusion的核心方法包括两个主要模块：
人类重建管道：该模块生成点云，并将其初始化为3D高斯位置。通过对输入图像进行单视图深度估计，生成对应的深度图，进而获取3D点云。
高斯属性扩散模块：该模块学习3D高斯属性的分布，利用条件扩散机制生成逼真的结果。为优化训练过程，HuGDiffusion采用了两阶段的工作流程：
第一阶段：通过对每个场景进行独立优化，创建代理的高斯属性集，确保生成的属性在不同场景中保持一致。
第二阶段：通过深度学习网络统一优化所有场景的高斯属性，减少属性之间的随机性和不一致性，从而提高模型的泛化能力。
此外，HuGDiffusion还集成了人类中心特征提取流程，以增强条件扩散机制的有效性，确保生成的3D高斯属性更为准确和丰富。
在实验中，我们使用了480个人类模型数据集进行HuGDiffusion的训练和验证。通过对比不同方法的性能，包括GTA、SiTH、LGM和SHERF，HuGDiffusion在多个指标上均表现出色，尤其是在PSNR和SSIM等视觉质量评估方面。我们还进行了消融实验，验证了两阶段构建流程和点云变换器的必要性，结果表明这种方法显著提高了生成的3D高斯属性的一致性和准确性。此外，HuGDiffusion在多种数据集上展现了良好的泛化能力，能够在不同场景和条件下生成高质量的人类图像。
HuGDiffusion的工作原理可以简单理解为一个两步走的过程。首先，它从一张普通的照片中提取出人类的三维信息，想象成将这张照片“变魔术”成一个可以在空间中旋转的3D模型。这个过程就像用点云来构建一个“骨架”，这些点代表了人体的各个部分。接下来，HuGDiffusion会利用一个智能的学习系统，逐步为这个骨架添加细节，比如皮肤的颜色、衣服的纹理等，最终生成一个看起来非常真实的3D人类形象。通过这种方法，即使只用一张照片，它也能创造出多种不同角度的图像，展现出丰富的细节和真实感。
2025-01-25｜ZhiyuanCollege,SJTU
http ://arxiv.org/abs/2501.15058v1
本研究提出了一种新的文本到动作生成方法KETA（Kinematic-Phrases-EnhancedText-to-MotionGeneration），旨在解决文本与动作之间的内在差距。现有的文本到动作（T2M）生成技术通常依赖于扩散模型，利用文本提示生成相应的运动，但由于语言的抽象性与动作空间的连续性之间的差异，现有方法在生成与文本一致的运动时面临重大挑战。为此，研究者们引入了运动短语（KinematicPhrases,KP）作为中间表示，旨在通过细粒度对齐来增强运动生成的质量。KETA通过将文本分解为多个细化的短语，并将这些短语与生成的运动进行对齐，从而为扩散模型提供了更为准确的指导信号。实验表明，KETA在多种指标上均优于现有的T2M模型，展示了其在运动生成领域的潜力。
KETA的核心方法包括以下几个步骤：
文本分解：利用大型语言模型（LLM）将输入文本分解为多个细化的短语，这些短语能够更好地反映运动的时序和空间关系。
KP提取：从数据集中提取运动短语（KP），捕捉运动的空间和时间特征，并为每个KP分配相应的约束。
文本-KP对齐：通过训练一个对齐模型，将分解后的文本与KP进行对齐，最小化它们在KP空间中的距离，从而实现运动生成过程中的物理一致性。
扩散模型的迭代优化：在生成阶段，KETA利用文本-KP对齐结果作为指导信号，通过多轮迭代优化生成的运动，确保其与输入文本保持一致。
这一方法的创新之处在于引入了细粒度的文本-KP对齐机制，使得生成的动作不仅在语义上与文本相符，而且在时间和空间上也具备更高的准确性。
在实验部分，KETA的性能通过与多种基准模型的比较进行评估。研究者们使用了HumanML3D数据集，该数据集包含丰富的人类动作数据。评估指标包括R-Precision（文本与生成动作的相似度）、FID（生成样本与真实样本的分布差异）和多样性。实验结果显示，KETA在R-Precision和FID值上均显著优于基线模型，达到了1.19倍和2.34倍的提升。此外，KETA在生成的动作与文本之间的时序和空间一致性方面表现出色，成功解决了现有模型在这些方面的不足。通过对比分析，KETA展现了其在T2M生成中的潜力，尤其是在处理复杂的文本描述时。
KETA的工作原理可以简单理解为一个“翻译器”，它将我们说的动作转换为计算机可以理解的运动形式。首先，当我们输入一段描述，比如“一个人走向前，然后蹲下去捡东西”，KETA会将这段话拆分成几个小的部分，每个部分描述一个具体的动作。接着，KETA会从已有的运动数据中提取出与这些小部分相对应的运动特征，这些特征就像是动作的“说明书”。然后，它会将这些小部分与提取的运动特征进行对比，确保生成的动作与描述一致。最后，KETA会通过不断调整生成的动作，确保它们在时间和空间上都与我们的描述相符。这样，KETA就能生成更加自然、流畅且符合我们期望的运动。
2025-01-27｜A*STAR
http ://arxiv.org/abs/2501.15839v1
在计算机视觉领域，可控手部抓取生成（ControllableHandGraspGeneration）和手物交互（HOI）生成已经成为一个重要的研究方向。手部抓取生成是有效控制手部几何形状的关键步骤，然而现有方法通常依赖于三维（3D）信息，这限制了其灵活性和可控性。为了解决这个问题，本文提出了一种基于更高阶几何表示（HigherOrderRepresentations,HORs）的新框架，结合谱图理论和向量代数，以改进生成手部姿势的质量。该框架不仅能够生成更为真实的手部抓取姿势，还能在用户指定的位置和方向上进行控制，从而提升整体的手物交互生成效果。
本研究的方法主要包括以下几个方面：
几何特性建模：我们将手部姿势视为离散图结构，利用几何先验信息进行建模。这种建模方式能够更好地捕捉手指关节角度、手掌朝向等几何特性。
高阶几何表示（HOR）：通过引入高阶几何表示，本文展示了如何利用图论和谱图理论来提高手部姿势生成的质量。具体包括根相对距离、骨长和指骨方向等描述符的计算。
扩散模型：提出了一种新型的扩散模型，能够在仅使用二维（2D）图像数据的情况下生成可控的手部抓取姿势。模型通过逐步添加噪声并反向去噪，确保生成的手部姿势与给定的上下文信息相一致。
评估框架：我们设计了一套高效且稳定的评估指标，以克服现有评估方法的不足，确保生成的手部姿势在质量上的一致性和可靠性。
在实验部分，本文使用HOI4D数据集进行详细的定量和定性评估。我们比较了不同手部姿势生成方法的性能，包括基线方法PoseVanillaNet和我们提出的PoseNet系列模型。实验结果表明，采用高阶几何表示的模型在抓取一致性和生成质量上显著优于传统方法。此外，我们还通过不同损失函数的组合，评估了模型在处理不同任务时的表现。实验结果验证了我们提出的评估指标在手部姿势生成中的有效性和稳定性。
在我们的研究中，我们提出了一种新方法来生成手部抓取姿势，这样可以让机器更好地理解如何抓取物体。首先，我们把手的姿势看作一个图形，这样可以更容易地计算手指之间的距离和角度。然后，我们利用一种叫做扩散模型的技术，逐步添加噪声来生成手势，最后再把噪声去掉，从而得到清晰的手部姿势。这个过程就像在画画时，先用铅笔轻轻勾勒出轮廓，然后再用橡皮擦去多余的部分，最后留下我们想要的图形。我们还设计了一些新的评估方法，确保生成的手势是准确和自然的。通过这些步骤，我们的模型能够在不同的情况下生成更好的手部抓取姿势。

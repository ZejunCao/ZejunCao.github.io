2025-02-05｜NTU,StepFun,ShanghaiTech,WestlakeU｜⭐️🟡
http ://arxiv.org/abs/2502.03207v1
本文提出了一种新颖的图像到视频生成方法，称为MotionAgent，旨在通过文本输入实现对视频生成的细粒度控制。其核心技术是运动场代理（motionfieldagent），该代理将文本提示中的运动信息转化为明确的运动场，从而提供灵活且精确的运动指导。具体而言，MotionAgent能够提取文本中描述的对象运动和相机运动，并将其转换为对象轨迹和相机外部参数。这种方法不仅提高了视频生成的控制精度，还在多个评估指标上超越了现有的先进模型，尤其是在视频与文本之间的运动信息对齐方面表现出色。
MotionAgent的设计分为几个关键步骤：
运动信息解析：代理首先解析文本中的运动信息，将其分解为描述对象运动和相机运动的独立部分。
对象轨迹绘制：通过识别文本中提到的对象，代理确定对象在图像中的位置，并基于文本描述绘制出对象的运动轨迹。
相机外部参数生成：根据文本中描述的相机运动，代理直接生成相机的外部参数，确保相机运动与对象运动的协调。
光流组合模块：通过分析光流数据，将对象运动和相机运动的信息合并，生成统一的光流图，用于后续视频生成。
I2V生成模型控制：最后，使用这些运动表示作为条件，控制基础的图像到视频扩散模型，以生成符合文本描述的动态视频。
在实验中，作者首先在一个公共的视频生成基准VBench上评估了MotionAgent的性能。实验结果表明，该方法在相机运动控制精度上显著提高，并在视频质量上与其他先进模型相当。此外，作者构建了一个新的视频运动基准子集，用于评估文本中的运动信息与生成视频之间的对齐情况。通过用户研究，结果显示MotionAgent生成的视频在运动信息与文本描述的一致性方面表现优于其他模型。最后，通过消融实验验证了每个模块的有效性，进一步支持了该方法的有效性和创新性。
MotionAgent的工作原理可以简单理解为一个智能助手，它能够根据你给的文字描述来制作视频。首先，它会分析你说的内容，比如“一个热气球向右飞”，然后它会找出这个热气球在图像中的位置，并画出它的飞行轨迹。同时，如果你提到“相机向下移动”，它也会确定相机的运动方式。接下来，MotionAgent会把这些信息结合起来，生成一个流畅的动画视频，确保视频中的动作跟你的描述完全一致。这样，无论你想要什么样的场景，MotionAgent都能通过简单的文字来实现它，帮助你轻松制作动态视频。
2025-02-04｜MBZUAI,CMU｜⭐️
http ://arxiv.org/abs/2502.02690v1
可控视频生成是一个重要的研究领域，尽管近年来取得了一些进展，但仍面临挑战。现有的方法通常将视频视为一个整体，忽视了细粒度的时空关系，从而限制了控制的精确性和效率。为了解决这些问题，本文提出了一种新的可控视频生成模型——可控视频生成对抗网络（CoVoGAN），该模型通过概念的解耦合来实现对视频内容的独立控制。具体来说，研究者们首先将静态和动态潜在变量进行解耦，然后利用充分变化性质实现动态潜在变量的逐步识别，从而实现对动作和身份的独立控制。通过理论分析，本文验证了该方法的有效性，并设计了一个时间转换模块以增强潜在动态的解耦表现，最终在多个视频生成基准测试中展示了该方法在生成质量和可控性方面的显著提升。
本文提出的CoVoGAN模型包含以下几个关键步骤：
静态与动态潜在变量的解耦：首先，模型将视频中的静态内容（如身份）与动态内容（如动作）分开，以便于独立控制。
充分变化性质的利用：通过最小化潜在动态变量的维度，确保在时间上具有条件独立性，从而实现动态变量的可识别性。
时间转换模块的设计：该模块在生成过程中对潜在动态进行解耦，确保模型可以在不影响身份的情况下独立调整动作。
对抗网络的集成：将上述模块作为插件集成到生成对抗网络中，以增强生成过程的可控性和质量。
理论分析：通过严谨的理论分析，证明了模型在解耦合方面的有效性，确保了生成过程的可解释性。
在实验部分，研究者对CoVoGAN进行了广泛的定性和定量评估，使用了多个真实世界数据集，包括面部伪造、天空延时和房地产视频数据集。实验评估的指标包括视频生成质量的Frechet视频距离（FVD）和生成的可控性。结果表明，CoVoGAN在各个数据集上都显著优于现有的基线模型，生成的视频质量更高，控制精度更好。此外，通过对比实验，研究者们还验证了模型的鲁棒性和计算效率，显示出CoVoGAN在处理复杂视频生成任务时的优势。
在这个研究中，科学家们想要创造一种可以更好控制视频生成的模型。想象一下，你在看一段视频，视频里有一个人正在走路。传统的方法可能会把这个视频看作一个整体，而不去考虑这个人是如何走的、他的表情如何等细节。为了改善这种情况，研究者们设计了一种新方法，叫做CoVoGAN。这个方法首先把视频中的“动作”（比如走路的方式）和“身份”（比如这个人是谁）分开，这样你就可以单独调整动作而不改变这个人的身份。为了实现这一点，他们还设计了一个特别的模块，帮助模型更好地理解和生成这些动作。通过大量实验，他们发现这种方法不仅生成的视频质量更高，而且在控制视频内容方面也更加灵活和有效。
2025-02-03｜CMU,MITMediaLab｜NeurIPS2024｜⭐️🟡
http ://arxiv.org/abs/2502.02610v1
本研究旨在通过开发一个全自动化的个性化音乐视频生成管道，来提升音乐体验。该系统不仅让听众成为音乐视频的消费者，还使他们能够成为共同创作者。通过分析歌词、节奏和音乐情感，系统能够生成个性化且符合上下文的视觉内容。研究中引入了CHARCHA，一个面部身份验证协议，以确保用户身份的安全，防止未经授权的面部使用，同时收集用户授权的图像以实现视频个性化。该框架结合了多模态翻译和生成技术，提供了一种安全且创新的方法来创建深度个性化的音乐视频。
本研究的方法分为几个关键步骤：
音乐到视频的转换：
使用OpenAI的自动语音识别模型Whisper提取音频文件中的歌词和时间戳。
利用音乐情感识别模型分析音乐情感，并结合歌词生成图像提示。
采用预训练的文本到图像潜在扩散模型（如StableDiffusion1.5）生成与歌词同步的图像。
情感提取：
通过开源音频特征提取工具openSMILE提取音频特征。
训练神经网络模型来预测情感的唤起度和效价，并追踪情感变化。
LLM条件生成：
利用大型语言模型（LLM）将歌词和情感信息转化为丰富的图像提示，以保持叙事的一致性。
个性化与安全性：
通过低秩适应（LoRA）技术，利用用户提供的图像生成个性化的音乐视频。
CHARCHA协议通过实时动作验证用户身份，确保安全性。
在实验部分，研究团队首先评估了生成的视频质量和个性化程度。通过不同的模型对RickAstley的《NeverGonnaGiveYouUp》进行实验，比较不同模型生成的视觉风格和文本提示的对齐效果。实验结果表明，所选模型对生成风格有显著影响，个性化模型能够准确再现参与者的外貌。CHARCHA协议的有效性通过多阶段测试验证，确保用户身份的真实性并防止滥用。最后，通过定量分析生成视频中的面部识别准确性，评估了该系统在生成个性化内容方面的能力。
在我们的研究中，我们开发了一种新方法，帮助人们制作个性化的音乐视频。首先，我们会从音乐文件中提取歌词和情感信息。接着，利用这些信息生成与音乐节奏和情感相匹配的图像。为了让每个人的视频都独一无二，我们还会使用用户的照片来创建个性化的角色。最重要的是，我们引入了一个叫做CHARCHA的安全验证系统，确保用户的身份不会被恶意使用。整个过程简单易懂，即使没有技术背景的人也能轻松上手，创造出属于自己的音乐视频。

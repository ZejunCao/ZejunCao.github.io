2025-02-06｜USTC,MSRA｜⭐️🟡
http ://arxiv.org/abs/2502.03997v1
在现代数字化时代，计算机辅助设计（CAD）已成为各行业不可或缺的工具。CAD模型的创建通常是一个迭代过程，设计师通过草图绘制2D曲线以定义形状的边界，并将其挤压成3D形状。尽管基于文本的CAD编辑在自动化CAD模型修改方面具有极大潜力，但目前的研究仍较为稀缺。现有方法主要集中在设计变体生成或基于文本的CAD生成，但往往缺乏对文本控制的支持或忽视了现有CAD模型的约束。为此，本文提出了CAD-Editor，这是第一个针对基于文本的CAD编辑的框架，通过自动化数据合成来解决训练数据的需求，从而推动CAD设计工作流程的变革。
CAD-Editor框架将基于文本的CAD编辑任务分解为两个主要子任务：定位和填充。具体方法如下：
自动化数据合成：通过利用设计变体模型生成原始和编辑CAD模型的配对，并使用大型视觉语言模型（LVLMs）总结它们之间的差异，以生成准确的编辑指令。
定位阶段：在此阶段，利用大型语言模型（LLMs）生成一个掩码CAD序列，标记出需要修改的区域，未修改的部分则从原始CAD序列中复制。
填充阶段：在填充阶段，LLMs使用掩码CAD序列及编辑指令生成最终的编辑CAD序列，确保对未修改部分的保持和对修改区域的精确填充。
这种分阶段的方法不仅提高了生成质量，还增强了与用户指令的对齐程度，使得CAD-Editor在处理复杂的文本指令时表现更为优越。
为了验证CAD-Editor的有效性，进行了一系列实验。首先，使用DeepCAD数据集进行数据合成，生成了大量的CAD模型和对应的编辑指令。通过与其他基线模型进行比较，CAD-Editor在有效性、现实性和生成质量等指标上均表现优异。实验结果显示，CAD-Editor的有效比率达到了95.6%，大幅超越其他方法。此外，使用人类评估的方式，CAD-Editor在生成的CAD模型质量上也得到了较高的认可，显示出其在实际应用中的潜力。
CAD-Editor的工作原理可以看作是一个“找东西然后填补”的过程。首先，它会根据用户的指令来定位CAD模型中需要修改的部分，就像在一张地图上找出需要改变的地点。接着，系统会根据指令来填补这些区域，确保修改后的模型看起来符合用户的要求。这个过程分为两步，第一步是找出要改的地方，第二步是把这些地方改好。通过这样的分步处理，CAD-Editor能够更好地理解用户的意图，从而生成更高质量的CAD设计。
2025-02-06｜Xiaohongshu,SJTU｜⭐️
http ://arxiv.org/abs/2502.04326v1https ://jaaackhongggg.github.io/WorldSense
本文介绍了WorldSense，这是首个用于评估多模态视频理解的基准测试，涵盖了视觉、音频和文本输入。与现有基准相比，WorldSense具有几个显著特点：首先，强调多模态的协作，设计任务要求模型有效利用音频和视频的协同感知；其次，视频和任务的多样性，WorldSense包含1,662个音频-视频同步视频，系统分类为8个主要领域和67个细分子类，并提供3,172个多选问答对，覆盖26种不同任务；最后，高质量的注释，所有问答对都由80名专家进行手动标注，并经过多轮校正以确保质量。通过WorldSense，研究团队对多种最先进的模型进行了广泛评估，结果显示现有模型在理解现实场景方面面临重大挑战，最佳准确率仅为48.0%。希望WorldSense能为评估模型在构建和理解多模态上下文的能力提供一个平台。
WorldSense的设计基于三个关键原则：第一，多模态协作，强调音频和视觉信息的相互作用。每个问题的设计都要求同时使用两种模态，缺一不可，以确保模型能够同时处理多种感官输入。第二，视频和任务的多样性，WorldSense涵盖了1,662个音频-视频同步视频，分为8个主要领域和67个细分领域，提供3,172个多选问答对，涵盖从基础感知到复杂推理的26种任务，确保全面评估模型的多模态理解能力。第三，高质量的注释，所有问答对由80名专业注释员手动标注，并采用严格的质量控制流程，包括人工审核和自动化验证，以确保每个问题都具有清晰的表述和适当的难度，且确实需要多模态信息来回答。
在实验部分，研究团队对多种现有的多模态大语言模型（MLLMs）进行了全面评估，包括开源视频模型和专有模型。实验结果揭示了当前模型在理解和推理多模态信息方面的显著局限性。开源视频-音频模型的表现仅在25%左右，几乎接近随机猜测，而专有模型如Gemini1.5Pro在同时处理音频和视频时达到了最高的48%准确率，但当仅提供单一模态时，准确率显著下降约15%。此外，研究还进行了详细的消融研究，探讨不同模态对模型性能的影响，发现视觉信息在理解中起着重要作用，音频信息的加入（无论是原始信号还是转录字幕）均能提升结果。总体上，实验结果强调了多模态协作在现实理解中的重要性，并指出当前模型在多模态整合能力方面的显著不足。
在WorldSense的研究中，科学家们想要了解模型如何处理来自不同来源的信息，比如视频和声音。为了做到这一点，他们设计了一些问题，要求模型同时使用视频和声音的信息来找到正确答案。比如，问题可能会问视频中一个人正在做什么，而仅仅看视频可能无法得出答案，因为需要结合声音来理解场景。为了确保评估的全面性，研究团队收集了很多不同类型的视频，涵盖各种主题和情境，并且所有问题都经过专业人员的仔细审查，以确保它们既清晰又有挑战性。通过这种方式，研究者们希望能找到更好的方法来提高模型理解现实世界的能力。
2025-02-06｜YorkU,VectorInstitute,Harvard,FARAI｜⭐️
http ://arxiv.org/abs/2502.03714v1
在这项研究中，作者提出了一种新的框架——通用稀疏自编码器（UniversalSparseAutoencoders,USAEs），旨在从多种预训练深度神经网络（DNNs）中发现和对齐可解释的共同概念。与传统的单一模型概念解释方法不同，USAEs通过同时训练多个模型的激活，学习一个共享的、可解释的概念空间，从而捕捉不同任务、架构和数据集中的共同特征。该方法不仅增强了对模型内部表示的理解，还为多模型系统的风险管理和创新架构的开发提供了新的视角。此外，研究还展示了USAEs在“协调激活最大化”中的应用，揭示了不同模型如何编码相同概念的有趣方式。
USAEs的核心思想是训练一个单一的、过完备的稀疏自编码器，该自编码器能够同时处理来自多个模型的激活，并重构每个模型的激活。具体方法如下：
数据输入：将来自不同模型的激活数据汇总成一个矩阵。
稀疏编码：通过模型特定的编码器对每个模型的激活进行编码，形成共享的稀疏代码Z。
重构激活：通过各自的解码器从共享的稀疏代码中重构每个模型的激活，确保重构的激活尽可能接近原始激活。
训练目标：通过最小化重构误差的Frobenius范数来训练模型，确保所有模型的特征可以通过共享的稀疏概念进行准确恢复。
协调激活最大化：通过优化每个模型的输入，使得它们在共享的概念空间中最大化相同的概念，从而实现跨模型的概念可视化。
实验部分分为多个部分，首先介绍了实验的实现细节，然后进行了对USAEs发现的通用概念的定性分析。接下来，通过重构激活的R²评分进行定量评估，分析概念的普遍性和重要性，并考察USAEs与独立训练的稀疏自编码器（SAEs）之间的一致性。最后，通过协调激活最大化的应用，展示了不同模型如何以不同方式编码相同概念的可视化结果。实验结果显示，USAEs能够有效捕捉和重构不同模型之间的共享特征，并揭示了多种不同层次的概念，从基础颜色到复杂的对象部件。
在这项研究中，科学家们开发了一种新的工具，称为通用稀疏自编码器（USAEs），用来帮助我们理解不同人工智能模型如何看待和处理信息。想象一下，多个不同的AI模型就像是不同的艺术家，他们在画布上用自己的风格描绘同一幅画。USAEs就像一个指导老师，帮助这些艺术家们找到共同的元素，比如颜色、形状或主题。具体来说，研究者们首先将来自不同模型的信息汇集在一起，然后让这些模型通过一个共享的“概念字典”来学习和表达这些信息。这样，不同模型就能在同一个框架下进行交流，从而更好地理解彼此的表达方式。最后，他们还展示了一种新方法，可以让所有模型同时“激活”同一个概念，帮助我们更好地可视化和比较这些模型的表现。
2025-02-06｜UBristol,LeidenU,SMU,UBath｜⭐️
http ://arxiv.org/abs/2502.04144v1http ://hd-epic.github.io
HD-EPIC是一个全新的、详细注释的自我中心视频数据集，专注于厨房环境中的日常活动。该数据集包含41小时的多日无脚本视频，记录了69种食谱和59,000多种细粒度动作。每段视频均经过手动注释，涵盖了食谱步骤、成分及其营养价值、移动物体、音频事件和视线方向等信息。重要的是，所有注释都基于3D数字双胞胎模型，确保了高精度的空间定位。通过这些丰富的数据，HD-EPIC为视频理解模型提供了一个全新的挑战性基准，尤其是在视觉问答（VQA）任务上，展示了当前模型的不足。
HD-EPIC的数据收集和注释方法主要包括以下几个方面：
视频收集：参与者在自然环境中记录自己的厨房活动，使用多传感器平台（如ProjectAria眼镜）捕捉视频和音频，确保数据的多样性和真实性。
注释流程：
食谱和成分：视频中的每个食谱步骤和成分都被时间标注，并记录营养信息，参与者在准备食材时需称重并记录。
细粒度动作：通过参与者的叙述，详细记录每个动作的开始和结束时间，描述其执行的方式和原因。
数字双胞胎：为每个厨房场景创建3D模型，标注所有厨房设施，并结合参与者的视线方向来关联动作和物体。
对象和手部动作追踪：通过手动和自动化方式，记录每个物体的移动和手的操作，确保对每个物体的详细跟踪。
在实验部分，HD-EPIC数据集被用于一系列基准测试，评估当前模型在视频理解中的表现。首先，构建了一个包含26,000个问题的视觉问答基准，涵盖食谱、成分、营养、细粒度动作、3D感知、物体运动和视线等多个方面。实验结果显示，现有模型在这些任务上表现不佳，尤其是在复杂的长视频输入方面。此外，数据集还进行了动作识别、音频识别和长期视频对象分割的性能评估，结果表明当前模型在这些任务中的准确率仍有很大提升空间。
在HD-EPIC的研究中，数据的收集和注释过程可以简单理解为一个详细的记录和分析厨房活动的项目。参与者在家中自然地进行烹饪，并使用特殊的眼镜记录下整个过程。研究团队通过这些视频，仔细标注每一步做菜的细节，比如每种食材的重量、营养信息，以及每个动作的具体做法和原因。此外，他们还创建了一个虚拟的厨房模型，帮助理解在厨房中物体是如何移动和被使用的。这种方法不仅收集了丰富的数据，还确保了信息的准确性，使得未来的机器学习模型能够更好地理解和模拟人类的厨房活动。

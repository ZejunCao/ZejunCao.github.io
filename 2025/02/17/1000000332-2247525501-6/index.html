2025-02-05｜NUS,NTU,SkyworkAI｜⭐️⭐️
http ://arxiv.org/abs/2502.03465v1https ://github.com/Nut-World/NutWorld
在本研究中，我们提出了NutWorld，一个创新的框架，旨在高效地表示捕获的单目视频，通过动态高斯斑点（DynamicGaussianSplatting）在一个前馈过程中实现。NutWorld的核心是结构化的时空对齐高斯（STAG）表示法，这种方法允许在没有每场景优化的情况下，优化场景建模，并有效地处理深度和流动的正则化问题。与现有的2D和2.5D技术相比，NutWorld通过将单目视频视为动态三维世界的投影，克服了复杂运动、遮挡和几何一致性等挑战。通过全面的实验，我们证明NutWorld不仅能实现高保真的视频重建，而且能够在实时中支持多种下游应用，包括视频分割、深度估计和视频编辑。
NutWorld的实现包含三个关键组成部分：
结构化时空对齐高斯（STAG）表示：在规范空间中建立高斯表示，确保每个动态高斯与特定的像素位置和时间戳相对应，从而保持空间一致性。
无优化的动态封装：通过一个基于变换器的网络，NutWorld能够在单个前馈过程中将视频映射到STAG表示。该网络利用预先校准的深度和流动先验，避免了深度模糊和运动不确定性。
深度和流动正则化：通过结合校准的单目深度和光流先验，NutWorld解决了空间模糊和运动外观纠缠的问题。这种正则化不仅增强了模型的稳定性，还提高了重建质量。
我们在MiraData和RealEstate10K数据集上进行了大规模的实验，以验证NutWorld在视频重建和多种下游任务中的表现。实验结果表明，NutWorld在视频重建中实现了显著的性能提升，具备更高的PSNR和SSIM值，同时在推理速度上也表现优异，能够在1.8秒内重建90帧视频。此外，我们还进行了消融实验，验证了STAG表示、流动和深度正则化对模型性能的贡献，结果显示去掉任何一项都会显著降低重建质量。
在NutWorld的方法中，研究团队开发了一种新的方式来处理视频数据。首先，他们创建了一种叫做STAG的结构，这种结构帮助他们把视频中的每一帧与特定的时间和空间位置联系起来，就像给每个画面打上标签一样。接下来，他们使用了一种特殊的网络模型，这个模型可以在处理视频时，不需要每次都重新计算场景的细节，而是直接从已有的数据中提取信息。最后，他们还加入了一种技术来确保视频中的深度和运动信息保持一致，这样即使视频中的物体在移动，系统也能准确地理解它们的位置和变化。这种方法使得NutWorld能够快速而准确地重建视频，甚至在一些复杂的场景中也能保持高质量的表现。
2025-02-05｜UCLA,UUtah｜⭐️🟡
http ://arxiv.org/abs/2502.03449v1https ://dress-1-to-3.github.io/
在数字资产创建的领域，尤其是在虚拟现实、影视和游戏等应用中，生成逼真的3D衣物模型至关重要。传统的数字人类和服装创建流程繁琐且耗时，通常需要多个步骤，包括概念设计、材料选择、服装建模等。为了解决这一问题，本研究提出了“Dress-1-to-3”管道，它能够从单视图图像中生成可用于物理仿真的分离服装和人类模型。该方法结合了多视图扩散模型和物理仿真技术，确保生成的服装不仅在视觉上真实，而且在物理上可行，适用于动态服装动画和虚拟试衣等应用。
“Dress-1-to-3”管道的核心方法包括以下几个步骤：
服装模式生成：从单视图图像中提取初步的服装缝制模式，利用预训练的多视图扩散模型生成一致的多视图RGB图像和法线图。
人体姿态重建：使用生成的多视图图像来估计人体姿态和形状，以确保服装能够正确地适配到人体模型上。
可分离的服装重建：通过将2D缝制模式映射到3D人体模型上，生成可分离的、符合物理原则的服装。
物理仿真优化：利用差分物理仿真技术对服装进行动态仿真，优化其形状和物理参数，使其在运动中表现自然。
纹理生成：通过视觉语言模型生成服装的纹理，确保最终效果的真实感。
在实验部分，研究者对“Dress-1-to-3”管道进行了广泛的测试，以验证其有效性和通用性。实验结果表明，该方法能够成功重建各种类别的服装，甚至包括训练数据集中未出现的形状。通过与现有技术进行比较，展示了本方法在生成服装的几何对齐、物理真实性和视觉一致性方面的优势。同时，实验还评估了生成的服装在不同视角下的表现，确保其在动态场景中的适用性。通过这些实验，研究者证明了“Dress-1-to-3”方法在实际应用中的潜力。
在我们的研究中，首先，我们从一张普通的照片中获取服装的初步设计。这就像是从一张图纸上提取出衣服的缝制模式。接下来，我们使用一些先进的技术来生成这件衣服在不同角度下的样子，就好比我们用多个相机拍摄同一件衣服的不同侧面。然后，我们将这些设计应用到一个3D模型上，这个模型代表一个穿着这件衣服的人。最后，我们使用物理模拟技术来确保这件衣服在运动时看起来是自然的，就像真实的衣物一样。通过这种方式，我们不仅能创造出美观的服装，还能确保它们在虚拟环境中真实可用。
2025-02-05｜MSU｜⭐️🟡
http ://arxiv.org/abs/2502.02891v1
在本研究中，我们提出了一种名为INST-Sculpt的交互式神经签名距离函数（SDF）雕刻框架，旨在实现对3D形状的实时、基于笔划的编辑。传统的3D建模工具虽然在灵活性和直观性上表现良好，但在处理神经隐式表示时却面临着诸多挑战。通过允许用户定义自定义的笔刷轮廓并在神经SDF上进行笔划编辑，我们的框架提供了一种新颖的方式来实现复杂的形状修改。该方法利用了高效的管状取样技术，使得在用户定义的曲线沿线进行平滑、连续的变形成为可能，从而增强了对3D造型的控制和表现力。
我们的框架主要依赖于以下几个关键步骤：
用户定义的笔刷轮廓：用户可以通过选择控制点来定义自定义的笔刷轮廓，这些控制点在[-1,1]的窗口内通过三次样条插值进行平滑处理。
笔刷笔划的绘制：用户在交互式查看器中绘制笔划，系统会根据笔刷轮廓和笔划路径的不同对SDF进行相应的变形。
管状区域取样：在笔划路径周围建立一个移动的u-v-n坐标框架，进行管状取样，以确保在曲面上均匀应用笔刷效果。
多层感知机（MLP）微调：通过采样的点对神经SDF的MLP进行微调，以实现流畅的变形效果。
多视角编辑结果展示：最终的变形效果可从多个角度进行观察，确保视觉上的一致性和真实感。
我们在多个3D对象上评估了INST-Sculpt的性能，并将其与传统的点编辑方法以及基于网格的编辑进行比较。实验中使用了Chamfer距离作为主要的度量指标，以量化编辑前后几何形状的差异。我们还测试了不同采样点数量对编辑速度和质量的影响。结果表明，INST-Sculpt在编辑速度上相较于传统方法有显著提升，且在保持形状细节方面表现优异。通过对比，我们发现该框架在交互性和编辑效果上均超越了现有的3D雕刻工具。
在我们的研究中，我们开发了一种新的3D建模工具，它允许用户通过在模型表面上绘制笔划来进行编辑。想象一下，您正在用画笔在粘土上雕刻，您可以自由地设计笔刷的形状和大小。首先，您可以选择笔刷的形状，像是一个圆形、方形或其他自定义形状。然后，您在模型上绘制笔划，系统会自动根据您的笔划来改变模型的形状。为了确保每次编辑都能平滑自然，我们还设计了一种聪明的取样方法，这样即使在复杂的曲面上，笔刷的效果也能均匀分布。最终，您可以从不同的角度查看您修改后的模型，确保它看起来既真实又美观。这个工具极大地提高了3D建模的灵活性和创造力，让每个用户都能轻松实现他们的设计想法。
2025-02-01｜ShandongU,UGlasgow,ETHZurich,CrownCAD｜⭐️
http ://arxiv.org/abs/2502.02607v1
本研究提出了一种名为MIND（MicrostructureINverseDesign）的新型框架，旨在生成具有特定物理属性的三维可拼接微结构。该系统利用生成模型，结合潜在扩散技术和Holoplane混合神经表示，能够同时编码几何和物理属性，从而有效地优化微结构设计。与传统方法相比，MIND在设计灵活性和结构多样性上具有显著优势，能够生成多种类型的微结构，满足特定的物理性能要求。该框架不仅能处理复杂的组合设计空间，还能实现异构微结构的生成，进而推动新型超材料的开发。
MIND的核心方法包括以下几个步骤：
Holoplane表示：我们引入了一种新的混合神经表示——Holoplane，能够有效地捕捉微结构的几何对称性和物理属性。该表示形式结合了显式和隐式编码，确保几何和物理属性的精确对齐。
潜在扩散模型：通过潜在扩散模型生成Holoplane，这一过程允许在潜在空间中条件生成微结构，确保生成的微结构能够满足预定义的性能标准。
物理感知嵌入：在训练过程中，结合物理先验知识来增强模型的能力，使其能够更好地捕捉几何与物理之间的复杂关系。这种嵌入方法有助于提高生成微结构的物理属性匹配精度。
边界兼容性优化：在异构设计中，确保相邻微结构之间的边界兼容性至关重要。我们采用了边界兼容性损失函数，以引导生成过程，确保生成的微结构在接口处紧密结合。
为了验证MIND的有效性，我们构建了一个多类数据集，包含多种几何形态的微结构，包括桁架、管道、壳体和板结构。通过与现有方法的比较，实验结果表明，MIND在生成微结构的物理属性准确性和几何有效性方面具有显著优势。我们评估了生成微结构的性能，确保其符合目标物理属性，并且能够无缝集成到复杂的装配中。此外，我们还进行了打印可行性测试，结果显示生成的微结构在不同打印精度下均能保持良好的物理特性，证明了MIND在实际应用中的潜力。
MIND方法的核心在于它如何生成微结构。首先，我们使用一种叫做Holoplane的技术来表示微结构的形状和特性。这就像绘制一幅画，画中不仅包含形状，还包含颜色和纹理。接下来，我们利用潜在扩散模型，这是一种智能算法，可以根据我们想要的特性来“想象”出新的微结构。想象一下，如果你想要一块特定颜色的蛋糕，这个算法就会根据你的要求生成不同的蛋糕设计。为了确保这些设计不仅好看，还能在实际中使用，我们在模型的训练过程中加入了物理知识。这就像在制作蛋糕时，还要考虑到蛋糕的味道和口感。最后，我们还确保不同微结构之间的连接是顺畅的，这样它们在一起使用时不会出现缝隙或不匹配的问题。通过这些步骤，MIND能够创造出既美观又实用的微结构设计。

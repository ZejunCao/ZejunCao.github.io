2025-02-06｜CUHK,AdobeResearch,MonashU｜⭐️⭐️
http ://arxiv.org/abs/2502.04299v1https ://motion-canvas25.github.io/
本文提出了一种名为MotionCanvas的方法，旨在提升图像到视频生成（I2V）的能力，特别是在电影镜头设计方面。该方法允许用户以场景感知的方式控制摄像机运动和物体运动，从而实现更灵活的创作流程。传统的I2V生成系统常常面临两个主要挑战：一是如何有效捕捉用户在镜头设计中的意图，二是如何将这些运动信息转化为视频生成模型能够利用的信号。MotionCanvas通过结合经典计算机图形学的知识与现代视频生成技术，成功解决了这些问题。该系统能够在不依赖昂贵的3D训练数据的情况下，实现3D感知的运动控制，展示了其在数字内容创作中的潜力。
MotionCanvas的核心在于其三大模块：运动设计模块、运动信号翻译模块和运动条件视频生成模型。具体方法如下：
运动设计模块：用户可以通过输入静态图像作为运动设计的画布，捕捉用户的运动意图。该模块支持用户在3D场景中规划运动，并将其转化为屏幕空间的控制信号。
运动信号翻译模块：该模块负责将3D运动路径转化为2D点轨迹，以便视频生成模型可以理解。它通过随机抽样静态背景点，并结合深度信息来生成相应的屏幕空间轨迹。
运动条件视频生成模型：基于预训练的扩散模型，MotionCanvas将用户定义的运动信号与视频生成结合，生成符合用户意图的动态视频。该模型能够处理不同长度的视频生成，支持自动回归生成，确保视频的流畅性和自然过渡。
为了验证MotionCanvas的有效性，作者进行了多项实验，涵盖了不同的实际图像内容和镜头设计场景。实验结果表明，该方法在生成多样化、高质量视频方面表现优异，能够准确反映用户的运动意图。通过与现有的基线方法进行比较，MotionCanvas在摄像机和物体运动控制的精度和质量上均表现出显著优势。实验还包括用户研究，评估用户对运动一致性、运动质量和帧保真度的偏好，结果显示MotionCanvas在这些方面均优于其他方法，进一步证明了其在创意应用中的潜力。
MotionCanvas是一种新颖的工具，帮助用户将静态图像转化为动态视频。它的工作原理可以分为三个简单的步骤。首先，用户可以在一张图片上设计想要的运动，比如摄像机的移动或物体的动作。接着，MotionCanvas会将这些设计转换成计算机能够理解的信号，就像把3D动作变成2D的轨迹。最后，这些信号被输入到一个视频生成模型中，模型会根据这些信号生成出符合用户想法的动态视频。这样的流程使得用户可以非常直观地控制视频的每一个细节，创造出更具个性化和艺术感的作品。
2025-02-05｜UCLA,SnapInc.｜⭐️🟡
http ://arxiv.org/abs/2502.03639v1https ://snap-research.github.io/PointVidGen/
在视频生成领域，传统模型往往局限于2D图像的处理，缺乏对三维几何和动态的理解，这导致在复杂场景中生成的视频存在形状扭曲等问题。为了解决这一问题，本文提出了一种新颖的视频生成框架，通过结合三维点云的增强与正则化，提升视频生成的质量。该方法通过增强2D视频的三维点轨迹，并在像素空间中对其进行对齐，构建了一个名为PointVid的三维感知视频数据集。通过对该数据集进行微调，利用潜在扩散模型，本文的方法能够更好地追踪2D物体的三维坐标，并在视频生成过程中保持物体的形状和运动一致性，从而消除不必要的伪影。实验表明，所提出的模型在处理人手与物体交互等任务导向视频时，能够显著提高生成视频的质量和真实感。
本文提出的方法主要由三个关键部分构成：
PointVid数据集构建：首先，从标准的三通道视频中提取第一帧作为参考图像，并进行语义分割以获取前景物体的掩膜。然后，随机选择前景中的像素进行三维点跟踪，生成对应的三维坐标，并将其与视频数据对齐。
联合训练策略：在训练过程中，将RGB视频与三维点云数据结合，形成一个增强的视频点模型。通过调整UNet架构，使其能够同时处理视频与三维点云数据，促进对三维几何和动态的理解。
三维正则化策略：为了解决生成结果中的非物理伪影，本文引入了正则化项，通过对生成的点云进行约束，确保其在运动和形状上保持一致性。这一过程通过引入刚度损失和重建损失，增强了生成视频的空间一致性和时间平滑性。
为了验证所提出方法的有效性，研究团队收集了大约70K个视频样本，经过处理后形成448×256分辨率的训练数据集。实验结果显示，所提出的模型在视频生成质量、运动平滑性及物理一致性等指标上均优于基线模型I2VGen-XL。定性分析表明，模型在处理复杂的手-物体交互场景时，能够有效保持物体的形状和运动，消除形状扭曲的现象。此外，定量评估通过VBench和VideoPhy基准测试，进一步证明了模型在生成视频时的物理常识得分显著提高，展现了良好的时序一致性和物理合理性。
在这项研究中，研究者们提出了一种新方法，可以让计算机生成更自然、更真实的视频。为了实现这一目标，他们首先创建了一个新的数据集，里面包含了许多视频和相应的三维信息。接着，他们设计了一种训练方式，让计算机不仅能理解视频的2D部分，还能学习到物体在三维空间中的运动方式。这就像是给计算机装上了一双“3D眼睛”，能够看到物体的形状和位置，而不仅仅是平面的图像。此外，为了确保生成的视频看起来更真实，研究者们还引入了一些规则，帮助计算机在生成过程中保持物体的形状和运动的平滑性。最终，他们的实验结果显示，这种方法在生成视频时表现得更好，能够更自然地展现人们的手与物体之间的互动，减少了不自然的变形和其他问题。
2025-02-05｜WIS,PikaLabs｜⭐️🟡
http ://arxiv.org/abs/2502.03621v1https ://dynvfx.github.io
本文提出了一种新方法DynVFX，旨在通过用户提供的简单文本指令，将动态内容无缝添加到真实视频中。该方法能够在不需要复杂参考的情况下，合成与原视频场景自然交互的新动态对象或效果。通过利用预训练的文本到视频扩散模型和视觉语言模型，DynVFX能够在保持原视频完整性的同时，增强其表现力。研究团队的目标是实现一个完全自动化的框架，使得用户只需输入简单的编辑指令，即可生成与原视频相协调的高质量编辑效果。
DynVFX的方法主要包括以下几个关键组成部分：
视觉语言模型（VLM）作为VFX助手：通过VLM解析用户指令，生成详细的场景描述，确保新内容与原始视频的动态特性相一致。
锚定扩展注意力机制：在生成过程中，利用扩展注意力机制来引导新内容的定位，从而确保其与相机运动、遮挡和场景深度相匹配。这种方法允许在保持灵活性的同时，保留原始场景的关键空间线索。
内容和谐化的迭代优化：为了提高新内容与原视频的融合度，采用迭代更新的方式逐步减少每一步的噪声，从而优化新内容的细节和动态表现。
在实验部分，研究团队对DynVFX进行了广泛的评估，包括定性和定量分析。通过对18个公开视频的测试，比较了该方法与多个基线方法的效果。实验结果显示，DynVFX在视频编辑的保真度和内容整合方面表现优异，能够自然地将各种视觉效果融入到真实场景中。用户研究也表明，参与者普遍更倾向于选择DynVFX生成的视频，认为其在保留原始视频特征的同时，成功地整合了新内容。
DynVFX的工作原理可以简单理解为一个智能助手，它能够根据你说的话来修改视频。首先，你只需告诉它想在视频中添加什么，比如“加一只小狗”。接着，这个助手会分析原视频的内容，理解其中的运动和位置。然后，它会把小狗“放”到视频中，确保小狗的动作和位置看起来都很自然，就像它真的在那个场景里一样。此外，如果小狗的某些部分看起来不太对劲，助手会进行调整，确保它和视频的其他部分协调一致。这样，最终生成的视频就能完美地结合原有内容和新添加的元素。
2025-02-06｜PKU,PengChengLab｜⭐️
http ://arxiv.org/abs/2502.04076v1
随着新一代视频生成模型如Sora的出现，人工智能生成内容（AIGC）视频的质量评估面临着新的挑战。这些模型显著减少了先前模型中常见的闪烁伪影，并能够处理更复杂和多样的运动模式。然而，传统的视频质量评估（VQA）方法往往无法有效评估这些内容丰富的视频。为此，本文提出了CRAVE（内容丰富的AIGC视频评估器），专门用于评估Sora时代的AIGC视频。CRAVE通过多层次文本-时间融合技术，将复杂的文本语义与视频动态进行对齐，并结合混合运动保真建模来评估时间伪影。为了支持这一评估过程，研究团队还构建了CRAVE-DB，一个包含1228个复杂文本驱动视频的基准数据集。
CRAVE的评估过程从三个方面进行：
视觉和谐：借鉴传统VQA方法，CRAVE首先评估视频的美学和技术失真，确保视频在视觉上和谐。
多层次文本-时间融合：该模块结合了来自多模态理解的方法，通过交叉注意力机制将文本信息与视频视觉信息进行有效融合。它将文本分解为不同的粒度（如短语和词），以捕捉更细致的语义信息。
混合运动保真建模：该方法专注于捕捉AIGC视频中特有的运动失真，利用光流数据提取低级运动特征，并结合抽象的运动信息，以便更全面地评估视频的时间质量。
在实验部分，CRAVE在多个AIGCVQA基准上进行了广泛的测试，包括T2VQA-DB和CRAVE-DB。通过10折交叉验证，研究团队评估了模型的表现，结果显示CRAVE在多个指标上均取得了领先的表现。具体而言，实验结果表明CRAVE在主观评分上与人类感知高度一致，特别是在处理复杂的文本驱动视频时，展示了其强大的评估能力。此外，研究还进行了消融实验，验证了各个模块对整体性能的贡献，进一步巩固了CRAVE的有效性。
CRAVE的工作原理可以简单理解为一个“三步走”的评估流程。首先，它会检查视频的视觉效果，比如画面是否美观，是否有瑕疵。接着，CRAVE会把视频中的内容与描述这些内容的文字进行比对，确保视频中的场景和动作与文本描述一致。最后，CRAVE还会特别关注视频中的运动效果，比如是否有不自然的动作或失真现象。通过这三步，CRAVE能够全面评估视频的质量，确保生成的视频既美观又符合描述，使得观众能够获得更好的观看体验。
2025-02-06｜TeleAI,ChinaTelecom｜⭐️🟡
http ://arxiv.org/abs/2502.03897v2https ://uniform-t2av.github.io/
随着深度学习的迅速发展，音视频生成领域迎来了新的机遇。传统的生成方法通常将音频和视频的生成过程分开进行，这种独立的方法在生成质量和跨模态一致性方面存在不足。为了解决这一问题，本文提出了UniForm，一个统一的扩散变换器（DiffusionTransformer），旨在同时生成音频和视频，提升两者之间的内在关联性。UniForm通过在一个统一的潜在空间中同时处理音频和视频信息，学习生成高质量且相互匹配的音视频对。实验结果表明，该方法在多种生成任务中表现优异，包括文本到音频视频生成、音频到视频生成及视频到音频生成，展现了其在跨模态生成领域的强大潜力。
UniForm模型的设计基于扩散模型的原理，具体方法可以分为以下几个步骤：
潜在编码：首先，UniForm使用短时傅里叶变换（STFT）将音频信号转换为频谱图，同时通过3D变分自编码器（VAE）对视频帧进行编码。这样可以将音频和视频信息映射到一个共享的潜在空间中。
多任务嵌入：在训练过程中，UniForm引入任务标识符（tasktokens），以便模型能够识别当前所需生成的任务类型（例如，文本到音频视频、音频到视频或视频到音频）。这些标识符帮助模型更好地理解任务要求。
生成过程：通过扩散过程，UniForm在潜在空间中逐步生成音频和视频。模型使用自注意力机制来捕捉音频与视频之间的关系，并通过优化损失函数来提高生成质量。
解码与输出：生成的音频潜在表示通过预训练的解码器转换为音频波形，而视频潜在表示则被重构为视频帧，最终生成完整的音视频内容。
在实验部分，UniForm模型在多个数据集上进行了评估，包括AudioSet和VGGSound。实验结果显示，UniForm在音视频生成任务中表现出色，尤其是在生成质量和多模态对齐方面。具体而言，UniForm在音频到视频生成任务中实现了最低的FVD（FrechetVideoDistance）分数，表明其生成的视频质量高。通过对比多种基线方法，UniForm在视频生成的多样性和一致性上均优于其他方法。此外，模型在文本引导生成方面的表现也得到了显著提升，显示出文本信息对生成质量的积极影响。
UniForm模型的工作原理可以简单理解为一个“多合一”的音视频生成器。首先，它把音频和视频信息都转化为一种“内部语言”，让它们在同一个空间里交流。然后，在生成过程中，UniForm会根据不同的任务（比如根据文本生成音视频，或者从音频生成视频）来调整工作方式。通过这种方式，UniForm能够同时处理音频和视频，让两者之间的关系更加紧密，生成的内容也更加自然和协调。最终，UniForm会把这些“内部语言”翻译回来，变成我们可以听到的音频和看到的视频。这种方法使得生成的音视频质量更高，且彼此之间的配合更好。

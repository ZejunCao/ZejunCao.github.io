2025-02-05｜ZJU,AlibabaGroup,A*STAR｜⭐️🟡
http ://arxiv.org/abs/2502.03426v1
在现代时尚行业中，Pose-GuidedPersonImageSynthesis(PGPIS)技术的需求日益增加。该技术旨在生成在特定姿势下的个体图像，同时保持其身份特征，尤其是面部和服装的细节。然而，传统的基于扩散的PGPIS方法在保持源图像中的服装细节时常常面临挑战，尤其是在源姿势与目标姿势之间存在显著差异时。为了解决这一问题，本文提出了一种新的方法——人类解析引导的注意力扩散（TRUEPOSE），该方法通过引入人类解析图来有效地保持面部和服装的外观，生成高质量的图像。通过对比多种基线方法，实验结果表明，TRUEPOSE在保留面部特征和服装细节方面具有显著优势。
TRUEPOSE的核心是一个人类解析感知的Siamese网络，主要由三个关键组件构成：
双重UNet架构：包括TargetNet用于扩散去噪，SourceNet用于提取源图像嵌入。这种设计确保了源图像信息的有效传递。
人类解析引导的融合注意力（HPFA）：该模块根据源解析图的指导，调整TargetNet的注意力，确保生成图像中面部和服装区域的特征得以突出。
CLIP引导的注意力对齐（CAA）：通过结合不同语义区域的一致性约束，进一步精细化TargetNet的嵌入，确保生成图像的高质量和一致性。
这些模块可以插入到TargetNet的不同层次中，以增强生成图像的细节和一致性，特别是在复杂姿势转换的情况下。
为了验证所提出方法的有效性，研究团队在多个数据集上进行了广泛的实验，包括在店内服装检索基准和最新的野外人类编辑数据集。实验结果表明，TRUEPOSE在多项指标上均优于13种最先进的基线方法。具体而言，TRUEPOSE在保持面部外观和服装图案方面表现出色，尤其是在复杂的姿势转换场景中。通过定量和定性评估，研究结果显示该方法在生成高质量图像的同时，能够有效传递源图像中的细节，减少了对额外摄影的需求。
在我们的研究中，我们开发了一种新的技术，叫做TRUEPOSE，它可以帮助我们在不改变个人身份的情况下，将人们的姿势转换成不同的样子。想象一下，你有一张朋友穿着特定衣服的照片，但你想看他们在不同姿势下的样子。我们的技术可以做到这一点，同时保持他们的面部特征和衣服的细节。我们使用了一种特殊的网络结构，像双胞胎一样分别处理源图像和目标姿势，这样可以确保从源图像中提取到的信息不会丢失。此外，我们还引入了人类解析图，这是一种帮助我们关注图像中特定区域（如面部和衣服）的工具，确保生成的图像看起来自然且真实。通过这些方法，我们的技术在实验中表现出色，能够生成高质量的图像，帮助时尚行业更有效地展示产品。
2025-02-05｜AaltoU｜⭐️🟡
http ://arxiv.org/abs/2502.03330v1
本文提出了一种基于扩散模型的可控界面生成方法，旨在帮助设计师在早期阶段快速探索多种低保真界面设计。设计师通常需要在设计初期生成多种草图以探索设计空间，但现有工具往往要求输入过多细节，限制了创意的发挥。我们的模型允许设计师使用三种输入：文本提示、线框图和视觉流方向，灵活组合这些输入以生成多样化的设计方案。该方法的创新之处在于，它能在输入要求较低的情况下，快速生成大量设计样本，从而帮助设计师更有效地进行创意探索。
我们的方法基于扩散模型，并引入了两种适配器来控制界面的局部和全局特性。具体方法如下：
问题定义：将界面探索任务形式化为可控的界面生成问题，设计师可以通过文本提示、线框图和视觉流方向三种输入灵活引导生成过程。
局部属性控制：使用ControlNet适配器来管理局部属性，如元素的类型和位置，确保生成的界面与输入的线框图相一致。
全局属性控制：使用Flow适配器来指导整体视觉流，设计师可以提供样本界面或指定流向，以引导生成的界面设计。
训练过程：采用无分类器引导的损失函数和DDIM方法进行训练，确保生成的界面在多样性和与输入条件的对齐度之间取得平衡。
我们通过定性和定量的方式评估了模型的性能。定性实验展示了模型在不同输入条件下生成多样化界面的能力，包括文本提示、线框图和视觉流。定量实验则比较了模型与其他现有生成模型在输入对齐和输出质量上的表现。结果表明，本文提出的模型在生成的界面设计上更能准确反映输入条件，同时在效率上也表现出色，能够在短时间内生成大量设计样本。
在我们的研究中，我们提出了一种新方法，可以帮助设计师快速生成多种界面设计，减少他们在创意初期的工作量。设计师可以输入三种信息：一是描述他们想要的界面内容的文字，比如“一个鸡尾酒推广网站”；二是一个简单的线框图，表示界面的大致布局；三是希望用户注意的视觉流向，比如希望用户的视线从左下角移动到右下角。通过这些输入，模型能够快速生成多种不同风格的设计，帮助设计师探索更多创意，而不需要花费太多时间在细节上。这种方法不仅提高了设计效率，还能激发更多的创意灵感。
2025-02-05｜CMU,HKU,PKU,William&Mary,AMD,MBZUAI｜⭐️🟡
http ://arxiv.org/abs/2502.03444v1https ://github.com/Hhhhhhao/continuous_tokenizer
近期的研究表明，潜在扩散模型在高分辨率图像合成中表现出色，然而如何构建一个有效的潜在空间仍然是一个未被深入探讨的问题。本文提出了一种新方法——MAE-Tok（MaskedAutoencoderTokenizer），该方法利用自编码器和掩码建模技术，旨在学习一个语义丰富且结构良好的潜在空间，从而提升扩散模型的生成效果。通过实验，作者验证了潜在空间的结构对生成质量的重要性，并且发现与传统变分自编码器（VAE）相比，简单的自编码器（AE）在生成任务中表现出更高的效率和准确性。
MAE-Tok的设计主要包含以下几个关键部分：
架构设计：MAE-Tok基于1D自编码器架构，采用了视觉变换器（ViT）作为编码器和解码器。输入图像被分割成多个图像块，每个块被映射为嵌入向量，同时引入可学习的潜在标记。
掩码建模：在训练过程中，随机选择40%-60%的图像块进行掩码处理，目的是通过未掩码的块来重建掩码块，从而促使编码器学习到更具辨别性的潜在空间。
辅助解码器：通过引入浅层辅助解码器，MAE-Tok在重建过程中不仅关注像素级别的重建，还包括HOG、DINO和CLIP等特征的预测。这一设计使得潜在空间的学习更加全面和有效。
像素解码器微调：在训练完成后，冻结编码器，仅微调像素解码器，以提高重建的像素级别质量。这种策略可以在不牺牲潜在空间结构的情况下，恢复高质量的生成效果。
实验部分通过多种设置验证了MAE-Tok的有效性。首先，作者在256×256和512×512分辨率的ImageNet数据集上进行训练，比较了MAE-Tok与AE和VAE的性能。结果表明，MAE-Tok在生成FID（FréchetInceptionDistance）和重建FID方面均优于其他模型。此外，通过对掩码比率、重建目标和辅助解码器深度的消融实验，作者分析了这些设计选择对潜在空间学习和生成质量的影响。最终，MAE-Tok在生成任务中展示了显著的性能提升，证明了潜在空间结构的重要性。
MAE-Tok的方法可以简单理解为一个通过“掩码”学习的自编码器。想象一下，我们有一张图片，但我们把其中一部分遮住了。在训练过程中，模型的任务就是根据剩下的部分来猜测被遮住的部分是什么。这样，模型不仅要理解图像的整体结构，还要学习到每个部分之间的关系。此外，MAE-Tok还引入了一些额外的“老师”，比如一些特定的图像特征，让模型在学习时能够更全面。这种方法的好处在于，模型在生成新图像时能够更快、更准确，因为它已经掌握了如何从一个较小的、结构良好的信息中恢复出完整的图像。
2025-02-05｜KTH,UppsalaU｜⭐️🟡
http ://arxiv.org/abs/2502.03081v1
本研究探讨了如何利用人类对图像的感知对大脑信号进行视觉解码。通过对比传统的图像编码方法与人类对图像的感知对齐的编码方法，研究者们提出了人类对齐图像编码器的使用，旨在提高从大脑活动中提取图像的准确性。实验结果表明，采用人类对齐的图像编码器可以在图像检索准确性上提高多达21%，相较于当前最先进的解码方法。研究涵盖了多种脑电图（EEG）架构、图像编码器及对齐方法，验证了该方法在不同脑成像模态上的一致性表现提升。
本研究采用了一种基于人类对齐图像编码器的视觉解码框架，具体步骤如下：
脑信号编码：使用特定的脑信号编码器对来自EEG的多维时间序列数据进行处理，将其转化为嵌入表示。
图像编码：引入预训练的图像编码器，生成与图像相关的嵌入表示。研究中使用了多种视觉模型，包括与人类感知对齐的编码器。
对齐机制：通过对比损失函数（如InfoNCE），将脑信号嵌入与图像嵌入进行对齐，确保在共享的潜在空间中相似的图像和脑信号彼此靠近。
检索过程：在训练完成后，使用测试的脑信号嵌入计算与图像数据库中所有图像的距离，并根据距离进行排名，最终检索出最相关的图像。
实验部分包括多个阶段，首先评估了不同图像编码器和对齐方法对视觉解码性能的影响。研究者们使用了ThingsEEG2数据集，记录了10名参与者在快速视觉刺激下的脑电活动。通过对比人类对齐的图像编码器与未对齐编码器的性能，结果显示前者在图像检索任务中表现显著优于后者。随后，研究者们还探索了模型在不同参与者之间的泛化能力，发现人类对齐模型在新参与者的数据上也表现出更好的效果。此外，研究还扩展到其他脑成像模态（如MEG），进一步验证了人类对齐方法在视觉解码中的有效性。
在这项研究中，科学家们想要找到一种更好的方法来从我们的脑电波中识别我们看到的图像。为了实现这一目标，他们使用了两种主要的工具：一种是用来解码脑信号的设备，另一种是用来分析图像的设备。研究者们发现，如果将图像编码器与人类的感知能力对齐，那么从脑信号中提取图像的准确性会大幅提升。具体来说，他们训练了一个系统，让它学习人类是如何看待和理解图像的，然后再将这个知识应用到解码脑信号的过程中。通过这种方法，科学家们能够更准确地从大脑活动中找出我们看到的图像，这为未来的脑-计算机接口技术带来了新的可能性。
2025-02-05｜THU,Shengshu,PazhouLab｜ICLR2025｜⭐️🟡
http ://arxiv.org/abs/2502.02922v1
本文探讨了一种名为“Analytic-Precond”的新型预处理技术，旨在优化一致性蒸馏（consistencydistillation）中的模型训练，特别是针对扩散模型（diffusionmodels）。一致性蒸馏是一种通过训练学生模型在教师模型的概率流（PF）常微分方程（ODE）轨迹上进行反向遍历的方法。预处理技术通过线性组合输入数据和网络输出，构建一致性函数，从而稳定训练过程。然而，现有的预处理方法通常是手工设计的，可能存在次优选择。本文首次从理论角度分析了预处理的设计标准，并提出了Analytic-Precond，该方法能够根据一致性差距（即教师去噪器和最佳学生去噪器之间的差距）进行优化。通过此方法，学生模型的轨迹与教师模型的轨迹对齐，显著加速了训练过程。
本文提出的Analytic-Precond方法基于以下几个关键步骤：
分析一致性函数：首先，研究一致性函数在一致性轨迹模型（CTMs）中的形式，识别其参数化为跳跃连接的特性。确保一致性函数满足边界条件和与去噪器的对齐。
导出广义ODE形式：通过将教师ODE转换为广义形式，引入自由参数，生成新的预处理函数。这种方法能够推导出一系列满足约束的新预处理方法。
优化预处理：通过最小化一致性差距，提出了一种原则性的方法来分析和优化预处理。该方法不仅在训练过程中高效计算，还能显著提升学生模型的学习能力。
在实验部分，本文通过在多种标准数据集（如CIFAR-10、FFHQ和ImageNet）上验证了Analytic-Precond的有效性。主要实验目标包括：
训练加速：比较Analytic-Precond与原始预处理方法在单步和多步生成中的表现，观察其在训练时间上的提升，结果显示训练加速达2到3倍。
生成质量分析：通过不同步长的生成过程，评估生成样本的质量，分析与教师模型的轨迹对齐情况。实验结果表明，Analytic-Precond在保持生成质量的同时，显著改善了模型的收敛性和稳定性。
Analytic-Precond的核心思想是通过数学方法来优化模型训练，尤其是在生成图像时的速度和质量。首先，我们分析了如何将输入数据和模型的输出结合起来，确保模型在训练时能够遵循教师模型的轨迹。接着，我们引入了一种新的数学模型，使得我们可以灵活地调整模型的行为。最后，通过最小化模型之间的差距，我们能够高效地计算出最佳的预处理参数，从而在训练过程中加速学习。总之，这种方法不仅让模型训练变得更快，还提高了生成的图像质量。

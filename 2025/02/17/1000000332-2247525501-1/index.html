2025-02-06｜KAUST｜⭐️🟡
http ://arxiv.org/abs/2502.04050v1https ://partedit.github.io/PartEdit/
本文提出了一种基于预训练扩散模型的文本驱动图像编辑方法，称为PartEdit。该方法旨在解决现有扩散模型在细粒度编辑方面的不足，尤其是在对象部分的理解和编辑能力上。通过引入特定的文本标记，PartEdit能够更好地定位和编辑图像中的特定部分，而不是简单地对整个对象进行编辑。研究表明，该方法在多项指标上优于现有技术，并且在用户研究中获得了77%至90%的偏好支持。这一创新不仅提高了图像编辑的精确性和质量，还为创作者提供了更大的控制力。
PartEdit的核心在于其独特的标记优化过程，具体步骤如下：
学习部分标记：针对特定对象部分，收集相应的图像及其分割掩膜，使用预训练的扩散模型进行编码，并通过随机初始化的文本嵌入来优化标记。这些标记旨在生成与对象部分相对应的交叉注意力图。
特征混合和自适应阈值：在每个去噪步骤中，利用优化后的标记生成局部化掩膜。通过特征混合策略，将源图像和编辑图像的特征进行平滑融合，确保编辑的自然过渡。
时间步和UNet层选择：研究表明，选择中间时间步进行优化可以有效提高大部分的定位精度。同时，选择合适的UNet层以减少计算开销，从而在保证效果的同时提高效率。
为了验证PartEdit的有效性，研究者建立了一个包含多个对象部分的基准测试，并与现有的文本驱动编辑方法进行比较。实验结果显示，PartEdit在合成和真实图像编辑任务中均表现优异，特别是在定位精度和无编辑区域的保持上。用户研究结果表明，参与者更倾向于使用PartEdit进行细粒度编辑，表明其在用户体验上的优势。此外，研究还进行了消融研究，以探讨不同参数和策略对编辑效果的影响，进一步验证了方法的有效性和灵活性。
PartEdit的工作原理可以想象成给图像中的特定部分贴上标签。首先，我们收集关于每个部分的信息，比如一张汽车的图片和车身的轮廓。然后，我们通过一种特殊的学习方法，让计算机理解这些标签的意义。接下来，当我们想要编辑一张图片时，比如把汽车的颜色改成红色，PartEdit会在图片中找到对应的部分，并只对这个部分进行修改，而不会影响到其他区域。这个过程就像是用一个精准的画笔，只在想要的地方涂色，确保编辑后的图片看起来自然且美观。这样一来，用户就能更轻松地实现他们的创意想法，而不必担心整体效果受到影响。
2025-02-05｜UWashington｜⭐️🟡
http ://arxiv.org/abs/2502.03629v1
在当今数字时代，图像编辑需求日益增长，用户希望能够轻松地对照片进行修改和美化。然而，现有的图像编辑模型往往无法满足真实用户的多样化需求。为了解决这一问题，研究团队提出了REALEDIT，这是一个大规模的图像编辑数据集，来源于Reddit，包含真实用户的编辑请求及对应的人工编辑结果。REALEDIT数据集包含48,000个训练示例和9,300个测试示例，能够更好地反映用户在实际应用中的需求。通过在REALEDIT上进行微调，研究团队的模型在多个评估指标上超越了现有的最佳模型，展现了显著的性能提升，尤其是在用户主观评价中获得了更高的Elo分数。
研究团队的主要方法包括以下几个步骤：
数据集构建：从Reddit的两个主要子版块（r/PhotoshopRequest和r/restoration）收集用户的编辑请求和相应的人工编辑结果。通过对261,000个帖子和1.1百万条评论的分析，团队提取了编辑请求和输入图像，构建了一个包含57,000个编辑示例的数据集。
数据处理：使用视觉语言模型（VLM）对编辑指令进行处理，提取出关键的编辑需求，并通过GPT-4对请求进行简化，以去除噪声和无关信息。
模型训练：基于InstructPix2Pix模型，研究团队对其进行了微调，使用经过处理的REALEDIT数据集进行训练，确保模型能够更好地理解和执行用户的真实编辑请求。
评估机制：采用多种自动化评估指标（如VIEScore和VQAScore）来量化模型的表现，并进行人类评估以验证模型的实际应用效果。
在实验部分，研究团队将其模型与六个开源基线模型进行比较，使用REALEDIT数据集进行评估。结果显示，研究团队的模型在VIEScore等多个指标上均优于其他模型，特别是在语义一致性和视觉质量方面表现突出。通过在Reddit上实际部署模型，团队收到了用户的积极反馈，进一步验证了模型在实际应用中的有效性。此外，团队还与一家非营利组织合作，利用REALEDIT数据集提升了深度伪造检测模型的性能，显示了该数据集在广泛应用中的潜力。
在这个研究中，团队主要是想让图像编辑变得更加简单和有效。他们从Reddit上收集了很多用户的真实编辑请求，比如“把我的狗的眼睛修得更好”或者“把这张照片里的背景换掉”。然后，他们把这些请求整理成一个大数据集，里面有很多真实的编辑实例。接下来，团队使用一种叫做InstructPix2Pix的模型来学习如何根据这些请求进行编辑。他们还设计了多种评价方法，确保模型做出的修改不仅符合用户的要求，还能看起来非常自然。通过这些步骤，研究团队希望能够创造出一个更智能的图像编辑工具，帮助用户轻松实现他们的创意。
2025-02-06｜ZJU,UBuffalo｜⭐️🟡
http ://arxiv.org/abs/2502.03726v1
本文提出了一种新颖的方法，称为DICE（DistillingClassifier-FreeGuidanceintoTextEmbeddings），旨在通过优化文本嵌入来提升文本到图像生成的质量。尽管现有的文本到图像扩散模型能够生成高质量的图像，但这些图像常常无法与给定的文本提示紧密对齐。当前流行的分类器无关引导（CFG）技术虽能改善文本与图像的对齐，但其引入了显著的计算开销，并偏离了扩散模型的理论基础。DICE通过对文本嵌入的精炼，去除了对CFG的依赖，同时保留了其带来的好处，从而实现了高质量、对齐良好的图像生成，且采样速度显著加快。
DICE的核心在于增强文本嵌入，具体步骤如下：
文本嵌入编码：使用预训练的文本编码器将文本提示转换为嵌入。
轻量级增强器：设计一个独立于主文本到图像模型的轻量级增强器，该增强器通过训练来优化文本嵌入，目的是使其能够模拟CFG引导的方向。
优化过程：在训练过程中，使用CFG的监督信号来更新增强器的参数，确保生成的嵌入能够有效提升图像生成质量。
无引导采样：通过增强后的文本嵌入进行无引导采样，避免了CFG带来的额外计算开销，同时保持了生成图像的质量。
在多个文本到图像模型上进行了广泛的实验，包括StableDiffusionv1.5和PixArt-α。实验设置包括使用MS-COCO2017数据集进行训练和评估，评估指标包括FrechetInceptionDistance(FID)、CLIPScore(CS)和AestheticScore(AS)。结果表明，DICE在生成质量上与CFG引导的采样方法相当，但在计算效率上具有显著优势。此外，DICE还支持负采样，用于图像编辑和质量提升，展示了其在不同模型和数据集上的广泛适应性和有效性。
DICE方法的核心在于如何让计算机更好地理解文本并生成相应的图像。首先，计算机会把文本信息转化为一种“语言”，这种语言叫做嵌入。接着，我们设计了一个小工具，叫做增强器，它的任务是改进这些嵌入，使它们在生成图像时更有效。为了让增强器学习如何改进嵌入，我们使用了一些现有的高质量图像和文本对作为参考。最后，通过这些经过改进的嵌入，计算机可以快速生成高质量的图像，而不需要额外的复杂步骤，这样既节省了时间，也保持了图像的质量。
2025-02-06｜UCIrvine,ChanZuckerbergInitiative,UCI｜⭐️🟡
http ://arxiv.org/abs/2502.03686v1
在本文中，我们提出了一种名为DiffusionTrajectoryMatching(DTM)的新方法，旨在改善扩散模型的引导能力。扩散模型在生成样本质量方面表现出色，但现有的引导方法往往需要额外的模型训练或仅适用于特定任务。我们从变分推断和控制的角度重新审视扩散模型中的引导过程，将其建模为一个变分控制问题。DTM能够将多种引导方法统一在一个框架内，并且无需对预训练模型进行额外训练或修改。我们的实验结果表明，在多种线性和非线性逆问题上，DTM方法的性能优于现有的最先进方法，尤其是在图像去模糊任务中显著提高了样本质量。
我们的方法基于变分控制理论，具体步骤如下：
控制信号的引入：在每个时间步引入控制信号，旨在引导扩散过程朝向满足特定终端条件的目标状态。
优化过程：通过优化控制信号，以最小化引导轨迹与无引导轨迹之间的差异，同时确保生成样本尽量接近无条件样本的分布。
终端成本和瞬时成本：我们定义了终端成本，以确保最终状态满足外部约束，同时瞬时成本则用于限制引导轨迹的偏离程度。
无训练引导：DTM框架可以直接应用于预训练的扩散模型，而无需重新训练模型，从而实现更高的灵活性和效率。
我们在多个数据集上对DTM进行评估，包括FFHQ和ImageNet，重点关注非线性去模糊和线性逆问题（如超分辨率和随机填补）。实验结果显示，DTM在多个任务上均优于现有的基线方法。具体而言，在非线性去模糊任务中，DTM显著提升了生成图像的感知质量和细节保留能力。在定量评估中，我们使用了多种指标（如LPIPS、FID和KID）来衡量生成样本的质量，结果表明我们的模型在样本质量和计算效率上均有显著优势。
在我们的方法中，首先，我们在每个时间步引入一个控制信号，像是给扩散过程添加了一个“方向感”，引导它朝着我们想要的目标状态前进。接着，我们通过不断调整这个控制信号，确保生成的图像既要符合外部要求，又要尽量保持与未引导状态的相似性。这就像是在绘画时，既要遵循某种风格，又要尽量保持画布的原始特征。我们的框架可以直接使用已经训练好的模型，这样就省去了重新训练的麻烦，使得整个过程更加高效。通过这样的方式，我们的方法成功地提高了生成图像的质量，并在多个任务中表现出色。
2025-02-06｜UTokyo｜⭐️
http ://arxiv.org/abs/2502.03826v1
随着文本到图像（T2I）模型的迅速发展，生成内容的能力显著提高，但这些模型在训练过程中使用的大规模数据集常常包含固有的社会偏见。这些偏见不仅在生成的图像中得以反映，还可能加剧社会不平等。为了解决这一问题，本文提出了一种名为FairT2I的框架，利用大型语言模型（LLM）来检测和减轻T2I生成中的社会偏见。该框架由两个主要组件组成：首先是基于LLM的偏见检测模块，能够识别文本提示中潜在的社会偏见；其次是属性重新平衡模块，旨在微调模型中的敏感属性，从而减轻检测到的偏见。通过广泛的实验，FairT2I展示了在降低偏见的同时，仍能生成高质量的图像，推动了伦理内容创作的发展。
FairT2I的方法包括以下几个关键步骤：
潜变量指导：我们采用潜变量模型来描述图像生成过程，明确敏感属性如何影响最终结果。通过这种方式，可以直接控制生成的图像所反映的社会偏见。
LLM辅助偏见检测：利用大型语言模型动态识别输入提示中的偏见，超越静态的预定义敏感属性集。这种方法允许更广泛的输入提示处理，并能检测人类观察者难以察觉的偏见。
属性重新平衡：我们引入了基于Boltzmann分布的技术，重新平衡敏感属性的分布，以促进公平的图像生成。通过调整温度参数，可以在生成过程中实现多样性，确保生成的图像更符合现实世界的分布。
在实验中，我们使用了StableDiffusion3.5-large作为文本到图像模型，并采用GPT-4o进行偏见检测。我们评估了FairT2I在两个数据集上的表现：StableBiasProfessionDataset和PartiPromptDataset。实验结果显示，FairT2I在生成的图像质量、提示遵循性和多样性方面均表现出色。通过人类评估，我们发现与基线方法相比，FairT2I显著提高了生成图像的多样性，并且在社会偏见的表现上有所改善。此外，利用非参数评估方法，我们能够更深入地分析不同方法在生成过程中对人口统计特征的表现，从而验证FairT2I在减轻偏见方面的有效性。
FairT2I的工作原理可以简单理解为：首先，它使用一种强大的语言模型来识别文本中的潜在偏见，比如某些性别或种族的刻板印象。接着，FairT2I会根据这些偏见调整生成图像的方式，确保不同的性别、种族或其他敏感属性在生成的图像中能够公平地展现。想象一下，如果你输入“一个程序员的肖像”，FairT2I不仅会生成高质量的图像，还会确保这个程序员的性别和种族多样化，而不是只显示一种类型的人。通过这种方式，FairT2I帮助我们创造出更公平、更具包容性的图像内容。

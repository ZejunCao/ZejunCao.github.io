2025-02-13｜KAIST｜⭐️
http ://arxiv.org/abs/2502.08914v1https ://power.lowyinstitute.org
本研究探讨了文本到图像（T2I）扩散模型在生成文化特定图像方面的能力，特别是它们在不同文化背景下的表现。尽管这些模型在生成高质量图像方面取得了显著进展，但其在地理和文化包容性方面的表现仍然存在不足。我们提出了CULTDIFF基准测试，以评估当前最先进的扩散模型在生成涵盖十个国家的文化特定图像的能力。研究结果显示，这些模型在生成特定文化的建筑、服装和食物等文化物品时，往往未能准确反映文化细节，尤其是在代表性不足的国家和地区。
我们的方法包括以下几个关键步骤：
数据集构建：
选择十个国家，涵盖不同的资源水平。
针对建筑、服装和食物三类文化物品，生成50个与每个国家相关的提示。
从Wikipedia和文化遗产网站收集真实图像，确保数据的多样性与代表性。
图像生成：
使用三种不同的扩散模型（StableDiffusionXL、StableDiffusion3Medium、FLUX）根据生成的提示生成图像。
为每个文化物品收集多张真实图像，以减少因环境因素造成的偏差。
人类评估：
通过问卷调查，招募来自十个国家的评估者，评估生成图像的相似性、描述匹配度和现实感。
评估使用Likert量表进行，确保评估的可靠性和有效性。
相似度度量：
开发CULTDIFF-S相似度度量，通过对比真实和生成图像，评估文化和上下文的相关性。
在实验中，我们首先创建了一个包含150幅图像的调查，涵盖建筑、服装和食物三大类。每类图像包括三幅真实图像和一幅AI生成图像。评估者根据图像的相似性和描述匹配度进行评分。结果显示，扩散模型在高资源国家（如美国和英国）生成的图像表现较好，而在低资源国家（如埃塞俄比亚和阿塞拜疆）的表现则较差。此外，通过对比不同模型的输出，我们发现CULTDIFF-S度量在评估文化相关性方面的表现优于传统的相似度度量方法。
在我们的研究中，我们的目标是看看一些先进的图像生成模型能否准确地反映不同文化的特色。我们首先选择了十个国家，并为每个国家创建了一些与建筑、服装和食物相关的提示。接下来，我们使用三种不同的计算机模型，根据这些提示生成图像。同时，我们也从网上收集了真实的文化图像，以便进行比较。为了确保我们的评估结果可靠，我们邀请了来自不同国家的评估者来判断这些生成图像与真实图像之间的相似性。最后，我们还开发了一种新方法来衡量这些图像的文化相关性，发现一些模型在表现上存在较大差异，特别是在一些文化代表性不足的国家。
2025-02-13｜UWA,UMelbourne,CurtinU｜⭐️
http ://arxiv.org/abs/2502.08927v1
随着深度生成模型的发展，尤其是文本到图像的扩散模型，视觉内容的生成变得更加真实和可及。然而，这种技术的广泛应用也引发了诸多伦理问题，包括知识产权保护和合成媒体的误用。为了解决这些挑战，本文提出了一种新颖的多阶段水印框架，旨在为扩散模型嵌入水印，以便确立版权并追溯生成图像的来源。该框架包括固定水印和动态水印的双重嵌入机制，确保在保持图像质量的同时，增强了对生成内容的追踪能力。
本研究提出的双重水印嵌入方法分为两个主要分支：模型水印和图像水印。具体方法如下：
模型水印：嵌入一个固定的二维码水印到扩散模型的噪声分布中，确保模型所有权的验证。
通过修改高斯核实现水印嵌入。
使用触发密钥来调整扩散路径，使水印与生成内容相结合。
图像水印：在生成的图像中嵌入动态水印，确保水印的不可感知性和对攻击的韧性。
在特征空间中，通过计算余弦相似度来调整水印的形状和颜色。
在像素空间中，利用结构相似度指数（SSIM）来保持图像的视觉质量。
这些方法结合使用，使得生成的图像不仅能够有效防止未经授权的使用，还能在遭受攻击时保持较高的可识别性。
为了验证所提出方法的有效性，研究团队进行了多项实验。首先，通过提取模型水印和图像水印，评估了水印的可视性和结构完整性。实验结果显示，提取的二维码水印在视觉质量上保持了较高的PSNR和SSIM值，表明水印的有效性。其次，针对不同类型的攻击（如旋转、模糊和压缩），评估了水印的鲁棒性。结果表明，在大多数情况下，水印的存在和分类准确率均高于90%。此外，通过统计分析，水印对生成图像质量的影响较小，变化在5%以内，确保了水印的隐蔽性和有效性。
在这项研究中，我们提出了一种新颖的方法来保护生成的图像版权。我们的方法可以分为两个部分：首先，我们在生成模型内部嵌入一个固定的二维码水印，这就像给模型打上了一个独特的标签，确保我们知道这个模型是谁的。其次，我们在每一张生成的图片中加入一个动态的水印，这个水印会根据图片的内容变化，但人眼几乎看不见。这样做的好处是，即使有人试图修改或盗用这些图片，我们仍然可以通过这些水印追踪到它们的来源。此外，我们还进行了多次实验，验证了这些水印在不同情况下的有效性，确保它们不仅难以被去除，还能在图片质量上几乎没有影响。
2025-02-13｜WHU,SYSU,ZJU｜⭐️
http ://arxiv.org/abs/2502.09434v1https ://github.com/liuxiao-guan/IET_AGC
本研究探讨了扩散模型在生成高质量图像时可能出现的数据记忆问题，特别是其对隐私的潜在威胁。尽管现有的记忆缓解方法主要集中在文本生成任务中，但它们的适用性受限于特定条件。因此，本文提出了一种新颖的方法，专注于视觉模态，以更普遍的方式解决记忆问题。通过将训练数据划分为多个数据片段并训练多个代理模型，然后将这些模型的参数聚合以形成最终模型，我们能够有效减少模型的记忆能力。实验结果表明，该方法在降低记忆分数方面取得了显著成效，同时保持了生成图像的质量。
本研究提出的方法主要包含两个关键模块：迭代集成训练（IET）和反梯度控制（AGC）。具体步骤如下：
迭代集成训练（IET）：
将训练数据分为多个片段，每个片段训练一个代理模型。
在每个训练阶段后，聚合所有代理模型的参数，形成一个新的模型，以减少直接记忆的风险。
反梯度控制（AGC）：
通过跟踪样本的训练损失，识别那些损失值异常低的样本，这些样本更容易被记忆。
对于损失值低于设定阈值的样本，选择跳过这些样本以避免记忆。
采用动态增强策略，针对那些未被跳过但损失接近阈值的样本，进行数据增强以提高其多样性，从而进一步降低记忆的风险。
此外，引入了记忆样本重分配（MSR）策略，以确保那些容易被记忆的样本在不同的片段之间进行交换，从而避免过度跳过。
在多个数据集上进行的实验表明，该方法在降低记忆能力的同时维持了生成图像的质量。具体来说，在CIFAR-10、CIFAR-100和AFHQ-DOG数据集上，与默认训练方法相比，记忆图像的数量分别减少了90.1%、74.6%和91.2%。在对预训练的扩散模型进行微调时，记忆分数降低了46.7%。实验结果还表明，采用该方法生成的图像在多样性和质量上均有所提升，且在文本与图像的对齐度上表现良好。
我们的方法可以简单理解为通过分组训练来减少模型的记忆能力。首先，我们将所有的训练数据分成几个小组，每个小组训练一个小的模型。这样做的好处是，每个小模型只看到自己小组的数据，从而减少了记忆的机会。然后，我们会定期将这些小模型的学习成果汇总成一个更大的模型。为了进一步防止模型记住某些特定的数据，我们还会监控每个样本的学习情况。如果某个样本的学习损失值过低，说明它很可能被模型记住，我们就会选择跳过这个样本。对于那些损失值接近跳过阈值的样本，我们会进行一些变化，如增强图像的多样性，从而降低它们被记住的风险。通过这种方式，我们能够有效减少模型的记忆，同时生成高质量和多样化的图像。
2025-02-13｜NUAA,CUHK,JUFE｜⭐️
http ://arxiv.org/abs/2502.08921v1
在文本到图像生成领域，尽管技术取得了显著进展，但恶意概念的上传和分享也带来了安全隐患。恶意概念可能被伪装成无害的内容，诱使用户下载并生成有害图像。为了解决这一问题，研究者提出了一种新颖的方法——ConceptQuickLook，旨在无需生成图像即可快速检测恶意概念。该方法通过分析概念文件的内容，利用特定的检测模式来判断其是否存在恶意。通过大量实验，ConceptQuickLook展示了其在概念分享平台中有效识别恶意内容的潜力，进而保护用户和平台的信誉。
ConceptQuickLook的核心在于其检测流程，主要分为两个工作模式：概念匹配和模糊检测。具体方法如下：
概念匹配：该模式通过比较输入概念的嵌入向量与已知概念类的嵌入向量，判断其是否属于某个已知的概念类。系统会输出与输入概念相匹配的文本描述，帮助用户理解概念的性质。
模糊检测：在缺乏明确文本描述和示例图的情况下，该模式通过计算概念嵌入向量之间的距离，判断输入概念是否属于某个特定的概念类。这种方法尤其适用于无法直接识别的概念，提供了一种灵活的检测手段。
这两种模式结合使用，确保了在不同情况下都能有效识别恶意概念，减少用户接触有害内容的风险。
在实验部分，研究者进行了全面的评估，以验证ConceptQuickLook的有效性。实验包括：
数据集构建：从多个概念分享平台下载概念文件，标注其内容，并生成相应的训练数据集。
性能评估：使用准确率、假阳性率、假阴性率等指标评估模型的性能。实验结果显示，ConceptQuickLook在恶意概念检测中表现优异，准确率达到97.82%，并且在处理未见过的概念时仍保持良好的性能。
用户评估：通过问卷调查收集用户对检测结果的反馈，进一步验证模型的实用性和用户满意度。
实验结果表明，ConceptQuickLook不仅能快速识别恶意概念，还在资源消耗上表现出色，适合在实际应用中推广。
ConceptQuickLook的工作原理可以想象成一个智能的“安全检查员”。当用户想要下载一个新概念时，ConceptQuickLook会先检查这个概念的“身份证”，也就是它的嵌入向量。这个向量就像是概念的指纹，可以帮助系统判断这个概念是否安全。系统主要有两种检查方式：第一种是“概念匹配”，就像在找一个熟悉的朋友，系统会比对这个新概念的指纹和已有的安全概念指纹，看看它是否属于安全的类别。第二种是“模糊检测”，当没有明确的指纹时，系统会通过计算距离来判断这个概念是否与某个已知的安全概念相似。通过这两种方式，ConceptQuickLook可以快速而准确地判断出一个概念是否可能是恶意的，从而保护用户不被有害内容所伤害。
2025-02-12｜FIU｜IJCAI2025｜⭐️
http ://arxiv.org/abs/2502.08821v1https ://github.com/Noodulz/dejAIvu
随着生成性人工智能模型的发展，AI生成图像在网络上的使用迅速增加，这引发了对信息真实性、数字伪造和艺术归属等问题的广泛关注。现有的AI图像检测方法往往存在不够透明的问题，用户难以理解为何某些图像被判定为AI生成。为了解决这一问题，本文提出了DejAIvu，这是一款基于Chrome浏览器的扩展程序，能够实时检测AI生成的图像，并通过显著性图（saliencymaps）提供可解释性，从而增强用户的信任感。DejAIvu不仅可以在用户浏览常见图片分享网站时自动分析图像，还能在保持用户隐私的同时，确保低延迟的性能。
DejAIvu的工作流程包括以下几个步骤：
图像拦截：扩展程序在用户浏览网页时，动态识别网页中的图像元素（标签）。
预处理与模型推理：将拦截到的图像调整为256×256像素，进行标准化处理，并输入到经过ONNX优化的模型中进行分类。
显著性热图生成：如果模型判断图像为AI生成，则计算并生成一个基于梯度的显著性图，以突出图像中与AI生成相关的特征。
可视化叠加：将显著性图实时叠加在原始图像上，使用户能够直观地理解哪些特征影响了AI生成的判定。
该模型在一个包含270,000多幅人类和AI生成艺术作品的精心策划的数据集上进行了训练，以提高检测的准确性和鲁棒性。
为了评估DejAIvu的性能，研究团队对其检测模型进行了多种主流架构的比较，评估了准确性、精确度、召回率、损失和模型大小等指标。结果显示，ResNet50模型在准确性上达到了97.1%，在保持合理文件大小的同时，显著提高了检测效果。此外，使用ONNX.js进行模型转换，较TensorFlow.js减少了约35毫秒的延迟，确保了实时推理性能。通过对多个数据集的验证，DejAIvu展示了其在浏览器内的高效性和实用性，能够有效应对AI生成内容的检测挑战。
DejAIvu的工作方式可以简单理解为一个智能助手，它在你浏览网页时悄悄地监视所有图片。当你查看图片时，这个助手会自动识别出哪些是AI生成的，并且用一种特殊的色彩图层来标记这些图片。这个色彩图层显示了哪些部分的图像让它认为是AI制作的，帮助你理解这些图片的来源。整个过程是自动进行的，你无需手动上传任何图片，助手会在后台默默工作，确保你的隐私不被侵犯。这样一来，无论你是在寻找艺术作品还是其他类型的图片，DejAIvu都能帮助你轻松辨别出哪些是由人工智能生成的内容。
2025-02-13｜HUJI｜⭐️
http ://arxiv.org/abs/2502.09611v1
在条件生成模型的研究中，流基生成模型因其在文本到图像生成等任务中的优异表现而受到关注。然而，现有方法往往将一般的单峰噪声分布转换为目标数据分布的特定模式，导致训练和推理效率低下。本研究提出了一种新颖的方法，旨在通过构造条件特定的先验分布来改进流基生成模型的性能。通过更精确地建模先验分布，我们能够实现更高效的条件生成，从而在训练和推理过程中减少采样步骤并提高生成样本的质量。
本研究的方法主要包括以下几个步骤：
条件映射：给定输入条件（如文本提示），我们首先将其映射到数据空间中的一个点，代表与同一条件模式的所有数据点的“平均”数据点。
流匹配：利用流匹配公式，将样本从一个以该点为中心的参数化分布映射到条件目标分布。我们选择高斯混合模型（GMM）作为先验分布，确保每个模式与特定的条件分布相关联。
训练与推理：在训练过程中，我们通过优化目标函数来拟合先验分布，以减少源分布与目标分布之间的平均距离，从而提高生成效率。在推理时，使用与训练相同的先验统计特征，以确保生成样本的质量。
我们在多个数据集上进行了实验，以验证所提出方法的有效性。首先，在一个简单的二维玩具示例中，我们展示了与传统方法相比，使用条件特定先验分布的优势。随后，我们在真实数据集上进行了评估，包括MS-COCO（文本到图像生成）和ImageNet-64（类条件生成）。实验结果表明，我们的方法显著提高了训练速度和生成样本的质量，特别是在较少的采样步骤下，FID、KID和CLIP得分均优于基线方法。此外，我们还展示了模型对未见类的良好泛化能力，表明该方法具有广泛的应用潜力。
在我们的研究中，我们提出了一种新方法来帮助生成图像，这种方法特别关注如何更好地处理输入条件，比如一段描述或一个类别。首先，我们将这些条件转换成一个具体的点，这个点代表所有与之相关的图像的“平均”特征。接下来，我们使用一种叫做流匹配的技术，将这个点周围的样本映射到我们想要生成的图像上。这样做的好处是，我们能够更快速地生成高质量的图像，因为我们减少了在生成过程中需要的步骤。最后，我们在多个数据集上测试了这种方法，结果显示它比传统方法更快、更准确，能够生成更好的图像。

VariationalRectifiedFlowMatching（VRFM）是一种新颖的框架，旨在改进传统的流匹配方法，通过引入多模态速度场来捕捉数据中的复杂性。在经典的流匹配中，样本通过解常微分方程（ODE）从源分布转移到目标分布，然而，这种方法在面对多模态数据时，常常无法有效捕捉样本之间的复杂关系。VRFM通过引入潜在变量，允许在每个数据位置建模多种流动方向，从而克服了经典方法的局限性。研究表明，VRFM在合成数据集、MNIST、CIFAR-10和ImageNet等多个数据集上表现出色，能够生成更为真实且多样化的样本。
VRFM的核心在于如何有效地学习和利用速度向量场，以便在数据域时间域中捕捉多模态性。其方法步骤如下：
潜在变量引入：通过引入潜在变量z，允许模型对速度向量场进行多模态建模。潜在变量的使用帮助解耦不同的流动方向。
变分推断：在训练过程中，使用变分推断来近似速度场的后验分布。具体而言，模型通过一个编码器q(z|x0,x1,xt,t)来获得潜在变量的样本。
速度场学习：通过最大化对数似然函数，模型学习速度向量场v(x,t,z)，并通过最小化与真实速度场的差异来优化参数。
ODE积分：在推断过程中，利用学习到的速度场通过ODE从源分布样本x0推导到目标分布样本xt。
在实验部分，研究团队在多个数据集上评估了VRFM的有效性。在合成一维数据实验中，VRFM成功捕捉到多模态速度分布，相比于经典的流匹配方法，生成的样本在质量上有显著提升。对于二维合成数据，VRFM展示了更好的流动轨迹，能够处理复杂的样本间关系。在MNIST和CIFAR-10数据集上，VRFM不仅在生成图像的质量上超越了基线模型，同时还在控制性和可解释性方面表现优异。最后，在ImageNet数据集上，VRFM同样取得了令人满意的FID得分，进一步证明了其强大的生成能力。
在VRFM方法中，我们通过引入一个潜在变量，使得模型能够在同一个位置上考虑多种不同的流动方向。简单来说，想象一下在一个城市中，交通流量可以向多个方向分散，而我们的模型就像是一个交通指挥系统，能够根据当前的交通状况灵活调整流向。我们首先通过一个编码器来获取这些不同流向的“交通信息”，然后利用这些信息来训练一个模型，使其能够在不同的时间点上准确预测车辆的流动情况。通过这种方式，VRFM不仅能更好地理解复杂的交通模式，还能生成更为真实的交通流动图像，帮助我们在实际应用中更好地处理多模态数据。

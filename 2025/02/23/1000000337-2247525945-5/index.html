主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）发布日期｜2025年2月13日｜周四
2025-02-12｜Tel-AvivU,ReichmanU,MIT｜⭐️⭐️
http ://arxiv.org/abs/2502.08642v1https ://swiftsketch.github.io/
SwiftSketch是一种基于扩散模型的图像到矢量草图生成方法，能够在不到一秒的时间内生成高质量的矢量草图。尽管近年来在大规模视觉-语言模型（VLMs）方面取得了显著进展，现有技术仍依赖于耗时的优化过程，限制了其实用性。SwiftSketch通过逐步去噪高斯分布中的笔画控制点，克服了这一限制。该方法采用变换器-解码器架构，能够有效处理矢量表示的离散特性，并捕捉笔画之间的全局依赖关系。此外，为了应对缺乏专业质量配对矢量草图数据集的挑战，研究团队构建了一个合成数据集，包含超过35,000个高质量矢量草图，涵盖100个类别。
SwiftSketch的方法主要由三个关键组成部分构成：
ControlSketch：一种基于优化的技术，用于生成高质量的矢量草图。通过结合深度控制网络，ControlSketch能够在生成过程中提供精确的空间控制。
合成配对图像-草图数据集：利用ControlSketch生成的图像和对应的矢量草图，构建了一个包含35,000个样本的合成数据集，以支持模型训练。
SwiftSketch模型：采用变换器-解码器架构，学习从输入图像生成矢量草图。模型通过逐步去噪高斯噪声，结合图像嵌入和时间步长信息，生成最终的草图。
在实验部分，SwiftSketch的性能通过与其他最先进的方法进行比较来评估。实验结果显示，SwiftSketch在生成草图的速度和质量上均优于现有技术，能够在不到0.5秒的时间内生成高质量的矢量草图。研究团队还通过定量评估和用户研究，验证了SwiftSketch在不同类别图像上的泛化能力，尤其是在处理未见过的类别时，尽管性能有所下降，但仍然能够捕捉到输入图像的基本特征。
在SwiftSketch的方法中，主要有三个步骤。首先，ControlSketch技术帮助我们从输入图像中生成草图。它通过分析图像深度和注意力图，确保草图能够准确反映图像的主要特征。接着，我们创建了一个包含大量图像和草图的合成数据集，以便训练我们的模型。最后，SwiftSketch模型使用了一个智能的变换器架构，它能通过逐步清理噪声来生成草图。这个过程就像在画布上慢慢描绘出一个图案，直到最终的草图清晰可见。这样，SwiftSketch不仅能快速生成草图，还能确保它们的质量和准确性。
2025-02-05｜HUST,JD.COM,SYSU｜WWW2025｜⭐️🟡
http ://arxiv.org/abs/2502.06823v1https ://github.com/Chenguoz/CAIG
在数字广告中，广告图像的生成对用户的吸引力和广告效果至关重要。现有的广告图像生成方法大多集中于美学质量，未能有效提升在线点击率（CTR）。为了解决这一问题，研究者们提出了一种基于多模态大型语言模型（MLLM）的广告图像生成方法，称为CTR驱动的广告图像生成（CAIG）。该方法通过优化CTR作为主要目标，结合多模态特征，确保生成的背景内容与产品特性相一致，从而提高广告的相关性和有效性。研究表明，该方法在多个在线和离线指标上达到了先进的性能。
本研究的方法分为几个关键步骤。首先，采用大规模的电子商务多模态数据集进行针对性的预训练，以增强MLLM对电子商务领域的理解。具体步骤包括：
图像理解：根据产品图像描述产品或背景。
多模态内容理解：利用产品信息（如标题、类别、标签）生成产品背景描述。
提示生成：基于多模态产品信息生成或重写描述提示。
其次，构建了一个基于MLLM的奖励模型（RM），用于模拟用户偏好，并通过强化学习（RL）算法对生成模型进行微调。该RM通过对比生成的图像对的CTR，优化生成过程。最后，采用产品中心的偏好优化策略（PCPO），确保生成的背景与产品特性一致，从而提高CTR。
在实验部分，研究者们使用了公共和商业数据集对所提出的方法进行了评估。公共数据集包含500K个产品样本和1.2M个独特广告图像，而商业数据集则包含1M个产品样本和3.4M个广告图像。通过对比不同模型的表现，研究者发现所提出的CAIG方法在多个关键指标上均显著优于现有的基线模型。此外，实验还表明，采用PCPO策略可以有效提高生成图像的相关性和CTR，进一步增强广告效果。
在这个研究中，科学家们想要创造更吸引人的广告图像，以提高用户的点击率。他们使用了一种叫做“多模态大型语言模型”的技术，这种技术能够理解和处理不同类型的信息，比如文字和图像。研究的第一步是让模型学习如何描述产品和它们的背景，比如一双鞋子需要一个运动感的背景。接下来，他们建立了一个“奖励模型”，这个模型会根据用户的反馈来评估生成的图像是否吸引人。最后，他们使用了一种特别的方法，确保生成的背景与产品的特性相匹配，从而提高广告的点击率。总的来说，这个研究的目标是让广告图像更具吸引力，从而帮助商家获得更多的点击和销售。
2025-02-11｜CSU,NUC,Microsoft,NUS｜⭐️🟡
http ://arxiv.org/abs/2502.07870v1https ://textatlas5m.github.io
TextAtlas5M是一个新推出的大规模数据集，旨在解决长文本图像生成中的挑战。随着文本条件图像生成技术的不断进步，生成包含长文本的图像仍然面临困难。现有的数据集通常专注于短文本，缺乏对复杂长文本的支持。TextAtlas5M包含500万张图像，涵盖多种数据类型，包括真实和合成图像，旨在为文本条件图像生成模型提供一个全面的评估平台。此外，研究团队还构建了一个专门的测试集TextAtlasEval，以测试生成模型在处理长文本时的表现。该数据集的多样性和复杂性使其成为未来研究的重要基准。
TextAtlas5M的构建分为几个关键步骤：
数据收集：
合成图像部分：通过三种复杂度逐步增加的合成数据集创建，起初是简单的文本图像，随后是文本与视觉元素的交织，最终形成自然场景中无缝集成的文本。
真实图像部分：从不同来源收集真实世界中的图像，如PowerPoint演示文稿、书籍封面和学术论文，确保数据的多样性和真实性。
长文本评估基准：
设计专门的测试集TextAtlasEval，以评估模型在生成长文本图像时的性能，确保模型能够处理复杂的文本信息。
综合模型评估：
对多种开放源代码和专有模型进行全面评估，以揭示当前模型在长文本生成方面的局限性，并为未来研究提供指导。
在实验部分，研究团队对现有的文本到图像生成模型进行了评估，使用TextAtlasEval测试集来比较不同模型的性能。实验结果显示，许多当前的先进模型在处理长文本时表现不佳，尤其在文本的布局和准确性方面。通过使用FréchetInceptionDistance(FID)和CLIPScore等指标，研究团队分析了生成图像与真实图像之间的相似性。此外，研究还探讨了OCR（光学字符识别）在生成图像中的准确性，发现一些模型在生成复杂文本时存在显著的识别错误。这些实验结果不仅揭示了当前模型的不足之处，也为未来的改进提供了方向。
在TextAtlas5M的数据集中，研究人员采取了一些简单易懂的步骤来创建这个庞大的资源。首先，他们从各种来源收集了大量的图像，包括真实的广告、书籍和演示文稿。这些图像中包含了很多复杂的文本，能够帮助模型学习如何将文字和图像结合起来。其次，他们还制作了合成图像，这些图像从简单的文字开始，逐渐增加了复杂的背景和视觉元素，最终形成了自然场景中的文本。这种方法确保了数据集的多样性和实用性。最后，研究人员通过特定的测试集来评估不同模型的表现，看看它们在生成长文本图像时的能力。这些步骤共同构成了一个强大的工具，帮助我们改进未来的图像生成技术。
2025-02-10｜ULiverpool,CAS,UEdinburgh,UBristol,UExeter｜⭐️🟡
http ://arxiv.org/abs/2502.07829v1
本论文探讨了将偏好对齐与扩散模型（DMs）结合应用于图像生成和编辑的最新进展。随着生成式人工智能的发展，研究者们逐渐认识到在图像生成过程中，如何更好地反映人类的价值观和审美偏好至关重要。尽管已经有多项研究尝试将偏好对齐策略与扩散模型相结合，但目前仍缺乏系统性的综述。为此，本文对偏好对齐与扩散模型的结合进行了全面的调查，旨在填补这一领域的知识空白。通过对当前技术的回顾，本文不仅总结了偏好对齐在图像生成和编辑中的应用，还探讨了未来可能的发展方向。
本文的方法部分主要聚焦于如何将偏好对齐与扩散模型结合，具体包括以下几个方面：
框架设计：我们将偏好对齐过程视为一个强化学习（RL）问题，涉及策略模型、奖励模型、动作空间和环境等关键组件。
训练策略：采用多种优化技术，如强化学习与人类反馈（RLHF）、直接偏好优化（DPO）等，以提升模型与人类偏好的对齐程度。
数据收集：分为人类标注和AI生成两种主要策略，旨在收集高质量的偏好数据，以训练模型。
任务优化：通过在线和离线方法，优化模型在图像生成和编辑任务中的表现，确保生成内容符合用户的期望。
在实验部分，本文通过实际应用案例展示了偏好对齐与扩散模型结合的效果。我们评估了多种模型在不同任务中的表现，包括自动驾驶、医疗成像和机器人领域。实验结果表明，采用偏好对齐策略的模型在生成图像的质量和对用户偏好的响应能力上显著优于传统方法。此外，本文还探讨了在不同应用场景中遇到的挑战与解决方案，为未来的研究提供了重要的参考。
在这篇论文中，作者们提出了一种新的方法来让计算机生成的图像更符合人们的喜好。想象一下，如果你想让计算机绘制一幅画，通常它可能不会完全理解你想要的样子。为了解决这个问题，研究者们使用了一种叫做“强化学习”的技术，简单来说，就是让计算机通过不断尝试和学习来改进自己的表现。首先，计算机会根据你给的提示生成图像，然后人类会对这些图像进行评分，告诉计算机哪些是好的，哪些是不好的。计算机会根据这些反馈不断调整自己，最终生成的图像就会越来越符合大家的审美。通过这种方法，计算机不仅能够生成美丽的图像，还能更好地理解人类的需求和偏好。

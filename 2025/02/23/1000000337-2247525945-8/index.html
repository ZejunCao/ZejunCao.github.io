299-A3&B｜视频到4D生成，可重打光体积视频生成；评估6D空间理解能力；人类中心基础模型综述；自监督图像视频表示学习

主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）发布日期｜2025年2月13日｜周四
2025-02-12｜MUST,UQueensland,CASIA｜⭐️🟡
http ://arxiv.org/abs/2502.08377v1
随着视频生成动态三维（4D）内容的技术不断进步，现有方法在处理视频帧信息时，往往未能有效区分动态区域与静态区域。这种混合导致动态区域信息的忽视，尤其是在静态区域占比较大的情况下，最终生成的结果常常出现模糊的纹理。为了解决这一问题，本文提出了一种动态-静态特征解耦模块（DSFD），该模块通过区分动态和静态特征，增强动态表示，从而提高4D内容生成的质量。此外，设计了一种时空相似性融合模块（TSSF），进一步增强来自不同视角的动态表示，确保准确的运动预测。实验结果表明，所提出的DS4D方法在视频到4D生成任务中实现了最先进的性能，生成的内容具有清晰的纹理和高质量的动态表现。
本文的方法主要包括三个部分：初始化、动态-静态特征解耦（DSFD）和时空相似性融合（TSSF）。
初始化：从单视角视频开始，通过大规模重建模型生成静态3D高斯点，作为后续优化的基础。这一过程确保了几何先验的稳定性。
动态-静态特征解耦（DSFD）：
选择参考帧特征，计算当前帧特征与参考帧特征之间的差异，以识别动态特征。
将动态特征与静态特征进行解耦，利用几何和纹理信息来增强动态表示，最终生成丰富的动态特征。
时空相似性融合（TSSF）：
通过视角投影，从高斯点中提取解耦特征，结合不同视角的动态信息。
设计适应性融合机制，选择相似的动态信息，增强生成内容的动态表现，确保在4D空间中的一致性和准确性。
在实验部分，研究者使用Consistent4D和Objaverse数据集对所提出的方法进行了评估。实验表明，DS4D方法在多个指标上均超过了现有的最先进技术，特别是在生成质量、运动保真度和时空一致性方面表现突出。此外，研究者还进行了消融实验，以验证DSFD和TSSF的有效性，结果显示，解耦动态和静态特征显著提高了生成效果。最终，DS4D在真实场景数据集上的表现也证明了其在动态场景生成任务中的实用性和高效性。
在这项研究中，作者提出了一种新方法，旨在从视频中生成更清晰、更真实的动态三维内容。首先，他们从视频中提取每一帧的特征，就像拍摄每一张照片一样。接下来，他们通过比较这些特征，找出哪些部分是动态的（比如运动的物体），哪些部分是静态的（比如背景）。这就像在一个繁忙的街道上，找出人们在走动的地方和静止的建筑物。最后，作者利用这些动态特征，结合不同角度的信息，来生成更真实的4D场景。这个过程就像拼图，确保每个拼块都能完美结合，从而创造出完整、清晰的画面。
2025-02-12｜ShanghaiTech,NeuDim,DGene｜⭐️🟡
http ://arxiv.org/abs/2502.08297v1
在这项研究中，我们提出了BEAM，一个新颖的管道，用于将四维高斯表示与物理基础渲染（PBR）结合，以生成可重光照的体积视频。体积视频能够捕捉动态三维场景，提供身临其境的体验，广泛应用于虚拟现实、教育和远程呈现等领域。传统的方法在固定光照条件下表现不佳，而神经网络方法在效率、质量和适应性方面存在权衡。BEAM通过从多视角RGB视频中恢复详细几何形状和PBR属性，克服了这些局限性。该方法的关键在于通过一系列高斯技术，精确地恢复几何和PBR属性，从而生成高质量、可重光照的动态场景。
BEAM的实现分为几个关键步骤：
高斯建模与几何优化：
利用高斯性能跟踪与几何感知光栅化相结合，采用粗到细的优化框架，恢复空间和时间一致的几何形状。
通过引入法线一致性损失和光度损失，进一步提升几何优化效果。
PBR材料分解：
采用生成模型快速生成粗糙度纹理，并将其与密集高斯关联。
通过2D到3D的策略，估计环境光遮蔽（AO）和基础颜色，并优化到相应的高斯属性中。
物理基础渲染：
结合生成的4D高斯，使用基于高斯的光线追踪器计算可见性信息，从而实现准确的AO和基础颜色映射。
采用延迟着色和离线渲染技术，支持实时和高质量的离线渲染。
为验证BEAM的有效性，我们进行了多项实验。首先，通过81台工业相机捕捉人类表演，生成高质量的动态视频。我们将BEAM与现有的最先进技术进行了比较，结果表明，BEAM在AO、基础颜色和重光照质量上均优于其他方法。定量评估显示，我们的方法在各项指标上均取得了最佳表现。此外，通过用户研究，95.65%的用户更倾向于BEAM的时间重建质量，87%的用户选择了其法线一致性。这些结果表明，BEAM在生成高质量动态视频方面具有显著优势。
在我们的研究中，我们设计了一种新方法，叫做BEAM，可以让你在任何光照条件下观看动态视频。想象一下，你在看一个三维视频，这个视频中的人物和物体可以在不同的光线下看起来完全不同。我们的BEAM方法首先从多个角度拍摄视频，然后使用一些聪明的数学工具来提取出这些视频中的细节，比如物体的形状和表面特性。接着，我们通过一些计算，给这些物体添加颜色和光影效果，使它们在不同的光照下都能看起来真实。最后，我们的系统能够在电脑上快速渲染这些视频，让你在虚拟现实中获得身临其境的体验。
2025-02-12｜JHU,DEVCOMArmyResearchLaboratory｜⭐️
http ://arxiv.org/abs/2502.08636v2
在近年来，随着大规模多模态模型（LMMs）的发展，视觉场景的理解和推理能力得到了显著提升。然而，这些模型在处理复杂的三维空间推理方面仍然存在不足，尤其是在六自由度（6D）空间理解的能力上。现有的基准测试主要集中在二维空间理解，而缺乏对6D空间推理的全面评估。为此，研究团队提出了PulseCheck457，这是一个可扩展且无偏的合成数据集，旨在评估LMMs在6D空间推理方面的能力。该数据集涵盖了多种空间推理能力，包括多物体识别、二维位置、三维位置和三维方向等，并构建了七种不同类型的问题，涵盖了从基本的单物体识别到复杂的6D空间推理任务。通过对多种LMMs在PulseCheck457上的评估，研究发现随着任务复杂性的增加，模型的表现普遍下降，尤其是在3D推理和6D空间任务中。
PulseCheck457的构建采用了系统化的评估结构，重点关注四个核心能力：
多物体识别：这是场景理解的基础，模型需要能够识别图像中的多个物体。
二维位置：通过二维平面上的位置关系，模型可以初步理解物体之间的空间关系。
三维位置：此能力扩展了对三维空间的理解，涉及到物体的深度和距离，尤其在处理遮挡问题时至关重要。
三维方向：模型需要理解物体的旋转状态，从而推断出物体在场景中的精确姿态。
为了全面评估这些能力，研究团队设计了五个难度级别的问题，涵盖了从基本的语义识别到复杂的6D空间关系。这种分级设计使得基准测试能够深入分析每个模型在不同空间推理任务中的表现，揭示其强项与弱点。
在实验中，研究团队生成了1000幅图像，并针对七种问题类型进行了测试，包括单物体问题、多个物体问题、二维空间关系、三维位置理解等。通过对不同模型在PulseCheck457上的表现进行评估，研究者引入了相对性能下降率（RPDR）这一指标，量化了模型在引入新因素后的表现下降。实验结果显示，所有模型在3D推理方面均表现不佳，尤其是GPT-4o和GeminiPro-1.5在三维方向任务上表现最差，而其他模型则在三维位置理解上遇到困难。此外，研究还分析了模型预测中的偏见，发现一些模型在颜色和姿态预测上存在明显的偏差，指出了未来改进的方向。
PulseCheck457的构建方法可以简单理解为一个分级测试系统，旨在帮助我们评估计算机视觉模型在理解空间关系时的能力。首先，我们定义了四个主要的能力，比如识别多个物体、确定物体在图片中的位置、理解物体的三维位置和方向。这就像我们在观察一个房间时，不仅要知道里面有什么物体，还要知道它们在哪里、朝哪个方向。接着，我们设计了一系列问题，从简单的物体识别开始，逐渐增加难度，直到需要模型进行复杂的空间推理。这样分步测试的方式，可以帮助我们清楚地看到模型在不同任务中的表现，找出它们的优缺点。
2025-02-12｜CUHK,ZJUCAD&CG,THU,HKUST｜⭐️🟡

近年来，围绕人类外观、情感、身份和行为的理解以及生成技术取得了显著进展。人类中心的基础模型（HcFMs）应运而生，旨在统一多种人类相关任务，包括外观、身份、运动和意图等。这些模型受到大型语言模型和视觉模型成功的启发，旨在通过一个统一的框架超越传统的任务特定方法，从而降低开发成本并提高性能。本文综述了HcFMs的现状，并提出了一种分类法，将当前的方法分为四类：人类中心的感知模型、生成模型、统一感知与生成模型，以及人类中心的代理模型。每种模型均满足一定的标准，如泛化能力、广泛适用性和高保真度，旨在推动数字人类和人形表现的研究与应用。
本文提出的HcFMs方法分为四个主要类别：
人类中心的感知基础模型：这些模型通过多模态数据捕捉细粒度特征，应用于2D和3D理解任务，如姿态估计和人类解析。它们使用自监督学习和多任务监督学习，通过对人类身体结构的先验知识进行学习，来生成通用的人类表示。
人类中心的生成模型：这些模型专注于生成高保真、多样化的人类相关内容，训练时依赖于丰富的人类数据。它们利用无监督学习和多模态监督学习来提高生成的真实感和多样性。
统一感知与生成模型：这些模型整合了感知和生成能力，能够同时处理多种人类相关任务，提升人类理解和合成的效果。
人类中心的代理模型：这些模型不仅限于感知和生成，还学习人类智能和互动行为，适用于复杂的人形任务。它们通过视觉-语言-动作的整合，增强了模型的交互能力。
在实验部分，研究者们通过对比不同模型在多种人类中心任务上的表现，验证了HcFMs的有效性。实验结果表明，统一的基础模型在多任务学习中表现优于传统的任务特定模型，尤其在数据稀缺的情况下，展现出更强的泛化能力。此外，研究还探讨了模型在生成高保真度人类图像、视频和3D头像方面的应用潜力，强调了这些模型在实际应用中的广泛适用性和灵活性。
在这项研究中，科学家们开发了一种新型的人工智能模型，专门用于理解和生成与人类相关的内容。可以把这些模型想象成一种“智能助手”，它能识别和模拟人类的外观、动作和情感。研究者们将这些模型分为四类：第一类专注于“看”人类，比如识别姿势；第二类则是“创造”人类图像，比如生成照片或动画；第三类结合了前两者，既能看又能创造；第四类的模型更进一步，能够理解人类的智能行为，像是与人类互动。通过这些模型，科学家们希望能在未来的应用中，创造出更加真实和智能的数字人类形象。
2025-02-08｜UMoratuwa,MBZUAI,UAuckland,WIS
http ://arxiv.org/abs/2502.07811v1
CrossVideoMAE是一种创新的自监督学习框架，旨在通过跨模态对比学习来提升视频理解中的时空和语义表征能力。当前的视频基础MaskedAutoencoders（MAEs）往往侧重于从视觉角度学习有效的时空表征，但常常忽视了对特定交互或序列的细微语义特征的捕捉，这限制了模型对复杂动作的理解能力。CrossVideoMAE通过同时学习视频和采样帧的特征，整合了视频中的时空信息与帧中的空间信息，从而增强了对动态场景的理解。该方法通过对比学习策略，利用无标签数据提取丰富的语义信息，显著提高了动作识别的性能，超越了现有的最先进方法。
CrossVideoMAE的核心在于其独特的对比学习策略，主要分为以下几个部分：
网络架构：该方法采用双分支结构，包括视频分支和图像分支。视频分支采用时空MAE进行自监督学习，图像分支利用预训练的MAE从采样帧中提取空间特征。
对比学习：在视频和帧级别上，CrossVideoMAE同时实施内部和跨模态对比学习。内部对比学习确保视频特征在不同数据增强下的一致性，而跨模态对比学习则对齐视频特征和图像特征，增强模型的表征能力。
损失函数：模型的整体损失函数结合了对比损失、重建损失和均方误差损失，旨在最大化同一实例的特征相似性，同时最小化不同实例之间的相似性，从而提高特征提取的有效性。
在多个动作识别数据集（如UCF101、HMDB51、Kinetics-400和Something-SomethingV2）上，CrossVideoMAE的性能经过严格评估。实验结果表明，CrossVideoMAE在这些数据集上均超越了现有的最先进方法，尤其在SSv2数据集上表现尤为突出。通过细致的消融实验，研究者验证了各个组件对模型性能的贡献，发现跨模态和内部对比学习的结合显著提升了特征的可迁移性。此外，通过测试时适应技术，模型在推理阶段进一步优化了权重，从而提高了在不同数据集上的适应能力。
在CrossVideoMAE的工作原理中，我们可以把它想象成一个两部分的系统。第一部分是视频分支，它从视频中提取信息，理解视频中的动作是如何随着时间变化的；第二部分是图像分支，它从单独的帧图像中提取信息，捕捉每一帧的细节。为了让这两部分更好地合作，CrossVideoMAE使用了一种叫做对比学习的方法，这种方法可以帮助模型学习到哪些特征是相似的，哪些是不同的。举个简单的例子，就像我们在观察一段舞蹈时，既要关注舞者的动作（视频信息），也要注意每一个舞姿的细节（帧信息）。通过这样的方式，CrossVideoMAE能够更全面地理解视频内容，从而提高动作识别的准确性。

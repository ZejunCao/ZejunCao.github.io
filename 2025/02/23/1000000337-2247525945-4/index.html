2025-02-13｜CUHKMMLab,CUHKMiuLarLab,ByteDance,NEU,UPenn,NJU,CUHK(SZ),ShanghaiAILab
http ://arxiv.org/abs/2502.09621v1https ://mmecot.github.io/
本研究提出了MME-CoT，一个专门用于评估大型多模态模型（LMMs）在链式思维（CoT）推理方面的基准。尽管CoT在大型语言模型（LLMs）中显著提升了推理能力，但对LMMs的系统评估仍然不足。MME-CoT涵盖数学、科学、光学字符识别（OCR）、逻辑、时空和一般场景六个领域，旨在深入分析LMMs的推理质量、鲁棒性和效率。研究中引入了三种新颖的度量标准，分别评估推理的质量、鲁棒性和效率，以获取更细致的结果。通过高质量的数据和独特的评估策略，研究揭示了当前LMMs在CoT推理能力上的不足，尤其是在复杂任务和感知任务中的表现。
本研究的方法主要包括以下几个关键步骤：
数据集构建：MME-CoT数据集由六个主要领域及其子类别组成，经过严格的人类验证以确保高质量。数据包括推理问题和相应的图像输入，以丰富视觉推理的场景。
评估指标设计：
质量评估：引入了两个主要指标：回忆率（Recall）和精确率（Precision）。回忆率衡量生成步骤中包含的真实解决方案的比例，而精确率则评估生成步骤的准确性。
鲁棒性评估：通过对比直接回答和CoT提示在感知任务上的表现，评估CoT是否会干扰模型的感知能力。
效率评估：分析CoT在长推理过程中的效率，使用相关性率和反思质量来评估每一步对最终答案的贡献。
实验设计：通过对现有最先进的LMMs进行广泛实验，评估其在不同任务中的推理能力，并总结关键发现，旨在为未来的多模态推理研究提供基础。
在实验部分，研究对多种LMMs进行了系统评估，重点分析了它们在CoT推理中的表现。实验结果显示，具有反思机制的模型在CoT质量上表现优越，但在感知重任务上，CoT提示往往会导致性能下降，表明可能存在过度思考的问题。此外，尽管CoT的质量较高，具有反思的LMM在正常响应和自我纠正阶段的效率显著降低。研究还发现，当前的开源模型在关键的链式思维推理方面仍落后于一些闭源模型，如GPT-4o和Kimik1.5。通过这些实验，研究揭示了当前LMMs在多模态推理中的关键问题，并为未来的研究方向提供了重要见解。
在这项研究中，研究人员开发了一个新的评估工具，叫做MME-CoT，专门用来测试大型多模态模型的推理能力。这个工具通过六个不同的领域，比如数学和科学，来全面检查这些模型的表现。他们首先建立了一个高质量的数据集，确保每个问题都有准确的答案和相关的图像。然后，研究人员设计了一些新的评估标准，像是回忆率和精确率，帮助他们了解模型在推理时的表现如何。最后，他们进行了大量实验，结果发现，虽然一些模型在思考时表现得很出色，但在处理需要视觉识别的任务时却可能会出错。这些发现不仅揭示了当前模型的不足之处，也为未来的研究提供了新的方向。
2025-02-13｜NWPU,ShanghaiAILab,CUHK,THU
http ://arxiv.org/abs/2502.09620v1https ://github.com/Ivan-Tang-3D/ENEL
本研究旨在探讨无编码器架构在三维大型多模态模型（3DLMMs）中的潜力。尽管近年来在二维视觉领域对无编码器架构的研究有所增加，但在三维理解场景中的应用仍然是一个未解之谜。传统的3DLMMs通常依赖于强大的编码器，这些编码器在处理不同分辨率的点云时表现不佳，并且其提取的特征未能满足大型语言模型（LLMs）的语义需求。为了解决这些问题，本文提出了一种新的无编码器3DLMM架构——ENEL，旨在通过将编码器的功能直接集成到LLM中，克服现有模型的局限性。
研究中提出的无编码器架构主要包括以下几个关键策略：
LLM嵌入语义编码（LLM-embeddedSemanticEncoding）：在预训练阶段，采用轻量级的token嵌入模块来捕捉点云的高层语义信息，确保LLM能够直接处理原始点云数据。
混合语义损失（HybridSemanticLoss）：通过应用多种自监督损失函数（如重建损失、对比损失等），优化LLM在处理点云时的语义理解能力。
分层几何聚合（HierarchicalGeometryAggregation）：在微调阶段，利用聚合和传播操作，使LLM能够更好地捕捉点云的局部几何细节，增强模型对复杂三维结构的理解。
通过对比实验，研究团队使用ENEL模型在多个3D任务上进行了评估，结果显示其在3D分类、字幕生成和视觉问答等任务上均取得了优异的成绩。例如，在Objaverse基准测试中，ENEL在3D分类任务上获得了55%的GPT评分，显示出与现有最先进模型相媲美的性能。此外，实验还表明，采用的无编码器架构在处理多模态信息时，能够有效减少信息丢失，并提高语义理解的准确性。
在这项研究中，我们提出了一种新的方法来处理三维数据，而不依赖传统的编码器。首先，我们使用一个简单的模块来将三维点云数据转化为模型可以理解的格式。接着，我们设计了一种新的学习方法，帮助模型更好地理解这些点云的高层次语义信息。最后，我们引入了一种分层的方法，确保模型在学习过程中能够关注到细节，而不仅仅是全局信息。这种方法的好处在于，它使得模型能够在处理三维数据时更加灵活和高效，从而在各种任务中表现出色。
2025-02-13｜TencentWeChatAI,HKUST,JHU｜NAACL2025
http ://arxiv.org/abs/2502.08946v1https ://physico-benchmark.github.io/
本研究旨在探讨大型语言模型（LLMs）是否真正理解其生成的内容，尤其是在物理概念理解方面。为此，研究团队设计了一个名为PHYSICO的评估任务，旨在解决LLMs在物理概念理解中的“随机鹦鹉”现象。该任务通过网格格式的输入来减轻模型的记忆问题，网格展示了从核心现象到应用实例再到其他抽象模式的不同理解层次。通过对PHYSICO任务的全面研究，发现当前最先进的LLMs在理解物理概念时表现明显低于人类，约有40%的准确率差距。此外，研究表明LLMs在自然语言描述方面表现良好，但在网格任务中却显得力不从心，进一步验证了“随机鹦鹉”现象的存在。
本研究采用了总结性评估的方法来测量LLMs对物理概念的理解，具体步骤如下：
任务设计：基于布鲁姆的教育目标分类法，设计了反映不同理解水平的任务，分为低级理解和高级理解两个子任务。
低级理解子任务：使用自然语言格式的问题，评估LLMs对物理概念的记忆能力。
高级理解子任务：采用抽象表示格式，考察LLMs是否能够超越记忆，进行深层理解。
数据集构建：PHYSICO任务涵盖52个物理概念，设计了多种问题，以确保不同难度和多样性，确保评估结果的有效性。
评估方法：通过让LLMs回答设计的问题并计算其准确率，评估它们对物理概念的理解程度。任务的设计和评估都遵循了总结性评估的原则，以确保结果的可靠性。
在实验阶段，研究团队对多种LLMs进行了全面的评估，包括文本和多模态模型。实验主要集中在以下几个方面：
低级理解子任务的表现评估，LLMs在自然语言格式下的准确率普遍较高，接近95%。
高级理解子任务的表现评估，LLMs的准确率显著低于人类，显示出在深层理解方面的不足。
多模态输入的实验，测试LLMs在视觉输入下的表现，发现尽管某些模型在视觉任务上表现良好，但在高级理解任务中依然存在明显的性能差距。
在这项研究中，我们想要了解大型语言模型（LLMs）是否真的理解它们所说的话。为此，研究人员设计了一个叫PHYSICO的任务，使用了一种特殊的网格格式来帮助模型理解物理概念。这个任务分为两个部分：一个是简单的记忆测试，看看模型能否回忆起物理概念的定义；另一个是更复杂的理解测试，要求模型在抽象的情况下解释这些概念。通过对多种模型进行测试，我们发现，虽然LLMs在简单的记忆任务中表现很好，但在更深入的理解任务上却远远落后于人类，这说明它们在真正理解方面还有很大的提升空间。
2025-02-13｜NJUST,NJU,Tongji,PKU,SYSU,NUS
http ://arxiv.org/abs/2502.09533v1
在长时间的TalkingFace生成中，保持面部运动的连贯性和一致性是一个主要挑战。为了应对这一问题，本文提出了一种新的模型，称为运动先验条件扩散模型（MCDM）。该模型利用历史和当前视频片段的运动先验，增强面部运动的预测能力，并确保生成视频的时间一致性。MCDM由三个主要模块组成：归档剪辑运动先验模块、当前剪辑运动先验扩散模型和内存高效的时间注意机制。通过这些模块，MCDM能够有效地维护身份一致性和运动连续性，从而生成高质量的TalkingFace视频。此外，研究团队还发布了TalkingFace-Wild数据集，包含超过200小时的多语言视频数据，为进一步的研究提供了重要资源。
MCDM的架构设计围绕三个核心模块展开：
归档剪辑运动先验模块：该模块整合历史帧和参考帧，通过帧对齐注意力机制增强身份表示，确保在长时间序列中保持一致的面部特征。
当前剪辑运动先验扩散模型：此模块通过多模态因果关系和时间交互来预测面部运动状态，包括头部、嘴唇和表情的运动。它通过解耦当前身份和运动特征，促进运动的准确预测。
内存高效的时间注意机制：该机制动态更新和整合历史运动特征与当前特征，减少错误累积的影响，确保生成视频的时间一致性和流畅性。
通过这些模块的有效结合，MCDM能够在生成TalkingFace视频时保持高水平的真实性和连贯性。
在实验部分，研究团队对MCDM的性能进行了全面评估，使用了多个数据集，包括HDTF和CelebV-HQ。通过与其他最先进的方法进行比较，MCDM在多个指标上表现出色，特别是在音频与视频同步、结构相似性和生成视频质量等方面。实验结果表明，MCDM在保持身份一致性和运动连续性方面具有明显优势。此外，研究团队还进行了用户研究，评估生成视频的质量，结果显示MCDM在用户偏好测试中获得了最高评分，进一步验证了其有效性。
在我们的研究中，我们设计了一种新的模型，叫做运动先验条件扩散模型（MCDM），它可以帮助计算机生成更自然的TalkingFace视频。这个模型有三个主要部分。首先，归档剪辑运动先验模块就像一个记忆库，它会记住之前的面部动作，帮助模型在生成视频时保持人物的身份一致。其次，当前剪辑运动先验扩散模型会根据音频和图像来预测面部的运动，比如嘴巴的动作和表情。这部分确保了视频的每一帧都能跟上音频的节奏。最后，内存高效的时间注意机制就像一个聪明的助手，它能动态更新和整合过去的动作信息，避免生成的视频出现不连贯的情况。通过这三个部分的合作，我们的模型能够生成更加真实和流畅的TalkingFace视频。
2025-02-13｜UIUC｜ICLR2025
http ://arxiv.org/abs/2502.09617v1https ://wenj.github.io/LIFe-GoM/
本研究提出了一种名为LIFE-GOM的通用人类渲染方法，旨在从稀疏输入中重建可动画的人类头像。该方法依赖于从大规模数据集中提取的数据先验和归纳偏置，以避免场景特定的优化，并实现快速重建。研究中面临的主要挑战包括：首先，必须在推理时以单次传递的方式重建人类形状表示，而非通过迭代梯度调整；其次，渲染过程需要在计算效率和高分辨率之间取得平衡。为了解决这些问题，研究者们提出了一种增强的双重形状表示，结合了网格和高斯点的优点，并设计了迭代反馈更新框架，以逐步改善重建过程中的人类形状表示。同时，通过研究耦合的多分辨率高斯-网格表示，实现了高效且高质量的渲染。实验结果表明，该方法在THuman2.0等数据集上表现优异，重建时间少于1秒，渲染速度达到95.1FPS。
本方法的核心在于两个主要贡献：耦合的多分辨率高斯-网格表示和迭代反馈重建过程。具体步骤如下：
输入处理：接收一组源图像及其对应的二进制掩码、相机参数和人类姿态信息。
耦合多分辨率表示：
使用低分辨率网格进行快速动画和简单几何处理，同时将高斯点附加到高分辨率网格上，以提高渲染质量。
通过线性混合蒙皮将低分辨率网格的顶点变换到目标姿态，并在高分辨率网格上附加高斯点。
迭代反馈更新：
在重建过程中，通过迭代反馈机制，逐步优化低分辨率网格的顶点和高分辨率网格的高斯参数。
每一步更新使用前一步的结果和源图像特征，以融合多源信息，提升重建质量。
训练：采用端到端可微分的网络结构，使用L1损失、感知损失和拉普拉斯损失来优化网络参数。
本研究在多个数据集上进行了实验验证，包括THuman2.0、XHuman和AIST++。实验结果表明，LIFE-GOM在重建质量和速度上均优于现有的最先进技术。在THuman2.0数据集上，该方法在3视图和5视图设置下分别达到了24.65和25.57的PSNR，显著超过其他方法。同时，重建过程的时间保持在907.92毫秒，渲染速度达到95FPS。此外，研究还验证了该方法在跨域泛化能力上的表现，能够在未进行微调的情况下，直接应用于不同数据集，表现出色。通过一系列消融实验，研究者探讨了迭代步数和网格分辨率对重建质量和速度的影响，最终选择了最佳的参数设置以平衡性能和效率。
在这个研究中，科学家们想要创建一个可以快速生成动画人类头像的方法。首先，他们从多张图片中提取信息，比如人类的姿势和相机的设置。接着，他们使用一种特殊的网格模型，这种模型结合了低分辨率和高分辨率的特点。低分辨率的部分可以快速移动，而高分辨率的部分则能产生更细致的图像。然后，研究者设计了一种迭代反馈机制，这意味着每次生成图像时，系统会根据之前的结果进行调整，从而不断提高图像质量。最后，他们在多个数据集中测试了这种方法，发现它不仅速度快，而且生成的图像质量也很高。总的来说，这种方法能够帮助在虚拟现实和增强现实中更好地展示人类角色。
2025-02-13｜ClemsonU｜ICLR2025｜⭐️🟡
http ://arxiv.org/abs/2502.09613v1https ://latent-radiance-field.github.io/LRF
本研究提出了一种新颖的潜在辐射场（LatentRadianceField,LRF）框架，旨在将3D感知引入2D潜在表示，以提升3D重建的质量。当前的潜在3D重建方法在将2D特征转换为3D表示时，面临着领域间差距的问题，从而导致渲染效果下降。为了解决这一挑战，本文设计了一个包含三个阶段的框架：首先，通过一种对应感知的自编码方法增强2D潜在表示的3D一致性；其次，构建LRF以将这些3D感知的2D表示提升到3D空间；最后，实施VAE-辐射场（VAE-RF）对齐策略，以改善从渲染的2D表示中解码图像的效果。通过广泛的实验，我们的方法在合成性能和跨数据集的泛化能力上超越了现有的最先进技术，特别是在各种室内和户外场景中，展示了其优越的照片级真实感3D重建能力。
本研究的方法分为三个主要阶段，旨在增强潜在空间的3D感知能力。
对应感知自编码：在第一阶段，我们引入了一种新的约束机制，以增强变分自编码器（VAE）编码器的3D感知能力。通过在潜在空间中施加几何一致性约束，确保2D表示在编码过程中保持多视图一致性。
构建潜在辐射场（LRF）：第二阶段是创建LRF，以利用经过3D感知调整的2D表示来表示3D场景。LRF通过将3D-aware2D表示提升到3D潜在空间，从而实现高效的3D重建。
VAE-辐射场对齐：在最后阶段，我们引入了VAE-RF对齐策略，以减少因神经渲染引起的潜在分布偏移。这一策略通过对解码器进行微调，使得从LRF渲染结果中获得的图像质量得以提升，进一步增强了3D重建和生成的效果。
为了验证我们的方法的有效性，我们在多个真实世界数据集上进行了广泛的实验，包括MVImgNet、NeRF-LLFF、MipNeRF360和DL3DV-10K。实验结果表明，我们的方法显著提升了2D潜在表示在3D重建任务中的能力，成功减少了由于3D不一致性导致的伪影和颜色失真。同时，我们的方法在多个数据集上的PSNR、SSIM和LPIPS等指标上都取得了最先进的性能，特别是在少量视图合成任务中表现出良好的泛化能力。这些结果表明，通过我们的三阶段框架，潜在空间的3D一致性得到了显著改善，从而实现了高质量的3D重建效果。
在这项研究中，我们提出了一种新方法来帮助计算机更好地理解和重建3D场景。我们的方法分为三个步骤。首先，我们利用一种特殊的技术，让计算机在处理2D图片时，能够更好地理解这些图片在3D空间中的位置和关系。接着，我们构建了一个系统，能够将这些2D图片的信息提升到3D空间，这样计算机就可以生成更真实的3D图像。最后，我们还微调了计算机的解码器，以确保从这些3D图像中生成的最终图像更清晰、更真实。这种方法不仅提高了计算机生成3D图像的质量，还使得它在不同的场景中表现得更加出色。

2025-02-12｜DLUT,CUHK,Kuaishou｜⭐️⭐️
http ://arxiv.org/abs/2502.08639v1https ://cinemaster-dev.github.io/
本研究提出了CineMaster，一个全新的框架，旨在实现3D感知和可控的文本到视频生成。CineMaster的目标是赋予用户与专业电影导演相当的控制能力，使其能够精确地在场景中放置对象，灵活操控对象和相机，并对生成的帧进行直观的布局控制。该框架的设计分为两个阶段：首先是一个交互式工作流程，允许用户在3D空间中构建条件信号；其次，这些信号被用于引导文本到视频的扩散模型，以确保生成用户所期望的视频内容。通过这一框架，用户可以更直观地进行视频创作，克服了现有方法在3D对象运动和相机姿态标注方面的不足。
CineMaster的工作流程分为两个主要阶段。
阶段一：构建3D感知控制信号
用户通过调整3D边界框和相机位置，直观地定义场景中的对象和相机动态。
引入了一个预览机制，用户可以在每次修改后查看渲染效果，便于实时调整。
阶段二：条件视频生成
使用在第一阶段生成的控制信号（包括渲染的深度图、相机轨迹和对象类别标签）来指导文本到视频的扩散模型。
引入了语义布局控制网络，以融合3D空间布局和语义信息，确保生成的视频符合用户的设计意图。
设计了一个自动化数据标注管道，从大规模视频数据中提取3D边界框和相机轨迹，克服了缺乏标注数据的问题。
为评估CineMaster的性能，进行了广泛的定性和定量实验。通过与现有的最先进方法（如MotionCtrl和Direct-A-Video）进行比较，CineMaster在多个指标上都表现出色。实验结果显示，CineMaster在对象运动和相机运动的控制能力上明显优于其他方法，特别是在生成视频的质量和准确性方面，显著提高了生成视频的均值交并比（mIoU）和轨迹偏差（Traj-D）。此外，通过消融实验验证了不同组件对整体性能的贡献，证明了引入深度图和语义信息对于提升生成质量的重要性。
CineMaster的工作原理可以简单理解为一个两步走的过程。首先，它允许用户在一个虚拟的3D空间内自由地移动和调整对象和相机。用户可以想象自己是在导演一部电影，通过调整每个物体的位置和相机的角度，来实现自己想要的场景效果。这个过程就像在拍摄电影时，导演会不断调整演员和摄影机的位置，以获得最佳的拍摄效果。然后，CineMaster会利用用户设置的这些信息，生成一段视频，确保视频中的每个元素都按照用户的意图呈现。通过这种方式，CineMaster使得视频创作变得更加简单和直观，任何人都可以像专业导演一样，创造出令人惊叹的视觉作品。
2025-02-12｜SJTU,USTC,CUHK,HKUST,Stanford,ShanghaiAILab｜⭐️🟡
http ://arxiv.org/abs/2502.08590v1https ://bujiazi.github.io/light-a-video.github.io/
在本研究中，作者提出了一种名为Light-A-Video的训练无关视频重照明方法。该方法旨在解决现有视频重照明技术中的时间一致性问题，尤其是在保持视频帧之间的光源一致性方面。传统的图像重照明模型虽然在处理静态图像时表现优异，但在视频场景中，由于光源变化和帧间抖动，往往导致生成视频出现闪烁和不一致的现象。Light-A-Video通过结合预训练的图像重照明模型和视频扩散模型，能够在无需额外训练的情况下，实现平滑和高质量的视频重照明。该方法的核心在于引入了两个关键技术：一致光注意力模块和渐进光融合策略，从而有效提升了视频重照明的时间一致性和视觉质量。
Light-A-Video的实现分为几个主要步骤：
输入处理：首先，输入视频被编码并加噪声，随后通过视频扩散模型（VDM）进行去噪处理。
一致光注意力（CLA）模块：在自注意力层中引入CLA模块，增强跨帧交互，确保光源在视频帧间的一致性。CLA通过结合时间平均特征，稳定生成背景光源，减少光源抖动。
渐进光融合（PLF）策略：PLF策略通过线性融合原始视频的外观和重照明结果，逐步引导去噪过程，确保光照变化的平滑过渡。该策略遵循光传输独立性原则，保证视频的视觉连贯性。
输出生成：最终，经过CLA和PLF处理后，生成的重照明视频具有更高的时间一致性和视觉质量。
在实验部分，作者对Light-A-Video进行了全面的定性和定量评估。通过与现有的最先进图像重照明方法进行比较，作者展示了其在视频重照明任务中的有效性。实验结果表明，Light-A-Video在多个设置下均表现出色，能够支持整个输入视频的重照明以及输入前景序列的重照明与背景生成。定量评价采用了多种指标，包括CLIP得分、光流相似度和FID得分，以评估生成视频的时间一致性和图像质量。实验结果证实，Light-A-Video在保持高质量视觉效果的同时，显著改善了视频的时间一致性，解决了传统方法中存在的光源不一致和闪烁问题。
在Light-A-Video的方法中，我们可以将整个过程想象成一位艺术家在为一部电影重新上光。首先，艺术家会把电影的每一帧都拍成一张张静态图片，然后通过一种特殊的技术给这些图片添加光源。为了让每一帧的光线看起来更自然，艺术家使用了两种工具。第一种是“一致光注意力模块”，就像是艺术家在观察整个场景，确保每一帧的光源都保持一致，避免出现闪烁。第二种是“渐进光融合策略”，就像艺术家在慢慢调整光线的强度和方向，让整个场景的光照变化看起来更加平滑和自然。通过这两个工具，Light-A-Video能够在不需要额外训练的情况下，把输入的视频变得更加美丽和连贯。
2025-02-12｜POSTECH,MSRA｜⭐️🟡
http ://arxiv.org/abs/2502.08244v1https ://jinwonjoon.github.io/flovd_site/
本文提出了一种新颖的视频扩散模型FloVD，该模型结合了光流技术，以实现更好的相机控制视频生成。FloVD利用光流图来表示相机和移动物体的运动，具有两个主要优势。首先，由于光流可以直接从视频中估计，该方法可以使用任意训练视频，而无需真实的相机参数。其次，背景光流编码了不同视点之间的三维关联，使得通过利用背景运动来实现详细的相机控制成为可能。为了合成自然的物体运动并支持精细的相机控制，FloVD采用了一个两阶段的视频合成管道，包括光流生成和基于光流的视频合成。实验结果表明，FloVD在相机控制的准确性和自然物体运动合成方面优于现有的方法。
FloVD的工作流程分为两个主要阶段：光流生成和基于光流的视频合成。具体步骤如下：
光流生成阶段：
相机光流生成：从输入图像和相机参数中生成相机光流图，反映三维结构。通过使用单张图像的深度估计模型，将每个像素坐标从图像空间转换到三维空间，然后再投影回二维平面，计算位移向量以生成光流图。
物体光流合成：使用物体运动合成模型（OMSM）生成表示物体运动的光流图。OMSM通过从输入图像中编码信息，并利用噪声特征进行迭代去噪，来合成物体运动光流图。
基于光流的视频合成阶段：
将生成的相机光流图和物体光流图结合，形成一个完整的光流图，称为相机-物体光流图。该光流图随后被输入到流条件视频合成模型（FVSM），以合成最终的视频帧。
在实验部分，研究者对FloVD进行了广泛的评估，比较了其在相机控制和视频合成质量上的表现。实验采用了多种数据集进行训练和测试，包括一个包含500K视频片段的内部数据集，以及一个经过筛选的静态场景数据集。评估指标包括相机参数的估计误差、视频合成的样本质量和物体运动合成的质量。结果显示，FloVD在所有指标上均优于现有的相机可控视频合成方法，如MotionCtrl和CameraCtrl，特别是在自然物体运动的合成和相机控制的准确性方面。
FloVD的工作原理可以简单理解为一个两步走的过程。首先，系统从一张图片和相机设置中生成光流图，光流图就像是视频中物体和相机运动的“地图”。在第一步中，它会分析图片的深度信息，帮助理解物体在三维空间中的位置变化。然后，它会生成一个表示物体运动的光流图，确保视频中的物体看起来在自然移动。
接下来，进入第二步，FloVD会把这两个光流图结合起来，形成一个完整的运动图，作为生成视频的基础。最后，使用这个运动图，FloVD就能合成出新的、流畅的视频，确保视频中的相机运动和物体运动都看起来很自然。这样，FloVD就实现了在不需要真实相机参数的情况下，生成高质量、可控的视频。
2025-02-04｜UTAustin,Meta｜⭐️🟡
http ://arxiv.org/abs/2502.07802v1https ://jeff-liangf.github.io/projects/movieweaver/
本研究介绍了“MovieWeaver”这一创新的视频个性化生成模型，旨在解决多概念视频生成中的身份混合问题。传统方法通常依赖于单一概念的个性化，限制了其在复杂场景中的应用。为此，MovieWeaver通过引入“锚定提示”（anchoredprompts）和概念嵌入（conceptembeddings）来准确关联每个概念与其对应的参考图像，从而实现多概念的个性化视频生成。此外，该模型采用了一种无需调优的策略，允许用户通过不同的参考图像组合生成高质量的视频，极大地提高了灵活性和应用范围。
MovieWeaver的核心方法包括以下几个方面：
锚定提示：通过在文本提示中嵌入独特的图像锚定标记（如[R1]、[R2]等），将每个概念与其对应的参考图像明确关联。这一机制有效地解决了多概念视频生成中的身份混合问题。
概念嵌入：为每个参考图像分配独特的嵌入，以编码图像的顺序信息。这一方法确保了模型在处理多个参考图像时能够准确区分不同概念的顺序，避免了由于顺序不当导致的生成错误。
自动数据策划：实现了一种自动化的数据策划流程，旨在生成包含多种参考组合的训练数据，支持多样化的输入配置，如人脸、身体以及动物图像等。
在实验部分，MovieWeaver的性能通过与现有的多概念视频个性化方法进行了比较。实验结果表明，该模型在身份保留和视频质量方面均优于传统方法。具体而言，MovieWeaver能够在多种配置下生成高质量视频，且能够有效保持不同参考图像的身份特征。此外，模型在处理复杂场景时展现出的灵活性，使其能够适应多种用户需求。实验还包括对锚定提示和概念嵌入的消融研究，进一步验证了这些方法对提升生成质量的重要性。
在MovieWeaver的方法中，主要有两个关键概念：锚定提示和概念嵌入。锚定提示就像是在文本中添加标记，以便模型知道每个描述对应哪个图像。例如，如果我们想生成一个视频，描述为“一个女人在公园里遛狗”，我们可以在描述后面加上[R1]来指明这个女人的图像，使用[R2]来指明狗的图像。这样，模型就能很清楚地知道每个描述要用哪个图像来生成视频。概念嵌入则是帮助模型记住这些图像的顺序，确保生成的视频中，图像的顺序是正确的。这种方法让电影制作变得更加灵活，用户可以用不同的图像组合生成个性化的视频，而不需要复杂的调优过程。
2025-02-12｜CUHK,THUSIGS,HKU｜⭐️🟡
http ://arxiv.org/abs/2502.08189v1https ://anycharv.github.io
在角色视频生成领域，AnyCharV框架的提出为用户提供了更高的灵活性和控制能力。传统的视频生成方法往往只能在固定的背景场景中对角色进行简单的动画处理，缺乏对角色细节和复杂动作的保留。为了解决这一问题，AnyCharV采用了一种创新的两阶段训练策略，结合了细致到粗略的引导机制，使得用户能够将任意角色自然地融入到目标视频中，同时保持角色的真实外观和复杂的人物交互。通过自我增强的训练方法，AnyCharV不仅提升了生成视频的质量，还显著改善了角色的运动表现和场景的融合度，展现了其在实际应用中的潜力。
AnyCharV的核心方法包括两个主要阶段：
基础模型训练：
在第一阶段，利用自我监督的方式，构建一个基础模型，该模型通过细致的分割掩码和2D姿态序列来集成参考角色与目标场景。
参考角色的外观通过CLIP编码器进行编码，以确保角色的身份被有效保留，并通过去噪UNet模型进行细致的生成。
自我增强训练：
第二阶段引入自我增强策略，通过使用粗略的边界框掩码来指导生成过程。这种方法旨在减少由于细分掩码形状差异造成的负面影响，从而更好地保留角色的细节。
在这一阶段，模型通过生成多个源-目标视频对，利用已生成的视频进行再训练，进一步提升生成视频的质量和一致性。
这种分阶段的训练策略不仅提高了生成视频的质量，同时也增强了角色与场景的交互表现。
在实验部分，AnyCharV框架的性能通过构建一个包含5482个视频的CharVG数据集进行评估。实验结果表明，AnyCharV在生成质量上优于现有的最先进方法，尤其在角色身份的保留和运动一致性方面。通过定量评估，使用FVD、DOVER++和CLIP图像评分等指标，AnyCharV在多个维度上都显示出显著的优势。此外，用户研究也表明，参与者对AnyCharV生成的视频质量给予了高度评价，特别是在复杂人机交互和角色运动的流畅性方面。
AnyCharV的工作原理可以简单理解为一个两步走的过程。首先，系统会学习如何将一个参考角色（比如某个动画人物）与一个目标视频（比如一个运动场景）结合在一起。在这个过程中，它会使用一些特殊的工具来确保角色的外观和动作都能准确地反映在视频中。接下来，系统会通过“自我提升”的方式，利用已经生成的一些视频来进一步训练自己，让角色的细节和动作更加自然。这样，用户就可以很方便地创建出看起来真实、动作流畅的角色视频，无论是用于游戏、动画还是其他创意项目。
2025-02-10｜HKUST,THU,SYSU,TencentAILab｜⭐️
http ://arxiv.org/abs/2502.07825v1
随着视频生成技术的迅速发展，预训练的视频生成模型在生成高保真和时间一致性视频方面取得了显著成功。然而，这些模型通常只能基于静态提示（如文本或图像）生成视频，限制了其在动态交互场景中的应用。为了解决这一问题，本文提出了一种新的方法——动态世界模拟（DynamicWorldSimulation,DWS），旨在将预训练的视频生成模型转变为可控的世界模拟器，能够执行指定的动作轨迹。DWS通过引入轻量级的动作条件模块，精确对齐条件动作与生成的视觉变化，同时强调一致的动态过渡建模，以此构建强大的世界模拟器。此外，DWS还引入了一种运动强化损失，提升了模型对动态变化的捕捉能力。实验结果表明，DWS能够广泛应用于扩散模型和自回归变换模型，显著提高生成的可控动态视频的质量，尤其在游戏和机器人领域。
DWS方法的核心在于其架构的高效性和灵活性，主要包括以下几个方面：
动作条件模块：该模块通过两个线性层嵌入到现有的生成模型中，使每一帧的生成能够直接受到对应动作的影响，而不是依赖全局描述。这种细粒度的控制确保了每个动作与生成帧之间的直接对应关系。
运动强化损失：传统的训练损失函数通常会均匀关注所有视觉元素，而DWS引入了运动强化损失，通过在训练过程中强调帧间动态变化，提升模型对动作引起的状态变化的捕捉能力。这使得模型在生成视频时更加关注动态元素，而非静态背景细节。
优先想象技术：为了提高模型在强化学习中的应用效率，DWS采用了一种优先想象的方法，专注于最具学习潜力的状态转移，从而提升样本效率和策略学习的性能。
在实验部分，DWS的有效性通过两个主要方面进行评估：动作模拟能力和模型基的强化学习效果。首先，利用BAIR数据集（机器人推物）和Procgen、Atari游戏数据集，评估DWS在生成可控视频方面的表现。实验结果显示，DWS在多种基模型（如Open-Sora和iVideoGPT）上均显著提升了动作条件视频的生成质量，尤其在动态一致性和背景保持方面表现出色。其次，在模型基强化学习的应用中，DWS经过训练的世界模型在多个游戏环境中展示了优越的策略学习效果，特别是在样本效率上显著优于传统方法。综合实验结果，DWS方法不仅提升了视频生成质量，还增强了模型在强化学习任务中的实用性。
DWS的方法可以简单理解为一个“智能视频生成器”，它不仅能生成视频，还能根据我们给出的具体动作来调整视频内容。首先，它通过一个特别的模块，让每一帧视频的生成都能直接反映出我们想要的动作，而不是用一个笼统的描述。其次，DWS还引入了一种新的训练方式，强调视频中每一帧之间的动态变化，确保生成的视频更真实、更连贯。最后，DWS还采用了一种聪明的策略，重点关注那些能帮助学习的关键状态，这样可以更快地提升模型的学习效果。总的来说，DWS让视频生成变得更加智能和高效，能够更好地适应我们想要的各种场景和动作。

本研究提出了一种新的任务——关键步骤技能生成（KS-Gen），旨在简化人类技能视频的生成过程。现有的视频生成模型在处理复杂的人类技能时面临挑战，因为这些技能通常涉及多个步骤和复杂的场景变化。为了解决这一问题，KS-Gen专注于生成技能完成所需的关键步骤视频片段，而不是完整的技能视频。通过引入一个精心策划的数据集和多种评估指标，研究团队希望提高人类技能学习的效率，并为机器人提供合成经验以学习技能。
本研究的方法框架主要包括三个核心组件：
多模态大型语言模型（MLLM）规划：根据给定的初始图像和技能描述，生成关键步骤的详细文本描述。为提高输出的准确性，采用检索增强的方法来优化步骤的顺序和内容。
关键步骤图像生成（KIG）：针对每个关键步骤生成相应的图像。由于后续步骤缺乏初始图像作为参考，KIG通过利用初始图像和步骤描述来生成每个步骤的第一帧图像，确保生成的图像在视觉上保持一致性。
视频生成模型：使用生成的步骤描述和关键步骤图像，生成对应的关键步骤视频片段。该模型旨在确保生成的视频片段在时间上具有高度一致性，从而形成连贯的技能展示。
在实验部分，研究团队评估了不同生成器在标准设置下的表现，并通过与基线模型的比较，验证了其新框架的有效性。实验结果显示，结合MLLM、KIG和视频生成模型的框架在生成关键步骤视频方面表现优越。此外，研究还进行了详细的消融实验，评估了图像和视频生成模型在不同条件下的效果。结果表明，提出的KIG显著改善了生成图像的质量和一致性，并且所构建的数据集和评估指标为KS-Gen任务的性能提供了有力支持。
在我们的研究中，我们想要让机器学习如何生成复杂的技能视频，比如做抹茶。我们的方法分为几个步骤：首先，我们使用一种智能的语言模型来根据一张图片和技能描述生成每个关键步骤的详细文字说明。接下来，由于后续步骤没有初始图像作为参考，我们设计了一种新的图像生成器，能够为每个步骤生成第一张图像，这样可以确保每个步骤的视觉效果一致。最后，我们利用这些步骤的描述和生成的图像，制作出展示这些步骤的视频片段。通过这样的方式，我们的目标是让机器能够更好地学习和展示复杂的技能，帮助人们更快地掌握这些技能。

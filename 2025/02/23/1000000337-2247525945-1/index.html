2025-02-13｜Stanford,MIT｜⭐️⭐️
http ://arxiv.org/abs/2502.09608v1https ://sketchseg.github.io/sketch-seg/
在本研究中，我们提出了一种名为SketchSeg的方法，旨在对场景草图进行实例分割。草图作为一种视觉表达工具，广泛应用于创意探索和设计过程中。然而，由于草图的稀疏性和风格多样性，传统的图像分割模型在处理草图时面临许多挑战。SketchSeg通过结合先进的图像分割和目标检测模型，采用类无关的微调方法，并利用深度信息来优化分割掩码。我们还构建了一个合成的场景草图分割数据集，以展示我们方法的鲁棒性和有效性，并促进该领域的进一步研究。
SketchSeg的工作流程包括几个关键步骤：
草图输入与目标检测：首先，将输入的草图图像传递给经过微调的GroundingDINO模型，以生成候选对象的边界框和相应的置信分数。
掩码提取：接下来，利用SegmentAnything(SAM)模型从生成的边界框中提取初始掩码。这一过程确保了草图中每个实例的像素被适当地分组。
分割掩码的精细化：通过深度信息和形态学操作，对初始掩码进行精细化处理，以解决重叠区域的模糊问题。深度信息帮助确定前景对象的优先级，确保分割的准确性。
层次化处理与修复：最后，根据分割结果生成有序的图层，并使用预训练的修复模型填补被遮挡区域，从而支持草图的高级编辑功能。
为了评估SketchSeg在多样化草图上的性能，我们构建了一个合成的注释场景草图数据集，涵盖了不同的绘图风格、笔触风格和对象类别。我们的方法在多个基准测试中表现出色，相较于其他现有方法，SketchSeg在检测和分割草图实例方面均显著提高了准确性。实验结果显示，我们的方法在处理复杂场景和多种对象类别时，能够有效地应对挑战，尤其是在处理如SketchAgent等具有高度抽象风格的草图时。
在我们的方法中，首先，用户输入一幅草图，系统会使用一种经过训练的模型来识别草图中的不同对象。接着，系统会为每个识别出的对象生成一个边框，并提取出这些对象的具体形状。然后，模型会利用草图的深度信息来判断哪些对象在前面，哪些在后面，确保在重叠部分能够准确分配给各个对象。最后，系统会将这些对象分成不同的层次，并填补任何被遮挡的区域，这样用户就可以轻松地编辑草图，比如移动或缩放其中的某个对象。通过这些步骤，我们的方法能够更好地理解和处理各种风格的草图。
2025-02-13｜SZU｜Eurographics2025｜⭐️⭐️
http ://arxiv.org/abs/2502.09064v1https ://zichongc.github.io/StyleBlend/
在文本到图像（T2I）扩散模型的研究中，生成既符合文本提示又具有特定艺术风格的图像仍然是一个重大挑战。本文提出了一种新方法——StyleBlend，旨在通过从有限的参考图像集中学习和应用风格表示，来增强风格特定的内容生成。StyleBlend方法将风格分解为两个主要组成部分：构图和纹理。通过分别学习这两个组件，StyleBlend能够在生成内容时实现文本语义的对齐，同时保持风格的一致性。与现有方法相比，该方法有效解决了文本不对齐和风格表达不强的问题，具有显著的定性和定量优势。
StyleBlend的核心在于其双分支合成框架，分别处理构图风格和纹理风格。具体方法如下：
风格分解：将图像风格分为构图风格（关注图像的语义结构和布局）和纹理风格（关注细节和外观）。
独立学习：采用不同的训练策略来分别学习构图风格和纹理风格。构图风格通过生成带有噪声的参考图像并进行去噪处理来捕捉，而纹理风格则通过优化全局嵌入来学习。
特征注入：在合成过程中，通过互相注入两个分支的特征来实现风格的有效融合。具体来说，从构图分支提取的Q特征与纹理分支的K和V特征进行交换，以确保最终生成图像的构图与纹理风格的和谐一致。
在实验中，StyleBlend被应用于多个样例，以验证其在风格特定图像生成中的有效性。研究者收集了多种风格的图像，并进行了定量和定性比较。结果显示，StyleBlend在处理3-shot和1-shot案例时，均能有效保持风格一致性和文本对齐。通过与现有的几种基线方法（如DreamBooth和TextualInversion）进行比较，StyleBlend在生成的图像质量、风格相似性和文本对齐度上均表现出色，证明了其优越性。
StyleBlend的工作原理可以用一个简单的比喻来解释：想象你在做一个拼图，拼图的每一块都有特定的形状和颜色。StyleBlend就像是一个聪明的拼图工具，它能够帮助你把不同的拼图块（构图和纹理）完美地组合在一起。首先，它会分析每一块拼图的形状（构图）和颜色（纹理），然后在合成新图像时，确保这些元素能够相互配合，最终形成一个既美观又符合你想要表达的内容的完整图像。通过这种方式，StyleBlend能够在有限的参考图像基础上，创造出多样化且高质量的图像，解决了以往方法中常见的风格不一致和文本不对齐的问题。
2025-02-13｜Tel-AvivU,NVIDIA,ReichmanU｜⭐️🟡
http ://arxiv.org/abs/2502.09411v1https ://rotem-shalev.github.io/ImageRAG
ImageRAG是一种动态图像检索方法，旨在增强基于文本的图像生成模型的能力，尤其是在生成稀有或未见概念时。尽管扩散模型已在图像生成领域取得显著进展，但它们在处理特定用户需求和稀有概念时仍存在局限性。ImageRAG通过动态检索相关图像并将其作为上下文参考，来指导生成过程，从而提高生成质量。与以往需要专门训练的检索增强生成方法不同，ImageRAG利用现有的图像条件模型，避免了额外的训练需求，显示出对不同模型的高度适应性，显著改善了稀有和细粒度概念的生成效果。
ImageRAG的核心在于其动态图像检索和生成过程，主要包括以下几个步骤：
初始图像生成：根据给定的文本提示，使用文本到图像（T2I）模型生成初步图像。
图像匹配与缺失概念识别：通过视觉语言模型（VLM）判断生成的图像是否与文本提示匹配。如果不匹配，VLM会识别出缺失的概念。
图像检索：根据识别出的缺失概念，生成详细的图像描述并从外部数据库中检索相关图像，以作为生成过程的参考。
生成过程指导：将检索到的图像作为上下文信息传递给T2I模型，以帮助其更好地生成符合文本提示的图像。
这一方法不仅提升了生成的准确性，还能够处理用户特定的需求和稀有概念的生成。
为了验证ImageRAG的有效性，研究者进行了定量和定性实验，主要对比了使用ImageRAG和不使用该方法的图像生成效果。实验涉及两个不同类型的模型：OmniGen和SDXL。通过在不同数据集（如ImageNet、iNaturalist等）上进行生成任务，结果表明，ImageRAG显著提高了生成稀有和细粒度概念的能力。此外，研究还进行了用户研究，评估参与者对使用ImageRAG生成的图像在文本一致性、视觉质量和整体偏好上的反馈，结果显示ImageRAG的生成图像在这几个方面均优于其他方法。
ImageRAG的工作原理可以简单理解为一个“智能助手”，它帮助图像生成模型更好地理解用户的需求。首先，系统会根据用户给出的描述生成一张初步的图像。接着，它会检查这张图像是否符合描述。如果不符合，它会询问自己：“缺少了什么？”然后，它会从一个大型图像库中找到与这些缺失部分相关的图像，像是找到了可以借鉴的参考资料。最后，它将这些参考图像提供给生成模型，帮助其生成更符合用户描述的图像。这个过程就像是一个艺术家在创作时，借鉴他人的作品来激发灵感，从而创作出更好的作品。
2025-02-13｜KAIST｜⭐️🟡
http ://arxiv.org/abs/2502.09164v1
E-MD3C（高效掩蔽扩散变换器）是一种为零-shot对象图像定制而设计的高效框架。与现有依赖资源密集型U-Net架构的工作不同，E-MD3C采用轻量级的掩蔽扩散变换器，主要在潜在补丁上操作，从而显著提高了计算效率。该框架整合了三个核心组件：首先是处理自编码器潜在特征的高效掩蔽扩散变换器；其次是确保紧凑性并保持背景对齐和细节的解耦条件设计；最后是一个可学习的条件收集器，能够将多个输入整合成紧凑的表示，以实现高效去噪和学习。E-MD3C在VITON-HD数据集上的表现超越了现有方法，在多个指标上如PSNR、FID、SSIM和LPIPS均显示出明显的优势。此外，该模型的参数数量仅为现有U-Net模型的四分之一，推理速度提升了2.5倍，并将GPU内存使用减少至一半。
E-MD3C框架由三个核心模块组成，分别是：
去噪变换器基础扩散网络（DTDNet）：该模块作为生成高质量目标图像的主干，通过迭代去噪过程生成图像。它利用预训练的StableDiffusionVAE将图像投影到潜在表示中，并通过逐步添加高斯噪声来训练模型参数，以最小化噪声预测的均方误差。
解耦掩蔽扩散模块（DMDNet）：此模块通过掩蔽设计建模目标图像中的上下文关系，利用提示和噪声潜在特征作为强正则化机制，增强变换器的学习能力。
条件收集网络（CCNet）：该模块集成了多个关键输入，包括目标框条件特征、提示图像特征和全局源图像特征，生成一个紧凑的条件向量，指导去噪过程。通过随机掩蔽30%的潜在补丁，进一步提高模型的学习效果。
在实验部分，E-MD3C在多个公开数据集上进行了评估，特别是在VITON-HD数据集上进行了详细的比较分析。我们采用标准的评估指标，包括FID、PSNR、SSIM、LPIPS和L1，以量化模型的性能。实验结果表明，E-MD3C在生成高质量图像方面超越了现有的其他方法，如AnyDoor，同时在推理速度和内存使用方面表现出色。此外，通过消融研究，我们分析了模型中各个组件的影响，验证了解耦条件和掩蔽建模对模型性能的提升作用。最终，E-MD3C展示了其在零-shot对象定制任务中的强大能力和高效性。
E-MD3C的工作原理可以简单理解为一个高效的图像生成器，它通过三个主要部分来实现目标：首先，它使用一种叫做去噪变换器的技术来逐步清晰化图像，像是在模糊的画面上慢慢擦去涂抹的涂料；其次，它通过解耦的方式将不同的信息分开处理，确保生成的图像既保留了原始图像的细节，又能适应新的对象；最后，它有一个聪明的条件收集器，可以把不同的输入信息整合成一个小巧的向量，帮助生成更准确的图像。这样，E-MD3C不仅能快速生成高质量的图像，还能在计算资源有限的情况下运行得很好，非常适合实际应用。
2025-02-12｜HIT(SZ)｜⭐️🟡
http ://arxiv.org/abs/2502.08674v1
在时尚推荐领域，如何生成兼容的服装搭配已成为一个备受关注的研究主题。现有的研究主要集中在评估现有服装搭配的兼容性，鲜有关注如何基于给定的时尚单品生成新的、兼容的时尚单品。本研究提出了一种新任务，即在给定任意数量的时尚单品的基础上，生成互补且兼容的时尚单品。为此，我们设计了一种名为COutfitGAN的服装生成框架，旨在合成与给定服装兼容的其他时尚单品的逼真图像。该框架包括一个金字塔风格提取器、一个服装生成器、一个基于UNet的真实/虚假判别器以及一个搭配判别器。通过对一个包含超过20万套服装和80万件时尚单品的大规模数据集进行训练和评估，实验结果表明COutfitGAN在相似性、真实性和兼容性指标上均优于其他基线方法。
我们的方法分为几个主要部分：首先是问题的形式化描述，接着介绍COutfitGAN框架的各个组件。具体来说，框架包括：
金字塔风格提取器：用于提取给定时尚单品的风格特征，以便在生成互补时尚单品时进行学习和保留。
服装生成器：逐步融合给定时尚单品的轮廓信息和风格特征，以合成互补的时尚单品。
真实/虚假判别器：用于监督合成图像的视觉真实性，确保生成的图像在全局和局部视角上都符合真实图像的特征。
搭配判别器：通过对时尚单品的全局风格进行建模，评估生成的时尚单品与给定单品的兼容性。
此外，我们还设计了训练损失函数，以确保生成的时尚单品在视觉上与真实图像相似，并保持搭配的兼容性。
为验证COutfitGAN的有效性，我们构建了一个名为OutfitSet的大规模数据集，包含200,000套服装和800,000件时尚单品。通过对该数据集进行训练，我们评估了模型在生成互补时尚单品方面的表现。实验结果显示，COutfitGAN在生成的时尚单品的相似性、真实性和兼容性方面均显著优于其他基线方法。此外，实验还展示了不同类别时尚单品的生成效果，证明了模型在处理多样化时尚单品方面的能力。
COutfitGAN的工作原理可以简单理解为一个多步骤的过程。首先，我们有一个金字塔风格提取器，它从给定的时尚单品中提取出它们的风格特征，就像从不同角度观察一件衣服，提取出它的颜色、图案等信息。接下来，这些提取的风格特征会被送入一个服装生成器，它会根据这些特征和轮廓信息，逐步合成出新的时尚单品。为了确保生成的图像看起来真实，我们使用了一个类似于“审查员”的判别器，它会检查生成的图像是否符合真实的时尚单品的特点。此外，我们还设计了一个搭配判别器，它会评估新生成的时尚单品与原有服装的搭配是否协调。通过这样的多层次监督，COutfitGAN能够生成既美观又兼容的时尚搭配。
2025-02-12｜SNU,CornellTech｜⭐️🟡
http ://arxiv.org/abs/2502.08690v1
在文本到图像生成的任务中，传统的文本编码器在内存使用上存在显著的低效问题。虽然这些编码器只需一次前向传递便能生成文本嵌入，但其内存需求却高达去噪模块的八倍。为了解决这一问题，本文提出了“跳过和重用层”（Skrr）的方法，专门针对文本编码器进行优化。Skrr通过选择性地跳过或重用某些层，利用变换器块中的冗余性，显著降低内存消耗，同时保持生成图像的质量。实验结果表明，在高稀疏性条件下，Skrr的性能与原始模型相当，且在多个评估指标上均优于现有的块级剪枝方法，展示了其在内存效率和性能保持方面的优势。
Skrr方法主要分为两个阶段：跳过（Skip）和重用（Re-use）。在跳过阶段，首先评估每个多头自注意力（MHA）和前馈网络（FFN）子块的重要性，使用特定于文本到图像任务的度量标准进行排序。重要性较低的块将被逐步移除，并通过一种基于束搜索的算法探索多种可能的组合，以优化剪枝过程。重用阶段则评估剩余层的有效性，确保重要信息得到保留，从而减少性能损失。具体步骤如下：
跳过算法：使用度量标准评估每个子块的输出差异，基于此差异进行排序并选择性移除低重要性的块。
束搜索策略：探索多条剪枝路径，以更好地考虑块间的相互作用，确保剪枝的有效性。
重用算法：评估被跳过层的相似性，并决定是否重用相邻未跳过的层，以恢复被跳过层的功能。
为了验证Skrr的有效性，本文将其与多种现有的基于大型语言模型的块级剪枝技术进行比较。实验使用了PixArt-Σ数据集，评估了生成图像的质量和文本对齐度。结果显示，Skrr在不同稀疏性水平下均保持了与原始模型相当的性能，尤其在高稀疏性条件下，Skrr的生成图像质量得以维持，且在FID、CLIP和DreamSim等多个评估指标上均表现优异。此外，Skrr在内存使用和计算成本方面也显著优于其他方法，表明其在实际应用中的潜力。
Skrr的方法可以简单理解为一个两步走的过程，旨在让文本编码器在生成图像时更省内存。首先，在“跳过”阶段，我们会检查每个处理文本的部分，看看哪些部分不太重要，可以省略掉。接着，使用一种聪明的搜索策略，确保我们不会错过那些可能很有用的部分。然后，在“重用”阶段，我们会查看那些被省略的部分旁边的部分，看看是否可以把它们的功能重新利用，以弥补省略造成的信息损失。通过这种方式，Skrr能够在不牺牲图像质量的情况下，显著降低内存使用，提升文本到图像生成的效率。

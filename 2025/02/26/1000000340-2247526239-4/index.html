2025-02-18｜MicrosoftResearch,UMD,UWM,KAIST,UWashington｜⭐️🟡
http ://arxiv.org/abs/2502.13130v1https ://microsoft.github.io/Magma
Magma是一种新型的基础模型，旨在为多模态人工智能代理提供支持。与传统的视觉-语言模型不同，Magma不仅具备语言理解能力，还能够在视觉-空间环境中进行规划和执行动作。该模型通过利用大量的异构数据集进行预训练，从而在UI导|航、机器人操作等任务中表现出色。Magma的核心特性是其多模态理解和空间-时间推理能力，使其能够在数字和物理环境中执行复杂的代理任务。研究表明，Magma在多项任务上超越了专门针对这些任务设计的模型，展现了其广泛的适用性和强大的泛化能力。
Magma的构建方法主要包括以下几个方面：
预训练目标：为了整合不同类型的任务，Magma采用了一种统一的预训练接口，旨在通过预测UI导航的2D坐标、机器人操作的3D位置以及文本输出，来克服不同任务之间的领域差异。
数据扩展：为了解决现有视觉-语言-动作数据集的数量和多样性不足的问题，Magma利用视频数据中的丰富人类动作和人机交互信息，生成了大量标注数据，以辅助模型学习。
**Set-of-Mark(SoM)和Trace-of-Mark(ToM)**：Magma引入SoM和ToM技术，前者用于图像中的可操作区域标注，后者则用于视频中的动作轨迹预测。这一组合不仅提升了模型的空间-时间智能，还能够有效利用未标注的视频数据进行预训练。
在实验中，Magma被评估于多种任务，包括UI导航和机器人操作。通过零-shot评估，Magma在多个基准测试中表现优异，显示出其在多模态理解和动作预测方面的强大能力。特别是在机器人操作任务中，Magma的成功率显著高于其他基于视觉的模型，表明其在空间智能学习方面的优势。此外，Magma通过有效的微调策略，能够迅速适应不同的下游任务，进一步验证了其作为多模态AI代理基础模型的潜力。
Magma的工作原理可以简单理解为它像一个聪明的助手，能够理解和处理不同类型的信息，比如图像、视频和文字。首先，它通过观察大量不同的数据（比如视频中的动作和图片中的按钮）来学习这些信息。然后，它使用一种叫做Set-of-Mark的技术来找到图片中可以点击的地方，或者在视频中预测未来的动作。此外，Trace-of-Mark的技术帮助它理解在视频中人们如何移动和互动。通过这些学习，Magma能够在各种任务中表现出色，比如在手机应用中导航或者让机器人完成特定动作。总之，Magma就像一个全能的助手，能够帮助我们处理复杂的多模态任务。
2025-02-18｜TAMU,UMichigan,UIUC,UNCChapelHill
http ://arxiv.org/abs/2502.13146v1https ://github.com/taco-group/Re-Align
随着大型视觉语言模型（VLMs）的出现，单模态的大型语言模型（LLMs）在跨模态应用中展现了巨大的潜力。然而，VLMs在处理视觉和语言信息时，仍然容易出现幻觉现象，即生成不准确或虚构的信息。为了解决这一问题，研究者们尝试通过强化学习和人类反馈（RLHF）来对LLMs进行对齐，但这些方法通常忽视了视觉信息在对齐过程中的重要性。本文提出了一种新的框架RE-ALIGN，利用图像检索技术构建双重偏好数据集，有效地结合了文本和视觉偏好信号，并引入了一种扩展的直接偏好优化（rDPO）方法，以在微调过程中增强对齐效果。实验结果表明，RE-ALIGN在减轻幻觉方面表现优异，并在视觉问答（VQA）任务中取得显著性能提升，展示了其在多种VLM架构和规模下的鲁棒性和可扩展性。
RE-ALIGN框架的核心在于结合图像检索与直接偏好优化（DPO）来改善VLM的对齐。其主要步骤包括：
偏好生成：
策略性掩蔽：从输入的文本和视觉对中生成响应时，策略性地掩蔽与图像内容相关的词汇，从而创建掩蔽响应。
图像检索：使用原始视觉编码器将训练集中所有图像嵌入，并通过余弦相似度检索与输入图像相似的前k个图像。
诱导幻觉：基于掩蔽响应和检索到的图像生成候选响应，若其与选择的响应之间的相似度低于阈值，则将其标记为拒绝响应。
偏好优化：
将生成的偏好数据集用于微调VLM，采用rDPO目标函数，该函数整合了文本和视觉偏好信号，以增强对齐过程的效果。
通过以上方法，RE-ALIGN不仅能够更有效地减轻幻觉现象，还能在多个VQA任务中提升模型的整体性能。
本文通过三个类别的实验验证了RE-ALIGN的有效性。首先，评估了RE-ALIGN在减轻幻觉和提高VQA任务通用性方面的能力，结果显示其在基准测试中优于多种基线方法。其次，考察了RE-ALIGN在不同模型规模和架构下的对齐效果，发现其在文本到图像及统一模型中均表现出色，且相较于传统方法有显著性能提升。最后，评估了rDPO在偏好优化中的作用，结果表明其在对齐VLM时能够持续超越标准DPO，体现了视觉偏好信号在优化过程中的重要性。
RE-ALIGN的工作原理可以简单理解为一个“智能助手”在学习如何更好地理解和生成信息。首先，它会从大量的图片和文字中找到相关的内容，并在生成回答时有意识地“隐藏”某些信息，以便更好地理解这些信息的上下文。接着，它会检索出与当前问题最相似的图片，帮助自己更准确地回答问题。最后，通过比较生成的回答和真实的回答，RE-ALIGN不断调整自己的学习方式，以便在未来能给出更准确的答案。这个过程就像是一个学生在学习时，不断从老师那里获取反馈，并根据这些反馈来改进自己的学习方法。这样，RE-ALIGN能够在处理视觉和语言信息时，减少错误并提高整体表现。
2025-02-18｜HUST,HorizonRobotics
http ://arxiv.org/abs/2502.13145v1https ://github.com/hustvl/mmMamba
本研究提出了一种名为mmMamba的框架，旨在通过逐步蒸馏技术，将现有的多模态大型语言模型（MLLM）转化为线性复杂度的解码器，仅依赖于解码器，而不需要预训练的循环神经网络或视觉编码器。传统的多模态模型由于其二次计算复杂度，导致在部署时面临挑战。mmMamba通过从高复杂度的解码器模型HoVLE中提取知识，成功实现了模型的简化，同时保持了良好的多模态理解能力。通过这种方法，mmMamba-linear和mmMamba-hybrid两种模型架构分别实现了在速度和内存使用上的显著优化，尤其在处理长序列时，mmMamba-linear表现出20.6倍的速度提升和75.8%的内存节省。
mmMamba的方法主要包括以下几个步骤：
初始化策略：从预训练的Transformer模型中继承参数，并根据Mamba-2架构的需求，仔细初始化新引入的状态空间模型（SSM）特定参数。这一过程确保了模型从一开始就能模仿Transformer的行为。
逐层蒸馏：该方法分为三个阶段：
阶段一：冻结继承的Transformer参数，仅训练新引入的SSM参数，通过均方误差（MSE）损失对齐每一层的行为。
阶段二：解冻并优化所有Mamba-2层的参数，以更好地对齐Transformer的行为。
阶段三：进行端到端的蒸馏，通过KL散度损失实现最终的模型对齐，确保mmMamba能够完全捕捉到教师模型的多模态理解能力。
模型架构：mmMamba-linear完全由Mamba-2层构成，而mmMamba-hybrid则结合了Transformer层与Mamba-2层，以实现更灵活的性能与效率平衡。
在实验部分，mmMamba在多个公共基准测试上进行了评估，涉及六个通用的VLM基准和三个视觉问答任务。实验结果表明，mmMamba-linear在六个基准上超越了现有的SOTA模型EVE-7B，并且在参数数量上减少了2倍，而mmMamba-hybrid则在所有基准上均表现出更优的性能。具体而言，mmMamba-linear在103K的上下文长度下，展示了20.6倍的速度提升和75.8%的内存节省，相比于传统的Transformer模型HoVLE。实验还验证了逐层蒸馏的有效性，表明该方法在多模态任务中具备强大的应用潜力。
mmMamba的工作原理可以简单理解为一种“知识转移”技术。首先，它从一个强大的模型（叫做HoVLE）中“学习”，就像学生从老师那里获取知识一样。这个过程分为三步：第一步是把老师的知识传给学生，但学生只学习新知识；第二步是让学生同时学习老师的所有知识；第三步则是让学生在做具体任务时，尽量模仿老师的表现。通过这种方式，mmMamba能够在保持高效能的同时，减少计算资源的消耗，使得在处理复杂任务时更加迅速和节省内存。这样，即使在需要处理大量信息的情况下，mmMamba也能表现得非常出色。
2025-02-18｜USYD,DeepGlint,Imperial(ICL)
http ://arxiv.org/abs/2502.12513v1https ://garygutc.github.io/RealSyn
本文提出了RealSyn，一个有效且可扩展的多模态交错文档转换范式，旨在提升视觉-语言表示学习的能力。随着移动网络和社交平台的迅速发展，大规模数据的增长为视觉-语言表示学习提供了坚实基础。然而，现有方法主要依赖于配对数据，而现实中非配对数据（如多模态交错文档）尚未得到充分利用。为了充分利用这些未配对文档，研究者们建立了一个现实世界数据提取管道，以提取高质量的图像和文本，并设计了分层检索方法来高效关联每个图像与多个语义相关的文本。通过这些创新，构建了RealSyn数据集，结合了现实和合成文本，并在多个下游任务上取得了显著的性能提升。
RealSyn的构建方法包括以下几个关键步骤：
数据提取：从OBELICS数据集中提取118M个交错图像-文本文档，生成336M张图像和2.13B句子。
图像过滤：实施两阶段过滤，确保数据质量。首先剔除低质量图像，然后利用EVA02-CLIP模型消除感知和语义冗余，最终保留198M高质量图像。
句子过滤：对提取的句子进行质量、语义和冗余过滤，确保剩余句子具有足够的信息量，最终保留0.84B句子。
分层检索：设计分层检索机制，首先在集群间检索相关集群，然后在同一集群内检索语义相关的句子，以优化计算效率。
图像语义增强生成：结合大型语言模型生成合成文本，以补充从现实文本中提取的信息，从而提升图像理解的细粒度。
语义平衡采样：通过对图像-文本对进行语义平衡采样，提升数据集的多样性，更好地学习长尾概念。
在实验部分，研究者对RealSyn数据集进行了全面评估，结果显示其在视觉-语言表示学习中的有效性和可扩展性。通过线性探测和零-shot转移测试，RealSyn在多个下游任务上均表现出色，特别是在与YFCC和LAION等现有数据集的对比中，RealSyn在大多数评估指标上均有显著提高。具体来说，RealSyn在15M、30M和100M规模下的数据集上，分别在多个任务中实现了1.3%至6.9%的性能提升，证明了其在视觉-语言表示学习中的重要性和潜力。
RealSyn的构建方法可以简单理解为几个步骤。首先，研究者从一个大型数据集中提取出很多图像和文本。接着，他们对这些图像进行筛选，确保只有质量好的图像被保留下来。然后，他们也对文本进行过滤，去掉那些不够有用的句子。接下来，研究者设计了一种聪明的检索方式，能够快速找到与每张图像相关的多个文本，这样可以更好地理解图像的内容。为了让生成的文本更准确，他们还使用了大型语言模型来帮助生成更详细的描述。最后，他们通过一种叫做“语义平衡采样”的方法，确保数据集中的信息多样化，能够更好地学习到各种不同的概念。这样，RealSyn就能帮助计算机更好地理解图像和文本之间的关系。

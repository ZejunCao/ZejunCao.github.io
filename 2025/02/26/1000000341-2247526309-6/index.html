2025-02-19｜AlibabaGroup｜⭐️
http ://arxiv.org/abs/2502.13923v1https ://github.com/QwenLM/Qwen2.5-VL
Qwen2.5-VL是阿里巴巴集团Qwen视觉语言系列的最新旗舰模型，展现了在基础能力和创新功能上的显著进步。该模型在视觉识别、物体定位、文档解析和长视频理解等方面取得了重大突破。其独特之处在于能够精确地使用边界框或点进行物体定位，并从发票、表格和图表中提取结构化数据。Qwen2.5-VL引入了动态分辨率处理和绝对时间编码，使其能够处理不同大小的图像和长达数小时的视频，同时实现事件的秒级定位。通过从零开始训练动态分辨率的视觉变换器（ViT）并结合窗口注意力机制，Qwen2.5-VL在保持原始分辨率的同时显著降低了计算开销。该模型不仅在静态图像和文档理解方面表现出色，还能作为一个互动视觉代理，在实际场景中执行计算机和移动设备操作等任务。
Qwen2.5-VL的架构包括三个主要组成部分：大语言模型、视觉编码器和多层感知器（MLP）基础的视觉语言合并器。具体方法如下：
大语言模型：使用预训练的Qwen2.5语言模型作为基础，调整了位置编码以适应多模态理解。
视觉编码器：采用重新设计的视觉变换器（ViT）架构，结合2D旋转位置嵌入和窗口注意力机制，以支持原始输入分辨率并加速计算。输入图像在训练和推理时会被调整为28的倍数，并通过分割成补丁生成特征集。
视觉语言合并器：为了提高效率，首先将空间上相邻的四个补丁特征组合，然后通过两层MLP进行维度投影，以减少计算成本并灵活处理不同长度的图像特征序列。
此外，Qwen2.5-VL还引入了动态分辨率和帧率处理，能够适应多种输入格式，提升了模型对视频内容的理解能力。
在实验部分，Qwen2.5-VL的性能与当前最先进的模型进行了比较。通过多个基准测试，Qwen2.5-VL在数学、视觉问答和文档解析等任务中表现优异。例如，在MMMU基准测试中，Qwen2.5-VL-72B模型得分达到70.2，超越了之前的开源模型。在数学任务中，Qwen2.5-VL-72B在MathVista中获得了74.8的高分，显示出强大的数学能力。在视觉问答方面，该模型在MMBench-EN中得分为88.6，超越了先前的最佳成绩。这些实验结果证明了Qwen2.5-VL在多模态理解和任务执行中的强大能力。
Qwen2.5-VL的工作原理可以理解为一个复杂的系统，主要由三个部分组成。首先，它有一个强大的语言理解部分，像一个聪明的助手，能够理解和生成文本。其次，它有一个视觉部分，能够识别和分析图像，就像人类的眼睛一样。最后，为了让这两个部分更好地合作，它使用了一种叫做多层感知器的技术，能够将视觉信息和语言信息结合在一起。这使得Qwen2.5-VL不仅能看懂图片，还能理解图片中的内容，并根据这些内容进行反应和操作。通过这种方式，Qwen2.5-VL能够处理各种复杂的任务，比如回答问题、识别物体，甚至在计算机和手机上执行操作。
2025-02-19｜UCAS,PKU,BaichuanInc
http ://arxiv.org/abs/2502.13383v1https ://github.com/Aurora-slz/MM-Verify
在多模态推理领域，尽管大型语言模型（LLMs）在处理文本和图像等多种输入上取得了显著进展，但仍然缺乏强大的多模态验证器。为了解决这一问题，本文提出了MM-Verifier和MM-Reasoner两种模型，以增强多模态推理的能力。MM-Verifier通过长推理和更强的验证机制来提升模型的推理能力。研究表明，结合MM-Verifier和MM-Reasoner可以在MathVista基准测试中实现65.3的准确率，超越了现有的强大模型如GPT-4o（63.8）。此外，MM-Reasoner在数据量增加时显示出良好的扩展性，进一步验证了该方法的有效性。
本研究采用了两种新颖的数据合成方法来训练MM-Verifier和MM-Reasoner。具体方法包括：
MM验证数据合成：
第一阶段：通过基于模拟的树搜索生成长链推理（COT）数据。该方法利用模拟算法为每个节点生成多个子节点，并对每个子节点进行模拟，最终通过反馈机制优化数据生成。
第二阶段：使用拒绝采样进一步提升验证的效率。通过对第一阶段生成的数据进行清洗和过滤，确保训练集的高质量，从而提升模型的推理能力。
MM-Verifier的设计：
MM-Verifier使用了一种新的多模态结果奖励模型（ORM），能够有效评估推理过程的每一步，确保最终结果的正确性。
通过这两种方法的结合，MM-Verifier和MM-Reasoner能够在多模态数学推理任务中展示出色的性能。
在实验部分，研究团队对MM-Verifier和MM-Reasoner进行了广泛的评估，使用了MathVista和MathVerse等基准测试。实验设置包括对比多种基线模型，评估不同数据量对模型性能的影响。结果表明，MM-Verifier在MathCheck基准中超越了所有其他模型，MM-Reasoner的性能也随着训练数据的增加而显著提升。具体来说，MM-Verifier在MathVista基准中达到了65.3的准确率，展现了其在多模态推理任务中的强大能力。此外，MM-Reasoner在处理不同规模数据时，表现出良好的扩展性，证明了其在实际应用中的潜力。
在这项研究中，作者提出了一种新方法来帮助计算机更好地理解和解决涉及文本和图像的问题。首先，他们设计了一个系统，叫做MM-Verifier，负责检查计算机的推理过程是否正确。这个系统通过两步来工作：第一步是生成大量的推理示例，第二步是用这些示例来训练MM-Verifier，使它能够更准确地判断答案是否正确。为了让这个过程更高效，研究者还引入了一种叫做拒绝采样的技术，来过滤和选择最有用的数据。通过这些方法，计算机在解决数学问题时，能够比以前更准确、更快速。
2025-02-19｜OsakaU,NICT,NIITokyo｜⭐️
http ://arxiv.org/abs/2502.13606v1https ://sites.google.com/view/lavca-llm/
本研究提出了一种新颖的方法，称为LLM-assistedVisualCortexCaptioning（LaVCa），旨在通过利用大型语言模型（LLMs）生成与人类视觉皮层中每个体素（voxel）相关的自然语言描述，从而深入理解人类视觉功能。尽管现有的编码模型已成功预测了体素的活动，但由于深度神经网络（DNNs）的“黑箱”特性，解释体素反应的机制仍然面临挑战。LaVCa通过数据驱动的方法，生成更为准确和详细的描述，超越了传统方法如BrainSCUBA，能够更好地捕捉到体素的视觉选择性和其内部及之间的细微差异。这一方法的提出不仅为理解人脑的视觉表示提供了新的视角，也为下一代计算机视觉模型的发展奠定了基础。
LaVCa方法分为四个主要步骤：
构建编码模型：为每个参与者构建体素级的编码模型，以预测观看自然图像时的脑活动。这一过程使用了来自CLIP的视觉特征。
识别最佳图像集：通过计算体素的编码模型与外部图像数据集的内积，识别出每个体素最能激活的前N幅图像，形成“最佳图像集”。
生成图像描述：利用多模态LLM（MLLM）为这些最佳图像集生成描述性文本。这一步骤允许LLM对图像进行理解并生成相应的自然语言描述。
生成体素描述：从生成的图像描述中提取关键词，并通过“句子构建器”将这些关键词整合成简洁的体素描述，形成最终的体素级自然语言描述。
在实验部分，研究者使用了NaturalScenesDataset(NSD)进行数据收集，涉及多个参与者在观看约10,000幅图像时的脑部活动。通过对比LaVCa与现有方法（如BrainSCUBA），研究者评估了生成的描述在体素活动预测准确性上的表现。结果显示，LaVCa在句子和图像层面均显著优于BrainSCUBA，且在描述的多样性和细致性方面表现突出。实验还通过量化分析展示了LaVCa在不同区域的表现，强调了其在理解视觉皮层功能方面的潜力。
LaVCa的方法可以简单理解为一个逐步生成描述的过程。首先，研究者会为每位参与者的脑部活动建立一个模型，这样可以预测在观看特定图像时，脑中的哪些区域会被激活。接下来，他们会找出每个脑区域最能激活的几张图像。然后，使用一种强大的语言模型，研究者会为这些图像生成描述，比如说“这是一只白色的狗在草地上奔跑”。最后，他们会从这些描述中提取关键词，并将它们组合成一个简短的句子，形成对该脑区域的最终描述。这样的方法让我们能够更好地理解大脑是如何处理视觉信息的，并且提供了更丰富、更准确的描述。
2025-02-19｜DartmouthCollege｜NAACL2025
http ://arxiv.org/abs/2502.13363v1
在视频字幕生成领域，传统方法通常需要复杂的设计以适应视频输入，且开发成本高昂。本文提出了一种创新的方法，通过将现有的图像文本模型BLIP-2进行简单的调整与后训练，成功将其转变为高效的视频字幕生成器。这一方法仅使用了6000对视频文本数据，远低于其他模型所需的数百万数据对，同时在多个标准测试集上取得了优越的性能，尤其是在MSR-VTT和MSVD数据集上排名第二。研究的核心在于探讨如何在资源有限的情况下，利用轻量级的图像模型实现视频字幕生成，挑战了对复杂视频特定设计的依赖。
本研究的核心方法包括三个关键方面：
模型规模优化：通过对BLIP-2模型进行适当的调整，优化其规模。我们发现中等规模的语言模型在特定任务上的微调性能优于大型模型，表明针对性优化比单纯增加模型规模更有效。
数据效率最大化：利用大量的图像文本对进行预训练，使得模型能够在视频任务上实现转移学习。经过6000对视频文本对的后训练，模型在视频字幕生成上表现出色，证明了数据的高效利用。
强化学习的引入：采用强化学习方法，直接优化与人类语言质量标准相符的非可微分CIDEr指标，确保生成的字幕更符合人类的描述习惯。
在实验部分，研究团队通过多轮实验验证了上述方法的有效性。首先，通过对模型组件的不同训练配置进行消融实验，评估了各个部分在视频字幕生成中的适应性和性能。其次，比较了不同大小的语言模型在生成视频字幕时的表现，发现中型模型如Flan-T5-XL-3B在多个指标上表现最佳。此外，实验还探讨了使用不同分辨率视频对模型性能的影响，结果显示低分辨率视频同样能够有效生成高质量字幕，进一步降低了计算资源的需求。
在这项研究中，研究者们发现可以用一种简单的方法将现有的图像处理模型转变为视频字幕生成模型。具体来说，他们通过几个步骤来实现这一目标。首先，他们优化了模型的规模，发现中等大小的模型在处理特定任务时表现得更好。接着，他们使用了大量的图像和文本数据来训练这个模型，让它能够理解视频内容。最后，他们引入了一种新的学习方式，这种方式帮助模型生成的字幕更符合人类的表达习惯。通过这些步骤，研究者们成功地提高了视频字幕生成的效率和质量，同时减少了所需的计算资源。
2025-02-18｜YorkU,VectorInstitute,SamsungAICentreToronto,GoogleDeepMind
http ://arxiv.org/abs/2502.13335v1https ://geomvi.github.io
本研究提出了一种新颖的几何感知扩散模型，旨在解决多视角场景的补全问题，特别是在图像部分被遮挡的情况下。传统的3D场景修复方法通常依赖于3D辐射场来确保跨视角的一致性，但这些方法往往会导致模糊的图像输出。为此，我们的模型不再依赖于显式或隐式的辐射场，而是通过学习空间中的跨视角信息融合，生成几何和外观一致的图像。我们的几何感知条件生成模型能够在仅有少量视角的情况下进行修复，这一特性在以往的方法中较少见。通过在两个数据集SPIn-NeRF和NeRFiller上进行评估，我们的方法在3D修复任务中表现出色，尤其是在少视角修复的场景中。
我们的方法包括几个关键步骤：
场景几何估计：我们使用DUSt3R模型来重建场景几何，能够有效地估计深度和相机参数。这个步骤为后续的图像修复提供了必要的几何信息。
几何感知图像修复：我们设计了一个条件扩散模型，该模型在进行图像修复时，能够同时利用来自参考视图的几何和外观线索。通过将参考图像的外观和几何信息映射到目标视图中，我们能够实现高质量的图像修复。
自回归修复过程：在每次迭代中，我们选择尚未修复的图像，并基于其他视图进行修复。这个过程不仅更新了图像内容，还不断优化场景几何，确保修复结果在多视角间的一致性。
我们在三个主要任务上评估了所提出的方法：狭基线物体去除、宽基线场景补全和少视角修复。实验结果表明，我们的方法在多个指标上超越了现有的最先进技术，尤其在图像清晰度和一致性方面表现突出。对于狭基线任务，我们的方法通过单视图修复实现了高质量的结果；而在宽基线任务中，尽管面临更大的挑战，我们的方法依然保持了较高的性能。对于少视角修复任务，我们的模型展现了出色的鲁棒性，能够在缺乏完整视角信息的情况下生成高质量的修复图像。
我们的修复方法可以简单理解为一个三步走的过程。首先，我们用一种叫做DUSt3R的工具来“看”整个场景，了解每个物体的深度和形状。接下来，我们构建了一个聪明的修复模型，它不仅能记住场景的外观，还能利用这些外观信息来帮助填补被遮挡的部分。最后，我们通过一个循环的方式，逐步修复每个视角的图像，每次修复后都会更新场景的几何信息，确保所有视角之间的修复效果都是一致的。这样的设计使得即使只有少量视角，我们也能够生成清晰、自然的修复图像。
2025-02-19｜SaarlandU,MPI-INF
http ://arxiv.org/abs/2502.13968v1https ://vcai.mpi-inf.mpg.de/projects/separable-recon/
本研究提出了一种新颖的多视角可分离三维重建方法，旨在解决多个相互作用物体的重建问题。传统的三维重建方法往往专注于单一物体，忽视了多个物体之间的相互遮挡和交互边界的复杂性。我们的方法能够在处理严重遮挡和交互时，清晰地分离物体表面，避免几何体的交叉渗透，并实现观察场景的全新视角合成。通过引入一种新型的神经隐式方法，我们可以同时重建两个物体的几何形状和外观，且该框架是端到端可训练的，利用新颖的α混合正则化技术确保几何体之间的良好分离。本文还引入了一个新的数据集，专注于人类与物体之间的紧密交互场景，以验证我们方法的有效性。
我们的方法包括以下几个关键步骤：
场景编码：首先，使用共享的多分辨率哈希网格对多视角图像进行编码，以捕获场景中的几何和颜色信息。
几何体分离：通过两个独立的多层感知机（MLP）头来解码为两个独立的有符号距离场（SDF），分别对应于两个相互作用的物体。
透明度计算：为每个物体计算个体透明度，确保在渲染过程中能够清晰分离物体边界，避免几何体的交叉渗透。
颜色渲染：通过体积渲染技术，结合每个点的透明度和颜色信息，生成最终的图像输出。
训练优化：使用新的α混合正则化损失函数，确保每个物体的透明度在重叠区域内保持一致，从而减少交叉渗透现象。
我们在多个数据集上对所提出的方法进行了评估，包括人类与物体、物体与物体之间的交互场景。实验结果表明，我们的方法在重建质量上显著优于现有的最先进技术，如ObjectSDF++和SegmentedNeuS2。我们使用了峰值信噪比（PSNR）、结构相似性指数（SSIM）和学习感知图像块相似性（LPIPS）等指标来评估新视角合成的效果，并通过方向性Chamfer距离评估三维重建的准确性。实验结果显示，在处理不同规模和复杂度的物体时，我们的方法表现出更好的几何体重建能力和更少的表面变形。
我们的方法可以简单理解为一个三步走的过程。首先，我们通过多个相机拍摄的图像来获取场景信息，就像用不同的角度观察同一个物体。接下来，我们把这个信息转换为两个独立的形状模型，分别代表两个相互作用的物体。然后，我们计算每个物体的透明度，这样在渲染图像时，能够确保它们的边界清晰可见，不会相互穿透。最后，我们利用这些信息生成最终的图像，确保每个物体的外观都被准确重建。通过这种方法，我们不仅能看到物体的形状，还能清楚地分辨它们之间的关系。

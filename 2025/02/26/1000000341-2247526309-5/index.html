2025-02-19｜Tel-AvivU,NVIDIA｜⭐️⭐️
http ://arxiv.org/abs/2502.13951v1https ://ip-composer.github.io/IP-Composer/
IP-Composer是一种新颖的图像生成方法，旨在通过结合多个视觉概念来创造出独特的图像。该方法的核心在于利用CLIP（ContrastiveLanguage-ImagePre-training）模型的嵌入空间，允许用户从一组输入图像中提取特定概念，并将这些概念合成为新的图像。与传统的文本引导生成方法相比，IP-Composer能够更精确地控制图像的视觉细节，避免了文本描述中常见的模糊性和不确定性。此外，IP-Composer不需要进行昂贵的训练过程，这使得其在实际应用中更加灵活和高效。通过对不同视觉来源的概念进行组合，IP-Composer不仅支持多样化的创意表达，同时也为内容创作者提供了一个强大的工具，帮助他们在创作过程中实现更高的自由度和创造力。
IP-Composer的实现包括以下几个关键步骤：
构建概念投影矩阵：首先，收集描述特定概念的文本，通过大型语言模型（LLM）生成一组涵盖该概念属性的文本描述。接着，使用CLIP的文本编码器将这些文本转换为嵌入向量。
提取概念方向：通过对文本嵌入矩阵进行奇异值分解（SVD），识别出与所需概念相关的方向。这些方向用于构建概念的投影矩阵，帮助提取特定概念的嵌入。
组合嵌入生成新图像：使用提取的概念嵌入与参考图像的嵌入进行组合，形成新的复合嵌入。最后，将该复合嵌入输入到预训练的IP-Adapter模型中，生成包含多个视觉概念的新图像。
这一方法的优势在于能够灵活处理多种概念，并且在生成过程中保留了输入图像的特征，从而实现更高的图像生成质量。
在实验部分，研究团队通过多种场景验证了IP-Composer的有效性。首先，通过定性结果展示了该方法在生成图像时的多样性和创新性，能够成功合成出具有不同属性的图像，例如在现有场景中插入新对象、改变时间场景等。其次，定量评估表明，IP-Composer在多个任务上均表现出色，尤其是在与其他基于训练的方法比较时，显示出更高的准确性和鲁棒性。此外，用户研究结果也表明，参与者普遍更倾向于选择IP-Composer生成的图像，认为其更好地结合了输入图像的视觉概念。总体来看，实验结果证明了IP-Composer在创意内容生成中的潜力与优势。
IP-Composer的工作原理可以简单理解为一个“拼图游戏”。首先，我们从不同的图像中提取出想要的部分，比如一朵花的颜色或一只动物的姿势。然后，我们使用一种智能程序（类似于一个聪明的助手），让它帮助我们找到这些部分的描述和特征。接下来，我们把这些特征组合在一起，就像把拼图块拼成一幅新的画。最后，利用这些组合的特征，我们能够生成出一幅全新的图像，这幅图像不仅包含了我们想要的元素，还保留了原始图像的风格。这样，我们就可以轻松地创造出有趣和独特的图像，而不需要从头开始绘制每一个细节。

在图像生成领域，FlexTok是一种新颖的图像编码器，旨在将2D图像转换为可变长度的1D令牌序列。这种方法不仅提高了图像生成的效率，还允许生成模型根据图像的复杂性动态调整生成的令牌数量。传统的图像令牌化方法通常依赖于固定的2D网格，这限制了模型对不同图像复杂性的适应能力。FlexTok通过引入灵活的令牌长度和层次化的表示，能够以粗到细的方式逐步生成图像内容，从而在生成过程中实现更高的灵活性与质量。
FlexTok的核心方法包括以下几个方面：
灵活长度的令牌化：FlexTok可以将图像压缩为1至256个令牌，具体数量取决于图像的复杂性。通过结合嵌套丢弃和因果注意力机制，FlexTok能够生成有序的令牌序列，从而实现高质量的重建。
视觉词汇的构建：与传统的2D令牌化方法不同，FlexTok采用层次化的方式生成令牌，使得高层语义信息优先被编码，而后续令牌则逐步增加细节。这种方法确保了即使在使用较少令牌的情况下，也能有效满足生成条件。
端到端的训练：FlexTok使用整合的流模型作为解码器，能够处理变长度的令牌序列，确保在极端压缩比例下仍能保持重建质量。通过对令牌进行量化和噪声处理，模型能够生成符合条件的高质量图像。
在实验部分，FlexTok的性能通过与其他令牌化方法（如TiTok）进行比较来评估。实验结果显示，FlexTok在使用8至128个令牌的情况下，FID（FréchetInceptionDistance）值低于2，表现优于TiTok，并与当前最先进的方法相匹配。此外，FlexTok还在文本条件的图像生成任务中表现出色，能够根据不同的文本提示生成高质量的图像。通过对不同数量的令牌进行生成，FlexTok展示了其在满足条件生成方面的灵活性和高效性。
FlexTok的工作原理可以想象成一个智能的图像拼图游戏。首先，它把一幅完整的图像分解成许多小块（这些小块就是“令牌”）。不同于传统的拼图方法，FlexTok可以根据图像的复杂程度来决定使用多少小块。如果图像比较简单，可能只需要几块就能拼成一幅不错的图；而对于复杂的图像，可能就需要更多的小块才能拼出完整的细节。FlexTok通过一种聪明的方式来安排这些小块的顺序，先放上大概的形状和颜色，再逐步加上细节，这样生成的图像既快速又清晰。总的来说，FlexTok让图像生成变得更加灵活和高效。
2025-02-19｜BNU,Baidu｜⭐️
http ://arxiv.org/abs/2502.13855v1
MagicGeo是一种新颖的框架，旨在从文本描述自动生成几何图形，具有显著的应用潜力，尤其是在教育和学术领域。传统的几何图形生成方法通常依赖于手动输入，既耗时又资源密集，而MagicGeo通过将图形生成过程视为一个坐标优化问题，避免了这些不足。该框架利用大型语言模型（LLMs）强大的语言处理能力，并结合形式化数学求解方法，确保生成的几何图形具有几何正确性。此外，MagicGeoBench作为一个新的基准数据集，包含220个几何图形描述，展示了MagicGeo在定性和定量评估中的优越性，超越了现有的生成方法。
MagicGeo的生成过程分为三个主要阶段：
自动形式化（AutoformalizationwithLLM）：通过大型语言模型（LLMs）将自然语言输入转换为一个结构化的形式化表示，定义点和相关的几何约束。这一过程利用了LLMs在数学概念形式化方面的能力，确保输入的几何关系被准确捕捉。
求解与验证（SolverwithVerification）：在这一阶段，使用计算几何原理来寻找满足所有约束的解。如果未能找到有效解，系统会回退到自动形式化阶段，重新提取必要信息。这一反馈机制有助于提高生成的准确性。
坐标感知生成（Coordinate-awareGeneration）：利用提取的点坐标生成TikZ代码，作为几何图形的中间表示。这种方法确保生成的图形在结构上清晰，并与输入文本保持高度一致。
为了评估MagicGeo的性能，研究团队构建了MagicGeoBench数据集，并与两种基准模型进行了比较。实验结果表明，MagicGeo在几何图形生成的定性和定量评估中均表现优异，尤其是在图形的准确性和视觉质量方面。此外，实验还探讨了验证机制和求解器的有效性，结果显示引入验证步骤显著提高了生成的准确性。通过用户研究，参与者对MagicGeo生成的图形表现出更高的偏好，进一步验证了其在实际应用中的有效性。
MagicGeo的工作原理可以简单理解为一个三步走的过程。首先，它会把用户输入的文字描述转换为一种机器能理解的形式，像是把复杂的几何关系翻译成一组规则。接着，系统会利用这些规则来寻找符合条件的点位置，这就像在解数学题一样。如果找不到合适的解，系统会回到第一步，重新解析描述。最后，MagicGeo会把这些点的位置转化为图形代码，生成出清晰的几何图形。这个过程的好处在于，它不需要手动绘图，能够快速生成准确的几何图形，非常适合教育和学术使用。
2025-02-18｜NationalTaiwanU,NVIDIA｜⭐️🟡
http ://arxiv.org/abs/2502.13234v1https ://b09902097.github.io/motionmatcher/
本文提出了一种名为MotionMatcher的运动定制框架，旨在通过运动特征匹配来微调预训练的文本到视频（T2V）扩散模型。虽然现有的T2V扩散模型在根据文本提示合成逼真视频方面表现出色，但仅依赖文本描述往往无法精确控制对象的运动和摄像机的构图。MotionMatcher通过使用用户提供的参考视频作为运动指导，解决了这一问题。与传统的微调方法不同，MotionMatcher在特征层面进行调整，避免了从参考视频中泄露内容，同时有效捕捉复杂的运动信息。实验结果表明，该框架在运动定制性能上达到了最先进的水平，验证了其设计的有效性。
MotionMatcher的核心是通过运动特征匹配来实现运动定制，具体方法包括以下几个关键步骤：
问题定义：给定一个参考视频和相关的文本提示，调整预训练的T2V扩散模型，使得生成的视频能够准确复制参考视频中的对象运动和摄像机构图。
特征提取：使用预训练的T2V扩散模型作为特征提取器，提取参考视频和生成视频的运动特征。这一过程利用了两个重要的注意力机制：交叉注意力（CA）图和时间自注意力（TSA）图，前者用于描述摄像机构图，后者用于捕捉对象运动。
运动特征匹配：通过最小化提取的运动特征之间的L2距离，确保生成视频中的运动与参考视频一致。
低秩适应（LoRA）微调：在保持预训练知识的同时，通过微调模型的低秩参数来实现运动定制，确保生成的视频既符合文本提示，又忠实于参考视频的运动特征。
为了评估MotionMatcher在运动定制方面的能力，研究团队收集了42个视频-文本对，涵盖了多种运动类型。实验中，MotionMatcher与四种基线方法进行了比较，包括传统的微调方法和训练无关的方法。使用多个自动化评估指标（如CLIP-T、图像奖励、帧一致性和运动差异）对生成视频的质量、文本对齐和运动一致性进行了定量评估。此外，还进行了用户研究，以获取对视频质量、文本对齐和运动一致性的主观评价。实验结果显示，MotionMatcher在所有评估指标上均优于基线方法，证明了其在复杂运动控制方面的有效性。
MotionMatcher的工作原理可以简单理解为一个“视频运动复制器”。首先，用户提供一个参考视频，MotionMatcher会分析这个视频中物体的运动和摄像机的移动。接下来，它利用一个强大的预训练模型来提取出这些运动的特征信息，而不是直接对视频的每一帧进行逐像素的比较。这样可以避免在生成新视频时出现不必要的视觉干扰。最后，MotionMatcher会根据用户的文本描述来生成新的视频，确保新视频中的物体运动和摄像机的视角与参考视频保持一致。通过这种方式，用户不仅可以控制视频中的内容，还能精确地调整运动效果，使得生成的视频既真实又符合预期。
2025-02-19｜Stanford｜⭐️

本研究提出了一种名为MoVer的运动验证领域特定语言（DSL），用于检查运动图形动画的时空属性。随着大型视觉-语言模型的发展，尽管这些模型能够从文本提示生成运动图形动画，但它们常常无法完全捕捉到提示中描述的所有时空特性。MoVer通过将这些特性表示为一阶逻辑谓词，提供了一种有效的验证机制。我们的工作不仅实现了运动图形的合成，还通过MoVer程序对生成的动画进行验证，确保动画符合输入的文本提示。我们构建了一个包含5600个文本提示和相应MoVer程序的合成数据集，并展示了该系统在迭代修正动画方面的有效性。
MoVer语言的设计基于认知心理学的研究，重点在于如何用形式逻辑表示自然语言中的时空概念。具体方法包括：
谓词定义：MoVer通过七个原子属性（如运动类型、方向、大小等）定义运动的基本特征，并使用逻辑运算符组合这些属性。
时空关系：采用艾伦的时序间隔代数和矩形代数，MoVer实现了多种时空关系的谓词，能够处理复杂的空间和时间关系。
执行引擎：MoVer的执行引擎能够对SVG格式的运动图形动画进行验证，检查每个逻辑谓词在动画中的有效性，并生成验证报告，指出哪些谓词成功，哪些失败。
迭代修正：通过将验证报告反馈给LLM（大语言模型），系统能够自动修正动画，提升生成的动画与文本提示的一致性。
为了评估MoVer的有效性，我们构建了一个包含5600个文本提示及其对应MoVer程序的合成数据集。实验结果表明，在没有迭代修正的情况下，系统能够成功生成58.8%的符合要求的动画；而经过多达50次修正迭代后，这一成功率提升至93.6%。我们还比较了MoVer与传统的语义解析器，发现MoVer在处理复杂的时空关系时表现更为出色。此外，用户可以根据验证报告手动调整动画，进一步提高生成内容的准确性和满意度。
在我们的研究中，MoVer是一种帮助检查动画是否符合描述的工具。想象一下，你告诉一个程序要让一个橙色圆圈移动到一个矩形的上面，并顺时针旋转字母”H”。MoVer会把这个描述转换成一系列简单的规则，比如检查圆圈的颜色和形状、字母”H”的旋转角度等。然后，它会逐帧查看动画，看看这些规则是否都满足。如果某个规则没有通过，MoVer会告诉你哪里出错了。更厉害的是，这个工具可以把这些错误信息反馈给生成动画的程序，让它自动进行修正。这样，用户不仅能得到符合要求的动画，还能通过简单的描述来实现复杂的动画效果。
2025-02-19｜KAIST｜⭐️

视频帧插值（VFI）旨在生成连续输入帧之间的中间视频帧。近年来，事件相机作为一种新型生物启发的传感器，因其能够以微秒级的时间分辨率捕捉亮度变化而受到关注。尽管已有多项研究利用事件相机提升VFI的性能，但现有方法往往仅依赖事件数据或其近似值来估计双向帧间运动场，无法有效处理复杂的真实世界运动场。为了解决这一问题，本文提出了一种新颖的基于事件的VFI框架——交叉模态不对称双向运动场估计（EIF-BiOFNet），该方法结合了事件和图像的特征，直接估计帧间运动场。此外，我们还开发了一种交互式注意力框架合成网络，能够高效利用基于扭曲和合成的互补特征，并构建了一个大规模的事件基VFI数据集ERF-X170FPS，以克服现有数据集的局限性。实验结果表明，所提出的方法在多个数据集上显著优于现有最先进的VFI方法。
本文的方法主要由两个部分组成：EIF-BiOFNet和交互式注意力框架合成网络。具体方法如下：
问题定义与概述：
输入为第一帧和最后一帧以及它们之间的事件流。我们将事件流分为左侧事件和右侧事件，以便在任意时间步进行插值。
EIF-BiOFNet架构：
该网络采用多尺度金字塔结构，利用事件和图像特征进行双向运动场的直接估计。
通过事件体素网格将事件数据转化为网络输入，直接估计不对称的双向运动场。
通过多个轻量级模块对特征进行处理，以提取关键信息并实现高效的运动场估计。
交互式注意力框架合成网络：
该网络结合了扭曲和合成方法的优点，通过自注意力机制和交叉注意力机制来提高帧合成的质量。
通过多头自注意力捕捉长距离依赖关系，增强了对复杂运动的处理能力。
我们在多个数据集上对所提出的方法进行了实验，包括合成事件数据集和真实事件数据集。实验步骤如下：
数据集构建：
构建了ERF-X170FPS数据集，包含高帧率和多样的场景，以验证方法的有效性。
性能评估：
通过PSNR和SSIM等指标定量评估方法在不同数据集上的表现。
与现有的最先进方法进行比较，验证所提方法的优越性，尤其是在处理大范围和非线性运动时。
结果分析：
实验结果显示，所提方法在多个基准数据集上均显著优于现有VFI方法，尤其是在大幅度运动场景下表现突出。
在这项研究中，我们提出了一种新方法来改善视频帧插值的效果。简单来说，视频帧插值就是在两帧视频之间插入一帧新图像，使得视频播放更加流畅。我们的创新之处在于使用了一种新型的相机，它可以非常快速地捕捉到场景中的亮度变化。我们的方法分为两个主要部分。第一部分是一个叫做EIF-BiOFNet的网络，它能够直接从事件和图像中提取运动信息，而不是依赖于传统的方法。第二部分是一个交互式的合成网络，它能够结合不同的特征来生成更高质量的中间帧。通过这些创新，我们的研究在多个测试中表现优异，尤其是在面对快速移动物体时，能够更准确地生成插帧。

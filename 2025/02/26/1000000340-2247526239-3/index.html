303-A3｜组件对齐图生3D场景，均匀分数蒸馏文生3D；基于溅射引导扩散的NVS，基于部件的INR，非最优运输流用于点云生成

主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）发布日期｜2025年2月19日｜周三
2025-02-18｜ShanghaiTech,Deemos,HUST｜⭐️⭐️
http ://arxiv.org/abs/2502.12894v1https ://sites.google.com/view/cast4
CAST（基于RGB图像的组件对齐3D场景重建）是一种创新方法，旨在从单一的RGB图像中恢复高质量的3D场景。当前的技术在处理特定领域的限制和低质量对象生成方面面临挑战。CAST通过提取物体级的2D分割和相对深度信息，结合GPT模型分析物体间的空间关系，使得生成的3D场景更加连贯和真实。该方法不仅生成每个物体的完整几何形状，还通过物理感知的修正步骤确保场景的空间一致性和物理合理性。实验结果表明，CAST在单图像3D场景重建的质量上显著优于现有的基准方法，具有广泛的应用前景，如虚拟内容创作、游戏环境和电影制作。
CAST方法主要包括三个核心步骤：
预处理：首先，利用2D基础模型（如Florence-2）进行物体识别和分割，生成每个物体的描述和边界框。随后，使用GPT-4v进行语义筛选，提取有意义的物体，并生成场景级点云，为后续重建提供结构基础。
感知3D实例生成：该步骤分为两个模块。首先，使用遮挡感知的3D对象生成模块，从部分图像和点云中生成高保真物体网格。其次，进行姿态对齐生成，确保生成的物体与场景中的其他物体空间一致。
物理感知修正：通过构建场景关系图，CAST优化物体的旋转和位移，确保物体间的实际接触和支撑关系，解决物体重叠和漂浮等物理不合理现象。
在实验中，CAST方法在多个数据集上进行评估，包括室内和户外场景。通过与现有的单图像场景重建方法（如ACDC和Gen3DSR）进行对比，CAST在几何质量、视觉质量和物理合理性方面均表现优异。定量评估中，CAST在CLIP评分和用户体验调查中均得到了较高评价，证明其在生成真实感和可编辑性方面的优势。同时，通过用户研究，CAST展现出在虚拟环境创建中的广泛应用潜力。
CAST的工作原理可以简单理解为三个主要步骤。首先，它从一张普通的图片中识别出所有的物体，并为每个物体生成一个“轮廓图”。接着，CAST会利用这些轮廓图来生成每个物体的3D模型，确保这些模型不仅看起来真实，还能在空间中准确地放置在一起。最后，CAST会检查这些物体之间的关系，确保它们不会相互穿透或漂浮在空中，而是像在现实世界中一样自然地相互支撑。通过这种方式，CAST能够从一张图片中创建出一个生动且真实的3D场景。
2025-02-18｜SCUT,SMU｜ICLR2025｜⭐️⭐️
http ://arxiv.org/abs/2502.12640v1https ://github.com/chansey0529/RecDreamer
在文本到三维（Text-to-3D）生成领域，当前的方法常常面临几何不一致性的问题，尤其是在生成的3D资产在不同视角下表现出重复的纹理和图案。这种现象被称为“多面贾努斯问题”（Multi-FaceJanusproblem），其根源在于现有模型在处理不同姿态时存在偏差，导致生成的3D模型在姿态变化时缺乏一致性。为了解决这一问题，本文提出了一种新方法——RecDreamer，通过重塑基础数据分布来实现更一致的姿态表示。该方法的核心思想是通过引入辅助函数来调整数据的先验分布，确保姿态变化均匀分布，而不是偏向某一标准姿态，从而有效改善生成过程中的几何一致性。
RecDreamer的主要目标是通过对预训练扩散模型中的数据分布进行修正，来缓解多面贾努斯问题。具体方法如下：
数据分布修正：通过引入加权函数，调整原始数据密度，使得姿态的边际分布符合目标均匀分布。
均匀分数蒸馏（UniformScoreDistillation,USD）：将修正后的数据分布纳入现有的分数蒸馏框架中，以优化3D表示，确保生成的3D资产在不同姿态下的一致性。
姿态分类器：设计一个轻量级的姿态分类器，该分类器能够在不需要额外训练的情况下，基于预训练特征提取器和用户提供的图像模板，准确估计姿态类别。
后验分布近似：通过动态处理噪声图像估计，确保在优化过程中姿态估计的稳健性和可靠性。
为验证RecDreamer的有效性，研究者进行了多组实验，使用22个不同的对象描述进行比较。实验中，RecDreamer与三种基线方法（SDS、SDS-Bridge、VSD）以及三种针对多面贾努斯问题的开源方法（PerpNeg、Debiased-SDS、ESD）进行了对比。评估指标包括生成质量的FrechetInceptionDistance（FID）、分类熵（cEnt和pEnt）以及CLIP评分，以量化生成结果的质量和一致性。实验结果表明，RecDreamer在生成质量、几何一致性和多视角信息的整合方面均优于其他方法，成功缓解了多面贾努斯问题。
RecDreamer的工作原理可以简单理解为一个“智能调整工具”。想象一下，你在画一个物体的三维模型，但每次从不同的角度看，它的形状都不太一样，甚至有些地方重复。RecDreamer就像是一个助手，它会帮助你调整这个模型的“视角”，确保无论你从哪个方向看，都能看到一致的形状。它首先会分析已有的数据，找出哪些地方需要改进，然后通过一种叫“均匀分数蒸馏”的方法，确保在生成模型时，所有可能的角度都能被均匀地考虑到。最终，它还设计了一个轻便的分类器，帮助识别不同的姿态，这样即使没有额外的训练，它也能准确地判断出物体的方向。通过这些步骤，RecDreamer有效地减少了模型在不同视角下的不一致性，使得生成的3D模型更加逼真和多样化。
2025-02-18｜ETHZürich,DisneyResearch|Studios｜⭐️🟡
http ://arxiv.org/abs/2502.12752v1
在现代计算机视觉和计算机图形学领域，从单一或稀疏图像生成高保真度的新视图（NovelViewSynthesis,NVS）仍然是一个重大挑战。现有的基于像素喷溅（splatting）的方法往往会因喷溅误差而导致几何形状的失真，而基于扩散（diffusion）的方法则常常面临纹理幻觉的问题。为了解决这些问题，本文提出了一种新颖的模型SplatDiff，它结合了像素喷溅和扩散的优点，旨在从单一图像合成高保真的新视图。该模型通过对目标视点和几何一致性的精确控制，克服了现有方法的局限性，并在多个任务中表现出色，展现了良好的跨领域和跨任务性能。
SplatDiff的核心方法包括以下几个关键组成部分：
对齐合成策略（AlignedSynthesis）：通过引入训练对齐（TrainingPairAlignment,TPA）的方法，利用目标视图生成与输入视图对齐的训练对，以确保生成视图在几何和纹理上的一致性。
纹理桥接模块（TextureBridgeModule）：该模块用于聚合来自喷溅视图的特征，以提高纹理的保真度。通过自适应特征融合，纹理桥接模块能够有效地保留输入图像的细节。
喷溅误差模拟（SplattingErrorSimulation,SES）：为了增强模型对喷溅误差的鲁棒性，SES通过在训练过程中模拟喷溅错误，帮助模型学习如何纠正这些错误，从而生成几何一致的新视图。
在实验部分，SplatDiff在多个数据集上进行了评估，包括RealEstate10K和DL3DV-10K。通过与现有的NVS方法进行比较，SplatDiff在单视图合成和稀疏视图合成任务中展现了显著的性能提升。实验结果表明，SplatDiff在保真度、几何一致性和纹理保持方面均优于现有技术。此外，SplatDiff在立体视频转换任务中也表现出色，展示了其跨领域应用的潜力。定量评估使用了多种指标，包括PSNR、SSIM、LPIPS等，确保评估的全面性和准确性。
SplatDiff的工作原理可以简单理解为：当我们想从一张照片生成新的视图时，首先我们会预测出这张照片的深度信息，然后利用这种深度信息将照片中的像素进行重新排列，形成一个新的视图。接着，我们会使用一种特殊的技术来确保生成的新视图在形状和细节上都与原始照片保持一致。为了让生成的纹理看起来更真实，我们还设计了一种“纹理桥”，它能够从多个视图中提取最佳的细节信息，确保新视图的质量。最后，通过模拟一些常见的错误，SplatDiff可以学习如何纠正这些错误，从而生成更加自然和真实的新图像。这样，即使只有一张照片，SplatDiff也能够创造出令人惊叹的新视图。
2025-02-18｜EPFL,BilkentU｜⭐️🟡
http ://arxiv.org/abs/2502.12985v1https ://github.com/cvlab-epfl/PartSDF
在工程应用中，准确的三维形状表示至关重要，尤其是在设计、优化和仿真等领域。传统的三维形状建模往往将对象视为整体，缺乏对其组成部分的明确建模，限制了其在实际设计任务中的应用。为了解决这一问题，本文提出了PartSDF，这是一种基于部件的隐式神经表示框架，能够显式地建模复合形状中的独立部分，同时保持形状的一致性。PartSDF采用简单的单解码器架构，在重建和生成任务中表现出色，超越了现有的监督和非监督基线。此外，该方法在工程应用中作为结构化形状先验，提供了对个别组件的精确控制，确保了整体形状的连贯性。
PartSDF的核心是一个复合形状解码器，该解码器为每个部分输出一个签名距离函数（SDF），并通过潜在向量和姿态参数进行参数化。具体方法可以分为以下几个步骤：
部分编码：每个部分通过潜在向量和姿态参数进行描述，确保部分之间的独立性和一致性。
解码器训练：采用自编码的方式训练解码器，模型参数和潜在向量共同优化，确保全局和部分级别的监督。
非交集损失：引入非交集损失，确保不同部分之间的空间分离，避免重叠。
部分生成：通过训练好的解码器，可以将新模态（如点云或图像）映射到潜在向量和姿态，从而重建形状。
在实验部分，研究者验证了PartSDF在多个任务中的有效性，包括形状重建、生成和优化。使用了三个经过精心策划的数据集，分别为汽车、混合器和椅子。实验结果显示，PartSDF在重建精度、生成能力和部分优化方面均优于现有的基线方法。通过对比不同模型的重建效果，PartSDF在Chamfer距离、体积交集和图像一致性等指标上均表现突出，证明了其作为复合形状先验的有效性和灵活性。
PartSDF的方法可以简单理解为一个智能的三维形状建模工具。首先，它把复杂的物体分解成几个独立的部分，比如一辆车的车身和车轮。每个部分都有自己的特征和位置。然后，这个工具通过学习来理解这些部分是如何组合在一起的，并能在需要时进行修改，比如改变车身的形状而不影响车轮。这个过程就像是拼装乐高，先把每个块的形状和位置搞清楚，再把它们组合成一个完整的模型。最后，这个工具还可以根据用户的需求生成新的物体，或者优化现有物体的设计，比如提高汽车的空气动力学性能。这样，工程师们就能更灵活地设计和调整他们的产品。
2025-02-18｜CUHK,NVIDIA｜ICLR2025｜⭐️
http ://arxiv.org/abs/2502.12456v1https ://research.nvidia.com/labs/genair/not-so-ot-flow
生成3D点云是3D建模中的一个基本问题，广泛应用于形状生成、3D重建和机器人感知等领域。近年来，扩散模型和流匹配方法成为学习3D点云生成模型的主流框架。然而，这些方法在处理点云的置换不变性时存在不足，即点的顺序变化不应影响所表示的形状。为了解决这一问题，本文提出了一种新的生成模型，称为“非最优运输流”，旨在提高大规模点云生成的效率和质量。通过对现有模型的分析，发现它们在处理大规模点云时表现不佳，因此本文提出的模型通过离线计算大规模点云和噪声样本之间的近似最优运输（OT）来解决这一问题，从而提高生成效率。
本文的方法主要分为以下几个步骤：
预计算OT映射：首先，通过计算一个稠密点云和稠密噪声样本之间的OT映射，来获得一组高质量的训练对。这一过程在训练前完成，避免了在线计算的高昂成本。
在线随机子抽样：在训练过程中，从预计算的OT映射中随机抽样数据-噪声对。通过这种方式，确保了生成模型在学习过程中能够获得更直观的流动路径，降低了计算复杂度。
混合耦合方法：为了简化目标流模型的复杂性，本文提出了一种混合耦合方法，通过引入小的高斯噪声来调整OT耦合，使得生成模型在学习时更加稳定。该方法有效地平衡了生成质量和模型复杂性。
在实验部分，本文使用ShapeNet数据集评估所提出的方法，重点比较其与现有扩散模型和流匹配模型的性能。实验结果显示，在不同的推理步骤下，所提模型在生成质量上优于现有方法，尤其在较少的推理步骤下表现突出。此外，在形状补全任务中，所提模型能够在仅五个推理步骤内生成合理的形状，远超其他方法的性能。通过定量和定性评估，验证了混合耦合策略的有效性和所提方法在处理大规模点云生成时的优势。
在我们的方法部分，首先，我们会在训练前计算出一个“最佳匹配”图，这个图可以帮助我们在生成点云时更高效。想象一下，如果我们有很多点和噪声（比如在一个混乱的房间里），我们可以提前找到这些点和噪声之间的最佳连接方式，这样在训练时就能轻松找到合适的搭配。接着，在训练过程中，我们从这些预先计算好的搭配中随机选择一些进行训练，这样就能避免计算复杂的匹配过程。最后，为了让模型学习得更顺利，我们还会在这些搭配中加入一点小的随机噪声，这样可以减少模型在学习时的困难，使得生成的点云更加流畅和真实。通过这些步骤，我们的方法能够在生成3D点云时表现得更好，尤其是在快速生成和高质量输出方面。

2025-02-20｜NTUS-Lab｜⭐️⭐️
http ://arxiv.org/abs/2502.14316v1https ://songlin1998.github.io/Textured-3D-Morphing/
本文提出了一种新的方法用于纹理化3D对象的再生形态变换，利用3D扩散模型来生成平滑且可信的变换序列。与以往的方法依赖于明确的点对点对应关系和形状变换不同，本方法消除了这些额外的需求，能够在没有类别特定对齐数据的情况下进行模型训练。这种方法的创新之处在于其能够处理多种类别的3D对象，生成的变换不仅在形状上自然过渡，同时在纹理上也保持一致性，适用于电影和视觉效果等创意应用。
本研究的核心方法分为三个主要步骤：
基本插值：通过3D扩散模型对源对象和目标对象的信息进行插值。该过程包括：
初始噪声生成：从高斯分布中获取初始噪声，并将其与源对象和目标对象的数据结合。
模型参数调整：对模型进行微调以适应特定的源和目标对象。
条件特征提取：使用CLIP编码器提取源对象和目标对象的条件特征。
平滑性提升：采用注意力融合机制来增强变换的平滑性。通过结合自注意力和交叉注意力的方式，确保在生成过程中保持语义一致性。
可信度提升：引入两种策略来提高生成结果的可信度：
令牌重排：在插值过程中对3D对象的令牌进行重排，以更好地引导隐式对应关系。
低频增强：强调低频信号在生成过程中的重要性，以改善3D表面的质量，减少高频噪声对生成结果的干扰。
实验部分通过对比多种现有方法，展示了所提方法在纹理化3D形态变换上的优势。使用多个指标（如FID、STP-GPT等）对生成结果的质量进行评估，结果显示本方法在平滑性和可信度方面均优于其他基线方法。此外，针对不同的参数设置和策略的组合进行了消融实验，验证了每个策略对最终效果的重要性。实验结果表明，该方法能够有效处理跨类别的3D对象变换，生成的序列在视觉上更为自然且一致。
在这项研究中，研究者们开发了一种新方法，旨在使3D物体之间的变换看起来更自然。想象一下，我们有两个不同的物体，比如一个蓝色的椅子和一个甜甜圈。传统的方法需要我们手动找到这两个物体之间的对应点，这非常麻烦，而且效果也不总是理想。而这项新方法则使用了一种叫做3D扩散模型的技术，能够自动处理这些变换，而不需要明确的对应点。
这个方法的工作流程可以简单地理解为三个步骤：首先，它会生成两个物体的基础信息，然后通过一种智能的注意力机制来确保变换过程中的每一步都平滑过渡。最后，为了让生成的结果看起来更真实，它会优化细节，确保在视觉上没有突兀的变化。总的来说，这种方法让3D物体的变换变得更加简单和高效，同时也能提升视觉效果，适合在电影和游戏中使用。
2025-02-20｜TencentXRVisionLabs,ANU｜⭐️🟡
http ://arxiv.org/abs/2502.14247v1https ://github.com/Tencent/Tencent-XR-3DGen
Pandora3D是一个全面的框架，旨在从多种输入提示中生成高质量的3D形状和纹理，包括单幅图像、多视角图像和文本描述。该框架由两个主要部分构成：3D形状生成和纹理生成。3D形状生成部分利用变分自编码器（VAE）将隐式3D几何体编码到潜在空间中，并通过扩散网络根据输入提示生成潜在表示。同时，研究还探索了一种替代的艺术家创建网格（AM）生成方法，针对简单几何体取得了良好效果。纹理生成则是一个多阶段的过程，包括生成正面图像、多视角图像、RGB到PBR纹理转换以及高分辨率多视角纹理细化。每个阶段都集成了一种一致性调度器，以确保多视角纹理在推理过程中的像素级一致性，从而实现无缝集成。
Pandora3D的生成方法主要分为两个部分：3D形状生成和纹理生成。具体步骤如下：
3D形状生成：
变分自编码器（VAE）：将3D几何体压缩到潜在空间，以便高效表示和处理。
扩散网络：生成基于输入提示的潜在表示，模型经过改进以提高性能。
多分辨率训练策略：逐步增加采样点和潜在嵌入维度，以捕捉更细致的几何细节。
纹理生成：
正面图像生成：根据深度图或输入图像生成正面图像。
多视角图像生成：从正面图像生成多视角RGB图像，并结合位置映射。
多视角PBR图像生成：利用图像到图像的扩散模型生成相应的PBR图像。
高分辨率细化：通过多次上采样步骤提高纹理的视觉质量。
像素一致性强制：确保多视角生成阶段的像素级一致性。
在实验部分，Pandora3D展示了其在处理多种输入格式方面的有效性。通过生成的3D模型和纹理，系统能够忠实地恢复细致的形状和纹理，且在空间中没有漂浮物体。实验结果表明，该框架在生成高质量3D内容方面表现出色，能够处理复杂的输入提示，并生成一致性强的多视角输出。
在Pandora3D的生成过程中，首先通过一个叫做变分自编码器的工具，将3D物体的形状压缩到一个更小的空间中，这样可以更高效地处理这些形状。然后，系统会根据输入的图片或文本提示，利用扩散网络生成这些压缩形状的详细信息。接下来，在纹理生成部分，系统会先生成一个正面的图像，然后再根据这个图像创建多个视角的图像，最终生成高质量的纹理。为了确保不同视角的图像在细节上保持一致，系统会在每个生成步骤中进行调整。这样一来，用户就能得到一个既美观又真实的3D模型，适合用于游戏、动画等多种场景。
2025-02-19｜UCSD｜⭐️
http ://arxiv.org/abs/2502.14129v1
在计算机图形学和视觉领域，从多视角的照片中重建3D物体是一项复杂的任务。尽管近年来神经辐射场（NeRF）在生成高质量3D图像方面取得了显著进展，但其处理成本依然较高。为了解决这一问题，研究者们提出了3D高斯点云（3D-GS）方法，该方法提升了训练效率并支持实时渲染。然而，3D-GS在重建光滑表面时存在局限，主要是因为其使用的球谐函数（SH）无法有效捕捉高频信息。为此，本文提出了一种新的逆渲染框架GlossGau，旨在在保持3D-GS性能的同时，提升光滑表面的重建质量。GlossGau通过显式建模表面法线和双向反射分布函数（BRDF）参数，并结合各类优化技术，成功实现了高效的光滑面逆渲染，且在训练和渲染速度上与传统3D-GS相当。
GlossGau的核心在于其高效的逆渲染框架，具体方法如下：
高斯Surfels重建：首先，通过高斯Surfels方法重建场景中的物体，每个Surfels携带几何属性（位置、协方差和不透明度）和外观属性。
BRDF参数估计：采用各向异性球面高斯（ASG）来建模BRDF参数，以增强光滑表面的表示能力。与传统方法相比，ASG能更好地捕捉光线与表面之间的复杂交互。
可见性计算：利用改进的点基光线追踪方法计算可见性，从而实现真实的阴影效果。
正则化技术：引入物理基础的正则化方法来减少表面属性与光照效果之间的歧义，确保准确的几何表示。
分阶段训练：采用分阶段的训练策略，首先优化光照和材质参数，然后再优化几何属性，以提高重建精度。
在多个数据集上对GlossGau进行了广泛的实验，以评估其性能。实验结果表明，GlossGau在光滑表面的重建上表现优异，尤其是在NeRF合成数据集和光滑合成数据集上。与现有的基于3D高斯点的逆渲染方法相比，GlossGau在视觉质量上保持了竞争力，同时在训练时间和渲染速度上显著减少了开销。具体而言，GlossGau的优化时间减少了多达66%，并且在内存使用上也表现出色，适合实时应用。此外，实验还通过定量指标（如PSNR、SSIM和LPIPS）对比了多种基线方法，结果显示GlossGau在多个指标上均优于对比算法。
在GlossGau的工作中，研究者们采用了一种新颖的方式来重建光滑表面的图像。首先，他们创建了一种叫做“高斯Surfels”的技术，这种技术可以将物体表面的细节转换为数学模型，使得每个小部分都能准确描述其形状和颜色。接着，他们使用了一种特殊的数学工具，称为各向异性球面高斯（ASG），来更好地模拟光线如何在表面反射。这种方法能够捕捉到光滑表面的光泽和反射效果。为了让图像看起来更真实，他们还计算了光线的可见性，这样在渲染图像时就能产生真实的阴影。最后，他们分阶段地训练模型，先调整光照和颜色，然后再优化形状，这样可以确保最终的图像既美观又真实。通过这些步骤，GlossGau能够在处理速度和图像质量上都表现出色。
2025-02-20｜TianjinU｜ICASSP2025｜⭐️
http ://arxiv.org/abs/2502.14178v1https ://nerf-3dtalker.github.io/NeRF-3Dtalker/
本研究提出了一种新颖的神经辐射场（NeRF）模型，命名为NeRF-3DTalker，旨在通过音频驱动合成逼真的3D说话人头部视频。现有的方法主要集中在渲染正面人脸，难以生成多视角的清晰说话人头部。此外，音频信息的丰富性（如节奏和音色）常常干扰模型的学习过程，导致生成的嘴型与音频不同步。为了解决这些问题，NeRF-3DTalker结合了3D形状先验和音频解耦模块，能够将输入音频分离为与三维运动和说话风格相关的特征，从而更好地对齐声学空间与视觉空间。通过这一方法，研究团队成功提升了合成图像的质量和嘴型同步效果。
NeRF-3DTalker的整体架构主要包括四个核心组件：
3D先验提取模块：该模块利用3D可变形模型（3DMM）提取面部的形状和外观特征，包括身份、表情和反射特性。通过此模块，模型能够获得与说话相关的3D信息。
3D先验辅助音频解耦模块：此模块将输入音频解耦为两类特征：与三维运动相关的特征和与说话风格相关的特征。通过这种方式，模型能够更有效地对齐声学信息与视觉信息。
神经辐射场渲染模块：该模块使用解耦后的特征作为条件输入，生成高质量的说话人头部图像。通过多层感知器，模型估计每个三维点的特征向量和密度。
标准化空间设计：为了将生成的框架调整回真实空间，设计了一个标准化空间，从全局和局部语义角度优化生成的图像。该模块通过引入语音相关的动作单元（AUs）信息，增强了生成图像的语义一致性。
在实验中，研究团队使用多个视频数据集，评估了NeRF-3DTalker在生成说话头部视频方面的性能。定量结果显示，该方法在多项指标上超越了现有的最先进技术，包括图像质量（SSIM）、感知差异（LPIPS）和嘴型同步（AU准确率）。此外，进行了消融实验，以验证各个模块对整体性能的贡献。结果表明，3D先验辅助音频解耦模块和标准化空间的引入显著提高了合成质量和同步效果。定性结果也表明，生成的嘴型图像与真实图像的相似度较高，展示了该方法在多视角合成中的优势。
在这项研究中，科学家们开发了一种新技术，能够让计算机根据声音合成出栩栩如生的说话人头部视频。为了实现这一目标，他们使用了一种叫做神经辐射场（NeRF）的算法，结合了3D面部模型和音频处理。具体来说，研究人员将输入的音频分成两部分：一部分与嘴巴的运动有关，另一部分与说话的风格有关。这样，计算机就能更好地理解如何将声音转换为嘴型运动。此外，他们还设计了一个标准化空间，可以帮助计算机将生成的图像调整到更真实的状态。通过这些技术，生成的视频不仅看起来真实，而且嘴型与声音的同步效果也得到了显著改善。
2025-02-19｜UPenn,HKU｜⭐️🟡
http ://arxiv.org/abs/2502.14140v1https ://yh2371.github.io/modskill/
本研究提出了一种新颖的技能模块化学习框架ModSkill，旨在将复杂的全身运动技能分解为可组合的模块化技能，以便独立控制身体各个部分。通过从大规模运动数据集中学习，这些模块化技能可以灵活组合，控制模拟角色实现多样的动作，例如乌赛因·博尔特（UsainBolt）经典姿势，并可无缝应用于各种下游任务。与以往方法相比，ModSkill通过引入技能模块化注意力层，使得低层控制器能够更精确地执行每个身体部分的运动。此外，研究还提出了一种主动技能学习方法，结合生成自适应采样技术，利用大型运动生成模型来增强在复杂跟踪场景中的策略学习。实验结果表明，该框架在精确全身运动跟踪方面超越了现有方法，并且学到的技能嵌入可以有效地应用于多种目标驱动的任务。
ModSkill框架的核心由两个主要组件组成：技能模块化注意力层和低层控制器。具体方法如下：
技能模块化注意力层：该层负责将观察到的策略信息转换为模块化技能嵌入，捕捉每个身体部分的特定技能。通过分析各身体部分之间的关系，生成的技能嵌入能够指导独立的低层控制器。
低层控制器：为每个身体部位设计独立的控制器，利用生成的技能嵌入来产生具体的运动指令。这些控制器基于高斯分布模型，能够灵活调整各个部位的运动。
主动技能学习：通过生成自适应采样策略，该方法从大型运动生成模型中生成新的样本，以便在策略训练中使用。这种方法不仅生成全身运动样本，还针对特定身体部位合成新的运动样本，以增强学习效果。
在实验部分，研究者对ModSkill框架进行了全面的评估，主要测试其在全身运动跟踪和下游任务中的表现。实验结果显示，ModSkill在多个基准测试中均表现优异，成功率和运动误差均优于其他现有方法。在全身运动跟踪任务中，ModSkill能够有效减少跟踪误差，并在多种生成任务（如达到、转向和打击）中展示出良好的技能重用能力。此外，研究者还进行了消融实验，验证了每个组件的有效性，结果表明，模块化设计和生成自适应采样策略显著提高了运动模仿的准确性和灵活性。
在我们的研究中，我们设计了一种新方法，叫做ModSkill，来帮助计算机模拟人类的运动。这个方法把复杂的运动技能分解成小的、可组合的部分，就像乐高积木一样。我们首先创建了一种技能模块化的注意力机制，这个机制就像一个聪明的助手，能够根据身体各个部分的状态，生成适合的运动指令。接下来，每个身体部位都有自己的小控制器，负责执行这些指令。为了让我们的模型更聪明，我们还使用了一种主动学习的方法，从已有的运动数据中生成新的运动样本，以帮助模型更好地学习。通过这些方法，我们的模型在模拟人类运动方面表现得非常好，能够完成各种不同的动作，并且在多种任务中表现出色。

303-A1｜人类偏好对齐与测试时采样，个性化图像生成十年综述，球面密集文生图；无噪声条件化的去噪生成模型；评估NSFW内容消除

2025-02-18｜NJU,AlibabaGroup｜⭐️🟡
http ://arxiv.org/abs/2502.12579v1
在文本到图像生成领域，扩散模型已成为一种主流的生成框架。然而，现有的模型在实现强大的文本-图像对齐、高生成质量和符合人类审美标准方面仍面临显著挑战。为此，本文提出了一种新颖的生成框架CHATS（结合人类对齐优化与测试时采样），首次探索了人类偏好对齐与测试时采样之间的协作。CHATS通过分别建模偏好和不偏好分布，利用代理提示的采样策略来有效利用这两种分布中包含的信息。实验表明，CHATS在多项标准基准测试中超越了传统的偏好对齐方法，展现出卓越的数据效率和生成性能。
CHATS方法主要包括两个核心部分：偏好模型和不偏好模型。具体而言，方法的步骤如下：
偏好模型：该模型通过人类专家对图像进行分类，学习生成图像的偏好分布，旨在捕捉人类所偏好的特征。
不偏好模型：与偏好模型相对，该模型则专注于捕捉不偏好和无条件分布的信息，避免生成不符合人类审美的图像。
训练目标：通过引入一种基于直接偏好优化（DPO）的理论基础目标函数，确保两个模型在训练过程中独立优化其各自的奖励分数。
采样策略：在采样阶段，CHATS结合了偏好模型和不偏好模型的信息，通过代理提示的方式来引导生成过程，确保生成的图像更符合人类的审美标准。
在实验部分，我们使用了两个主要的生成模型：扩散模型和流匹配模型，并对其进行了详细的评估。实验设置包括使用不同的偏好优化数据集（如Pick-a-Picv2和OpenImagePreferences）进行训练和测试。我们与当前最先进的方法Diffusion-DPO进行了比较，评估了CHATS在多项基准测试中的表现。结果显示，CHATS在多个任务中均表现优异，尤其在复杂的多对象和属性任务上，展现出更高的准确性和生成质量。此外，我们还进行了消融实验，以识别CHATS中的关键组成部分，验证了双模型架构和协同采样策略对于提高生成图像质量的重要性。
CHATS的工作原理可以简单理解为一个双模型系统。首先，它有两个模型，一个是专门学习人们喜欢的图像特征（偏好模型），另一个则学习人们不喜欢的特征（不偏好模型）。在训练时，这两个模型分别优化自己的目标，确保生成的图像尽量符合人类的审美标准。然后，在生成图像时，CHATS会结合这两个模型的信息，通过一种聪明的采样策略来创建更好的图像。简单来说，CHATS就是通过“学习喜欢”和“学习不喜欢”来帮助生成更符合人类审美的图像，确保最终的结果既高质量又美观。
2025-02-18｜HIT,PolyU,TomorrowAdvancingLife｜⭐️🟡
http ://arxiv.org/abs/2502.13081v1https ://github.com/csyxwei/Awesome-Personalized-Image-Generation
近年来，生成模型的快速发展极大促进了个性化内容创作的进步。个性化图像生成技术使得用户能够基于少量特定图像和文本描述生成符合其需求的图像。随着生成对抗网络（GAN）、扩散模型（DM）和自回归模型（AR）的演变，这些技术的应用也变得愈加广泛。本文综述了个性化图像生成的最新进展，提出了一个统一框架，涵盖了不同生成模型中的个性化过程，包括反演空间、反演方法和个性化方案。通过对现有方法的比较分析，本文揭示了个性化图像生成技术的共同点和独特之处，并探讨了未来的研究方向和面临的挑战。
在个性化图像生成中，方法可以分为三个主要步骤：反演空间、反演方法和个性化方案。首先，反演空间的选择至关重要，它决定了如何将用户的概念图像映射到生成模型可操作的表示空间。常见的反演空间包括GAN的潜在空间和扩散模型的噪声空间。其次，反演方法涉及将目标图像映射到选定的反演空间，主要有优化基、学习基和混合三种方法。优化方法直接优化潜在表示以最小化生成图像与目标图像之间的差异；学习方法则使用编码器将输入图像转换为潜在表示；混合方法结合了两者的优点，以提高效率和准确性。最后，个性化方案将反演得到的表示与用户的文本描述结合，生成符合用户需求的个性化图像。
本文通过对多个生成模型的个性化技术进行比较，评估了不同方法的效果和性能。实验结果表明，基于GAN的个性化方法在细节保留和图像质量上表现优异，而扩散模型在生成多样性和文本一致性方面具有优势。此外，采用自回归模型的个性化方法则在处理复杂场景时显示出更好的灵活性。通过分析不同模型的优缺点，本文为未来的个性化图像生成技术的发展提供了有价值的见解，并指出了当前研究中的空白和未来可能的研究方向。
在个性化图像生成的过程中，可以将其想象成一个三步走的过程。首先，我们需要选择一个“反演空间”，这就像是为我们要生成的图像选择合适的画布。接下来，我们使用“反演方法”来把已有的图像转换成模型能够理解的形式，这个过程可以通过直接优化或利用已有的知识来完成。最后，我们将这个转换后的信息与用户给定的文本描述结合起来，生成出符合用户需求的个性化图像。简单来说，这个过程就像是先选择了合适的画布，然后用特定的工具把已有的图像转换成模型能理解的形式，最后根据用户的要求来创作出新的图像。
2025-02-18｜DFKI,RPTUKaiserslautern-Landau&DFKIGmbH｜⭐️
http ://arxiv.org/abs/2502.12691v2
近年来，文本到图像生成（T2I）技术取得了显著进展，然而在布局控制和全景图像生成方面仍面临挑战。为了解决这些问题，研究者们提出了密集文本到图像（DT2I）和球面文本到图像（ST2I）模型，但尚未形成统一的解决方案。传统的生成方法无法有效处理球面图像的扭曲和无缝过渡。本文提出了球面密集文本到图像（SDT2I）模型，通过将无训练的DT2I方法与微调的全景模型相结合，提出了多缝合扩散（MSTD）和多全景融合（MPF）两种新方法。此外，为了评估模型的性能，研究团队构建了一个新的合成数据集——Dense-Synthetic-View（DSynView），以测试生成的全景图像的质量和布局一致性。
本文的核心方法是开发SDT2I模型，主要包括以下几个步骤：
多缝合扩散（MSTD）：将StitchDiffusion与区域基础的多扩散（MD）相结合，输入包括一个全局文本提示和多个局部提示的掩码。通过对掩码进行预处理和边缘扩展，确保在缝合步骤中保持一致性。
多全景融合（MPF）：基于PanFusion模型，集成MD到推理过程中。该方法通过将掩码与潜在表示一起旋转，来保持对齐，确保生成的图像在不同分支之间的一致性。
引入引导机制：在生成过程中，采用引导机制来改善对象的放置准确性和布局遵循性，尤其是在前景物体的处理上。
数据集构建：创建DSynView合成数据集，以多样化的掩码和提示来评估模型的表现，确保生成的全景图像具有丰富的布局和内容。
为了评估提出的模型，研究团队进行了系统的实验，包括以下几个方面：
数据集与指标：使用DSynView数据集进行实验，结合多种掩码和提示，生成多种场景的全景图像和透视图像。评估指标包括图像质量、布局遵循性和多样性。
超参数调整：通过系统地调整关键超参数（如掩码大小、步长和引导机制）来优化模型性能，并与基线模型进行比较。
定量与定性分析：对生成图像进行定量分析，比较不同模型在IoU（交并比）、FID（FréchetInceptionDistance）等指标上的表现，同时进行定性分析，观察生成图像的视觉效果和布局一致性。
在这项研究中，科学家们想要创造一种新方法，可以根据文字描述生成360度的全景图像。为了实现这一目标，他们设计了两种主要的方法。首先是多缝合扩散（MSTD），它能够将不同区域的图像拼接在一起，确保各部分之间没有缝隙。其次是多全景融合（MPF），它通过结合不同的图像生成策略，使得生成的图像更加丰富多彩。为了测试这些方法的效果，研究人员创建了一个新的合成图像数据集，里面包含了许多不同的场景和布局。通过这些实验，他们能够比较新方法与旧方法的优劣，确保生成的图像不仅美观，而且符合用户的要求。
2025-02-18｜MIT｜KaimingHe｜⭐️🟡
http ://arxiv.org/abs/2502.13129v1
在现代的去噪生成模型中，噪声条件化被广泛认为是实现有效去噪的关键因素。然而，本研究提出了一个挑战这一常见信念的观点。通过对多种去噪生成模型在没有噪声条件的情况下进行系统比较，研究发现许多模型在缺乏噪声条件化时表现出良好的性能，并且在某些情况下，去噪效果甚至优于有条件的模型。研究者们通过理论分析探讨了去噪过程中去除噪声条件化所带来的误差，并与实证观察结果相一致。此外，研究还提出了一种无噪声条件的模型，在CIFAR-10数据集上取得了竞争性的FID分数，显示出去噪生成模型的潜在新方向。
本研究的核心是对去噪生成模型的重新构思，特别是噪声条件化的必要性。研究者们采用以下方法进行分析：
模型比较：系统地比较了多种去噪生成模型，包括有噪声条件和无噪声条件的模型，探讨其在去噪效果上的差异。
理论分析：分析了去噪生成模型在缺乏噪声条件化时的行为，特别是噪声水平的不确定性及其对生成结果的影响。
误差界限：提出了一个可以在不涉及任何训练的情况下计算的误差界限，该界限依赖于噪声调度和数据集，帮助评估模型在没有噪声条件化时的表现。
无噪声条件模型设计：设计了一种新的去噪生成模型，专门针对无噪声条件的场景，证明其在CIFAR-10数据集上能够实现竞争性的性能。
实验部分主要评估了去噪生成模型在不同噪声条件下的性能，具体包括：
实验设置：在CIFAR-10、ImageNet和FFHQ等数据集上进行无噪声条件和有噪声条件的比较，使用FrechetInceptionDistance(FID)作为评估指标。
模型表现：实验结果显示，许多模型在去除噪声条件化后仍能保持良好的生成质量，部分模型甚至在无噪声条件下表现出更好的FID分数。
误差分析：通过理论分析与实验结果的结合，研究者们发现无噪声条件模型的误差界限与其生成性能之间存在显著的相关性，为未来的模型设计提供了指导。
在这项研究中，科学家们想要了解去噪生成模型是否真的需要噪声条件化。噪声条件化就像给模型提供一个“背景信息”，让它知道输入的数据有多“吵闹”。研究发现，许多模型即使在没有这个背景信息的情况下，也能很好地去噪，甚至在某些情况下表现得更好。为了验证这一点，研究者们进行了大量的实验，比较了不同模型在有噪声和无噪声条件下的表现。他们还提出了一种新模型，专门设计用于无噪声条件的情况，结果显示它在处理图像时性能非常出色。这项研究的发现可能会改变我们对去噪生成模型设计的理解，未来可能会有更多的模型不再依赖于噪声条件化。
2025-02-18｜ECNU｜⭐️
http ://arxiv.org/abs/2502.12527v1
本研究针对文本到图像的扩散模型中不当内容（NSFW）的消除进行了全面评估和分析。尽管这些模型在多个领域展现了出色的创意潜力，但其强大的泛化能力也使得它们在生成图像时可能无意中产生NSFW内容，从而对其安全部署造成潜在风险。为了解决这一问题，研究者们提出了多种概念消除方法，包括数据集过滤、参数微调和后处理修正等。然而，目前对这些方法的有效性缺乏系统性的评估。因此，本研究旨在建立一个全面的基准，系统分析11种最先进的消除方法及其14种变体，以确保在实际应用中能够有效增强T2I模型的安全性和可靠性。
本研究的方法分为三个层面进行评估：任务层面、工具层面和洞察层面。
任务层面：我们设计了一个框架，从六个不同的评估视角分析消除方法的性能，包括消除比例、图像质量、语义对齐、过度消除、显性和隐性不安全提示的影响，以及鲁棒性。
工具层面：进行NSFW数据集的毒性分析，并比较不同NSFW分类器的性能，以提供更深入的洞察和评估指标的汇编。
洞察层面：通过综合各种评估结果，分析影响消除方法性能的潜在因素，并总结研究发现，为未来的研究提供启示和方法论。
在实验部分，我们对11种消除基线方法进行了广泛的测试，涉及五个特定的NSFW主题以及整体NSFW内容。通过比较不同方法在消除特定主题方面的有效性，评估它们在不同场景下的表现。使用VQA分类器进行图像分类，生成的图像被标记为NSFW或安全。实验结果显示，不同方法在消除效果、图像质量和语义对齐方面的表现存在显著差异，特别是在处理显性和隐性不安全提示时。最终，我们提出了一个全面的评估框架，为后续的研究提供了明确的指导。
在这项研究中，我们采用了一种系统的方法来评估如何从生成的图像中消除不当内容。首先，我们从多个角度分析消除方法的效果，比如消除了多少不当内容、生成的图像质量如何，以及这些图像是否仍然与输入的文本描述匹配。其次，我们还对现有的NSFW数据集进行了分析，以了解不同分类器的性能，确保我们使用最准确的工具来判断图像的安全性。最后，通过对实验结果的综合分析，我们总结了哪些方法最有效，并为未来的研究提供了有用的见解。这种方法使得我们能够更好地理解如何在生成图像时提高安全性，避免不当内容的生成。

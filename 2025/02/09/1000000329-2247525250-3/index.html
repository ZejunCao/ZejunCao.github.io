2025-02-03｜DHU｜⭐️
http ://arxiv.org/abs/2502.01626v1
本文介绍了一种新的虚拟试穿模型，名为MFP-VTON（Mask-FreePerson-to-PersonVirtualTry-On），旨在通过扩散变换器技术提高人对人虚拟试穿的效果。传统的虚拟试穿方法通常依赖于标准的服装图像，而在实际应用中，获取这些标准服装图像常常面临困难。MFP-VTON通过构建一个专门的数据集并引入无掩模的框架，旨在简化这一过程，同时保持目标人物的细节不变。研究结果表明，该模型在生成高保真度的虚拟试穿图像方面表现优异，能够在电商行业等多个领域找到应用。
MFP-VTON的实现主要包括以下几个步骤：
数据准备：构建一个专门的个人对个人虚拟试穿数据集，利用现有的服装对人数据集，随机抽取配对实例以生成新的数据对。
无掩模虚拟试穿：采用FLUX.1-Fill-dev作为基础模型，利用其先进的图像修复能力来处理输入图像。与传统方法不同，MFP-VTON避免在目标人物身上遮罩服装区域，从而保留更多的原始细节。
专注注意力损失：引入FocusAttentionLoss来指导模型在训练过程中更有效地关注参考图像中的服装区域及目标人物外部细节。通过优化注意力机制，确保模型能够生成更为自然的试穿效果。
实验部分的设计包括模型的实现、定性和定量比较。模型在VITON-HD数据集上进行微调，并将其与现有的服装对人方法进行比较。通过使用SSIM、LPIPS、DISTS等多种评估指标，定量分析生成图像的质量。同时，通过视觉比较，展示MFP-VTON在保留人物姿态和细节方面的优势。实验结果表明，MFP-VTON在个人对个人虚拟试穿任务中表现出色，生成的图像在多个指标上均优于其他方法。
在MFP-VTON的实现过程中，我们采用了一些简单易懂的步骤。首先，我们从已有的服装和人物图像中随机挑选出一些配对的样本，形成一个新的数据集。接着，我们使用一种叫做FLUX的模型，这个模型能够在图像中填补缺失的部分，而不需要在目标人物身上遮住任何区域，这样可以保留更多的细节。最后，我们引入了一种新的损失函数，叫做FocusAttentionLoss，它可以帮助模型更好地关注到服装和目标人物之间的细节，从而生成更加自然和真实的试穿效果。通过这些方法，我们的模型能够在虚拟试穿中表现得更加出色，给用户带来更好的体验。
2025-02-03｜HIT(SZ)｜IEEETCSVT｜⭐️
http ://arxiv.org/abs/2502.01080v1
在时尚领域，利用生成对抗网络（GAN）进行服装合成的研究逐渐受到关注，尤其是在生成视觉上相互搭配的服装方面。现有的方法大多只能一次合成一件服装，这限制了用户在选择时的多样性。为了解决这一问题，本文提出了一种新的批量服装生成框架BC-GAN，该框架能够同时合成多件视觉上相互搭配的服装。BC-GAN通过引入新的时尚兼容性鉴别器，充分利用服装之间的搭配关系，提升合成结果的时尚兼容性。通过在自建的大规模数据集上进行测试，BC-GAN在多样性、视觉真实性和时尚兼容性方面的表现超过了现有的最先进方法。
BC-GAN的核心在于其框架设计和训练过程。以下是其主要方法步骤：
问题定义：BC-GAN的目标是基于给定的上衣或下装，合成多件兼容的服装。
生成模块：
使用预训练的StyleGAN模型，BC-GAN的生成器由映射网络、编码网络和合成网络组成。
映射网络将随机噪声映射到风格嵌入空间，编码网络将输入的服装与生成的服装合并后编码为风格嵌入，合成网络则生成目标服装图像。
鉴别器设计：
设计了风格嵌入鉴别器，用于判断生成的风格嵌入是否真实，从而确保生成图像的视觉真实性。
采用对比学习方法构建兼容性鉴别器，以有效区分兼容和不兼容的服装对，并为生成器提供反馈。
训练目标：训练过程中，生成器的目标包括视觉真实性、合成多样性和时尚兼容性，结合不同的损失函数进行优化。
在实验部分，研究团队首先构建了一个名为DiverseOutfits的大规模时尚数据集，包含31,631套兼容的服装组合。然后，进行了一系列实验以验证BC-GAN的有效性。实验结果显示，BC-GAN在合成多样化服装方面表现优异，具备较高的视觉真实性和时尚兼容性。通过与现有方法进行比较，BC-GAN在多项评估指标上都取得了显著的提升。此外，研究还进行了消融实验，以验证各个模块对整体性能的贡献。
BC-GAN的工作原理可以简单理解为一个智能的服装设计助手。首先，它会接收一件你喜欢的衣服作为输入，然后利用一个强大的“记忆库”——预训练的StyleGAN模型，来生成多件与之搭配的服装。这个过程就像是在为你挑选合适的搭配，而不是只给你提供一件选择。为了确保这些搭配不仅好看，而且风格相符，BC-GAN还使用了两个“助手”：一个是风格鉴别器，帮助判断生成的服装是否真实好看，另一个是兼容性鉴别器，确保这些服装能够和原始衣服搭配得当。通过这些步骤，BC-GAN能为用户提供多样化、时尚且和谐的服装选择。
2025-02-03｜HIT(SZ),CUHK(SZ),HFUT｜⭐️
http ://arxiv.org/abs/2502.00992v1
本研究提出了一种新颖的框架FCBoost-Net，用于生成多个视觉上协调且多样化的时尚服装组合。传统的时尚组合生成方法通常局限于基于给定服装项生成唯一的服装组合，而缺乏多样性和灵活性。FCBoost-Net通过利用预训练的生成模型，随机合成多组时尚单品，并通过一个创新的时尚兼容性增强模块，逐步提高这些合成服装组合的时尚兼容性。该方法的灵感源于提升算法，能够在多个步骤中不断优化生成的服装组合。大量实验结果表明，FCBoost-Net在视觉真实性、多样性和时尚兼容性方面均优于现有的多模态图像到图像翻译方法。
FCBoost-Net的核心方法可以分为以下几个部分：
问题定义：定义了一个通用的时尚单品集合，包含上衣、下装、包包和鞋子等四个类别，目标是生成多个与给定单品相匹配的服装组合。
生成模块：采用预训练的StyleGAN生成模型，设计了多个生成器，每个生成器负责合成一个特定类别的时尚单品。通过映射网络和合成网络的组合，输入随机噪声，生成视觉上协调的服装组合。
时尚兼容性增强模块：在生成的每一轮中，使用时尚兼容性增强模块来计算当前服装组合的兼容性。该模块通过比较当前服装项与之前生成项的距离，逐步优化服装组合的兼容性，同时保持多样性。
在实验部分，FCBoost-Net在OutfitSet数据集上进行了验证，该数据集包含2万组由时尚专家挑选的视觉上协调的服装组合。实验分为训练和测试两个阶段，使用80%的数据进行训练，20%的数据用于测试。对比了FCBoost-Net与多种基线方法（如MUNIT、DRIT等）在视觉真实性、多样性和时尚兼容性方面的表现。通过多种评价指标（如FID、LPIPS和F2BT）进行量化分析，结果显示FCBoost-Net在所有指标上均表现优异，生成的服装组合在视觉上更具真实性，并且兼容性和多样性显著提高。
FCBoost-Net的工作原理可以简单理解为一个“服装搭配助手”。首先，它会根据用户给定的一些服装单品，随机生成多种不同的搭配选择。接着，FCBoost-Net会通过一个“时尚兼容性检查器”来分析这些搭配的好看程度，确保搭配的服装在视觉上是协调的。在每一轮生成中，这个检查器会不断比较新生成的搭配与之前的搭配，逐步优化每个搭配的兼容性，同时保持搭配的多样性。最终，用户可以获得多个既好看又独特的服装组合，帮助他们找到最适合自己的风格。
2025-02-03｜IITMadras｜⭐️🟡
http ://arxiv.org/abs/2502.01507v1
Text-to-Image(T2I)合成是一项复杂的任务，涉及文本和图像两种模态之间的复杂交互。近年来，许多最先进的方法采用了基于对比损失的预训练图像对齐文本嵌入来启动学习过程。然而，这些嵌入往往是通用的，未针对特定的合成网络进行优化。本文提出了一种新方法，通过端到端的学习方式，专门为T2I合成网络设计文本嵌入。该方法结合了生成和对比训练，使用了两个不同的嵌入：一个优化生成图像的真实感，另一个增强文本与图像的对齐性。通过在三个文本到图像基准数据集（Oxford-102、Caltech-UCSD和MS-COCO）上进行全面实验，结果表明，使用两个独立的嵌入比共享嵌入的效果更佳，并且在与使用预训练文本编码器的其他方法比较时表现出色。
本文提出的“DualTextEmbeddingGAN”（DTE-GAN）框架由三个核心组件组成：1）双文本嵌入设置，2）单阶段生成器G，3）判别器D。首先，输入文本T通过双文本嵌入过程编码为两种句子嵌入：生成器侧句子嵌入S和判别器侧句子嵌入S。双文本嵌入设置由两个独立的双向LSTM编码器和各自的词嵌入组成，分别用于生成和判别过程。生成器G接收噪声向量和两个句子嵌入，生成图像并通过多模态对比损失与真实图像进行比较。判别器D则用于判断图像的真实性，并提取图像特征以计算对比损失。通过这种方法，生成器和判别器的嵌入可以独立优化，分别捕捉生成和判别过程中的不同特征，从而提高合成图像的真实感和文本对齐性。
在实验部分，DTE-GAN在三个数据集上进行评估：Caltech-UCSD鸟类数据集、Oxford-102花卉数据集和MS-COCO数据集。使用InceptionScore(IS)、FréchetInceptionDistance(FID)和R-precision三个指标来评估生成图像的质量和文本-图像对齐效果。结果显示，DTE-GAN在CUB数据集上将FID从14.06降至13.67，在Oxford-102数据集上从40.31降至30.07，且在R-precision指标上表现优异。此外，DTE-GAN的生成图像在视觉上与其他方法进行比较，显示出更好的细节捕捉和一致性，证明了双文本嵌入的有效性和灵活性。
在这项研究中，研究人员开发了一种新的方法来生成图像，方法的核心是使用两种不同的文本嵌入。想象一下，当你描述一只鸟时，可能会用不同的词来表达它的颜色、形状和特征。研究人员将这种想法应用到计算机生成图像的过程中。首先，他们使用一种特殊的程序将文本信息转换成两种不同的表示方式：一种是为了生成图像的真实感，另一种是为了确保生成的图像与文本描述相符。接着，这些信息被送入一个生成器，这个生成器根据这些信息和一些随机噪声生成图像。同时，还有一个判别器在旁边工作，它的任务是判断生成的图像是否真实。通过这种双重嵌入的方法，计算机能够更好地理解文本和图像之间的关系，从而生成更准确和真实的图像。
2025-02-03｜RutgersU,VinAIResearch,MonashU｜ICLR2025｜⭐️🟡
http ://arxiv.org/abs/2502.01441v1https ://github.com/quandao10/sLCT/
近年来，生成模型逐渐受到重视，尤其是在图像和视频生成领域。尽管扩散模型在生成高质量样本方面表现出色，但其在大规模数据集上的应用仍面临挑战。本文提出了一种改进的训练技术，专注于潜在一致性模型（latentconsistencymodels），旨在缩小这些模型与扩散模型之间的性能差距。通过深入分析像素空间和潜在空间之间的统计差异，研究发现潜在空间中的冲击性异常值显著影响模型性能。为了解决这一问题，本文采用了更为稳健的Cauchy损失替代传统的Pseudo-Huber损失，并引入了扩散损失和最优传输（OptimalTransport）匹配，以提升模型的表现。此外，研究还设计了一种自适应缩放调度器和非缩放层归一化（Non-scalingLayerNorm），以更有效地捕捉特征统计并降低异常值的影响。
本文的方法分为几个关键部分：
统计分析：首先，重新实现了现有的潜在一致性训练框架，发现潜在数据包含极端的冲击性异常值，这些异常值在训练过程中导致不稳定。
Cauchy损失：针对潜在空间中的异常值，采用Cauchy损失函数来替代Pseudo-Huber损失，以更有效地抑制极端异常值的影响。
扩散损失：在小时间步上引入扩散损失，以减少一致性训练的过度拟合现象，确保模型在低噪声水平下的性能。
最优传输匹配：采用最优传输技术以降低训练过程中的方差，提高模型的稳定性。
自适应缩放调度器：设计了一种动态调整缩放参数的策略，以便在训练过程中更好地控制模型的鲁棒性。
非缩放层归一化：在网络架构中引入非缩放层归一化，以增强对特征统计的捕捉能力，同时减小对异常值的敏感性。
在实验部分，本文评估了所提出的训练技术在多个数据集上的表现，包括CelebA-HQ、FFHQ和LSUNChurch。通过生成50,000个潜在样本并将其传递至VAE解码器，使用FrechetInceptionDistance（FID）和Recall指标对模型性能进行评估。结果表明，所提出的方法在所有数据集上均显著优于基线模型iLCT，尤其在1-NFE采样下，FID值明显降低，表明生成图像的质量有了显著提升。此外，采用Cauchy损失和扩散损失的实验结果显示，模型在处理潜在空间异常值时的鲁棒性得到了增强，从而实现了高保真度图像的生成。
在本文中，我们提出了一种新方法来改进潜在一致性模型的训练。首先，我们发现潜在数据中有些极端值（我们称之为异常值）会影响模型的表现，因此我们决定使用Cauchy损失，这是一种更能抵抗这些异常值的损失函数。接着，我们在训练过程中引入了扩散损失，尤其是在处理噪声较小的情况时，这能帮助模型更好地学习。此外，我们还使用最优传输技术来减少训练中的波动，使模型更加稳定。为了进一步提升性能，我们设计了一个智能的缩放策略，能根据训练进展动态调整参数，最后，我们还改进了模型的归一化方法，以更好地处理特征数据并降低异常值的影响。通过这些改进，我们成功提高了生成图像的质量，使得模型在潜在空间的表现更接近于当前最好的扩散模型。
2025-02-03｜UCambridge｜⭐️
http ://arxiv.org/abs/2502.01309v1
本研究提出了一种新颖的方法，通过异构图数据对基于扩散的图像合成模型进行条件化。传统的方法通常将条件变量直接嵌入模型架构中，使用交叉注意力层或图像拼接等方式。然而，这些方法在处理多样化和复杂的条件变量时存在局限性。为了解决这一问题，本文引入了异构图像图（HeterogeneousImageGraphs,HIG），将条件变量和目标图像建模为两个互联的图，从而实现对可变长度条件输入及其关系的高效处理。此外，本文还提出了一种保持幅度的图神经网络（GNN），将HIG集成到现有的扩散模型中，以提高生成过程中的语义和关系控制能力。实验结果表明，该方法在多个数据集上表现出色，超越了现有的最先进技术（SOTA）。
本研究的方法主要包括以下几个步骤：
弱点分析：指出现有条件化方法在处理复杂图像合成任务时的不足，特别是在多模态和稀疏输入条件下的表现。
异构图像图（HIG）构建：将图像特征转换为图节点，形成图像图，并将条件图的节点和边与图像图连接，构建条件图。每个节点代表一个特征，边则表示节点之间的关系和依赖性。
模型架构：在现有的扩散模型架构中集成HIGnn（异构图像图神经网络）作为条件网络。该网络能够在图像特征和图像节点之间动态切换，支持对不同输入类型的处理。
信息传播：使用自定义的图卷积操作，通过图神经网络对HIG进行处理，保持幅度并整合来自不同节点的信息，以实现更精细的图像控制和生成。
在实验部分，研究团队在COCO-Stuff和VisualGenome两个数据集上验证了所提方法的有效性。通过对模型的训练和评估，使用多种指标（如FID、DiversityScore和YOLOScore）对生成的图像质量、丰富性和可控性进行了量化分析。实验结果显示，所提方法在各项指标上均优于现有方法，特别是在生成图像的细节和一致性方面。此外，研究还进行了消融实验，以评估不同架构设计选择对模型性能的影响，进一步证明了HIG的有效性。
在我们的研究中，我们提出了一种新方法，旨在通过图形数据来帮助计算机生成更好的图像。我们发现，传统的方法在处理复杂的图像生成任务时有些不够灵活，因此我们创造了一种新的表示方式，叫做异构图像图（HIG）。简单来说，我们把图像和与之相关的条件（如标签或描述）看作两个相互连接的图。这样，我们可以更好地处理输入信息的不同种类和复杂关系。我们的模型能动态切换图像特征和图节点，确保生成的图像不仅质量高，而且能根据不同的条件进行调整。这种方法不仅提高了图像的生成质量，还使得我们能更精确地控制生成的内容，比如改变图像中的物体位置、颜色等。通过实验，我们验证了这种方法在多个数据集上的优越表现。

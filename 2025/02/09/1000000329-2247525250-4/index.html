2025-02-03｜ByteDance｜⭐️⭐️
http ://arxiv.org/abs/2502.01061v1https ://omnihuman-lab.github.io/
近年来，基于扩散变换器（DiffusionTransformer）的技术在视频生成领域取得了显著进展，尤其是在生成真实人类动画方面。然而，现有方法在数据规模扩大和多模态输入的处理上仍存在局限。本文提出了一种新型的模型——OmniHuman，该模型通过混合条件训练策略有效整合多种驱动信号（如音频、视频和姿势），从而显著提升了生成视频的质量与多样性。OmniHuman不仅能够生成高质量的人类视频，还支持多种输入形式，包括面部特写、半身和全身图像，能够处理复杂的手势和物体交互，克服了传统方法在数据利用效率和生成能力上的不足。
OmniHuman模型的核心在于其混合条件训练策略，具体包括以下几个方面：
模型架构：OmniHuman基于先进的扩散变换器（DiT）架构，能够处理多种输入信号，包括文本、音频和姿势。
条件设计：通过结合音频特征、姿势热图和参考图像，OmniHuman能够在生成过程中实现多模态信息的相互作用，增强视频生成的自然性和准确性。
训练策略：采用逐步混合条件训练，首先使用较弱的条件（如文本），逐步引入更强的条件（如姿势），以此优化模型的学习效果。
数据扩展：通过利用多种条件信号，OmniHuman能够有效扩展训练数据，减少因数据过滤而丢失的重要运动模式，从而提升模型的泛化能力。
在实验部分，研究者们详细描述了OmniHuman的实现细节和评估方法。首先，使用了包含18.7K小时人类相关数据的训练集，经过精心筛选以确保数据质量。其次，通过与多种现有方法进行比较，OmniHuman在生成质量（如FID和FVD评分）上显著优于其他模型。此外，研究还进行了多种条件下的消融实验，以验证不同条件组合对生成效果的影响，从而优化训练策略，确保模型在生成过程中能够有效利用各种输入信号。
OmniHuman的工作原理可以简单理解为一个多才多艺的动画生成器。想象一下，你想让一个动画角色说话或唱歌，OmniHuman能从你提供的图片和声音中学习，生成栩栩如生的视频。它的“秘密武器”在于同时使用多种信息，比如音频、姿势和参考图像，这样就能更好地捕捉到角色的动作和表情。而且，OmniHuman还会逐步学习，从简单的任务开始，慢慢增加难度，这样可以确保它在生成视频时能够更自然、更真实。通过这种方法，OmniHuman能够处理不同类型的输入，无论是近距离的面部特写还是全身动作，都能生成高质量的动画效果。
2025-02-03｜AdobeResearch,URochester｜⭐️⭐️
http ://arxiv.org/abs/2502.00972v1https ://yiconghong.me/HTH
近年来，视觉生成领域取得了显著进展，尤其是在图像和视频生成方面。尽管Transformer架构在这一领域表现出色，但其计算复杂度在处理长序列时面临效率挑战。为了解决这一问题，研究者们开始探索状态空间模型（SSM），这些模型在处理长视觉序列时表现出色。本文提出了一种创新的Hydra-Transformer混合模型（HTH），旨在推动SSM在高分辨率图像和长视频生成中的应用。该模型结合了双向HydraSSM与自注意力机制，能够生成高达2K分辨率的图像和360p、8秒的视频，且在推理速度上优于传统Transformer架构。
HTH模型的设计基于以下几个关键方面：
数据编码器：模型通过文本和视觉编码器对输入进行预处理，采用FLAN-T5-XXL语言模型获取文本表示，同时使用3D变分自编码器（VAE）压缩视觉输入，生成低维潜在表示。
去噪模型：HTH的去噪过程由多个模块组成，包括交叉注意力层、令牌混合层（可以是Hydra或自注意力）和前馈网络。交叉注意力层将文本信息与视觉潜在表示结合，以指导生成过程。
扫描模式：为提高模型的表达能力，HTH采用双向扫描模式，交替使用水平和垂直扫描，确保在处理视觉数据时能够更好地捕捉空间和时间特征。
条件机制：通过简单有效的方式，将扩散过程和文本条件结合，确保生成的图像和视频与输入文本保持一致。
在实验中，HTH模型经过多阶段训练，首先在256p数据集上训练图像生成模型，然后在180p和360p的视频数据上进行调整。实验结果表明，HTH模型在生成高分辨率图像和视频方面表现出色，尤其在视觉质量和生成一致性上超过了现有的多种方法。具体来说，HTH在MSCOCO-30K和MJHQ基准测试中均表现优异，生成的图像在FID和CLIP评分上均显示出竞争力。此外，模型的推理效率也得到了显著提升，尤其是在处理长序列时，展示了SSM架构在视觉生成中的巨大潜力。
在我们的研究中，我们构建了一种新的图像和视频生成模型，叫做Hydra-Transformer混合模型。这个模型的工作原理可以简单理解为：首先，它通过文本和图像的编码器来理解输入的信息。然后，它使用一种叫做去噪的技术，逐步改善生成的图像或视频，使其更加清晰和真实。为了更好地处理图像和视频，模型采用了一种特别的扫描方式，就像在阅读一本书时，我们会从左到右、从上到下地扫视内容。最后，模型还能够根据输入的文本指令来生成相应的图像或视频，确保它们与描述相符。通过这些方法，我们的模型不仅生成的图像和视频质量高，而且速度也很快，能够处理更加复杂的视觉任务。
2025-02-03｜ZJU｜⭐️⭐️
http ://arxiv.org/abs/2502.01101v1https ://csfufu.github.io/vid_sketch/
随着生成性人工智能的快速发展，已有研究成功实现了从手绘草图生成静态美术图像的任务。然而，现有方法在生成视频动画方面存在局限，无法利用手绘草图控制动画生成。为了解决这一问题，本文提出了一种名为VidSketch的创新方法，它能够直接从任意数量的手绘草图和简单文本提示生成高质量的视频动画。VidSketch通过引入了基于层级的草图控制策略，使得不同绘画水平的用户在生成过程中能够自动调整草图的指导强度，从而满足多样化的创作需求。此外，本文还设计了一种时空注意力机制，以增强生成视频的时空一致性，显著提高了帧间的连贯性。这一方法不仅降低了视频创作的门槛，还使普通用户能够轻松实现高质量动画的创作。
VidSketch的核心方法包括三个主要部分：手绘草图驱动的视频生成、基于层级的草图控制策略以及时空注意力机制。首先，在手绘草图驱动的视频生成中，系统通过对用户提供的草图序列和文本提示进行处理，生成对应的视频动画。具体步骤如下：
草图处理：用户输入的第一帧草图经过预处理，生成初始图像作为视频生成的参考。
反向扩散过程：通过选择性添加噪声的方式，确保第一帧信息不受损，同时逐步生成后续帧。
生成中间帧：通过线性插值生成中间帧，确保动画的平滑过渡。
接着，基于层级的草图控制策略动态评估草图的抽象水平，并根据用户的绘画技能调整生成过程中的控制强度。这一策略确保了生成动画的质量与用户的创作能力相匹配。最后，时空注意力机制则通过提取关键帧的信息，优化帧间的一致性，进一步提升生成动画的质量。
在实验部分，我们对VidSketch进行了全面的评估，主要包括定量和定性分析。首先，定量实验中，我们使用传统的评估指标如PickScore和均方误差（MSE），并借助VBench工具评估多个方面的表现，如主题一致性、背景一致性和视觉质量。实验结果显示，VidSketch在这些指标上均优于现有的基线方法，如IP-Adapter和ControlNet，证明了其在生成高质量视频动画方面的有效性。此外，我们还进行了用户研究，邀请参与者在多个维度上对生成的视频进行评价，结果表明，用户对VidSketch生成的动画在美学、时空一致性和细节丰富度等方面给予了更高的评分。通过消融实验，我们进一步验证了各个模块对最终生成质量的重要性，强调了层级草图控制策略和时空注意力机制的必要性。
在VidSketch的方法部分，我们可以把它理解为一个简单的过程，帮助用户将自己的手绘草图变成动画。首先，用户只需画出想要的草图，然后输入一些文字描述。系统会首先处理用户提供的第一幅草图，生成一个起始画面。接下来，系统会通过一种特殊的方法，确保第一帧的内容保持不变，同时逐步生成接下来的帧，形成连贯的动画。为了让不同绘画水平的用户都能获得满意的结果，VidSketch会根据草图的复杂程度自动调整生成动画的指导强度。最后，系统通过一个聪明的机制，确保每一帧之间的过渡自然流畅，提升整体动画的质量。这样，即使是初学者也能轻松制作出高质量的动画作品。
2025-02-01｜WyomingSeminary,SimonsInstitute,UCBerkeley,SYSU｜⭐️🟡
http ://arxiv.org/abs/2502.00500v2
本文提出了一种高效的视频建模方法，称为视频潜在流匹配（VideoLatentFlowMatching,VLFM）。与以往的研究不同，VLFM不再随机抽样潜在图像块进行视频生成，而是利用当前先进的预训练图像生成模型，基于文本引导的流动模型对潜在图像块进行建模，从而解码为时间相关的视频帧。我们首先假设视频的多帧图像在某个潜在空间中是可微分的，接着引入了HiPPO框架以近似最优的多项式投影来生成概率路径。该方法具有理论上界定的通用逼近误差和时间尺度鲁棒性。此外，VLFM还具备任意帧率下的视频插值和外推能力。通过在多个文本到视频数据集上的实验，展示了我们方法的有效性。
VLFM的核心方法包括以下几个步骤：
视频帧反演：利用强大的预训练视觉解码器将视频帧反演为潜在图像块，形成时间依赖的流动模型。
HiPPO框架：采用HiPPO框架来近似连续视频分布，利用高阶多项式投影实现对潜在图像块的最佳逼近。
条件视频潜在流：定义一个时间依赖的流动函数，通过输入潜在图像块和时间参数，生成与输入数据匹配的潜在流。
训练目标：通过最小化模型预测与真实流动的距离，优化模型参数，确保生成的视频潜在块与真实视频的相似度达到预设的精度要求。
在实验部分，我们首先介绍了实验设置，包括使用的视觉解码器和数据集。接着，展示了VLFM在不同帧率下进行文本到视频生成的能力，结果表明VLFM在生成质量和效率上表现优异。此外，我们还验证了VLFM在视频插值和外推方面的强大性能，实验结果显示该方法在处理不同帧率和时间尺度时能有效恢复视频内容。最后，通过消融实验，比较了使用流匹配算法与直接生成潜在图像块的效果，进一步证明了流匹配在提升生成质量方面的重要性。
在本研究中，我们提出了一种新的视频生成方法，叫做视频潜在流匹配（VLFM）。简单来说，VLFM通过一个强大的图像生成模型，将视频的每一帧转换成一种“潜在状态”，这些潜在状态就像是视频的“蓝图”。接着，我们使用一种叫做HiPPO的数学工具来确保这些蓝图能够准确地反映出视频的时间变化。这就像是把每一帧视频的图像拼接成一个流动的动画。我们还设计了一种训练方式，让模型在生成视频时尽量减少与真实视频之间的差距。通过大量的实验，我们发现VLFM不仅能够生成高质量的视频，还能在不同的播放速度下保持良好的效果。这使得我们的方法在视频生成领域具有很大的潜力。
2025-02-03｜HKU,BaiduInc｜⭐️🟡
http ://arxiv.org/abs/2502.01549v1https ://github.com/HKUDS/VideoRAG
本文介绍了VideoRAG，这是一个专为处理和理解极长视频而设计的检索增强生成框架。尽管检索增强生成（RAG）在文本内容的应用中取得了显著成功，但在多模态视频知识的整合上仍然相对较少。VideoRAG的核心创新在于其双通道架构，结合了基于图的文本知识基础和多模态上下文编码，旨在有效捕捉跨视频的语义关系，并高效保留视觉特征。通过构建跨越多个视频的知识图谱，VideoRAG能够处理无限长度的视频，保持语义依赖关系。实验结果显示，VideoRAG在Lecture、Documentary和Entertainment等多个类别的视频理解任务中，表现出显著的性能优势，为视频知识提取和教育内容分析等应用开辟了新的可能性。
VideoRAG的设计旨在解决极长视频处理中的几个关键挑战。其方法分为以下几个部分：
多模态视频知识索引：该模块将视频内容转化为结构化的文本和视觉表示。通过图基文本知识构建，保留了跨视频的语义关系，同时利用多模态上下文编码捕捉细粒度的跨模态交互。
知识基础的多模态检索：结合文本语义和视觉内容匹配，通过构建的知识图谱和嵌入，识别最相关的视频内容。该过程包括查询重构、实体匹配、片段选择和视频片段检索等步骤。
查询感知内容整合：在检索到的视频片段基础上，采用两阶段内容提取过程，结合LLM（大语言模型）提取关键词和视觉分析，生成全面的响应。
在实验部分，研究者对VideoRAG进行了全面的实证评估，使用了一个名为LongerVideos的基准数据集，该数据集包含超过160个视频，总时长超过134小时，涵盖了讲座、纪录片和娱乐等多个类别。实验主要围绕以下几个研究问题展开：VideoRAG在处理长视频内容时的有效性、与现有RAG方法和大视觉模型的比较、关键组件的有效性以及在实际应用场景中的表现。实验结果表明，VideoRAG在多个评估维度上均优于基线方法，尤其在信息的全面性和增强性方面表现突出，验证了其在多模态视频理解中的潜力。
VideoRAG的工作原理可以简单理解为一个智能助手，它能帮助我们从长视频中快速找到我们需要的信息。首先，它会把视频分成小段，像是把一本书的章节分开。然后，对于每一小段，它会用图形化的方式记录下重要的内容，比如说视频里的对话、场景和动作。接着，当我们提出一个问题时，VideoRAG会通过这些记录来找到最相关的视频片段，并结合这些片段的信息来生成一个详细的回答。就像在图书馆里找书一样，VideoRAG可以快速找到我们想要的“书”，并告诉我们里面的内容，这使得理解长视频变得简单而高效。
2025-02-03｜FDU｜⭐️🟡
http ://arxiv.org/abs/2502.01046v1https ://demoface-ai.github.io/
在当今数字化时代，情感化语音合成技术正在迅速发展，尤其是在面向人机交互的应用中。本文探讨了一种新任务——情感面部语音合成（eF2S），旨在通过面部表情直接生成情感语音。传统的面部语音合成（F2S）方法虽然在捕捉身份特征上表现出色，但在情感表达和多样性方面却面临挑战。为了解决这些问题，研究者们提出了一种新的生成框架DEmoFace，该框架利用离散扩散变换器（DiT）和课程学习，结合多层神经音频编解码器，旨在根据面部情感和身份特征生成高质量的语音。通过这种方法，用户不仅能够获得与发言者身份一致的语音，还能感受到丰富的情感表达，这为虚拟角色配音和辅助有表达语言障碍的个体提供了新的可能性。
DEmoFace的核心在于其创新的生成框架，具体方法包括以下几个关键步骤：
任务公式化：将eF2S任务定义为从多模态条件（身份、情感和文本）生成语音。模型需要在生成的语音中同时反映出身份特征和情感状态。
离散扩散变换器（DiT）：采用多模态DiT来动态对齐文本和语音，同时根据面部情感和身份调整语音风格。该模型通过逐步细化的课程学习算法来提高训练效率和生成质量。
增强的预测无引导：为了处理多样的条件场景，DEmoFace引入了一种增强的无预测引导方法，促进了多条件生成和复杂属性的有效解耦。
音频编解码：使用残差向量量化（RVQ）技术对语音进行编码和解码，以实现高质量的音频生成，并优化生成过程中的效率和多样性。
实验部分展示了DEmoFace在自然性和一致性方面的优越性能。通过与传统F2S方法和其他表达性文本到语音（TTS）方法的对比，DEmoFace在生成的语音中表现出更自然的情感表达和一致性。实验结果表明，DEmoFace能够在不依赖于任何语音提示的情况下，仅通过面部表情生成情感丰富的语音。此外，模型的多模态条件生成能力使其在复杂场景下表现出色，能够有效处理身份和情感的解耦。
在我们的研究中，我们提出了一种新的技术，叫做DEmoFace，它可以通过人的面部表情来生成语音。这种方法的工作方式很简单：首先，我们会分析面部表情，了解这个人当前的情感状态，比如他们是高兴、伤心还是惊讶。接着，我们会把这些信息与文本内容结合起来，让计算机知道该说什么。然后，DEmoFace会使用一种特殊的算法，逐步生成与该面部表情和文本内容相匹配的语音。这个过程分为几个阶段，先从简单的开始，逐渐增加复杂性，确保生成的语音听起来既自然又富有情感。最终，用户可以听到与他们所看到的面孔相符的声音，这在虚拟角色配音和帮助有语言障碍的人士方面都有广泛的应用前景。

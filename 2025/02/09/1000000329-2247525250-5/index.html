292-A3｜物理引导文生组合性3D场景，从多视角语义信息生成3D形状；动态3D人类重建，情感驱动说话头合成；3D场景理解与分割

2025-02-02｜SHU｜⭐️🟡
http ://arxiv.org/abs/2502.00708v1
PhiP-G（Physics-GuidedText-to-3DCompositionalSceneGeneration）是一种创新的文本到3D场景生成框架，旨在通过自然语言输入生成高质量的3D场景。该方法结合了多模态大语言模型（LLM）和3D高斯生成技术，克服了现有方法在生成复杂场景时所面临的多个挑战，包括物理一致性、资产生成的自动化以及对复杂场景描述的理解。PhiP-G通过分析复杂的场景描述，生成场景图，并利用物理池和视觉监督代理来指导场景布局，从而实现物理合理且符合文本描述的3D场景生成。通过一系列实验，PhiP-G在生成质量和效率方面均表现出色，达到了当前技术的最先进水平。
PhiP-G的框架主要分为两个核心组件：3D资产生成和物理3D场景布局。
3D资产生成：
使用关键词提取代理（AG-extractor）对复杂场景描述进行解析，生成场景图。
通过2D生成代理（如DALL·E3）将解析后的资产描述转换为2D图像，并使用3D高斯生成模型将这些图像快速转换为3D资产。
生成过程中，采用循环生成和图像检索库以确保生成的2D图像与资产描述之间的语义一致性。
物理3D场景布局：
利用物理池和关系分类代理构建初步布局，确保生成的场景遵循物理法则。
通过视觉监督代理进行迭代优化，评估布局的合理性并提供调整建议。
该过程结合了反向推理，利用生成的场景图和描述进行布局的动态调整。
在实验部分，研究者通过定量和定性分析验证了PhiP-G的有效性。定量分析中，使用CLIP和T3Bench等评价指标对比了PhiP-G与当前最先进的方法，结果显示PhiP-G在生成质量和语义一致性方面均优于其他方法。此外，用户研究表明，参与者对PhiP-G生成的场景在文本忠实度、场景质量、美学和物理合理性等方面给予了更高的评分。通过消融实验，研究者进一步验证了各个模块对整体性能的贡献，结果表明AG-supervisor和物理池在优化场景布局方面至关重要。
在PhiP-G的工作中，首先我们需要理解场景描述，比如“一个鸟站在椅子上”。然后，系统会使用一个智能代理来提取这个描述中的关键词，帮助生成一个场景图。接下来，系统会利用2D图像生成工具把这些关键词转换成图像，比如生成一张包含椅子和鸟的图片。之后，这些2D图像会通过一个快速的3D生成模型转化为3D对象，形成一个可以在虚拟空间中使用的3D模型。最后，系统会通过物理规则来安排这些3D对象的位置，确保它们看起来合理，比如鸟确实是在椅子上，而不是漂浮在空中。整个过程非常智能化，用户只需提供简单的文字描述，系统就能自动生成符合要求的3D场景。
2025-02-01｜USTC,CardiffU｜⭐️⭐️
http ://arxiv.org/abs/2502.00360v1https ://shapefromsemantics.github.io
本研究提出了一种名为“从语义生成形状”的新型生成问题，旨在通过多视角的语义信息生成高质量的3D模型。与传统的“从X生成形状”方法不同，该方法不依赖于特定的视觉输入（如RGB图像或深度图），而是利用语义描述作为输入，极大地扩展了设计空间，使用户能够创造出更具想象力和吸引力的3D对象。该框架的关键在于通过语义输入生成的3D模型在不同视角下能够保持几何形状和外观的一致性，确保生成的形状不仅在视觉上吸引人，而且在实际制造中可行。研究表明，利用多重语义信息能够有效地提升模型的创造性和实用性，这为增强现实（AR）、虚拟现实（VR）及艺术创作等领域提供了新的可能性。
本研究的方法分为三个主要阶段，以实现从多视角语义生成3D模型的目标：
多语义SDS基础形状设计：首先，利用语义导向的扩散模型生成粗略的几何形状和纹理。通过对不同语义标签进行采样，生成多个视角的图像，并使用ScoreDistillationSampling（SDS）损失指导生成过程，确保初始形状与语义输入一致。
细节生成与多视角监督：在第二阶段，基于生成的高质量图像，使用图像恢复模型和视频生成模型进行细节增强。这一过程确保了生成的3D模型在视觉上更加丰富，同时保持与输入语义的一致性。
精细形状重建：最后，采用神经隐式距离场（SDF）表示法对细化的形状进行重建，提取出具有复杂细节的网格。此阶段的目标是生成可制造的高质量网格，确保几何形状、纹理和过渡的平滑性。
在实验部分，研究者们采用多个案例来验证所提方法的有效性。通过与现有技术的对比，展示了生成模型在语义一致性和视觉吸引力方面的优势。具体实验设计包括500次迭代的训练，使用3D高斯样本进行形状优化。实验结果表明，生成的3D模型在不同视角下能够有效传达输入的语义信息，且在制造上具有可行性。此外，研究团队还进行了用户研究，收集参与者对生成形状的语义准确性和整体偏好的评分，结果显示本方法在用户偏好上显著优于传统方法。最终，通过3D打印验证了生成形状的实际制造能力，结果与预期设计高度一致。
在这个研究中，科学家们提出了一种新方法，利用人类可以理解的描述（比如“绿色的杯子”或“飞翔的鸟”）来创建3D模型。这个过程可以分为三个简单的步骤。首先，他们会根据这些描述生成一个粗略的形状和颜色，确保这个形状看起来像描述的对象。接着，他们会对这个形状进行细节的增强，比如让颜色更加鲜艳，形状更加生动。最后，他们使用一种特殊的技术来确保生成的3D模型不仅美观，而且可以通过3D打印等方式实际制造出来。通过这种方法，用户可以轻松创造出复杂且富有创意的3D设计，甚至不需要专业的绘图技能。
2025-02-03｜UTDallas,HKU,HKUST,BNU-HKBUUIC,TAMU｜⭐️
http ://arxiv.org/abs/2502.01045v1https ://wyiguanw.github.io/WonderHuman/
在这篇论文中，研究者们提出了一种新的动态3D人类重建方法，称为WonderHuman。该方法能够从单目视频中重建出高保真度的人类，尤其是在视频中未被完全观察到的人体部分。以往的动态人类重建技术通常需要全面覆盖的多视角输入，但在实际应用中，常常只能获取到有限的视角，如单一的前视视频，这使得重建隐藏部分的任务变得复杂。WonderHuman通过利用2D生成扩散模型的先验知识，结合双空间优化技术，能够有效地生成高质量、逼真的动态人类模型，尤其是在处理那些难以观察的身体部位时表现出色。
WonderHuman的核心方法分为两个主要阶段。首先，在第一阶段，系统从部分可见的单目视频中重建人类的可见外观。这一过程包括以下几个步骤：
高斯参数预测：利用高斯解码器预测可见部分的人体几何形状和外观。
动态人类渲染：通过结合线性混合蒙皮函数与高斯点渲染，生成动态3D人类模型。
法线图渲染：生成法线图以捕捉详细的表面几何信息，从而提高渲染质量。
在第二阶段，系统则专注于重建不可见部分的外观。主要步骤包括：
双空间优化：在标准空间和观察空间中应用得分蒸馏采样（SDS），确保生成的内容在视觉上的一致性。
视图选择：分析哪些区域的可见性较差，以便进行优化。
姿势特征注入：将姿势相关特征注入解码器，以增强生成模型的动态表现。
在实验部分，研究者们在多个基准数据集上评估了WonderHuman的效果，包括ZJU-Mocap、Monocap和MVHumanNet数据集。结果显示，该方法在生成可见和不可见部分的人体外观时，均优于现有的最先进技术。具体的性能指标如峰值信噪比（PSNR）、结构相似性指数（SSIM）和感知相似性（LPIPS）均显示出显著的提升。此外，WonderHuman在处理复杂动态场景时，能够有效减少常见的伪影和不一致性，从而提高重建的真实感和细节表现。
在WonderHuman的工作中，研究者们想要从普通的视频中创造出3D人类模型，这个过程可以分为两个主要步骤。首先，他们从视频中提取出可以看到的人体部分，利用一种叫做高斯解码器的工具来预测这些部分的形状和颜色。接着，他们通过一些数学方法将这些信息转化为3D模型，这样我们就可以从不同的角度看到这个人。
在第二步中，WonderHuman会处理那些视频中看不到的身体部分。他们使用一种叫做双空间优化的技术，确保这些隐藏的部分在视觉上与我们能看到的部分一致。此外，他们还会选择哪些部分需要特别关注，并注入一些与姿势相关的信息，以便让模型更加真实。通过这样的方式，WonderHuman能够生成出更完整、更真实的人类模型，尤其是在视频中那些看不到的部分。
2025-02-02｜UNIST,Inriaet.al.｜⭐️
http ://arxiv.org/abs/2502.00654v1
本研究提出了一种新颖的3D情感驱动的说话人合成框架，称为EmoTalkingGaussian。该框架旨在通过利用音频输入及情感的连续变化（即愉悦度和唤醒度）来生成高质量的说话人图像。与现有的说话人合成方法相比，EmoTalkingGaussian能够更好地表达多样化的情感，并在合成过程中保持嘴唇与音频同步。研究中采用了一种自监督学习的方法，以提高嘴唇同步的准确性，并通过引入情感面孔生成器来增强合成的情感表现力。该方法的优势在于能够在不需要额外数据捕捉的情况下，利用已有的视频数据生成富有表现力的说话人图像。
EmoTalkingGaussian的核心方法包括以下几个步骤：
基础架构：采用TalkingGaussian作为基础，通过音频特征和动作单元来合成3D高斯分布的说话人图像。
情感调节：引入情感分支，以愉悦度和唤醒度为输入，调整面部表情，允许动态变化的情感表达。
嘴唇同步：利用自监督学习方法，通过文本到语音的网络生成多样化的音频数据，以确保嘴唇动作与音频内容的高度一致。
情感面孔生成器：通过引入一个唇部对齐的情感面孔生成器，确保合成图像中的嘴唇形状与输入图像相匹配，从而提升合成图像的自然度和情感表达的准确性。
损失函数设计：设计多种损失函数，包括正常图损失和同步损失，以优化合成图像的质量和嘴唇的同步性。
在实验部分，研究团队在多个公开数据集上对EmoTalkingGaussian进行了评估。实验包括自重建、跨域音频和情感驱动的场景。通过与现有的基线方法进行比较，结果显示，EmoTalkingGaussian在图像质量、嘴唇同步和情感表达的准确性上均表现优越。具体而言，研究使用了PSNR、SSIM、LPIPS等指标来评估图像质量，并通过嘴唇标记距离（LMD）和同步误差（Sync-E）来评估嘴唇的同步性。此外，情感分类准确性（E-Acc）也被用来验证模型在情感表达方面的能力。最终，实验结果表明，该方法在各种场景下均能有效表达目标情感，且在图像质量上与其他方法相当。
EmoTalkingGaussian的工作原理可以简单理解为一个智能的动画制作工具。它通过分析输入的音频，比如一个人说话的声音，来生成一个说话的人物图像。这个工具不仅能让这个人物的嘴巴跟着说话的声音动，还能让他/她的表情根据你想要表达的情感（例如快乐、惊讶或悲伤）而变化。为了让嘴巴的动作和声音更协调，EmoTalkingGaussian使用了一种聪明的学习方法，能够从已有的声音和图像中“学习”如何更好地同步。最后，通过一个特殊的面孔生成器，确保生成的人物图像看起来更真实，嘴巴的形状也与输入的图像相匹配。这样，无论是制作动画视频还是进行虚拟对话，这个工具都能提供更加生动和自然的表现。
2025-01-31｜MagicLeapInc.｜⭐️
http ://arxiv.org/abs/2502.00173v1
在当前的计算机视觉领域，3D场景理解和实例分割是重要的研究方向。本文提出了一种名为“LiftingbyGaussians”（LBG）的方法，旨在实现高效的3D实例分割。LBG利用现有的2D基础模型（如SAM、CLIP和DINO）来直接融合2D分割掩码和稠密特征至3D高斯场（3DGS）中，避免了传统基于学习的方法所需的复杂训练过程。与现有的学习方法相比，LBG的处理速度快一个数量级，能够在增强现实和虚拟现实等新应用中实现物体操作等功能。该方法不仅简化了3D场景的分割过程，还提高了分割的语义质量，展示了其在现有3DGS重建上的广泛适用性。
LBG方法的核心在于其高效的2D到3D提升策略，主要可以分为以下几个步骤：
输入准备：LBG接收两个输入：一组带姿态的2DRGB图像和一个预训练的3DGS场。
2D掩码提取：使用2D基础模型（如SAM）从每帧图像中提取2D分割掩码，并为每个掩码提取语义特征。
像素到高斯的映射：通过将每个像素与其对应的最大贡献高斯进行关联，生成每帧的3D物体碎片。每个像素都被分配一个唯一的对象ID。
增量合并：对每一帧的物体碎片进行增量合并，基于几何和语义相似性将新检测到的物体碎片与现有对象进行匹配。
分层细化：在合并的基础上，进一步将对象分解为部件和子部件，形成层级结构的场景图。
实验部分对LBG方法的有效性进行了全面评估。研究者使用了两个数据集：LERF和3D-OVS，分别用于评估2D掩码和3D实例分割的性能。实验中，LBG的分割结果在多个标准指标上均优于其他现有方法，显示出更高的视觉质量和准确性。具体而言，通过对比PSNR、SSIM和LPIPS等度量，LBG在提取3D资产时表现出更少的噪声和更高的清晰度。此外，LBG在处理时间上也显著缩短，展示了其在实际应用中的潜力。
LBG方法的工作原理可以简单理解为一个高效的“翻译器”，它将2D图像中的物体信息快速转化为3D模型。首先，LBG从每张2D图片中提取出物体的轮廓和特征，就像我们在照片上用笔圈出物体一样。接着，它将这些信息与3D高斯模型结合，确保每个物体在三维空间中都能被准确标识。然后，LBG会逐帧检查这些物体，并将相似的物体合并在一起，就像拼图一样，把相邻的碎片组合成完整的图像。最后，它还会对物体进行细分，确保每个物体的各个部分都能被清楚地识别和分开。这种方法不仅快，而且能处理各种复杂的场景，使得3D建模变得更加简单和高效。

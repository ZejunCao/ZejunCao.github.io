292-B｜人类身体图像恢复，跨主体fMRI视觉解码；NeRF傅里叶频率控制，探索3D场景假设推理，3D场景理解与问答综述

主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）发布日期｜2025年2月4日｜周二
2025-02-03｜SJTU,VIVO
http ://arxiv.org/abs/2502.01411v1https ://github.com/gobunu/OSDHuman
人类身体恢复（HBR）是一种图像恢复的特定应用，广泛应用于多个领域。然而，由于缺乏标准化的基准数据集，HBR的研究进展面临挑战。本研究提出了一种高质量数据集的自动裁剪和过滤（HQ-ACF）管道，利用现有的对象检测数据集和其他未标记图像自动裁剪和筛选高质量的COCO人类图像。基于这一管道，我们构建了一个以人物为基础的恢复数据集，名为PERSONA，涵盖了109,052张高质量的512×512人类图像。该数据集在内容丰富性和质量上显著优于其他人类相关数据集。此外，我们提出了一种新颖的一步扩散模型OSDHuman，专门用于人类身体恢复，能够在视觉质量和定量指标上超越现有方法。
本研究的方法分为以下几个关键部分：
高质量人类数据集管道（HQ-ACF）：首先，从多个常用的大型对象检测数据集中收集图像，使用YOLOv5进行人类检测。接着，通过拉普拉斯算子计算图像的方差，筛选出高质量图像。最终，裁剪并调整人类边界框，生成约440,000张高质量图像。
人物基础恢复数据集（PERSONA）：利用HQ-ACF管道，我们创建了一个包含复杂对象和自然活动的人物基础恢复数据集。该数据集不仅包含单人图像，还涵盖了多人交互场景，确保数据的多样性和复杂性。
一步扩散模型（OSDHuman）：该模型采用变分自编码器（VAE）架构，通过高保真图像嵌入器（HFIE）提取适合HBR的提示，从而提高恢复质量。训练过程中结合了多种损失函数，以确保生成图像与自然图像的分布对齐。
在实验部分，我们对PERSONA数据集进行了定量分析，评估了模型的性能。通过多种无参考图像质量评估（IQA）指标，我们的PERSONA数据集在各项指标上都优于其他人类相关数据集。此外，我们将OSDHuman模型与多种现有的恢复方法进行了比较，结果显示，OSDHuman在视觉质量和定量指标上均表现出色。实验还包括对不同训练数据集的比较，结果表明，使用PERSONA数据集训练的模型在恢复性能上显著优于使用其他数据集训练的模型。
在我们的研究中，我们采用了一种新颖的方法来恢复人类图像。首先，我们建立了一种自动化的系统，能够从大量图像中识别并裁剪出包含人类的高质量图像。这就像一个智能的筛子，能够把好的图像挑出来。接着，我们把这些图像整理成一个数据集，确保它们不仅清晰而且内容丰富。然后，我们设计了一种新的计算机模型，叫做OSDHuman，它能够从这些高质量图像中学习，帮助它更好地恢复低质量的图像。这个模型就像一个高级的图像修复师，能够精准地还原出人类的细节和表情。通过这些步骤，我们的研究不仅提高了图像恢复的质量，还为未来的相关研究提供了更好的数据支持。
2025-02-01｜THU,BIT,BUAA｜ICASSP2025｜🟡
http ://arxiv.org/abs/2502.00412v1
本研究提出了一种名为TROI（TrainableRegionofInterest）的新型两阶段数据驱动方法，旨在提升跨主体的fMRI视觉解码性能。传统的fMRI解码方法依赖于手动标注的感兴趣区域（ROIs），这往往会引入冗余信息和噪声，影响解码效果。此外，缺乏自动化的ROI标注方法也限制了fMRI视觉解码技术在新受试者中的应用。TROI通过利用跨主体数据集中的标注ROI进行预训练，从而高效优化新受试者的输入层，避免了从头开始重新训练整个模型。该方法在小样本数据集上进行评估，结果显示其在视觉检索和重建任务中均优于现有的最先进方法MindEye2。
TROI方法分为两个阶段：第一阶段为稀疏掩模训练，第二阶段为学习率回滚。具体步骤如下：
稀疏掩模训练：
使用LASSO正则化来进行稀疏掩模训练，以减少输入数据中的噪声。
结合低通滤波器快速生成体素掩模并确定输入层维度。
通过引入一个0-1ROI掩模来选择性提取来自粗略分割的脑区的体素，主要集中在枕叶。
学习率回滚：
在第二阶段，利用第一阶段训练得到的稀疏掩模，从头开始重新训练TROI模块，这种策略被称为学习率回滚，旨在提高模型的解码性能。
通过联合优化多种任务的损失函数，确保模型在有限样本情况下的有效性。
本研究在一个配备L40GPU的单机上进行实验，利用NaturalScenesDataset（NSD）进行模型训练。以Subject1作为目标受试者，模型首先在其他受试者的数据上进行预训练，然后使用Subject1的小样本数据进行微调。在第一阶段中，经过15个训练周期，将掩模中的体素数量减少到约3000个；在第二阶段中，继续训练150个周期。实验结果表明，TROI方法在视觉检索和重建任务中都优于MindEye2，尤其在小样本场景下表现突出。
在这项研究中，研究人员提出了一种新方法来帮助我们从大脑扫描中解码视觉信息。这个方法叫做TROI，它分为两个主要步骤。首先，在第一步中，研究人员使用一种叫做稀疏掩模的技术，来选择性地提取大脑中特定区域的信号。这就像在一堆信息中挑选出最重要的部分，以减少噪声和冗余。为了做到这一点，他们还使用了低通滤波器来过滤掉不必要的信号。第二步则是对模型进行进一步的训练，以确保它能够更好地处理新受试者的数据，这个过程叫做学习率回滚。通过这两个步骤，TROI能够在小样本情况下更准确地解码大脑的视觉活动，从而帮助我们更好地理解大脑如何处理视觉信息。
2025-02-03｜ÉcolePolytechnique｜3DV2025｜🟡
http ://arxiv.org/abs/2502.01405v1
在本研究中，作者提出了一种名为FourieRF的新方法，旨在解决少量输入数据下的神经辐射场（NeRF）重建问题。该方法通过逐步控制傅里叶频率，显著减少了高频伪影的出现，确保了训练过程的稳定性和最终渲染质量。与以往依赖于预训练模块或特定场景数据驱动先验的方法不同，FourieRF采用了一种通用的方法，能够有效重建现实场景，且成本远低于竞争对手的技术。实验结果表明，该方法在少量视角下的场景重建中表现出色，具有广泛的应用潜力。
FourieRF的方法主要包括以下几个关键步骤：
傅里叶频率控制：通过在傅里叶域中对信号频率进行显式控制，逐步引入高频信息，以此来平衡低频和高频的特征学习，避免早期优化过程中的高频伪影。
课程训练策略：采用课程训练策略，初始阶段仅使用低频信息进行训练，随着训练的进行，逐渐增加可用的高频信息。这种方法有助于网络在学习初期建立稳健的基础，从而在后续阶段能够更好地捕捉细节。
网格基础表示：FourieRF基于网格的特征表示，利用向量-矩阵分解和CP分解来减少计算复杂性，同时保持高效的重建能力。这种方法使得在少量视角下的重建变得更加高效。
为了验证FourieRF的有效性，作者在合成和真实场景数据集上进行了广泛的实验。实验设置包括使用不同数量的训练视角（如3、4、6视角），并与现有的最先进方法（如ZeroRF和FreeNeRF）进行了比较。结果显示，FourieRF在重建质量和训练速度上均优于这些方法，训练时间仅需约10分钟，且在不同场景中的表现一致良好。定量评估结果表明，FourieRF在PSNR、SSIM等指标上均达到了新的最佳水平，展示了其在少量视角下高效重建的能力。
FourieRF的核心方法可以简单理解为“逐步学习”。想象一下你在学画画，开始时你只用铅笔勾勒出大致的轮廓（低频信息），随着你对画面的理解加深，逐渐加入颜色和细节（高频信息）。FourieRF就是通过类似的方式来训练计算机，让它先专注于理解物体的基本形状，再慢慢添加细节，这样可以避免在开始时就产生错误的细节（伪影）。此外，FourieRF还利用了一种数学工具——傅里叶变换，来帮助它更好地控制学习的过程，确保最终得到的图像既清晰又真实。
2025-02-02｜Imperial(ICL)
http ://arxiv.org/abs/2502.00954v2https ://matchlab-imperial.github.io/Hypo3D
本文研究了在没有实时场景数据的情况下，模型在三维场景中的假设推理能力。随着视觉-语言基础模型的兴起，研究者们希望缩小人类与机器在三维场景理解方面的差距。现有的三维推理基准通常假设实时场景数据的可用性，但在现实世界中，动态场景的更新成本高昂。为此，研究者们提出了Hypo3D基准，旨在评估模型在假设场景变化后的推理能力。Hypo3D包括7,727个上下文变化和14,885个问答对，覆盖700个室内场景，揭示了当前模型在处理运动变化和方向推理时的显著性能差距，尤其是在与人类的比较中。
Hypo3D的任务定义为3D视觉问答（VQA），模型需在没有实时场景数据的情况下，通过想象变化后的场景来回答问题。具体方法包括：
场景表示：模型首先接收一个3D场景的表示，包含点云、RGB-D视频和顶视图地图等多种形式。
世界框架定义：为确保一致性，所有场景都被赋予一个基于锚定物体的世界框架，以标准化方向术语。
上下文变化描述：研究设定了五种类型的上下文变化，包括移动、移除、属性变化、添加和替换，模型需根据这些变化想象新的场景状态。
问答生成：在上下文变化的基础上，生成与变化相关的问题，确保问题的独特性和可回答性。
数据集构建：通过人工标注和大型语言模型（LLMs）的结合，构建了包含多样性和高质量的上下文变化与问答对的数据集。
在实验部分，研究者们对多个基础模型进行了评估，包括大型语言模型（LLMs）、二维视觉语言模型（2DVLMs）和三维视觉语言模型（3DVLMs）。采用精确匹配（EM）和部分匹配（PM）两种指标来评估模型性能。实验结果显示，尽管模型在某些任务上表现良好，但整体上仍显著低于人类的表现，尤其是在运动变化和方向推理的问题上。研究发现，现有模型在推理假设变化的场景时，往往会出现幻觉现象，导致对无关上下文变化的错误调整。这一发现强调了当前模型在假设推理能力方面的不足，并为未来改进提供了方向。
在Hypo3D的研究中，科学家们想要了解计算机如何在没有实时场景图像的情况下进行推理。为了做到这一点，他们设计了一种方法，首先给计算机一个三维场景的图像，然后告诉它场景发生了什么变化，比如“桌子被移动了”。接着，计算机需要想象这个变化后的场景，并根据这个新场景回答问题。整个过程就像在玩一个需要想象力的游戏，计算机必须“看到”变化，而不是依赖于实际的图像。为了确保计算机能够理解这些变化，研究者们还设定了不同类型的变化，并根据这些变化生成问题。通过这种方式，研究者们希望能够评估和提升计算机在复杂场景中的推理能力。
2025-02-01｜HunanU,UMelbourne,USYD,AnhuiU
http ://arxiv.org/abs/2502.00342v1
3D场景问答（3DSQA）是一个跨学科的任务，结合了3D视觉感知和自然语言处理，旨在使智能代理能够理解和与复杂的3D环境进行交互。近年来，随着大规模多模态建模的进展，出现了多样化的数据集，并推动了3DSQA中指令调优和零-shot方法的发展。然而，这一快速进展也带来了挑战，尤其是在实现不同数据集和基准之间的统一分析和比较方面。本文首次全面回顾了3DSQA，系统评估了数据集、方法论和评估指标，同时强调了数据集标准化、多模态融合和任务设计方面的关键挑战和未来机遇。
在3DSQA中，模型的任务是理解一个3D场景（S）和一个查询（Q），以生成文本答案（T）和可选的空间信息（B）。该任务的形式化定义为F:(S,Q)→(T,B)，涉及多模态推理和空间理解。方法主要分为以下几个方面：
数据集结构：数据集在3DSQA中至关重要，影响任务的范围和复杂性。现有数据集在场景表示、规模和查询复杂性上存在显著差异。
查询生成：早期数据集依赖于人工注释，近年来引入了模板生成和大语言模型（LLM）辅助生成的方法，以提高QA对的多样性和规模。
模型架构：从早期的自定义架构（如PointNet++和BERT）发展到基于LLM的模型，采用指令调优和零-shot学习技术，减少对特定任务注释的依赖。
本文总结了不同数据集和方法在3DSQA任务中的应用与表现，特别关注各类模型在特定任务上的效果。通过对比不同方法的性能，研究了基于任务的、预训练的和零-shot学习的方法。这些实验不仅帮助识别了当前方法的优缺点，还为未来的研究方向提供了见解，尤其是在动态和开放世界场景中的应用潜力。
在3D场景问答中，模型的工作是理解一个3D环境和一个问题，然后给出答案。这个过程可以分为几个步骤。首先，模型需要从数据集中获取3D场景的信息，这些场景可以用点云或多视图图像表示。接下来，模型会接收一个问题，这个问题可以是简单的文本描述，也可以是包含空间信息的复杂查询。然后，模型会结合这些信息，使用特定的算法来处理和分析数据，最终生成一个文本答案。为了提高模型的表现，研究人员还会使用大语言模型（LLM）来帮助生成问题和答案，这样模型在面对新的问题时也能更好地适应。这种方法的目标是让计算机能够像人一样理解和回答有关3D环境的问题，从而在机器人、虚拟现实等领域发挥作用。

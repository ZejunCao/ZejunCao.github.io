2025-02-03｜NUSShowLab｜⭐️⭐️
http ://arxiv.org/abs/2502.01572v2https ://github.com/showlab/MakeAnything
本研究提出了一种新颖的框架，名为MakeAnything，它利用扩散变换器（DiffusionTransformers）来生成多领域的程序性序列。这一工具能够根据文本描述或条件图像，生成如绘画、手工艺和烹饪等活动的逐步程序性教程。当前的技术面临三个主要挑战：缺乏多任务程序性数据集、保持步骤间的逻辑连续性和视觉一致性，以及在多个领域间的泛化能力。为了解决这些问题，研究团队构建了一个覆盖21个任务、超过24,000个程序序列的多领域数据集，并通过引入对称低秩适应（LoRA）和ReCraft模型，增强了生成的一致性和逻辑性。实验结果表明，MakeAnything在程序生成任务上超过了现有方法，设立了新的性能基准。
方法部分主要包括以下几个关键组成部分：
扩散变换器（DiT）的预备知识：DiT模型通过逐步去噪的方式生成图像，处理图像和文本条件的token，以实现高质量的序列生成。
整体架构：MakeAnything的训练分为两个阶段，首先使用对称LoRA方法在MakeAnything数据集上进行训练，生成创意教程。然后，将LoRA与Flux基础模型合并，形成ReCraft模型的基础。
对称低秩适应（LoRA）：通过共享知识和任务特定矩阵的结合，LoRA有效提高了多任务性能，避免了单一数据集训练导致的过拟合。
ReCraft模型：该模型允许用户上传图像，生成与参考图像高度一致的步骤序列，提升了生成过程的可控性和解释性。
MakeAnything数据集：该数据集包含21个任务，经过专业团队收集和注释，确保了数据的多样性和高质量。
在实验部分，研究团队实施了多项实验以验证MakeAnything的有效性。首先，设置了对比基线，包括ProcessPainter和Flux等现有方法。实验结果显示，MakeAnything在生成的程序序列的逻辑性、一致性和实用性等方面优于这些基线。具体的评估指标包括文本与图像的对齐度、生成序列的连贯性和用户的可用性反馈。此外，用户研究表明，参与者在对比不同生成方法时，普遍认为MakeAnything生成的序列更具实用性和逻辑性。通过综合评估，MakeAnything在21个任务上均表现出色，确立了新的研究基准。
在MakeAnything的工作中，我们使用了一种叫做扩散变换器的技术，这种技术可以帮助计算机理解和生成图像和文本。首先，我们创建了一个包含很多不同类型教程的数据集，比如绘画和手工艺。接着，我们设计了一种新的训练方法，叫做对称低秩适应，这样可以让计算机更好地学习如何生成这些教程，而不会因为数据量少而出错。我们还开发了一个叫做ReCraft的模型，用户只需上传一张图片，计算机就能生成与这张图片相关的制作步骤。通过各种实验，我们发现MakeAnything生成的教程比其他方法更清晰、更有用，能够帮助人们更好地学习如何完成各种活动。
2025-02-03｜NUSShowLab｜⭐️⭐️
http ://arxiv.org/abs/2502.01105v1https ://github.com/showlab/LayerTracer
LayerTracer是一种基于扩散变换器的框架，旨在解决生成认知对齐的分层SVG（可缩放矢量图形）的挑战。现有的方法往往产生过于简化的单层输出或因优化而导致形状冗余，无法满足专业设计的需求。LayerTracer通过学习设计师的分层SVG创作过程，利用一个新颖的序列设计操作数据集，提供了一种更符合设计师认知的生成方式。该方法分为两个阶段：首先，基于文本条件的扩散变换器生成模拟人类设计工作流程的多阶段栅格化构建蓝图；其次，通过路径去重的分层矢量化生成干净、可编辑的SVG。这种方法不仅提高了生成质量，还增强了可编辑性，确保输出符合专业标准。
LayerTracer的核心方法包括以下几个关键模块：
认知对齐的SVG合成：通过学习设计师的构建逻辑（如元素排序、图层分组和空间推理），确保生成的SVG符合专业设计标准。
分层模型训练：利用LoRA（低秩适应）进行模型微调，以生成分层像素图像，适应文本描述的需求。
图像到层的模型：将参考图像编码为条件令牌，通过对图像的上下文进行学习，推导出图层构建的顺序。
层级矢量化：将生成的像素图像转化为高质量的矢量图形，采用邻接层之间的差异分析进行优化，确保保留图形的结构完整性。
这些模块的结合使得LayerTracer能够有效地生成既符合逻辑又具有可编辑性的分层SVG，克服了传统方法的局限性。
在实验部分，LayerTracer通过与多种基线方法（如SVGDreamer、Vecfusion和DiffSketcher）进行比较，展示了其在文本到SVG生成和层级矢量化任务中的优越性能。实验采用了包含20,000个分层SVG的自建数据集，并进行了一系列定量和定性评估。结果表明，LayerTracer在生成质量、路径数量和运行时间等指标上均优于其他方法。同时，用户研究显示，参与者更倾向于选择LayerTracer生成的结果，认为其更符合设计要求和逻辑层次。
LayerTracer的工作原理可以简单理解为一个智能设计助手。首先，它学习专业设计师如何一步一步地创建图形，比如先画背景再画前景。接着，它利用这些知识来根据文字描述或图片生成分层的SVG图形。这个过程分为几个步骤：首先，LayerTracer会根据输入的描述生成一个粗略的图形草图；然后，它会将这些草图细化为多个图层，使每一层都可以单独编辑。通过这样的方式，设计师可以轻松调整每个图层的内容，而不会影响到其他部分。总的来说，LayerTracer就像一个能够理解设计逻辑的智能工具，帮助设计师更高效地创建和修改图形。
2025-02-03｜NEU,AdobeResearch｜⭐️⭐️
http ://arxiv.org/abs/2502.01639v1https ://sliderspace.baulab.info
本文介绍了SliderSpace，一个用于自动分解扩散模型视觉能力的框架。扩散模型在生成多样化图像方面表现出色，但用户对其内部结构的理解仍然有限。SliderSpace旨在解决这一问题，通过从单一文本提示中发现可控且人类可理解的多个方向，帮助用户更好地探索模型的创造潜力。与现有方法不同，SliderSpace不需要用户事先定义编辑方向，而是通过无监督学习自动识别出多个可解释的方向。这些方向作为低秩适配器进行训练，使得用户能够组合和控制不同的视觉属性，从而实现更丰富的创作可能性。通过在多个先进的扩散模型上进行实验，SliderSpace展示了其在概念分解、艺术风格探索和多样性增强等应用中的有效性。
SliderSpace的核心是一个无监督的视觉发现过程，主要分为以下几个步骤：
问题表述：定义目标是从给定的文本提示中识别出独立的控制方向，这些方向可以捕捉模型生成的图像的主要变异模式。
分布采样：从扩散模型中生成多样化的图像样本，并提取特征。这一过程通过对随机种子进行变换来实现，确保样本的多样性。
语义分解：将生成的样本映射到语义嵌入空间（如CLIP），并计算出主成分，代表语义空间中的正交方向。
Slider训练：为每个主成分训练相应的低秩适配器，使其在语义空间中与特定方向对齐。训练目标是确保每个适配器在生成图像时能够引入一致的语义变化。
这些步骤共同构成了SliderSpace的工作流程，使得用户能够在不需要明确控制的情况下，探索和组合模型的视觉能力。
在实验部分，作者评估了SliderSpace在多个应用中的有效性，主要集中在概念分解、艺术风格探索和多样性增强。实验使用了SDXL-DMD，一个四步的蒸馏扩散模型。结果显示，SliderSpace能够有效分解高层概念，揭示模型内部的主要变异方向。此外，通过与手动策划的艺术风格列表进行比较，SliderSpace发现的方向在多样性上匹敌甚至超越了这些列表。用户研究表明，SliderSpace生成的图像在多样性、实用性和创造性方面均被用户更为青睐。最后，SliderSpace还被用于解决蒸馏模型中的模式崩溃问题，恢复生成图像的多样性，同时保持生成速度。
SliderSpace的工作原理可以简单理解为一个“滑块”系统，用户只需输入一个简单的提示，比如“画一只猫”，SliderSpace就会自动找到与这个提示相关的多个“滑块”。每个滑块代表一种特定的视觉变化，比如猫的颜色、姿势或背景。用户可以通过调节这些滑块，来组合不同的视觉效果，而不需要自己去定义每一个细节。这就像在调音台上滑动不同的音量控制，来创造出和谐的音乐。通过这种方式，SliderSpace让用户能够更直观地探索和创造图像，同时保持作品的主题一致性和丰富性。
2025-02-02｜HKUST(GZ),SJTU｜⭐️🟡
http ://arxiv.org/abs/2502.00848v1
在文本到图像生成领域，现有模型如StableDiffusionV3和Flux在图像合成方面取得了显著进展。然而，这些模型由于依赖于固定的训练参数，往往面临知识不足的问题，导致在生成新颖或细致的真实世界物体时出现幻觉或失真。为了解决这一问题，本文提出了一种新的框架——RealRAG（基于检索增强的现实图像生成），旨在通过学习和检索真实世界图像来弥补生成模型的知识缺口。RealRAG的核心在于结合缺失的知识，通过自反对比学习训练一个反射检索器，从而增强生成模型的生成能力，提高图像的真实感和准确性。
RealRAG的工作流程可以分为几个主要步骤：
生成图像：首先，使用特定的文本到图像生成模型，根据给定的文本提示生成初步图像。
反射负样本选择：接着，从真实世界图像数据库中检索与生成图像相似度较高的图像，作为反射负样本。这些样本与生成图像在特征空间中相对接近，但提供了补充的知识。
自反对比学习：通过自反对比学习训练反射检索器，使用文本提示作为正样本，同时利用选取的反射负样本进行训练。这一过程确保检索器能够有效地检索出弥补生成模型知识不足的图像。
增强生成：最后，将检索到的图像作为参考输入到生成模型中，生成更加真实的图像，从而有效解决幻觉问题。
为验证RealRAG的有效性，研究者在多个基准数据集上进行了实验，包括斯坦福汽车、斯坦福犬和牛津花卉等。实验结果显示，RealRAG在多个主流生成模型上均实现了显著的性能提升，例如在斯坦福汽车基准测试中，RealRAG相较于传统方法提升了16.18%的FID分数。此外，RealRAG还在生成未见新物体的能力上表现出色，生成的图像在真实感和细节上均优于基线模型。通过人类评估，参与者对RealRAG生成图像的准确性和真实感给予了较高的评分，进一步证明了其在细粒度和新颖物体生成方面的优势。
RealRAG的工作原理可以简单理解为一个“学习与借鉴”的过程。首先，系统会根据用户输入的描述生成一张图像。接着，它会查找一些与这张图像相似但包含更多细节的真实图片，作为“参考”。通过比较这些参考图像，系统能够识别出生成图像中缺失的部分，并学习如何填补这些空白。最后，利用这些额外的知识，系统会生成一幅更加真实、细腻的图像。这个过程就像是一个学生在写作文时，借鉴了很多优秀的范文，最终写出了一篇更精彩的作品。
2025-02-01｜PKU,CMU｜⭐️🟡
http ://arxiv.org/abs/2502.00433v1https ://github.com/ada-cheng/CAT-Pruning
本文提出了一种新颖的加速策略，称为CATPruning（Cluster-AwareTokenPruning），旨在解决文本到图像扩散模型中计算资源消耗巨大的问题。尽管扩散模型在生成任务中取得了显著的进展，尤其是在文本到图像合成领域，但其迭代去噪过程仍然需要消耗大量的计算资源。CATPruning通过结合令牌级裁剪和缓存技术，显著降低了计算成本，并保持了模型的性能。实验结果表明，该方法在计算成本上减少了50%至60%，同时提升了扩散模型的效率。
CATPruning方法主要包括三个关键步骤：
相对噪声幅度：通过计算当前预测噪声与前一步噪声之间的差异，来评估每个令牌的重要性。这一相对噪声幅度用于选择在当前迭代中需要更新的令牌。
选择频率监控：为了避免选择的令牌在后续迭代中出现“陈旧”现象，我们引入了选择频率的监控机制，确保在每次迭代中都能保持令牌选择的一致性。这一过程有助于平衡选择的令牌，避免过度集中在某些令牌上。
空间聚类：通过对令牌进行空间聚类，确保在选择令牌时考虑到空间结构的相关性。这一过程有助于保持生成图像的细节和一致性，从而提高生成质量。
在实验部分，研究者对多个预训练的扩散模型（如StableDiffusionv3和Pixart-Σ）进行了评估，使用了MS-COCO2017和PartiPrompts等数据集。实验采用28和50个采样步骤进行测试。结果显示，CATPruning在28个采样步骤下，计算成本从168.28T降低至90.28T，实现了1.90倍的速度提升，而在50个采样步骤下，速度提升达到约2倍。此外，CATPruning在生成质量方面与标准扩散模型相当，证明了其有效性。
CATPruning的核心方法可以简单理解为三个步骤。首先，研究人员会检查每个令牌的“噪声”，也就是它在图像生成中的重要性。通过比较当前和之前的噪声，他们决定哪些令牌需要更新。接着，他们会记录每个令牌被选择的频率，以确保不会总是选择同一组令牌，这样可以避免生成图像时出现的细节缺失。最后，他们使用一种叫做“聚类”的技术，将相邻的令牌分组，以确保在选择时考虑到它们的空间关系。这种方法不仅加快了生成速度，还提高了生成图像的质量，让图像看起来更加细腻和真实。

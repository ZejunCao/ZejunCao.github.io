293-B2｜高斯过程增强GS，几何神经过程场；开放词汇3D场景分割；数字人系列

主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）发布日期｜2025年2月5日｜周三
2025-02-04｜ManchesterMetropolitanU,PKU,PengchengLab,HDU,WHU｜🟡
http ://arxiv.org/abs/2502.02283v2
3DGaussianSplatting(3DGS)是一种高效的光照合成新视角生成方法，但其依赖于稀疏的结构光束运动（SfM）点云，导致场景重建质量不足。为了解决这一问题，本文提出了一种新的框架——高斯过程增强的3DGaussianSplatting（GP-GS），旨在通过多输出高斯过程（MOGP）模型实现对稀疏SfM点云的自适应和不确定性引导的密集化。该框架利用输入的2D像素和深度图推断新的候选点，并通过不确定性估计引导高方差预测的修剪，从而确保几何一致性并生成密集的点云。这些密集的点云为高质量的3D高斯体提供了良好的初始化，显著提升了重建性能。通过在多个合成和真实世界数据集上的广泛实验，验证了所提框架的有效性和实用性。
GP-GS框架的核心是多输出高斯过程（MOGP），其方法步骤如下：4.问题定义：将点云密集化视为回归问题，目标是根据图像像素和相应的深度先验预测密集的点云。5.输入特征：MOGP的输入特征包括每个图像中的像素坐标和深度信息，输出目标为相应的三维位置和颜色信息。6.自适应采样策略：通过自适应邻域采样机制选择候选像素，以提高模型对稀疏SfM点云的密集化效果。7.不确定性引导过滤：在预测的点云中应用基于方差的过滤方法，去除高不确定性的预测，从而提高密集点云的质量。8.损失函数与优化：通过最大化边际似然和L2正则化的组合损失函数来训练MOGP模型，确保模型的鲁棒性和可泛化性。
在实验部分，GP-GS框架在多个数据集上进行了评估，包括合成数据集和真实世界数据集。实验结果显示，所提方法在点云密集化和新视角合成方面显著优于现有的3DGS*方法。通过使用均值Chamfer距离（CD）、均方根误差（RMSE）和R²等指标评估密集化效果，结果表明，GP-GS在重建细节和几何一致性上表现出色。此外，在新视角合成的定量和定性评估中，GP-GS在PSNR、SSIM和LPIPS等指标上均优于对比方法，尤其在处理复杂场景和高频纹理时，能够有效减少伪影和增强细节保留。
在我们的研究中，我们提出了一种新方法来改善3D场景的重建质量，特别是当我们只有少量的3D点时。我们使用了一种叫做多输出高斯过程的技术，它可以根据我们从图像中获取的像素和深度信息，智能地推断出更多的3D点。这种方法的工作原理是，我们首先选择一些像素作为基础，然后利用这些像素的信息来生成新的3D点。接着，我们会对生成的点进行过滤，去掉那些不可靠的预测，以确保最终的3D场景看起来更加真实和细致。通过这些步骤，我们的方法能够在不同的场景中表现出色，特别是在那些复杂或光照条件不佳的情况下，能够更好地保留细节和结构。
2025-02-04｜UAmsterdam,CMU,NKI
http ://arxiv.org/abs/2502.02338v1
本研究提出了一种新的框架，名为几何神经过程场（GeometricNeuralProcessFields，G-NPF），旨在解决神经场（NeuralField，NeF）在新场景适应性方面的挑战。传统的NeF模型在面对新的三维场景和有限的上下文图像时容易出现过拟合，导致泛化性能差。G-NPF通过将NeF泛化视为一个概率建模问题，能够直接从稀疏的上下文图像中推断出NeF函数的分布。此外，该框架引入了一组几何基函数，旨在捕捉空间结构信息，从而增强模型的泛化能力。通过这种设计，G-NPF不仅提高了对新信号的适应性，还有效地捕获了不确定性。
G-NPF方法的核心在于将神经场泛化问题转化为概率建模，主要包括以下几个步骤：
概率神经过程框架：通过定义上下文集和目标集，G-NPF利用神经过程（NeuralProcesses）来推断潜在变量，从而捕捉函数级的不确定性。
几何基函数：设计一组几何基函数，这些函数通过编码空间结构信息（如高斯结构），实现对局部信息的聚合，增强模型对新场景的适应能力。
层次化潜变量模型：构建层次化潜变量模型，结合全局和局部潜变量，以便更有效地参数化NeF，确保在有限观察数据下仍能保持较好的泛化性能。
在实验部分，G-NPF在多个数据集上进行了广泛验证，包括1D信号回归、2D图像回归和3D新视图合成等任务。结果表明，G-NPF在这些任务中均优于现有的基线方法。在2D图像回归实验中，G-NPF在CelebA和Imagenette数据集上表现出色，显著提高了PSNR值。在3D新视图合成任务中，G-NPF能够在仅提供一到两个视图的情况下，生成高质量的图像，证明了其在复杂场景中的有效性。此外，实验还展示了G-NPF在处理有限上下文数据时的鲁棒性，能够准确重建高频细节。
在G-NPF的工作原理中，我们可以把它想象成一个聪明的画家，他在绘画时只看到了几张模糊的草图，但他能够通过这些草图推测出完整的画面。这个过程分为几个步骤：首先，画家会收集所有可用的信息（这些信息就像是我们提供的上下文图像），然后利用这些信息来构建一个关于画面的整体理解（这就是几何基函数的作用）。接着，画家会根据不同的细节（如颜色和形状）来调整他的画作，使之更接近真实的样子（这就是层次化潜变量模型的工作）。最终，画家不仅能根据有限的信息画出漂亮的画，还能在不同的场景中灵活应对，确保每幅画都能展现出应有的细节和生动感。
2025-02-04｜NVIDIA,POSTECH
http ://arxiv.org/abs/2502.02548v1https ://nvlabs.github.io/Mosaic3D/
Mosaic3D是一种新提出的基础数据集和模型，旨在解决开放词汇3D场景理解的问题。该研究通过引入一种新颖的数据生成管道和训练框架，针对3D区域的精确分割、全面的文本描述以及足够的数据集规模等三个关键要求，提出了一种高效的解决方案。Mosaic3D-5.6M是该研究的核心数据集，包含来自多个3D场景的数据，涵盖超过30,000个注释场景和5.6百万个掩码-文本对，显著超越了现有数据集的规模。这一模型结合了基于对比学习的3D编码器和轻量级的掩码解码器，能够实现开放词汇的3D语义和实例分割，推动了3D场景理解的前沿。
Mosaic3D的方法分为以下几个主要步骤：4.数据生成管道：利用最先进的开放词汇图像分割模型和区域感知的视觉-语言模型（VLM），自动生成高质量的3D掩码-文本对。该管道确保精确的区域分割和丰富的文本描述。5.3D编码器训练：采用稀疏卷积网络（SparseConvNets）作为基础模型，通过对比学习将每个点的特征与文本嵌入对齐，从而实现语言和几何特征的有效结合。6.掩码解码器训练：在3D编码器的基础上，训练一个轻量级的掩码解码器，该解码器能够直接从语言对齐的特征中预测对象实例，首次实现单阶段的开放词汇3D实例分割，无需真实标签。
实验部分主要包括三个方面的评估：首先，研究团队分析了数据集规模对模型性能的影响，逐步增加训练数据以观察模型的准确性提升；其次，Mosaic3D在多个开放词汇3D场景分割基准上与现有方法进行了比较，显示出优越的性能；最后，团队通过消融实验评估了数据生成引擎中各个组件的贡献，确认了数据规模和注释质量在开放词汇3D理解中的重要性。实验结果表明，Mosaic3D在多个基准测试中均取得了最佳性能，证明其在3D场景理解中的有效性和创新性。
Mosaic3D的方法可以想象成一个高效的工厂，专门用来制造3D场景的“说明书”。首先，这个工厂通过先进的技术和工具，自动识别和标记3D场景中的各种对象，比如沙发、桌子等，并为每个对象生成详细的描述。接着，工厂会将这些信息整合到一个大型数据库中，确保每个对象的特点和位置都被准确记录。最后，工厂还会训练一个“智能助手”，使其能够根据这些信息快速识别新场景中的对象，而不需要事先知道它们的具体类别。这种方法不仅提高了效率，还使得3D场景的理解更加灵活和智能。
2025-02-04｜SYSU,PengChengLab｜AAAI2025
http ://arxiv.org/abs/2502.02372v1
在计算机视觉领域，生成虚拟数字化身是一个重要的研究方向。近年来，基于神经辐射场（NeRF）的方法取得了显著的成果。然而，现有的研究通常假设训练数据是静态且固定的，这与现实世界中人物的外观和姿势不断变化的情况不符。为了解决如何在保持旧外观渲染能力的同时更新人类化身的问题，本文提出了一种基于持续学习的可维护化身（MaintaAvatar）。该方法通过引入全球-局部联合存储模块和姿态蒸馏模块，有效避免了模型在学习新外观时的灾难性遗忘，从而实现了对虚拟化身的持续学习与更新。
本研究的核心是MaintaAvatar模型，其主要由以下几个部分构成：
全球-局部联合存储模块：该模块能够分别存储不同外观的全局和局部信息，便于模型更好地区分各种外观之间的差异，从而防止颜色混合现象。
姿态蒸馏模块：通过提取过去任务的姿态信息作为监督信号，帮助模型在学习新任务时更好地保留过去的姿态信息，避免渲染错误的姿态。
预训练模型：利用多个不同人类身体的预训练模型作为初始化，以便在仅需少量图像的情况下快速调整新的化身模型。
在ZJU-MoCap和THuman2.0两个数据集上进行实验，结果表明MaintaAvatar在多种姿势、视角和外观的渲染上均表现优异。与其他现有方法相比，MaintaAvatar在图像质量（PSNR、SSIM、LPIPS等指标）上均取得了显著提升。具体而言，实验结果显示，MaintaAvatar能够在较少的训练样本下快速适应新外观，同时保持对先前外观的良好渲染能力。通过对比实验，验证了全球-局部联合存储模块和姿态蒸馏模块的有效性，进一步证明了该方法在持续学习中的优势。
MaintaAvatar的工作原理可以简单理解为一个不断学习和更新的虚拟人偶。想象一下，如果你有一个玩偶，它的衣服和姿势能够随着时间的推移而变化。MaintaAvatar就像这样的玩偶，它不仅能学会新的衣服样式，还能记住以前的样子，不会忘记。这个过程通过两个主要部分实现：首先，它有一个聪明的存储系统，能区分不同衣服的颜色和样式，避免混淆；其次，它有一个“记忆助手”，可以帮助它在学习新姿势时，保持对旧姿势的记忆。这样一来，无论你想让玩偶穿什么衣服、摆什么姿势，它都能轻松应对。
2025-02-04｜AMD
http ://arxiv.org/abs/2502.02468v1
本研究旨在开发一种新颖的方法，从普通的笔记本电脑摄像头生成高保真度的人类虚拟化身。传统的高保真虚拟化身生成需要昂贵的专业摄影设备和艺术劳动，而近期的研究虽然允许使用智能手机的RGB和红外传感器自动构建虚拟化身，但仍依赖于高分辨率相机和强大的服务器处理能力。为满足现代应用（如视频会议）对实时生成虚拟化身的需求，本研究提出了一种基于3D形态模型、关键点检测、照片级真实感纹理生成对抗网络（GAN）和可微渲染技术的自动化系统。该系统能够在低质量摄像头图像和边缘计算的限制下，生成可动画化的高保真虚拟化身，充分利用移动芯片的神经计算能力。
本研究的方法主要分为两个部分：形状生成和纹理映射生成。具体步骤包括：4.3D形态模型（3DMM）适配：通过迭代优化，调整3DMM的参数，以适应用户的面部特征。该过程使用FLAME模型，初始化时考虑头部姿势和场景照明。5.纹理映射生成：采用拉普拉斯金字塔融合技术，处理来自多个视角的用户图像，生成均匀照明的纹理。此过程包括从不均匀照明的源图像中提取高频内容，并将其与均匀照明的模板纹理结合。6.纹理细化网络：使用预训练的面部恢复模型对生成的纹理进行细化，提升细节和清晰度，确保最终生成的虚拟化身在视觉上更具真实感。
为了验证所提方法的有效性，研究团队从不同角度（左、正、右）捕获用户图像，所有图像均在1080p的普通笔记本电脑摄像头上拍摄。实验考虑了不同光照条件下的图像质量，包括均匀照明和方向性照明的组合。通过定性和定量评估，比较了生成的虚拟化身与现有方法（如Avaturn）的性能。结果显示，所提方法在处理低质量图像时，能够生成更高质量的虚拟化身，且在亮度对称误差等指标上表现优越，验证了其在资源受限环境下的实用性。
在这个研究中，科学家们想要用普通的笔记本电脑摄像头来创建非常真实的虚拟人像，像是你的数字化身。以往制作这样的虚拟人像需要昂贵的设备和专业的技术，但他们找到了一个新方法，能够在普通设备上完成这个任务。首先，他们使用一种特殊的3D模型来捕捉人脸的形状，然后通过处理不同角度的照片来生成肤色和细节。为了让这些细节更加真实，他们还使用了一种叫做“拉普拉斯金字塔”的技术，这可以帮助他们在不同光照条件下提取面部特征。最后，他们用一个智能算法来提高生成图像的质量，确保这些虚拟人像在视频会议等应用中看起来非常自然。

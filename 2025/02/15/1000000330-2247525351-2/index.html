2025-02-03｜UCBerkeley,MIT,NVIDIA,THU｜⭐️🟡
http ://arxiv.org/abs/2502.01776v1
在视频生成领域，扩散变换器（DiffusionTransformers,DiTs）因其出色的生成效果而备受瞩目。然而，其高昂的计算成本限制了在实际应用中的推广，通常在高性能GPU上生成几秒的视频需要数十分钟。本文提出了一种名为SparseVideoGen（SVG）的新框架，旨在通过利用3D全注意力中的固有稀疏性来提高推理效率。研究表明，注意力头可以根据不同的稀疏模式动态分为两组：空间头和时间头，分别关注同一帧内的空间相关性和跨帧的时间相关性。SVG通过在线分析策略捕捉这些动态稀疏模式，并结合硬件友好的张量布局转换和定制的内核实现，显著提升了视频生成的速度，同时保持了生成质量。
SVG的核心方法包括以下几个方面：
在线稀疏模式识别：SVG通过在线分析策略动态识别每个注意力头的稀疏模式。该策略仅随机采样1%的输入数据，计算其空间和时间稀疏性，并选择误差最小的模式进行后续推理，从而有效减少计算负担。
硬件高效布局转换：为了提高计算效率，SVG引入了一种张量布局转换方法，将时间头的非连续稀疏布局转变为连续的硬件友好型布局。这种转换使得在硬件加速器上进行计算时的效率大幅提升。
定制内核与量化：SVG还利用定制的CUDA内核和FP8量化技术，进一步加速视频生成过程，同时保持高质量的生成结果。通过优化内核的并行处理能力，SVG在保持质量的前提下显著提高了推理速度。
实验部分中，SVG在多个开放源代码的前沿视频生成模型上进行了评估，包括CogVideoX-v1.5和HunyuanVideo。结果显示，SVG在生成质量（如PSNR、SSIM和LPIPS）上均优于其他基线方法，同时实现了高达2.33倍的端到端速度提升。具体而言，在生成720p分辨率的视频时，SVG的平均PSNR超过29.99，表明其在保持生成质量的同时，显著提高了计算效率。此外，通过敏感性测试，SVG展示了其对在线稀疏模式识别比例的鲁棒性，证明即使只分析1%的输入数据，也能获得接近最优的生成质量。
SVG的工作原理可以简单理解为一个“智能选择系统”。首先，它通过一种叫做“在线分析”的方法，快速查看一小部分数据，找出哪些信息最重要，然后根据这些重要信息来决定后续的计算方式。就像在烹饪时，厨师只需挑选出最新鲜的食材，而不是检查所有的食材。接着，SVG会对这些选出的信息进行重新排列，使得计算机在处理时更加高效，类似于把零散的食材整齐地摆放在一起，方便烹饪。最后，SVG还使用一些特别的技巧来加快计算速度，比如定制的计算公式和减少数据存储空间的方法。这些步骤共同作用，使得SVG在生成视频时既快又好。
2025-02-04｜MetaGenAI,Tel-AvivU｜⭐️🟡
http ://arxiv.org/abs/2502.02492v1https ://hila-chefer.github.io/videojam-paper.github.io/
在视频生成领域，尽管近年来取得了显著进展，现有模型仍然面临在运动表现、动态和物理模拟方面的挑战。本文提出了VideoJAM框架，旨在通过引入有效的运动先验来显著增强视频生成模型的运动一致性。该框架通过促使模型学习联合的外观-运动表示，解决了传统像素重建目标造成的运动和外观之间的矛盾。VideoJAM的设计允许它在不需要对训练数据或模型规模进行修改的情况下，适用于各种视频生成模型。实验结果表明，VideoJAM在运动一致性方面达到了最先进的性能，超越了许多竞争对手，并且在视觉生成质量上也有所提升。
VideoJAM框架的核心由两个互补单元组成，分别在训练和推理阶段发挥作用。具体方法如下：4.训练阶段：
目标扩展：修改训练目标，使其同时预测生成的视频像素和对应的运动信息。通过将视频与其运动表示配对，模型能够学习到联合的外观-运动表示。
架构调整：在模型架构中添加两个线性层，以支持输入和输出的双重格式，分别用于处理外观和运动信息。
推理阶段：
Inner-Guidance机制：在生成过程中，利用模型自身的运动预测作为动态指导信号，引导生成过程朝向一致的运动。这一机制使得模型能够根据先前生成的运动信息，调整生成的内容，从而实现更连贯的运动表现。
为了验证VideoJAM的有效性，研究团队进行了广泛的定性和定量实验，评估其在不同模型规模和多种运动类型下的表现。实验结果显示，应用VideoJAM后，预训练视频模型的运动一致性显著提高，超越了现有的先进模型。具体实验包括：
基准测试：在VideoJAM-bench和MovieGen等基准上进行评估，确保覆盖各种真实世界的运动场景。
人类评估：通过人类评审员对生成视频的质量、运动一致性和文本对齐度进行比较，结果表明VideoJAM在所有评估中均表现优越。
在VideoJAM的工作原理中，首先在训练阶段，模型会学习如何同时生成视频的图像和运动信息。这就像在画一幅画时，不仅要画出图像的细节，还要考虑画中物体的运动方式。为此，研究人员在模型中添加了两个新的部分，帮助模型理解和处理这两种信息。
在生成视频时，VideoJAM引入了一个名为Inner-Guidance的机制。这个机制就像是一个教练，帮助模型在生成每一帧时，参考之前生成的运动信息，从而确保视频中的动作看起来自然且连贯。通过这种方式，VideoJAM能够生成更真实的运动效果，让观看者感受到更流畅的动态表现。
2025-02-02｜ZJUCAD&CG,ZJU｜⭐️🟡
http ://arxiv.org/abs/2502.01690v1https ://tankowa.github.io/HuViDPO.github.io/
随着生成对抗网络（GAN）和扩散模型的快速发展，文本到视频（T2V）生成技术得到了显著提升。然而，现有的T2V方法在与人类偏好的对齐方面仍面临挑战。本文提出了一种新方法HuViDPO，通过直接偏好优化（DPO）策略，旨在提升生成视频的美学质量和对人类偏好的对齐。研究中，作者们首先识别了现有T2V生成技术的局限性，包括缺乏有效的损失函数和配对视频偏好数据的稀缺性。为了解决这些问题，作者们提出了三项创新策略，首次将DPO引入T2V任务，构建小规模人类偏好数据集，并采用首帧引导策略来提高视频生成的灵活性和质量。
HuViDPO方法的核心在于有效地将DPO策略融入到T2V生成任务中。具体步骤包括：
损失函数设计：通过精心构造的损失函数，利用人类反馈来指导视频生成与人类偏好的对齐，避免使用单独的奖励模型，从而减少计算资源的依赖。
小规模人类偏好数据集的构建：为每个动作类别构建小规模的人类偏好数据集，并通过微调模型来提高生成视频的美学质量，降低训练成本。
首帧引导策略：利用首帧中的丰富信息来指导后续帧的生成，增强视频生成的灵活性。同时，采用稀疏因果注意力机制以提升生成视频的质量。
这些方法的结合使HuViDPO能够在保持生成灵活性的同时，确保生成视频的美学质量和时序一致性。
在实验部分，HuViDPO的性能通过与多种基线方法的比较进行评估。实验涵盖了八个动作类别，使用了小规模人类偏好数据集进行微调。结果显示，HuViDPO在生成视频的美学、时序一致性和与人类偏好的对齐方面均表现优越。具体来说，HuViDPO在CLIP评分、结构相似性（SSIM）和均方误差（MSE）等指标上均超过了其他基线方法。此外，通过用户研究，参与者对HuViDPO生成的视频在美学和一致性方面给予了较高评价，进一步验证了该方法的有效性和实用性。
HuViDPO的核心方法可以简单理解为通过人类的反馈来改进视频生成。具体来说，研究者们首先设计了一种特殊的“损失函数”，它可以帮助模型更好地理解人们喜欢什么样的视频，而不是依赖复杂的奖励系统。接着，他们创建了小型的数据集，里面包含人们对视频的偏好，这样模型就能在较少的数据下进行有效训练。最后，他们采用了一种“首帧引导策略”，即利用视频的第一帧来帮助生成后面的帧，这样可以确保视频的内容更加连贯。同时，他们还引入了一种新的注意力机制，进一步提升了视频的质量和流畅度。通过这些方法，HuViDPO能够生成更符合人类审美的视频，且生成过程更加高效。
2025-02-04｜FDU,ShanghaiAcademyofArtificialIntelligenceforScience,MeituInc.MTLab｜⭐️🟡
http ://arxiv.org/abs/2502.02088v2
随着视频生成技术的快速发展，视频基础模型在生成用户满意的视频方面仍面临诸多挑战，尤其是在保持主题一致性、运动流畅性和美学质量方面。为了解决这些问题，本文提出了一种新的方法——迭代偏好优化（IPO），旨在通过与人类偏好的对齐来提高文本到视频（T2V）生成的质量。IPO结合了后训练技术和人类反馈，利用多模态大语言模型和批评模型来优化生成过程，从而在不需要大量标注数据的情况下，提升生成视频的质量。此外，实验结果表明，IPO能够有效地使得一个仅有2B参数的模型超越5B参数的模型，显示出其在视频生成领域的潜力。
IPO框架的核心包括三个主要组成部分：偏好数据集的收集、批评模型的训练和迭代学习流程。具体方法如下：4.偏好数据集收集：首先，定义多个视频生成类别（如人类、动物、动作等），并随机组合生成视频。然后，手动为这些视频打上两种标签：一是成对排名标签，表示两个视频的偏好比较；二是逐点评分标签，描述视频质量为好或坏。5.批评模型训练：使用收集的偏好数据集训练批评模型，该模型能够进行成对排名和逐点评分。通过多模态大语言模型，批评模型能够生成更为准确的偏好标签，从而优化视频生成模型。6.迭代偏好优化：在每次训练迭代中，批评模型评估生成的视频，并提供反馈。根据这些反馈，生成模型进行调整，以更好地对齐人类偏好。这个过程可以多次重复，逐步提升生成视频的质量。
我们在VBench基准上进行了全面的实验，评估IPO的性能。实验结果显示，经过迭代优化后，2B参数的模型在多个指标上超越了5B参数的模型，尤其是在视频的一致性、流畅性和美学质量方面。具体来说，优化后的模型在时间一致性和语义对齐方面表现出显著改善，减少了视频的闪烁现象，并增强了运动的流畅性。此外，我们还进行了消融研究，以验证IPO框架中各个组件的贡献，结果表明，迭代训练显著提升了模型的性能和稳定性。
在我们的方法中，首先，我们收集了很多视频，然后请人们对这些视频进行评价，告诉我们哪个视频更好。接下来，我们使用这些评价来训练一个“批评者”模型，这个模型能够判断视频的质量。然后，在生成新视频时，我们会让这个批评者评价生成的视频，并根据反馈不断优化生成的过程。通过这种反复的调整，我们可以逐步提高视频的质量，使其更符合观众的期待。简单来说，就是通过不断地生成、评价和改进，让视频变得越来越好。
2025-02-03｜UNC-ChapelHill,UChicago,Oxford,Stanford｜⭐️🟡
http ://arxiv.org/abs/2502.01719v3https ://aiming-lab.github.io/MJ-VIDEO.github.io/
随着视频生成技术的进步，利用文本指令合成视频的能力已经显著提高。然而，当前的模型依然面临诸多挑战，包括指令不一致、内容幻觉以及生成不安全或有偏见的输出。为了应对这些问题，研究者们提出了MJ-BENCH-VIDEO，这是一个大规模视频偏好基准，旨在从五个关键方面评估视频生成的效果：对齐性、安全性、细致性、一致性和偏见与公平性。该基准包含28个细分标准，为视频偏好的全面评估提供了支持。基于此数据集，研究团队进一步提出了MJ-VIDEO，一个基于混合专家（MoE）的视频奖励模型，能够动态选择相关专家，以便根据输入的文本-视频对进行精确的偏好判断。通过对MJ-BENCH-VIDEO的广泛基准测试，MJ-VIDEO在视频偏好评估中表现出了显著的优势，整体和细分偏好判断分别提高了17.58%和15.87%。
MJ-VIDEO的设计基于多层次的混合专家架构，旨在提供细致的偏好评估，具体方法如下：7.数据收集：通过三种策略收集视频对和相应的文本提示，包括从现有视频偏好数据集中获取、将图像转化为视频以及直接从文本生成视频。8.数据过滤：使用GPT-4和人工评审相结合的方式，过滤掉与提示不一致或质量较低的视频对，确保数据集的质量。9.数据注释：人类注释者对每个视频对进行打分，依据28个细分标准进行评估，最终得出每个视频对的总体偏好分数。10.多阶段训练：采用三阶段训练策略，首先训练细分标准评分层，其次训练方面路由层，最后联合优化这两个层以确保整体偏好分数的有效性。
在实验中，研究团队利用MJ-BENCH-VIDEO和MJ-VIDEO探讨多个问题，包括现有大型视觉语言模型（LVLMs）在视频偏好评估中的有效性，以及细分偏好注释对视频奖励模型性能的提升。实验结果显示，MJ-VIDEO在多个基准上都优于现有模型，特别是在细分评估方面表现突出。通过将MJ-VIDEO引入偏好微调过程，生成的视频在对齐性方面得到了显著改善。此外，研究还进行了消融实验，验证了MoE架构在视频偏好判断中的优势。
MJ-VIDEO的工作原理可以简单理解为一个智能评审系统，它通过多个专业的“评审员”来判断视频的好坏。首先，系统会收集大量的视频和对应的文本描述，然后用一些智能工具来筛选出质量较高的视频。接下来，真正的评审员会根据28个不同的标准来打分，比如视频是否安全、是否符合描述、画面是否清晰等。最后，系统会利用这些打分来训练自己，让它在未来能更准确地评估新的视频。通过这样的方式，MJ-VIDEO不仅能提供精准的评估结果，还有助于提高未来视频生成的质量。
2025-02-04｜IIITHyderabad,UCanberra｜IUI2025｜⭐️⭐️
http ://arxiv.org/abs/2502.02172v1https ://editiq-ave.github.io/
在本研究中，我们提出了一种名为EditIQ的全自动视频编辑框架，旨在通过对话理解和视觉显著性分析，对静态宽视角视频进行电影化编辑。该系统的设计初衷是解决传统视频制作中由于设备、场地和人力资源限制而带来的挑战。EditIQ通过模拟多个虚拟摄像机，从单一的高分辨率静态摄像机捕捉的画面中生成多个镜头，进而形成一系列可用于编辑的“急件”。这些镜头不仅基于对话内容的理解，还结合了场景中显著的视觉元素，以确保最终输出的视频在叙事上连贯且视觉上引人入胜。
EditIQ的工作流程分为四个主要阶段：预处理、急件生成、潜力计算和镜头选择。具体方法如下：11.预处理：从输入的宽视角视频中提取关键特征，包括演员跟踪和对话转录，以便为后续步骤提供支持。12.急件生成：通过模拟虚拟平移缩放摄像机，自动生成多种镜头组合，涵盖不同的演员和场景元素。该过程使用优化算法确保镜头的构图符合专业摄像师的标准。13.潜力计算：利用大型语言模型（LLM）分析对话内容，结合视觉显著性预测，计算每个镜头在叙事和视觉上的重要性。14.镜头选择：将上述计算结果整合，形成一个离散优化问题，通过动态规划选择最佳镜头序列，以确保叙事的流畅性和视觉的吸引力。
为了验证EditIQ的有效性，我们在BBCOldSchool数据集上进行了心理物理实验，涉及20名参与者。该数据集包含多种类型的表演视频，旨在评估自动化编辑系统的表现。实验结果显示，EditIQ在叙事有效性、情感保留和整体观看体验方面超越了多种竞争方法，包括随机编辑和基于说话者检测的编辑策略。通过对比人类编辑的节目，EditIQ的输出在视觉连贯性和情感表达上得到了参与者的高度认可。此外，我们还扩展了评估范围，包含了多个高质量的4K剧场表演片段，以检验系统在不同场景下的适应性和表现。
EditIQ的工作流程可以简单理解为一个四步走的过程。首先，它会对输入的视频进行“清理”，提取出演员的位置信息和对话内容。接下来，它会像魔法一样，通过虚拟摄像机的方式，自动生成多个不同角度的镜头，让每个演员的表现都能被捕捉到。然后，系统会分析这些镜头，结合对话的内容和场景的视觉效果，评估每个镜头的重要性。最后，它会选择出最合适的镜头，确保整个故事流畅且引人入胜。通过这样的方式，EditIQ能够在没有人工干预的情况下，自动制作出高质量的影片。

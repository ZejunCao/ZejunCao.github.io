2025-01-31｜UTokyo｜⭐️🟡
http ://arxiv.org/abs/2501.19252v1https ://sites.google.com/view/t2v-dlbs
在文本到视频的生成领域，近年来的扩散模型取得了显著进展，能够生成高质量的照片级视频。然而，这些生成的视频常常存在不自然的运动、变形和静止场景等问题。为了解决这些问题，本文提出了一种新的推理时间对齐方法，称为扩散潜在束搜索（DiffusionLatentBeamSearch,DLBS），结合了前瞻性估计器（LookaheadEstimator）。该方法旨在通过优化生成视频与文本提示之间的对齐度，提升视频的感知质量。通过对现有评估指标进行加权校准，DLBS能够在不更新模型参数的情况下，显著改善生成视频的质量，并在相同的搜索预算下超越贪婪搜索和最佳采样方法。
本文的方法分为几个关键步骤：
扩散潜在束搜索（DLBS）：通过束搜索算法探索潜在空间，选择具有高对齐奖励的潜在样本。该方法能够在每个时间步生成多个潜在样本并评估其奖励，以便选择最佳结果。
前瞻性估计器：为了减少奖励评估中的噪声，前瞻性估计器利用确定性DDIM（DenoisingDiffusionImplicitModels）对当前潜在样本进行多步前瞻性采样，从而更准确地估计奖励。
奖励校准：通过将多个感知质量指标的线性组合进行加权，校准生成视频的奖励。这一过程确保了生成结果更符合人类评价标准，提升了视频生成的自然性和流畅性。
在实验部分，作者对DLBS进行了全面评估，与现有的最佳采样（BoN）和贪婪搜索（GS）方法进行了比较。实验结果显示，DLBS在不同的搜索预算下均能显著提高生成视频的质量，尤其是在预算较高时（如KB=16或32）。此外，增加前瞻性步骤和去噪步骤也对提升生成效果有显著帮助。通过使用AI反馈（如Gemini和GPT-4o）进行评估，DLBS不仅提升了校准奖励，还改善了原始视频质量的反馈，表明该方法在实际应用中的有效性。
在这项研究中，作者提出了一种新方法，叫做扩散潜在束搜索（DLBS），用于生成与文本描述相符的视频。简单来说，这个方法就像是一种智能的“视频制作助手”。它首先从一组可能的画面中选择出一些，然后对这些画面进行评估，看看哪些最符合我们的要求。为了让评估更准确，DLBS还会提前预测未来的画面效果，帮助我们选择出更好的选项。最后，DLBS通过调整评估标准，确保生成的视频不仅符合文本描述，还能让人觉得自然流畅。通过这样的方式，DLBS能生成更高质量的视频，帮助我们更好地实现创意和想法。
2025-01-31｜SJTU｜⭐️🟡
http ://arxiv.org/abs/2501.19331v1
视频上色是计算机视觉中的一项传统任务，其目的是将灰度视频序列转换为生动的彩色版本，同时保持时间一致性。现有的方法在颜色饱和度和时间一致性方面面临挑战。为了解决这些问题，本文提出了一种新颖的自动视频上色方法，称为基于调色板引导的稳定视频扩散（SVD）。该方法通过引入调色板作为全局指导，增强了生成颜色的丰富性和一致性，从而克服了传统方法中常见的颜色不饱和和时间不一致的问题。实验结果表明，所提出的方法在颜色饱和度和视频质量方面均优于现有技术。
本文的方法主要分为以下几个步骤：
视频扩散模型架构：采用稳定视频扩散（SVD）作为基础模型，将灰度视频序列与调色板引导相结合，以生成高质量的彩色视频。
调色板引导机制：
调色板生成：使用K均值聚类算法从参考帧中提取五种主要颜色，形成调色板。
颜色嵌入：将生成的调色板通过线性层转换为颜色嵌入，以确保颜色信息在特征表示中得到一致性整合。
去噪网络：在去噪过程中，将灰度潜在编码和采样的高斯噪声结合，进而生成彩色潜在编码。
灵活的调色板生成方法：支持多种生成方式，包括从参考图像提取、随机采样和用户指定等，以适应不同的应用场景。
为验证所提出方法的有效性，进行了三组实验：
与现有方法的比较：通过定性和定量分析，评估新方法在颜色饱和度、视觉质量和时间一致性方面的表现，结果显示新方法在各项指标上均优于对比方法。
消融研究：分析调色板引导的必要性及不同调色板生成方法对结果的影响，结果表明有调色板引导的模型在颜色丰富性和视觉效果上显著优于无调色板的模型。
调色板引导的灵活性评估：通过不同调色板生成方法的对比，展示了调色板对颜色化结果的影响，表明调色板引导机制能够提供多样化且可控的颜色化效果。
在这项研究中，我们提出了一种新方法来给黑白视频上色。首先，我们使用一种叫做“稳定视频扩散”的技术，这种技术帮助我们在处理视频时保持颜色的一致性。接着，我们设计了一种调色板引导的方法，这就像给视频上色时提供了一组颜色样本。我们从视频中的某些帧提取出主要颜色，形成一个调色板，然后在上色时使用这些颜色，确保每一帧的颜色都能协调一致。我们还允许用户选择不同的调色板生成方式，比如从其他图片中提取颜色，或者随机选择颜色。通过这些方法，我们的上色结果不仅颜色鲜艳，而且视频的每一帧之间也保持了良好的颜色一致性。实验结果表明，这种方法在上色质量上优于许多现有技术，能够更好地满足用户的需求。
2025-01-30｜StonyBrookU,ByteDance,Apple｜⭐️🟡
http ://arxiv.org/abs/2501.18801v1
本研究介绍了一种新颖的音乐驱动图像动画生成模型——MuseDance，旨在通过结合音乐和文本输入来生成个性化的舞蹈视频。尽管图像动画在多模态研究中已经取得了一定进展，但音乐驱动的舞蹈视频生成仍然是一个相对未被充分探索的领域。MuseDance通过使用参考图像、音乐片段和描述性文本提示，能够生成与音乐节奏同步的动态视频，满足用户的个性化需求。此模型的创新之处在于其消除了对复杂运动指导输入（如姿势或深度序列）的依赖，使得不同专业水平的用户都能轻松创建多样化的舞蹈视频。这一方法不仅提升了用户的参与度，还在电影、社交媒体和教育等多个行业中展现了广泛的应用潜力。
MuseDance的训练过程分为两个主要阶段。第一阶段专注于单帧生成，模型通过随机选择目标图像和参考图像来学习视觉特征，并利用DensePose提取精确的姿势信息。第二阶段则引入音乐和文本作为生成视频序列的驱动因素。在这一阶段，模型通过三个新模块——音乐理解模块、节拍对齐模块和运动对齐模块，确保生成的动画与音乐特征和节拍一致。具体步骤如下：
基础训练：模型先在单帧图像上进行训练，提取并分离外观和运动信息。
引入动态输入：在第二阶段，模型利用音乐动态和文本提示来生成与音乐节奏一致的舞蹈视频。
模块整合：通过自注意力机制，确保生成视频的时间一致性和内容连续性，同时实现音乐、节拍和运动之间的语义对齐。
本研究构建了一个新的多模态数据集，包含2904个舞蹈视频，并配有相应的背景音乐和文本描述。实验中，使用NVIDIAA100GPU进行训练，模型在两个阶段各进行了30000步的训练。通过与现有模型的对比，评估了生成视频的质量，使用了多种度量标准，包括PSNR、SSIM和FVD等。结果表明，MuseDance在生成视频的清晰度和一致性方面表现优越，尤其在生成非人类对象舞蹈视频方面，展示了其强大的适应性和创造力。
MuseDance的工作原理可以简单理解为一个两步走的过程。首先，模型会学习如何从一张静态的参考图像中提取出舞蹈动作的特征，比如舞者的姿势和动作。接着，在第二步中，模型会添加音乐和文本提示，让舞者的动作与音乐节奏同步。这就像是给一个人提供了一首歌和一些舞蹈说明，然后让他根据这些信息来跳舞。通过这种方式，无论是人还是其他物体，MuseDance都能生成流畅的舞蹈视频，帮助用户轻松创建个性化的内容。这个过程不需要用户有专业的舞蹈知识，只需简单的输入，便能看到令人惊叹的舞蹈效果。
2025-01-31｜WHU,ZJU｜⭐️🟡
http ://arxiv.org/abs/2501.18940v1https ://wangsai23.github.io/TV-Dialogue/
随着大型语言模型（LLMs）的快速发展，文本和图像的对话生成取得了显著进展，但视频对话生成仍处于探索阶段，面临独特的挑战。本文提出了一种新任务——主题感知视频对话制作（TVDC），旨在根据视频内容和用户指定的主题生成新的对话。为此，研究者们设计了TV-Dialogue框架，它结合了主题对齐和视觉一致性，确保生成的对话与视频中的情感和角色行为相匹配。通过实时的沉浸式交互，TV-Dialogue能够理解视频内容并生成符合给定主题的对话。为评估生成的对话质量，研究团队建立了一个多层次的评估基准，展示了TV-Dialogue在自收集数据集上的有效性，尤其是在零样本生成的能力上，显示出其广泛的应用潜力，如视频重创作和影片配音。
TV-Dialogue的设计分为几个关键模块，以确保生成的对话符合用户指定的主题并与视频内容一致。具体方法如下：
主题感知角色生成模块：根据用户指定的主题，创建新情节并为每个角色分配新角色。每个角色以第一人称视角进行沉浸式交互，从而生成与主题高度一致的对话。
视觉驱动对话预测模块：在对话生成期间，角色的细微视觉表现（如面部表情和肢体动作）指导对话内容的生成。每个角色在对话过程中需要保持与其视觉表现一致。
对话自我修正模块：为了进一步提高对话质量，TV-Dialogue引入了一种自我修正机制，能够评估生成的对话是否符合主题和上下文，并提供改进建议。若生成的对话不符合标准，系统会提示重新生成。
为验证TV-Dialogue的有效性，研究团队构建了一个名为多主题视频对话数据集（MVD），包含351个视频片段，涵盖多种应用场景。通过传统评估指标（如BERTScore、METEOR和ROUGE-L）和定性评估，研究者对生成的对话进行了全面评估。实验结果表明，TV-Dialogue在多个评估指标上均优于现有的主流模型，尤其是在逻辑连贯性和内容多样性方面。此外，研究还表明，TV-Dialogue生成的新对话在视频文本检索任务中表现出显著的提升，强调了其在多模态任务中的应用潜力。
在这个研究中，TV-Dialogue的工作方式可以想象成一场电影的排练。首先，系统根据观众的要求，编写一个新的剧本，这个剧本围绕着某个特定主题展开，比如“总统选举”。接着，系统为每个角色分配新角色，让他们在视频中进行对话。每个角色都像演员一样，从自己的视角进行表演，确保他们的对话不仅符合剧本，还与他们在视频中的情感和动作相匹配。为了确保对话的质量，系统会不断检查生成的对话是否合适，并给出修改建议，就像导演在排练中不断指导演员一样。这样，最终生成的对话就能让观众感受到身临其境的体验。

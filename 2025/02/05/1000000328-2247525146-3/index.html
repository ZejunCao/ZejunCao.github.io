2025-01-22｜HIT(SZ),THU｜⭐️⭐️
http ://arxiv.org/abs/2501.12909v1https ://filmagent.github.io/
FILMAGENT是一个基于大型语言模型（LLM）的多代理协作框架，旨在实现虚拟三维空间中的端到端电影自动化制作。该框架模拟了传统电影制作的工作流程，通过多个LLM代理分别扮演导演、编剧、演员和摄影师等角色，分阶段进行创意开发、剧本编写和摄影等关键环节。FILMAGENT的目标是通过代理之间的协作和反馈，减少制作过程中的错误和不一致性，从而提升影片的整体质量。通过对15个创意生成的视频进行人类评估，FILMAGENT在多个方面表现优异，平均得分为3.98（满分5分），显示了多代理协作在电影制作中的可行性和优势。
FILMAGENT的工作流程被分为三个主要阶段：创意开发、剧本编写和摄影。具体方法如下：
创意开发：
导演根据初步故事构思生成角色档案，并将创意扩展为详细的场景大纲，明确每个片段的情境和角色。
剧本编写：
编剧在导演的指导下撰写初稿，包含角色对话、位置和动作的注释。
采用“批评-修正-验证”策略，导演和编剧进行反复讨论，确保剧本的一致性和吸引力。
演员根据角色档案提供反馈，确保对话与角色形象相符。
摄影：
摄影师依据剧本中的每一行对镜头进行选择，采用“辩论-裁决”策略，通过讨论和对比不同镜头选择的优缺点，最终确定最佳镜头设置。
通过以上方法，FILMAGENT实现了一个高效的电影制作流程，确保了剧本和镜头设置的高质量。
在实验部分，研究团队手动构思了15个故事创意，并在构建的虚拟三维空间中实现这些创意。评估生成视频的关键方面包括剧本的主题一致性、镜头设置的适当性、剧本与角色档案的一致性、演员动作的准确性以及整体情节的连贯性。实验结果显示，FILMAGENT在这些方面的表现均优于基线模型，尤其是在情节连贯性和镜头选择的准确性上。此外，尽管使用了相对较少先进的GPT-4o模型，FILMAGENT在多代理协作的背景下仍然超越了单一代理模型的表现，证明了多代理系统的优势。
FILMAGENT的工作方式可以简单理解为一个团队合作的过程。首先，导演会根据一个简单的故事点子，设计出角色的背景和故事的框架。接下来，编剧会写出角色之间的对话，并根据导演的反馈不断修改剧本，确保每个角色的对话都符合他们的性格。此外，演员们也会参与进来，提出他们对角色的理解，以便让对话更加自然。最后，摄影师会选择最合适的镜头来拍摄每个场景，确保观众能更好地理解故事。整个过程就像一个团队在讨论和完善一个创意，确保最终的电影既有趣又连贯。
2025-01-31｜PKU,CMU｜ICLR2025｜⭐️⭐️
http ://arxiv.org/abs/2501.18982v1https ://wgsxm.github.io/projects/omniphysgs/
在3D资产重建和生成领域，物理交互的动态合成已取得显著进展。然而，现有方法通常假设所有材料属于特定的预定义类别，这种假设忽略了现实场景中多种异质物体的复杂组成，导致生成的动画缺乏物理合理性。为了解决这一问题，本文提出了OMNIPHYSGS框架，通过将每个3D资产视为一组构成3D高斯体，来合成更一般化的物理基础3D动态场景。OMNIPHYSGS的关键设计在于为每个高斯体引入12种物理领域专家子模型的集合，以表示其物理材料，这显著增强了模型的灵活性。通过用户指定的提示定义场景，并利用预训练的视频扩散模型监督材料加权因子的估计。实验表明，OMNIPHYSGS在多种材料的物理动态合成方面表现出更高的通用性和真实感，相较于现有方法在视觉质量和文本对齐度指标上提高了约3%至16%。
OMNIPHYSGS的核心方法包括以下几个部分：
3D高斯表示：采用3D高斯粒子表示场景，允许为每个粒子分配物理属性。这种表示形式兼具简单性和表现力。
构成高斯模型：将普通高斯扩展为构成高斯，通过引入可学习的构成模型，增强物理动态的表达能力。
物质点法（MPM）：利用物质点法模拟3D动态，结合构成高斯进行物理约束的优化。
视频扩散模型指导：通过预训练的视频扩散模型进行训练，使用得分蒸馏采样（SDS）优化生成的视频序列，使其符合用户提供的文本提示。
物理导向的网络：构建一个物理导向的网络结构，集成专家设计的构成模型，以指导学习过程，提高模型的稳定性和收敛性。
在实验部分，OMNIPHYSGS的方法经过了广泛的验证，包括与多种现有方法的比较。首先，使用多个静态3D场景生成动态视频，并通过不同材料的物体进行验证。实验结果显示，OMNIPHYSGS在生成多种材料动态行为方面的表现优于现有的最先进方法，尤其在塑性、粘弹性和流体材料的建模上。其次，通过定量和定性评估，使用CLIP相似度和SSIM指标对生成视频的质量进行了评估，结果表明OMNIPHYSGS在文本对齐和视觉质量方面均有显著提升。此外，研究还进行了消融实验，以验证构成高斯和训练策略的有效性，结果显示引入物理先验和分批训练策略显著提高了模型性能。
在我们的研究中，我们提出了一种新方法，称为OMNIPHYSGS，旨在生成真实的3D动态场景。首先，我们使用一种叫做3D高斯的技术来表示场景中的物体，这种方法可以将每个物体的物理属性（如弹性、重量等）与其表示关联起来。接着，我们为每个高斯体引入了一种可以学习的模型，能根据不同的物理条件自动调整物体的行为，比如如何在重力作用下下落。我们还借助一个强大的预训练视频模型来指导我们的训练过程，这样可以确保生成的视频不仅看起来真实，还能与给定的文本描述相符。通过这些步骤，我们能够生成多种材料的动态场景，例如一个橡胶熊在水中跳跃，或是一个木头树在风中摇摆。这种方法不仅提高了生成动态场景的质量，还能让我们在没有手动调整的情况下，自动生成各种材料的交互效果。
2025-01-30｜XMU,BaiduInc｜⭐️🟡
http ://arxiv.org/abs/2501.18672v1https ://quyans.github.io/Drag-Your-Gaussian
随着生成模型的迅速发展，3D场景编辑的技术也在不断进步。现有的方法通常依赖于生成模型来进行基于文本的3D编辑，然而这些方法在处理几何变更时常常表现不佳，尤其是在需要精确控制编辑结果的情况下。为了解决这些问题，本文提出了一种新的3D拖拽编辑方法——DragYourGaussian(DYG)，该方法基于3D高斯点云表示，允许用户通过3D掩码和控制点对编辑区域进行精确指定，从而实现灵活且高质量的3D场景编辑。实验表明，DYG在编辑效果和质量上超越了现有的基线方法，展现了其在多视角一致性和细粒度编辑方面的优势。
DYG方法的核心在于几个创新的模块，具体包括：
多分辨率三平面位置编码器（MTPEncoder）：该模块将3D空间分解为多个可学习的特征平面，帮助解决高斯原语的稀疏分布问题，从而实现平滑的几何编辑。
区域特定位置解码器（RSPDecoder）：该解码器用于预测被编辑区域的位移，确保编辑效果的局部一致性。
两阶段拖拽策略：第一阶段侧重于优化几何结构，建立编辑场景的基础框架；第二阶段则专注于细化纹理细节，以确保编辑结果的整体质量。
软局部编辑策略（SLE）：该策略通过识别与掩码区域相邻的高斯原语，进行局部优化，从而在编辑时保持其他区域的完整性。
通过这些模块的结合，DYG能够有效地进行3D场景的拖拽编辑，实现高质量的几何修改。
为了验证DYG的有效性，研究团队进行了多项实验，比较了DYG与其他基线方法（如GS-Editor和SC-GS）的编辑效果。实验结果显示，DYG在多个编辑任务中表现出色，尤其是在处理复杂的几何编辑时，能够保持高质量的视觉效果和一致性。此外，用户研究也表明，DYG在场景质量和编辑效果方面获得了用户的高度认可，86.1%的用户更倾向于选择DYG的编辑结果。通过这些实验，DYG证明了其在3D场景编辑中的广泛适用性和优越性。
DYG方法的工作原理可以简单理解为一个高科技的“数字雕刻工具”。想象一下，你在一个3D模型上使用鼠标来选择你想要编辑的区域，DYG允许你通过简单的拖动来改变这个区域的形状或位置。首先，用户通过3D掩码来标识想要编辑的部分，然后通过设置控制点来引导编辑的方向。接下来，DYG会使用智能算法来确保所做的修改看起来自然且和谐，不会影响到其他部分。这个过程就像是在雕刻一个粘土模型，先建立基础形状，然后细致打磨，确保每个细节都完美无瑕。通过这种方式，DYG使得3D编辑变得更加直观和灵活，适合各种复杂的编辑需求。
2025-01-27｜MIT,ColumbiaU,ETHZürich,GoogleResearch｜⭐️⭐️
http ://arxiv.org/abs/2501.18623v1
在计算机图形学领域，程序性材料（proceduralmaterials）作为功能节点图（functionalnodegraphs）被广泛应用于真实感材质外观设计。尽管这种材料允许用户进行直观且精准的编辑，但基于输入图像创建程序性材料仍然需要专业知识和大量的努力。本文提出了一种新方法，利用大型视觉-语言模型（VLM）生成程序性材料的标准Python程序。通过对预训练VLM的微调，结合开源的程序性材料数据集，研究者们能够从输入图像生成相应的程序性材料。此外，研究还引入了程序级数据增强，以提升生成材料的质量。经过广泛的评估，研究结果表明，该方法在合成和真实世界示例上均优于现有技术。
本研究的核心方法包括以下几个步骤：
数据收集：首先，收集超过550,000个Blender程序性材料的图像和程序对，确保数据集的多样性和代表性。
微调VLM：在收集的材料数据集上微调预训练的大型视觉-语言模型，使其能够从输入图像预测程序性材料的Python代码。
数据增强：采用基于程序结构和参数的增强策略，利用预训练的语言模型生成新的程序结构和参数变体，显著扩展数据集规模。
后优化：在生成初步的程序性材料后，利用无梯度的随机局部参数搜索算法（基于马尔可夫链蒙特卡洛方法）进一步优化材料的节点参数，以提高与输入图像的感知匹配度。
在实验部分，研究者们对所提出的方法进行了全面的评估，包括以下几个方面：
数据集构建：使用来自Blender和Substance的不同来源的图像进行测试，确保测试的全面性和多样性。
基准对比：将所提方法与其他四种基准方法进行比较，包括GPT-4o-mini和ConditionalMatFormer等，评估生成材料的质量。
评估指标：采用风格损失、切片Wasserstein距离和CLIP相似性等多种指标来量化生成材料与输入图像之间的感知相似性。
消融研究：通过消融实验验证数据增强和后优化对生成质量的影响，结果表明这两个步骤对提升生成材料的质量至关重要。
在这项研究中，研究者们希望通过一个智能程序来帮助设计师从图片中生成可以编辑的材料。首先，他们收集了大量的样本材料和对应的程序代码，这些代码可以在Blender软件中使用。接着，他们对一个已经训练好的智能模型进行了“微调”，让它学会如何根据输入的图片生成这些程序代码。为了让模型更聪明，他们还利用了一些技术来创造新的程序变体，增加了训练数据的多样性。最后，在生成初步材料后，他们使用一种特殊的算法来进一步优化材料的外观，使其更接近输入的图片。这种方法不仅提高了生成材料的质量，还使得设计师可以在3D软件中轻松编辑这些材料。
2025-01-31｜UToronto,USTC,MSRA｜⭐️🟡
http ://arxiv.org/abs/2501.19054v1
本研究提出了一种名为CADFusion的框架，旨在通过结合视觉反馈和大语言模型（LLMs），实现文本到计算机辅助设计（CAD）模型的自动生成。CAD模型的构建通常需要复杂的参数序列和多次迭代，传统方法往往需要专业的知识和经验。CADFusion通过引入两阶段的训练策略，即顺序学习（SL）和视觉反馈（VF），有效地简化了这一过程。顺序学习阶段使用真实的参数序列进行训练，以生成逻辑连贯的CAD参数序列；而视觉反馈阶段则通过评估生成的视觉对象的受欢迎程度来优化参数序列，从而提高生成模型的整体性能。实验结果表明，CADFusion在生成CAD模型的质量和效率方面显著优于现有方法。
CADFusion的训练过程分为两个主要阶段：顺序学习（SL）和视觉反馈（VF）。
顺序学习（SL）阶段
在此阶段，使用真实的CAD参数序列对LLMs进行微调，目的是学习参数序列的结构和操作。
采用文本描述作为输入，生成对应的CAD参数序列，确保生成的序列符合逻辑。
通过交叉熵损失函数来评估生成序列的准确性。
视觉反馈（VF）阶段
该阶段的目标是通过评估生成的视觉对象与用户期望之间的匹配程度来优化参数序列。
采用直接偏好优化（DPO）方法，通过对比生成的视觉对象与真实对象，给予生成序列奖励或惩罚，从而引导模型生成更符合视觉偏好的参数序列。
这一阶段确保模型在生成过程中考虑视觉效果，提高生成对象的质量。
这两个阶段交替进行，以确保模型在学习参数序列和视觉反馈之间保持平衡，最终实现高质量的CAD模型生成。
在实验部分，我们使用了DeepCAD数据集进行顺序学习阶段的训练，构建了包含20,000对文本描述和CAD参数序列的数据集。同时，在视觉反馈阶段，我们利用大型视觉语言模型（LVMs）生成偏好数据，以评估视觉对象的质量。实验结果显示，CADFusion在多个评估指标上超越了基线模型，如F1分数、Chamfer距离和无效率等。我们还进行了消融实验，以验证视觉反馈阶段和交替训练策略的有效性。结果表明，CADFusion在理解用户指令和生成高质量CAD对象方面表现出色，特别是在复杂几何体的生成能力上。
CADFusion的工作原理可以简单理解为一个两步走的过程。首先，在顺序学习阶段，系统会根据用户的文字描述，学习如何生成CAD模型的基础结构。这就像是教一个学生如何根据老师的讲解画出一幅图。接着，在视觉反馈阶段，系统会将生成的图像与真实的CAD模型进行比较，看看生成的图形是否符合预期。如果生成的图形不够好，系统会进行调整，尝试生成更符合要求的模型。这两个步骤反复进行，就像在练习和调整，最终使得生成的CAD模型既符合逻辑又能很好地呈现出用户想要的效果。通过这种方法，即使没有专业知识的人也能轻松生成高质量的CAD设计。

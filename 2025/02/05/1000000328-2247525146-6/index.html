2025-01-28｜UniversitàdellaSvizzeraitaliana｜⭐️🟡
http ://arxiv.org/abs/2501.18635v1
本研究旨在探讨在使用焦点渲染技术时，模糊对立体深度感知的影响。焦点渲染是一种通过将计算资源集中在视网膜中心区域（即“焦点”）的技术，来提高虚拟现实和增强现实中的图像质量，同时降低周边区域的渲染质量。尽管已有研究表明，周边视力的空间分辨率较低，但对于模糊对深度感知的影响仍未得到充分研究。为此，研究者设计了一项心理物理实验，以量化不同模糊强度和偏心度下的立体深度感知阈值，并提出了一个简单的感知模型，旨在描述不影响立体敏锐度的焦点渲染模糊程度。
本研究的主要方法包括以下几个步骤：
刺激设计：研究者设计了一个带有波动深度的纹理环，使用随机点模式来量化模糊对深度感知的影响。刺激的深度振幅和差异被调整，以便在2-选择强迫选择实验中确定可解析的深度阈值。
实验参数：在三个不同的偏心度（0°、10°和20°）下进行测量，并使用高斯滤波器对刺激进行模糊处理，模糊强度从0’到26.6’不等。每个条件下，参与者需判断刺激的深度信息。
数据收集与分析：采用“最佳PEST”程序来估算每种条件下的深度阈值，并使用Weibull累积分布函数拟合数据，提取出75%的阈值，以表示参与者的正确识别能力。
实验分为两个主要部分：首先是基础条件下的阈值估计，其次是复杂场景下的验证实验。在基础实验中，11名参与者在不同模糊强度和偏心度下完成了深度感知任务。参与者需在给定时间内判断刺激的深度特征，并记录其反应。随后在复杂自然场景中进行验证，参与者在森林和厨房场景中评估焦点和非焦点渲染的深度感知。通过比较两种渲染方式下的深度阈值，研究者验证了焦点渲染对深度感知的影响。
在这项研究中，科学家们想知道当我们看东西时，周围的模糊会如何影响我们判断物体的深度。为此，他们设计了一种特殊的实验，使用一个带有波动深度的环形图案，这个图案上有许多小点。参与者在实验中需要判断这个图案的不同部分的深度感知。在实验中，科学家们通过改变图案的模糊程度和观察的角度，来测试参与者的反应。最终，他们发现，即使在周围模糊的情况下，人们的深度感知能力依然保持稳定，这意味着焦点渲染技术不会影响我们对深度的判断。
2025-01-30｜ToyotaResearchInstitute,TTI-Chicago｜⭐️🟡
http ://arxiv.org/abs/2501.18804v1https ://mvgd.github.io
本研究提出了一种新的多视角几何扩散框架（MVGD），旨在从任意数量的已知视角图像生成新视角的图像和深度图。现有的三维重建方法通常依赖于中间的三维表示（如神经场或体素网格），而MVGD则直接进行像素级生成，克服了传统方法在处理新场景时的局限性。通过利用多视角信息和空间条件，MVGD能够在生成图像和深度图时保持一致性。该方法在超过6000万的多视角样本上进行训练，展示了其在新视角合成、立体和视频深度估计任务中的优越性能。
MVGD方法的核心在于其多任务生成架构，主要包括以下几个方面：4.扩散模型：通过学习从噪声到样本的状态转移函数，MVGD能够有效地生成新视角的图像和深度图。这一过程不需要中间的三维表示。5.条件嵌入：使用输入视角的相机几何信息（如射线映射）作为条件，增强了生成的图像和深度图的空间一致性。6.多任务学习：MVGD同时生成图像和深度图，通过可学习的任务嵌入引导扩散过程，使得模型能够根据特定任务生成相应的输出。7.场景尺度归一化：提出了一种新的归一化策略，自动提取场景的尺度信息，以确保生成的深度图与输入视角一致。8.增量微调策略：通过对小模型进行增量微调，MVGD能够高效地扩展到更大的模型，提升生成效果。
在多个基准测试中，MVGD展示了其在新视角合成和深度估计任务中的领先性能。实验结果表明，在使用2至9个输入视角的情况下，MVGD均优于现有的最先进方法。通过分析不同条件视角数量对生成质量的影响，MVGD在多视角条件下表现出色，尤其是在处理复杂场景时。此外，MVGD在立体和视频深度估计任务中也取得了显著的效果，表明其在多视角学习中的有效性和灵活性。
MVGD的工作原理可以简单理解为：它就像一位艺术家，能够根据几幅不同的画作（即输入图像）来创作出新的画作（新视角图像）和深度信息。这个艺术家使用了一种特殊的技巧，叫做“扩散”，这意味着他会从一幅模糊的草图开始，逐步细化，直到完成作品。为了使新作品看起来更真实，艺术家还会参考输入画作的特征和相机的位置，确保新画作中的深度感和细节都与原作一致。此外，MVGD还会不断学习和改进，能够在不同的场景中表现出色，即使面对全新的画面也能创作出高质量的作品。
2025-01-27｜USCICT｜⭐️🟡
http ://arxiv.org/abs/2501.18630v1
本文提出了一种新颖的三维场景重建方法——可变形BetaSplatting（DeformableBetaSplatting,DBS），旨在提升3D场景的几何和色彩表现。传统的3D高斯Splatting（3DGS）虽然在实时渲染方面表现出色，但由于其依赖于高斯核和低阶球谐函数，导致在捕捉复杂几何形状和多样化色彩时存在局限性。DBS通过引入可变形的Beta核，提供了更高的几何细节捕捉能力和更好的内存效率。此外，DBS还扩展了Beta核到颜色编码中，实现了漫反射与镜面反射成分的有效分离，从而在视觉质量上超越了基于球谐函数的方法。实验结果表明，DBS在实时渲染中仅使用45%的参数，渲染速度比3DGS快1.5倍，首次使得基于Splatting的方法在视觉效果上超越了最新的神经辐射场（NeRF）技术。
DBS的核心在于三个主要组成部分：9.Beta核用于几何表示：该核基于Beta分布，具有适应性形状控制和有界支持，能够准确捕捉平面表面、锐利边缘和光滑区域的几何细节。通过引入一个控制形状的参数b，Beta核可以从类似于高斯的形状逐渐调整为更适合复杂几何的形状。10.球面Beta用于颜色编码：该方法通过解耦漫反射和镜面反射成分，利用较少的参数（仅31%）来高效捕捉高频照明效应，从而提高了模型的效率和视觉效果。11.无核Markov链蒙特卡罗优化：通过将每个原始体的透明度视为概率，DBS优化过程采用了无核的MCMC策略，避免了与高斯核相关的复杂启发式策略。这种方法在优化过程中引入位置噪声以鼓励探索，防止过拟合。
在多种真实和合成数据集上评估DBS的性能，结果表明其在视觉质量和渲染效率上均优于现有的先进方法。通过在Mip-NeRF360、TanksandTemples等数据集上进行标准化评估，DBS在峰值信噪比（PSNR）、结构相似性指数（SSIM）和学习感知图像补丁相似性（LPIPS）等指标上均表现出色。此外，DBS的渲染速度比3DGS-MCMC快50%，并且在相同的原始体数量下，存储内存使用量减少至3DGS-MCMC的一半。通过消融实验，进一步验证了不同配置对性能的影响，结果显示DBS在捕捉细节和保持高保真度方面的优势。
在这项研究中，我们提出了一种新的技术来处理三维场景的渲染，叫做可变形BetaSplatting。想象一下，传统的3D渲染就像用模具制造一个模型，模具的形状限制了你能做的细节。而我们的技术就像是使用一种能够随意变形的模具，可以更好地捕捉到物体的细微特征，比如平坦的表面和锐利的边缘。我们还引入了一种新的方式来处理颜色，使得光线的反射和漫反射可以更加真实地表现出来。通过这种方法，我们不仅能让渲染看起来更好，而且速度也更快，使用的计算资源更少。这意味着在虚拟现实或游戏中，用户可以享受到更流畅、更真实的体验。
2025-01-31｜JagiellonianU,FMCS｜⭐️
http ://arxiv.org/abs/2501.19196v1https ://github.com/KByrski/RaySplatting
本文介绍了一种名为RaySplats的新型3D高斯点云渲染技术，该方法通过结合光线追踪技术来克服传统3D高斯点云（3DGS）在光照和阴影处理上的局限性。3DGS能够有效地将2D图像转化为高质量的3D场景，但其渲染过程主要依赖光栅化技术，难以准确模拟光线反射和阴影效果。RaySplats通过直接在高斯原语上应用光线追踪，利用信心椭圆表示高斯分布，从而实现更真实的光照效果。该方法在训练和推理阶段均采用光线追踪机制，显著提升了渲染的视觉效果和效率。
RaySplats方法的核心在于高斯分布与光线的交互计算。具体步骤如下：12.高斯分布表示：每个高斯分布被表示为信心椭圆，包含位置、协方差、颜色和不透明度等参数。13.光线与高斯交集计算：通过将光线的参数化方程代入高斯椭圆的方程，计算光线与高斯分布的交点，确定哪些高斯分布与光线相交。14.颜色聚合：沿光线聚合颜色时，考虑所有相交的高斯分布的颜色和不透明度，使用加权平均的方法计算最终颜色。15.光照效果处理：集成阴影、反射和透明度等光照效果，通过光线追踪技术实现真实感的渲染。16.反向传播：在反向传播阶段，计算损失函数的梯度，以优化高斯分布参数。
这种方法不仅提高了光照效果的真实性，还简化了与网格模型的集成，使得RaySplats能够在多种应用场景中表现出色。
在实验部分，研究团队对RaySplats进行了定量和定性的评估。使用了多个广泛应用的数据集，包括Mip-NeRF360、TanksandTemples和DeepBlending。实验结果显示，RaySplats在视觉保真度和渲染效率上均优于传统的3DGS方法。定量评估使用了PSNR、SSIM和LPIPS等指标，结果表明RaySplats在大多数场景中能够达到与经典3D高斯点云相当的效果。此外，研究还进行了消融实验，探讨了高斯数量、光线终止阈值等关键参数对渲染质量的影响，进一步验证了RaySplats的有效性。
RaySplats的方法可以简单理解为一种结合了光线追踪和高斯分布的渲染技术。首先，想象每个3D物体都是由许多小的高斯“点”构成的，这些点有自己的颜色和透明度。RaySplats通过光线追踪，像激光一样发出光线，检查这些光线是否与高斯点相交。如果光线碰到高斯点，就会根据这些点的颜色和透明度来决定最终的颜色。这样，RaySplats不仅能显示出物体的基本形状，还能模拟光的反射、阴影和透明度，使得3D场景看起来更加真实和生动。通过这种方式，RaySplats能够处理复杂的光照效果，而不需要依赖传统的光栅化技术，从而提升了渲染的质量和速度。
2025-01-27｜EPFL,NVIDIA
http ://arxiv.org/abs/2501.18627v1
本文提出了一种新颖的辐射场损失（RadianceFieldLoss）方法，以简化和加速发射表面的重建过程。该方法基于现有的发射体积重建算法，进行了细微但重要的修改，旨在提高重建的速度和鲁棒性。与传统的基于体积的方法不同，本文的方法通过在损失计算中直接监督训练图像，优化表面分布，从而在处理复杂的真实场景时表现出更好的稳定性。该方法的核心在于通过优化表面分布而不是集成光线的辐射场，显著提升了收敛速度和重建质量。
本文的方法主要包括以下几个步骤：17.非局部表面扰动：通过引入非局部表面扰动，考虑在可见表面上方添加小表面补丁的影响，以优化几何形状。这种方法使得算法能够在更高维度的损失空间中实现更快和更稳健的收敛。18.辐射场损失计算：针对单个候选表面，定义损失函数，使得候选表面能够根据目标颜色的匹配程度进行优化。通过对比候选表面的颜色和背景颜色，动态调整表面的存在性。19.多候选表面评估：扩展损失函数以处理多个候选表面，允许同时评估多个扰动的效果，从而加速收敛过程。通过将每个候选表面视为独立子问题，优化每个子问题的损失。20.空间方向损失：在多个光线方向上评估损失，从而扩展独立考虑的候选表面集合，形成综合损失函数，最终实现辐射场损失的计算。
实验部分展示了所提方法在多个场景下的性能，包括新视角合成和几何重建。通过与现有的NeRF模型进行比较，结果表明，本文的方法在视觉质量上几乎没有损失，同时在计算效率上实现了显著提升。具体来说，在相同的训练参数下，所提方法的平均PSNR仅比基线低0.1dB，但在处理复杂场景时展现出更高的鲁棒性。此外，实验还展示了在不同占用水平下的表面渲染效果，验证了所提方法在真实世界应用中的有效性和灵活性。
在我们的研究中，我们提出了一种新的方法来快速重建物体表面。简单来说，我们的方法就像一个智慧的“摄影师”，它不仅仅是拍摄照片，而是通过不断调整拍摄角度和光线，来优化和改善我们看到的物体表面。我们首先考虑在物体表面上添加一些小的补丁，这样可以帮助我们更好地理解物体的形状和颜色。接着，我们通过比较这些补丁的颜色与背景的颜色，来决定哪些补丁是有用的，哪些可以忽略。最后，我们会同时考虑多个补丁的效果，这样可以加快我们找到最佳形状的速度。通过这种方式，我们的方法不仅能快速生成高质量的物体表面，还能在复杂的现实场景中保持稳定性。
2025-01-31｜BUAA
http ://arxiv.org/abs/2501.19088v1
在日常交互中，手是最主要的接口，因此高质量数字人手的建模与真实图像的渲染是一个重要的研究课题。为了满足交互和渲染应用的要求，必须实现实时渲染和数字模型的驱动能力，而不降低渲染质量。为此，研究者们提出了JGHand（Joint-DrivenAnimatableHandAvatar），这是一种新颖的基于3D高斯点云（3DGS）的手部表示方法，可以在各种姿势和角色下实时渲染高保真手部图像。与现有的关节神经渲染技术不同，该方法引入了一种可微分的空间变换过程，基于3D关键点支持从标准模板到任意骨长和姿势的变形。此外，还提出了一种基于每像素深度的实时阴影模拟方法，以模拟手指运动引起的自遮挡阴影。通过全面的消融研究验证了该方法的有效性，实验结果表明JGHand在公共数据集上的实时渲染速度和图像质量均超越了现有的最先进方法。
JGHand的方法分为几个关键步骤：21.骨架变换计算：基于手的姿势，计算骨架变换矩阵B，以实现从标准姿势到目标姿势的零误差转换。22.3D高斯点云渲染：利用3D高斯点云驱动输入姿势的图像渲染，首先将高斯从标准空间转换到姿势空间，然后使用线性混合蒙皮（LBS）来处理变形。23.3D高斯属性估计：通过身份特征和学习特征动态调整高斯的属性，以确保准确的渲染。24.自遮挡阴影模拟：在渲染过程中计算深度图像，并通过可微分的阴影计算层来模拟手指运动引起的阴影，提高渲染图像的真实感。
通过这些步骤，JGHand能够实时生成高质量的手部图像，并在各种姿势下保持一致的渲染效果。
在实验中，JGHand与现有的最先进方法在两个数据集上进行了比较。使用InterHand2.6M数据集，该数据集包含多视角手部序列，验证了方法在真实捕捉场景中的表现。此外，使用HandCo数据集进行合成数据的验证。实验结果表明，JGHand在渲染质量和速度上均优于其他方法，尤其在细节恢复和渲染时间上表现突出。通过消融研究，验证了各个组件对最终效果的影响，结果显示阴影模拟和骨架变换的准确性对渲染质量至关重要。
在JGHand的方法中，研究者们首先计算手部的骨架变换，以便将手的标准姿势转换为目标姿势，这个过程可以确保转换过程中没有任何误差。接下来，他们利用3D高斯点云来帮助渲染手部图像，这就像在三维空间中使用小球来表示手的不同部分，并通过一些算法让这些小球在不同姿势下移动和变形。为了让渲染的手看起来更真实，他们还计算了手部的深度信息，并模拟了手指运动时产生的阴影。最终，这个方法可以在各种手势下快速生成高质量的手部图像，让人感到非常真实。

本研究提出了一种新颖的方法，利用CLIP特征蒸馏实现高效的语言引导3D分割。与以往依赖多尺度CLIP特征的技术不同，我们的方法通过直接蒸馏密集的CLIP特征，简化了工作流程，从而提高了3D场景的分割精度。为了解决在密集CLIP特征蒸馏过程中出现的噪音问题，我们引入了一种适配器模块，并采用自交叉训练策略。此外，为了提升分割边缘的准确性，我们设计了一种低秩瞬态查询注意机制。通过将分割任务转化为分类任务，我们确保在不同视角下相似颜色的分割结果的一致性。我们的实验结果表明，该方法在训练速度和性能上均超越了现有的先进技术。

在三维视觉应用中，点云完成是一项核心任务，旨在从部分观测中重建完整的三维形状。这一过程常常受到遮挡、传感器限制和噪声的影响，导致关键信息的丢失。传统的自编码器架构虽然可以处理这一问题，但其潜在的知识传递和重建过程仍未得到充分探索。为此，本文提出了一种新的方法——视图蒸馏点云完成网络（VD-PCN），通过多视图的蒸馏学习策略来完成三维形状的重建。该方法充分利用了二维像素的有序性、灵活性以及强大的二维网络能力，经过在PCN、ShapeNet55和MVP数据集上的广泛评估，证明了其在定量和定性上的有效性。
VD-PCN的设计分为几个关键模块。首先，模型接收不完整的点云数据，并通过一个多视图编码器处理来自多个视角的深度图像。具体步骤如下：30.多视图编码器：将来自k个视角的深度图像输入共享的二维网络，提取多视图特征。31.知识蒸馏：采用教师-学生结构进行知识转移。教师网络通过部分点云和完成的深度图进行训练，建立从深度图到完整点云的映射。学生网络在初始时使用教师的权重，并在较小的学习率下进行训练。32.双模态解码器：生成初步的点云种子，并通过上采样过程逐步生成高分辨率的完整点云。在这个过程中，结合了二维和三维的信息，以最大限度地减少信息损失。
在实验部分，VD-PCN在多个数据集上进行了评估，包括PCN、ShapeNet-55和MVP。实验设置包括不同的缺失点比例，以验证模型在不同难度下的表现。结果显示，VD-PCN在Chamfer距离（CD）上的表现优于现有的最先进方法，尤其是在处理复杂形状时，模型能够更好地恢复细节并减少噪声。此外，模型的训练过程使用AdamW优化器，并通过精细的损失函数设计来提高重建精度。实验结果表明，VD-PCN在各类数据集上均展现出优异的性能。
在VD-PCN的工作流程中，我们首先从多个角度拍摄物体的深度图像，就像用不同的相机拍照一样。然后，我们将这些图像输入到一个共享的二维网络中，这个网络就像一个聪明的助手，能够从这些不同的视角提取有用的信息。接下来，我们有一个“老师”和一个“学生”网络。老师网络已经学习了如何从部分信息中恢复完整的形状，而学生网络则在老师的指导下学习。最后，学生网络将生成的初步形状进行精细化处理，结合来自二维和三维的信息，逐步提升到高分辨率的完整点云。这种方法不仅提高了点云完成的准确性，还减少了在恢复过程中可能出现的细节丢失。

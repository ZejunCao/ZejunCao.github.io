主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）发布日期｜2025年2月3日｜周一
2025-01-31｜UWaterloo,VectorInstitute｜⭐️⭐️
http ://arxiv.org/abs/2501.19339v1https ://tiger-ai-lab.github.io/PixelWorld/
本研究提出了一种新颖的统一感知框架，名为“像素世界”（PixelWorld），旨在将文本、图像、表格和代码等多种模态都视为像素输入，以此来提升现有大型语言模型（LLMs）的性能。传统的模型通常将文本处理为标记，而视觉输入则被视为图像，这与人类的感知方式存在显著差异。随着体现在机器人等智能体中的应用需求日益增加，建立一个统一的感知框架显得尤为重要。通过“将一切视为像素”（PEAP）的方法，我们开发了PIXELWORLD评估套件，能够综合评估模型在多模态任务中的表现。研究结果表明，PEAP在多模态数据集上的表现优于基于标记的输入，尤其在结构复杂和多模态任务中，但在推理和编码能力上仍有待提升。
本研究的核心方法是PEAP，具体步骤如下：
数据集构建：我们将任务分为三类：仅文本、结构化和多模态。对于前两类，我们开发了图像合成管道，将文本输入转换为像素表示；对于多模态数据集，我们利用光学字符识别（OCR）提取图像中的文本。
像素输入处理：PEAP方法通过将所有输入视为像素，减少了对领域和模态敏感的预处理步骤，同时保留了重要的格式和布局信息，提升了模型的理解和推理能力。
效率优化：为了提高推理速度，我们引入了PEAP-Fast算法，通过去除空白像素区域来减少计算开销，确保在不损失精度的情况下加快处理速度。
我们在PIXELWORLD评估套件上对多种视觉语言模型（VLMs）进行了全面评估，实验结果按“文本输入”、“结构化输入”和“多模态输入”进行组织。我们建立了基线模型，并在相同的输入条件下进行比较，以确保公平性。实验中使用的指标包括ROUGE-L、准确率等，以评估模型在不同任务上的表现。结果显示，较大的模型在多模态任务中表现更佳，而较小的模型则在指令遵循方面存在显著不足。此外，复杂的推理任务在使用PEAP时表现出较大的性能下降，提示在未来的研究中需要加强模型的推理能力。
在我们的研究中，我们提出了一种叫做PEAP的方法，简单来说，就是将所有类型的信息（比如文字、图片和表格）都看作是“像素”。这样做的好处是，可以更好地模拟人类的感知方式，减少了处理这些信息时的复杂步骤。具体来说，我们首先将文本转变为图像，然后通过OCR技术提取图片中的文字，这样就可以在同一个框架下处理所有信息。此外，我们还设计了一种加速算法，能够去掉那些没有实际信息的空白区域，从而提高处理速度。通过这种方法，我们希望能够让机器更好地理解和处理来自不同模态的信息。
2025-01-31｜IowaStateU,NYU
http ://arxiv.org/abs/2501.18880v1
本文提出了一种新颖的框架RLS3，旨在通过强化学习（RL）增强视觉语言模型（VLM）在空间推理任务中的表现。随着深度学习的进步，VLM在自主感知中的应用越来越广泛，尤其是在需要理解视觉和文本信息的场景中。然而，VLM的训练通常依赖于高质量的数据集，而现有的数据集往往不足或不平衡，限制了模型的泛化能力。为了解决这一问题，RLS3框架通过RL代理生成合成数据，帮助VLM更有效地进行特定任务的微调，特别是针对空间推理的挑战。通过这种方法，研究者们能够提高模型对复杂空间关系的理解能力，从而实现更准确的场景识别和理解。
RLS3框架的核心在于其数据生成和微调过程，具体包括以下几个步骤：4.Unity环境构建：利用Unity游戏引擎创建多个室内场景，确保生成的图像符合可行性要求，并包含空间关系的描述。5.RL代理的数据生成：RL代理在Unity环境中执行动作，移动场景中的对象，以生成有效的图像-文本对。每次移动后，代理根据生成的场景的有效性获得奖励，确保生成的样本具有高质量和相关性。6.提示生成：在生成的图像基础上，使用提示生成器创建描述空间关系的文本。根据物体之间的相对位置，生成不同复杂度的描述，以丰富模型的训练数据。7.VLM微调：将生成的样本输入到VLM中进行微调，通过RL代理获取反馈，进一步优化数据生成过程，确保模型能够处理更复杂的空间推理任务。
实验部分主要评估RLS3框架在空间推理任务中的有效性。研究者们通过对比随机生成的数据和RL生成的数据，分析了不同生成策略对模型性能的影响。实验设置包括多个场景的动态数据生成，以确保模型在每次训练中都能接触到不同的环境和任务。此外，使用固定的验证集来评估模型的性能变化，确保结果的可重复性和可靠性。实验结果表明，RLS3框架显著提升了VLM在空间推理任务中的表现，尤其是在处理复杂的空间关系时，展现了RL代理在合成数据生成和模型微调中的潜力。
RLS3框架的工作原理可以简单理解为一个智能机器人在一个虚拟房间里学习如何识别和描述物体的位置。首先，研究者们利用一个游戏引擎（Unity）创建了一个模拟环境，里面有不同的房间和物品。然后，一个智能代理（就像机器人）在这个环境中移动物品，并观察这些物品之间的关系。每当它成功地放置一个物体而不让它们重叠或超出桌子边缘时，它就会得到奖励。接着，代理会根据这些物品的位置生成描述，比如“这个碗在那个盘子的右边”。这些生成的图像和描述会被用来训练一个视觉语言模型，使其更好地理解空间关系。通过这种方法，RLS3可以帮助模型在复杂的场景中做出更准确的判断。
2025-01-31｜HZU,CAS,ZJU,SUTD,USTC,Moxin(Huzhou)Tech.Co.,LTD.｜🟡
http ://arxiv.org/abs/2501.19329v1
本研究首次利用人类的自由手绘草图来提升在复杂图像分割任务中的表现，特别是在伪装物体检测（COD）方面。传统的分割方法依赖于边界框或点标注，而我们的创新框架允许用户通过绘制物体的粗略轮廓来进行直观标注。研究表明，草图输入显著提升了现有迭代分割方法的性能，超越了文本或边界框注释。同时，我们引入了网络结构的关键修改和一种新颖的草图增强技术，以充分利用草图输入的优势，从而进一步提高分割精度。此外，我们创建了KOSCamo+数据集，这是首个用于伪装物体检测的自由手绘草图数据集，所有数据和标注工具将开源，旨在推动这一领域的研究。
我们的研究方法主要包括以下几个方面：8.自由手绘草图绘制：我们进行了初步研究，评估不同注释方法（如点、框、草图）在COD任务中的有效性。结果显示，使用草图的效率更高，标注时间显著减少。9.网络架构设计：采用了基础的SAM模型，通过引入草图编码器来替代传统的输入提示编码器。草图特征与图像特征通过融合机制整合，以增强分割效果。10.注入领域特定信息：在网络中引入适配器以利用高频特征和补丁嵌入，帮助模型更好地理解伪装物体的特征。11.草图增强技术：为了应对草图创作中固有的抽象性和多样性，我们设计了一种草图增强方法，通过参数化贝塞尔曲线生成多样化的草图，以提高网络的鲁棒性。12.边界细化和自适应焦点损失：通过引入边界细化损失，增强网络对边界细节的学习，同时使用自适应焦点损失来改善模型对难以分类像素的学习效果。
在实验部分，我们通过多个基准数据集验证了所提方法的有效性。训练过程持续12小时，经过120,000次迭代，采用多种评估指标（如S-measure、E-measure和加权F-measure）进行性能评估。实验结果显示，与传统的注释方法相比，我们的方法不仅在分割精度上取得了显著提升，而且在注释效率上节省了超过120倍的时间。此外，我们的方法可直接用于训练其他深度学习模型，表现媲美于像素级标注。通过对比实验，我们发现草图输入在提升分割性能方面具有显著的优势，尤其是在处理复杂背景和伪装物体时，展示了我们方法的强大潜力。
在我们的研究中，我们使用了一种简单的方法来帮助计算机更好地识别图像中的物体。想象一下，如果你想让朋友找到你在一张复杂的照片中的某个物体，你可能会用手指指着它或画个圈。而我们的方法就是让用户用手绘的草图来标记物体，而不是用传统的框或点。这个过程不仅直观简单，而且速度快，用户只需在图像上快速勾勒出物体的轮廓。我们还设计了一种智能的计算机程序，它可以理解这些草图，并将其与图像结合，从而更准确地找到目标物体。通过这种方式，我们的研究不仅提高了识别的准确性，还大大减少了需要花费的时间，让人们在标注图像时变得更加轻松和高效。
2025-01-31｜THU,Kuaishou
http ://arxiv.org/abs/2501.18993v1https ://github.com/qyp2000/VARSR
图像超分辨率（ImageSuper-Resolution,ISR）旨在从低分辨率（Low-Resolution,LR）图像生成高分辨率（High-Resolution,HR）图像。近年来，生成对抗网络（GAN）和扩散模型等生成模型的引入为ISR带来了显著进展。然而，这些方法在生成图像的真实感和保真度之间存在权衡，同时计算复杂性也限制了它们的应用。为此，本文提出了一种新颖的视觉自回归建模框架VARSR，旨在通过下一尺度预测来增强图像的细节和真实感。该框架通过引入前缀令牌（PrefixTokens）来有效整合LR图像中的语义信息，并采用尺度对齐旋转位置编码（Scale-alignedRotaryPositionalEncodings）来捕捉空间结构。此外，利用扩散精炼器（DiffusionRefiner）来建模量化残差损失，从而实现像素级的保真度。通过大规模数据集的收集和训练过程的设计，VARSR在生成高保真和高真实感图像方面展现出了优越的效率和效果。
VARSR框架的设计包括多个关键步骤：13.视觉自回归建模：VARSR引入了自回归模型的思想，通过对图像进行离散量化，将图像块映射为一系列令牌（tokens），并通过前一个令牌序列和条件信息预测下一个令牌。14.条件控制生成：使用前缀令牌将LR图像编码为条件令牌，作为生成过程的全局指导。这种方式有效地整合了LR图像的语义信息，避免了直接添加LR特征所带来的空间结构丢失。15.尺度对齐位置编码：采用SA-RoPE，确保不同尺度的令牌在空间上的位置关系被准确捕捉，进而改善图像生成的结构保真度。16.扩散精炼器：通过对量化残差的预测，利用扩散模型来生成连续令牌，从而减轻量化过程中的信息损失，提升生成图像的细节。
在实验部分，VARSR框架在多个图像质量评估指标上表现优异。首先，利用大规模高质量图像数据集进行训练，确保模型的生成能力。通过对比实验，VARSR在PSNR、SSIM等传统指标上与现有的GAN和扩散基准方法进行比较，显示出在生成高保真和高真实感图像方面的优势。此外，VARSR还在多个非参考图像质量指标（如MUSIQ、MANIQA）上取得了显著提升。实验结果表明，VARSR在细节生成和结构保真度方面均优于传统方法，尤其是在处理高质量和低质量图像时，展现出更强的适应性和鲁棒性。
VARSR的工作原理可以简单理解为一个多步骤的图像修复过程。首先，它会将低分辨率的图像转化为一系列小的“令牌”，就像将一幅画分解成许多小块。然后，模型会根据这些小块和原始图像的信息，逐步预测出更高分辨率的图像。为了确保每个小块的位置关系不会被打乱，VARSR使用了一种特殊的编码方式，帮助它记住每个小块在整体图像中的位置。最后，VARSR还引入了一种“精炼器”，可以进一步修复图像中的细节，确保生成的高分辨率图像既真实又美观。这种方法不仅提高了生成图像的质量，还使得整个过程更加高效、快速。
2025-01-31｜HKUST(GZ),AlpsenTek｜ICLR2025
http ://arxiv.org/abs/2501.19129v1https ://github.com/yunfanLu/RGB-Event-ISP
本研究首次探讨了事件引导的图像信号处理（ISP），提出了一个新的RGB-事件配对数据集及基准测试，以提升传统RGB图像生成的质量。现有的方法大多集中在后处理RGB图像，而忽视了事件传感器在ISP中的挑战和潜力。为了解决这一问题，研究者们收集了3373幅RAW图像及其对应的事件数据，涵盖了24个场景以及多种曝光模式和镜头设置。此外，提出了一条传统ISP管道，用于生成高质量的RGB参考图像，同时对现有可学习ISP方法进行了分类与评估。最后，研究者们提出了一种简单的事件引导ISP方法，并在新数据集上进行了测试，旨在激励相关领域的研究。
本研究的方法主要包括以下几个步骤：17.数据集构建：使用新开发的HVS-Eiger传感器，收集具有像素级对齐的事件和RAW图像，确保数据集的多样性，包括不同的光圈、焦距和曝光时间。18.传统ISP管道：构建一个基于MATLAB的可控ISP框架，执行黑电平补偿、去马赛克、白平衡、去噪和色彩空间转换等基本操作，利用ColorChecker作为参考。19.现有ISP方法分类：将现有的可学习ISP方法分为三类，选择多种方法在新数据集上进行训练和评估，确保不同场景下的性能比较。20.事件引导ISP方法：提出一种简单的UNet-like神经网络，将事件数据与RAW图像融合，以提升ISP的性能，特别是在动态场景中。
实验部分主要集中在对比不同ISP方法在新数据集上的表现。通过使用PSNR、SSIM和L1距离等指标，对各种方法在室内和室外场景下的图像重建效果进行评估。实验结果显示，PyNet在室外场景中表现最佳，而UNet在室内场景中表现较好。进一步分析表明，事件数据的引入显著提高了在户外场景中的性能，但在室内场景中，由于人工光源的闪烁，性能有所下降。此外，研究还探讨了不同ISP方法在处理复杂照明条件下的局限性，提出未来研究应关注于优化ISP性能和解决闪烁问题。
在这项研究中，科学家们开发了一种新方法，利用事件传感器拍摄的图像来改善传统相机的照片质量。首先，他们收集了很多图像和事件数据，确保这些数据能代表不同的拍摄环境。接着，他们设计了一种图像处理流程，像是给照片“洗澡”，让它们更清晰、更真实。这个流程包括几个步骤，比如调整黑色部分的亮度、去除图像中的噪音、以及确保颜色的准确性。最后，研究人员还测试了不同的图像处理方法，发现利用事件数据可以显著提升照片在快速移动场景中的表现，但在室内环境下，由于灯光闪烁，效果就没有那么理想了。

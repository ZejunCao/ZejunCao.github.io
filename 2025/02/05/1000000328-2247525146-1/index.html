本研究提出了一种名为“修正梯度引导（REG）”的新方法，旨在提升条件扩散模型的生成性能。尽管现有的引导技术在实践中表现出色，但其理论基础和实际实现之间存在显著差距。通过分析现有引导方法的局限性，研究者们发现现有的缩放边际分布目标在理论上并不有效，因此提出了一个有效的联合分布目标。研究的核心贡献在于建立了一个统一的理论框架，重新审视了引导技术的实质，并通过大量实验验证了REG方法的有效性，表现出在生成图像和文本等任务中的优越性。
本研究的方法主要包括以下几个步骤：
理论重构：首先，研究者们指出现有引导技术在缩放边际分布时与扩散模型的反向去噪过程存在矛盾，提出应当使用缩放联合分布目标来替代。
引导信号生成：引入REG的过程涉及更新噪声预测网络的输出，将其作为原始输出和用户定义的引导信号的加权和。引导强度由超参数控制。
多种引导形式：研究者们还探讨了不同形式的引导，包括分类器引导、无分类器引导和自动引导，展示了REG在这些方法中的兼容性和有效性。

在这项研究中，作者提出了一种新方法，叫做“修正梯度引导（REG）”，来帮助计算机生成更好的图像和文本。首先，他们发现现有的方法在理论上有些不够完善，特别是在处理数据时的某些步骤。为了解决这个问题，作者设计了一种新的方式来生成引导信号，这个信号可以告诉计算机如何更好地生成想要的结果。简单来说，REG通过结合原始的计算结果和一些额外的指导信息，帮助计算机在生成过程中更好地理解目标。通过一系列实验，作者证明了这种方法能够显著提升生成图像的质量，让计算机生成的图像更加真实和准确。
2025-01-31｜USTC,HFUT｜⭐️🟡
http ://arxiv.org/abs/2501.19243v1
在内容生成领域，扩散变换器（DiffusionTransformer,DiT）是一种重要的技术，但其在采样时通常需要较长的时间。为了解决这一问题，许多研究尝试通过缓存机制来加速生成过程。现有的缓存方法主要通过重用前一时间步的特征来减少计算量，然而它们往往忽视了缓存引起的错误，导致生成内容的质量显著下降。为此，本文提出了一种新的错误优化缓存（Error-OptimizedCache,EOC）方法，该方法通过提取先前知识、优化缓存步骤及减少缓存错误等手段，显著提高了生成图像的质量，并有效降低了因过度缓存而产生的错误累积。
EOC方法主要包括三个关键步骤：4.先前知识提取（PriorKnowledgeExtraction）：该步骤通过记录模型在不同生成阶段的输出，分析和处理各个模块之间的差异。这种差异被称为“趋势”，用于后续的缓存优化。5.缓存优化判定（CacheOptimizationDetermination）：在这一阶段，系统评估每个缓存步骤是否需要优化。具体来说，若某个模块的输出与缓存的模块之间存在显著差异，则优先考虑对该缓存进行优化，以避免负面影响。6.缓存优化（CacheOptimization）：在确认需要优化的缓存模块后，通过将先前知识中的趋势信息嵌入到生成过程中，从而有效减少缓存引起的错误，提升生成内容的质量。
在实验部分，研究者们在ImageNet数据集上对EOC方法进行评估，比较了其与传统基于规则和训练的缓存方法的性能。实验结果显示，EOC方法在不显著增加计算负担的情况下，显著提高了生成图像的质量。具体而言，使用EOC后，生成图像的FrechetInceptionDistance（FID）值较之前显著降低，表明生成内容的质量得到了提升。此外，通过对不同缓存策略的对比，研究者们验证了EOC方法在减少错误累积和提高生成效率方面的有效性。
在我们的研究中，我们提出了一种新的方法来加速内容生成，尤其是在使用扩散变换器（DiT）时。这个方法叫做错误优化缓存（EOC），它的工作原理可以简单地理解为三个步骤：7.提取知识：我们首先记录模型在生成图像时的表现，看看不同时间步骤之间的变化。这就像观察一幅画的不同阶段，以了解哪些部分需要特别关注。8.判断是否需要优化：接下来，我们检查哪些步骤的结果不太好，如果发现某些步骤的输出和我们之前记录的有很大不同，我们就知道这部分需要优化。9.进行优化：最后，我们将之前观察到的变化趋势应用到生成过程中，这样可以减少因重复使用旧数据而导致的错误，从而提高生成图像的质量。
通过这种方式，我们不仅加快了生成速度，还确保了生成内容的质量。
2025-01-31｜BostonU,Runway｜⭐️
http ://arxiv.org/abs/2501.19066v1
在文本到图像生成模型的快速发展中，尽管取得了显著的进展，但这些模型也面临着生成不安全或不道德内容的风险。现有的解决方案通常依赖于对模型进行微调，以去除特定概念，这种方法不仅计算成本高，而且在可扩展性和生成质量方面存在不足。为了解决这些问题，本文提出了一种新颖的框架，利用k-sparse自编码器（k-SAEs）实现对生成内容的高效和可解释性控制。该方法能够在不影响整体图像结构和视觉质量的前提下，有效地操控生成过程中的具体概念，如去除不当内容或引入新概念。通过大量实验，结果表明该方法在不安全概念的去除、风格操控等方面表现优异，且速度比当前最先进的方法快约五倍。
本研究的核心在于利用k-sparse自编码器（k-SAE）来实现对生成模型的概念操控。具体方法包括以下几个步骤：10.训练k-SAE：首先，我们从包含语义概念的文本提示中提取嵌入，训练k-SAE以识别可解释的单语义概念。每个k-SAE的隐藏状态对应一个独立的单语义概念。11.生成过程中的操控：在生成过程中，当需要操控某个概念时，k-SAE会识别其对应的潜在方向，并精确调整该概念在生成结果中的存在量。12.无微调需求：该方法的显著特点是，k-SAE仅需训练一次，无需对基础模型进行微调或使用额外的适配器，从而保留生成能力并提高效率。13.鲁棒性：该方法对于对抗性提示的操控十分有效，确保在面对挑战性场景时仍能生成可靠的图像。
在实验部分，我们通过多项测试验证了所提方法的有效性。首先，我们使用不当图像提示（I2P）数据集评估了去除不安全概念的效果，并在Ring-A-Bell基准上测试了对暴力内容的抑制能力。实验结果显示，该方法在去除不当内容方面的成功率显著高于现有方法，同时保持了较高的视觉质量和与输入提示的对齐程度。此外，我们还进行了风格操控的实验，展示了该方法在处理多种摄影风格和对象属性时的灵活性。通过对比不同方法的性能指标，如攻击成功率（ASR）和FID分数，我们的方法在多项任务中表现出色，证明了其高效性和实用性。
在这项研究中，我们使用了一种称为k-sparse自编码器的技术来帮助生成图像时更好地控制图像的内容。简单来说，k-sparse自编码器就像一个聪明的助手，它可以在生成图像的过程中，识别并调整我们想要的具体元素，比如去掉不适当的内容或改变图像的风格。这个助手只需要训练一次，就能在后续的生成中快速工作，而不需要每次都重新学习，这样就节省了很多时间和计算资源。我们的实验表明，这种方法不仅能有效去除不当内容，而且在处理不同风格的图像时也表现得非常灵活，最终生成的图像质量也得到了保证。通过这种方式，我们可以更安全、更高效地使用文本到图像的生成模型。
2025-01-31｜MonashU,DSTG｜⭐️
http ://arxiv.org/abs/2501.18950v1https ://github.com/tuananhbui89/Adaptive-Guided-Erasure
在生成模型中，概念抹除技术被提出以降低有害内容生成的风险。传统的抹除方法通常将不希望的概念映射到固定的通用概念，这种方法在效果上并不理想，因为它忽视了抹除一个概念对其他概念的影响。本文提出了一种新的方法——自适应引导抹除（AGE），通过将概念空间建模为图形，分析抹除一个概念对其他概念的影响，从而选择最优的目标概念。研究发现，抹除的影响是局部的，即仅限于与被抹除概念相近的其他概念。AGE方法通过动态选择与不希望的概念相关的目标，显著提高了抹除效果，同时更好地保留了无害概念。
AGE方法的核心是选择适当的目标概念以实现有效的抹除，具体步骤如下：22.概念空间建模：将概念空间视为图形，节点代表概念，边的权重表示抹除一个概念对其他概念的影响。23.目标概念选择：在抹除过程中，选择与待抹除概念相关但不相同的概念作为目标，这样可以最小化对其他概念的负面影响。24.优化问题：通过解决一个极小极大优化问题，自动选择每个待抹除概念的最佳目标概念。该问题的目标是最小化抹除效果，同时保持模型对其他概念的生成能力。25.组合概念表示：为了丰富目标概念的选择，将其表示为多个单一概念的组合，增加了目标概念的表达能力和优化效率。
实验部分评估了AGE方法在多个抹除任务中的有效性，包括物体相关概念、NSFW属性和艺术风格的抹除。通过与现有最先进的方法进行比较，AGE在抹除不希望的概念的同时，能够更好地保留其他重要概念。例如，在抹除物体相关概念的实验中，AGE在保留成功率和抹除成功率上均优于其他方法。在NSFW属性的抹除实验中，AGE显示出最低的露出率，表明其在抹除不当内容方面的有效性。此外，在艺术风格抹除实验中，AGE在保留性能上也表现出色，显示了其在多种任务中的广泛适用性和优越性。
AGE方法的基本思路是帮助我们在生成图像时，去掉一些不想要的内容，比如不当的图像或特定的艺术风格。它通过分析不同概念之间的关系，找出最合适的替代概念来进行抹除。想象一下，如果你想去掉某种动物的图像，AGE会选择一个与之相关但不完全相同的动物作为替代，这样就不会影响到其他动物的图像生成。这个过程是通过数学优化来实现的，确保在去掉不想要的内容的同时，尽量保留其他无害的内容。这样，AGE方法就能在生成图像时更加智能和高效。
2025-01-31｜KNU｜⭐️
http ://arxiv.org/abs/2501.18877v1https ://github.com/aei13/DES
在文本到图像的扩散模型中，尽管这些模型在根据文本提示生成图像方面表现出色，但它们仍然存在生成不安全内容（如NSFW内容）的风险。现有的防御措施，如提示过滤和概念遗忘，往往无法有效抵御对抗性攻击，同时又不能保证生成图像的质量。为了解决这些问题，本文提出了一种名为“扭曲嵌入空间”（DistortingEmbeddingSpace,DES）的新型防御机制。该机制通过创新的嵌入空间控制，有效地将不安全的嵌入转化为经过精心计算的安全嵌入区域，从而防止不安全内容的生成，并在此过程中保持原有安全嵌入的生成质量。此外，DES还可以在不同的文本到图像模型中以即插即用的方式进行部署，且不增加推理开销，显示出其在防御能力和生成质量方面的卓越性能。
DES的核心在于其嵌入空间的控制，主要通过以下几个步骤实现：
目标向量生成阶段：首先，DES识别与不安全嵌入相似度最低的安全向量，以确保不安全向量被转化为安全区域。该过程通过计算余弦相似度来实现，选择与不安全向量相对较远的安全向量作为目标。
训练阶段：在训练过程中，DES采用三种损失函数来优化文本编码器，确保不安全嵌入被有效转化：
不安全嵌入中和（UEN）：最小化当前不安全向量与目标安全向量之间的余弦相似度。
安全嵌入保留（SEP）：确保安全向量保持与原安全向量的高相似度，避免性能下降。
裸体嵌入中和（NEN）：通过将裸体向量与无条件向量对齐，从根本上消除对裸体概念的直接利用。
损失函数的调节：采用邻近感知损失调整（PALA）来动态调整损失，确保在处理模糊表达时，安全向量的语义不被破坏。
在实验部分，研究团队利用多个模型版本（如StableDiffusionv1.4和v1.5）进行广泛的测试。通过黑箱和白箱攻击场景，DES的防御性能得到了验证。实验结果显示，DES在防止NSFW内容生成方面表现出色，攻击成功率（ASR）显著低于其他防御机制，同时在图像生成质量方面保持高标准。此外，DES的训练时间仅需90秒，且在推理时没有额外开销，展现出极高的实用性和效率。通过与其他方法的比较，DES在多项指标上均表现优异，证明了其在实际应用中的有效性和灵活性。
DES的工作原理可以简单理解为一个“安全守卫”，它帮助文本到图像的生成模型避免生成不适当的内容。首先，DES会识别出危险的提示（比如可能导致生成不雅图像的文字），然后它会把这些危险的“图像想法”转化为安全的“图像想法”。这个过程就像是把一个不合适的故事情节重新编写成一个适合所有观众的版本。为了确保这个过程不会影响到其他正常的内容，DES还会在训练时使用几种不同的策略，确保安全内容的质量不受影响。最终，DES不仅能有效防止不当内容的生成，而且在生成的图像质量上也保持高标准，非常适合实际应用。
2025-01-28｜BostonU｜⭐️
http ://arxiv.org/abs/2501.18642v1
在当前的生成式人工智能（AI）模型中，种族和性别偏见问题日益严重，尤其是在文本到图像生成的应用中。为了应对这一问题，研究者们提出了一种名为DebiasPI（通过提示迭代进行去偏见）的推理时干预过程，旨在通过用户控制生成图像中个体的人口属性分布来降低偏见。DebiasPI通过跟踪已生成的属性，利用模型的内部状态或外部分类器来引导模型选择尚未充分代表的属性。通过这种方法，研究者成功生成了在性别和种族上表现出平等代表性的图像，展示了与新闻标题相关的复杂概念。此外，研究还探讨了其他属性（如年龄、体型、职业和肤色）的变化，并分析了在干预过程中不同属性之间的相互影响。
研究采用了三种类型的提示来评估文本到图像生成模型的偏见。首先是基线提示，未明确要求模型注意任何属性，可能会放大模型的内部偏见。其次是带有属性列表的提示，允许用户选择特定属性生成图像。最后是带有属性分布的提示，用户可以设定目标属性分布。DebiasPI的过程包括以下几个步骤：4.设定目标分布：用户在提示中定义希望生成的属性分布。5.生成图像：将目标分布和文本输入发送给生成模型。6.属性跟踪：通过模型的内部信念或外部分类器来解析生成图像中的属性。7.调整选择：当某一属性的生成数量达到预设目标时，停止生成该属性，促使模型选择其他属性。8.迭代调整：持续调整生成过程，以确保最终输出符合目标属性分布。
研究团队在实验中使用了GPT-4生成200个关于人类成功故事的中立新闻标题，并通过DALL-E3生成相应的图像。基线实验显示，生成的图像中98.5%为男性，且大多数为白人。随后，研究者使用属性列表提示进行实验，发现模型倾向于生成两面板图像，其中职业成功前后的个体有时表现出不同的种族或性别，进一步加深了社会偏见。在进行伦理干预实验时，研究者调整了提示，使生成的图像在性别和种族上达到更均衡的分布。通过对比有无DebiasPI控制的实验，结果显示，使用DebiasPI时，生成的图像在性别和种族的代表性上显著改善。
在这项研究中，科学家们想要解决一个重要问题：如何让AI生成的图像在性别和种族上更加公平。为此，他们设计了一种名为DebiasPI的方法。这个方法就像是一个指挥家，帮助AI在生成图像时选择不同的属性，比如男女、不同肤色的人。首先，用户会告诉AI希望看到的属性比例，比如希望生成的图像中男女各占50%。然后，AI会根据这些要求生成图像，并在生成过程中记录下已经生成了哪些属性。当某个属性的数量达到要求时，AI会停止生成该属性，转而选择其他属性。这样，通过不断调整，AI最终能够生成符合用户期望的公平图像。这项研究不仅提高了AI生成图像的多样性，还引发了对如何使用AI的伦理思考。

291-A3.2｜可控动作生成，实时动作生成，运动扩散自编码器与空手道，共语姿势生成

2025-01-30｜AIGeeks｜⭐️
http ://arxiv.org/abs/2501.18726v1https ://aigeeksgroup.github.io/
人类运动生成是计算机视觉领域的重要研究方向，广泛应用于电影制作、视频游戏、增强现实/虚拟现实以及人机交互等多个领域。现有的方法主要依赖于扩散模型或自回归模型进行文本到运动的生成。然而，这些方法面临两个主要挑战：首先，生成过程耗时较长，限制了实时应用的可能性，如游戏和机器人操控；其次，现有模型通常以相对运动表示为基础，难以实现精确的关节级控制。为了解决这些问题，本文提出了一种简单但有效的架构，包括两个关键组件：优化的硬件效率和计算复杂性，以及引入的运动控制网络（MotionControlNet），使得运动生成更加精确和可控。
我们提出的方法主要包括两个部分：
高效运动变换器（EfficientMotionTransformer）：
通过自定义闪电线性注意机制，优化变换器模型在运动生成中的硬件效率和计算复杂性。
该机制允许在现代计算架构上更高效地处理运动数据，显著提升生成性能。
运动控制网络（MotionControlNet）：
该网络在运动潜在空间内进行定制，以实现更精确的关节级控制。
每一层都附加了一个零初始化的线性层，以消除初始训练步骤中的随机噪声，从而提高生成运动的准确性和一致性。
运动潜在一致性模型通过引入噪声并进行预测来加速生成过程，确保生成的运动序列与用户定义的轨迹一致。
为了验证我们提出的方法的有效性，计划进行一系列实验，包括比较研究和消融研究。我们将使用HumanML3D数据集对比我们的模型与其他最新模型的性能。此外，消融研究将帮助我们评估高效运动变换器、运动控制网络和运动潜在一致性模型等组件对整体性能的贡献。通过系统性地禁用各个组件并分析结果，我们希望深入理解每个部分如何影响模型的整体效果，以指导进一步的优化和改进。
在我们的研究中，我们提出了一种新方法来生成3D人类运动，特别是在游戏和机器人等实时应用中。首先，我们设计了一种叫做高效运动变换器的工具，它可以更快地处理运动数据，就像给计算机装上了更快的“引擎”，使得生成运动变得更加迅速。其次，我们引入了运动控制网络，这就像给运动添加了“遥控器”，让用户能够更精确地控制每个关节的动作。通过这两种技术的结合，我们希望能生成更加真实、自然的运动，同时减少生成所需的时间，使其适合实时应用。这样，无论是在游戏中还是机器人操作中，用户都能获得更好的体验。
2025-01-31｜UCL,Oxford｜⭐️
http ://arxiv.org/abs/2501.19083v1
在近年来多模态生成模型的快速发展中，MotionPCM作为一种新型的文本条件运动合成模型应运而生。该模型旨在解决传统扩散模型在实时应用中面临的高计算复杂度和长采样步骤的问题。通过引入相位一致性模型（PCM），MotionPCM能够在生成高质量人类运动的同时，实现实时合成。与以往的运动合成方法相比，MotionPCM在保持生成质量的基础上，显著提高了合成速度，特别是在处理文本描述时，展现出卓越的性能。这一方法的成功应用不仅为游戏、电影制作和虚拟现实等领域带来了新的可能性，也为人类动作生成的研究提供了新的思路。
MotionPCM的设计分为几个关键部分，以确保高效的实时运动合成。首先，模型利用变分自编码器（VAE）将运动序列压缩为低维潜在代码，从而减少计算量。其次，模型采用相位一致性模型（PCM），将整个运动轨迹划分为多个子轨迹，每个子轨迹在固定的时间间隔内进行一致性约束，避免了在多步采样过程中累积的随机噪声。具体步骤如下：
VAE预训练：通过VAE将运动序列转换为潜在空间中的低维表示，便于后续的扩散过程。
相位一致性模型的集成：将PCM引入合成管道，分段处理运动轨迹，以实现高效的两步推理。
对抗性损失的引入：通过引入一个判别器，增强生成结果的真实性和一致性，提升模型在低步设置下的表现。
生成过程：在推理阶段，从先验分布中采样潜在代码，利用PCM进行运动序列的生成，并通过解码器输出最终的运动序列。
在实验部分，研究团队在广泛使用的HumanML3D数据集上评估了MotionPCM的性能。通过与现有的多种运动合成方法进行比较，MotionPCM在多个评估指标上均表现出色，包括平均每句生成时间（AITS）、R-Precision、FrechetInceptionDistance（FID）和多模态距离（MMDist）。尤其在生成速度上，MotionPCM在1步采样下能够实现每秒超过30帧的运动合成，满足实时应用的需求。同时，模型在生成质量和一致性方面也超过了大多数基线方法，展示了其在文本驱动运动生成任务中的强大能力。
MotionPCM的工作原理可以简单理解为一个高效的“运动生成机器”。首先，它通过一个叫变分自编码器（VAE）的技术，把复杂的运动数据压缩成更小、更易处理的格式，就像把一本大书的内容提炼成几页摘要。接着，MotionPCM将整个运动轨迹分成几个小段，每段都单独处理，以确保每一步生成的动作都保持一致，不会因为多次处理而产生错误。这个过程就像在制作一部电影时，每个场景都要仔细拍摄和剪辑，确保最终的效果流畅自然。此外，MotionPCM还引入了一个“审查员”，这个审查员会检查生成的动作是否真实并符合预期，帮助提升生成的质量。最终，MotionPCM能够快速而准确地将文本描述转换为生动的运动，适用于各种实时应用场景。
2025-01-30｜UBremen｜⭐️
http ://arxiv.org/abs/2501.18729v1https ://github.com/anthony-mendil/MoDiffAE
本研究提出了一种名为运动扩散自编码器（MotionDiffusionAutoencoders,MoDiffAE）的新型方法，旨在实现人类运动数据的属性操控，尤其是针对空手道技术的应用。传统方法在处理人类运动时面临诸多挑战，包括运动的序列性和属性的时变性。本研究首次成功实现了对人类运动数据中高层属性的操控，具体表现为在保持其他属性不变的情况下，改变运动员的技术水平或执行的技术类型。为此，研究团队设计了一种基于旋转的姿态表示方法，使得运动轨迹与人体骨架的解耦成为可能，同时确保了原始解剖结构的准确重建。
该方法的核心在于以下几个关键步骤：
旋转基础的姿态表示：设计了一种新的姿态表示方式，通过解耦人体骨架和运动轨迹，使得运动的重建更为精准。
高层语义发现：使用变压器编码器（transformerencoder）来提取运动数据中的高层语义信息，从而实现对运动属性的操控。
扩散概率模型：采用扩散模型来处理运动数据中的随机变异，确保在操控某一属性时，其他未被操控的属性保持不变。
属性操控机制：通过在语义嵌入空间中沿特定方向移动，来实现对运动属性的操控，确保操控过程中的语义信息和随机变异的保留。
本研究采用了Kyokushin空手道数据集进行实验，数据集包含37名健康参与者的3308个样本。实验分为定量和定性两部分。定量评估中，主要分析了模型嵌入空间的线性可分性，验证了属性操控的有效性。定性评估则通过专家知识对运动特征进行预期，检验实际操控结果是否与预期匹配。结果显示，该模型在五种空手道技术中成功操控了三种技术的执行，并能够在保持技术类型的同时，改变运动员的技能等级。
在我们的研究中，我们开发了一种新的技术，叫做运动扩散自编码器，专门用来改变人类运动的某些特征，比如空手道的技术水平。我们首先设计了一种新方法来表示运动姿势，这样可以更好地理解和重建运动的细节。接着，我们用一种叫做变压器的工具来提取运动中的重要信息，这样我们就能知道如何改变运动的某些方面，而不影响其他部分。最后，我们通过一个扩散模型来处理运动中的随机变化，确保在我们改变运动属性时，其他部分依然保持一致。这种方法能帮助运动员提高技能，甚至可以用于动画制作，让角色的动作更自然。
2025-01-31｜URochester｜⭐️
http ://arxiv.org/abs/2501.18898v1https ://andypinxinliu.github.io/GestureLSM
在自然语言交流中，手势是表达情感和增强理解的重要非语言线索。然而，基于语音信号生成自然手势的任务在计算机视觉领域仍然面临诸多挑战。现有研究往往未充分考虑身体各部位在发言时的空间互动，导致生成的手势缺乏协调性和自然感。为了解决这些问题，本文提出了一种名为GestureLSM的框架，采用潜在快捷模型和空间-时间建模的方法。该框架通过对身体不同部位的交互进行显式建模，旨在实现流畅且实时的手势生成。GestureLSM通过对身体部位的分区和空间、时间注意力机制的结合，显著提高了手势生成的速度和质量，为数字人类和具身代理的开发提供了新的可能性。
GestureLSM的核心在于其独特的手势生成方法，主要分为以下几个步骤：
手势特征表示：将身体分为上半身、手部、下半身和面部表情四个部分，通过残差向量量化将这些局部运动转换为离散的运动表示。
空间-时间手势生成：采用基于变压器的模型，利用空间和时间注意力机制来捕捉身体各部位之间的关系。空间注意力关注同一时刻不同部位的相互作用，而时间注意力则关注各部位随时间变化的动态。
潜在快捷模型：通过引入潜在快捷学习，GestureLSM在量化手势表示的紧凑空间中进行学习，显著提高了生成速度和质量。该方法通过分析去噪模式和有效的时间分布采样策略，进一步提升了模型的生成效果。
本研究使用BEAT-X数据集进行模型的训练和评估，该数据集包含25名发言者的60小时高分辨率手势数据。我们通过定量和定性实验验证了GestureLSM的有效性。在定量评估中，使用FrechetGestureDistance(FGD)和BeatConstancy(BC)等指标，结果显示GestureLSM在所有评估指标上均优于现有方法，特别是在生成质量和实时推理速度方面。定性评估则通过用户研究进行，参与者对生成手势的自然性、与语音的同步性和流畅性进行了评分，结果显示GestureLSM生成的手势在这些方面均表现优越。
GestureLSM的工作原理可以简单理解为一个“智能手势生成器”，它通过分析我们说话时的身体动作来生成自然的手势。首先，它把我们的身体分成几个部分，比如手、手臂和脸，并为每个部分创建一个“动作库”。然后，当我们说话时，这个系统会根据说话的内容和节奏，选择合适的手势。它使用两种注意力机制：一种是“空间注意力”，用来确保不同身体部分在同一时刻的协调；另一种是“时间注意力”，用来确保手势随着说话的节奏自然变化。通过这些方法，GestureLSM能够快速生成流畅、自然的手势，让我们的虚拟角色在交流时看起来更真实。

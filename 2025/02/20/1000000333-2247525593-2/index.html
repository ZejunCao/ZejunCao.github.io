2025-02-07｜Imperial(ICL)｜ICLR2025｜⭐️⭐️
http ://arxiv.org/abs/2502.04923v1https ://github.com/Yqcca/CMLoRA
本研究提出了一种新颖的CachedMulti-LoRA(CMLoRA)框架，旨在解决在多概念图像生成中，低秩适应（LoRA）技术组合时所面临的语义冲突问题。尽管LoRA技术能够有效地渲染多个独特元素（如角色和背景），但当组合多个LoRAs时，生成的图像质量往往受到影响。本文通过分析LoRAs在去噪过程中的作用，特别是在傅里叶频域中的表现，发现不同LoRAs在高频和低频特征的处理上存在显著差异。基于这些发现，研究提出了一种基于频域的调度策略，以确定在推理过程中LoRAs的最佳组合顺序，从而提高图像生成的质量和效率。
本研究的方法主要分为以下几个部分：
LoRA的傅里叶分析：通过傅里叶变换分析LoRAs在去噪过程中的频率响应，识别出高频和低频LoRAs，并根据其特征进行分类。
优先级排序策略：根据高频特征的变化幅度对LoRAs进行排序，形成优先级策略O，确保在去噪的早期阶段优先使用高频LoRAs，以增强图像的细节和纹理。
CachedMulti-LoRA框架：在推理过程中，CMLoRA框架通过动态选择高频和低频LoRAs的组合，利用缓存机制来提高生成图像的质量。具体而言，框架在去噪的初期使用高频LoRAs，并在后期引入低频LoRAs，减少因多LoRA组合导致的语义冲突。
在实验中，研究使用了ComposLoRA测试集，对比了CMLoRA与其他多LoRA组合方法的性能。实验结果表明，CMLoRA在CLIPScore和MiniCPM-V的评估中均表现优异，尤其在图像的美学质量和语义准确性方面，CMLoRA的表现显著优于现有的LoRA组合方法。此外，实验还验证了缓存机制对于提升低频LoRA的贡献和稳定性的有效性，进一步证明了CMLoRA在多概念图像生成任务中的潜力。
在这项研究中，我们提出了一种新方法，叫做CMLoRA，旨在帮助计算机更好地生成复杂图像，比如将不同角色、风格和背景结合在一起。首先，我们分析了不同LoRA（低秩适应）技术在图像生成中的表现，发现有些LoRA更擅长处理细节和纹理（高频），而有些则更关注整体结构（低频）。为了更有效地组合这些LoRA，我们制定了一个优先顺序，让计算机在生成图像时，首先使用那些能增强细节的LoRA，然后再引入其他LoRA，确保生成的图像既美观又一致。通过这种方式，CMLoRA不仅提高了图像的质量，还减少了不同LoRA之间可能产生的冲突，使得生成的图像更加自然。
2025-02-07｜USurrey,AdobeResearch｜⭐️🟡
http ://arxiv.org/abs/2502.05165v1
本文介绍了一种名为Multitwine的生成模型，能够同时进行多对象合成，并通过文本和布局进行控制。该模型的独特之处在于，它不仅可以在场景中添加多个对象，还能捕捉对象之间的复杂交互，如“拥抱”或“弹吉他”。通过结合文本输入和布局信息，Multitwine能够自动生成与所描述动作相关的附加元素，例如在“自拍”场景中添加相机。这种模型的设计旨在克服当前单一对象合成模型的局限性，使得多个对象的交互更自然、更流畅。
Multitwine模型采用了一种基于扩散的架构，能够在控制的情况下同时合成多个对象。其主要方法分为以下几个方面：
输入处理：模型接收背景图像、布局信息（如每个对象的边界框）、多个对象图像及描述文本。该布局信息帮助模型理解每个对象在场景中的位置和关系。
多模态嵌入：通过将文本描述和对象图像进行编码，生成多模态嵌入。这一过程确保文本和视觉信息的平衡，使得模型在缺少某一输入时仍能有效工作。
训练策略：模型采用联合训练策略，同时优化合成和定制任务，以增强文本与图像的对齐能力。通过随机丢弃输入的某一模态，模型能够在不同情况下保持性能。
数据生成管道：利用多种数据源（如视频和手动收集的图像），构建多样化且高质量的训练数据集，确保模型能学习到丰富的对象交互和场景上下文。
在实验部分，研究者对Multitwine模型的性能进行了全面评估，包括与现有单对象合成模型的比较。实验结果表明，Multitwine在同时合成多个对象时表现优越，特别是在对象交互的自然性和图像质量方面。使用定制化任务作为辅助训练，模型在文本对齐和对象身份保持方面也取得了显著改进。此外，用户研究显示，用户普遍偏好Multitwine生成的图像，认为其在真实感和交互性方面优于其他基线模型。实验还表明，模型在处理复杂场景时，能够有效地捕捉对象之间的关系，生成更和谐的合成结果。
Multitwine模型的工作原理可以简单理解为一个智能的图像编辑器。它能够根据用户输入的文字和布局信息，自动将多个对象合成到一幅背景图中。首先，用户提供一个背景图像和一些描述，比如“在公园里，两个人在拥抱”。模型会根据这些信息，找到合适的图像来代表这些对象，并将它们放置在正确的位置。接着，它会确保这些对象看起来和谐，甚至在需要时添加一些额外的元素，比如在拥抱的场景中加入一只狗。通过这种方式，Multitwine不仅能创建出美观的图像，还能让这些图像看起来更真实、更自然。
2025-02-07｜HKU,RIKEN,ShanghaiAILab｜⭐️🟡
http ://arxiv.org/abs/2502.04725v1
本研究探讨了扩散模型（DMs）在学习图像中隐含特征规则方面的能力，特别是对于图像中存在依赖关系的特征对（如太阳高度与影子长度）。尽管扩散模型在生成真实图像方面表现出色，但在捕捉特征间复杂关系时仍存在显著的不足。通过对主流扩散模型（如StableDiffusion3.5）的实证评估，研究发现模型在处理光影关系和物体反射等情况下产生不一致的结果。为此，研究者设计了四个合成任务，以评估扩散模型在学习隐含特征规则方面的能力。结果表明，尽管扩散模型能够识别粗略的规则，但在精细规则的学习上却显得力不从心。
本研究采用了一种系统化的方法来评估扩散模型的规则学习能力，主要包括以下几个步骤：
合成任务设计：设计四个合成任务，分别模拟光影关系、反射/折射、形状与颜色的关系等，每个任务设定了粗粒度和细粒度的规则。
模型训练：使用无条件的扩散模型（DDPM）进行训练，设置合适的训练参数和数据样本，以便于评估模型对特征规则的学习能力。
特征提取与评估：通过特定的特征提取管道，对生成图像进行分析，验证其是否符合设计的特征规则。该管道包括颜色掩膜、元素计数和关键特征提取等步骤。
引导扩散：为了解决模型在学习细粒度规则时的不足，引入了分类器指导的策略，利用对比学习构建正负样本对，进一步提升模型生成规则符合性的能力。
在实验部分，研究者生成了多个样本并对其进行评估。每个合成任务生成2000个样本，并通过特征提取方法来验证生成图像是否遵循预设的规则。实验结果显示，扩散模型在粗粒度规则方面表现良好，生成的样本大多数符合这些规则。然而，当面对细粒度规则时，模型的表现明显不足，生成的样本与期望的规则存在显著偏差。尤其是在非空间规则的任务中，模型的学习能力更为薄弱。通过引入引导扩散策略，研究者观察到生成样本的质量有所改善，但仍然无法完全克服细粒度规则学习的挑战。
在本研究中，科学家们想要了解一种叫做扩散模型的图像生成技术是否能学会图像中不同特征之间的隐含规则。为了测试这一点，他们设计了几个简单的任务，比如让模型理解太阳的位置与影子的长度之间的关系。研究发现，虽然这种模型可以很好地识别一些简单的规则，比如太阳和影子应该在不同的位置，但在处理更复杂的规则时，它们却常常出错。为了帮助模型更好地学习这些复杂的规则，研究者们还引入了一种新的方法，通过引导模型学习更多的细节。尽管如此，模型在学习细节方面依然面临挑战，显示出它们在处理复杂图像关系时的局限性。
2025-02-06｜SYSU,PengchengLab｜⭐️🟡
http ://arxiv.org/abs/2502.04412v1
随着文本到图像生成技术的进步，扩散模型（DiffusionModels）在生成艺术性和细节丰富的图像方面展现了显著的能力。然而，当前的扩散模型在理解复杂文本提示时存在局限性，特别是在捕捉语义和逻辑关系方面。本研究提出了一种新的方法，将解码器结构的大型语言模型（LLMs）与扩散模型相结合，旨在提升其文本到图像生成的效果。通过引入LLMDiff-Adapter，我们的模型能够更好地理解用户意图和文本中的实体关系，从而生成更高质量和更具逻辑性的图像。实验结果显示，使用该适配器的模型在生成质量和可靠性方面超越了当前最先进的模型。
本研究的方法主要包括以下几个步骤：
模型架构选择：我们采用解码器结构的LLMs（如GPT系列），因为其在语义理解和生成任务中表现优越。与传统的编码器-解码器架构相比，解码器结构能够更好地处理长文本提示。
LLMDiff-Adapter设计：该适配器作为一个插件模块，连接LLMs与扩散模型。它通过集成来自LLMs的文本编码，增强扩散模型对文本提示的理解能力。
文本编码过程：在生成图像之前，适配器将输入文本转化为一系列的文本编码。这个过程包括利用LLMs的自注意力机制来捕捉文本中的上下文信息和语义关系。
训练与优化：模型通过最小化均方误差（MSE）损失函数进行训练，以提升生成图像的质量和细节表现。适配器的参数在训练过程中进行动态调整，以适应不同的文本输入。
为了验证我们提出的方法的有效性，进行了多项实验：
数据集选择：我们使用了来自GRIT和Midjourney的数据集，确保训练样本的多样性和丰富性。总共使用了约100万条数据进行训练。
基准模型比较：将我们的LLMDiff-Adapter与现有的文本编码模型（如CLIP和T5）进行比较，评估生成图像的质量、复杂性和美感。
评估指标：采用SigLIPScore评估生成图像与文本提示的匹配程度，并使用CLIP-IQA评估图像的质量和细节。用户研究也被用于评估生成图像的逻辑性和整体表现。
结果分析：实验结果显示，使用LLMDiff-Adapter的模型在各项指标上均优于对比模型，尤其是在处理复杂文本提示时，能够更准确地生成符合用户意图的图像。
在我们的研究中，我们提出了一种新方法来改进图像生成模型的性能。我们使用了一种名为LLMDiff-Adapter的工具，这个工具可以帮助计算机更好地理解用户输入的文本，比如描述一幅画的内容。这个过程大致可以理解为：首先，我们使用一种强大的语言模型来分析文本，这个模型像一个聪明的助手，能够抓住文本中的重要信息和细节。然后，我们将这些信息输入到图像生成模型中，这样模型就能根据这些指示生成更符合用户期望的图像。通过这种方式，我们的模型不仅能够生成美丽的图像，还能确保这些图像与用户的描述高度一致，甚至在复杂场景的生成上表现得更好。
2025-02-05｜JLU｜⭐️
http ://arxiv.org/abs/2502.04369v1
在本研究中，作者提出了一种新的风格转移模块，称为HolisticStyleInjector（HSI），旨在提高任意风格转移（AST）的效率和效果。传统的风格转移方法通常通过直接合并内容和风格特征，导致生成图像的局部失真和不和谐。尽管基于注意力机制的方法在合成风格细节方面表现出色，但它们往往过于关注局部模式，忽略了全局风格特征。为了解决这些问题，HSI模块通过全局风格表示进行风格化，避免了局部不和谐的图案，并通过动态双重关系学习机制，确保生成图像能够保留原始内容并提高风格保真度。实验结果表明，HSI在有效性和效率上均优于现有的最先进方法。
HSI的设计包含三个主要特征：全球风格提取、动态双重关系构建和线性复杂度传输过程。
全球风格提取：HSI通过提取一系列全局风格统计特征（如均值、方差、偏度和峰度）来建立与内容特征的直接联系，从而避免了局部区域造成的风格偏差。
动态双重关系构建：该模块同时构建局部内容到全局风格和全局内容到全局风格的关系，以增强风格化质量。通过这种方式，当内容图像与风格图像在语义上接近时，可以增强整体和谐性。
线性复杂度传输过程：HSI采用逐元素相乘的方式建立内容与风格之间的关系，显著降低了计算复杂度，相较于传统的矩阵乘法，HSI的计算复杂度为线性级别。
本研究在MS-COCO和WikiArt数据集上进行了广泛的实验，使用多种分辨率的图像进行风格转移。实验结果表明，HSI在视觉质量和计算效率上均优于六种最先进的AST方法，包括AdaIN、SANet和StyA2K等。通过比较内容损失、风格损失、LPIPS和FID等多项指标，HSI展现出更低的风格损失和更好的内容保留能力。此外，HSI能够在高分辨率图像上有效运行，避免了其他方法在处理高分辨率图像时的内存溢出问题。定量与定性的评估结果均表明，HSI在风格转移任务中具有显著优势。
HSI方法的核心在于如何更好地将艺术风格融入到普通图像中。首先，它会提取风格图像的整体特征，而不是仅仅关注某一个小区域，这样可以确保风格的整体性不会被局部失真所影响。其次，HSI会同时考虑内容图像和风格图像的关系，确保两者在视觉上能够和谐地结合。最后，HSI使用简单的数学运算（逐元素相乘），这样处理速度快且不容易出错，尤其是在处理大图像时表现得尤为出色。这种方法不仅提升了图像的美观度，还大大减少了计算机处理时的负担，让风格转移更加高效。
2025-02-05｜UWaterloo,NUS,UCMerced,UAlberta,UniversityatBuffalo,UQueensland｜⭐️
http ://arxiv.org/abs/2502.04364v1
随着扩散模型的快速发展，文本引导的图像编辑工具如InstructPix2Pix和ControlNet等，已变得越来越普遍。这些工具允许用户对合成内容进行精确和迭代的修改，然而，这也带来了内容真实性和可追溯性的新挑战。特别是在对图像进行多次编辑的情况下，如何有效识别和追踪图像的来源变得尤为重要。本文提出了一种名为LAMBDATRACER的新型归属方法，旨在从潜在空间中识别和区分真实与被操控的图像。通过适应性地调整重构损失，LAMBDATRACER能够在各种编辑过程中有效地保持准确性，确保内容的真实性并降低误用的风险。
LAMBDATRACER的核心在于其创新的归属机制，主要包括以下几个步骤：
图像生成与编辑：首先，使用稳定扩散模型生成图像，然后通过文本引导的编辑工具进行修改。
潜在空间插入与损失评估：将生成或编辑的图像编码到潜在空间中，并计算其重构损失，使用均方误差（MSE）作为评估标准。
数据变换与动态λ选择：引入Box-Cox变换以优化损失函数，并通过最大似然估计、偏度最小化和峰度最小化等策略选择最优的λ值，以适应不同的编辑模式和分布特征。
通过这些步骤，LAMBDATRACER能够有效地处理复杂的图像编辑历史，并在开放环境中保持高效的归属性能。
在实验部分，研究者使用了多种生成和编辑模型，包括StableDiffusion和InstructPix2Pix等，构建了一个包含7200张图像的对抗编辑数据集（AE-Dataset）。实验中，作者将LAMBDATRACER与基线方法LATENTTRACER进行了比较，重点评估其在不同编辑迭代次数下的检测性能。评估指标包括精确率、召回率和F1分数，结果显示LAMBDATRACER在识别被操控图像方面表现出色，尤其是在多次编辑的情况下，显著提高了检测的准确性和鲁棒性。通过消融研究，作者还验证了Box-Cox变换在提升潜在空间可分离性方面的重要性。
在LAMBDATRACER的工作流程中，首先我们生成一张图像，然后通过用户输入的文本来进行编辑，比如添加特效或调整风格。接下来，我们将这张图像转化为一种特殊的数学表示（潜在空间），并计算它与原始图像之间的差异。为了确保我们能够准确区分原始图像和经过多次编辑的图像，我们使用了一种叫做Box-Cox变换的技术，这种技术可以帮助我们调整图像的特征，使得即使经过多次修改，图像的身份依然清晰可辨。通过这种方式，LAMBDATRACER能够在各种复杂的编辑场景中，帮助我们追踪图像的真实来源，从而保护创作者的版权。

2025-02-07｜UTAustin,NVIDIA｜⭐️🟡
http ://arxiv.org/abs/2502.05178v1https ://nvlabs.github.io/QLIP/
在本研究中，我们提出了一种新的视觉标记化方法，称为量化语言-图像预训练（QLIP），旨在统一处理多模态理解和生成任务。QLIP结合了先进的重建质量和零样本图像理解能力，通过训练一种基于二元球面量化的自编码器，采用重建和语言-图像对齐目标。我们首次展示了这两种目标可以有效结合，而不必相互对立。通过动态平衡训练过程中的两种损失项，我们的两阶段训练管道成功地将图像-语言预训练所需的大批量要求与重建目标所带来的内存瓶颈相结合。最终，QLIP展示了在多模态理解和文本条件图像生成方面的有效性，能够以一个模型处理多种任务，从而开辟了统一多模态理解与生成的机会。
QLIP的设计涉及以下几个关键步骤：
视觉标记化：我们采用了一种基于二元球面量化（BSQ）的自编码器，旨在将输入图像压缩为一组离散的视觉标记。这一过程包括视觉编码器、量化瓶颈和视觉解码器三个组件。
损失函数平衡：在训练过程中，重建损失和对齐损失之间的平衡至关重要。我们引入了一种自动加权机制，根据它们的后验损失值动态调整损失项的权重，以优化训练效果。
两阶段训练策略：第一阶段结合对齐损失和均方误差（MSE）损失进行训练，使用内存高效的Transformer架构。第二阶段冻结视觉编码器，仅优化量化器和解码器，从而降低内存使用并提高训练效率。
通过上述方法，QLIP能够在理解和生成任务上表现出色，并有效地将视觉和文本信息结合在一起。
在实验部分，我们使用多个数据集来验证QLIP的有效性，包括DataComp-1B、LAION-COCO和ImageNet等。我们评估了QLIP在视觉理解和重建任务上的表现，使用零样本准确率、重建FID、PSNR和SSIM等指标。结果表明，QLIP在零样本分类精度上与CLIP基线相当，同时在图像重建质量上也表现良好。此外，在文本条件图像生成任务中，QLIP生成的图像在FID评分和文本-图像对齐方面优于传统的方法。最终，QLIP作为一个统一的多模态模型，能够处理文本生成、图像到文本和文本到图像等多种任务。
QLIP的核心方法可以理解为一个高效的图像处理系统，它将图像转换成一组简化的视觉标记，就像把一幅复杂的画作变成几个简单的图形。这个过程分为几个步骤：首先，系统会分析图像并提取重要的信息；然后，它会将这些信息压缩成易于处理的标记；接下来，系统会通过平衡不同的学习目标，确保既能准确理解图像内容，又能保持图像的细节。最后，QLIP通过两次训练，逐步提高处理的效率和效果，最终让模型能够灵活地完成多种任务，比如根据文字描述生成图像，或从图像中提取信息并生成相应的文字。这样，QLIP不仅可以帮助计算机更好地理解图像和文字之间的关系，还能在实际应用中提供更高质量的生成结果。
2025-02-04｜ULuxembourg｜⭐️
http ://arxiv.org/abs/2502.04353v1https ://cognartive.github.io/
本研究探讨了如何利用大型语言模型（LLMs）和多模态模型（MLLMs）来自动化艺术分析，揭示艺术作品中的美学元素。艺术作为一种普遍语言，能够传达多重含义和复杂的情感，传统的艺术分析往往侧重于视觉元素如颜色、线条和形状，而忽视了更深层次的社会文化背景和艺术家创作的动机。随着AI技术的发展，尤其是视觉能力强大的LLMs的出现，艺术分析的方式得到了根本性的改变。研究者们通过结合计算机视觉、机器学习和自然语言处理等技术，能够从数字图像中提取出有意义的信息，从而更深入地理解艺术作品的技术和表现特征。
本研究采用的主要方法包括以下几个步骤：
数据集构建：研究团队从WikiArt收集了超过15,000幅来自23位著名艺术家的数字艺术作品，涵盖了广泛的艺术风格和历史时期。
分析框架：使用GPT-4V模型，研究者设计了一个包含八个问题的API请求，旨在从技术和概念两个角度对艺术作品进行评估。这些问题涉及形状、光线、材料、色彩等多个方面。
结果处理：通过GPT-4和Gemini2.0对模型返回的结果进行清理和处理，提取出定量和定性指标，以便进行更深入的分析。
相似度计算：最终，通过计算艺术风格与分析结果之间的余弦相似度，评估分析的准确性和有效性。
实验部分主要集中在艺术作品的技术和表现特征的分析上。研究者们对收集到的艺术作品进行了以下几方面的考察：
形状和规模：分析不同艺术家在作品中使用的形状类型及其变化趋势，发现自然形状在各个时期都占据主导地位，而几何形状在近年逐渐受到重视。
色彩分布：研究色彩的使用与情感主题之间的关系，发现某些色彩在特定历史时期中更为突出，且色彩的情感表达随着时间的推移而变化。
光影与对比：探讨光线和对比在艺术作品中的运用，发现艺术家们利用光影效果来增强作品的深度和情感表达。
运动与材料：分析艺术作品中运动的表现方式，发现现代艺术作品趋向于更抽象的运动表现，而材料的使用上，油画和帆布仍然是主流。
在这个研究中，科学家们使用了一种新的方法来分析艺术作品，就像用放大镜观察细节一样。他们从一个大型艺术数据库中收集了大量的艺术作品，并利用一种叫做GPT-4V的智能程序来帮助分析这些作品。首先，他们设计了一些问题，像是“这幅画使用了什么颜色？”或者“艺术家用了什么材料？”然后，把这些问题和每幅画的数字图像一起输入到智能程序中。接着，程序会给出答案，科学家们会把这些答案整理起来，看看它们是否能准确反映出艺术作品的风格和情感。最后，他们还计算了这些答案和已有的艺术风格描述之间的相似度，以确保他们的方法是有效的。这种方法不仅能帮助我们更好地理解艺术作品，还能让艺术分析变得更加高效和客观。
2025-02-07｜Microsoft,StonyBrookU｜ICLR2025｜⭐️🟡
http ://arxiv.org/abs/2502.05153v1https ://roar-ai.github.io/hummingbird
Hummingbird是一种新型的图像生成模型，旨在解决现有扩散模型在场景感知任务中的局限性，特别是在视觉问答（VQA）和人机交互（HOI）推理任务中。传统的图像生成方法在生成图像时往往无法准确保留与参考图像及其伴随文本指导相关的场景属性。Hummingbird通过引入一种多模态上下文评估器，能够在生成图像时同时优化全局语义和细粒度一致性奖励，从而确保生成的图像在保持多样性的同时，忠实地保留场景属性。这一模型不仅在生成图像的多样性和保真度上表现出色，还为复杂视觉任务提供了一个新的基准评估框架。
Hummingbird的核心方法包括以下几个步骤：
多模态上下文输入：模型接收一个包含参考图像和文本指导的多模态上下文，确保生成的图像与这些输入保持一致。
上下文描述生成：通过多模态大型语言模型（MLLM），Hummingbird将文本指导转换为上下文描述，提取出需关注的场景属性。
图像和文本嵌入：使用CLIP模型分别对参考图像和上下文描述进行编码，生成图像和文本的嵌入表示。
生成图像：将图像嵌入和文本嵌入输入到扩散模型的去噪器中，生成新的图像。
奖励机制：引入全局语义奖励和细粒度一致性奖励，以优化生成图像的保真度和多样性，确保生成的图像能够忠实反映输入的场景属性。
在实验中，Hummingbird被评估于多个基准数据集，包括MME感知和BongardHOI数据集。实验结果显示，Hummingbird在生成图像的保真度和多样性上均优于现有技术。通过与其他最先进的图像生成方法进行比较，Hummingbird在VQA和HOI推理任务中显著提高了准确率，证明了其在场景感知任务中的有效性。此外，Hummingbird还在图像分类任务中表现出色，展示了其在不同视觉任务中的广泛适用性。
Hummingbird的工作原理可以简单理解为一个智能的图像生成助手。首先，它接受一个参考图像和一些文字说明，比如“这个图里有多少个人？”接着，它通过一个强大的语言模型来理解这些文字说明，并提取出需要关注的场景信息。然后，Hummingbird会把参考图像和这些信息转化为数字形式，输入到一个图像生成模型中。这个模型会生成一张新图像，同时尽量保持与原图的不同之处，比如改变某些物体的颜色或位置，但仍然确保这些变化符合文字说明中的要求。最后，Hummingbird会根据生成的图像与文字说明之间的一致性进行优化，确保生成的图像不仅看起来新颖，而且准确反映了场景中的关键细节。这样，Hummingbird就能在复杂的视觉任务中提供高质量的图像输出。
2025-02-07｜CISPA,UToronto,VectorInstitute｜⭐️
http ://arxiv.org/abs/2502.05066v2https ://github.com/sprintml/ToxicBench
本文探讨了当今先进的视觉生成模型（如扩散模型和视觉自回归模型）在生成图像时嵌入不当文本（NSFW文本）的问题。尽管以往的研究成功地减少了图像中的不当视觉内容，但生成的图像中可能包含的冒犯性文本（如侮辱、种族歧视和性暗示词汇）却被忽视。这种情况不仅影响用户体验，还可能引发伦理问题。研究表明，当前所有主流的生成模型都容易产生此类不当文本，而现有的缓解措施在处理这些嵌入文本时效果不佳，甚至可能降低无害文本的生成质量。为此，作者提出了一种新的安全微调方法，并创建了ToxicBench，一个开放源代码基准，用于评估生成模型中NSFW文本的生成能力。
为了应对生成模型中NSFW文本的问题，本文提出了一种安全微调的方法，主要步骤包括：
文本编码器的安全微调：针对主要的扩散模型架构，选择CLIP文本编码器进行微调。
定制数据集：构建一个包含NSFW词汇及其语法相似的无害替代词的数据集，以训练文本编码器，减少不当文本的生成。
损失函数设计：定义一个损失函数，使得NSFW文本的嵌入能被映射到相应的无害词汇，同时保持生成图像和文本的整体质量。
评估机制：为了验证微调效果，使用一系列指标评估生成文本的NSFW程度和图像质量，确保无害文本生成的能力不受损害。
在实验部分，作者使用ToxicBench基准对不同生成模型的NSFW文本生成能力进行了评估。具体步骤包括：
数据集构建：使用218个不同的提示模板和2954个NSFW词汇，创建了73466个训练数据点和21800个测试数据点。
模型评估：对比现有的NSFW检测方法和作者提出的微调方法，观察其在生成NSFW文本和无害文本时的表现。
性能指标：使用多个指标（如F1分数、KID、Levenshtein距离等）来评估模型在生成无害文本和控制NSFW文本方面的效果。
结果分析：实验结果表明，作者提出的安全微调方法显著降低了NSFW文本的生成，同时保持了无害文本生成的质量，优于现有的基准方法。
在这项研究中，作者主要是想解决一个新问题：当我们使用现代计算机生成图像时，往往会发现这些图像中可能包含一些冒犯性或不当的文字。为了应对这个问题，研究者们选择了一个重要的工具——文本编码器，进行了一种“安全微调”的训练。这个过程就像给计算机教导如何用更合适的词汇来表达，而不是使用那些可能让人不舒服的词。为此，他们创建了一个包含不当词汇和它们的无害替代品的数据集，确保计算机在生成图像时能够避免使用这些不当的词。最后，通过一系列测试和评估，研究者们发现这种新方法不仅有效减少了不当文本的生成，还保持了无害文本的质量。这项研究的目标是让未来的图像生成技术更加安全和可靠。
2025-02-07｜ShanghaiAILab,NJUST,CUHK,THU,KIT｜⭐️🟡
http ://arxiv.org/abs/2502.05034v1
本研究提出了一种名为MindAligner的框架，旨在通过有限的功能性磁共振成像（fMRI）数据实现跨主体的视觉解码。脑解码的目标是从人类的脑活动中重建他们所感知的视觉刺激，这对理解大脑的感知机制至关重要。然而，现有的方法往往局限于单一主体的范式，导致个体间的泛化能力不足，尤其在脑部差异显著的情况下。MindAligner通过引入显式的脑功能对齐机制，克服了这些挑战。其核心在于学习一个脑转移矩阵（BTM），将新主体的脑信号映射到已知主体的信号，从而实现跨主体的视觉解码和功能分析。
MindAligner的方法主要包括以下几个关键组成部分：
脑转移矩阵（BTM）：该矩阵的作用是将新主体的fMRI信号投影到已知主体的信号空间。通过对矩阵进行优化，可以实现高效的跨主体对齐。
脑功能对齐模块（BFA）：此模块在训练阶段使用，旨在通过多层次的对齐损失实现软性跨主体映射。由于缺乏严格配对的fMRI数据，BFA采用了跨刺激神经映射技术，以便在不同视觉刺激下进行信号转换。
多层次脑对齐损失：该损失结合了信号级重建损失和潜在对齐损失，确保生成的信号与真实信号之间的一致性，并且在潜在空间中加强了功能对齐的精度。
推理阶段：在推理过程中，仅使用训练好的BTM进行功能映射，将生成的fMRI信号直接输入预训练的解码模型中进行视觉重建。
实验部分展示了MindAligner在视觉解码方面的有效性，特别是在数据有限的情况下。研究使用了自然场景数据集（NSD），并通过定量和定性方式评估了模型的性能。结果表明，MindAligner在不同指标上均优于现有的最先进方法，尤其在脑信号的重建质量上表现突出。此外，研究还通过交叉主体的功能可视化分析，揭示了大脑区域间的变异性，提供了关于视觉皮层和高层视觉区域的有价值神经科学见解。
在MindAligner的研究中，科学家们想要解决一个问题：如何从不同人的脑部扫描中提取出相似的视觉信息。为了做到这一点，他们设计了一种新的方法，首先创建一个“脑转移矩阵”，就像一个翻译器，可以把一个人的脑信号转换成另一个人的信号。接着，他们使用一个叫做“脑功能对齐模块”的工具，帮助这个翻译器学习如何在不同的视觉刺激下更好地工作。通过这种方式，即使只有很少的数据，MindAligner也能有效地理解和重建人们看到的图像。最后，实验结果显示，MindAligner在视觉解码方面表现优异，能够揭示不同人之间大脑活动的共性和差异。
2025-02-06｜UniversityofNotreDame｜⭐️
http ://arxiv.org/abs/2502.04391v1
面部解析是计算机视觉中的一项基本任务，涉及对面部特征如眼睛、鼻子和嘴巴进行细致的分割。这项技术在身份验证、面部编辑和可控图像合成等应用中至关重要。然而，现有的面部解析模型往往缺乏公平性和鲁棒性，导致在不同人群中的分割表现不均，以及在噪声、遮挡和领域转移等情况下的错误。本文提出了一种多目标学习框架，旨在同时优化准确性、公平性和鲁棒性。通过引入一种动态调整目标重要性的同伦损失函数，我们的方法能够在训练过程中灵活平衡这些目标，从而在生成模型中提升面部合成的质量和一致性。
我们的多目标学习框架包括以下几个关键步骤：
问题定义：构建一个数据集，其中每个面部图像都配有相应的分割掩码，目标是训练一个函数来预测这些掩码，同时优化准确性、公平性和鲁棒性。
数据准备：使用CelebAMask-HQ数据集，按训练、验证和测试集进行划分，并提取人口统计特征以计算公平性指标。
模型架构：采用U-Net架构，结合ResNet-34编码器，输出19个通道以对应不同的面部区域，确保高效的计算和准确的分割。
多目标训练：通过优化准确性损失、鲁棒性损失和公平性损失的加权和来训练模型，动态调整损失权重以适应不同训练阶段。
生成模型集成：将训练好的U-Net与Pix2PixHDGAN和ControlNet集成，利用分割图来生成高质量的合成面部图像。
我们通过将多目标和单目标U-Net模型集成到GAN基础的面部合成管道中，评估其对生成质量的影响。实验包括对不同人群的公平性和鲁棒性评估，使用指标如平均交并比（mIoU）和FrechetInceptionDistance（FID）。结果显示，多目标模型在多个条件下的表现优于单目标模型，能够生成更具真实感和一致性的面部图像。此外，我们还进行了初步实验，探讨分割质量如何影响基于扩散的生成模型（ControlNet）的性能，验证了我们方法的有效性。
在我们的方法中，我们首先定义了一个任务，即从每个人的面部图像中提取出眼睛、鼻子等特征。为了做到这一点，我们使用了一个叫做U-Net的模型，这是一种非常擅长图像分割的神经网络。我们将每张面部图像与其对应的分割图（显示特征位置的图）配对，并通过训练模型来提高它的准确性。为了确保我们的模型在不同人群中表现公平，我们还引入了一个“公平性”指标，这样模型在不同人群中都能有相似的表现。此外，为了让模型在面对噪声或遮挡时仍然能够准确分割，我们增加了一个“鲁棒性”指标。通过这些措施，我们的模型不仅能准确分割面部特征，还能在生成的面部图像中保持公平和一致性。

2025-02-07｜CASIA,KAUST｜⭐️⭐️
http ://arxiv.org/abs/2502.04762v1
本文提出了一种新颖的树木生成方法，名为HourglassTree，结合了高效的数据结构、变换器架构和适当的令牌排序策略，以实现对树木结构的高质量生成。树木生成在计算机图形学和仿真领域中具有重要意义，传统方法往往依赖于程序化技术或规则基础的模拟，难以灵活适应不同树种和环境。HourglassTree通过引入深度生成模型，特别是自回归变换器，克服了上述限制，能够更好地捕捉树木的层级结构和生长特性。该方法不仅支持静态树的生成，还可以实现动态生长树的模拟，具有广泛的应用前景，包括图像到树、点云到树的转换等。
HourglassTree的核心在于其创新的树木表示和Hourglass变换器架构。具体方法包括：
树木表示：将树木视为一组分支，每个分支通过两个端点和对应的半径进行表示。这种表示方式减少了生成模型所需的参数数量，提高了计算效率。
令牌化：将连续参数离散化为令牌，以便于自回归变换器的处理。每个连续值被量化为256个离散标签，确保了模型在训练时的高效性。
分支排序：采用深度优先搜索（DFS）或广度优先搜索（BFS）策略对分支进行排序，以保持树的层级依赖性，确保生成结构的连贯性。
Hourglass变换器架构：该架构结合了下采样和上采样阶段，通过跳跃连接实现多分辨率处理，从而有效捕捉树木数据中的全局和局部模式，提高了生成效率。
在实验部分，研究者对HourglassTree进行了多项测试，包括无条件生成和条件生成。通过不同的令牌排序策略（如DFS和BFS）评估生成质量，结果显示DFS方法在多个指标上表现最佳。此外，实验还探讨了模型在不同数据集规模下的可扩展性，结果表明随着训练数据量的增加，模型生成的树木质量显著提高。研究还包括对不同树种的生成能力测试，验证了模型在捕捉树木细节和多样性方面的有效性。最终，实验结果表明HourglassTree在生成速度、内存消耗和生成质量上都优于传统方法。
HourglassTree是一种智能的树木生成工具，想象一下，它就像一个设计师，能够根据不同的需求和环境来创造各种独特的树木。首先，它把树木看作是由许多分支组成的，每个分支都有两个端点和一个宽度，这样可以节省大量的计算资源。然后，它通过把树木的特征转换成一系列简单的数字（我们称之为“令牌”），使得计算机更容易理解和处理这些信息。接着，它采用了一种聪明的排序方法，确保树木的结构是连贯的，就像搭积木一样，先搭好底部再搭上面的部分。最后，这个工具不仅能生成静态的树木，还能模拟树木的生长过程，适应不同的场景和需求，帮助我们更好地理解和应用树木生成技术。
2025-02-07｜NYCU,NVIDIA
http ://arxiv.org/abs/2502.05176v1https ://kkennethwu.github.io/aurafusion360/
AuraFusion360是一种创新的基于参考图像的360°无界场景修复方法，旨在解决在三维场景中去除物体和填补空洞时的视图一致性和几何准确性问题。该方法结合了高效的高斯散射表示和生成模型的能力，能够在复杂的360°场景中有效去除物体并填补缺失区域。AuraFusion360引入了三项关键技术：深度感知的未见区域生成、适应性引导深度扩散，以及基于SDEdit的细节增强，确保在多个视角下保持一致性和真实感。此外，研究团队还构建了360-USID数据集，为360°无界场景修复提供了全面的基准评估。
AuraFusion360的方法流程主要包括以下三个步骤：
深度感知未见区域生成：该步骤通过深度扭曲技术自动生成每个视角的未见区域掩码，确保精确识别被遮挡的区域。使用深度图和背景信息，系统能够区分可见区域和需要修复的未见区域。
参考视图初始高斯对齐：在去除物体后，选择一张参考图像，并通过适应性引导深度扩散技术将其投影到三维空间。该过程确保高斯初始点的准确位置，从而为后续的细节增强提供结构性指导。
基于SDEdit的细节增强：在初始化高斯后，使用DDIM反演生成结构化噪声，以保持参考视图的结构信息，同时对细节进行增强。这一过程确保了在多个视角下的纹理一致性和几何准确性。
在实验部分，AuraFusion360的方法在360-USID数据集上进行了评估。该数据集包含七个场景，提供了丰富的训练和测试视角。实验结果显示，AuraFusion360在PSNR和LPIPS指标上均优于现有的最先进方法，特别是在复杂几何和大视角变化的场景中。通过对比不同方法的性能，AuraFusion360展示了在360°无界场景修复中保持视图一致性和细节保留的卓越能力。此外，研究团队还进行了消融实验，以验证每个组件对最终效果的贡献，进一步证明了其方法的有效性。
AuraFusion360的工作原理可以简单理解为一个三步走的过程。首先，系统会识别出哪些区域是被其他物体遮挡的，这个过程叫做生成未见区域掩码。接着，选择一张参考图片，系统会将这张图片的内容准确地投影到三维空间中，以便填补那些被遮挡的区域。最后，系统会通过一种特殊的技术来增强细节，确保修复后的区域与周围环境看起来一致且自然。通过这三步，AuraFusion360能够在复杂的360°场景中有效去除物体，并填补空缺，创造出更真实的视觉效果。
2025-02-07｜MetaRealityLabs,UCBerkeley,UToronto
http ://arxiv.org/abs/2502.05175v1https ://ethanweber.me/fillerbuster/
Fillerbuster是一种新颖的多视角场景补全方法，旨在解决在日常捕捉中由于拍摄角度不足而导致的3D场景信息缺失问题。传统的3D重建技术通常依赖于完整的视角数据，而Fillerbuster则通过利用大量的输入图像，联合建模图像和相机位姿，来有效填补这些缺失区域。该方法特别适合于快速拍摄的随意捕捉场景，这些场景往往只有稀疏的图像数据。Fillerbuster的核心在于其生成模型，能够在缺乏完整相机参数的情况下，预测缺失的图像内容，并生成新的视图。这一方法的优势在于其能够处理多种输入和输出框架，使得生成的场景更加完整和连贯。
Fillerbuster的工作原理主要包括以下几个步骤：
联合建模：该模型使用多视角感知的扩散模型，通过对已知图像和相机位姿的联合建模，来推测缺失的内容。
输入处理：模型能够处理多达50张输入图像，用户可以指定哪些内容已知，哪些内容未知，甚至可以处理部分完整的图像。
潜在扩散变换器：Fillerbuster采用了一种潜在扩散变换器架构，能够在潜在空间中进行图像和相机位姿的去噪，生成新的视图。
掩码策略：在训练过程中，模型使用混合掩码策略，随机选择图像中的已知和未知区域，以增强模型的学习能力。
序列生成：模型能够在推理过程中生成多个新视图，支持长序列的生成，以提高场景的完整性。
在实验中，Fillerbuster在多个数据集上进行测试，特别是针对日常捕捉场景的补全效果。实验结果表明，Fillerbuster在生成新视图时，能够显著提升图像的质量和一致性。与其他方法相比，Fillerbuster不仅能够处理已知和未知的图像，还能在缺失内容的情况下生成连贯的场景。此外，Fillerbuster在“未标定场景补全”任务中表现优异，能够从一组未标定的图像中推断出相机位姿并生成新视图。通过与其他基线方法的比较，Fillerbuster在多个指标上均表现出色，验证了其在实际应用中的有效性和潜力。
Fillerbuster的工作方式可以简单理解为一个“智能拼图”游戏。当你用手机快速拍摄一个场景时，可能会遗漏一些角落或细节。Fillerbuster就像一个聪明的助手，它能根据你已拍摄的照片，推测出那些你没有拍到的部分。它会分析这些照片中的信息，判断哪些地方是空缺的，然后生成新的图像来填补这些空白。这个过程就像是用很多小拼图块拼出一幅完整的画面。通过这种方式，Fillerbuster让你的捕捉场景看起来更完整、更真实，仿佛你在现场一样。
2025-02-07｜HKUST,SYSU,USC｜ICLR2025
http ://arxiv.org/abs/2502.04734v1http ://www.chenyingshu.com/sc-omnigs/
在360度图像的收集和处理领域，SC-OmniGS（自校准全向高斯点云系统）提出了一种新颖的方法，旨在快速且准确地重建全向辐射场。传统的辐射场技术在处理360度图像时面临诸多挑战，尤其是在相机参数的准确性上。SC-OmniGS通过将360度图像视为一个整体球面，而非传统的立方体映射，开发了一种数学框架，能够直接进行全向相机姿态校准及3D高斯优化。此外，研究还引入了一种可微分的全向相机模型，以纠正现实世界数据中的畸变，从而提高性能。通过最小化加权球面光度损失，系统能够联合优化全向相机的内参、外参和3D高斯，显著提升了在复杂场景下的重建质量。
SC-OmniGS的核心方法包括以下几个步骤：
全向高斯点云表示：采用3D-GS技术，通过一组具有明确位置、协方差和球面谐波系数的3D高斯来表示辐射场，以加速优化过程。
可微分光栅化：开发了一种可微分的光栅化器，将3D高斯点投影到单位球面上，并通过扩展到二维图像进行渲染。这一过程允许在渲染过程中计算全向相机姿态的梯度，以优化噪声相机姿态。
相机模型优化：提出了一种新的全向相机模型，能够通过学习复杂的畸变模式来纠正输入图像的畸变。这一模型通过可微分优化实现，从而提供无畸变的全向图像。
联合优化：通过最小化球面光度损失和各项正则化损失，联合优化3D高斯、相机姿态和相机模型，确保优化过程的稳定性和高效性。
为了验证SC-OmniGS的有效性，研究者进行了大量实验，包括在合成数据集OmniBlender和真实世界360Roam数据集上的测试。实验结果表明，SC-OmniGS在不同的相机姿态条件下均能保持高质量的辐射场重建，尤其是在没有相机姿态先验信息的情况下。与现有的最先进技术（如BARF和L2G-NeRF）相比，SC-OmniGS在PSNR、SSIM和LPIPS等指标上表现出色。此外，实验还展示了SC-OmniGS在处理真实世界数据时的鲁棒性，能够有效校准相机参数并减少畸变影响，提升重建质量。
SC-OmniGS的工作方法可以简单理解为一个高效的“相机和图像处理助手”。首先，它将360度的图像视为一个完整的球体，而不是将其分成多个小块，这样可以更全面地捕捉场景。接下来，系统使用一种叫做“3D高斯”的技术，这就像在图像中放置许多小点，帮助我们更快速地重建图像。为了确保图像没有失真，SC-OmniGS还设计了一种智能的相机模型，能够自动纠正图像中的畸变。最后，通过不断调整这些小点和相机位置，SC-OmniGS能够在没有任何先验知识的情况下，生成高质量的图像。这种方法不仅高效，而且在处理复杂场景时表现得非常出色。
2025-02-07｜UNC,DartmouthCollege,RiceU｜⭐️🟡
http ://arxiv.org/abs/2502.04630v1
本研究提出了一种新颖的传感器融合技术，旨在实现高速动态三维场景的重建。通过结合RGB相机、事件相机和深度相机的数据，我们能够在快速变化的场景中捕捉到高质量的三维图像。这种方法利用了不同成像模式的互补优势，RGB相机提供丰富的颜色信息，事件相机以微秒级的分辨率捕捉快速场景变化，而深度相机则确保了三维空间的准确性。研究表明，该方法在低光照、高动态范围和快速运动等挑战性条件下，显著提升了场景重建的细节和准确性，为计算机图形学、机器人技术以及其他相关领域提供了新的解决方案。
本研究的方法主要包括以下几个步骤：
传感器融合框架：利用RGB、事件和深度相机的组合捕获场景数据，形成一个多模态的输入数据集。
可变形3D高斯模型：通过训练可变形的3D高斯模型来动态表示场景。这些高斯模型能够适应场景的变化，并在时间上保持一致性。
高斯点云渲染：将3D高斯模型投影到二维相机平面，利用高斯的均值和协方差矩阵进行渲染，并通过最小化重建图像与实际观测图像之间的差异来优化参数。
事件监督学习：在训练过程中，结合事件相机捕获的时间变化信息，以提高模型在动态场景中的表现。这一过程通过优化时间窗口内的亮度变化，确保模型能够准确捕捉快速运动的细节。
为验证所提出方法的有效性，研究团队进行了多组实验，涵盖合成数据和真实世界场景。合成实验使用了多种动态场景，例如快速拍打的蜂鸟、旋转的球体和快速解体的雕塑。通过对比不同条件下的表现，结果显示该方法在重建质量和结构准确性方面均优于现有技术。此外，真实世界实验则在不同光照条件和运动速度下进行，结果表明该方法在复杂环境中表现出色，成功捕捉到细微的动态细节。实验结果进一步证明了传感器融合对提升高速动态三维成像的关键作用。
在这项研究中，科学家们开发了一种新方法，可以帮助我们更好地捕捉和重建快速移动的三维场景。想象一下，当你拍摄一只快速飞翔的鸟时，传统相机可能无法清晰地记录下它的每一个细节。但通过结合三种不同类型的相机——一种记录颜色的相机（RGB相机）、一种能捕捉极快动作的相机（事件相机）和一种能测量距离的相机（深度相机），研究人员能够更全面地捕捉到鸟的飞行轨迹和形态。这种方法就像把三种不同的视角结合在一起，形成一个立体的画面。同时，他们还使用了一种特殊的数学模型，帮助这些相机更好地理解场景的变化，从而在各种光线和运动条件下，生成清晰、详细的图像。这使得我们在观看这些动态场景时，能够感受到更真实的视觉体验。

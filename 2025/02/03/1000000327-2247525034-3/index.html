288-A3｜图像扩散用于可扩展3D高斯生成，逆程序化点云到建筑抽象，单图重建点云；轻量级物理感知可控动作生成，动作文本联合生成

2025-01-28｜PKU,ByteDance｜ICLR2025｜⭐️⭐️
http ://arxiv.org/abs/2501.16764v1https ://chenguolin.github.io/projects/DiffSplat
在3D内容生成领域，尤其是从文本或单幅图像生成3D模型方面，研究者们面临着高质量3D数据集稀缺和2D多视图生成不一致性的问题。为了解决这些挑战，研究团队提出了DIFFSPLAT，这是一种新颖的3D生成框架，能够有效利用大规模的2D图像扩散模型，直接生成3D高斯斑点，并保持3D一致性。DIFFSPLAT的创新之处在于其通过轻量级重建模型快速生成多视图高斯斑点网格，从而支持可扩展的数据集策划。此外，该框架还引入了3D渲染损失，以增强不同视图之间的3D一致性。通过广泛的实验，DIFFSPLAT在文本和图像条件生成任务中表现出优越性，并通过消融研究验证了设计选择的有效性。
DIFFSPLAT的核心方法分为三个部分：
数据策划：结构化斑点重建利用结构化的多视图高斯斑点网格来表示3D对象。通过一个小型网络，能够在不到0.1秒内从多视图图像中回归每个像素的斑点。网络训练采用渲染损失，以确保生成的高斯原语与真实图像的相似性。
斑点潜在表示为了将高斯斑点网格编码到图像扩散潜在空间中，采用了变分自编码器（VAE）进行独立编码。该过程同时使用重建损失和渲染损失，以提高编码质量。经过训练后，生成的斑点潜在表示将用于后续的生成模型。
DIFFSPLAT生成模型该模型能够根据文本提示或单视图图像生成3D内容。通过两种不同的拼接方式（视图拼接和空间拼接），将多视图斑点潜在表示整合到生成网络中。训练目标包括常规的扩散损失和额外的渲染损失，以确保生成内容的3D一致性。
在实验部分，研究团队在G-Objaverse数据集上训练了模型，该数据集包含来自265K个3D对象的38个不同视角的高质量图像。为了评估文本条件生成的性能，使用了来自T3Bench的300个文本提示，并通过CLIP相似度分数和图像奖励等指标进行定量比较。在图像条件生成任务中，DIFFSPLAT展示了出色的3D内容生成能力，尤其是在复杂场景中，生成的3D对象与输入图像的几何结构保持一致。此外，研究还进行了消融研究，分析了不同设计选择对生成效果的影响，验证了DIFFSPLAT在3D生成中的有效性。
DIFFSPLAT的工作原理可以简单理解为一个三步法。首先，它通过一个小型的网络快速从多张图片中提取出3D对象的结构信息，这就像把不同角度的照片拼接成一个3D模型。接着，这些结构信息会被转换成一种可以被计算机理解的“潜在表示”，这就像把复杂的信息压缩成一个更简单的格式，方便后续处理。最后，DIFFSPLAT利用这些潜在表示，根据用户的文本描述或单张图片生成全新的3D对象，确保生成的内容在不同视角下看起来都很自然和一致。整体来说，DIFFSPLAT通过结合2D图像的丰富信息和3D生成的技术，提升了3D内容生成的质量和效率。
2025-01-28｜MPI-IS,GoogleDeepMind,Google｜⭐️⭐️
http ://arxiv.org/abs/2501.17044v2
本研究提出了一种通过反转程序化建筑模型来生成建筑的三维抽象表示的方法。该方法利用变换器模型（transformer）对程序化建筑的几何特征和结构进行学习，能够从点云数据中推断出相应的建筑抽象。这一过程首先通过构建一个包含多种技术组件的数据集来实现，该数据集将程序化建筑模型与相应的点云模拟数据配对。通过这种方式，研究者们能够保留程序化模型在游戏和动画中的优良特性，如高效渲染和对规则性、对称性的强先验知识。最终，该方法在几何重建和结构一致性推断方面表现出色，尽管仍面临程序化模型灵活性不足的挑战。
本研究的方法主要包括以下几个步骤：
数据集生成：利用程序化建筑模型生成341,721对建筑抽象和点云数据的配对。每个建筑抽象由多个建筑资产（如立面段）按照一定规则组合而成。
贝叶斯推理框架：我们定义了一个贝叶斯推理模型，通过训练一个推断模型来近似后验分布。该模型接受建筑点云作为输入，并推断出对应的参数化描述（抽象）。
模型架构：采用变换器的编码器-解码器结构，首先对输入的点云进行编码，然后通过语言变换器预测相应的抽象表示。该过程包括对点云的分块、特征嵌入以及通过交叉注意力机制整合全局信息。
优化目标：训练过程中使用最大似然估计，优化推断模型以实现高效和准确的建筑抽象推断。
在实验部分，研究者们评估了训练好的模型在持出子集上的表现，比较了推断出的建筑抽象与真实数据之间的结构属性和几何拟合度。结果显示，模型在多个结构变量（如楼层数、立面数等）上的准确率超过95%。此外，研究者们还对模型在不同点云噪声水平下的重建性能进行了测试，发现推断建筑的几何误差相对较小，且能够有效填补缺失信息，表现出较强的鲁棒性。尽管如此，模型在处理真实建筑数据时仍需进一步优化，以应对模拟与真实数据之间的域转移问题。
在这项研究中，科学家们开发了一种新方法，可以从建筑的点云数据中生成简化的三维模型。想象一下，建筑的点云就像是一个复杂的拼图，而他们的工作就是找到这个拼图背后的简单图案。首先，他们收集了很多建筑的虚拟数据，并将这些数据与实际的点云配对。接着，他们使用了一种叫做变换器的智能算法，它可以理解和处理这些数据。这个算法就像是一个聪明的助手，当给它一个建筑的点云时，它能很快推断出这个建筑的简化描述。最后，研究人员通过大量实验验证了这个方法的有效性，发现它在重建建筑的结构和外观方面非常准确。总之，这项技术可以帮助我们更好地理解和生成建筑设计，尤其是在没有大量手动标注数据的情况下。
2025-01-28｜XJTLU,ULiverpool,DKU｜⭐️
http ://arxiv.org/abs/2501.16737v1
本研究探讨了如何从单张图像中重建3D点云，提出了一种名为一致性扩散模型（ConsistencyDiffusionModel,CDM）的新方法。该模型旨在通过结合2D和3D先验知识，在贝叶斯框架下增强重建过程中的一致性。传统的3D重建方法通常依赖于多视角图像，但在只有单视角图像的情况下，重建效果往往不尽如人意。CDM通过引入从初始3D点云中提取的结构先验，作为约束条件来提高重建的一致性和准确性。此外，该模型还整合了来自输入图像的2D信息，以增强训练过程中的指导，从而在合成和真实世界的数据集上都取得了新的基准性能。
在方法部分，我们的CDM模型通过以下几个关键步骤实现高效的3D重建：
扩散模型基础：首先，CDM利用扩散模型的框架，通过逐步引入噪声来生成3D点云。模型从高斯噪声开始，并通过优化过程逐步调整点云。
3D结构先验的引入：我们提取初始3D点云的多视角结构信息，作为额外的3D先验。在训练过程中，这些先验被用作约束条件，帮助模型更好地对齐生成的点云与目标结构。
2D信息的整合：从输入图像中提取的2D信息被映射到3D点云上，作为辅助指导。这一过程确保了模型能够有效利用图像中的深度和轮廓信息，提高重建的质量。
损失函数的设计：我们设计了一种新的损失函数，通过最大化证据下界（ELBO）来增强模型的训练效果，确保重建过程中的一致性。
在实验部分，我们对CDM进行了广泛的评估，使用了合成数据集ShapeNet和真实世界数据集Co3D。我们与现有的PC2和BDM模型进行了比较，结果表明CDM在重建精度和一致性方面均表现出色。具体而言，我们使用了Chamfer距离和F1分数作为评估指标，结果显示CDM在多个类别上均超越了其他模型。此外，我们还进行了消融实验，以验证2D和3D先验的有效性，并探讨了不同先验组合对重建性能的影响。实验结果一致表明，CDM在合成和真实场景中均取得了最先进的性能。
在我们的研究中，我们提出了一种新方法，叫做一致性扩散模型（CDM），用于从一张图片中创建3D模型。想象一下，你有一张照片，CDM能够根据这张照片生成一个立体的物体，比如一个椅子或玩具。这个方法的关键在于它结合了两种信息：一是从已有3D模型中提取的结构信息，二是从照片中提取的深度和轮廓信息。通过这种方式，模型不仅能更准确地理解物体的形状，还能在生成3D模型时保持一致性。我们通过大量实验验证了这一方法的有效性，结果显示CDM在生成的3D模型的准确性和细节上都优于之前的方法。简单来说，CDM就像是给模型加上了“眼睛”，让它能更好地“看”到物体的真实形状。
2025-01-28｜UBC,DalhousieU｜⭐️🟡
http ://arxiv.org/abs/2501.16778v1
FlexMotion是一种新颖的框架，旨在生成轻量级、物理感知和可控的人体运动序列。该框架解决了现有方法在计算效率、物理真实感和空间可控性之间的权衡问题。FlexMotion利用一种轻量级的扩散模型，操作在潜在空间中，从而避免使用物理模拟器，确保训练过程快速高效。它采用了多模态预训练的Transformer编码器-解码器，整合了关节位置、接触力、关节激活和肌肉激活等信息，以确保生成运动的物理合理性。此外，FlexMotion还引入了一个插件式模块，允许对多种运动参数（如关节位置和肌肉激活）进行空间可控性调整，从而提升生成运动的现实性和可控性。
FlexMotion的核心方法包括三个主要组成部分：
物理感知多模态自编码器：该部分负责将运动的运动学和动力学特性映射到潜在空间，并通过物理约束确保生成运动的物理合理性。它利用Transformer架构处理多种运动模态，如关节位置、旋转和肌肉激活等。
潜在空间运动扩散模型：此模型在潜在空间中生成运动序列，基于输入文本提示进行条件生成。通过逐步去噪的过程，模型能够生成多样化且真实的运动序列。
空间可控性模块：该模块允许用户对生成的运动进行精细控制，包括关节位置、速度和肌肉激活等参数，从而确保生成运动符合特定的空间约束和动态需求。
在实验部分，FlexMotion在三个流行的数据集上进行了评估：HumanML3D、KIT-ML和Flag3D。通过数据增强，使用OpenSim工具集成了详细的肌肉激活、接触力等生物力学信息，以提升运动数据的真实感。评估指标包括自然性、文本相关性、多样性、物理合理性和空间控制精度等。实验结果表明，FlexMotion在生成运动的自然性和物理合理性方面表现优越，且显著提高了运动生成的可控性，超越了多种现有的最先进方法。
FlexMotion的工作原理可以简单理解为一个高科技的“运动生成器”。首先，它有一个“学习大脑”，这个大脑会分析不同运动的细节，比如关节的位置和肌肉的活动，然后把这些信息转化成一种“代码”来理解运动。接着，FlexMotion会根据输入的文字或指令，逐步生成运动，就像在给演员下达指令一样，确保动作既真实又符合物理规律。最后，它还允许用户对动作进行精细调整，比如改变关节的角度或肌肉的用力程度，就像导演在拍摄时对演员进行指导一样。这种方法使得生成的运动不仅生动而且科学，适用于动画、虚拟现实等多个领域。
2025-01-27｜UWashington｜⭐️
http ://arxiv.org/abs/2501.16551v1
本研究提出了PackDiT，这是一个创新的扩散模型，旨在实现人类运动和文本生成的联合任务。该模型的核心创新在于利用互相提示的机制，整合多个扩散变换器（DiTs），从而在不同模态之间实现无缝的生成。PackDiT能够执行多种生成任务，包括文本到运动、运动到文本、运动预测以及无条件生成等。通过在HumanML3D数据集上进行训练，PackDiT在文本到运动生成任务中达到了0.106的FID分数，显示出其在多模态生成方面的优越性能。此外，实验结果还表明，PackDiT在运动预测和其他中间任务中也表现出色，为多模态生成奠定了坚实的基础。
PackDiT的设计包括两个独立的扩散变换器模型：MotionDiT和TextDiT。其方法包括以下几个关键步骤：
无条件生成：在此阶段，模型利用运动和文本的输入生成相应的输出。每种模态的生成模型独立运行，确保了灵活性和可扩展性。
互相提示机制：通过在两个变换器之间引入互相提示的块，模型能够在生成过程中实现信息的交互，从而提高生成的准确性和一致性。
联合训练：在训练过程中，两个模型共同优化，确保特征的对齐和联合生成能力。训练分为多个阶段，包括无条件预训练、联合生成训练和任务特定的微调。
任务微调：在完成联合训练后，针对特定任务（如文本到运动或运动到文本）进行进一步的微调，以提升模型在特定任务上的表现。
在HumanML3D数据集上进行了一系列实验，以评估PackDiT在多个生成任务上的性能。实验结果显示，PackDiT在文本到运动生成任务中，达到了0.106的FID分数，显著优于现有的其他方法。此外，在运动预测和运动中间生成任务中，PackDiT同样表现出色，显示出其强大的生成能力。通过对比不同的超参数设置和训练策略，研究还发现互相提示的机制和无条件预训练阶段对模型的最终性能至关重要。整体而言，PackDiT在多模态生成任务中展现了卓越的灵活性和高效性。
PackDiT的工作原理可以简单理解为一个“双工”系统，分别负责生成运动和文本。首先，系统会在没有任何特定输入的情况下，独立生成运动和文本，这个过程称为无条件生成。接下来，系统通过一种称为“互相提示”的机制，让两个生成模型相互交流，确保生成的运动和文本能够相互匹配。这样，当你输入一个描述，比如“一个人走路”，系统就能生成相应的运动，表现出这个人的走路方式。最后，系统会根据特定的任务进行微调，以便在实际应用中提供更准确的生成结果。通过这种方式，PackDiT不仅能生成运动，还能理解和生成与之相关的文本，展现出强大的多模态生成能力。

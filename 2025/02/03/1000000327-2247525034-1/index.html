主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）发布日期｜2025年1月29日｜周三
2025-01-27｜QualcommAIResearch｜ICLR2025｜⭐️⭐️
http ://arxiv.org/abs/2501.16559v1
随着大型基础模型（LFMs）的普及，针对这些模型的高效微调方法需求日益增加。低秩适应（LoRA）作为一种参数高效的微调策略，能够在仅增加少量参数的情况下实现与全模型微调相当的性能。然而，当基础模型被替换时，所有相关的LoRA模块需要重新训练，这通常需要访问原始训练数据或生成足够代表性的合成数据，而这在隐私或许可问题上常常不可行。为了解决这一挑战，本文提出了一种新的适配器——跨模型低秩适应（LoRA-X），该方法允许在不同基础模型之间无训练地转移LoRA参数，消除了对原始或合成训练数据的需求。LoRA-X通过在源模型的子空间中操作适配器，确保适配器在转移过程中保留了有效性和可适应性。
LoRA-X的设计基于两个主要原则：首先，LoRA能够放大特定下游任务中关键特征的表现；其次，转移过程中只能依赖目标模型的预训练权重。为了实现有效的转移，LoRA-X将适配器限制在源模型的子空间内，这样可以确保适配器在转移到目标模型时保持兼容性。具体方法分为以下几个步骤：
适配器结构：通过奇异值分解（SVD）将权重更新表示为低秩矩阵的组合，确保适配器的更新保持在源模型的子空间。
权重投影：在源模型的适配器权重与目标模型的权重之间进行投影，以确保在转移过程中保留相似的子空间特征。
相似性度量：使用无权相似性度量来评估源模型和目标模型之间的模块相似性，从而确定适配器的可转移性。
转移成本计算：通过构建转移成本矩阵，量化从源模型到目标模型的适配器转移难度，以指导训练过程中的转移决策。
为了评估LoRA-X的有效性，本文进行了文本到图像生成的实验。实验分为两种场景：一种是“训练”场景，其中在特定基础模型上训练LoRA-X；另一种是“转移”场景，其中将训练好的LoRA-X从一个基础模型转移到另一个基础模型。实验中使用了多个公共数据集，如BlueFire和Paintings，测试了LoRA-X在不同目标模型上的表现。结果显示，转移后的LoRA-X在图像生成质量上与训练得到的LoRA-X相当，表明其在无训练条件下的有效转移能力。此外，通过对比分析不同模型之间的转移成本，进一步验证了LoRA-X在模型更新和替换过程中的适用性。
在这项研究中，LoRA-X就像是一个灵活的适配器，能够在不同的机器学习模型之间转移而不需要重新训练。想象一下，如果你有一台设备（源模型）已经安装了特定的功能（LoRA适配器），而你又想把这些功能转移到另一台新设备（目标模型）上，而不想重新设置所有的参数。LoRA-X通过保持功能在原设备的“工作空间”内，确保它能适应新设备的结构。具体来说，研究者们使用了一种叫做奇异值分解的方法来简化这个过程，确保在转移时不会丢失重要的信息。此外，他们还设计了一种相似性评分系统，帮助判断不同设备之间转移的难易程度。通过这些方法，LoRA-X能够有效地将已有的知识和功能转移到新模型上，大大减少了重新训练所需的时间和资源。
2025-01-28｜ETHZurich,ZMOAIInc.,ZJU,NTUS-Lab｜⭐️⭐️
http ://arxiv.org/abs/2501.17159v1https ://www.youtube.com/watch?v=By-y3IH6ejM
IC-Portrait是一种创新的个性化肖像生成框架，旨在解决现有模型在保持个体身份一致性方面的挑战。尽管现有的扩散模型在生成具有身份保留特征的图像方面展现出较大潜力，但在面对用户提供的多样化外观和照明条件时，个性化肖像生成依然面临困难。IC-Portrait通过将肖像生成过程重构为两个互补的子任务：照明感知拼接和视角一致适应，有效地应对了这些挑战。通过引入高比例遮罩自编码技术和合成的视角一致性数据集，该框架能够生成具有高度一致性和响应照明条件的个性化肖像，显著提升视觉质量和身份保留的稳定性。
IC-Portrait的核心方法分为两个主要部分：
照明感知拼接：该部分利用高比例遮罩的自编码技术（约80%），从输入图像中提取有效的自监督学习特征，以学习照明特性。这种技术能够有效减少用户提供的肖像图像与风格参考图像之间的适应差距，保留局部照明线索和全局阴影效果。
视角一致适应：该部分通过利用合成的视角一致性图像数据集，学习上下文对应关系，使得参考图像可以在不同姿态下进行变形。这种方法确保了在不同视角下生成的肖像能够保持一致性，从而提高生成结果的空间对齐能力。这两个子任务通过将潜在变量连接在一起，形成类似于ControlNet的监督信号，进一步增强了生成的肖像在照明和视角变化下的表现。
为了验证IC-Portrait的有效性，研究团队进行了大量的实验评估，比较了该方法与现有最先进技术的表现。实验结果表明，IC-Portrait在身份保留和视觉质量方面均显著优于其他方法。具体来说，IC-Portrait在ArcfaceID度量上达到了0.674的成绩，显示出其在肖像生成中的优越性。此外，该方法还展示了在复杂照明条件下的3D感知重光能力，能够在多种环境下生成高保真度的个性化肖像。通过在多视角合成数据集上进行训练，IC-Portrait能够有效处理多个人物的肖像生成，展现出其在数字内容创作和虚拟头像等应用中的广泛潜力。
IC-Portrait的工作方法可以简单理解为两个主要步骤。首先，系统会分析输入的照片，使用一种叫做“照明感知拼接”的技术，这就像给照片打上一个“灯光标签”，帮助系统理解照片中的光线变化。这样做的目的是确保生成的肖像在不同光照下看起来都很自然。其次，系统会利用另一种技术，叫做“视角一致适应”，来确保即使在不同的姿势下，生成的肖像也能保持一致的外观。想象一下，如果你在不同的角度拍照，IC-Portrait能让每张照片都看起来像是同一个人。通过这两个步骤，IC-Portrait能够生成高质量、个性化的肖像，保证无论在何种情况下，肖像的身份特征都能得到完美保留。

http ://arxiv.org/abs/2501.17162v1https ://cubediff.github.io/
在本文中，我们提出了一种名为CubeDiff的新方法，用于从文本提示或图像生成360°全景图。该方法利用了多视角扩散模型的最新进展，通过联合合成立方体映射的六个面来实现全景图的生成。与以往依赖于处理等矩形投影或自回归生成的方法不同，我们的方法将每个面视为标准的透视图像，从而简化了生成过程，并使现有的多视角扩散模型得以应用。我们展示了这些模型可以适应生成高质量的立方体映射，而无需复杂的对应注意力层。该模型不仅支持精细的文本控制，生成高分辨率的全景图，而且在超出训练集的情况下也能很好地泛化，达到了定性和定量上的最先进结果。
CubeDiff的核心在于其生成高质量全景图的架构设计。我们的方法包括以下几个关键组成部分：
模型架构：CubeDiff由变分自编码器（VAE）和潜在扩散模型（LDM）组成，采用与传统文本到图像扩散模型相似的结构。我们对每个组件进行了精心调整，以实现有效的多视角全景图生成。
同步组归一化：为了解决生成全景图时颜色不一致的问题，我们在VAE中采用同步组归一化。这一方法通过在空间和视图维度上共同归一化特征激活，确保生成的全景图在不同视角间保持一致的色调。
位置编码：为了使LDM具备空间意识，我们通过计算立方体的UV坐标为每个立方面上的点添加位置编码。这些编码帮助模型理解各个潜在补丁在立方体面上的空间位置，从而生成几何和对象关系一致的全景图。
重叠预测：在训练和生成过程中，我们引入重叠预测策略，通过扩大每个面视场的范围，鼓励模型学习相邻面之间的一致性。这一策略有效避免了在立方体面边界处出现的可见接缝。
无分类器引导：在训练过程中，我们采用无分类器引导方法，通过随机丢弃文本提示和条件图像，增强生成的多样性。
我们通过一系列实验评估了CubeDiff的性能，包括定量和定性评估。我们在多个数据集上进行训练和测试，使用了来自公共资源的室内和室外环境的全景图，确保了模型的泛化能力。实验结果表明，CubeDiff在生成质量上显著优于现有的最先进方法。在定量评估中，我们使用了FrechetInceptionDistance(FID)和CLIP分数等多种指标进行比较，CubeDiff在所有指标上均表现出色。此外，我们还进行了用户研究，结果显示参与者更倾向于选择CubeDiff生成的全景图，进一步验证了其在视觉质量和一致性方面的优势。
CubeDiff的工作原理可以简单理解为一个智能的画家，它能够根据你给出的文字描述或图片，绘制出一个360度的全景画。首先，CubeDiff将画面分成六个面，就像一个立方体的六个面一样，每个面代表一个视角。接着，它使用一种叫做扩散模型的技术，这种技术就像在画布上逐渐添加细节，让画面变得更加生动。为了确保每个面之间的色彩和细节一致，CubeDiff会在处理这些面时，使用一种特殊的归一化方法，确保颜色不会出现差异。此外，它还会在生成时让相邻的面有一些重叠，这样可以避免在拼接时出现缝隙。通过这些步骤，CubeDiff能够生成高质量的全景图，让人感觉就像身临其境一样。

http ://arxiv.org/abs/2501.16757v1
ITVTON（虚拟试穿扩散变换器）是一种新兴的虚拟试穿技术，旨在提高服装与人物图像之间的交互效果。该方法结合了服装和人物图像的空间通道输入，显著提升了服装拟合的准确性。与以往基于生成对抗网络（GAN）的虚拟试穿方法相比，ITVTON利用扩散模型实现更高的图像真实感，同时克服了复杂场景和动态姿态下的挑战。其核心贡献包括采用Flux-Fill模型作为预训练基础，简化输入表示，以及精细化训练策略，这些都有效提高了模型的稳定性和输出质量。实验结果表明，ITVTON在定性和定量上均优于现有的基准方法，确立了虚拟试穿任务的新标准。
ITVTON的方法主要分为以下几个方面：
Flux-Fill模型作为预训练基础：该模型加速了训练过程并提升了性能。只训练单一扩散变换器（Single-DiT）块中的注意力参数，从而降低了计算成本。
简化输入表示：通过将服装和模型图像沿空间维度拼接，并集成多图像文本描述，消除了复杂的预处理步骤，确保自然的服装覆盖效果，尤其对于复杂纹理和图案的服装。
过滤IGPair数据集进行训练：从IGPair数据集中挑选出10,257对图像，涵盖多种服装类型和不同姿态，生成对应的掩码，以提升模型的泛化能力和虚拟试穿图像的质量。
本研究的实验采用了两个公开的时尚数据集：VITON-HD和IGPair。VITON-HD数据集包含13,679对图像，IGPair数据集则包含324,857对图像。通过OpenPose和HumanParsing技术生成独立于服装的掩码，并统一调整图像分辨率。我们对比了ITVTON与其他最新虚拟试穿方法的定量评价，结果显示ITVTON在结构相似性指数（SSIM）、FréchetInceptionDistance（FID）和KernelInceptionDistance（KID）等指标上均表现优异，证明了其简化网络架构和参数微调策略的有效性。
在ITVTON的工作中，研究者通过一种新颖的方式来进行虚拟试穿。首先，他们使用了一种叫做Flux-Fill的模型来帮助快速训练，这样可以更快地得到高质量的图像。其次，他们将服装和模特的图片拼接在一起，简化了输入的处理过程，不需要复杂的步骤，比如姿势估计，这样可以更自然地让衣服“穿”在模特身上。最后，他们从一个大型数据集中挑选了很多不同的服装和模特的图像进行训练，确保模型能够适应各种不同的场景和姿势。这种方法不仅提高了图像的真实感，还减少了生成过程中可能出现的问题，使得虚拟试穿的效果更加理想。

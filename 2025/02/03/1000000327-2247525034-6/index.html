2025-01-30｜NVIDIA,MIT,THU,Playground,PKU｜⭐️🟡
http ://arxiv.org/abs/2501.18427v1https ://github.com/NVlabs/Sana
本文介绍了SANA-1.5，一个用于文本到图像生成的线性扩散变换器，旨在提高训练和推理的计算效率。相较于SANA-1.0，SANA-1.5引入了三项关键创新：首先，采用了高效的训练扩展策略，支持从1.6B参数扩展至4.8B参数，同时显著降低计算资源消耗，并结合了内存高效的8位优化器；其次，提出了模型深度剪枝技术，通过对模型的重要性分析，实现了在质量损失最小的情况下，灵活压缩模型至任意规模；最后，采用推理时间扩展策略，通过重复采样来平衡计算与模型容量，使得小模型在推理时能够达到与大模型相当的质量。这些创新使得在不同计算预算下实现高效模型扩展成为可能，同时保持高质量的生成效果。
SANA-1.5的设计围绕三个主要策略展开：
高效模型扩展：通过从预训练模型中扩展至N+M层，SANA-1.5在保留已学知识的基础上，优化了参数初始化策略，确保新层的有效知识转移。具体采用了部分保留初始化策略，使得新层在训练初期表现为恒等函数，从而实现稳定的训练动态。
内存高效的CAME-8bit优化器：通过结合CAME与8位量化，显著降低了内存使用量，使得在单个消费者GPU上进行大规模模型的微调成为可能。该优化器在保持训练稳定性的同时，减少了约8倍的内存占用。
推理时间扩展：通过多次生成样本并利用视觉语言模型（VLM）进行选择，SANA-1.5能够在推理时使用较小的模型生成与大模型相当的结果。这一策略显示出计算资源可以有效地替代模型容量，从而打破了大模型总是更好的传统观念。
在实验部分，SANA-1.5的性能通过与其他先进文本到图像生成模型的比较得以验证。模型在GenEval基准上达到0.72的分数，经过推理时间扩展后，该分数进一步提升至0.80。实验表明，SANA-4.8B相比于SANA-1.6B在训练收敛速度上提升了2.5倍，并且在不同规模的模型中均表现出较高的生成质量。此外，采用深度剪枝的模型在经过微调后，能够在较小的计算成本下实现与全模型相当的性能。通过这些实验，SANA-1.5展示了在保持高生成质量的同时，如何在不同资源限制下实现高效的模型扩展。
在SANA-1.5的设计中，科学家们想出了三种聪明的办法来让模型更高效。首先，他们不再从零开始训练一个大模型，而是从一个已经训练好的小模型开始，慢慢增加新的部分，这样可以节省时间和计算资源。其次，他们开发了一种新型的优化器，叫做CAME-8bit，这种优化器可以在不牺牲模型性能的情况下，显著减少所需的内存。这意味着即使在普通的电脑上，也能训练大模型。最后，他们还设计了一种推理策略，通过生成多个图像并选择最好的一个，这样即使是小模型也能生成高质量的图像。总之，这些方法让高质量的图像生成变得更加容易和高效。
2025-01-30｜CMU,ToyotaResearchInstitute｜CHI2025｜⭐️🟡
http ://arxiv.org/abs/2501.18588v1
随着文本到图像（T2I）模型的发展，产品设计师开始利用这些生成模型来增强其创意过程。然而，设计师在使用这些工具时常常面临设计固定化的问题，导致探索新想法的能力受限。为了解决这一问题，研究团队开发了Inkspire，这是一个基于草图的工具，旨在支持设计师在产品设计中进行概念原型的迭代和探索。Inkspire通过类比启发和草图-设计反馈循环，促进设计师从抽象概念中获得灵感，并有效指导AI生成设计。通过与专业设计师的交流，研究团队确定了改进T2I交互的设计目标，并通过对比研究验证了Inkspire在激发灵感和探索设计方面的有效性。
Inkspire的设计方法主要包括以下几个方面：
类比启发：通过AnalogicalPanel，设计师可以从已知领域（如自然、建筑、时尚）中提取类比概念，以帮助生成新的设计灵感。
草图引导生成：在SketchingPanel中，设计师可以通过简单的草图输入与AI进行交互。每次草图绘制后，AI会生成新的设计，并将设计结果以草图支架的形式呈现，帮助设计师在设计过程中保持灵活性。
反馈循环：通过Design2Sketch组件，Inkspire将AI生成的高保真设计转换为低保真草图，作为设计师后续创作的基础。这一过程使设计师能够在草图基础上进行迭代，而不是被高保真图像所固定。
为了评估Inkspire的有效性，研究团队进行了一个对比实验，参与者包括专业设计师和新手用户。参与者在使用Inkspire和基线工具ControlNet进行设计任务后，填写了问卷以评估其创意支持水平。实验结果显示，使用Inkspire的参与者在灵感激发、设计探索和人机协作的感知上显著优于使用ControlNet的参与者。设计师反馈称，Inkspire的草图支架功能帮助他们更好地理解AI的生成状态，并有效指导后续的设计迭代。
在Inkspire的使用方法中，设计师可以通过简单的草图与AI进行互动。首先，设计师在一个叫做AnalogicalPanel的地方输入一个抽象的设计概念，比如“保护性”，然后选择一些与这个概念相关的灵感，比如“龟壳”或者“坦克”。接着，设计师在SketchingPanel中绘制草图，AI会根据这个草图生成新的设计，并在草图下方显示一个透明的草图支架，帮助设计师看到AI生成的设计轮廓。这样，设计师就可以在这个基础上继续修改和完善自己的设计，而不是只依赖于AI生成的高保真图像。通过这样的方式，设计师能够更灵活地探索不同的设计方向，避免被固定在某一个思路上。
2025-01-29｜UChicago,2ndsight.ai｜⭐️🟡
http ://arxiv.org/abs/2501.18033v1
生成性人工智能（GenerativeAI）正在图像合成领域引发变革，推动着高质量、多样化和逼真视觉内容的创造，广泛应用于设计、媒体、医疗保健和自主系统等行业。随着图像到图像转换、文本到图像生成、领域转移和多模态对齐等技术的进步，自动化视觉内容创建的范围不断拓展。这些技术的进步主要得益于生成对抗网络（GANs）、条件框架和基于扩散的模型（如稳定扩散）等模型的推动。本文对图像生成技术进行了结构化分类，按照输入的性质对方法进行了组织，探讨了这些模型背后的原理，重点介绍了DALL-E、ControlNet和DeepSeekJanus-Pro等关键框架，并讨论了计算成本、数据偏差和输出与用户意图对齐等挑战。通过这种以输入为中心的视角，本文为研究人员和从业者提供了一个全面的资源，以利用生成性人工智能进行实际应用。
本文提出了一种基于输入类型的图像生成技术分类框架，主要包括以下几个类别：
噪声向量：如生成对抗网络（GANs）和扩散模型，利用随机噪声作为输入，通过对抗训练生成多样且逼真的图像。
潜在空间表示：变分自编码器（VAEs）通过将输入数据编码为压缩的潜在表示，并解码生成新图像，便于对潜在空间的结构化操作。
条件输入：依赖于特定输入（如标签、草图或结构数据）来指导生成过程，产生与用户定义需求相符的定制输出。
文本描述：文本提示作为输入，利用文本到图像生成模型（如DALL-E和稳定扩散）将自然语言转换为视觉内容，整合语言和视觉模态。
这种分类方法为理解不同图像生成方法的能力和实际应用提供了基础结构，帮助研究人员和从业者应对复杂挑战，创造定制的视觉内容，探索生成性人工智能在计算机视觉中的无限可能。
在实验部分，本文探讨了不同生成模型的性能和应用实例，特别关注GANs和扩散模型。通过对比这些模型在图像生成中的表现，研究显示GANs在生成高质量、真实感强的图像方面表现出色，而扩散模型则在生成多样化和高保真图像时更具稳定性。具体实验包括使用VanillaGAN进行数据增强、使用Pix2Pix进行配对图像转换、使用CycleGAN进行无配对图像转换，以及使用StyleGAN进行高质量图像合成。此外，扩散模型如稳定扩散在医疗成像和低光照条件下的图像增强中显示出良好的应用效果。通过这些实验，本文验证了不同方法在实际应用中的有效性和适用性。
在生成性人工智能的研究中，方法部分可以简单理解为如何利用不同类型的信息生成图像。首先，有些模型使用随机噪声作为起点，像是在一张空白画布上开始创作，逐渐形成一幅完整的图像。其次，还有一些模型则使用潜在空间的概念，像是在一个三维空间中找到合适的位置来重建或生成新图像。再者，条件输入模型则类似于给艺术家提供草图或主题，让他们在此基础上进行创作。最后，文本到图像生成模型则是将自然语言描述转化为图像，像是根据故事书中的描述来画画。通过这种方式，研究人员可以利用不同的输入类型来创造各种各样的视觉作品，从而满足不同的需求和应用场景。
2025-01-30｜UCSD,MetaGenAI｜⭐️🟡
http ://arxiv.org/abs/2501.18593v1https ://yinboc.github.io/dito/
本研究提出了一种新颖的图像标记器——扩散自编码器（DiTo），旨在简化和提高图像生成模型的训练效率。DiTo利用单一的扩散L2损失函数来学习紧凑的视觉表示，这一方法与现有的图像标记技术形成鲜明对比。传统的标记器通常依赖于多种损失函数的组合以及预训练的监督模型，导致训练过程复杂且难以调整。DiTo的设计基于理论基础，并通过简化的训练目标实现了可扩展性，从而在图像重建和下游生成任务中获得了竞争力的表现。研究表明，DiTo不仅能有效减少训练复杂性，还能在视觉质量和重建忠实度上优于当前的最先进模型。
DiTo的核心方法包括几个关键步骤：
图像输入与编码：输入图像经过编码器E处理，生成紧凑的潜在表示（tokens）z。
扩散解码器：潜在表示z作为条件输入，传递给扩散解码器D，以建模图像的条件分布p(x|z)。
单一损失函数：训练过程中，使用单一的扩散L2损失函数来优化编码器和解码器。这种方法简化了训练过程，避免了多种损失函数之间的复杂平衡。
噪声同步化：为提高潜在表示的有效性，DiTo引入了噪声同步化机制，确保在潜在空间和像素空间之间的噪声添加过程保持一致，从而增强生成模型的表现。
实验部分对DiTo的性能进行了全面评估，使用ImageNet数据集进行训练和测试。研究者比较了DiTo与传统的GAN-LPIPS标记器（GLPTo）在图像重建和生成任务中的表现。结果表明，DiTo在图像重建质量上优于GLPTo，尤其在处理文本和结构化视觉输入时。定量分析使用了FrechetInceptionDistance（FID）作为评估指标，显示DiTo在重建FID上表现出色。此外，研究还进行了人类偏好评估，结果表明在相同规模下，DiTo的视觉质量得到了更高的认可。
DiTo的工作原理可以简单理解为一个“图像压缩与解压”的过程。首先，输入的图像通过一个称为“编码器”的部分被压缩成一个较小的表示，这个表示就像是图像的“缩略图”。接着，这个缩略图被送入另一个部分，叫做“解码器”，它的任务是根据这个缩略图重新生成原始图像。DiTo使用一种叫做“扩散L2损失”的方法来训练这两个部分，这种方法就像是给它们设定了一个明确的目标，让它们在学习过程中不断优化。为了确保压缩和解压的过程能够顺利进行，DiTo还引入了一种噪声同步的机制，使得在处理图像时的噪声添加过程保持一致。这样，DiTo就能更好地生成高质量的图像，且训练过程比传统方法简单得多。
2025-01-30｜SJTU｜⭐️🟡
http ://arxiv.org/abs/2501.18314v1https ://agav-rater.github.io
在当今AI生成内容的时代，确保音视频质量至关重要。本文提出了一种新的质量评估方法AGAV-Rater，旨在解决AI生成音视频内容（AGAV）中的独特失真问题。现有的音视频质量评估方法往往无法有效处理AI生成内容中的不一致性和不自然的音频元素。为此，研究团队构建了第一个大规模AGAV质量评估数据集AGAVQA，包含3,382个AGAV样本，涵盖16种视频到音频（VTA）方法。该数据集分为AGAVQA-MOS和AGAVQA-Pair两个子集，前者提供多维度的音频质量评分，后者则用于选择最佳的AGAV配对。AGAV-Rater模型基于大型多模态模型（LMM），能够从多个维度对AGAV进行评分，并选择最优的生成结果，从而提升用户体验。
AGAV-Rater的设计和训练过程可分为以下几个步骤：
数据集构建：
收集386个高质量的AI生成视频，从多个VTA方法生成3,088个AGAV样本。
通过主观实验收集9264个均值意见分数（MOS），涵盖音频感知质量、音视频内容一致性和整体质量三个维度。
模型训练：
预训练阶段：构建50,952个指令-响应对，利用真实世界音频字幕数据集进行训练，自动标注为“优秀”或“差”。
微调阶段：使用AGAVQA-MOS子集中的三维数值评分对模型进行微调，以提高对音视频质量的评估能力。
模型结构：
AGAV-Rater基于VideoLLaMA2架构，利用视频编码器和音频编码器分别处理图像序列和音频信号，最终将其映射到同一向量空间中，以便进行统一的多模态质量评估。
在实验部分，AGAV-Rater在AGAVQA-MOS、文本到音频（TTA）和文本到音乐（TTM）数据集上进行了性能评估。通过与现有的多种音视频质量评估方法进行比较，AGAV-Rater在音频质量、内容一致性和整体质量三个维度上均表现出色。实验结果显示，AGAV-Rater在选择最佳AGAV方面的准确性显著高于其他方法，80%的用户认为AGAV-Rater的使用提升了他们的音视频体验。此外，AGAV-Rater的训练和评估过程中采用了5折交叉验证，以确保模型的泛化能力和鲁棒性。
AGAV-Rater的工作原理可以简单理解为一个“智能评分员”。首先，它从网络上收集了很多AI生成的视频和音频，像是在准备考试一样。然后，它通过两个阶段来训练自己：第一阶段是学习如何根据一些简单的评价标准（比如“好”或“差”）来判断音视频的质量；第二阶段是更深入的学习，使用真实用户的评分来提高自己的判断能力。最后，AGAV-Rater就像一个专家，可以快速分析不同的音视频内容，告诉我们哪些是最好的选择。这一过程帮助用户在海量的AI生成内容中快速找到最优质的音视频，提升观看体验。

2025-01-28｜ByteDance｜⭐️⭐️
http ://arxiv.org/abs/2501.16612v1https ://github.com/bytedance/CascadeV
在视频生成领域，随着扩散模型在文本到图像生成中的成功，越来越多的研究开始关注其在文本到视频（T2V）应用中的潜力。然而，扩散模型的计算需求极高，尤其是在生成高分辨率和高帧率的视频时，面临着显著的挑战。为了解决这一问题，本文提出了CascadeV，一种基于级联潜在扩散模型（LDM）的架构，能够生成2K分辨率的视频。实验结果表明，该模型在高压缩比的情况下，仍能有效重建高频细节，并显著降低高质量视频生成的计算负担。此外，CascadeV还引入了一种时空交替网格3D注意力机制，以更好地整合空间和时间信息，从而确保生成视频帧之间的一致性。
CascadeV的整体架构包括两个主要的级联组件：基础的T2V模型和基于潜在扩散的变分自编码器（LDM-VAE）解码器。具体方法如下：
基础T2V模型：该模型生成与文本语义信息对齐的潜在表示，作为LDM-VAE的条件输入。
LDM-VAE解码器：该解码器由潜在扩散模型和标准VAE组成，旨在增强基础模型输出的高频细节，达到32:1的解码比率。
3D注意力机制：引入时空交替网格3D注意力设计，使得空间和时间信息的整合更为有效，降低计算复杂度，同时提高生成视频的真实性。
连接基础模型与LDM-VAE：在推理阶段，LDM-VAE处理来自基础模型的潜在表示，并将其解码到标准VAE潜在空间中，完成高质量视频的生成。
本研究使用Intern4k数据集进行实验，数据集包含1000个4K分辨率、60FPS的高质量视频。我们将CascadeV的重建能力与现有的开源视频模型进行比较，评估指标包括传统图像质量指标（如PSNR、SSIM和LPIPS）以及视频质量指标（如主题一致性和运动平滑度）。实验结果显示，尽管CascadeV采用较高的压缩比，其生成的视频在细节和时间一致性上表现优异，尤其在视频格式的评估中，CascadeV展现出更好的时序一致性。此外，通过将LDM-VAE与现有模型级联，CascadeV能够有效提升视频的分辨率和帧率。
CascadeV的工作原理可以简单理解为一个两步生成过程。首先，基础的T2V模型接收输入的文本信息，并生成一个简化的视频表示，像是一个草图。接着，这个草图会被送入一个更复杂的解码器（LDM-VAE），这个解码器的任务是将这个草图转化为细节丰富的高分辨率视频。为了让这个过程更加高效，CascadeV使用了一种聪明的3D注意力机制，它能够同时考虑视频的空间和时间信息。这就像在看电影时，我们不仅关注画面的美丽，还要注意场景的变化和动作的连贯性。通过这种方法，CascadeV能够在保持视频质量的同时，减少计算资源的消耗。
2025-01-27｜Netflix,UCLA｜⭐️⭐️
http ://arxiv.org/abs/2501.16550v1
PhysAnimator是一个创新的框架，旨在从静态动漫插图生成动态且具有物理真实性的动画序列。传统的手绘动画制作过程耗时且需要专业技能，而PhysAnimator通过结合物理模拟与数据驱动的生成模型，极大地简化了这一过程。该方法能够捕捉动漫中流畅与夸张的动态特性，同时允许用户通过指定外部力量和绑定点来实现个性化效果。最终生成的动画不仅在视觉上令人信服，还具有时间一致性，展示了其在动态动漫风格动画创作中的有效性。
PhysAnimator的核心方法可以分为几个关键步骤：
对象分割与网格生成：首先，从给定的静态动漫插图中提取对象，使用分割算法生成2D三角化网格表示，以便进行后续的物理模拟。
可变形体动力学模拟：通过建立物理模型，计算对象在外部力量作用下的动态变化。具体而言，利用牛顿第二定律来描述物体的运动，并通过数值方法求解运动方程，生成光流场。
用户交互与控制：用户可以通过可定制的能量笔划来定义外部力量的影响，并通过绑定点来指定所需的运动轨迹。这种交互方式增强了动画的可控性。
草图引导的视频扩散模型：将提取的草图与光流场结合，使用视频扩散模型生成高质量的动画帧，确保生成的动画在风格上与原始插图一致。
数据驱动的卡通插值模型：最后，应用数据驱动的插值模型来增强动画效果，添加额外的动态元素，使得生成的动画更加生动。
在实验部分，PhysAnimator的性能与现有的视频生成模型进行了全面比较。研究者构建了一个包含20个动漫图像的测试集，生成每个图像10个视频样本，并使用FrechetInceptionDistance(FID)和VideoScore等指标评估生成结果的质量。实验结果表明，PhysAnimator在视觉质量、时间一致性、动态程度和事实一致性等多个指标上均优于其他基线方法。此外，用户研究采用双选择实验（2AFC）评估用户对生成视频的偏好，结果显示，PhysAnimator在视觉质量、运动合理性和整体感受等方面均获得了更高的用户满意度。
PhysAnimator的工作原理可以简单理解为几个步骤。首先，它从一幅静态的动漫画中提取出主要的角色和物体，并把它们转化为一个可以动态变形的网格，就像用橡皮泥做模型一样。接着，系统利用物理定律来模拟这些物体在风吹、重力等外力作用下的运动。用户可以通过简单的操作来指定物体的运动方向和力度，这就像在玩一个互动游戏。然后，系统会根据这些运动生成一系列动画帧，确保这些帧看起来既流畅又符合原始画风。最后，如果需要，系统还会加入一些额外的动态效果，让动画看起来更加生动。通过这种方式，PhysAnimator能够帮助用户轻松创造出高质量的动漫动画，而无需具备专业的绘画技能。
2025-01-28｜BUAA,Meituan｜⭐️🟡
http ://arxiv.org/abs/2501.16714v1
本研究提出了一种新颖的方法，通过定制文本到视频扩散模型（T2V-DM），实现运动与外观的分离，从而更好地生成符合特定运动概念的视频。运动定制的目标是适应扩散模型，使其能够根据一组参考视频生成具有指定运动的高保真视频。传统方法在编码运动概念时，往往会同时编码参考视频的外观信息，导致生成视频的外观与文本描述不一致。为了解决这一问题，本文提出了两个创新策略：时间注意力净化（TAP）和外观高速公路（AH），旨在有效分离运动与外观，从而提高生成视频的质量和一致性。
本研究的方法包括以下几个关键步骤：
基础模型选择：采用典型的预训练T2V-DM，如ZeroScope或ModelScope，作为运动定制的基础框架。
时间注意力净化（TAP）：在时间变换器中，仅对键（Key）嵌入进行调整，利用低秩适应（LoRA）技术重塑时间注意力，以减少外观信息的干扰，确保运动概念的准确性。
外观高速公路（AH）：修改U-Net中跳跃连接的起始点，将其从LoRA适应的时间变换器的输出改为非适应的空间变换器的输出，以保持原始T2V-DM的外观生成能力，并进一步防止外观信息的泄露。
分阶段LoRA集成（PLI）：在去噪过程中，前期使用适应了TAP和AH的模型，后期则使用原始的T2V-DM，以降低外观泄露的风险。
为验证所提方法的有效性，研究团队进行了多项实验。实验首先使用LOVEU-TGVE-2023数据集进行单视频的评估，通过与现有运动定制方法（如MotionDirector和VMC）的比较，自动评估指标显示，本文方法在文本对齐、时间一致性和美学评分等方面均表现优越。此外，通过人类评估，参与者对生成视频的外观多样性和运动保真度给予了更高的评分。在少量视频的评估中，所提方法同样展现出优秀的性能，能够生成与给定文本描述高度一致且多样化的视频。
在我们的研究中，我们想要解决一个问题：如何让计算机生成的视频不仅能表现出特定的运动，比如“骑自行车”，而且外观也要符合文本描述，比如“在花园里”。为此，我们设计了两种新方法。第一种是“时间注意力净化”，它就像给计算机的运动部分打上了一个保护罩，只让它关注运动，而不去考虑参考视频的外观。第二种是“外观高速公路”，这就像为计算机创造了一条快速通道，让它在生成视频时可以更好地使用原有的外观信息，而不是受到新运动的干扰。通过这些方法，我们能够生成更自然、更符合人们期望的视频。

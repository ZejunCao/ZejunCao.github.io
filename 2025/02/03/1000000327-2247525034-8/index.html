主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）发布日期｜2025年1月31日｜周五
2025-01-30｜ETHZürich,TUMunich,HKBU,AaltoU,UBonn,Oxford,LamarrInstitute,EPFL
http ://arxiv.org/abs/2501.18592v1https ://github.com/donghao51/Awesome-Multimodal-Adaptation
在实际应用中，实现领域适应和泛化面临着重大挑战，特别是在未知目标分布的情况下。多模态领域适应和泛化（MMDA和MMDG）更为复杂，因为不同模态具有独特的特征。近年来，该领域取得了显著进展，应用涵盖了动作识别、语义分割等多个方向。随着大规模预训练的多模态基础模型（如CLIP）的出现，研究者们开始利用这些模型来提升适应性和泛化性能。本综述首次全面回顾了从传统方法到基础模型的多模态适应与泛化的最新进展，涵盖了多模态领域适应、多模态测试时适应、多模态领域泛化等五个主题，并分析了相关数据集和应用，指出了当前面临的挑战及未来的研究方向。
本研究的主要方法分为以下几个部分：
多模态领域适应（MMDA）：该方法旨在利用源领域的标注数据和目标领域的无标注数据，通过对抗性学习和自监督学习来调整模型，使其在目标领域上表现良好。
多模态测试时适应（MMTTA）：MMTTA专注于在在线环境下适应预训练的多模态模型，模型在没有源领域数据的情况下，通过实时调整来应对目标领域的变化。
多模态领域泛化（MMDG）：该方法旨在在没有目标领域数据的情况下，利用多个源领域的训练数据，学习一个具有良好泛化能力的模型。
基础模型的辅助适应与泛化：利用大规模预训练的多模态基础模型（MFMs），如CLIP，来提升模型的适应性和泛化能力。
多模态基础模型的适应：探索如何将多模态基础模型有效地调整到特定的下游任务中。
在实验部分，研究者们通过多种任务验证了所提出方法的有效性，包括动作识别和语义分割等。具体而言，MMDA方法在处理视频、音频和光流等多模态数据时，通过对抗性学习和自监督学习显著提升了模型的准确性。MMTTA则通过动态调整模型参数，成功适应了在连续流入的目标数据中出现的分布变化。此外，MMDG方法在不同源领域上表现良好，能够在未见过的目标领域上保持较高的预测精度。这些实验结果表明，利用基础模型的多模态学习策略能够有效提升模型在复杂任务中的表现。
在这项研究中，科学家们提出了一些方法，让机器学习模型能够更好地适应不同类型的数据。首先，他们通过使用来自多个来源的数据，帮助模型在面对新数据时做出更准确的判断。其次，他们设计了一种方法，允许模型在没有旧数据的情况下，实时调整自己，以便更好地处理新数据。此外，他们还利用一些大型的预训练模型，这些模型已经在很多数据上进行了训练，因此它们可以快速适应新的任务。通过这些方法，研究者们在动作识别和图像分割等任务上取得了很好的效果，证明了这些新方法的有效性。简单来说，这些方法就像给机器学习模型装上了“眼睛”和“耳朵”，让它们能够更聪明地理解和适应周围的世界。
2025-01-30｜UCambridge,METU,TheAlanTuringInstitute
http ://arxiv.org/abs/2501.18192v1
本研究首次探讨了在使用脑电图（EEG）数据进行抑郁症检测时，机器学习（ML）算法的公平性问题。研究提出了两个核心研究问题：RQ1：现有EEG数据和抑郁症检测的ML算法是否存在偏见？RQ2：现有的偏见缓解方法在提升ML公平性方面的有效性如何？通过对三种EEG数据集（Mumtaz、MODMA和Rest）进行实验，结果显示数据和算法偏见确实存在。此外，研究还发现不同的偏见缓解策略在不同数据集和算法中展现出不同的有效性，强调了在抑郁症检测中考虑公平性的重要性。
本研究将抑郁症检测视为分类问题，利用包含敏感属性（如性别）的EEG数据集进行模型训练。具体方法包括：
数据准备：对原始EEG数据进行预处理，确保数据集的平衡性。
偏见缓解方法：
拒绝选项分类：通过重新分类接近决策边界的样本，提升公平性。
重加权：为每个样本分配权重，修改损失函数以减轻偏见。
正则化：通过修改损失函数，确保不同群体的结果公平。
数据增强：使用Mixup技术生成新的样本，以平衡性别比例。
数据调整：对数据进行标签修改，以实现性别间的标签均衡。

处理中：
后处理阶段：
评估指标：采用准确率、精确率和F1分数等指标评估模型性能，同时使用统计公平性和机会均等等公平性指标评估模型的公平性。
实验中使用了三个EEG数据集进行模型训练和偏见评估。每个数据集的样本分布不同，Mumtaz数据集的性别比例失衡最为明显。研究使用了三种深度学习模型（Deep-Asymmetry、GTSAN和1DCNN-LSTM），并对每种模型应用了五种不同的偏见缓解策略。实验结果表明，所有数据集均存在一定程度的偏见，且偏见缓解方法在不同模型和数据集上的效果不一。通过对比分析，发现数据增强在某些情况下能有效提高模型的公平性，但整体上，现有的偏见缓解方法未能完全消除偏见，提示后续研究需更深入探讨公平性问题。
在这项研究中，我们主要想了解在使用脑电图（EEG）数据来检测抑郁症时，是否存在对某些人群的不公平现象。为此，我们采用了一些方法来减少这种偏见。首先，在数据准备阶段，我们通过生成新的样本来确保不同性别的样本数量相等。接着，在模型训练过程中，我们调整了样本的权重，让模型更关注那些少数群体的样本。此外，我们还在模型的输出结果中进行了一些调整，以提高不同群体的预测准确性。最后，我们通过多种指标来评估模型的表现，确保不仅仅关注预测的准确性，还要考虑到模型对不同性别的公平性。通过这些方法，我们希望能够更好地理解和改善机器学习在抑郁症检测中的公平性问题。
2025-01-30｜SKKU,HippoT&C,IBS｜AAAI2025
http ://arxiv.org/abs/2501.18269v1
本研究提出了一种名为MAMS（Model-AgnosticModuleSelection）的框架，旨在解决视频字幕生成中的关键问题。传统方法通常从视频中提取固定数量的帧，这可能导致重要信息丢失或冗余信息的产生。MAMS框架的核心在于其自适应性，能够根据视频内容动态选择适当数量的帧和生成模块，从而提高字幕生成的准确性和完整性。具体来说，该框架包括两个主要功能：选择合适的字幕生成模块以及构建对应的视觉标记子集。此外，研究还引入了一种新的自适应注意力掩码机制，进一步增强对重要视觉标记的关注。通过在三个不同的基准数据集上进行实验，结果显示，MAMS显著提升了现有视频字幕生成模型的性能。
MAMS框架的架构由三个主要模块组成：1）视频编码器，用于将视频转换为视觉标记；2）文本编码器，将字幕转换为文本标记；3）字幕生成模块，负责生成最终的字幕。具体方法步骤如下：
视觉标记提取：从视频中提取视觉标记，形成完整的视觉标记集。
模块选择：使用模块选择器，根据每帧的视觉标记重要性选择合适大小的字幕生成模块。框架内设定了一个评分机制，以确定各帧的重要性。
子集构建：若选择较小的模块，则从全视觉标记集中构建与所选帧相对应的子集；若选择较大的模块，则使用所有帧的视觉标记。
自适应注意力掩码：通过自适应注意力机制，框架能更专注于对字幕生成贡献较大的视觉标记，确保生成的字幕更为准确。
为验证MAMS框架的有效性，研究在三个主要数据集（MSVD、MSRVTT和YOUCOOKII）上进行了广泛的实验。实验中，MAMS与多种视频字幕生成模型（如SwinBERT、UniVL和mPLUG-2）结合使用，结果显示MAMS显著提升了这些模型的性能。评估指标包括BLEU-4、METEOR、ROUGE和CIDEr，结果表明，MAMS在所有指标上均优于传统方法，尤其是在生成准确性和语言流畅性方面。通过对比不同配置的模型，研究还探讨了模块选择和注意力掩码对字幕生成的影响，进一步证明了MAMS框架的有效性和广泛适用性。
在视频字幕生成中，MAMS框架的工作方式可以简单理解为“灵活选择”。首先，系统会从视频中提取出许多画面信息（我们称之为视觉标记），然后根据这些信息的重要性，决定需要使用多少画面来生成字幕。比如，如果视频内容丰富，系统会选择更多的画面；而如果视频内容简单，系统则会减少选择的画面数量。接着，MAMS会根据选择的画面，决定使用哪个生成模块来创建字幕。同时，MAMS还会使用一种智能的注意力机制，确保在生成字幕时，更加关注那些对字幕内容影响最大的画面信息。这样一来，生成的字幕不仅更准确，而且更具连贯性。
2025-01-30｜USurrey,UWollongong,KCL｜🟡
http ://arxiv.org/abs/2501.18594v1https ://github.com/vgthengane/Awesome-FMs-in-3D
本论文对3D点云的基础模型进行了全面的调查与展望。3D点云作为一种重要的空间数据表示形式，能够有效保留物理世界的几何特征，促进对复杂3D环境的理解与交互。尽管人类能够轻松理解物体之间的复杂关系及其空间布局，人工智能系统仍未完全具备这一能力。为了弥补这一差距，研究者们开始探索将多种模态（如图像、文本和音频）与点云数据结合的可能性。基础模型（FMs）是能够跨模态进行无缝集成与推理的模型，近年来在2D模态（如图像和文本）的发展上取得了显著进展，但3D领域因标注数据稀缺和计算资源需求高而滞后。本文旨在填补这一文献空白，提供3D视觉理解领域中使用基础模型的最新方法的系统性回顾，并为未来的研究方向提供见解。
本研究的方法主要分为三个部分：直接适配、双重编码器和三元组对齐。
直接适配：该方法利用2D基础模型（如卷积网络和视觉变换器）对3D点云进行表示。通过简单的调整，直接将预训练的2D模型应用于3D任务，减少了对新模型的需求。
双重编码器：此方法在2D基础模型的基础上增加一个独立的3D点云编码器，利用对比学习和知识蒸馏的方式进行训练，以提取2D和3D特征的共同信息。
三元组对齐：该方法通过对齐点云、图像和文本三种模态，采用对比损失来确保各模态之间的特征一致性，从而提升模型的性能。这种方法通常基于CLIP框架，能够有效整合多种模态的信息。
在实验部分，研究者们对不同模型在多个3D任务上的性能进行了评估，包括分类、检测和分割等任务。实验结果显示，利用2D基础模型进行3D理解的策略在多个基准数据集上表现出了优越的性能。在直接适配方法中，模型能够在没有大量3D训练数据的情况下，依然实现较高的准确率。双重编码器方法通过引入知识蒸馏，有效提升了模型的特征提取能力，而三元组对齐方法则在多模态任务中展示了更好的表现，尤其在开放世界的识别任务中，能够更好地处理复杂的场景。
在这项研究中，科学家们提出了几种方法来帮助计算机更好地理解3D世界。首先，他们使用了“直接适配”的方法，这就像把一个已经会做事情的机器人教会新技能，只需稍微调整一下就能完成新的任务。接下来是“双重编码器”，可以理解为一个双管齐下的策略，计算机同时使用两个不同的工具来处理信息，一个处理图像，另一个处理3D数据，从而更全面地理解场景。最后，“三元组对齐”是一个更复杂的过程，它让计算机尝试将图像、文字和3D点云三者结合起来，就像在同一张图上画出不同的视角，确保它们之间的关系清晰可辨。通过这些方法，研究者希望计算机能够更聪明地理解和互动于我们生活的三维世界。
2025-01-29｜DartmouthCollege｜🟡
http ://arxiv.org/abs/2501.17978v2
本文提出了一种名为VoD-3DGS（View-opacity-Dependent3DGaussianSplatting）的新方法，旨在改进3D场景重建中的光照交互表现。传统的3DGaussianSplatting（3DGS）模型在处理视角依赖内容时存在局限，无法有效区分物体表面与光照的交互，尤其是在反射和高光表现方面。为此，研究者引入了一种可学习的对称矩阵，以扩展标准的标量不透明度模型，使其成为视角依赖函数。这一改进使得模型能够根据观察者的视角抑制某些高斯分布，从而更准确地表现视角依赖的反射和高光，同时保持场景的完整性。通过在多个数据集上的验证，VoD-3DGS在渲染速度和图像质量上均达到了先进水平，能够实现超过60帧每秒的实时渲染。
VoD-3DGS方法的核心在于对3DGaussianSplatting模型的扩展，具体包括以下几个方面：
视角依赖的不透明度模型：通过引入一个对称矩阵，将标量不透明度扩展为与视角相关的函数。这一矩阵的参数可通过学习获得，使得不透明度能够根据观察者的视角变化，增强场景中高光和反射的表现。
密度控制机制：在优化过程中，模型通过修剪和细化技术来调节高斯分布的数量，以保持渲染效率和内存使用率。这一机制确保了在不同视角下的高斯数量能够有效控制，避免冗余。
视角一致性正则化损失：为了确保在相似视角下的高斯表现一致，作者引入了一种新的损失函数，使得在优化过程中，来自相似视角的高斯能够保持相似的不透明度。这种正则化策略进一步提升了视觉一致性。
在实验中，VoD-3DGS方法在多个数据集上进行了评估，包括Mip-NeRF-360、Tanks&Temples、DeepBlending和NeRF-Synthetic。实验结果表明，该方法在多项指标上均表现优异，尤其是在PSNR（峰值信噪比）和SSIM（结构相似性指数）方面，VoD-3DGS在所有测试数据集中都达到了领先水平。尽管在内存使用上有所增加，但渲染速度保持在每秒超过60帧，显示出该方法在实时渲染中的潜力。此外，与其他先进方法相比，VoD-3DGS在处理复杂场景中的反射和高光表现上有显著提升，验证了其有效性和实用性。
在VoD-3DGS方法中，研究者通过引入一个特殊的矩阵，使得3D场景中的颜色和亮度表现更加真实。想象一下，当你从不同的角度看一面镜子时，镜子里映出的东西会有所不同。传统的方法无法很好地处理这种变化，而VoD-3DGS则通过根据你的视角调整每个部分的透明度，来更好地表现光的反射和高光。这就像在画画时，根据光源的位置调整颜料的混合比例，以使画面看起来更生动。为了确保在不同角度下的画面保持一致，研究者还添加了一种新的方法，确保从相似角度看同一物体时，它的表现不会有太大差别。通过这些改进，VoD-3DGS能够在渲染速度和图像质量上都取得令人满意的效果。

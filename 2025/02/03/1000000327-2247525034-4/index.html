288-B｜评估VLM物理世界理解的综合性基准测试，显式时间建模用于视频理解；单目视频流预测未来时刻4D场景表示

AIGCResearch
主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）发布日期｜2025年1月29日｜周三
2025-01-27｜USC,UCBerkeley,ToyotaResearchInstitute｜ICLR2025｜⭐️⭐️
http ://arxiv.org/abs/2501.16411v2https ://physbench.github.io/
理解物理世界是具身人工智能（embodiedAI）的基本挑战，这对于使智能体能够在现实环境中安全有效地执行复杂任务至关重要。尽管视觉-语言模型（VLMs）在推理和任务规划方面显示出巨大潜力，但它们在理解物理现象方面的能力仍然非常有限。为了解决这一问题，本文提出了PhysBench，这是一个综合性的基准测试，旨在评估VLMs在物理世界理解能力上的表现。PhysBench包含10,002个交错的视频-图像-文本数据条目，覆盖四个主要领域：物体属性、物体关系、场景理解和基于物理的动态。这些领域进一步细分为19个子类和8个不同的能力维度。通过对75个代表性VLMs的广泛实验，我们发现这些模型在常识推理方面表现良好，但在理解物理世界时却面临显著挑战，这主要归因于训练数据中缺乏物理知识和嵌入的物理先验知识。为了解决这一缺陷，我们引入了PhysAgent，一个新的框架，将VLMs的广泛性与视觉模型的专门知识相结合，显著提升了VLMs在多种任务中的物理理解能力。
在本研究中，我们提出了PhysBench和PhysAgent两个核心方法。PhysBench的构建过程如下：
数据收集：通过网络搜索、模拟和现实世界捕捉，收集视频和图像。
视频标注：人类标注者对收集的原始视频进行剪辑和物理原则的注释。
问题设计：基于标注的视频生成与物理相关的问题，采用手动设计和自动化工具相结合的方法。
文件组织：将有效问题按任务、子任务和能力类型进行分类。
质量检查：经过人工审核，确保问题与物理世界相关，且答案准确。
PhysAgent则是一个统一的框架，结合了视觉基础模型和物理知识记忆，以增强VLMs的物理世界理解能力。它通过分析VLMs在PhysBench上的错误来源，识别出感知不准确和知识不足是主要原因。因此，PhysAgent引入了视觉基础模型来提升感知能力，并集成了一个知识记忆模块，以嵌入关于物理世界的基本知识。
在实验部分，我们对75个代表性VLMs进行了评估，以确定它们在PhysBench上的物理世界理解能力。实验设置分为三种配置：图像VLMs、视频VLMs和通用VLMs。评估结果显示，大多数模型的平均准确率约为40%，远低于人类水平。即使是表现最好的模型GPT-4o，其准确率也仅为49.49%。这些结果表明，当前的VLMs在物理世界理解方面存在显著差距。为了改善这一情况，我们对VLMs进行了微调，使用PhysBench数据集进行训练，结果显示它们的性能得到了显著提升。此外，我们还通过在MOKA平台上进行的机器人操作实验，验证了增强物理理解能力对具身智能体部署的积极影响。
在我们的研究中，我们提出了一个名为PhysBench的工具，它帮助我们评估计算机模型对物理世界的理解。这个工具的建立分为几个步骤：首先，我们从互联网和现实世界收集了很多视频和图片；然后，团队成员对这些数据进行了详细标注，确保每个视频和图片都能与物理概念相对应。接着，我们设计了一系列问题，帮助我们测试这些模型是否能够理解物理现象，比如物体的质量、位置和运动等。最后，我们用这些问题对不同的计算机模型进行测试，结果发现大多数模型在理解物理方面的表现还不够好。因此，我们还开发了一个名为PhysAgent的框架，它结合了视觉技术和物理知识，帮助这些模型更好地理解物理世界。这些工作将帮助我们在未来构建更智能的机器人，能够在复杂的现实环境中安全地执行任务。
2025-01-28｜CSIROData61,TikTok,UMelbourne
http ://arxiv.org/abs/2501.16786v1
本研究探讨了显式时间建模在多模态大型语言模型（MLLMs）中对视频理解的作用。随着视频数据的日益普及，如何有效地捕捉时间关系成为了一个重要的研究课题。现有的方法主要分为隐式时间建模和显式时间建模两种。隐式建模依赖于LLM解码器来推断帧之间的时间关系，而显式建模则通过辅助时间编码器来直接建模时间依赖性。为此，研究者提出了可堆叠时间编码器（STE），以便在不同的时间感知尺度上进行灵活的显式时间建模。通过系统比较这两种建模方式，研究揭示了显式时间建模在提高视频理解性能方面的重要性，并为未来的研究提供了可行的设计思路。
本研究的核心方法是引入可堆叠时间编码器（STE），其设计旨在灵活捕捉视觉时间信息。具体方法如下：
整体架构：STE与LLaVA系列的开源模型相结合，保留其核心结构，增强其时间建模能力。
时间编码：STE通过卷积操作对输入帧进行处理，提取时间依赖性，并生成具有所需输出形状的嵌入。这一过程允许对输入和输出帧的比率进行调整，便于灵活编码和压缩。
训练过程：训练分为两个阶段：预训练阶段利用现有模型的能力初始化STE，以有效捕捉时间信息；监督微调阶段则对整个模型进行微调，以增强对视觉输入的理解。
评估与比较：通过在多个视频基准测试上评估STE的表现，比较其与隐式时间建模的效果，验证显式时间建模的有效性。
实验部分主要通过在六个视频基准测试上评估STE的性能，结果显示显式时间建模显著提升了模型的理解能力。具体实验设置包括：
实验设置：使用LLaVA-OV和LLaVA-Video作为基础模型，分别评估其在不同时间压缩比下的表现（如2:2、4:3等）。
性能比较：在不同的时间建模策略下，比较模型在视频理解任务中的准确性，发现显式时间建模相较于隐式建模有显著的性能提升。
任务评估：重点分析模型在时序理解、动作识别和感知能力等任务上的表现，结果表明，采用STE的模型在多个基准测试中表现优于基线模型。
在这项研究中，研究者们想要了解如何更好地让计算机理解视频中的时间信息。为此，他们设计了一种新的工具，叫做可堆叠时间编码器（STE）。这个工具像一个聪明的助手，能够帮助计算机在观看视频时，记住不同画面之间的关系。研究者们首先把这个助手和已有的模型结合在一起，然后教它如何识别视频中的时间变化。接着，他们进行了很多实验，发现这个助手确实能帮助计算机更准确地理解视频内容，尤其是在处理复杂的时间关系时。通过这样的方式，研究者们不仅提升了计算机的理解能力，也为未来的研究提供了新的思路和方法。
2025-01-28｜StepFun,MEGVII,FDU
http ://arxiv.org/abs/2501.16617v1
本研究提出了一种新颖的动态场景三维表示预测框架，旨在利用单目视频流生成未来时刻的三维场景表示。与以往主要预测未来帧的方法不同，本框架通过明确的三维表示来捕捉动态场景的变化。该方法的核心设计包括两个方面：首先，采用以观察者为中心的无界三平面来显式表示动态物理世界；其次，开发了一个4D感知变换器，以从单目视频中聚合特征并更新三平面。这种设计的结合使得模型能够在自监督的方式下，利用大规模单目视频进行训练。实验结果表明，该模型在动态辐射场预测方面表现优异，具备很强的泛化能力，能够应对未见场景的挑战。
本方法的核心在于从单目视频流中预测未来时刻的三维场景表示。具体步骤如下：
特征提取：使用图像编码器从输入的单目视频中提取特征。
特征聚合：通过4D感知变换器，将提取的特征进行聚合，以更新以观察者为中心的三平面表示。
三维表示生成：更新后的三平面经过上采样处理，生成目标时刻的最终三维表示。
模型优化：利用光度损失函数对比生成的目标视图和真实视图，优化模型参数。此外，引入基于时间的三维约束，以增强三维场景表示的一致性。
我们在多个数据集上进行了广泛的实验，以验证所提方法的有效性。实验使用了NVIDIA动态场景数据集、DAVIS数据集等，评估指标包括PSNR、SSIM和LPIPS等。结果显示，本方法在动态场景的三维未来预测中，优于现有的基线模型，如GNT和PGDVS，尤其在动态区域的表现上，PSNR值显著提高。同时，本模型在未见场景上的泛化能力也得到了验证，展示了良好的适应性和鲁棒性。
在这个研究中，我们的目标是从普通的视频中预测未来的三维场景。首先，我们用一个程序来分析视频中的每一帧，提取出有用的信息，就像人类观察事物一样。接下来，我们使用一个智能的变换器，它能够把这些信息整合起来，并更新一个特殊的三维表示，这个表示就像是一个三维模型，帮助我们理解场景的变化。最后，我们通过比较生成的三维图像和真实图像，来不断优化我们的模型，确保它的预测尽可能准确。这样的方法不仅让我们能更好地理解动态场景，还能在未见过的情况下做出合理的预测。

2025-03-12｜TencentGames,TAMU,CUHK｜⭐️⭐️
http ://arxiv. org/abs/2503. 09439v1
本研究提出了一种名为SuperCarver的3D几何超分辨率框架，旨在为给定的粗糙3D模型生成高保真且与纹理一致的表面细节。传统的3D建模流程通常需要大量人工雕刻，效率低下且耗时。尽管近年来AI技术在3D内容生成方面取得了显著进展，但生成的网格表面往往过于平滑，缺乏细致的几何特征。SuperCarver通过将原始纹理模型渲染到图像域，并利用多视角的法线扩散模型，自动增强表面细节。这一方法不仅提高了3D模型的视觉真实感，还显著加快了高质量网格资产的生产效率，适用于广泛的应用场景，如电影、游戏和虚拟现实等。
SuperCarver的工作流程分为两个主要阶段。第一阶段是构建一个确定性先验引导的2D扩散架构，用于预测细节增强的法线图。具体步骤如下：
从指定视角渲染输入的3D纹理模型，获取相应的彩色图像、法线图和深度图。
利用法线图生成低分辨率法线的特征表示，并通过扩散模型进行细节增强。
结合图像的语义和几何先验信息，优化法线图以提高生成的表面细节质量。第二阶段则是通过反向渲染技术，将多视角的法线图预测结果转化为高精度的3D网格。此阶段采用了一种抗噪声的逆法线渲染方案，确保在优化过程中保持几何结构的稳定性，最终生成的3D模型表面细节与原始纹理一致。
为了验证SuperCarver的有效性，研究团队在多个现有3D生成框架生成的粗糙模型上进行了实验，包括InstantMesh、Unique3D和CLAY。使用Objaverse和ABO数据集的800K+3D资产，经过精心筛选和清理，最终获得约80K高质量网格模型用于训练。实验结果表明，SuperCarver能够有效提升模型的表面细节，生成的法线图在细节和准确性上均优于现有方法。具体评估采用了均方根误差（MAE）等指标，结果显示SuperCarver在生成高保真法线图方面表现优异。此外，研究还通过对比不同的法线预测和网格优化方案，进一步验证了其在处理复杂表面细节方面的优势。
SuperCarver的工作方式可以简单理解为“给3D模型加细节”。首先，系统会从不同的角度拍摄这个模型，获取它的颜色、法线（表面方向）和深度信息。然后，通过一个智能算法，系统会分析这些信息，并预测出更细致的法线图，就像给模型“画上阴影和纹理”。接着，这些细节会被用来改进模型的表面，使其看起来更加真实。这个过程就像是用一张模糊的照片，经过处理后变得清晰可见，细节丰富。通过这种方法，SuperCarver不仅能快速生成高质量的3D模型，还能大大减少人工雕刻的时间和成本，让3D艺术家们的工作变得更加轻松和高效。
2025-03-12｜HUJI｜⭐️🟡
http ://arxiv. org/abs/2503. 09601v2https ://itaychachy. github.io/reward-sds/
RewardSDS是一种创新的得分蒸馏采样方法，旨在通过奖励加权采样来优化生成模型的输出，从而更好地对齐用户意图。该方法特别适用于数据稀缺的场景，如3D内容生成，利用2D扩散模型的先验信息来提升生成质量。与传统的得分蒸馏采样(SDS)相比，RewardSDS通过引入奖励模型来对采样的噪声进行加权，优先考虑那些能够产生高奖励输出的噪声样本。这种方法不仅提高了生成的图像质量和多样性，还增强了与用户期望的对齐程度。研究中还提出了RewardVSD，将该方法扩展到变分得分蒸馏(VSD)中，进一步验证了其有效性。
RewardSDS的核心在于通过奖励模型对噪声样本进行加权，以优化得分蒸馏的过程。其方法步骤包括：
噪声样本生成：从高斯分布中抽取N个噪声样本，并为每个样本计算对齐分数。
图像渲染：使用参数化生成器g(θ)渲染初始图像，并将噪声应用于该图像。
计算奖励分数：通过去噪处理得到的图像输入到奖励模型中，获取每个样本的奖励分数。
损失计算：根据奖励分数对得分蒸馏损失进行加权，计算加权损失并反向传播更新生成器参数。
扩展到VSD：在变分得分蒸馏中，应用类似的策略，通过优化多个粒子来增强生成效果。
在实验部分，研究者分别在零-shot文本到图像生成、文本到3D生成及图像编辑等任务上评估了RewardSDS和RewardVSD的表现。通过对比标准的SDS和VSD，发现RewardSDS显著提升了生成质量和与用户意图的对齐度。在零-shot文本到图像生成中，使用不同的奖励模型进行评估，结果显示RewardSDS在多项指标上均优于基线模型。此外，针对文本到3D生成和图像编辑的实验也表明，RewardSDS方法不仅提升了生成的视觉吸引力，还增强了生成内容的多样性。研究还进行了广泛的消融实验，以分析不同设计选择对性能的影响。
RewardSDS的工作原理可以想象成一个评委在选拔比赛中的优胜者。首先，系统会生成一些候选图像，每个图像都有一些“噪声”添加进来。接着，系统会使用一个评委模型来给这些图像打分，评委会根据图像的质量和与描述的匹配程度来评判。然后，系统会根据这些分数来决定哪些图像更好，给那些表现优秀的图像更多的“关注”。最后，系统会根据这些加权的评分来调整生成图像的参数，以便下一次生成更符合要求的图像。这样，通过不断地评估和调整，RewardSDS能够生成更符合用户期待的高质量图像。
2025-03-12｜CCNU,NJUA,CUHK,Autodesk,USTB,LNU｜⭐️⭐️
http ://arxiv. org/abs/2503. 09160v2
WonderVerse是一种新颖且有效的框架，旨在生成可扩展的3D场景。与传统方法依赖于迭代深度估计和图像修复的方式不同，WonderVerse利用视频生成模型的强大能力，从多视角创建高沉浸感和几何一致的3D环境。该框架的主要创新在于引入了一种可控的3D场景扩展技术，显著增加生成环境的规模。此外，WonderVerse还结合了异常序列检测模块，通过分析摄像机轨迹来解决生成视频中的几何不一致性问题。通过与多种3D重建方法的兼容性，WonderVerse实现了高效且高质量的生成，实验结果表明其在生成可扩展和高度真实的3D场景方面显著优于现有的复杂架构。
WonderVerse的方法分为几个关键模块。首先，输入文本描述后，系统生成一个循环的视频序列。然后，通过扩展初始视频，生成左右两个新的视频，保持风格一致性。接下来，框架利用COLMAP进行摄像机参数估计，以确保生成的3D场景在几何上保持一致。为了应对视频生成过程中可能出现的几何不一致性，WonderVerse设计了异常序列检测模块，通过监测摄像机运动中的不连续性来识别并再生不稳定的视频段。最后，利用3D高斯点云表示法进行3D场景重建和渲染，确保生成的场景在视觉上具有高保真度和一致性。这一系列步骤的结合，使得WonderVerse能够生成高质量、可扩展的3D场景。
在实验部分，WonderVerse的实现细节和评估指标被详细介绍。首先，从文本提示出发，生成初始的循环视频，并通过四次扩展迭代生成多个场景。每次扩展都创建新的视频，最终形成一个完整的场景视频。评估指标包括CLIP-Score、Q-Align和NIQE，用于衡量生成场景的语义对齐、结构一致性和感知质量。实验结果显示，WonderVerse在室内和室外场景生成方面均优于现有方法，尤其是在语义对齐和几何一致性上表现突出。此外，通过对关键模块的消融研究，验证了异常序列检测和场景扩展策略的有效性，进一步强化了WonderVerse在生成高质量3D场景方面的能力。
WonderVerse的工作原理可以简单理解为一套生成3D场景的流程。首先，它会根据用户输入的文字描述，生成一个环绕场景的视频。接着，WonderVerse会将这个视频向左右两侧扩展，创造出更大的场景。为了确保视频中的画面看起来连贯自然，系统会使用一种叫COLMAP的工具来计算摄像机的运动轨迹，确保每个画面之间的连接是平滑的。如果在视频生成过程中发现某些画面不太一致，WonderVerse会自动检测并重新生成这些画面。最后，使用3D高斯点云技术将这些画面合成一个完整的3D场景，从而确保最终生成的场景不仅真实且富有细节。这一系列步骤使得WonderVerse能够轻松创建出大规模、可扩展的3D场景。
2025-03-11｜YonseiU｜⭐️🟡
http ://arxiv. org/abs/2503. 08737v1
本研究提出了一种新型变分自编码器（VAE）框架，称为COD-VAE，旨在通过压缩潜在空间来提升三维扩散模型的效率。传统的三维生成模型面临着计算成本高和重构质量不足的问题，尤其是在处理复杂的三维形状时。COD-VAE通过将三维形状编码为一组紧凑的1D潜在向量，显著减少了对潜在向量的需求，从而实现了更高效的生成过程。实验结果表明，COD-VAE在保持高质量重构的同时，潜在向量数量减少至64个，达到了20. 8倍的生成速度提升，表明其在三维生成任务中的有效性和潜力。
COD-VAE的核心在于其两阶段自编码器架构，具体包括以下几个关键组件：
编码器设计：编码器首先通过注意力机制对输入的三维点云进行下采样，生成中间点补丁。这些补丁在后续的自注意力层中被处理，最终映射为紧凑的1D潜在向量。
解码器设计：解码器利用三平面作为中间表示，从潜在向量中重建三维形状。该设计通过使用变换器块来提高解码效率，同时减少计算开销。
不确定性引导的令牌修剪：为了进一步提高解码效率，模型引入了一种不确定性引导的令牌修剪机制，优先处理重建复杂区域的计算，同时跳过简单区域，降低总体计算负担。
在实验部分，研究者在ShapeNet和Objaverse数据集上评估了COD-VAE的重构质量和生成能力。实验结果显示，COD-VAE在重构任务中表现优异，尤其是在使用仅64个潜在向量的情况下，其重构质量甚至超越了使用512个潜在向量的VecSet模型。针对不同的生成任务，COD-VAE展示了在类条件生成和无条件生成中的强大性能，尤其在处理复杂物体时，生成效率显著提升，达到了20. 8倍的速度提升。此外，模型的训练效率也得到了显著改善，表明COD-VAE在实际应用中的潜力。
COD-VAE的工作原理可以简单理解为一个“聪明的压缩器”。首先，它像一个精明的工匠，将复杂的三维形状“拆解”成小块（中间点补丁），并用一种特殊的方式将这些小块压缩成更简单的1D向量。这个过程就像把一个大图像压缩成更小的文件，同时保留重要的信息。在解码阶段，COD-VAE会把这些小向量“重组”回三维形状，就像拼图一样，但它还会根据哪些部分更复杂来决定优先处理哪些部分，从而提高效率。通过这种方式，COD-VAE不仅能快速生成高质量的三维形状，而且只需要较少的存储空间，表现得更为高效。
2025-03-12｜AlibabaGroup｜⭐️🟡
http ://arxiv. org/abs/2503. 09527v1https ://combatvla. github.io/
CombatVLA是一种专为3D动作角色扮演游戏（ARPG）设计的高效视觉-语言-动作（VLA）模型。尽管现有的VLA模型在体现在智能决策方面取得了显著进展，但在复杂3D环境中的实时决策仍然面临诸多挑战，如高分辨率感知和动态条件下的战术推理。为了解决这些问题，CombatVLA通过整合视频-动作对训练数据，采用了一种创新的“行动思维”（AoT）序列格式，并在此基础上构建了一个高效的动作执行框架。实验结果表明，CombatVLA在战斗理解任务中不仅超越了现有的模型，还实现了50倍的速度提升，并在多个战斗任务中超过人类玩家的成功率。
CombatVLA的构建遵循三阶段的渐进学习策略，以提升模型在战斗任务中的表现。首先，开发了一个“动作追踪器”，用于在游戏中收集玩家的动作数据。这些数据随后被整理成“行动思维”（AoT）格式，便于模型进行战斗推理。第二阶段是粗粒度视频级别的AoT调优，模型通过分析视频中的整体内容来理解战斗环境。接下来是细粒度帧级别的AoT调优，模型需要快速响应并理解环境变化，以便做出及时决策。最后，CombatVLA采用了一种特殊的“截断策略”，通过引入⟨TRUNC⟩标记来加速推理过程。此外，模型的训练还结合了自适应动作加权损失，以优化视觉与动作之间的对应关系。
在实验中，CombatVLA在多个战斗理解基准（CUBench）和一般性能基准上进行了评估。实验结果显示，CombatVLA在战斗理解的平均得分上达到了63. 61，超越了其他闭源和开源模型，尤其是在高层次的动作推理能力上表现突出。通过与人类玩家的对比，CombatVLA在多个任务中展现出了更高的成功率和更快的执行速度，证明了其在实时战斗场景中的有效性。此外，实验还对模型的推理延迟进行了评估，CombatVLA的推理延迟仅为1. 85秒，显示出显著的效率提升。通过对不同阶段的渐进学习和自适应损失的消融研究，进一步验证了模型在战斗任务中的强大能力。
CombatVLA是一种新型的智能游戏模型，专门用来帮助游戏角色在复杂的战斗场景中做出快速决策。它的工作原理可以分为几个简单的步骤。首先，模型通过一个叫“动作追踪器”的工具，记录玩家在游戏中的每一个动作，比如按下哪个键或移动到哪里。然后，这些动作被整理成一种特殊的格式，帮助模型理解在不同情况下应该如何反应。接下来，模型会在视频中学习，一开始是从整体视频入手，慢慢地再去关注每一帧的细节，确保能够快速做出反应。最后，为了加快决策速度，模型会在生成动作时省略一些不必要的信息。这样一来，CombatVLA就能在游戏中像人类一样快速而准确地作出反应，帮助玩家更好地应对敌人。

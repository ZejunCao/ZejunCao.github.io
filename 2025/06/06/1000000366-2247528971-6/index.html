2025-03-11｜TongjiU,ByteDance,UEdinburgh
http ://arxiv. org/abs/2503. 08741v2
在多模态大型语言模型（MLLMs）的发展中，训练数据的缺乏成为了一大瓶颈。尽管已有一些方法尝试合成多模态数据，但这些方法往往依赖于复杂的输入和人力干预，导致数据的多样性和质量受限。因此，本文提出了一种新颖的合成方法Oasis，旨在仅利用图像生成高质量的多模态指令数据。Oasis通过简化数据合成流程，消除了对多种输入文本的需求，从而显著提高了数据的多样性和质量。研究表明，Oasis可以有效提升MLLMs在多个基准测试中的表现，并且为特定领域的任务提供了强有力的支持。
Oasis的合成流程分为三个步骤。第一步是数据合成，利用“钩子”提示从优化后的多模态语言模型（MLLM）中提取响应。具体而言，输入仅为图像，模型根据图像生成相关的指令，而不需要手动设计的文本提示。第二步是数据分类，通过设计分类机制，将生成的数据分为指令跟随和描述性文字两类。我们利用LLM作为分类器，确保只保留符合指令跟随标准的数据。第三步是指令质量控制，针对生成的指令进行四个维度的评估，包括解决能力、清晰度、幻觉和语义合理性，筛除低质量的指令。通过这种方式，Oasis不仅提高了合成数据的质量，也确保了生成结果的多样性。
在实验部分，研究团队收集了超过50万条Oasis数据，并在LLaVA-NeXT模型上进行了广泛测试。实验结果显示，Oasis合成的数据显著提升了MLLMs在14个不同基准测试中的表现。具体而言，使用Oasis数据的模型在多个领域的任务中均表现出色，特别是在OCR和文档分析等任务中，均取得了明显的性能提升。此外，Oasis数据的多样性和高质量使得模型在处理特定领域的问题时更具优势。通过对比不同合成方法，Oasis的表现显著优于现有的合成数据，验证了其在多模态数据合成中的有效性和实用性。
Oasis方法的核心在于简化多模态数据的生成过程。首先，它只需要一张图像作为输入，而不是传统方法中常用的复杂文本提示。接下来，Oasis会分析这张图像，并自动生成与之相关的指令，比如“这只猫在做什么”。然后，Oasis会对生成的指令进行分类，确保只保留那些能引导用户完成特定任务的指令。最后，Oasis会对这些指令的质量进行严格把关，确保它们不仅清晰明了，而且能够准确反映图像的内容。这种方法使得合成的数据更加丰富多样，同时也减少了人力成本，极大地提高了数据的使用效率。
2025-03-10｜Caltech
http ://arxiv. org/abs/2503. 08723v1https ://github. com/Raphoo/DCSM_Ideal_CLIP
ContrastiveLanguage-ImagePre-Training(CLIP)是一种广泛应用于视觉语言模型（VLM）预训练的方法，旨在将图像和文本编码器联合训练，从而将两种模态映射到一个共享的潜在空间。尽管CLIP在许多下游任务中表现出色，但其潜在空间在处理复杂的视觉与文本交互方面存在显著的局限性。这些缺陷导致CLIP无法准确表示属性绑定、空间关系和否定等重要语义特征。本文通过对CLIP的几何性质进行深入分析，揭示了其潜在空间的基本限制，并提出了一种新的评分方法——DenseCosineSimilarityMaps（DCSM），以解决CLIP的根本性缺陷。该方法不仅能改善CLIP在多种基准测试上的性能，还为复杂的视觉语言任务提供了更强的支持。
本研究提出的DenseCosineSimilarityMaps(DCSM)方法旨在克服CLIP的几何限制，具体步骤如下：
保留嵌入信息：DCSM方法保留了CLIP中每个文本令牌和图像补丁的完整嵌入信息，而不仅仅依赖于单一的CLS和EOS令牌。
计算密集余弦相似性：通过计算所有文本令牌与所有图像补丁之间的成对余弦相似性，生成一个密集的相似性图。这一过程能够保留局部信息，提供更细致的语义匹配。
引入功能行：为了解决功能词（如“左边的”、“右边的”等）在DCSM中的干扰，研究者将这些行替换为常量向量，以减少噪音并提高模型的判别能力。
轻量级CNN模型：使用一个简单的两层卷积神经网络（CNN）来处理DCSM，显著减少参数量，同时保持高效的性能。
在实验部分，研究者将DCSM方法与多种现有模型进行了比较，包括CLIP、NegCLIP、CoCa和BLIP等。实验涵盖了多个基准测试，包括属性绑定、空间推理和否定等任务。结果表明，DCSM在各项任务中均显著超越了CLIP及其变种，尤其是在处理复杂的视觉语言组合时。具体而言，DCSM在CLEVR-bind、NCD和VGattribution等数据集上的表现均优于其他模型。此外，DCSM还展现出良好的泛化能力，能够处理未见过的属性和空间概念，表明其在实际应用中的潜力。
在本研究中，研究者提出了一种新的方法来改善CLIP，这种方法叫做DCSM。简单来说，DCSM是通过保留更多信息来提高图像和文本之间的匹配能力。首先，它不再只关注单一的图像或文本部分，而是计算每个文本词与所有图像部分之间的相似度，这样可以更全面地理解它们之间的关系。其次，研究者还解决了功能词的干扰问题，确保模型能够更准确地判断图像和文本的匹配程度。最后，通过一个简单的神经网络，DCSM能够在多个任务中表现得更好，尤其是在需要理解复杂关系的情况下。这种方法让模型更加智能，能够更好地处理我们日常生活中的视觉和语言信息。
2025-03-12｜ZJU,BJUT,ShanghaiAILab,PolyU,QDU,SUSTech
http ://arxiv. org/abs/2503. 09445v1
Astrea是一种基于混合专家（MoE）架构的视觉理解模型，旨在解决多任务异构性和模型通用性之间的平衡问题。随着视觉-语言模型（VLMs）的发展，它们在多模态理解任务中扮演了核心角色，能够同时处理图像和文本数据。然而，任务的复杂性和多样性使得在异构视觉专家之间协调负载平衡变得具有挑战性。Astrea通过引入三项关键创新：异构专家协调机制、动态知识融合策略和增强优化框架，旨在解决这些挑战。其设计允许将四个专门的模型（检测、分割、分类和描述）整合到一个全面的专家矩阵中，从而覆盖视觉理解的核心元素。通过在12个基准任务上的广泛评估，Astrea表现出优越性，平均性能提升达到4. 7%，为开发通用多模态代理奠定了新的方法论基础。
Astrea的训练方法采用两阶段训练策略，旨在实现异构专家协作。第一阶段是对齐阶段，模型从粗粒度到细粒度逐步理解视觉信息。具体步骤包括：
渐进训练：从全局视角开始，逐步细化到更详细的局部信息，以确保模型能够捕捉到图像的语义、空间位置和对象之间的关系。
随机残差连接：在训练过程中引入随机残差连接，防止模型在学习新任务时遗忘先前任务的知识，增强任务间的相互协调。
动量对比学习：通过动量对比学习增强特征准确性，利用历史样本来提升当前批次的学习效果。
动态权重分配：根据数据特征动态调整各个专家的贡献，确保在决策过程中选择最合适的专家参与。
这种方法有效地减少了训练负担，并通过动态知识融合策略提升了模型的整体性能。
在实验部分，Astrea在12个基准任务上进行了评估，包括视觉问答（VQA）、图像描述和跨模态检索等。实验采用了多分辨率设置，利用AdamW优化器进行训练，并设定了具体的训练参数和学习率策略。通过对比多种现有模型，Astrea在多个任务中均表现出色，尤其是在知识、OCR和视觉中心任务上，展现出强大的理解和推理能力。实验结果表明，随着训练步骤的增加，模型性能逐渐提升，且通过逐步对齐的训练策略，Astrea能够快速适应简单的VQA任务，并在复杂的视觉推理对话任务中获得显著改善。此外，模型在视频理解任务中也表现突出，显示出其在多模态大语言模型领域的竞争力和适应性。
Astrea的训练方法可以简单理解为一个分阶段的学习过程。首先，模型在学习时会从整体图像开始，逐步深入到每个细节，这就像是先看一幅画的全貌，再观察画中每一个细节。为了确保模型不会忘记之前学到的知识，它会在学习新内容时，偶尔回顾之前的内容，这就好比在学习新知识时，不时复习旧知识。接着，模型使用一种叫“动量对比学习”的技巧，这意味着它会借助过去的学习经验来提升当前的学习效果，就像在考试前复习旧题目一样。最后，Astrea会根据不同任务的需要，灵活调整不同专家的参与程度，确保在解决问题时总是选择最适合的专家来帮助解决。这种灵活的学习方式让Astrea在处理各种复杂任务时，表现得更加出色和高效。
2025-03-12｜ShanghaiTech,TencentPCG｜ICLR2025
http ://arxiv. org/abs/2503. 09046v1https ://foundation-model-research. github.io/NeuronPath/
本研究聚焦于揭示视觉变换器模型（VisionTransformers）中的重要神经元路径，以理解其内部工作机制。尽管视觉变换器在图像分类等任务中表现出色，但其决策过程仍然不透明，给实际应用带来了挑战。已有研究主要集中在输入归因和单个神经元分析上，但缺乏对层级信息和整体信息流路径的考虑。本文提出了一种新的方法，通过联合影响测量来评估神经元对模型输出的贡献，并设计了一种逐层选择神经元的算法，以识别从输入到输出的关键神经元路径。实验结果表明，该方法在发现影响力神经元路径方面优于现有基准，揭示了视觉变换器在处理同一类别图像时的特定内部机制。
本研究提出了一种新颖的方法，称为“神经元路径”（NeuronPath），旨在识别视觉变换器模型中最具影响力的神经元路径。具体方法包括以下几个步骤：
联合影响测量：首先定义一个联合归因度量，评估不同神经元对模型输出的共同贡献。
逐层神经元定位算法：采用逐层选择算法，逐渐构建神经元路径。在每一层中，算法选择对当前层输出贡献最大的神经元。
路径定义：根据用户定义的标准（如最大激活值），确定神经元路径，以便追踪信息从输入到输出的流动。
计算和优化：通过计算联合归因分数，优化所选神经元路径，确保所识别的路径在模型推理中发挥关键作用。
这种方法不仅考虑了单个神经元的影响，还综合了多个层次神经元的共同作用，提供了对模型行为的更全面理解。
为验证提出的方法，研究团队进行了多项定量和定性实验。首先，通过与现有基准方法的比较，评估所发现的神经元路径的有效性。实验包括神经元干预实验，即对选定神经元进行去除或增强操作，分析其对模型分类能力的影响。使用的指标包括去除准确率偏差和增强准确率偏差，量化操作后模型输出的变化。此外，研究还分析了不同类别图像的神经元路径，揭示了在相同类别中神经元的聚类特性。最后，进行网络剪枝实验，探讨保留关键神经元路径对模型性能的影响，结果表明，大部分神经元是冗余的，只有少数神经元对模型性能至关重要。
在这项研究中，我们提出了一种新的方法来找出视觉变换器模型中最重要的神经元路径。简单来说，我们首先确定哪些神经元对模型的最终判断最有帮助。然后，我们逐层检查模型，从每一层中选出最关键的神经元，最终形成一条信息流通的路径。这个过程就像在一条河流中找出水流最强的部分。通过这种方法，我们不仅能看到哪些神经元在工作，还能理解它们如何协同作用，帮助模型做出决策。我们还通过实验验证了这种方法的有效性，比如在对模型进行调整时，发现了哪些神经元对于模型的表现至关重要。这种深入的分析帮助我们更好地理解视觉变换器的内部机制，也为未来的模型优化提供了依据。
2025-03-11｜SJTU,EIT,UCAS,CAS,CASNC
http ://arxiv. org/abs/2503. 08751v1https ://qiwang067. github.io/diswm
本研究提出了一种名为“DisentangledWorldModels”（DisWM）的新框架，旨在解决在复杂环境中进行视觉强化学习（VRL）时所面临的样本效率低下问题。传统的强化学习方法通常依赖于大量的环境交互来学习有效的策略，但在具有视觉干扰和动态变化的环境中，这种方法的效率显著降低。DisWM通过从干扰视频中提取语义知识，利用离线到在线的潜在蒸馏技术，促进了跨领域的知识转移。这种方法使得代理能够在未见过的任务上提高学习效率，从而应对环境变化带来的挑战。实验证明，DisWM在多个基准测试中表现优越，展示了其在不同环境下的适应能力和有效性。
DisWM框架主要包括三个阶段：首先是“离散表示的预训练”，在这一阶段，通过对干扰视频进行无动作的视频预测模型训练，提取出离散特征。其次是“离线到在线的潜在蒸馏”，该步骤将从预训练模型中提取的语义知识转移到世界模型中，确保模型能够适应新的任务环境。最后是“离散世界模型的适应”，在这一阶段，通过结合在线交互中的动作和奖励信息，对世界模型进行微调，增强模型的表示能力。具体而言，使用β-VAE编码器来实现潜在变量的解耦，并通过最大化重构损失和Kullback-Leibler散度来优化模型。这一系列步骤确保了模型在面对环境变化时，能够快速恢复并保持较高的样本效率。
在实验部分，我们在DeepMind控制套件（DMC）和MuJoCoPusher等基准测试上评估了DisWM的性能。实验结果显示，DisWM在多种任务中均优于现有的视觉强化学习基线，如DreamerV2和APV等。通过与这些基线的比较，我们观察到DisWM在样本效率和任务性能上具有显著的提升。此外，我们还进行了消融研究，以验证潜在蒸馏和解耦约束的有效性。结果表明，去除潜在蒸馏会导致模型性能的明显下降，进一步证明了该方法在处理环境变化时的重要性。通过对不同视频数据集的预训练，DisWM能够有效利用从干扰视频中获得的知识，提升在下游任务中的表现。
DisWM的工作原理可以简单理解为三个步骤。首先，研究人员利用没有动作的干扰视频来训练一个模型，这个模型能够识别和提取出视频中的重要特征，这一步被称为“预训练”。接下来，研究人员将这些提取的特征转移到另一个模型中，这个模型能够在新的任务中使用这些特征，这个过程叫做“潜在蒸馏”。最后，研究人员会根据新的任务要求，微调这个模型，确保它能够在面对不同的环境变化时，依然表现良好。这种方法的核心在于利用已有的视频数据来提高模型的学习效率，从而让它在新任务中更快地适应和学习。
2025-03-12｜ZJU,VIVO
http ://arxiv. org/abs/2503. 09509v1
本研究提出了一种名为ViM-VQ的高效后训练向量量化方法，旨在解决视觉Mamba网络（ViMs）在低比特量化中的性能挑战。ViMs基于选择性状态模型（Mamba），在处理视觉任务时展现出显著的潜力。然而，现有的向量量化（VQ）方法在应用于ViMs时面临着量化误差、内存消耗过高和校准过程冗长等问题。具体而言，ViMs的权重中存在许多异常值，这些异常值显著放大了量化误差。此外，现有的VQ方法在多层网络中表现不佳，导致了次优的性能。为此，ViM-VQ结合了快速的凸组合优化算法和增量向量量化策略，旨在有效地更新和确认最佳代码字，从而减轻量化过程中的截断误差。实验结果显示，ViM-VQ在各种视觉任务中均实现了最先进的低比特量化性能。
ViM-VQ方法的核心在于其两个创新组件：首先，采用快速凸组合优化算法，该算法能够高效地初始化和更新多个候选代码字和软分配比率。通过利用欧几里得距离快速识别最接近的候选代码字，ViM-VQ能够显著加速最佳代码字的搜索过程。其次，增量向量量化策略则通过逐步确认最佳代码字，缓解了由于分配转换引入的截断误差。在每次校准迭代中，算法会动态调整候选代码字，以确保量化子向量保持在最优范围内。此外，ViM-VQ还引入了基于任务目标的损失函数，确保在校准和推理阶段网络输出的一致性。这种方法不仅减少了GPU内存的使用，还降低了校准步骤的复杂性，确保了在极低比特量化设置下的优越性能。
为了验证ViM-VQ的有效性，研究团队在两个主流视觉Mamba网络（ViMs）上进行了广泛的实验，涵盖了图像分类、语义分割、目标检测和实例分割等四大计算机视觉任务。实验中，ViM-VQ在不同的比特宽度下表现出色，尤其是在2-bit和1-bit量化条件下，仍能保持高准确率。具体而言，在使用ViM-T网络进行图像分类时，ViM-VQ在3-bit、2-bit和1-bit量化下分别达到了74. 79%、72. 17%和69. 93%的准确率，显著超越了其他先进的VQ方法。此外，在MSCOCO数据集上的目标检测和实例分割实验中，ViM-VQ在2-bit量化下也表现出色，证明了其在极低比特量化条件下的稳健性。整体而言，实验结果表明ViM-VQ在低比特量化中提供了极具竞争力的性能，适合在资源受限的边缘设备上部署。
ViM-VQ是一种新方法，旨在帮助计算机更好地处理视觉任务，比如识别图像或视频。它的工作原理是将复杂的模型“压缩”成更小的版本，以便在手机或其他小型设备上运行。这种方法主要由两个部分组成。第一部分是快速找到最合适的“代表”数据点，这些点可以帮助计算机理解和处理视觉信息。第二部分是逐步确认这些代表数据点，以减少在转换过程中可能出现的错误。这种方法的好处在于，它不仅能减少计算机所需的存储空间，还能加快处理速度。通过一系列实验，研究者发现ViM-VQ在处理图像时表现得非常优秀，即使在数据量较小的情况下，仍能保持高准确率。这使得ViM-VQ非常适合在资源有限的设备上使用，比如智能手机。

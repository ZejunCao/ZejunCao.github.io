2025 Google I/O 大会开发者主旨演讲干货盘点

仅用于站内搜索，没有排版格式，具体信息请跳转上方微信公众号内链接

在今年的GoogleI/O大会上，我们展示了如何在Google的各个平台进行应用构建，并利用GoogleDeepMind的卓越AI模型实现创新，大展拳脚。以下是开发者主旨演讲中的重磅亮点内容，不容错过。
GoogleAIStudio是评估模型并着手利用GeminiAPI开始构建的最便捷途径。
GoogleAIStudio
http ://aistudio. google.com/
着手利用GeminiAPI开始构建
https ://blog. google/technology/developers/google-ai-developer-updates-io-2025/
GoogleAIStudio助你轻松驾驭GeminiAPI:我们已将Gemini2. 5Pro集成到原生代码编辑器中，方便大家更快完成原型开发。通过与GenAISDK的深度优化，无论是文本、图像还是视频提示，皆可即刻生成Web应用。不妨从简单指令开始尝试，或者从我们的示例应用中汲取灵感。
示例应用
https ://aistudio. google.com/apps
GeminiAPI赋能，打造代理式体验:通过GeminiAPI及URLContext等创新工具，充分发挥Gemini2. 5的卓越推理能力，构建智能体。URLContext让模型仅需一个链接，便能轻松获取网页中的上下文信息。此外，我们还宣布了GeminiSDK将支持模型上下文协议(MCP)定义，简化开源工具的集成与使用。
Gemini2. 5Flash原生音频登陆LiveAPI:打造专属代理式应用，不仅\“能听会说\“，更能随心定制模型的声音、语调、语速和整体风格，支持多达24种语言。Gemini2. 5Flash原生音频功能精准地捕捉对话节奏，有效过滤背景杂音，让人机沟通如行云流水般自然顺畅。
能听会说
https ://aistudio. google.com/live
借助Stitch，生成高质量UI设计:这款AI赋能的创新工具能自动生成精美用户界面和相应前端代码。通过聊天即可迭代设计、调整主题，还能将成果一键导出到CSS/HTML或Figma，方便后续工作。立即试用Stitch，让UI设计快人一步:
http ://stitch. withgoogle.com/
异步编程智能体Jules开启公测:Jules是一款能与GitHub代码库直接对接的并行异步编程智能体。无论是版本升级、编写测试、功能更新还是修复bug等，Jules都能代劳。它能启动云虚拟机，在代码库中进行协同编辑、运行测试，待你确认无误后，便可从其分支发起拉取请求。
Jules
https ://jules. google/
了解我们为简化跨设备卓越体验开发所做的努力。
跨设备卓越体验开发

生成式AI重塑应用体验:生成式AI能让应用变得更智能、更懂你并具备代理能力，全面提升用户体验。我们已推出基于GeminiNano的全新MLKitGenAIAPI，助力端侧常见任务。我们展示了一个AI示例应用Androidify，这个应用能将自拍变为专属安卓机器人。深入了解Androidify的搭建过程，并查阅开发者文档，即刻开启你的创想。
MLKitGenAIAPI
https ://android-developers. googleblog.com/2025/05/on-device-gen-ai-apis-ml-kit-gemini-nano. html
Androidify的搭建过程

轻松适配五亿设备，打造至臻应用体验:Android移动应用是手机、折叠屏设备、平板电脑及ChromeOS的基石；今年，我们将帮助大家进一步将其扩展至车载系统和AndroidXR。别忘了运用Material3Expressive，让你的应用脱颖而出！
AndroidXR

Material3Expressive

AndroidStudio集成Gemini-AI智能体助力高效开发:AndroidStudio中的Gemini是一款由AI驱动的编码助手，旨在提升开发者在各个开发阶段的工作效率。我们带大家预览了Journeys功能，这个智能体能协助开发者编写和执行端到端测试；还有VersionUpgradeAgent(版本升级智能体)，让依赖项的更新变得更加简单方便。欢迎探索AndroidStudio中的Gemini，体验这些智能体如何让你事半功倍，快速打造卓越应用。
AndroidStudio中的Gemini
https ://developer. android.com/gemini-in-android

我们致力于帮助开发者简化网页开发过程，无论是优化UI、加速调试，还是开发AI驱动的新功能，让开发者可以更轻松地打造卓越的网页体验:
https ://developer. chrome.com/blog/web-at-io25
只需几行CSS和HTML代码，轮播界面唾手可得:用CSS打造精美轮播界面，首次渲染即可交互。Chrome135引入了若干全新的CSS构建基元，显著简化了轮播界面和其他屏外UI元素的开发！开发者可以运用熟悉的CSS技能，以极少时间成本打造出内容丰富、交互自然、顺滑流畅且体验友好的轮播界面。

Gemini能为你的开发工作带来无限可能，但有时，训练和优化自己的专属模型往往能带来更佳表现。为此，我们推出了Gemma-我们的开放模型系列，致力于提供前沿的技术，并让模型可以轻松适配各种设备。
Gemma3n开放早期预览:得益于前沿研究创新，该模型仅需2GB内存即可流畅运行。它是首款采用全新、先进的移动优先架构构建的模型，该架构亦将驱动下一代GeminiNano，旨在为便携设备提供顶尖的端侧AI性能。

MedGemma-我们功能最强大的医学开放模型，擅长医学文本与图像多模态理解:它基于Gemma3开发，如果你想打造医疗健康相关AI应用，可以以此模型为基础，进行微调和改进。模型体积小，推理效率高；开源设计确保你能灵活微调，并在自己熟悉的平台运行。MedGemma已整合入HealthAIDeveloperFoundations，欢迎体验。
HealthAIDeveloperFoundations
https ://developers. google.com/health-ai-developer-foundations/medgemma
ColabAI智能体赋能，重塑编码体验:在Gemini2. 5Flash的强力驱动下，Colab能帮助你轻松驾驭模型微调等复杂任务。我们展示了以AI为先的全新Colab如何助力开发者构建UI，帮助大家节省大量编码时间。

DolphinGemma，全球首个专为海豚设计的大语言模型:我们携手佐治亚理工学院与WildDolphinProject(野生海豚项目，非营利组织)的科研团队，利用数十年来积累的珍贵野外研究数据对DolphinGemma进行微调，助力科学家进一步理解海豚的交流方式。

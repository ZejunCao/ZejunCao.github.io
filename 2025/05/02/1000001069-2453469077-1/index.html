Claude深夜更新！DeepResearch+MCP入侵你的私人数据，威力很大！

AGI Hunt
发布日期:   2025-05-02

Claude深夜更新！DeepResearch+MCP入侵你的私人数据，威力很大！

仅用于站内搜索，没有排版格式，具体信息请跳转上方微信公众号内链接

这次的Claude，要让AI真正「连接」到你的整个数字世界！
Anthropic今天正式宣布了两项重量级更新：「Integrations」集成功能和「AdvancedResearch」高级研究能力。
这可不是简单的功能叠加，而是将彻底改变了人们与AI的交互方式！
见视频：
Claude的「触手」伸向了所有应用——
在去年11月发布的「ModelContextProtocol」（MCP）时，它只能在ClaudeDesktop上通过本地服务器使用。
而现在，Claude可以通过「Integrations」功能，像章鱼一样把「触手」伸向用户连接的各种应用和工具中！
这意味着，Claude不再是个「孤立」的助手，而是可以「看见」并「操作」你的整个工作环境！
真正入私到每个人的私人数据，数字空间中来了！
目前已支持连接的应用包括：
Atlassian的Jira和Confluence（码农们应该都知道）
Zapier（这个厉害了，意味着可以连接数千个应用）
Cloudflare
Intercom
Asana
Square
Sentry
PayPal
Linear
Plaid
而且Stripe和GitLab等公司，还有各个产品也即将加入这个阵营！
这有些什么用呢？
举几个例子：
连接Zapier后
——Claude可以自动从HubSpot拉取销售数据，并根据你的日历自动准备会议简报。
连接Atlassian后
——Claude可以协助你构建新产品、更有效地管理任务，甚至一次性创建多个Confluence页面和Jira工作项。
连接Intercom后
——Claude可以更快地响应用户反馈。当用户报告问题时，Intercom的AI代理Fin可以在Linear中提交bug。你可以与Claude聊天来识别模式，并使用Intercom的对话历史和用户属性进行调试。
最让人兴奋的是，开发者可以在30分钟内创建自己的Integration，将Claude与任何工具连接起来！
任何工具！
BryanHelmig（Zapier联合创始人兼CTO）对此表示兴奋不已：
Zapier很高兴成为这次发布的一部分！
除了Integrations，Anthropic还升级了Claude的Research功能，叫做AdvancedResearch——
现在Claude不仅会搜索，而且会「研究」，而且能力更是大大增强：
能研究的时间更长：从5分钟到45分钟不等（视问题复杂度而定）
搜索范围更广：不仅搜索网络和GoogleWorkspace，还能搜索你连接的所有应用
报告更全面：汇总数百个内部和外部来源，并提供清晰的引用链接
这是一个怎样的体验？
想象一下：你问Claude一个复杂的问题，它不会立即回答，而是先把问题分解成多个部分，对每部分进行深入调查，然后编写一份全面的报告。
这个过程可能需要45分钟，但这可是通常需要人工花好几个小时才能完成的研究啊！
FlorianS（AI研究者）已经联想到：
所以这是一种用于DeepResearch的MCP？让Claude可以深度搜索任何类型的连接应用（而不仅仅是传统搜索中的网络），长达45分钟🤯这实际上是一个非常酷的想法！
DeepResearch+MCP入侵私人数据，威力无穷！
简单说就是——
你的真•数字牛马分身来了！
有网友已经开始畅想各种可能的应用场景了：
业务数据库集成：经理们可以问Claude「我们在欧洲的第四季度损失是多少？」，然后它可以执行SQL，查询你的业务数据库，并给出答案。不需要分析师。
电子邮件应用集成：问Claude「过去几年谁参与了项目X，他们说了什么，还有什么悬而未决？」，它可以扫描你的电子邮件，找出角色、意见、任务，并给你一个总结。
MikeRussell（CreatorMagic创始人）表示：
非常酷！我刚刚在我的YouTube频道添加了一个视频，讲述了使用Claude+Zapier的所有惊人方式。团队工作不错！
还有用户SyedIjlalHussain打趣道：
按照这个速度，Claude很快就会开始要求休假并加入Zoom会议了🤔
不过，也有用户表示希望这些功能能够尽快向所有用户开放：
我什么时候能用上这个高级功能？这些功能对日常生活来说是必不可少的🙏🥹
目前，Integrations和AdvancedResearch功能已在Max、Team和Enterprise计划的测试版中提供，并将很快在Pro计划中推出。
也就是说，我的小小Pro账号还用不了……
同时，网络搜索功能现已在全球范围内向所有Claude.ai付费计划开放。
这些Claude的更新，标志着AI助手从单纯的「对话框」向真正的「工作伙伴」转变。
而Anthropic陆续推出MCP、Integrations、AdvancedResearch背后的野心，也正逐渐显露——
通过深度连接我们的数字生态系统，Claude将成为一个更加智能、无孔不入的你在数字空间里的牛马分身。

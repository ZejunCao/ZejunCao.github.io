坐标：北京
部门：ML&MIIC
工作内容（任一）

2. 参与具身智能相关VLA/VLN算法的训练和实机部署；
3. 参加具身智能相关算法竞赛。
岗位要求
1. 工程能力强，有工程项目经验；
2. 了解具身智能相关知识，进行过相关实验；
3. 能够保证四个月的实习期。
1. 熟悉Isaac相关仿真平台，有强化学习经验；
2. 有机器人实机经验，具有电气/硬件能力；
3. 有LLM/VLM训练/sft经验，了解大模型训练/推理流程；
4. Kaggle/ACMICPC获奖。
福利待遇
1. 提供有竞争力的实习生薪资待遇；
2. 参与具身智能前沿实机研究的实践经验；
3. 优秀的工作有机会凝练成顶会论文。

上海算法创新研究院-算法实习生
坐标：上海徐汇区
岗位职责
1. 负责空间智能、3D视觉相关研究，包括基于经典方法、Transformer、MLLM的三维空间和物体感知、理解与重建；
2. 负责具身智能仿真环境、具身合成数据等相关研究。
岗位要求
1. 计算机、自动化、电子等相关专业的在读本硕博学生，或者相关专业GapYear学生；
2. 需要在以下至少一个领域具备3个月以上的研究或项目经验：3D视觉、3D重建、具身仿真环境，具身合成数据；
3. 作为主要作者在NeurIPS/CVPR/RSS等顶会上发表过相关论文者优先；
4. 扎实的代码能力和算法能力，在Kaggle等AI和机器人相关比赛中获奖者将优先；
5. 较强的学习能力，创新能力，自驱力强，能够全时线下实习6个月以上。
你将获得
1. 前沿的空间智能、具身智能算法设计与系统研发经验；
2. 交大优秀导师带队，年轻且充满活力的团队氛围，与优秀的人一起做有挑战的事；
3. 参与具有成长潜力的项目，以及探索并解决开放性问题的成就感；
4. 行业领先的薪资。
投递邮箱：bo. zhao@sjtu. edu.cn
滴滴-算法实习生
坐标：北京
岗位职责
1. 在多种网约车品类交易模式与业务背景下，负责乘客与乘客间、乘客与司机间的匹配交易算法设计与优化，既保障司乘体验，又提高交易效率。
2. 针对不同时空场景下变化的用户和司机供需关系，设计灵活且自适应的价格调控策略，提升网约车整体收益，并与业务经营战略相配合。
3. 从数据驱动的视角，利用深度学习、机器学习、统计回归等方法，结合真实业务认知，参与品类生态算法体系的底层能力建设，构建智能决策链路，为品类交易与价格调控提供模型与数据支持。
4. 借由大模型的技术浪潮，利用AI打通数据、技术和业务的链路，通过现实语境赋予AI职责，使其在数据挖掘、智能经营、效率提升方面颠覆传统工作模式。
岗位要求
1. 计算机、电子、数学等相关专业毕业，具备扎实的数据结构与算法基础，具备良好的数学思维能力。
2. 熟悉常用数据挖掘算法及其应用，掌握机器学习算法（如LR、GBDT、XGBoost、RL、DNN等），并具备实战经验。
3. 编程基础扎实，熟练掌握C++、Java、Python、Scala或Golang等语言之一。
4. 了解一定优化理论（如ILP、凸优化、矩阵优化）、深度/机器学习（如强化学习、GAN、GMM）、概率统计与决策（如贝叶斯回归、MCMC、MDP）、随机过程（如Markov过程、泊松过程、随机微分方程）及特征工程等相关的数学知识。
5. 了解有关Transfomer、GPT、MoE等方面的知识，对Llama，DeepSeek等大模型框架与其在业务应用、效率提升方面的实践有相关经历。
6. 拥有海量数据处理与分析经验，熟悉Hadoop、Hive、Spark等大数据平台，并能结合SQL、Python、CSV等工具分析挖掘出重要洞察，以支持策略设计。
7. 热衷于了解业界前沿技术，洞悉人工智能国际会议的研究动态，不断提升自己在机器学习、运筹优化、机制设计、数理统计等领域的能力。
8. 专家级需求：具备机器学习相关的深厚工业界经验或学术积累，拥有成功的相关项目经验者优先，具备项目管理与团队管理经验者优先。
加分项
1. 有分布式机器学习和数据分析经验者优先；
2. 有大模型背景和深度学习训练与应用经验者优先；
3. 在人工智能相关顶级会议发表过论文者优先；
4. 具备ACM/ICPC、CCPC、NOI等经历，编程能力优秀者优先；
5. 拥有大型系统架构设计与实践经验者优先；
6. 在Kaggle等比赛中取得优异成绩者优先；
7. 有英语浏览和书写能力者优先。
投递邮箱：caoyujie@didiglobal. com
达摩院-医学影像分析实习生
坐标：北京
岗位职责
1. 探索AI+医疗的科研创新，包括但不限于：基于AIGC的癌症早筛算法研发、多模态医疗影像生成（CT/超声等）、扩散模型/flow在医学图像分析中的应用；
2. 结合真实医疗数据，实现算法原型开发、优化及产业落地验证；
3. 参与高质量学术论文撰写与顶会投稿。
岗位要求
1. 海内外在读硕士/博士，可长期在北京base线下实习；
2. 熟练掌握Python/PyTorch，具备多模态大模型（尤其是生成方向）实践经验；
3. 在CVPR/ICCV/MICCAI等顶会或AI+医疗领域发表过论文者优先；
4. 对医疗数据特性（如医学标注、病理先验）有强烈兴趣，能主动学习医学知识；
5. 具备优秀的问题解决能力和团队协作精神。
岗位待遇
1. 具有竞争力的实习补助与餐补；
2. 深度参与科研与产业结合的一线项目，积累论文与产品落地双重经验；
3. 表现优异者可获转正机会；
4. 提供顶级计算资源与医疗数据支持。
